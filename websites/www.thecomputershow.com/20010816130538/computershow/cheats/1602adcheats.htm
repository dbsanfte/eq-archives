    Downloads: PLATFORM: PC

    Cheat mode Hold [Ctrl] + [Shift] + [Alt] + W during game play. A prompt will appear at the bottom left corner of the screen. Type 2061 and press [Enter] at the prompt. Then, press A then press [Enter] to enable cheat mode. Enter one of the following codes to activate the corresponding cheat function.

      Code Result
    1. Shift + M Money
    2. Shift + Z Bricks
    3. Shift + K Cannon
    4. Shift + H Wood
    5. Shift + T Tools
    6. gold Gold
    7. fastgame All Building

    $10,000 Enter Columbus and give yourself a yellow flag for $10,000. Information in this section was contributed by ckaiser1.

    Free items When you buy something and your income is red or low, save the game. Load it again and you will have your money back.

    Text Editor Cheats All campaigns Note: This procedure involves editing a game file; create a backup copy of the file before proceeding. Use a text editor to edit the "game.dat" file in the game directory. Locate the line "Volume: -750, 0, -5??" and change it to "Volume: -750, 0, -581". Note: The "750" value may vary from game to game.
