13.12.2001 -  Ausfälle der Website
Nach häufigen Ausfällen der Website und erneutem Nachfragen bei ecore, wo das Problem liegt, bekam ich folgende Antwort:
 - Der Server bei Ecore, der die Nebelwelt hostet läuft zur Zeit sehr instabil. Die Firewall tut ihr übriges dazu, sie hatte irgendwelche Probleme verursacht, der den Server dazu hin zu häufigen Abstürzen führen lies. Ecore wechselt den Server aus. Das kann allerdings noch ein paar Tage gehn, aber sie arbeiten fieberhaft an dem Problem. 
Probleme können auftauchen, von daher werde ich abwarten, was sich da die nächsten Tage tut. Allerdings kann es wohl noch sein, dass die nächste Zeit die Website des öfteren ausfällt.
