/who <name> Zeigt alle Spieler auf, deren Name mit <Name> beginnen
/who <class> Zeigt alle Spieler im Spiel auf mit der angegebenen Klasse

Hotkeys

Hotkeys sind wohl voreingestellt, jedoch empfiehlt es sich bei der Einwahl, kurz vor dem Charakterscreen, das Optionsmenü aufzurufen und die aktuelle Belegung der Hotkeys anzuschauen. Je nach Ländercode variieren diese etwas, außerdem kann es sehr sinnvoll sein diese so anzupassen, dass sie denen bereits gespielter Spiele ähneln. Das hilft ungemein, im Spiel selber sich leichter zurecht zu finden.

 

Funktionen, denen man Hotkeys zuweisen kann:
Attack Mode Schaltet den Angriffsmodus an oder aus
Chat Log Der Chat wird in einer Datei mitgeschnitten oder der Mitschnitt wieder beendet
Chat Page Down Im Chatfenster eine Seite nach unten blättern
Chat Page Up Im Chatfenster eine Seite nach oben blättern
Consider Zeigt den Levelunterschied zum markierten Ziel an
Crawln/Down Charakter kriechen lassen oder Treppen, Leitern usw. herunterklettern
Get/Drop Item Markiertes Item nehmen oder fallen lassen
Ground Target keine Ahnung - wer Infos dazu hat bitte email an webmaster@nebelwelt.de
Help Ruft das Hilfssystem von DaoC auf
Jump/Up Charakter springen lassen oder Treppen, Leitern usw. hinaufklettern
Look Up Blickfeld nach oben richten
Look Down Blickfeld nach unten richten
Look Center Blickfeld zentrieren 'Normaleinstellung'
Mouselook Schaltet Mouse-Look an oder aus
Mouse Toggle Schaltet die Maussteuerung an oder aus
Move Backward Bewegt den Charakter rückwärts
Move Backward 2 Zusätzliche Taste um den Charakter rückwärts zu bewegen
Move Forward Bewegt den Charakter vorwärts
Move Forward 2 Zusätzliche Taste um den Charakter nach vorn zu bewegen
Nearest Enemy Der am nächsten stehende Feind wird markiert
Nearest Friend Der am nächsten stehende Freund wird markiert
Nearest Object Das am nächsten befindliche Objekt wird markiert
QuickDestroy Item Zerstört das auf dem Cursor befindliche Objekt
Quicksell Item Verkauft das auf dem Cursor befindliche Objekt an den Händler, ohne den Sell-Button mit der Maus drücken zu müssen

Gildenkommandos

Allgemeine Gildenbefehle
Befehle, welche von den Gildenmitgliedern ausgeführt werden können
/gc allgemeiner Gildenkanal Befehl, in der Regel gefolgt von einem weiteren Kommando 
/gc who Zeigt, wer von der Gilde online ist (funktioniert momentan nur in der selben Zone, PvP-Bereiche sind eine andere Zone, aber daran wird gearbeitet)
/gc info Zeigt alle Gildeninfos an
/osend oder /os Damit kann man im Offizierskanal schreiben, wenn man Zugang dazu hat.
/gu message oder /gu damit kann man im Gildenkanal schreiben, wenn man Zugang dazu hat.
/gc promote # der Befehl gefolgt von einer Nummer weist dem im Target befindlichen Gildenmitglied einen höheren Rang in der Gilde zu.
/gc demote # der Befehl gefolgt von einer Nummer weist dem im Target befindlichen Gildenmitglied einen niedereren Rang in der Gilde zu.
/gc invite <Username> Läd den Charakter mit dem Namen in die Gilde ein, geht nur, wenn der Charakter online ist.
/gc remove <Target> Entfernt das Subjekt im Target aus der Gilde.
/gc quit Beendet die eigene Gildenmitgliedschaft.

 

Befehle, die nur der Gildenleiter ausführen kann
/gc motd Damit kann der Gildenleiter ein "Motto des Tages" eingeben, das von allen Gildenmitgliedern beim einloggen gelesen werden kann.
/gc emblem Damit kann man das Gildenemblem und die Gildenfarben festlegen.
/gc aaccept Akzeptiert eine Einladung zu einem Gildenbündnis durch eine andere Gilde.
/gc adecline Lehnt eine Einladung zu einem Gildenbündnis durch eine andere Gilde ab.
/gc acancel Löscht eine bereits bestehende Einladung zu einem Gildenbündnis, die von der eigenen Gilde ausgesprochen wurde.
/gc ainvite Läd eine andere Gilde ein ein Gildenbündnis mit der eigenen Gilde einzugehen.
/gc aremove # Wenn man mehr wie ein Bündnis leitet werden die Bündnispartner durchnummeriert. Dieser Befehl löst dann das Bündnis mit der Gilde mit der entsprechenden Nummer. (#)
/gc aremove leader Dieser Befehl löst ein Bündnis mit einer anderen Gilde auf.

 

Befehle zur Gildengründung und Gildeneinstellungen
bei der Gildengründung müssen 8 Leute in einer
Gruppe sein und die Formierung der
Gilde bestätigen.
/gc form Formt eine neue Gilde, wenn alle Gruppenmitglieder zustimmen.
/gc accept Muß von allen 8 Gruppenmitgliedern bei der Gründung angegeben werden
/gc decline Verhindert das Gründen der Gilde
Gildenstruktur Nur vom Guild-Leader einzustellen.
/gc leader TARGET Gibt das Amt des Guildleaders weiter
/gc edit wird eingegeben mit einer Nummer (#)  zwischen 1 und 9, welche den Rang angibt, der bearbeitet wird.
/gc edit # ranklevel # verändert den Rang eines Gildenslots. Erste Nummer ist der aktuelle Rang, zweite Nummer die des neuen Rangs.
/gc edit # title NAME Weist einem Gildenrang einen Namen zu
/gc edit # invite (y/n) stellt ein, ob dieser Rang neue Mitglieder aufnehmen darf oder nicht
/gc edit # ochear (y/n) stellt ein, ob dieser Rang den Offizierskanal lesen darf
/gc edit # ocspeak (y/n) stellt ein, ob dieser Rang im Offizierskanal schreiben darf
/gc edit # gchear (y/n) stellt ein, ob dieser Rang den Gildenkanal lesen darf
/gc edit # gcspeak (y/n) stellt ein, ob dieser Rang im Gildenkanal schreiben darf
/gc edit # remove (y/n) stellt ein, ob dieser Rang Mitglieder niederen Ranges aus der Gilde entfernen darf
/gc edit # promote (y/n) stellt ein, ob dieser Rang Mitglieder im Rang verändern darf
/gc edit # emblem (y/n) stellt ein, ob dieser Rang Das Gildenemblem tragen darf oder nicht
/gc edit # view (y/n) stellt ein, ob dieser Rang den Gilden Info Screen abrufen darf
/gc edit # deposit (y/n) stellt ein, ob dieser Rang Geld auf das Gildenkonto einzahlen darf
/gc edit # withdraw (y/n) stellt ein, ob dieser Rang  Geld aus dem Gildenkonto abheben darf
/gc edit # achear (y/n) Alliancekanal noch nicht implementiert
/gc edit # acspeak (y/n) daher vorerst noch ohne Bedeutung
