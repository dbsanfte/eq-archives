04.10. Neues Artwork

Heute wurden Konzeptzeichnungen von einzelnen Kreaturen und Klassen veröffentlicht. Wer will kann sie sich hier ansehen. Sie sehen einfach nur noch gut aus.

27.09. Erste Bilder vom Interface

Es wurden heute zwei Bilder vom Interface von Duelfields veröffentlicht. Es wird einen Bereich für das Schlachtfeld geben auf dem die Kreaturen vom Spieler bewegt werden. Daneben zeigt ein Fenster die Animationen, wenn die Kreatur eine Aktion durchführt. Darunter befinden sich die Statistiken der Kreatur.

26.09. Start des Beta - Tests

Heute wurden die ersten Mails verschickt. Der Test hat nun offiziel begonnen. Viel Spass!!

21.09. Termin für den Beta - Test

Am 26. September ist es soweit und der Beta Test fängt an. Leider kann man sich nicht mehr für den Beta - Test anmelden. Es werden anfangs nur 50 Leute sein, die eine Benachrichtigung erhalten werden. Wenn alles wie geplant klappt, dann werden täglich immer mehr angemeldete Tester mitmachen können.
