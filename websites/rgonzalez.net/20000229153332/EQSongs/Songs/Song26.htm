ANGSTLICH's APPALLING SCREECH

Level: 26

Effect: Fear

Skill: Brass

Target: Area effect

Location: Kelethin

Does not stack with: Instill Doubt skill

Taunt: High

Quantified: N/A

Analysis by Rokenn Swiftsong, Karana server

Comments: This song will send out an area effect fear spell.

Strategies:

If things go bad deep in a dungeon this can give your casters time to get off an Evac or gate.  This can be especially important if your cleric can get out, as they can resurrect you saving you a long run back to your corpse.
Outdoors this song is a very powerful offense tool, whether grouped or solo.  Combined with Consonant Chain or a snare spell and you can take out blues/whites with ease, and make yellows a lot easier.
When using Chain/Fear be sure to use your Instill Doubt skill as well.  It cannot hurt and is a good way to practice that under used skill
This song will work in PvP zones on non-PvP servers.
[Mayner] This song can be used to open locked doors. (as can any spells that cause fear) by fearing the critter it runs opens the locked door and you follow. Careful you don't lock yourself in! This may also work on inoperable doors (that aren't locked but don't open at all) I do not know for sure.

Cautions:

[Rokenn] On the non-PvP servers you will not be able to start this song if you have a PC targeted. 
[Rokenn] This song has no effect in PvP combat. This is on the PvP servers only, unsure of the effect on duels on non-PvP servers.
[Della] Latest news from Team-PvP servers: This song will effect players that are not on your team, even if you are grouped with them.
When using this song outdoors be careful to not pickup unwanted green trash that may wander by.
This song will cause MASSIVE trains in most dungeons, so be careful using it underground.
The fear effect from Instill Doubt (ID) does not stack with Screech.  If you successfully ID while the target is feared you will still do the token kick damage, but the fear effect will not stick. But if you have ID'd a target then Screech the effects will stack
