Data Point Guidelines

 

Please use the following methodology for gathering data on song effectiveness.

1) Use only normal store bought instruments, ie Hand Drum, Lute/Mandolin, Horn, Wooden Flute.

2) Test the effects both with and without instruments.

3) Test Hymn of Restoration for standing regeneration rate on a Non-Troll race.  If you have any sort of item the speeds HP recovery (ie Rubicite breastplate) be sure to remove it first.

4) Do not adjust PvP damage figures.

5) If testing attack speed up/slowdown effects be sure you have a clean connection with little or no lag.

6) When sending in a report be sure to include your level :)
