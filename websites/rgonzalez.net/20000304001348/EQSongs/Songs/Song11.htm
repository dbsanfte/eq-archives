CINDA's CHARISMATIC CARILLON

Level: 11

Effect: Increases faction with NPCs

Skill: Wind

Target: Single/Zone

Location: Kelethin

Does not stack with: Itself, see cautions.

Taunt: N/A

Quantified: On average it seems to raise you one or two faction rankings. The size of each faction rank seems to be different so the song may raise you anywhere from 0 to 2 ranks. Using a wind instrument seems to enhance the effect slightly. People have claimed that in some cases using a flute raised their faction one more rank than just singing the song

Table data provided by Elthan.

Starting Faction Adjusted Faction
Ready to attack No Change
Threateningly Dubiously
Dubiously Indifferent
Apprehensively Amiable
Indifferently Amiable
Amiable No change*
Kindly No change*
Warmly No change*
Fondly No change - best faction possible

If you are near the borderline to the next faction level this may push you over the edge.

Analysis by Kaearia <Platinum Sphere> Karana server:

Comments: This is a really nifty song at the lower levels when money is scarce. At higher levels, I find myself using it only for quests and occasionally when buying/selling big ticket items to guarantee I'm getting the best possible price.  To use this song target the NPC whose faction you wish to adjust and sing.  The effect lasts about 15 to 20 mins or till you zone.  It will affect all NPCs in the zone that have the same faction as the one you targeted, ie use it on the Militia Guard at the city gate and your faction is adjusted for all Militia Guards in the zone. This is the first wind based song. It is not a good practicing song as the song only lasts a few seconds. If you want to practice your Wind Instruments skill, wait until level 16 and use Tarew's Aquatic Ayre.

Strategies:

[Rokenn] This song can give you access to quests that you normally would not be able to get/complete.  It can also help with getting a better reward on random item type quests, as the better rewards seem to come with a higher faction.
[Rokenn] You can use this song to get around border line KoS (Kill on Sight) NPCs.  If a NPC cons as 'Glares at you threateningly' they will normally attack if you hang out near them for too long.  With this song you can adjust their attitude so they only 'Glower at you Dubiously' allowing you to safely travel in the zone.
[Liquidated] How to get the best price from a Glowering merchant: Normally using this song will raise a Glowering merchant to Apprehensive allowing you to buy and sell.  A trick you can use to get better prices is to sneak behind them (this gives an Indifferent con) then sing this song.  Liquidated reports that this will sometimes raise faction to Amiably.  This could also be used on quest NPCs.

Cautions:

This song uses the same icon as Lament, Lullaby, and Charm. Be careful that you're using the Carillon and not one of these offensive songs. The Carillon will give an immediate message "A wave of enchantment washes over you." If you do not see this message immediately upon starting the song, STOP WHATEVER SONG IT IS YOU'VE STARTED.
Of all things, counters will block this song. The spell effects will go off as if it worked, but you will get a message saying "you cannot see your target," and your faction will not change. Make sure you have clear line of sight with the target. Small differences in height (if your target is standing in a slightly recessed area for instance) will cause it to fail as well.
[Rokenn] Only one faction group at a time can be effected by this song.  So if you used it to raise faction with the Freeport Militia, then used it on the merchant to get a better price, you will cancel the faction raise with the Militia. So be careful if you are using this to avoid borderline KoS NPCs.
[Rokenn] This song has no effect if the NPC is already aggro'd on you, the only way to fix this is to kill them, or zone (unless there is a Cleric or Enchanter nearby that can Atone/Mem Blur).
[Rokenn] Some mobs have no faction that can be adjusted (the Avaiks in South Karana for example) and will be unaffected by this song.
