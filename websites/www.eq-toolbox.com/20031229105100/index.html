  EQ Instant Messenger take 2.
Posted by Ramsus on Monday, December 08 @ 22:41:08 EST (34 reads)
(comments? | Score: 0)
Synapse writes:

I know a lot of people have been looking for alternatives to the buggy worthless piece of shit EQIM client, so I thought I'd share a piece of information.

An open source client for EQIM can be found at http://teqim.sourceforge.net. There is a console version (text only, no GUI) called EQIMu, and a Trillian plugin (requires Trillian Pro) called TEQIM under development. Both are available in source code and binary form. They work for all servers without modifications, and dont crash... unlike the EQIM client from SOE.
