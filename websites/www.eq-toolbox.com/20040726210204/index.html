  New EQWindows type program
LAX writes...
I thought I'd drop you a line that WinEQ, a free EQW clone from the same guy (me) that brought everyone EQPlayNice and the open source EQIM clients, has reached the beta stage and is ... amazing.

Here's a brief description:
------------
WinEQ is an application designed to increase the playability of EverQuest, as well as facilitate multitasking and ability to run multiple sessions of EverQuest.

Features
# Forces EQ to use Windowed mode
# WinEQ's manual mouse release prevents EQ from re-capturing the mouse automatically. It must be manually re-captured with the mouse release hotkey
# Automatically releases the mouse during loading and zoning (the cursor may not be visible while above the window until loading and zoning is complete, but it is released), and automatically re-captures unless the mouse was manually released
# Keeps a list of up to 50 separate profiles, each with different settings (EQ Path, window size and position, custom window title, global activation hotkey, and others) - Note that every window using a particular profile will be in the same position, even right after you finish moving one of them. If you wish to have two windows visible side to side, make sure they use different profiles!
# Allows resizing of the window
# Has a "Full Screen Emulation" mode, which is still windowed but hides the window border and title bar and uses your desktop resolution
# Allows the window to be set "Always On Top"
# Task tray icon waits patiently for you to use it to change WinEQ settings or to start EverQuest through its context menu
# Each WinEQ session has a right click-context menu available while the mouse is released. This temporarily pauses EQ, but allows quick access to some settings including profile selection
------------

I'm currently adding the product page and screenshots to the web site, but the news on the front page has the download link http://teqim.sourceforge.net

Highly recommended! Check it out! -R
