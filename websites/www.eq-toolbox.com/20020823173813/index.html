  If you crash getting into EQ
Posted by Gondola on Wednesday, August 14 @ 10:50:46 EDT (71 reads)
(comments? | Score: 0)
If you crash getting into EverQuest:

1. Edit your eqclient.ini file, and change the line that starts with UISKIN= to UISKIN=default
2. Delete or rename the UI_charactername_servernumber.ini file. This will be, for instance, UI_Thissis_21.ini for a character named Thissis on Cazic Thule. This will be different for everyone, AND for EACH CHARACTER you've loaded the new UI with.

Caution: Deleting files can be hazardous if you don't know wtf you are doing! Ask your friendly neighborhood 12 year-old to help you if you are afraid!
