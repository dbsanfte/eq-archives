MELEE COMBAT SKILLS (1H Slash, 1H Blunt, Piercing)

Available at Level: 1

Effect: Increases chance of hitting with weapon types.

Maximum skill: 200

Quantified: (lvl + 1)*5

Analysis by Rokenn Swiftsong, Karana Server

Comments: Not much to say here, these skills all go up pretty fast. It is generally a good idea to work on all them, as there is at least one Bard only weapon of each type in the game. 

Strategies:

If you have neglected one or more of these skills, do not despair, as they advance fairly quickly.  To maximize increase checks use the fastest weapon of each type you can find.  As more swings = more skill checks.
