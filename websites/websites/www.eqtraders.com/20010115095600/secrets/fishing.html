Fishing Lures

With the advent of the new tackle box and hooks experimentation has begun on "recipes" that use it. Such recipes have been confirmed to exist, but NO text (quests, recipes, etc.) is provided in-game at this point. Below are results of successful bait combinations from our message board. It is still unsure what the benefits are of using these lures, if anything. Skill checks for making these are based on fishing skill.

I've received a report that all lures on this list are trivial by skill 75, but have not had a chance to verify it.

Note that silk thread is made via the tailoring skill, not merchant-sold.
