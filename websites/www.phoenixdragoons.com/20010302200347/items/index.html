EVERQUEST ITEMS

NOTES
Hover the mouse over each item to get information about:

  • Loc :- its location
  • NPC :- the NPC who drops it
  • Desc :- a brief description
  • By :- who submitted this entry

You can also view the alphabetical complete item image list (112 items) collected thus far.

If you wish to help (something I could really use!) build this section, please see the bottom of this page.


VERY INTERESTING ITEMS
Only very rare items are collected here.

INTERESTING ITEMS
Interesting items are not too difficult to find and are often sold for two-digit plat amounts.

NOT SO INTERESTING ITEMS
These are common items that don't often spark interest (or don't hold that interest for long).

UNLISTED ITEMS
Most Everquest items are not yet listed here -- this section will continue to grow as long as there are items missing from these pages! If you would like to contribute by taking screenshots and properly capturing the item image and description, send me an email (siuan@phoenixdragoons.com) and I will coordinate with you.

Consistency
Consistency is the name of the game here -- every image is created using Paintshop Pro v6.0 in exactly these steps:

  1. Put the item inside a backpack (if possible), right-click on the icon, hold the right mouse button and move the cursor off the icon, then take the screenshot (press '-' on the keypad)
  2. Open the .BMP in Paintshop Pro, press Shift-G and choose 1.50 as the gamma correction value (this brightens the image)
  3. To capture the description, create a selection rectangle with coordinates (157,337) - (472,467)
  4. To capture an icon, ensure it is inside a backpack (if possible) and select the 47x47 region (this eliminates the background text found on inventory slots)
  5. Copy the item icon into the description image at (256, 7) as a new layer
  6. Save the description as: 'eqitem[NameOfItem].psp' and the icon as 'eqicon[NameOfItem].psp' (eg: eqitemSmallBronzeBoots.psp and eqiconSmallBronzeBoots.psp)
