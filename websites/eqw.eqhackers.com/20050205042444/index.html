4/12/03 - new version 2.35a
2.35a is up here. This version addresses the issue of the mouse locking in other windows while 1 is zoning. The method I am using has caused problems for EQW users in the past, primarily with sound. If you have new problems with this version set Multithreaded=FALSE in eqw.ini and post your experiences in the Bugs forum.

Keep in mind the EQW window switch hotkey currently doesn't work while zoning so you will have to use alt-tab to switch windows.
