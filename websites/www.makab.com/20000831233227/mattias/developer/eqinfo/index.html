EQ Info

A feature packed program for EverQuest players.
EQ Info is a nice program for players of the very popular on-line game EverQuest.
Features include: Skill information and monitoring, mana calculator, server monitor, class and race combination and starting cities, etc.
It all started as a mana calculator as the previously available ones did some minor things wrong (Intelligence over 200 not handled correctly and mana only for every 5th int). But I kept adding more things I would like to have myself. I hope you find this useful too!
More Information is available in the Readme file (for example upcoming features).

How to start using it:

  1. Install Java 2 version 1.3 if you do not already have it. If you are not sure you most likely do not have it.
  2. Download the program from the files section.
  3. Then it depends a little on the platform you are using. If on Windows for example you can just double click on that file, setup a shortcut to it or whatever. On most other plaforms run the command: 
	java -jar EQInfo_x_y.jar
EQInfo_x_y.jar is the file you downloaded. Where x is the major and y the minor version number.

Features:

  • Skill information.
  • Skill monitoring.
  • Mana calculator.
  • Server monitor.
  • Zone locations (zone entrances and other placess of interest) and descriptions (in character).
  • Spawns for each zone! Name, hp, level, race, class, location.
  • Parse log files with multiple characters!
  • Guild members listings.
  • pre 51 and 51+ skill caps.
  • List of lvl 51+ class names.
  • Nerfs and Enhancements to EverQuest since the release.
  • Race and Class combinations
  • Starting cities for all races.
  • Printing support. Print a character sheet!
  • Small (286KB).
  • And more...

Requirements:

  • Any computer with Java 2, version 1.3 or higher (Java 2). The Java Runtime Environment is enough (about 5MB)
