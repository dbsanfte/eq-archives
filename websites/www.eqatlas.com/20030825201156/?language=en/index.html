LoY Cartography System
Saturday, February 15th

Friday brought the announcement of the new cartography system that will be going in with the Legacy of Ykesha expansion. I haven't got the details yet, but hopefully we'll be able to transfer the maps people make around. Doing the maps still requires you to run along the walls and collect points to draw the map out. Needless to say, I've been doing this for years now, and have a program that does the same thing with the log files, and it takes hours to make a good map, so I'll guarantee few of you will be making a hobby of this new tool. However, it does add the neat feature of giving you an arrow to tell you which direction you're going, and putting your own markers on the maps.

Anyway, I'm going to look into hosting the files if they are able to be transferred around between people (I imagine they have to be somehow if you have local data), as well as possibly coming up with a converter for many of the map parses I already have collected for people to download. There will also be the GUI XML Creator for people to use if they want to include other maps (like the ones on this page) and this coming week the EQ Atlas PDF will be coming out, so stay tuned.
