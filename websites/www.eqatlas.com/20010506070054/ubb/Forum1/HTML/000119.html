This item can be bought from the Sisters island in OOT, at the inn where alcholic beverages are sold... I'm not sure whether they are sold other places, but I haven't done any extensive looking...

It costs around 5-6pp (depending upon CHA)...

The effect is a light healing spell which only cures for 35 hps, so whether it is worth buying is totally up to you...

Because it's a lore item you can only have one, which severely limits the usefulness of this item. It does come in handy to have one in extreme emergnecies though...
