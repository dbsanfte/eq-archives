A Word about Web-Browser Compatibility

These web pages make copious use of Cascading Style Sheets, which are the recommended standard for controlling the appearance of a web page. This helps to separate the instructions for appearance from the actual content of a web page, which is very much a desirable feature. Many older web browsers, however, cannot handle the style sheets or don't handle them properly -- most notably Internet Explorer versions before 5.0 and Netscape versions older than 6.0. If you're using one of those older browsers, then ... well it's about time you upgraded. Geez, your browser can't even handle the minimum recommended standards for web pages!
