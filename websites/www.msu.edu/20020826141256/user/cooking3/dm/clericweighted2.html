This list of Magelo Profiles was complied over the week of June 17, 2002, in order to have some comparison on different stats. While much of the information will (has) become outdated all of the magelo profiles should remain unchanged. Please excuse the poor HTML on some of the pages. Microsoft Excell was used to sort and produce totals of each of the data sets, if the page does not display properly please re-load the page (possibly more then once), which usually corrects the formatting errors. Note that profiles that do not include HP/AC/Mana totals are not included in the average totals due to the incorrect AC information that would skew averages.
