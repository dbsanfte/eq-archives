Warrior ガイド
ここではThurgadinから始まる長編Questを紹介します。

2001年夏現在、7段階目まで用意されているこのQuestでは肩装備が貰えます。

段階を重ねる毎に少しずつ性能が良くなって行きます。

第一段階：AC 1, INT 1, WIS 1, SvC 1
第二段階：AC 2, INT 2, WIS 2, SvC 2
第三段階：AC 3, INT 3, WIS 3, SvC 3
第四段階：AC 4, INT 4, WIS 4, SvC 4
第五段階：AC 5, INT 5, CHA 5, WIS 5, SvC 5, SvF 5
第六段階：AC 6, INT 6, CHA 6, WIS 6, SvC 6, SvF 6
第七段階：AC 7, INT 7, CHA 7, WIS 7, SvC 7, SvF 7, SvP 7

このQuestの特徴は生産スキルを使用する事で、数多くの生産スキルを用います。

第一段階：無し
第二段階：無し
第三段階：料理
第四段階：陶芸
第五段階：裁縫
第六段階：裁縫
第七段階：陶芸、裁縫、弓、宝石、醸造

これらの生産スキルの上げ方は、Trade Skillsを参照して下さい。

このQuestではSoVのThurgadinのFactionが必要になります。SoVで何もしていない人であればKoSでは無いので第一段階のQuestを受ける事が出来ると思います。

また、Giant系のFactionが下がります。途中途中でGiantを殺してアイテムを奪う個所が有り、Giant系は捨てる事になると思います。

適正Lvと必要人数は以下の通りです。

第一段階：Lv35以上の3人以上を推奨、1時間程度
第二段階：Lv35以上の3人以上を推奨、1時間程度
第三段階：Lv45以上の3人以上を推奨、9時間程度
第四段階：Lv35以上の3人以上を推奨、3時間程度
第五段階：Lv50以上の6人以上を推奨、10時間程度
第六段階：Lv50以上の3人以上を推奨、8時間程度
第七段階：Lv50以上の12人以上を推奨、10時間程度
