Vert Mod Skin Version 1.1: Tested (8-14-02)
 Updated: 8-14-02
Author: Vertebrae / Assassin of Tholuxe Paelles [Guardians of Valour]
Installation:
- unzip to everquest/uifiles/vert
- Edit eqclient.ini file. Change the line UISkin=default to Ukin=vert
(make sure the UISkin line comes after NEWUI=true)
- Start everquest, chatwindow type: /load vert

Extras:
- Optional WIS and INT caster player info windows are in extras.
Just rename them to EQUI_PlayerWindow.xml and overwrite in the main directory.
- Regular mouse cursor by default. For Ragebringer mouse pointer just rename the file in extras to mouse.tga and overwrite in the main directory.
