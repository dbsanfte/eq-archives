LillyanneMS
03-15-2005, 02:11 PM
Memorize the spells in the order you want them. Then (assuming you are using the default UI) right click on the book icon beneath the spell gems and select "save spell set". This will bring up a window where you name the spell set. Then you can right click the book icon and select that spell set to memorize at nearly any time.
