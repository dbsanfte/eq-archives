As initially conceived, this page was designed to have the format of nested messages, with the initial message being on the first level, and then the replies being indented, and the replies to the replies being indented even further than that, and so on.

However, this has turned out to be somewhat cumbersome. Since many threads on this site will be on-going one-on-one conversations, the nesting will eventually run off the page and start wrapping around the screen in wholly unaesthetic ways. So to make things a bit more orderly, I have indented all the replies to one level and left the initial messages unindented (so as to distinguish them from the forest of replies). And I have added links within each message so that one can begin a thread and then follow the links through all the replies without having to come back here to this page.

Hopefully, this will make things go a bit more smoothly and reduce some of the clutter. Enjoy.
