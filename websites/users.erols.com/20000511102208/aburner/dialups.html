I started out making this page really nice and pretty, with everything formatted in a nice table and such. Then I realized that most folks who would access this page would probably prefer it in a machine-parseable format. So here it is. If you know of any additions, please email me at aburner@erols.com.

NB: The "official" designations mean that an admin for that domain has provided the dial-up information. "Unofficial" means that either (A) someone has run a scan on the particular block and it seems to be all dial-up address space; or (B) a large ISP has blocked that particular address space without any problems or complaints. While all of these netblocks should be treated as though there may be some non-dial-up addresses within them, the "unofficial" ones should, perhaps, be given the squinky eye moreso than the "official" ones.

Unless, of course, you trust us. Heh heh heh.
