About these lists:

First and foremost, understand that this is only a list of concerns to be brought to the Live Team. This is not a list of promises, and I'm certainly not telling you that any or all of these items will be changed or implemented.

Also, there are three types of items that I don't have listed here. Epic Quests, Magic Resistances in Velious and Mid-level quests. Those three items were mentioned by enough classes that I've pulled them out for separate examination.

All items are currently marked with NONE for status. Obviously we've already started working on some items, you've seen the lists from the patch messages on the Test Server. I will be updating the status for items early next week to reflect those potential changes and with any responses from the team. I just wanted to get these lists out to you fresh.

I want these lists to be final as you see them now, but I know there will need to be changes made. So this list is not final. But if you want something added to or removed from the list, please be convincing about the change. At this point I don't want to tack on too many more items, since the team is already reviewing the original list.

Thanks for the input so far.

Alan


Rogues:


ITEM: Evade.

DESCRIPTION: Evade was broken when the size of the hate list was expanded for NPCs. It has to be fixed this in such a way that there won't be too much of a change from the way that Evade worked prior to the hate list change. This will, however, 'nerf' the skill a bit compared to how it works now.

STATUS: None

---

ITEM: Damage Shields and missiles.

DESCRIPTION: Thrown weapons will cause a damage shield to damage the thrower. This seems wrong from a constancy standpoint, especially since damage shields do not cause damage to an archer in this fashion.

STATUS: None

---

ITEM: Locked Chests.

DESCRIPTION: It would be nice if locked chests were introduced into the game. Making them static would, obviously, be pretty silly. But they could drop off NPCs.

STATUS: None

---

ITEM: Report Backstab messages to the group.

DESCRIPTION: It seems like this might be able to boost morale and allow the rogue to be even more impressed with his own skills. If this is added, though, it would be good to also add a filter for it to reduce spam.

STATUS: None

---

ITEM: Sneak and Hide functionality.

DESCRIPTION: Hide is cancelled when you loot, shop or trade. But the buttons don't 'ungray', and you don't get a message that tells you that you've stopped hiding. Perhaps this could be changed. And perhaps sneak could break in the same fashion, since sneak currently does not break in these circumstances. Perhaps consistency here would be good.

STATUS: None

---

ITEM: Haste for missile weapons.

DESCRIPTION: How about allowing haste to affect the speed at which weapons are thrown or bows are fired?

STATUS: None

---

ITEM: Illusion Masks.

DESCRIPTION: The illusions can't be clicked off by the wearer. Is this intentional?

STATUS: None

---

ITEM: Hide/sneak overwriting invisibility.

DESCRIPTION: As it is now, when any invisibility spell (including invisibility to undead and bard songs) wears off, a hidden rogue will become visible. The wearing off of the invisibility spell also takes out the rogues hide invisibility. Is this as it was intended?

STATUS: None

---

ITEM: Increased Safefall.

DESCRIPTION: How about increasing the safefall skill cap. Not enough to overtake monks, but enough to allow a bit more leeway for the running rogue. A slight increase for post level 50 rogues might be in order.

STATUS: None

---

ITEM: Sneaking in water.

DESCRIPTION: For some reason you can sneak and then move into the water and continue to sneak. But you can't begin to sneak if you are in the water. This should be consistent. Either sneak breaks when you enter the water, or you can start sneaking while in the water.

STATUS: None

---

ITEM: Stackable Poisons.

DESCRIPTION: This is a pretty common request. They would like stackable poisons, or perhaps the ability to make multi-charge poisons.

STATUS: None

---

ITEM: Poisoning in Full Screen mode.

DESCRIPTION: It seems that applying poisons can't be done in full screen.

STATUS: None

---

ITEM: Sneak skill increases.

Sneak is a continuous skill. But skill increases only happen when the skill is first started. Perhaps it should work more like swimming, and give a skill check at regular intervals.

STATUS: None

---

ITEM: Speed increase while hide and sneaking.

DESCRIPTION: Allow the movement increases to start earlier, say at 41st level. There have also been requests to allow a larger speed increase overall.

STATUS: None

---

ITEM: Tracking.

DESCRIPTION: Many rogues have asked for some form of tracking, ranging from full-fledged tracking to dungeon or city only tracking.

STATUS: None

---

ITEM: Sense Traps.

DESCRIPTIONS: Perhaps the range of this ability is too short. Some also think that the reset time is too long, making it tedious to look for traps, and easier just to trigger them and deal with the consequences.

STATUS: None

---

ITEM: Stun with Backstab.

DESCRIPTION: A sort of backstab, but with blunt weapons. So if the rogue is behind the opponent and uses his backstab while wielding a one-handed blunt weapon there would be a chance to stun the target. Such a backstab attempt might also do normal (not backstab) damage, to compensate for the stun chance.

STATUS: N/A – This is an idea for future use, no action is expected on this item

---

ITEM: Poison on thrown weapons.

DESCRIPTION: How about the ability to apply poisons to thrown weapons.

STATUS: N/A – This is an idea for future use, no action is expected on this item at this time.

---

ITEM: Soulbound Lockpicks.

DESCRIPTOIN: Rogues would like to have lockpicks that they can't lose. Perhaps as the result of a quest.

STATUS: N/A – This is an idea for future use, no action is expected on this item

---

ITEM: Apply Poison Animation.

DESCRIPTION: How about making the rogue crouch when he applies poison in a fashion similar to Bind Wound, so they don't need to stand after applying

STATUS: N/A – This is an idea for future use, no action is expected on this item

---

ITEM: Throwing Weapons.

DESCRIPTION: It would be nice if there were some longer range, stackable throwing weapons.

STATUS: N/A – This is an idea for future use, no action is expected on this item

---

ITEM: Set Traps.

DESCRIPTION: It would be interesting if rogues could to set traps. Something like a pet that would spring out an bite the next thing to come by. Only one at a time, hard to acquire components…

STATUS: N/A – This is an idea for future use, no action is expected on this item

---

ITEM: Hide/Sneak across zone borders.

DESCRIPTION: It would be nice if zoning did not break sneak and hide.

STATUS: N/A – This is an idea for future use, no action is expected on this item

1 - I know it's been beaten into the dirt, but the way evade currently functions is the ONLY way to counterbalance, to some degree, the damage we do as rogues. Bandaging is hardly a class defining ability, and even if it were, I would think rogues would be the best at it, as monks use their mental discipline to mend their bodies, and warriors just have tons of hp to suck up damage and rely heavily on the magic of others to heal their wounds.
