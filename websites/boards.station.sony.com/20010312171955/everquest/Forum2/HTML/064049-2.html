Some folks were looking for a "reader's digest" version of my rather long post.

It really boils down to my hypothesis that there are currently significant issues with the game ranging from class balance to epic items. I futher hypothesized that most, if not all, of these issues could have been avoided if Verant invsted in a structured and vibrant discussion with its customers.

Specific recommendations were:
1) Verant employed, class specific, advocates whose mission is to truly understand the wants and needs of their advocacy group.
2) Such advocates are either members of, or have significant input to, the tuning team.
3) All issues managed by advocates are communicated and over-communicated to the interested community via class specific, verant managed, web pages.
4) Time lines for resolution to issues are published and any slippage communicated in a timely manner.

Thats the best digest version i can do. There are some other reasonable useful datapoints in the orginal post; if, or when, you have the inclination, give it a read.
