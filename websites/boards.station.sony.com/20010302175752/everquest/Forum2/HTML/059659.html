Okay, so any improvement to a class is a NERF (shall I say: Slap in the face!) to every other class. Any change that is not an improvement is also a NERF by the popular definition. So you are saying that Verant should never change anything. Period. Just wanted to make sure I understand. I for one would not want to play this game if it never changed.

"So druids got better heals, should the clerics be happy because they have someone to back them up? NO they are pissed because now there suddenly is 150,000 people who can take their job."

Druids are not the only class that received healing improvements. The 150,000 number is clearly just made up so we can all ignore this part.

"This is where all the whines are coming from, people realizing their average time to find a group just shot way up."

Ahh, here is the heart of it. There will still be the same number of players and so the same number of possible groups. What this change has done is to increase the number of combinations of players that will "work" in a group thus increasing the total number of groups. Clerics and Warriors may have to spend a little more time LFG now but that is offset by Shaman, Druids, Rangers, Paladins, and SK's having a lot less time LFG. So the average time of all players LFG should go down. This is what balance is. If a warrior had to spend an average of 1 hour LFG and a Paladin had to spend 2 hours, then changing things so that both had an hour and 20 minutes would be balancing. Even though the warrior would have to spend more time LFG than before.

Be objective.
