About these lists:

First and foremost, understand that this is only a list of concerns to be brought to the Live Team. This is not a list
of promises, and I'm certainly not telling you that any or all of these items will be changed or implemented.

Also, there are three types of items that I don't have listed here. Epic Quests, Magic Resistances in Velious and
Mid-level quests. Those three items were mentioned by enough classes that I've pulled them out for separate
examination.

All items are currently marked with NONE for status. Obviously we've already started working on some items,
you've seen the lists from the patch messages on the Test Server. I will be updating the status for items early
next week to reflect those potential changes and with any responses from the team. I just wanted to get these
lists out to you fresh.

I want these lists to be final as you see them now, but I know there will need to be changes made. So this list is
not final. But if you want something added to or removed from the list, please be convincing about the change. At
this point I don't want to tack on too many more items, since the team is already reviewing the original list.

Thanks for the input so far.

Alan


Rangers:


ITEM: Defense Skill Cap.

DESCRIPTION: It seems fair to move the ranger defense skill cap up to match the knights' offense skill cap.

STATUS: None

---

ITEM: Dodge and Riposte Skill Caps.

DESCRIPTION: If it seems that rangers still need a bit more defense at the upper levels, raising the Dodge skill cap
would seem the most logical way to give them better damage avoidance. If we want to give them more offense
and more defense, then raising the Riposte cap would make sense.

STATUS: None

---

ITEM: Archery.

DESCRIPTION: The concern is that this combat skill is not effective enough, primarily because of what seems like a
bug. Archery seems to have a lot trouble with z-axis issues that cause the ranger to miss far more often than he
probably should. Also, why not push the ranger skill cap on archery up some, actually allowing them to be better
than warriors at firing their bows?

STATUS: None

---

ITEM: Tracking.

DESCRIPTION: Obviously people are anxious to get the changes that have been suggested, like scrolling and
sorting.

STATUS: None

---

ITEM: Trueshot Discipline.

DESCRIPTION: Rangers are of the impression that this discipline is not tuned up properly yet. They would like to
see it do more damage and/or hit more often. The problem with targeting here might be closely related to the
situation when this discipline is used, combats where a lot of characters are trying to take swings at one enemy,
not leaving any lines of sight for the ranger. Perhaps this discipline could allow this ranger to ignore the normal
issues with intervening PCs and NPCs. Also, the discipline could be tweaked up a bit just by giving it some sort of
minimum damage, perhaps a portion of the base damage of the bow used.

STATUS: None

---

ITEM: More Hit Points.

DESCRIPTION: Rangers would like more hit points.

STATUS: None

---

ITEM: Jolt.

DESCRIPTION: Players would like to see this spell lowered in level (49th perhaps) and have another, more effective
version added in its old spot.

STATUS: None
