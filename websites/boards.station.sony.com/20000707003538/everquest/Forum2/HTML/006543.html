I have suspected Win98 Registry of being the problem for many of the Problems people are experiencing so after doing all of the ussual ie reinstalls, getting right drivers, messin with cmos ect. I rebooted pressed F8 to get to safe mode command prompt, type scanreg/fix after it is done type scanreg/opt after that is done reboot.

This will not hurt anything so maybe it will work for you especially if you are like me and have had several upgrades on the same pc. Hmm goin to bed played for 6 hours in kunark without a problem lol it felt so good.
