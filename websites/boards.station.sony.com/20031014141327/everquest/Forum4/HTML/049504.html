quote:

Is it possible to use the keys from Evolution to upgrade your existing account, lets say for LoY, PoP and Velious and give the rest of the license to your family member that does not have EQ yet?

Nope! You can either upgrade an existing account with the expansions you don't have, or start a new account with ALL expansions. You cannot use the ones you didn't use to start a second account, it won't work.

EQ Evolution only comes with one key. As Sellis already said, it can be used to upgrade an existing account with missing expansions, OR create a brand NEW account with all expansions. You can't use the key for both, because a key can only be registered with one account (station name).
