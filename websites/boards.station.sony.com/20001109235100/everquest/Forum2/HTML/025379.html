I think it's important to note that neither suggestion will remove crossteam guilds. Removing crossteaming is not anybody's aim here.

Removing the ability to exploit the code to allow for immortal heals and buffs is the aim.

Under plan B (see my message above), you could crossteam, but the code would stop you from casting beneficial spells on your other-team guildmates. This would obviously add some complication to PvE for crossteam guilds, but it would just mean they'd have to do more planning, it would not actually "wreck" their setup. It WOULD stop their exploits though, which is the goal here.
