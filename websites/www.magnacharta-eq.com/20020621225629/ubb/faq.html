Registration
Die Registration ist nur notwendig, wenn Sie neue Beiträge schreiben oder auf Beiträge antworten wollen. Die Registration ist kostenfrei, Sie müssen nicht Ihren realen Namen angeben. Eine gültige eMail-Adresse ist jedoch notwendig.

Smilies
Sie haben sicher gesehen, das andere Nutzer Smilies in ihren Beiträgen oder anderen Foren benutzen. Smilies sind Tastaturzeichen, die bestimmte Gefühle ausdrücken sollen. Das Forum wandelt die Tastaturzeichen automatisch in Grafiken um, z.B. wenn Sie :o schreiben wird die Grafik eingefügt. Folgende Smilies können benutzt werden:

:) Lächeln
:( Stirnrunzeln
:o Verlegenheit
:D großes Grinsen
;) Zwinkern

Verwenden von HTML und/oder UBB Code
Sie können in Ihren Beiträgen HTML und/oder UBB Code verwenden, sofern der Administrator/Moderator diese Funktion aktiviert hat. Die HTML und UBB Code Funktionen können getrennt eingeschalten/abgeschalten werden. Ausschließlich Foren-Leiter können dies Funktionen aktivieren. Immer wenn Sie neue Beiträge schreiben wollen, wird Ihnen angezeigt, ob UBB Code und/oder HTML für das Forum verfügbar ist. Wenn HTML aktiviert ist, können Sie alle HTML-Befehle benutzen, achten Sie jedoch auf den korrekten Syntax. Beachten Sie den Syntax nicht, muß der Moderator oder Administrator Ihren Beitrag ändern. UBB Code ist das gleiche wie HTML, jedoch werden nur bestimmte Grundfunktionen unterstützt: Hyperlinks, Anzeige von Bildern, Fettschrift und Kursivschrift. Eine komplette Liste der aktuellen UBB Codes finden Sie hier.

Moderatoren
Moderatoren kontrollieren individuelle Foren, sie können in ihren Foren Beiträge ändern, löschen oder kürzen. Sollten Sie Fragen zu einem bestimmten Forum haben, dann wenden Sie sich bitte direkt an den Moderator des Forums.

Ändern Ihres Profils
Sie können Ihre Profildaten jederzeit selbst ändern, indem Sie auf den "Profil"-Link an der Oberseite jeder Seite klicken. Identifizieren Sie sich mit Ihrem Benutzernamen & Ihrem Passwort und Sie sehen Ihre Angaben. Sämtliche Angaben können geändert werden, bis auf den Benutzernamen.

Voreinstellungen anpassen
Als ein registrierter Forum-Benutzer können Sie Ihren Benutzernamen und Ihr Passwort für ein Jahr im Speicher Ihres Browsers ablegen. Sie müssen dann nicht jedesmal Ihren Benutzernamen und Ihr Passwort eingeben. Zusätzlich können Sie die Standardansicht für Beiträge wählen. Klicken Sie auf "Voreinstellungen" und wählen Sie Ihre Einstellungen.

Cookies
Das Forum verwendet Cookies zum speichern von folgenden Informationen: Datum/Zeit Ihres letzten Besuchs, Ihren Benutzernamen & Ihr Passwort sowie Ihre eingestellte Standardansicht für Beiträge, welche Sie unter Voreinstellungen gewählt haben. Diese Cookies werden im Speicher Ihres Browsers abgelegt. Die Cookies werden nicht benutzt um Ihre Aktivitäten aufzuspüren oder andere Funktionen auszuführen, sie unterstützen jedoch den einfachen Gebrauch des Forums. Sollte Ihr Browser keine Cookies unterstützen oder Sie diese Funktion abgeschaltet haben, dann können Sie die zeitsparenden Funktionen nicht nutzen. Wenn Sie ein privates Forum betreten wollen, sind Cookies zwingend notwendig.

Sie können alle Cookies löschen, welche von diesem Forum gesetzt wurden. Wählen Sie dazu "Voreinstellungen" und danach den Link "Löschen der Cookies".

Änderung Ihrer Beiträge
Sie können Ihre Beiträge jederzeit ändern oder löschen. Wählen Sie dazu einfach Ihren Beitrag und drücken Sie danach auf das Ändern-Icon in der Zeile "geschrieben von ...". Kein anderer kann Ihren Beitrag ändern oder löschen, außer der Forum-Moderator oder Forum-Administrator. An der Unterseite des Beitrags wird ein Hinweis angefügt, wann der Beitrag geändert wurde. Beachten Sie: wenn Sei Ihren Beitrag löschen wollen und Ihr Beitrag der erste Beitrag der gesamten Nachricht war, dann wird der gesamte Beitrag gelöscht (alle Antworten auf Ihren Beitrag werden auch gelöscht).

Hinzufügen einer Signatur
Sie können eine Signatur (wird auch in eMail's benutzt) an Ihre Beiträge anfügen. Das ist eine neue Funktion. Sind Sie bereits seit längerem registriert, dann können Sie Ihre Standard-Signatur über den Link "Profil" an der Oberseite des Forums bearbeiten. Haben Sie eine Signatur gespeichert, dann können Sie beim schreiben von Beiträgen entscheiden, ob Sie Ihre Sigantur verwenden wollen (Box "Signatur anfügen" aktivieren) oder nicht. Der Forum-Administrator kann wählen, ob die Signatur-Funktion benutzt werden darf oder nicht. Ist die Signatur-Funktion deaktiviert, erscheint die Box "Signatur anfügen" nicht, selbst wenn Sie eine Signatur gespeichert haben. Ihre Signatur können Sie über den Link "Profil" ändern.

Hinweis: Benutzen Sie kein HTML in Ihrer Signatur, verwenden Sie dafür den UBB Code (außer UBB Bild-Code).

Dateien anhängen
Aus Sicherheitsgründen können Sie keine Dateien anhängen. Sie können Text in Ihren Beiträgen ausschneiden und einfügen, HTML und/oder UBB Code (wenn verfügbar) verwenden um Hyperlinks zu externen Quellen herzustellen.

Nach bestimmten Beiträgen suchen
Sie können nach bestimmten Beiträgen suchen, basierend auf: einem oder mehreren Wörtern in den Beiträgen, einem Benutzernamen oder einem Datum; in allen oder bestimmten Foren. Klicken Sie dazu auf den Link "Suchen" am oberen Rand des Forums. Hinweis: Ankündigungen werden in die Suche nicht integriert.

Ankündigungen
Ankündigungen erfolgen vom Forum-Moderator oder Administrator. Niemand kann auf diese Ankündigungen antworten. Wenn Sie über bestimmte Ankündigungen schreiben wollen, müssen Sie einen neuen Beitrag im entsprechenden Forum erstellen.

Beiträge zensieren
Der Forum-Administrator hat die Möglichkeit, bestimmte Worte in den Beiträgen zu zensieren. Bitte beachten Sie, daß das zensieren auf bestimmte Wörter eingestellt ist. Zensierte Wörter werden durch Sternchen ersetzt.

Benutzername und/oder Passwort vergessen
Sie können sich Ihren Benutzernamen und Ihr Passwort per eMail zusenden lassen, sofern die eMail Funktion des Forums eingeschaltet ist. Alle Seiten, in denen Sie sich identifizieren müssen, enthalten den Link "Passwort vergessen?" unter dem Sie sich Ihren Benutzernamen und Ihr Passwort per eMail zusenden lassen können, sofern Sie bei der Registration eine eMail-Adresse engegeben haben.

eMail Mitteilung
Wenn Sie einen neuen Beitrag schreiben, können Sie die Option wählen, eine Mitteilung zu erhalten, sobald eine Antwort auf Ihren Beitrag erfolgt ist. Aktivieren Sie dazu einfach die Box "eMail Mitteilung" beim schreiben des Beitrags, wenn Sie eine Mitteilung erhalten wollen.

Wozu dient die Liste "die aktiven Beiträge vom heutigen Tag"?
Wenn Sie das Forum betreten, sehen Sie am oberen Rand des Forums den Link "die aktiven Beiträge vom heutigen Tag". Klicken Sie auf diesen Link, wenn Sie eine Liste der Beiträge/Antworten sehen wollen, welche heute geschrieben wurden.

Kann ich in privaten Foren suchen?
Zur Zeit können nur Administratoren und Moderatoren in privaten Foren suchen. Wir planen, das auch autorisierte Nutzer zukünftig diese Funktion nutzen können.

Was sind Archive?
Archive sind Bereiche in denen nur gelesen werden kann. Sie können keine Antworten auf einen Beitrag schreiben. Archivierte Beiträge können aus dem Original-Forum gelöscht worden sein, oder noch in dem Original-Forum existieren.

Was sind Icons?
Abhängig von der Konfiguration des Forums, können Sie Ihrem Beitrag ein bestimmtes Icon zuordnen. Zur Zeit sind 14 verschiedene Icons verfügbar, die bestimmte Gefühle oder Kennzeichnungen ausdrücken. Diese Icons reichen vom Schuldschein über Traurigkeit bis zur Warnung. Die Icons erscheinen in der Nähe des Themas von jedem Beitrag. Sehen Sie keine Icons, dann hat der Forum-Administrator diese Funktion abgeschaltet. (Vielen Dank an Kombat03 für die Idee!)
