W A S      I S T      U B B      C O D E ?

UBB Code ist eine Sammlung von HTML-Befehlen. Er erlaubt Ihnen, bestimmte Funktionen hinzuzufügen oder Ihren Beitrag zu verändern, wozu Sie normalerweise HTML-Befelhe einsetzen müssten. Sie können UBB Code verwenden, selbst wenn die HTML-Funktion für das Forum abgeschaltet ist. Sie sollten UBB Code anstelle von HTML verwenden, selbst wenn die HTML-Funktion für das Forum eingeschaltet ist, da wenig Befehle verwendet werden und er sicherer zu benutzen ist (falscher Syntax kann zu wenigeren Problemen führen).

Aktuelle UBB Codes:

URL Hyperlinks

NEU! Ist UBB Code eingeschaltet, brauchen Sie nicht länger den [URL] Code benutzen, um Hyperlinks zu erstellen. Schreiben Sie einfach die komplette URL in irgendeiner der folgenden Weise und der Link wird automatisch erstellt:

  • http://www.IhreURL.de
  • www.IhreURL.de
    Wenn die Seite ohne "www" erscheinen soll, müssen Sie die komplette Adresse mit "http://" angeben. Die Autolink-Funktion können Sie auch für https und ftp URL's verwenden (wenn UBB Code eingeschaltet ist).

    Der alte [URL] Code arbeitet weiterhin nach der unten beschriebenen Methode. Verwenden Sie den Link wie im folgenden Beispiel: (UBB Code ist Rot).

    [url]www.IhreURL.de[/url]

    NEU! Sie können jetzt richtige Links mit dem [url] Code erstellen. Verwenden Sie folgendes Format:
    [url=http://www.psychoterror.de]Der Wahnsinn im Web.[/url]

    In diesem Beispiel generiert der UBB Code automatisch einen Link der URL, welche zwischen dem UBB Code liegt. Bei einem Klick, wird der Link in einem neuen Fenster geöffnet. Beachten Sie, das der "http://" Teil der URL nicht unbedingt notwendig ist. Im zweiten Beispiel wird die URL dem Text zugeordnet, die nach dem = Zeichen steht. Verwenden Sie innerhalb der URL keine Anführungszeichen.
e-Mail Links

Um einen e-Mail Link in Ihrem Beitrag zu verwenden, benutzen Sie folgendes Format (UBB Code ist Rot):

[email]psychomaster@psychoterror.de[/email]

Im Beispiel wird automatisch ein Link zur e-Mail Adresse generiert, welche innerhalb des UBB Code steht.
Fett- und Kursivschrift

Sie können Kursiv- oder Fettschrift verwenden, indem Sie den Text zwischen den UBB Code einfügen [b] [/b] oder [i] [/i].

Hallo, [b]Mario[/b]
Hallo, [i]Mary[/i]
Liste mit Punkten/nummerierte Listen

Sie können Listen mit Punkten oder nummerierte Listen erstellen (mit Nummern oder Buchstaben).

Liste mit Punkten:

[list]
[*] das ist die erste Zeile
[*] das ist die zweite Zeile
[/list]

Erzeugt folgende Liste:

  • das ist die erste Zeile
  • das ist die zweite Zeile

Beachten Sie, daß Sie einen [/list] Code am Ende der Liste einfügen.

Erstellen von nummerierten Listen ist sehr einfach. Fügen Sie einfach TYPE=A oder TYPE=1 innerhalb Ihres Eröffnungs [list] Code ein. Wählen Sie TYPE=A für eine Buchstabierung von A bis Z. Wählen Sie TYPE=1 für eine nummerierte Liste.

Hier ist ein Beispiel:

[list type=A]
[*] das ist die erste Zeile
[*] das ist die zweite Zeile
[/list]

Erzeugt folgende Liste:

  1. das ist die erste Zeile
  2. das ist die zweite Zeile
Einfügen von Bildern

Um eine Grafik in Ihren Beitrag einzufügen, können Sie die URL der Grafik innerhalb des UBB Code verwenden (UBB Code ist Rot):

[img]http://www.psychoterror.de/images/beispiel.gif[/img]

In diesem Beispiel macht der UBB Code die Grafik in Ihrem Beitrag automatisch sichtbar. Hinweis: der "http://" Teil der URL ist NOTWENDIG für den [img] Code. Beachten Sie weiterhin: einige UBB Foren unterstützen den [img] Code nicht, um unzulässige Bilder nicht anzeigen zu können.
Beiträge mit "Anführungszeichen"

Um sich auf einen Text zu beziehen, welcher von einem anderen Nutzer geschrieben wurde, kopieren Sie den Text einfach in Ihren Beitrag, innerhalb des UBB Code (UBB Code ist Rot):

[QUOTE]fragen Sie nicht, was Ihr Land für Sie tun kann...
fragen Sie, was Sie für Ihr Land tun können.[/QUOTE]

Im Beispiel wird der Text vom UBB Code automatisch in "Anführungszeichen" gesetzt.
Code Befehl

Ähnlich dem [QUOTE] Code, fügt dieser Code <PRE> Befehle hinzu, um Formatierungen beizubehalten. Das ist besonders hilfreich, um z.B. Programmiercode darzustellen.

[CODE]#!/usr/bin/perl

print "Content-type: text/html\n\n";
print "Hello World!"; [/CODE]

In diesem Beispiel behält der UBB Code die Formatierungen automatisch bei.

Hinweise
Sie dürfen HTML und UBB Code nicht gleichzeitig verwenden, um die gleiche Funktion auszuführen. Weiterhin brauchen Sie beim UBB Code nicht auf Groß-/Kleinschreibung zu achten (Sie können [URL] oder [url] verwenden).

Falscher UBB Code:

[url] www.prime-web.com [/url] - schreiben Sie keine Leerzeichen zwischen dem UBB Code und dem Text, welcher kodiert werden soll.

[email]ted@prime-web.com[email] - der End-Code enthält jeweils einen Schrägstrich ([/email])
