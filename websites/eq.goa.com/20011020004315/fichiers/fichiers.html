Magelo EQ Items

Magelo EQ Items est un petit programme qui enregistre les objets de vos personnages quand vous jouez à EverQuest. Les objets sont ensuite téléchargés automatiquement vers un serveur de collecte. Les nouveaux objets envoyés apparairont dans les 24h dans la liste d'équipement de notre site. Magelo peut être lancé avant ou pendant que vous jouez à EverQuest, il ne doit pas être installé dans le même répertoire qu'EverQuest. Tout nouvel objet ramassé sera pris en compte après le changement de zone suivant.

magelo_EQ_Items.zip - (11Ko)


Patch Clavier

EverQuest est configuré par défaut pour les claviers Qwerty américains, si vous utilisez un autre clavier, Azerty au hasard :) , il vous suffit de copier le fichier Keyboard.txt suivant dans votre répertoire EverQuest. Vous pouvez personnaliser votre configuration clavier en modifiant ce fichier à l'aide d'un éditeur hexadécimal comme UltraEdit (www.ultraedit.com). N'essayez pas de le modifier avec un éditeur de texte banal comme Notepad, Wordpad ou Word, vous risqueriez de l'endommager et de brouiller votre configuration clavier. En cas de problème vous pouvez toujours retrouver le clavier Qwerty original en supprimant le fichier du répertoire d'EverQuest.

Keyboard.txt - Configuration Azerty
Keyboard.txt.ch - Configuration suisse Qwertz (à renommer Keyboard.txt)


EQu

EQu est une boîte d'outils pour EverQuest, on y retrouve un excellent éditeur de fichier de configuration (eqclient.ini), un moniteur de serveurs très complet, un analyseur de journal et un visualiseur de captures d'écran. Le programme se patch automatiquement comme EverQuest, de sorte à avoir toujours la version la plus récente. Un must !

EQu 4.x.x (2.68Mo)


EverMon

Evermon est un petit programme qui vous permet de voir l'état des serveurs. Après sélection il vous lancera automatiquement le serveur désiré quand celui-ci sera à nouveau fonctionnel.


EQ Server Scan

EQ Server Scan est un autre moniteur d'état des serveurs.


EQ Option

EQ Option édite le fichier eqclient.ini et permet de configurer facilement la couleur des messages affichés dans EverQuest. EQ Option propose plusieurs fonds d'écran pour tester la lisibilité des textes. Ce logiciel est développé par Phillip Davis d'eqcaster.com

EQ Option 1.4 (303Ko)


EverAmp

EverAmp est un petit plug-in Winamp pour commander la lecture de vos mp3 directement depuis EverQuest. Indispensable !


EQ Caster

EQ Caster dissèque le fichier spdat.eff des informations détaillées sur les caractéristiques des sorts. Ce logiciel est développé par Phillip Davis d'eqcaster.com

EQ Caster 4.1.0 (274Ko)


EQ Inside

EQ Inside permet d'extraire les images, les sons, les modèles 3D et toutes les autres resources contenues dans les fichiers d'EverQuest. EQ Inside analyse également le fichier de définition des sortilèges (spdat.eff) fournissant nombres d'informations intéressantes sur les propriétés des sorts. Cet utilitaire est développé par HackersQuest

EQ Inside 0.2b (627Ko)


ZoneConverter

ZoneConverter est un utilitaire complémentaire de EQ Inside convertissant les fichiers 3D d'EverQuest en un format lisible par les principaux programmes de modélisation 3D. ZoneConverter est également développé par HackersQuest.

ZoneConverter 1.1 (75Ko)

Musiques

Voici l'intégralité des musiques d'EverQuest au format midi. Pour en profiter pleinement il vaut mieux charger les wavefont (les fichiers .sbk dans le répertoire everquest) si votre carte supporte cette fonctionnalité (SB AWE 64 et SB Live par exemple)
eqmidi.zip (462Ko)
eqmusic.zip (304Ko)


EQ Log Parser

EQ Parser analyse et extrait des statistiques en tout genre du fichier de log d'EverQuest. Inutile donc parfaitement indispensable ! :)


EQ Log

EQ Log est un autre analyseur de log mais celui ci se contente d'extraire les /say /shout /ooc... Très pratique pour séparer les conversations d'un enregistrement des autres informations. Seul défaut les emotes ne sont pas extraits.

EQ Monk Parser

EQ Monk Parser : La même chose mais spécialement pour les moines.
