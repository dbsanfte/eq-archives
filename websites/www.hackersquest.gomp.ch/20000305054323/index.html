02.04.00 :: Latest ShowEQ source now in CVS
The source of ShowEQ is now stored in an CVS archive,
containing all revisions and changes of the code since
release 1.0 - The purpose of the CVS archive is to allow
developers to collaborate better and also to allow users
to get bleeding-edge access to the source.

To get access to the CVS archive you need a CVS client,
here are the details you need for anonymous access:

    Connection: pserver
    Host: www.hackersquest.gomp.ch
    Username: showeq
    Password: showeq
    Project: showeq

Those developers who wish to contribute and require
write access to the archive, please contact Slicer
in #showeq on irc.giblets.com

01.20.00 :: Did the experiment fail?
Why ShowEQ was released anonymous.

    The whole release of ShowEQ was meant as test
    for Verant and for the EQ-users as an experiment.
    We wanted to see how the general public and
    Verant would react after such thing.
    The man to blame errr, I mean to congratulate for such a
    great program as ShowEQ is Megaton.

    After this patch we'll see what Verant did to stop it.
    (Given that I can use some friends account to log in,
    because I have canceled mine a week ago.)

    I have decided to not put up the ShowEQ files yet,
    since it is not going to work after the patch.

    Maybe we will release the windows version of ShowEQ
    too, but propably not as open source.

    Megaton and me are working on an update for
    ShowEQ, to work with the recent client patch.
    Verant changed the packet headers and also
    encrypted the NPS spawn packet.

    We'll keep you updated!

    P.S.: Updated the members list
    (which I should have done months ago).
