You can hack the tutorial to load any zone, although some will start you in a code room or place tumpy too close to you without being able to target him. Change the starting coords if you can find them.

Or you could start a rogue, maybe get to 5 for better sneak, go naked in a dungeon when it's crowded and look around. When you find a spot that isn't camped (like imps in solb) disconnect your cable and go as far as you can in 30 seconds or so. Hide/sneak and reconnect before the time is up, you might even get lucky and live.

I retested it ... it works fine. And is very easy to use. Read the Readme.txt file first and follow instructions, otherwise you will see the standard EQ tutorial. I had to change the offset for the tutorial in the eze.ini file.

Not all zones are fully explorable though. But this is not the fault of EZE, it is where the tutorial starts one in a zone. For example Cazic starts you in a darkened room. Once you try to leave the room the program exits. The program even states this in a comment to the right of the name though.

It also has some zones that were never implemented or a perhaps part of Kunark.
