hehe yep, that Combat Fury + Holyforge discipline trick is nice. I've been able to get cripplings for 750-800 damage with top ATK buffs and Avatar.

You need to make an important choice when deciding your AA advancement path and the path you choose directly affects the AA abilities you want to focus on. Most people will agree that there are two main types of AA paladin sets. There is a third which focuses on healing but I don't think many dedicated paladins would choose this route. Regardless of which path you take, your first six AA points WILL go into Runspeed 3 and Innate Regeneration.

THE DEFENSIVE PALADIN "Tankadin"
Objective: The objective of the Defensive Paladin is to concentrate primarily on AA skills that will minimise the damage that you will take and also to increase your peak endurance(time you can tank a mob before needing healing attention). This is an example skill path of a Tankadin:

General:
Innate Regen 1/2/3 - 3aa
Runspeed 1/2/3 - 3aa

Archetype:
Combat Stability 1/2/3 - 12aa*
Combat Agility 1/2/3 - 12aa
Combat Fury 1/2/3 - 12aa**
ND 1/2/3 - 12aa

* Do Combat Stability before Combat Agility
** Purely for Slay Undead because all paladins should have this ability regardless of skill set

Class:
Physical Enhancement 1/2/3 - 18aa
Slay Undead 1/2/3 - 18aa

Planar Advancement:
Innate Defence 1/2/3/4/5 - 30aa
Lightening Reflexes 1/2/3/4/5 - 30aa

Planar Ability:
Planar Durability - 9aa

TOTAL COST: 159aa points


THE OFFENSIVE PALADIN "Zealadin"
Objective: The name stems from a classic Diablo2 paladin skill set of the same name and purpose. The idea of the Zealadin is to maximise damage and attack ratings at the expense of defense. This set includes no defense bonuses, these can be added as felt necessary. Zealadins generally should use 2handed weapons to be able to make use of the 2h specific PoP AA abilities.

General:
Innate Regen 1/2/3 - 3aa
Runspeed 1/2/3 - 3aa

Archetype:
Combat Fury 1/2/3 - 12aa


Class:
Slay Undead 1/2/3 - 18aa
Double Riposte 1/2/3 - 18aa
2hand Bash - 6aa


Planar Advancement:
None Initially, Add as required/sought fit

Planar Ability:
Coup De Grace - 12aa
Flash of Steel - 18aa
Fury of the Ages - 9aa
Speed of the Knight - 12aa
Knight's Advantage - 18aa

Note: It is recommended that you invest in some basic defensive skills like Combat Stability(level 3) as it always helps to have atleast some enhanced damage mitigation.

TOTAL COST: 129aa points
