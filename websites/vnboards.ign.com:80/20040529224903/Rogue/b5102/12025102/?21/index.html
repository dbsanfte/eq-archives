It procs at level 15.

Bottom line:

If you want the strongest backstabs and the best hit points (for third-string tanking) you can get, you want to be a barbarian.

If you want to be small and have the second-best strength and stamina, as well as a good dexterity for a teensy bit more procing, you want to be a dwarf.

If you want to be smart and learn poisons and stuff like that more quickly and easily than others at the cost of not being very strong, you want to be a dark elf (or a gnome if you like being small and funny-looking and being able to tinker).

If you want to be small, have a really high agility which gets you a tiny bit more AC (big deal) as well as a high dexterity for procing and a not completely hopeless strength and stamina, you want to be a halfling.

If you want to basically have the same stats as a halfling, only not quite as good, but be sexier and a tiny bit smarter, you want to be a wood elf.

If you want to be slightly more dextrous than a wood elf, slightly less agile, and pretty close to the same otherwise (a tiny bit more strength and stamina and a lot less wisdom, like that matters), you want to be a half-elf.

If you want to be the worst possible race for your class, you want to be a human.

If you've figured out by now that gear is everything for meleers and stats mean very, very little and you just want to play a race you like, play that race and have fun. ^.^

Any questions?
