Once you set all of this up, you won't have to do it again. Sneaky Amp will 'remember' all of your settings and from this point on, all you'll have to do is start up WinAmp before you log onto Everquest. One neat feature is that you have full control of Winamp while zoning as well. For more info on how to control Sneaky Amp from inside Everquest, download the readme.doc file and read it.
