Club Passing - Notation

Throughout this document on club passing I make use of the notation described in this section.

In general all patterns are listed in table form. An example is given below for a two person, 3-count pattern.

Passers 1 2 3 4 5 6 ...
Passer A RPlsrs LPrsls...
Passer B RPlsrs LPrsls...

Each row of the table shows the passes performed by one of the passers. The first column lists the passers in the pattern (Or the name of the pattern/trick if only one row is present). Each subsequent column represents one beat of the pattern, and in this column we list the type of throw the person for that row performs. The pattern is assumed to repeat from the end although in some cases more than one cycle of the pattern is shown.

Several different types of information can be used to represent one throw or beat of the pattern. The table at the end of this section describes the elements that can be used to describe one throw.

When both passers do the same thing, as above, only one of the passers needs to be shown:

3-count RPlsrs LPrsls

All self throws (including handoffs) are written in lower case to place emphasis on the passes that are done.

When more than two passers are involved, each pass is in the form:

[passer]-[pass]-[receiver]

For example

A-Rx2-F

is a right to right handed crossing double from passer A to passer F.

The first element is not used in a table format while the last element is only necessary if there are more than two passers in the pattern.

The definition of a pass is:

[hand][throw type][direction][# of spins]

Where:

hand is right (R) or left (L)

throw type is pass (P), Dropback pass (K), self (S), hand off (H).

direction is straight (|) or crossing (x).

# of spins is single (1), double (2) etc.

Each of these elements is optional, and if left out the default (as specified in the above table) is used.

In patterns such as 4-count or 2-count the left hand only does self throws. In these cases we leave out the left hand to simplify things. Also in patterns involving multiple people we often only show who the passes go to and not what hands are used or what timing is used. For example a standard three person line feed is:

A BC
B As
C sA

For patterns in which the passers physically move around (eg. rotating feed) the pattern shape is shown above the passes so you can see how the position of the passers relates to the passes that are being done.
