Thursday November 14 2002
Razor's Edge - Update
EQ Skins - 7:31 AM PST
I received an email from Daggun this morning, whom is the original creator of the UI Razor's Edge up to version 4 (Razor's Edge v4). He has informed us that he has not released a version 5 of his UI, and if/when he releases a newer version, he will not use the version 5 label. I have asked Daggun to repost his version 4 of his UI creation. He has devoted a lot of time into creating this UI, and running a website to support it here. My sincerest apologies to Daggun for this mix-up.

I also received an email from XtremeK apologizing to everyone for the mistake. He had received the non-official version from a friend and decided to share it, not knowing it was infected. I reported earlier that his account was missing, but in my haste to clear this up I missed it.

This is a good reason why one should not upload files one did not create. If in doubt, email the author of the UI mod.

Again, my apologies to everyone who was affected by this.

For Those of you Using Custom Skins
EQ Skins - 1:33 PM PDT
As you may have noticed with the recent patch, some custom UIs may not work. Several UIs have already been updated in the download area, hopefully more to come.

This from EQLive:

    If you are using a custom interface and are encountering problems with it, you may need update this file:

    Equi_animations.xml

    We added the following new animations:

    a_checkboxnormal
    a_checkboxpressed

    Custom interfaces will need to be updated to incorporate these changes. Until then, you may need to copy the EQUI_animations.xml file into the directory for your custom skin.

    Alan

UI Downloads section changes
EQ Skins - 8:53 PM PDT
You may have noticed a few new features in the downloads section of EQSkins. There is now a listing of the last 10 files added to the downloads area, which is next to the Category listing on the main page.

Tuesday September 17 2002
SidlWidl - UI Utility
EQ Skins - 5:57 PM PDT
Several people have emailed me asking for a guide to skin making, or utilities to help with creating their ultimate skin.

I'm no XML expert, but there is a very nice utility called SildWidl. It is still in Alpha test, but is very usable, and has a lot of features built into it. Instead of having to manually edit the xml files, figure out coordinates, etc... you can click, drag, and drop your way to a customized interface. To use the utility, you will have to have the Microsoft .NET framework installed.

For those of you interested, it can be downloaded from the SildWidl website, or from www.eq-toolbox.com, which also has several other XML utilities as well.
