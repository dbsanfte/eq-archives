Well if anyone wants some "realism" languages could be made more necessiary. If a human walk up from Freeport to Rivervale, it'd be nice if he/she took the time to learn enough Halfling to get by. Certainly all the NPCs should speak Halfling. It ought ot be like any other "country" where some merchants speak other languages, well or not so well, and some just plain don't speak any but their own. Hearty adventurers would then need to acquire the basic set of languages just to get by better. Yes there's the game communication problem. So give everyone a medium rating in other player languages. Over time in any place they'll learn it. It's VERY easy to learn languages, so use language to add more "local color" to the game.

I agree that languages should have more significance. To date, the only time it's been of any use to me was in several encounters my 'Light' character had with some Teir'Dal. They were excellent RPers, muttering under their breath and cursing me in their native tongue, not knowing that I understood them. It made for an interesting encounter.

I also agree that there should be some marker before speech that shows you which tongue is being spoken, if you're skilled enough in that language to recognize it. Someone might wish to speak with you privately in a public area by way of 'say' instead of tell, perhaps to clue others in that secrets are being shared. It would be nice to not have to say first "Switch to faerie tongue please."
