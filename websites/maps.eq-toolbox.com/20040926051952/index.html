So you got a corrupted zip? posted by Ramsus on Sep 19, 2004, 03:48am
You tried to download some maps and open the zip and it didn 't work. It said it was corrupted. Or maybe the file was empty? Here is what we know about that problem.

If you save the file to disk and then open it, it seems to work fine.

If you click open instead of save to disk, you have a chance of it corrupting the zip.

This error does not occur with the Firefox browser. It DOES occur with Internet Explorer.
If you are still using Internet explorer, you should really consider changing browsers. But I know some of you won 't, so, until we can figure out exactly what it is that some browsers don 't like about trying to open these files, just save them to disk and open them from there.

Thanks.
