Miscellaneous Commands

There are so many commands within EverQuest that frankly, I didn't know exactly how to categorize some of them. But don't be fooled...just because they don't have their own category doesn't mean they aren't useful!
Command Action
/anon or /a Use this command to make your character anonymous. Being anonymous prevents others from seeing your class, level, guild and zone. This command toggles on and off.
/assist Target a player that is engaged in combat then type /assist. This will automatically target and engage you in the combat. If you already know who the assist person will be you can type /assist NAME or set it as a hot key.
/autosplit This command will automatically split all coins with everyone in your party.
/consent NAME This command gives the person NAME consent to drag your corpse.
/consider or /c This command will use the green, white/black, blue, yellow, red system to help you determine if the targetted NPC or PC is your level or not.
/corpse You must be near your corpse to use this command. It will summon your corpse to your side.
/decline This command declines the offer to duel someone.
/duel or /d This command is used to challenge or accept a duel.
/exit This command will immediately exit you from the game. However, when this command is used your character is actually still in the game for a few minutes and can therefore be attacked. Use caution when deciding to use this command.
/follow Target the person you wish to follow and type /follow. This is a very temperamental command .. do not use it in areas with lava or high mountains. Also, you must be able to run as fast (or faster) as the person you are following. You must be grouped in order to use this command.
/friend NAME Use this command to add or remove someone from your friends list. Using /who all friends will give you a list of all those on your list who are online.
/help Use this command to get a list of all the available commands within the game.
/loc Use this command to see your location. This command can be used to navigate to other locations as well.
/log off This command turns your logs off.
/log on This command turns your logs on.
/loot This command will loot the corpse of an NPC which is near you.
/played or /p This command will give you the birthdate and "real time" hours played with your character.
/quit or /q This command will immediately exit you from the game. However, when this command is used your character is actually still in the game for a few minutes and can therefore be attacked. Use caution when deciding to use this command.
/random X Y This command rolls a magical die which will give you a random number between 0 & X or X & Y. (X & Y represent numbers) - Example, /random 100 150 would give me any number between 100 and 150.
/roleplay or /role Use this command show others that you are a role-player. While in Roleplay mode others can not see class, level and zone. (similar to /anon only others still see your guild tag) This command toggles on and off.
/split P G S C Use this command to split a set amount of money with your group.
P = Platnuim, G = Gold, S = Silver, C = Copper.
/surname or /sur Use this command to set your last name after you have reached level 20.
/time This command will show you the time of day in Norrath as well as the time of day in the real world.
/who or /w This command will give you a list of all players with certain parameters:
/who NAME: lists all with that name in your zone
/who all NAME: lists all in the game with that name
