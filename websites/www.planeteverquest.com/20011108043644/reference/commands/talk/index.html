Communication Tools

These commands are used as a means to help you communicate with others in a variety of ways within the game.

Command Action
/afk This command toggles on and off and will inform people sending you /tells that you are Away From Keyboard. To add your own message you can type /afk [message]. (Example: /afk grabbing a coke would inform those sending you /tells that you are Away From Keyboard grabbing a coke.
/auction Use this command when you are trying to sell something. It will announce your auction throughout the zone you are in.
/bug or /b This command will take you to a bug-reporting screen.
/emote or /e You can use this command to create your own emotes by using /emote (or /e) and then typing your message. Example /emote kicks the tree. By doing this the message will say "YOURNAME kicks the tree."
/feedback or /f This command will take you to the feedback screen.
/filter This command toggles the bad language feature on and off.
/group or /g Use this command to speak with everyone inside of your group at once.
/guildsay or /gu Use this command to speak with everyone in your guild at once.
/ignore This command is for the ignore feature. If someone in the game is bothering you type /ignore NAME. To see a list of everyone on your ignore list type /ignore and the list will appear. To remove someone from the list, simply type /ignore NAME and it will remove them.
/log off This command turns your logs off.
/log on This command turns your logs on.
/ooc Use this command to make an Out of Character statement for the entire zone to see.
/petition MESSAGE Use this command to contact the ingame GMs when you have a problem and need their assistance.
/reply or "r" This command will reply to whoever last sent you a /tell.
/report NAME Use this command to send a report to the GM of your server. It will send the last ten visible lines of text to the GM. Use this if you are being harrassed by a particular player or if that player is saying things which are in voilation of Varent's rules of conduct.
/say Use this command to chat with people that are near your location.
/shout Use this command if you have something to say to everyone in the zone.
/tell NAME Use this command to send a private message to someone in particular.
/yell or /y You can use this command if there are people near you and you are in trouble. People around you will hear "YOURNAME yelled for help.
