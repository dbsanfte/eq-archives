This package replaces Debian GNU/Linux's prepackaged version of the
mod_put module for Apache. This Apache-licensed module implements
handlers for the HTTP PUT and DELETE methods.

Michael Alan Dorman <mdorman@debian.org> assembled this package
from original sources by Lyonel VINCENT <vincent@hpwww.ec-lyon.fr>.
The original source can be found at:

http://hpwww.ec-lyon.fr/~vincent/apache/mod_put.html
