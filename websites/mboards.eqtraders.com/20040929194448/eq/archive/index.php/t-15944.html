Step 4: HP and Mana formula
HP = (Max * factor) * hpf
Mana = (Max * factor) * hpf
Step 5: Combine
Example:
So now how to use this:
Lets say I want to know stats on Lemon-Buttered Hynid Pie.
Step 1:
Type = 2
Pri stat = Agi
Sec stat = Sta
Step 2:
Pri save = PR
Sec save = DR
Max = 16
Step 3:
Pri stat = 5
Sec stat = 3
Pri save = 6
Sec save = 3
factor = .75
hpf = .5
mpf = 1
Step 4:
Agi = Pri stat = 5
Sta = Sec stat = 3
PR = Pri save = 6
DR = Sec save = 3
HP = (16 * .75) * .5 = 6
MP = (16 * .75) * 1 = 12
Therefore:
Lemon-Buttered Hynid Pie has the following stats:
Agi: +5 Sta: +3 Sv Poison: +6 Sv Disease: +3 HP: +6 Mana: +12
