Kaysha Soulsinger
I took some time to create this template, listing all possible Race/Class/Deity combinations that the game allows at this moment. The generic armor name at each entry is followed by the race who can make it, in brackets.

For the Race/Class combinations that have the option of Agnostic, the other deity options for that Race/Class combination can wear the armor listed at the Agnostic part as well - I didn't explicitely write that out though; so whenever you check a non-Agnostic entry for armors that can be worn, also include the Agnostic entry.

The Race/Class combinations that cannot have an Agnostic combination have been given a Non-Imbued entry which replaces the Agnostic entry for the purpose that I just explained. Each armor entry following a deity entry assumes the armor Imbued for that particular deity.

It could be that I have overlooked some things, made an error or two, but it was a lot to work through, so bear with me. If you do notice an error, please post as such in a reply. Beyond that, enjoy reading!
