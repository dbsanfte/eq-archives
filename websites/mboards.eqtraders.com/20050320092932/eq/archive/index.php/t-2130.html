EZ Board message boards are all part of one massive system that is operated by ezboard.com. The EZ Board company hosts the board and provides the interface so you can set it up and use it. All login information and authentication stuff is handled through their central database.

PhpBB is software that you use to configure and operate message boards that you host yourself. Two different PhpBB message boards hosted by two different people aren't connected to each other any more than any two documents that were both created with Microsoft Word. Your login information is held only within that single board and is completely unrrelated to your login information on any other message board anywhere.

Therefore, the only way to have the same username and password on a phpBB board and any other board is to sign up for each with the same username and password. If it is denying you the one you want on another board, it's only because someone else already took it on that other board. If it is saying that your email address is already used, then you probably signed up there some time in the past...or someone is using your email. See if they have some kind of username and password retrieval feature, or contact the admin.
