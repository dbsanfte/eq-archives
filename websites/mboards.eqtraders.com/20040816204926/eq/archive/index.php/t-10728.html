I just want to be 100% sure that I understand "cultural" fletching with the addition of the Karana-fletching.

ANY Wood Elf (regardless of class or god) can MAKE the "Wood Elf Cultural Fletching" items such as Blessed Faydark Thunderbolt, Mithril Scout Arrows, and Blessed Champion Arrows.

Any race can utilize these items. However the "Blessed" items can only be utilized by worshippers of Tunare. The Bows can be used by Warriors, Palidans, Rangers, and Rogues. The normal arrows can be used by Warriors, Palidans, Rangers and Shadowknights. The "Blessed" arrows can be used by Warriors, Palidans, and Rangers.

ANY worshipper of Karana can MAKE the "Karana Diety Fletching" items such as The Blessed Hurricane Bow, Steel Phalanx Arrows, or Blessed Steel Guardian Arrows.

Any race can utilize these items. However the "Blessed" items can only be utilized by worshippers of Karana. The Bows can be used by Warriors, Palidans, Rangers, and Rogues. As can the arrows.

So Wood Elf worshippers of Karana can make BOTH types of items, but would only be able to use the normal mithril arrows and the Karana items?

So from a Ranger perspective, only Human and Half-Elf worshippers of Tunare do not have access to some sort of special flecthing.

Is this synopsis a fair reflection of the way it actually works?
