Slippyblade
1. When a rogue take 10 Vials of Poisons (or 5 Vials of Poisons), which were made using an Advanced Poison Vial... and combine them in the coffin poison bottle (10 Slot Tradeskill Container)

A. The Combine is NO-FAIL. It Always Succeeds
B. The Combine returns the 10 (or 5) Advanced Poison Vials empty.

Not sure about the NoFail part but B is true. Though now that I think of it, I've never failed on a Multi-Dose combine...

2. If a Rogue attempts to make a Single Dose Vial of Poison that uses an Advanced Poison Vial as the container, if the attempt fails, an Empty Advanced Poison Vial is returned.

True. This is the case with ALL poisons though. Even the old world ones, we get the vial back on a failed combine.

3. If a Rogue takes a Single Dose Vial of Poison (that used an Advanced Poison Vial), and applies it to his weapon (basically using it), does this return an EMPTY Advanced Poison Vial?

Nope. When a poison is actually used, the vial is gone. It is considered part of the product and is consumed.

4. If a Rogue uses any Single Dose Vial of Poison, and applies it to his weapon, does this application return an empty vial of the type needed to make the poison?

Once again, nope. =) See above answer, it's the same.

5. There is no way at this time to combine poisons that are not of the advanced type into multiple charge versions?

Rogues the world over would throw parties the likes of which Norath has never seen if Old World poisons could be multi-dosed. =)
