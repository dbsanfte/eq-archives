Verdandi
We do have that feature. It is the "Find recipes that use this item" link. If it says "There are no known recipes that use" the item, then there are currently no recipes in the database for it. We may have goofed in setting up the recipe(s) that should use it, or we may not have a use for it yet.

We don't usually put things in unless we know what they're for since there are a lot of items flagged as used in tradeskills that are only for obscure quests that we don't want to mess with. But sometimes, especially with the flood of new information with an expansion, it'll happen.
Krazick
This is easy... I will take your example item "white primrose".

1. Go to the main site page (www.eqtraders.com)... on the left you see various menu items (About, FAQ, Search...Quests, Links, PayPal, and a entry field with Find button next to it).

2. Into that entry field type in 'White Primrose" and Click the Find Button.

3. You will get a page with the basic information on the item, with a Link on the Name... Click on that Link

4. The next page will provide more detailed information... with a Link Find recipes that use this item. Click on this link

5. And this returns (for White Primrose) that No Recipes are found that use it. (we don't have any yet recorded).

If we change it to 'Chub'... we get Rujarkian Chub in the list.. Clicking on the Rujarkian Chub and then the 'Find recipes that use this item... gets back 4 Recipes back.
