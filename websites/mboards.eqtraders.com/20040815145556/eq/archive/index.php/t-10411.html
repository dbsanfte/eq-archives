Cedir
A new JC recipe book has been added. It talks about combining Geode, Amethyst, and Polished Quartz with Silver, Electrum, Gold, Platinum, and Velium bars. It gives some hints about the stats, as well.

My interpretation of the stat hints was that Geode would give STR and STA; Amethyst would give SvD, SvP, and HP; and Polished Quartz would give INT and WIS (and possibly MANA).

The three new stones, I presume, only drop in LDoN zones. They are dropable. I bought some Geode off a merchant for about 25p a piece.

It's worth noting that the book specifically says that combines may be made with or without enchanting the metal bars. Does that mean that you will get stats either way? I'm not sure. Until I've tested with enchanted bars, I'm not going to be wasting suplies on non-enchanted ones.

I was only able to do a few combines, but here's what I've got so far:

Geode + Enchanted Silver Bar = Silver Geode Ring
AC+1, STR+2, STA+2, slot type 1

Geode + Enchanted Electrum Bar = Electrum Geode Ring
AC+2, STR+3, STA+3, slot type 1

Geode + Enchanted Gold Bar = Gold Geode Ring
AC+3, STR+5, STA+5, slot type 1

Geode + Enchanted Platinum Bar = Platinum Geode Necklace
AC+4, STR+7, STA+7, slot type 1

Geode + Enchanted Velium Bar = Velium Geode Bracelet
AC+5, STR+8, STA+8, slot type 1

Polished Quartz + Enchanted Silver Bar = Silver Polished Quartz Ring
AC+1, WIS+2, INT+2, MANA+5, slot type 1

Polished Quartz + Enchanted Electrum Bar = Electrum Polished Quartz Ring
AC+2, WIS+3, INT+3, MANA+10, slot type 1

Amethyst + Enchanted Silver Bar = Silver Amethyst Ring
AC+2, HP+5, SvD+2, SvP+2, slot type 1

Amethyst + Enchanted Electrum Bar = Electrum Amethyst Ring
AC+3, HP+10, SvD+3, SvP+3, slot type 1

The combines with Silver, Electrum, and Gold were trivial at 250. The combines with Platinum and Velium were *not* trivial at 250. They all sold back to vendors for *mere* 1 gold, 9 silver... so these items must be intended to be marketable (it's far too big a loss to be used for skill).

I'm a little dissappointed that these items only have slot type 1, unlike the new smithing recipes. Oh well.

I will edit this post if I find more.

Edit 9/06 7:30pm EST - Found more. I have now successfully done 9 out of the 15 new recipies.
