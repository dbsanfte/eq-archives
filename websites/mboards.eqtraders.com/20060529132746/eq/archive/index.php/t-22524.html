Hello fellow tradeskillers,

I recently created this guide for working with / enhancing OoW augments for the benefit of my guildmates (Celestial Navigators, Vazaelle -now Maelin Starpyre- server).

Yes, that was a shameless plug for my wonderful guild.

Anyway, I received alot of positive feedback on it and decided I ought to make it more accessible to anyone who may want it, so I'm posting the link for it here.

This guide should explain OoW augments clearly and thoroughly both for people interested in creating the augments and for those who wish to simply have information handy so they can decide what they want and learn what they need to acquire should they wish someone else to do the combines on their behalf. I have worked hard to try and make it easy to read and understand. It is broken up into several sections and I have also made liberal use of color-coding for quick reference.

PLEASE NOTE: THIS GUIDE IS FORMATTED TO BE VIEWED IN-GAME USING THE STORY WINDOW

If you look at the text file in notepad, it's gonna look pretty wierd because of all the formatting embedded in the text. This guide is meant to be viewed using the in-game story window. If you are using default UI and haven't changed the default settings for it, then you can access the story window in-game with Alt-N.

Sooooooo, if you have any questions whatsoever about OoW augments, my little guide will probably answer them. It is very thorough.

So, how do I get my hands on this thing, you ask?

Easy as pie:

1) Right-click on the link below.
2) Choose "Save Target As" from the menu that pops up.
3) Select your /everquest/storyline folder
4) Click OK
5) Yer done! You got it! You can view it in-game with the Alt+n command!

I really hope you guys like this. I put alot of work into it. Enjoy!
