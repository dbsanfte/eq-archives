Fairly sure that at this site the effects are stored en bloc so are not searchable, and it would be a considerable effort to alter things to make them so.

Allakhazam's Item Search provides an answer, though I am not sure if this is only available to premium users. For example, this search (http://everquest.allakhazam.com/db/adv_search.html?classes=0&race=0&type=Inventory&effect=9999&fzone=0&foodtype=1&smod=0&stype=0&stat1=svmag&statlim1=1&stat2=None&statlim2=0&stat3=None&statlim3=0&pristat=svmag&priwt=1&secstat=none&secwt=1&tristat=none&triwt=0&level=&price=&qt=b&aug=y&st=) returns all Food/Drink items with a minimum MR of 1, ordered by MR (the former check is needed to avoid *all* food/drink items being returned).
