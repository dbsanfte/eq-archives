Catapults work just fine... they are tricky because the Shaped Ashwood Recurve Bow must have a hemp string. If it has one of the other 2 strings it DNCs. When the bow has the wrong string it still has the same name, but if you look at the stats they are different. The good news is that if you have the wrong bow you can stick it in a fletching kit with a hemp string and combine (no fail triv 16) to get the correct bow. Restringing the bow even if you buy a fletching kit will cost only about one plat.
