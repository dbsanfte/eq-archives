On the Rathe, the sonic wolves currently own the North caves, and the owlbears have the East caves and South isles. This hasn't changed for at least the last three days. 5. Submitted a /bug report. No response, which I know is normal for a /bug.
6. Submitted a /petition, so I could show a GM exactly where the despawn is happening. I got an update to my ticket saying "thanks for submitting a /bug, we are closing your ticket".

Design lock?!? Design lock!?!? {what} is that?!?!
Design lock basically means the devs are locked out of making any design changes (hence the name). This gives QA a chance to test out changes in their final state before they're pushed to live servers. You'll occasionally hear EQ devs talk about design lock or code lock (for the programming side) right before a major patch. If I recall right, code lock for a new expansion is something like a week before the expansion launch date -- only critical bug fixes are allowed in the last few days. (Note: I don't actually work for Sony, so all this is my impressions from reading various developer postings over the last few years.)
