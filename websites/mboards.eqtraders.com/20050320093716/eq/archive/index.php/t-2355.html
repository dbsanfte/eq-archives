To run EQ on two seperate machines, at the same time, you will need two copies of the game, and by that I mean the original EverQuest or Planes of Power

You can only access the other areas (Kunark, Velious, Luclin) if you purchase and install those expansion packs, again for BOTH machines.

Each expansion comes with its own registration key, possibly the Gold version comes with a universal key, that needs to be entered to enable the zones in game, but this key is unique. Yes you could install the same to both machines (ie from one set of cds') but you would encounter problems should you try and run them at the same time.

So in answer to your question : install Gold on one machine : have access to all EQ, Kunark, Velious and Luclin zones, but NOT POP, with a valid account

Install PoP on the other machine, and the other machine and have access to All EQ zones and Planes of Power zones. (as i understand it)

You will NOT get patched for the other zones, or if you do, you will not be able to get into them.

Hope that was clear.

Your account is basically separate from the installation process. If you want two Accounts, you need to create two accounts with valid unused Keys for the Expansions you want on that Account.

Any given Key is used when applied, and as such, once you have used your key (be it for creating a new Account, or adding Expansions to another), its gone.

The patcher will, based on what files you have, determine what expansions you need to patch.

Two EQ Gold, and two EQ PoP will allow you full access on two accounts to the entire game. Any other combination will leave one or the other short of contents. In addition, LoY has just been released for preordering & download, and each Account would have to purchase the Expansion to access the contents.

In short, your Account determines what you are allowed to access, and the installed files determines what you -can- access. If you're not allowed to go somewhere, you won't.

One thing to be aware of. Sony, like far too many companies, uses the same name for different products depending on where it's sold. I'm in the unusual position of having bought both the American and European "Gold" boxes. Here are some of the differences:

American:
[list] Gold-colored metallic "tin" box
Includes PoP
Includes five CD's -- one each for Classic, Kunark, Velious, Luclin, and PoP
Includes ONE key code to activate a new account with all five expansions, OR upgrade an existing account to use all five.[/list:u]

European:
[list] Black and gold cardboard box
Does NOT include PoP
Includes five CD's -- Classic/Kunark on one, Velious on one, and Luclin on three.
Includes three key codes: One for Classic/Kunark, and one each Velious and Luclin[/list:u]

With the European box, you can use the key codes to upgrade different accounts -- so you could create a new account with Classic/Kunark only, upgrade a second with Velious, and upgrade a third with Luclin if you so wish. I don't think you can create a new account with the Velious and Luclin codes.

To get access to a particular expansion, the following must both be true:
[list] The files for the expansion are installed on your machine, AND
The account you're logging in on has that expansion enabled[/list:u]

Note that you can install all the files (from either boxed set) on as many PC's as you wish. You just can't log in to one account from multiple machines at once.

Hope this helps clear things up a bit.
