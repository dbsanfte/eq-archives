I am new to tradeskills, ie-Baking. I have read about learning recipes and using autolearn.
What is autolearn and what do folks mean by learning and/or forgetting recipes from their recipe book?
Nolrog
Not all recipes will show up in your spit when you search for them. Some need to be successfully combined once, and then you learn it (i.e., it appears in the search list) so you can use the new UI to combine it the next time. This happens automatically (hence the term AutoLearn.) Some recipes cannot be learned though, so you won't see them in your list. Some are not yet correctly set up. The Developers are working on these issues, and I'm hopeful that we'll see some significant improvements in the next patch.
