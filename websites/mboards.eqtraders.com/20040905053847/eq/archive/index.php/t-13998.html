There is currently one bug regarding Vegetable Oil, to tradeskillers benefit.

When you search for vegetable oil using the new UI, you will get two identical matches. Same name, trivial, ingredients. Using the first recepy in list you gain 4 Vegetable Oils, while only 1 using the second recepy. Both uses 1 Water + Vegetable.

Doing this combine manually yields 1 Vegetable Oil.

My guess is that there has been a double entry in EQ's internal tradeskill database as long as there was Vegetable Oil, only that it wasn't possible to get the second alternative without the new tradeskill interface.
