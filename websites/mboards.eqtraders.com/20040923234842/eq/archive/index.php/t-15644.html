Because I currently only have about half a dozen unfinished EQ projects in the works, I figured I ought to start another. :upsidwn:

I want to create a list of all the keys in EQ. I mean only keys that are some kind of tangible item that allow you to click somewhere on something to get to somewhere else that is not accessible without the key. I don't want flags that do not involve an actual item that serves as the key. I want to include keys that can be added to the keyring and then deleted, so long as there was originally an actual key. Keys that do not actually look like keys (that look like bracelets or rings or idols) are still keys.

The info I want is the zone that contains the "lock" (the portal, door, or whatever you will click with the key), where it takes you to, if the key is no rent and if not if it's tradeable, the keys' names, and whether they fit on the keyring.
