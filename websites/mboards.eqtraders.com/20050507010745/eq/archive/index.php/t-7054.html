re: modifiers
<center>Items that modify your trade skills do not stack for the same skill,
only the single highest modifier applies.

i.e., if you have a +5% Geerlok device and a +5% Trophy
for the same skill then you only receive a +5% modifier.

Another example: if you have a +5% pole and +3% bait
you only recveive +5% fishing.</center>
