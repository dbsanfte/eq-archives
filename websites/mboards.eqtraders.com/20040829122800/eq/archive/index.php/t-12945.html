A copy of a guide I wrote up for another board:Chat channels are channels that can be joined by anyone on any server who knows the name of the channel and the password, if there is one. If there were a channel on your server named "greeneggs" and without a password, you would join it by typing:

/join greeneggs

If it had the password of "andham" you would type:

/join greeneggs:andham

You create a chat channel by being the first one to join it. If it does not have a password and everyone leaves, it ceases to exist. If it does have a password then it will require that password for up to 24 hours after everyone has left. That is why many public channels have passwords that are the same as the channel name. They prevent someone from joining when no one else is on and giving it a password so that no one else can get in.

If you create the greeneggs channel (no password) on Karana, people on other servers could join it by typing /join karana.greeneggs. Players on Karana could also join it by typing /join karana.greeneggs, but don't have to put the karana part on if they don't want to.

For channels that are intended to be available to people on all servers, you can designate them as such by using "serverwide" as the server name when they are created. Serverwide is just a generic term you can use to avoid having any particular server's name attached to the channel. Everyone on every server must use the serverwide part of the name to join that channel.

You can also automatically join channels. If you type:

/autojoin tradeskills:tradeskills,serverwide.eqtraders,greeneggs:andham

you will be joined to those three channels every time you log on. As of the October 9, 2003 patch, the autojoin channels are saved independently for each character you play; you must autojoin the channels you desire whenever you start a new character.
