When skilling up, remember this rule of thumb;
Skill up quickly. Skill up cheaply. Skill easily. You may only choose 2 options.

If a path is easy and quick, it will be expensive. If it is cheap and easy, it will be slow and tedious. If a path is expensive and fast, it will be hard to do. Knowing this little rule-of-tumb will make you a lot less insane, since it will keep you from seeking an option which is all three. :)
