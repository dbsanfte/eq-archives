I finally finished the brewing compendium that goes hand in hand with the baking compendium that I created earlier. It will work completely stand-alone but will work even better if you use the baking version also. Each recipe ingredient is tied to it's own recipe if one exist (the exception to this is water flasks and short ale, they are used far to often and are common merchant items)
I've also added a few recipes that weren't listed on the EQTraders brewing lists. I'll get them submitted soon.

You can download the file from here:
obsidian.efrontco.com/regression/LothBake_108.zip (Baking Version)
obsidian.efrontco.com/regression/LothBrew_104.zip (Brewing Version)
obsidian.efrontco.com/regression/LothPott_101.zip (Pottery Version)
obsidian.efrontco.com/regression/LothComp01July2003.zip Baking, Brewing and Pottery in one download

Just unzip the file to your storyline directory using full paths. No files should be overwritten unless you have a previous version installed. If you have a previous version installed, you may want to delete the files associtated with it just to get a fresh install.
