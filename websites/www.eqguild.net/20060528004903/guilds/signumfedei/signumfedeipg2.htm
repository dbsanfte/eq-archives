EQ WINDOWS 98 SE

INSTALLATION INSTRUCTIONS

*Do This First*

Insert the WIN98 BOOTUP floppy into the floppy drive.

Press the start button on the computer.

The Computer will bootup off the system written on WIN98 BOOTUP floppy disk.

After the standard PC bootup checks have been completed, the following three options will appear on the screen:

Start Windows 98 Setup from the CD-ROM

Start computer with CD-ROM support

Start computer without CD-ROM support

Select option 3 (using the keyboard arrows to select it) and press the ENTER (return) key located on the keyboard.

You will be presented on the screen with the DOS prompt pertaining to the Floppy Disk (A:) as set out below:

A:\>

At the DOS prompt type in:

C:

Press the ENTER (return) key located on the keyboard to execute these instructions.

You will then be presented with the DOS prompt pertaining to the hard drive (C:) as set out below:

C:\>

At the DOS prompt type in:

cd progra~1

Press the ENTER (return) key located on the keyboard to execute these instructions.

Note: the letters on the screen should read:

C:\>progra~1

At the DOS prompt type in:

cd everqu~1

Press the ENTER (return) key located on the keyboard to execute these instructions.

Note: the letters on the screen should read:

C:\>progra~1\everqu~1

Remove the WIN98 BOOTUP floppy from drive A and replace it with a clean formatted floppy disk

At the DOS prompt type in:

copy *.ini A:

Press the ENTER (return) key located on the keyboard to execute these instructions.

At the DOS prompt type in:

copy *.opt A:

Press the ENTER (return) key located on the keyboard to execute these instructions.

At the DOS prompt type in:

cd a:

Press the ENTER (return) key located on the keyboard to execute these instructions.

At the DOS prompt type in:

del eqhardware.ini

Press the ENTER (return) key located on the keyboard to execute these instructions.

At the DOS prompt type in:

del ss.opt

Press the ENTER (return) key located on the keyboard to execute these instructions.

At the DOS prompt type in:

del eqoptions1.opt

Press the ENTER (return) key located on the keyboard to execute these instructions.

At the DOS prompt type in:

del eqoptions2.opt

Press the ENTER (return) key located on the keyboard to execute these instructions.

Remove the floppy disk currently in drive A and, once again, replace it with the WIN98 BOOTUP floppy

*NOTE*

The above procedure will allow you to restore your EQ settings (exactly as they were before) once you have installed Everquest without the need to re-do hotkeys, and the like, by copying all of the files from the floppy disk back into your c:\program files\everquest folder

 

OVERVIEW

The following instructions assume you have:

A WIN98 SE Bootup Floppy Disk.

A WIN98 SE Installation CD.

and that you are going to perform a 'clean' install of Windows 98 SE by formatting the hard disk prior to installing the Windows software

 

 

PRELIMINARIES

Start the computer by pressing the start button .

Press the CD-ROM button to eject the CD-ROM tray.

Place the WIN98 SE INSTALLATION CD on the CD-ROM drive tray and then close the tray.

Shut down the computer.

BOOTING UP THE COMPUTER USING THE FLOPPY DISK

Insert the WIN98 BOOTUP floppy into the floppy drive.

Press the start button on the computer.

The Computer will bootup off the system written on WIN98 BOOTUP floppy disk.

After the standard PC bootup checks have been completed, the following three options will appear on the screen:

1. Start Windows 98 Setup from the CD-ROM

2. Start computer with CD-ROM support

3. Start computer with out CD-ROM support

You need to Format the hard disk prior to installing Windows 98, therefore:

Select option 2 (using the keyboard arrows to select it) and press the ENTER (return) key located on the keyboard.

You will be presented on the screen with the DOS prompt pertaining to the Floppy Disk (A:) as set out below:

A:\>

A: represents the floppy drive in MS-DOS. You need to change the DOS prompt to the CD-ROM drive (D:) to access the files on the CD. To do this.

At the DOS prompt type in:

D:

Note: the letters on the screen should read:

A:\>D:

Press the ENTER (return) key located on the keyboard to execute these instructions.

You will then be presented with the DOS prompt pertaining to the CD-ROM drive (D:) as set out below:

D:\>

FORMATTING THE HARD DISK (C: drive)

For a 'clean' install of Windows 98 (recommended), the computer's hard disk needs to be wiped clean of any old files and formatted prior to running the WIN98 installer.

The FORMAT software is located on the WIN98 CD in the win98 folder.

NOTE: The actual file location of the FORMAT software is D:\win98\format

To format the computer's hard disk, type in at the DOS prompt (D:\>)

win98\format C:

Note 1: There is a space between the word format and letter C

Note 2: The full text on the screen should read:

D:\>win98\format C:

Note 3: Translating the above MS-DOS 'code':

D: is the CD in the CD-ROM drive,

> is the prompt where you start typing instructions to the computer

win98 is the folder (on the CD) in which the format software is located

\ is the symbol used to define a new folder (called a sub-directory in MS-DOS land)

format is the instruction to format the specified disk

C: is the primary hard disk drive installed in the computer which is about to be formatted.

Press the ENTER (return) key on the keyboard to initiate these instructions.

You will be presented with the following warning:

WARNING, ALL DATA ON NON REMOVABLE DISK DRIVE C: WILL BE LOST!

Proceed with format (Y/N)?

Select yes by pressing the y key and then the ENTER (return) key on the keyboard.

Make a c:\windows\options\cabs directory and copy the files from the win98 directory on the CD-ROM to the cabs directory.

a:\> c:

c:>md windows

c:>cd windows

c:\windows>md options

c:\windows>cd options

c:\windows\options>md cabs

c:\windows\options>cd cabs

c:\windows\options\cabs>copy d:\win98\*.*

Where d: is your CD-ROM drive.

The c:/windows/options/cabs directory was a convention established with the standard installation some of the OEM (Original Equipment Manufacturer) versions of Windows 95. You could copy the win98 directory to any desired directory on your hard disk and install from there, but I continue to use the convention so I can remember where the cab files are located.

Cab or cabinet files are files containing one or more compressed files. Most of the Windows 98 files are in cab files on the CD.

Install Windows 98 by typing the following command at the MS-Dos (C:) prompt:

C:\windows\options\cabs>setup

At this point simply follow along by answering whatever questions Windows asks to the best of your ability.

FAT32 Your Drives

StartButton/Programs/Accessories/SystemTools/DriveConverter(FAT32), then follow the wizard.

 

Update Your Windows Files

Namely, download and install DirectX8.1 from http://windowsupdate.microsoft.com

 

Re-install any hardware drivers that you need by going to the manufacturer’s website and downloading the latest drivers…eg. video, sound, cable modem, etc…

 

Re-install Everquest

Once you have done this proceed to copy the files that were originally copied onto the blank floppy diskette back into your Everquest program folder.

Connect to the EQ servers and download all the necessary patches (automatically done once you connect) PS…don’t forget to download and install all the optional patches also.

Copy all the files that were copied onto the blank formatted diskette back into your Everquest folder.

Voila!

Shutdown and Restart your system and you’re ready to go…=)
