Q: How do I change the text colors?

A: Go into your everquest folder under C:/Program files. Find the file called "eqclient.ini". Find the section called "Text Colors". Use the following chart to determine which color you would like to change.

13- Other misses you
14- Some broadcasts (duels)
15- Skills (ups, non-combat use, etc.)

33- Others spells
34- Spell failures (resists, fizzles, missing comp)

All colors are to be set with Red, Green, and Blue values, in decimal. Valid settings for each color range between 0 and 255. "Black" (0 red, 0 green, and 0 blue) will be white in overlay mode, and black in normal mode.

Example: To change the color of guild chat, find guild on the table. Next find #4 under text colors on the .ini file. You will see 3 enteries under "user 4", one for red, one for green, one for blue. To find the number value for the color you wish to use, I recommend going into your "Paint" Program. Under "colors" on the menu bar, select "edit colors" and then select "define custom colors". Use the arrow all the way on the right, dragging it up and down to adjust lightness/darkness and then drag the cursor around on the color map until you find a color you like and then take note of the values next to red, green and yellow. I chose a dark purple, so now in the /ini file I would change User 4's entries to read

User_4_Red=149
User_4_Green=36
User_4_Blue=137

To change colors while in game use the following format, using the same table above to determine what number to use.

/usercolor [Chat Type] Rvalue Gvalue Bvalue. - For example, For Guild chat to be purple, you would type

/usercolor 4 149 36 137.
