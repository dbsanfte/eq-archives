Welcome Straight Dope and Fathom members. This is our fancy EverQuest guild page. Check it out! The first thing you need to do is head over to GuildMagic and create a user there if you don't have one already (if you have one, just use the 'create new character' to add your new character) and create your character there. Add yourself to the Legend Urbana guild. I'll check once a day or so to see who has done it and add you in (as the admin I have to approve each person)... You can check our fancy Roster page to see who has already done this. I use a script to include the GuildMagic data on our site so it matches all pretty-like.

Scheduling

Playing together is the whole idea here so while everyone is welcome to play whenever they want, we should also have some standard play times when we can expect to see each other online. This is all subject to change upon discussion, but for now I've set it up this way:

Weeknights starting at 9pm EST - nightly playtime
Wednesdays and Saturdays starting at 11am EST - lunch bunch
Sunday afternoons-evenings starting at 3pm EST - Main guild gathering time. Everyone try to make these as often as possible.

If you want to schedule any events, just use the calendar. Login directions are on the calendar page.
