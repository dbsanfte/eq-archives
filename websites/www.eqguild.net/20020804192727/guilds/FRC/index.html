July 31, 2002

Patch Message

Today's Patch updated the following:

  • /hitsmode functionality is now available in the new UI. Please check out the context (right-click) menus from New UI Chat windows. Hopefully this version is somewhat easier to understand.
  • NPC and PC names display settings should be saved between sessions.
  • Bazaar Search Window is now resizable.
  • Disabled Begging and Stealing in the Bazaar
  • The New UI Train window no longer cuts off any skill names for INT casters.
-Katn

July 30, 2002

Hitsmode Coming back soon!

Hitsmode will be returning soon, to the new interface. This is what Alan posted about it.

I know a lot of folks would like to see the functionality of Hitsmode reappear in the new user interface. So would we. In fact, we’re working on something that would replace it and should have the same or better functionality. We hope to have it ready pretty soon, if possible before the new interface goes fully Live.

Alan

-Katn
