Patch Message

Today's Patch updated the following:

  • /hitsmode functionality is now available in the new UI. Please check out the context (right-click) menus from New UI Chat windows. Hopefully this version is somewhat easier to understand.
  • NPC and PC names display settings should be saved between sessions.
  • Bazaar Search Window is now resizable.
  • Disabled Begging and Stealing in the Bazaar
  • The New UI Train window no longer cuts off any skill names for INT casters.
-Katn

Patch Message

Today's patch message is out and contains a lot of changes. Here's a summary:
The new interface is now available to players.
The Bazaar is now available to players.
A new zone: Jaggedpine Forest is now available to players.
Moving NPCs will no longer receive a reduction in DoT damage.
Fear & Charm will no longer work against any player on any server when cast by another player.
The bug to use charmed pets to have NPC's assist with kills has been fixed.
Vah Shir Models for the Old World have been added that are less memory intensive.
The Vex Thal key quest has been modified.
Pets will now resist spells differently.
A massive patch with lots of new stuff to explore.
