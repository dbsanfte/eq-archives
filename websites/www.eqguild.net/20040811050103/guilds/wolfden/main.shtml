******** Guild Update ********
Posted Monday, March 8, 2004 by wolfden

HI all, Well we ran in to a small problem but are again in a guild rebuilding mode and will be changing some of the rules in the charter. We are going to switch the guild to a NGB (need before greed) guild because we all feel this will make the guild a much more fun and enjoyable guild for all. We want to make this guild the funnest guild on The Rathe server. We are going to put the guild invite up to 45 lvl so any that join the guild can start raiding with us right away, or very shortly, but anyone currently in the guild will stay so do not worry NO ONE is getting booted just for there lvl. Please rememeber the guild invite is a guideline as we do know there are some great up and coming players that would make great members at any level. We have started revising the guild charter and we will be setting up polls so all in the guild can vote on the changes, so if you know someone that would make a great member and wants to help us with raids, let us or an officer know right away. Also, right now the only place you need to make sure you have signed up is at www.guildmagic.com the web site and messageboard are optional.

Woodlandmaster & Tiwini
