Changes To Heroic Opportunities 11/02/04

Heroic Opportunities changed a bit on the latest patch. No longer will just hitting a skill/spell start a heroic opportunity. They have added skills to each class (Fighter - Fighting Chance, Scout - Lucky Break, Mage - Arcane Augur, Priest - Divine Providence). These skills have no cast time, a 10 second recast time and are now what start heroic opportunities. This change should only effect the starter chains, the actual heroic opportunities do not seem to be effected. A new chart is in progress and after some further testing will be available. If you have anything to add, please stop by our forums and give us a hand.
