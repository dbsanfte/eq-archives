    Same Game は、同色の駒が『縦』か『横』に隣接していると駒を取ることができます。駒を次々に取って行き、最終的にすべての駒を無くすことを目的としたパズルゲームです。
    この『Same Same』では、各駒に『色』と『番号』を持たせて『色』と『番号』のどちらでも駒を取れるようにしました。
