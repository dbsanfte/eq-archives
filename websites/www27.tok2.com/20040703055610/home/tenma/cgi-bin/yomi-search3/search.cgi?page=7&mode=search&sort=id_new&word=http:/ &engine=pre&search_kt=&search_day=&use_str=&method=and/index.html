    基本ルールは本来の「おはじき」と一緒。画面に散らばったおはじきから、ターゲットのおはじきと飛ばすおはじきを選び、おはじきを飛ばす方向と飛ばすパワーを調整してぶつけます。ぶつけた２つのおはじきの間に上手にラインを引くと、おはじきをGETできます。　ぶつけた時に他のおはじきにぶつかったり、引いたラインがおはじきに触れてしまうとMiss。制限時間内に決められた個数のおはじきをGETするとステージクリアです。

    ■ Fire Fighting ver.1.01

    更新日：2003/07/04(Fri) 12:52 [修正・削除] [管理者に通知]
    [:パズルゲーム] [:思考ゲーム] [:その他のゲーム]
    水を使って火を消すだけの面クリア型のパズルです。ゲームのルールは以下のとおりです。
    水は障害物がなければ直進する。
    「普通の水」は「普通の火」は消せるが、「赤い火」は消せない。「赤い水」は「普通の火」も「赤い火」も両方消せる。
    「赤い火」の直線上にプレーヤーが来ると、火が襲ってきて(^^;プレーヤーは死んでしまう。
    これらの制約事項を頭に入れつつ、いかにすべての火を消すか考えてください。

    Take's HomePage
    http://takehiro.ciao.jp/
