NPC Name Fix
Submitted by: NiteRaven on Thursday, November 9th at 05:40am EST

I wrote a small script in the wee hours of the night that converted all NPC pictures to a new naming convention, in the format of NPC + id + file ending. My old way of doing it was pretty stupid and inefficient, it would basically take the name of whatever file was being uploading and appended the id of the NPC. This was fine as long as there were no spaces or weird characters in the NPC name, but many times that wasn't the case. This would cause a problem in Netscape browsers which couldn't display weird named images. It appears the procedure it was a success and I was able to copy all the images to their new name. If you find any broken links, please report them to me!

NPC Screenshots
Submitted by: Brutius on Wednesday, November 1st at 11:54am EST

I want to thank everyone for sending in the great NPC screenshots. Each screenshot helps us have a more robust and complete web site!

Unfortunately, a lot of the screenshots I receive are unusable for various reasons. Please try to follow the following guidelines when sending screenshots.

  1. Try and get all screenshots without other people/npcs in the way.
  2. Try and get far enough away from the monster so that you can see the name, but where the monster doesn't take up the whole screen.
  3. Do not edit the screenshots yourself. Send them in the raw BMP format. (zipped if possible).
  4. Try to include as much information about the NPC as you can, including zone, damage, faction etc.
If everyone follows these general procedures, I can add the NPC screenshots faster and easier! Thanks!
