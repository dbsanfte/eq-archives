PATCH CHANGES

The last patch was a biggie for the world of Norrath. I've high-lighted some of the more Paladin related changes.

  • Several spells will no longer be interrupted, spells such as the 'proc' generated by Instrument of Nife
  • Ghoulbane is now No-Drop
  • SoulFire charges can only be used by Paladins (though they can be used from inventory)

Saturday January 19, 2002

GHOULBANES TO BECOME NO-DROP AND SOULFIRE EFFECT TO BECOME CLASS EXCLUSIVE!

It was announced that Ghoulbanes will soon become No-Drop. These changes will be retroactive. The Ghoulbane change was due to the camping of Ghoulbane quest items to make a quick bit of plat. Also, another change has taken effect on the Soulfire, the effect is now useable only by Paladins. This change has occured due to the great number of Soulfires used in the recent ToT3.

- Submitted by Gloyn, 7:41 PM
 
 
Tuesday January 8, 2002

HORSE MEDITATION GONE...DURING COMBAT ONLY

Latest patch has fixed a "bug" that had characters on horses regaining mana at a meditation speed while on a horse. This "fix" only stops the mana increase during BATTLE...running/walking still regains the same amonut of mana.
