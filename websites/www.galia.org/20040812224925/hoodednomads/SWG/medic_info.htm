Star Wars Galaxy : The Alfie Lists


I've compiled a list of resources needed for medics, doctors and combat medics. Note that this information was gathered directly from my datapad and various forums, sites and might not be 100% accurate.

If there are errors, feel free to send an in game email to alfie @ Eclipse Server.

 

Medical Item Use Skill List

This is a list of medical items made with no ease-of-use experimentation using average ingredients (OQ of 700+,etc), using high quality ingredients can give you (with out experimentation) on average a reduction of 2-5 use. The Min skill values are reported values and might require extremely high quality ingredients and experimentation skills. Use at your own risk. Note that Medic/Doctor items uses Medicine Use, while Combat Medic items are Combat Medicine Use.

Enhance Pack Ease-of-Use is currently bugged and cannot be experimented. You will need novice doctor to resuscitate, and with a medicine use of 55, there is no need to experiment Res Packs.

The list is slightly incomplete as I need to gather a few more resources to complete the ones marked "?". I'm not a master Doctor, so I cant make the higher E packs. Anyone with info please email it to me at elvin (at) elvin.net. I'll get the rest done soon.
