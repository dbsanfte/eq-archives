FAQ - Frequently Asked Questions... and some answers to them

How much does this cost?

      Nothing. Of course with the low cost comes many constraints upon the user

Can I change the layout?

      No, you cannot. Nor should you be expecting me to do much tweaking to any guilds database either. I may, in the future, accept pictures of predetermined size to act as backgrounds for the menu tables.

How do I make specific calls to the database?

      Administrators will be given access to another section in which many calls will be covered.

How do I make  a user?

      All databases will be set to default access. This means that anyone can enter the ‘Add User’ menu in which players determine their own username:password. These self-made users will only have access to add/modify/delete their own records. Also, Administrators will have the ability to create new or delete existing users in their database.

How do I delete my profile?

      For non-Admin users: Log onto the database with your current username:password. Once you arrive at the Main Menu select the ‘Delete’ option. From here there are several ways to find out what character profile(s) you have entered in the database. The easiest way is to select the ‘List All’ option at the top of the Search Form; the database will then list ALL profile(s) your username has added to the database. From here, select the checkbox next to the profile you wish to delete and then select 'Delete Checked Record(s)'.  Of course, you could also search for profiles by Name/Level/Race/Etc. if you care to take that route.

For Admin users: Please visit the Administration section.

How do I modify my profile?

      For non-Admin users: Log onto the database with your current username:password. Once you arrive at the Main Menu select the ‘Modify’ option. Once again, the easiest way to select which profile you want to modify is to select the ‘List All’ option near the top of the search form. Once you have located the profile you wish to modify select the radio button next to it and click ‘Modify Record’ found at the bottom of the screen. The selected profile should then load in the browser. Make changes needed and again select ‘Modify Record’.
      For Admin users: Same as above, except that when you select ‘List All’ you will see ALL profiles currently in the database. If you have a lot of entries in your database it may be wise to search by Name/Level/Etc…
