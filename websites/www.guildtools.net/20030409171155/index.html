R5 - 2003-03-25
EQRDB5 is done(ish).
Pretty much the same as 4, with some new things thrown in since you can't download it.
- Added a field layout editor where you can either choose to show or hide all fields (except name) in the listing. You can also specify how wide you want the table to be, how wide each column should be, the names of 5 misc fields (add your own VT key colum now) the name of your roster, and the colors of alternating fields.

The instant signup "seems" to work. I found a few bugs right after I changed it, but I think I caught all the major ones. Let me know if you see something fishy.

Forum Hosting Available - 2003-03-23
I'm happy to announce forum hosting again for the first time in a long time. Forums are YaBB Gold 1.2 which I think you'll find are very slick.
When you sign up for a forum you become the admin - gaining complete control over categories, layout, users, privacy and a few more goodies.

Forums are limited to guild use for online games. Any deviation will result in board removal. Forum creation is instant, provided you supply the correct email address.

CSS not working in r4? - 2003-03-07
If your stylesheet is not working in your downloaded version of EQRDB 4, please redownload the file and use the new html.pl. The old html.pl had a / in the style relative link to the stylesheet that was throwing off some servers. Or, just edit your html.pl and go to the very bottom of the page, and remove the / at the end of the style sheet tag.
