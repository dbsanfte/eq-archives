EQ*DB Downloads and Release History

All versions of the RDB are a modified form of DBMan that Gossamer-Threads has so kindly allowed free use/modification of for non-profit organizations. The GuildTools Network is definitely a non-profit organization. In fact, you could call us an in-debt organization.

EQIDB

EQ IDB 1.0  Released: 4.7.2002
The first (and last) version of EQ Item Database. Fully functional with detailed listing. Check out the demo from the main page.

EQRDB

EQ RDB 3.1  Released: 10.1.2001
This release adds a few cosmetic changes, a skills listing, more detailed tracking information, and CSS globals for link manipulation.

EQ RDB 3.0  Released: 9.22.2001(Not Available)
This release was brought about by some suggestions from Geten of Grey Lords. It includes Time Zone listing for the normal 'List All' and also a new sub display '{short}'. The new display is mostly text as to increase the load time, yet you can still click on the characters name to see all the detailed information. Also, removed the search feature for Allakhazam's since he decided to make his site a pay-to-use site.

EQ RDB 2.01  Released: 8.28.2001(Not Available)
This release can be refered to as Operation Cleanup. I went through every line in the script and whipped it into shape so that people who want to set this up on their servers only have to edit about 10 lines of code. Go me!

EQ RDB 2.0  Released: 8.11.2001(Not Available)
This release was a built upon the frame of 1.0 and using what I learned making 1.1. It brought about many new fields for users to enjoy including equipment, stats, trade skills, resists, etc.. This release also continued the icon trend from 1.1 with many new icons pilfered from various sources (which slow down the load time, but it sure does look perty!). This version is no longer available for download.

EQ RDB 1.1  Released: 8.8.2001(Not Available)
This release added in the ability to link to show a picture/graphic and description of characters. It was later scrapped and is no longer available for download with the release of RDB 2.0.

EQ RDB 1.0  Released: 8.4.2001
This was the first release of RDB and is still available for download today. I really like the speed at which it can display a large number of members. This release is good for basic information but fails to go into the great detail later releases do.
