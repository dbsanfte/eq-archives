R5 - 2003-03-25

CSS not working in r4? - 2003-03-07
If your stylesheet is not working in your downloaded version of EQRDB 4, please redownload the file and use the new html.pl. The old html.pl had a / in the style relative link to the stylesheet that was throwing off some servers. Or, just edit your html.pl and go to the very bottom of the page, and remove the / at the end of the style sheet tag.
