EQRDB 4.0 Development

The following list are enhancements that *may* go into the final version of EQRDB. Please bear with me while I try to build this list from memory.

If you have any additions you would like to see added please click on the feedback link to the left.

Layout
- Ability to turn fields shown on the main listing OFF or ON by adding a 0 or 1 in the .cfg file.
- Alignment of the roster and/or banners (ie center/left/right) by changing one word in the .pl file.
- Ability to add a tiled background image instead of a color.
- CSS for form fields and buttons.
- Use a GuComics charicature as a personal image *undecided*
- Add a help section to the admin area that has some common Q&A.
- Add a detailed introduction and explanation of EQRDB to the HOME page. (or) Remove the HOME page altogether and make the first page upon login a listing of all records
- Add a credits section to acknowledge those that helped make v4.0 what it is.
- Change the HOME page to a listing of classes (ie X many Warriors Y many Clerics Z many Wizards).

Coding
- Rework of all the html coding.
- Possible perl reworking.
- Incorporate skills.pl, html.pl, and detail.pl into one .pl file. I think I tried this before and couldn't get it to work. Not sure.
- Complete the skills list (or) Remove the skills list to reduce load time *feedback needed*

Add-ons (need feedback on all these)
- Image upload. This will allow members to upload an image of their character to your server. However, this will most likely make the installation of the script much more difficult.
- Cookie Mod. Members user id/password will be stored on their PC.
- What's new? Will show if a record has been modified in the last X many days
