Security Reminder. at Jun 8, 2001 10:02:06 by Dimentox

As the population of Norrath increases, so too do issues regarding stolen or compromised accounts, and/or accounts that have been transferred from one player to another in violation of the User Agreement. These issues - and the time it takes to investigate and attempt to resolve them - hugely detract from the time that SOE's customer service staff have to devote to providing quality customer service to issues that impact upon the gameplay experience. Accordingly, SOE must remind all players that:

(1) Account security -- whether it be password protection, running virus checks, disabling file sharing, or any other element of making sure accounts are not stolen or compromised -- is the sole responsibility of the account owner.

(2)Any and all stolen and/or compromised accounts that are reported to SOE will be banned, and no items will be reimbursed. Any player found to be involved in an account theft will have any and all of his/her accounts banned.

- The EverQuest Customer Service Team

New Patch For graphics at Jun 7, 2001 11:43:03 by Dimentox

We have just put up a new patch. It's a small patch and it's of the type that will not require that we take an servers down. So you'll get the update the next time you log in.

The patch is a fix for the problem with older ATI cards and cards using Savage chipsets that caused them to crash during rain and snow and while using F10 to change views.

The patch may improve performance in general and might solve some other crashing problems.

Abisor on the New server at Jun 7, 2001 09:49:06 by Dimentox

Hello everyone. I've got some news for you that I'm sure you'll like.
We plan to have the new PvP Deity Teams server up and running some time tomorrow. I also have the name of the new server and the basic rule set for you below.

What we'll be doing is putting the new server up as a Test server for a short while, a few weeks or it may only be a few days, depending on how things go. In order to play on the server during the testing period you'll have to log in using testeverquest.exe. At the end of the test period we will be wiping all characters from the new server in order to restart it as a Live server.

The server will be open to anyone that wants to help us test it. You'll have to understand that the standard Test Server disclaimers will apply (see the note below). And our Development Team and Testing Teams will be running around on the server. Please cooperate with them and don't bother them with requests.

Please use /bug to tell us about any bugs you find with the new rules and PvP code.

Remember, this is a Test server, and there is no promise of customer service.

Don't let any of that intimidate you if you want to test out the new rules once we get the server up. Just run testeverquest.exe and start up a character.

Anyway, on to the rules. This isn't necessarily a final list, we are still thinking of adding and modifying some rules. But this is a pretty good summary.
