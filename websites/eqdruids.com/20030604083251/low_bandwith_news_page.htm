Norton Antivirus and EQ 5/26/03

Someone mentioned that they had heard that Norton Antivirus with auto protection was slowing down EQ.  So we decided to run some tests.  Our results shocked even us.

We clocked login time, and then zone time for each instance.  To clock the zone time, we stepped off the same edge of Kelethin and waited to see how long it took for us to load after death.

With Autoprotect running.  We had a loading time of 223 seconds from clicking on the EQ icon to actually being in the game.

With Autoprotect turned off.  We had a loading time of 191 seconds from clicking on the EQ icon to actually being in the game.

This is almost a 17% difference in speed!

Next we did our death test.

With Autoprotect running.  After dying it took 30 seconds to zone.

With Autoprotect turned off.  After dying it took 25 seconds to zone.

Again almost a 17% difference in speed!

To verify our results we tried several other programs.  Loading a very large Word document we saw almost a 50% faster loading time with the Autoprotect turned off, than on.  In Photoshop we reduced a 45 meg image down to a 450k image.  The speed difference also showed there but not as dramatically.

The only conclusion we can draw from these tests is that if the program you are running is loading from the disk often, the program will run 15-17% faster with Autoprotect off.

Now before you run out and turn off your autoprotection.  I am not saying to do that!  Norton's autoprotection definitely has a purpose on your system.  However, if you are going to be raiding or having a hard time with lag while in EQ this might be an option to help reduce problems (just turn autoprotect back on when you are done playing!!)

For those curious, these tests were run on a Pentium 4 computer with 512 megs of ram running Windows XP home.  I did run other programs in the background to make this test "real life" but those background programs were the exact same programs through each test.

I'm not saying that Norton Antivirus is at fault here, it could very easily be a conflict between Antivirus software and Windows itself (there have been other incidents of Windows and Antivirus programs butting heads) but this is something to be aware of.

  - Spiritual
