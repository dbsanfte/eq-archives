Tymmon announces changes to EqLuclin.com
Posted by: Bicrius
Tymmon essentially announced the following:

After what seems to be forever, I have finally gotten the EverQuest Server Status module up on www.eqluclin.com.

I wanted to alert people to another upcoming addition. I will be turning on the EverQuest Profiler in about 3 days. This will add Character Profiles to the site. This is the same profile tool seen at Insidiousblood.net for iB's roster. You will be able to add your stats, tradeskills, keys, and info to the site so it can be searched. This will aid people looking for tradeskill folk or whatever and it should be very interesing indeed to see the stats. This will be a manual process meaning if you want it up there, you add it and edit it. BTW, this is all GPLed, I will release it after you guys find a way to break it on the luclin site.

Is there anything else anyone wants to see made a module at www.eqluclin.com? We're working here for you all, help us make this a better tool and resource for the community.
