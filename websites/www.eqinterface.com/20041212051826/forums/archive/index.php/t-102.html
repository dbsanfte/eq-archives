All text in the new UI has a default of size 3.

Not sure how good his coding is but he could go into each file in the uidirectory (after he makes a copy of default and renames it)
He'll notice under each item which contains text this line
<!-- <Font>3</Font> -->
So verant have the line there they have just commented it out. Just change the line to
<Font>5</Font>

The only problem I can foresee is that the positioning of certain items relys on the smaller text. Oh and things like the hotbuttons have adjustable sizing depending on the amount of text in them.
