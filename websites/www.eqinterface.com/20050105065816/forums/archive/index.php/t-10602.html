It is actually pretty straight forward if you are used to using computers and text editors (Word, WordPerfect, Notepad, etc.) at all.

What you need to do is:

Open your custom folder.
Open the Patch (http://www.eqgui.com/showthread.php?s=&threadid=10596) thread.
Look at the list of files in the very first post of that thread.
See what files are in your custom folder that are listed in that post.
Make the changes for only those files. Ignore the info for the rest of the files that you don't have.

The actual "fixes" are pretty easy to follow (I think/hope?).
