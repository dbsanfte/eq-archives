IF you have a custom EQUI_OptionsWindow.xml file, open using a text editor (Word, WordPerfect, NotePad, etc.) and using the Search or Find function:

Find:

<Choices>3D Sky Off</Choices>
<Choices>Single Layer</Choices>
<Choices>Double Layer</Choices>

<Choices>Off</Choices>
<Choices>Simple</Choices>
<Choices>Complex</Choices>
