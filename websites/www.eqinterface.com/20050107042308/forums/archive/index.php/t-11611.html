Here's the trick:
Decide which UI you want as your BASE. Now, download your other items that you wish to include.

Then comes the important part -- Compare the contents of the UI Modifications. Check the following things;
Which XML files are modified? Which graphic files are modified? Do any of the packages modify the SAME file? Do any of the packages require/depend on other XML files in the same package? (Example: Does say the Trader mod require certain changes to the XML in the EQUI_Animations.xml file that are different than the ones in your Druid UI. This is just an example).
Once you've answered those questions, you'll be able to tell what your next step really should be.

If you want the features of more than on UI set, and the UI items you want to change each require other pieces of XML to be modified, you're just going to have to learn XML and code it. Learning enough to identify various items in the XML code should not be too difficult, and will almost certainly be less painful than posting the same question in 3 different forums and being frustrated waiting for an answer.

I know I haven't completely solved your problem. I'm just pointing you to the right road. To quote one of my favorite movies, "Endeavor to perservere."
