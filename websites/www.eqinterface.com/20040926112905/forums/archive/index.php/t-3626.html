ok if you start using an existing TGA file from the UI you will see by clicking on 'channels' that there is a alpha channel and if you select it yo ucan basically paint the 'opacity' in range from black (transparent) t owhite (opaque).

What you do is yo uselect tjhe layer in the color layers, switch the the channels tab and select the alpha channel, then while the selection is still on, use alt backspace to fill it in with white (press D for default black/white colors).

then you save your TGA and everything should show up.

to makea drop shadow for example, you leave black in the color channel, make a selection of the shadow shape, then in the alpha channel you fill it in with gray ...

hope that makes sense

if you start from scratch,
go the 'channels' tab and sim,ply click the new channel button, it wiall automatically adda ALPHA channel after the R G B channels (or the cmyk oneswhatever they are I dont remember right now)
