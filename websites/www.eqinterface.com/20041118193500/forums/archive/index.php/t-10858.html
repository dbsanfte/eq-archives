Go to: File >> New and make a new canvas. You need to make the pixel size one of the dimensions shown in this tutorial (http://www.eqgui.com/showthread.php?s=&threadid=458). Then make a new layer and that should allow you to delete the background layer making the background transparent.

Or you can open one of the tga's from the default interface... like one of the window peices tga's. Select the entire thing and CUT everything out and put what ever you want in it. Then just save it with a different filename.

Then once your done merge all the layers together and save it as a TGA or PNG.
