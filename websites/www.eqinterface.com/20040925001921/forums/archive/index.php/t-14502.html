Something simple I just thought of, for those SOE people that read here ( or someone that might be able to forward this to them )...

When you select "Always Chat Here" for one of you chat windows, why doesn't the UI automatically get rid of the input bar for all the other windows?

It's unnecessary, since the player has already said they only want to chat in one window, and it takes up screen space that could be used for one extra line of output. Also, it makes it hard to cleanly line up mutiple chat windows ( unless you want to display the empty line, or you want to deal with overlapping chat windows that keep moving focus around every time a line appears in a different one )

Anyway, just something I was thinking... might be easy to do on thier side and it would be really nice for those of us with mutiple tiled chat windows ( who really only want to SEND chat from one, and read stuff in others )
