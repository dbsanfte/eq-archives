Varent set it so it now Cache's 3 files during login - the solution that Oguler found that is the Easiest is to go into you eqclient.ini file find this line

UISkin=default

and change it to the folder that has the images you want to use.. for me I have a folder called "bard" that has all my colors and mods in it.. sooo i changed it to:

UISkin=bard

the scan will still repair the default directory if you put any files there.. BUT it loads the images from the "bard" directory and works fine (see screenshot in beta forum)

also do a /bug report on it.. maybe we can get it fixed in next patch.
