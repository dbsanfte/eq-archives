The next generation of StoneAngel

First, I would like to adress the users of my former skins. A HDD Crash a few month ago resulted in the loss of all
project files of my old versions. Because of that, I decided to call it a day with UIs. Now I found I couldn't leave my hands
of it. So if you have a zip file with an old StoneAngel 2.0 or 3.0 lying around somewhere, please tell me. Don't send me the file
right away, send me an E-Mail that you have it. My E-Mail account doesn't allow much storage, so I don't want it to crash because
6 people send me the same file. :)

Anyway, this is about the UI at hand.

====================================
INSTALATION
====================================

Simply unzip the file to a new folder in the Everquest/UIFiles directory. Make sure it's not a double folder.
Once thats done, rename the "UI_Lysiander_10.ini" File to "UI_<yourname>_<your servernumber>.ini" and copy it into your generall
Everquest Folder. Then Start EQ and enjoy the skin.

If you don't know your server number, simply check your existing ini file in the generall everquest folder. If you have never
used the new UI before, load up eq, log in a charakter, log out again, go to your EQ folder and do a search for the name of
the charakter you logged in with. The number at the end of the UI_<yourname>_<your servernumber>.ini file is your servernumber.

===================================
NOTES FROM THE AUTHOR
===================================

The basic layout is pretty common these days and since I like it I decided to use it. I was never a fan of changing
things if they worked in the first place.
A few things to be noticed:
The Backround Pictures are locked to those positions. The UI is coded for 1280x1024 resolution and will only work in that
resolution. While the windows will load in others, the backround pictures will be out of place. Because of this, it is pretty
important you use the *.ini file delivered with the UI.
Basically I suggest to use the UI the way it is in the screenshot. I will work in other ways, but you won't get the pictures.
Currently, the skills window is not movable. I know how it can be fixed, but I'm too lazy because noone uses that window anyway. :P

Well, enough of this, enjoy the skin. :o)
