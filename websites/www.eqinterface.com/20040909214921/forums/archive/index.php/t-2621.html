Shown here is my UI setup, I will take no credit for any of the artwork, it is mainly mixed and matched from stuff off of this website and others. As you can see in the Screenshot, I have implemented the EQAtlas map mod to the help window. I don't know about everyone else, but I get lost very often around Norrath, so this helps a lot more than that help menu they started us off with lol. I also have a modified spell book from Thangor, along with his vertical spell bar. I have yet to implement t.king's spell icons, once I get around to doing the xml coding since a lot of mix and match have a problem with the animations xml file. The bags are modded so that you don't have to open your inventory every time you want to switch out weps for instruments, this mod was done by Cairenn. The player window mod was done by Minka, and looks really nice in my opinion. I also have an anorexic inventory window now since I rarely need to open it up since I got the hotbutton mod done by Cairenn again.
That is the main features of this UI, if you have any requests for me to post this as a zipped file, just leave a post here. I also apologize if I have not given credit for your work.
