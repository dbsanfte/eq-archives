<a name="news">Latest News</a>
Default UI for Download: Oara and Celeritas of Solus have graciously set aside space for the Default UI Files (http://www.synex-flux.com/Other/defaultuifiles/index.html), making them available for download when modders can't get to their own files. Both a zip package and the individual files are available, and will be updated with each patch as necessary.

New T.King address: Per Thomus King (http://pub165.ezboard.com/fmaximus94788frm10.showMessage?topicID=74.topic), all his home- and art-pages will be changing from attbi.com to comcast.net on July 16; don't forget to update your bookmarks!

It's About Time! According to Alan (http://eqlive.station.sony.com/community/dev_view.jsp?id=55792), the Plane of Time has been entered for the first time! (Is that redundant?) I'm sure we'll be hearing details soon!

Old News - New Server: From the "Late But In Earnest" file, there's a new server in town: Stromm! From the Developer's Corner (http://eqlive.station.sony.com/community/dev_view.jsp?id=55686), the new server is No Transfer, so all characters start from level 1.
