It should be possible. Someone asked me to do that with my UI, so I'll probably play with it myself this weekend.

If you wanted to have functioning group buttons (invite etc) you'd have to put it all in the group window. If you want it to flash you need to use the player window. And you can't just delete the other xml files, the elements have to be there but drawn with no size or offscreen.

But I can't think of any reason you wouldn't be able to do it
