Fixes:
v0.6
- Group window completely compatible finally. =)
- Health and pet bars made a little longer.
- There should now be enough room for longer names. If they are real long though they may overlap the percentage.
v0.5
- Target Window, fixed problem with name showing up on the health bar and the second percentage showing up.
- Color changing squares on the Target window are now centered as they should be.
v0.4
- Combined my Target, Player and Group color changing mods into one series.
- Slim lined the windows.
- Fixed the name overlay problem on the target window.
- Fixed the problem with one row of pixels blinking even when above 40% health. This
problem existed in all 3 windows.
- Fixed some label and gauge names to allow compatibility with more mods.
- All of the background images have been changed a little to allow for the narrowing of the windows overall.

-------------------------------

Known issues:
v0.4
- Still one or two compatibility issues with the default EQ interface. This does not
affect game play at all and will be fixed before final release.
- Target window, still a little taller than I want it, expect it to be slimmed before
release as well.
- Report issues you come across to this thread.

-------------------------------

Description:
The Color Changing Window Series was designed with healers in mind. All three windows
included in this mod have squares that change colors as health is lost. In the player
window, as you lose health the square will change colors green->yellow->orange then
once below 40% it will flash from red to yellow. Same goes with the group window and
each member in your party. The target window does this as well. These mods can be used
as a full set or you can pick and choose which windows you want to use.
