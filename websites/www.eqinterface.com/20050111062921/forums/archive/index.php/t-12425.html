It is possible that the author of your interface has the chat windows set to transparent. Open your EQUI_ChatWindow.xml file, go to this section:

<Screen item="ChatWindow">
<RelativePosition>false</RelativePosition>
<Location>
<X>95</X>
<Y>280</Y>
</Location>
<Size>
<CX>421</CX>
<CY>200</CY>
</Size>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>

and make sure that the line I have in bold is set to "false" like I show it there.
