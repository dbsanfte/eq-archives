First of all, you should know that even if you make it so the window doesn't look square, that transparent cutout will still be considered by EQ to be part of the window; you can see through it but you can't click through it and if there's another window under it you won't be able to use the functions of that window.

Secondly, the key to using custom tga files for window backgrounds is fund in EQUI_Templates.xml. That has a bunch of templates that tell EQ how to draw the windows, including what background image file to use. Take a look at how the default window templates are set up, then try making your own.

Well...depends what you mean by 'work'. You can make a window that *looks* round, but the only way to define a window size in EQ is as a set of x,y dimensions, creating a rectangle. Hence you can have your pretty little round window but if you click on the transparent corners around the circle you'll get the window, not what's behind it. (exception is chatwindows- you can click through them.)
