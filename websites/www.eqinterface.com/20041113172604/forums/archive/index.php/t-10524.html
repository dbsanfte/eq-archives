I just dowloaded the update for GoD for you patcher (III) and I saw another >300k file that you can remove.
The Thumbs.db file is used by Windows to hold the thumbnail image for your folder on your computer.
By default it is a hidden system file, but you must have changed your Folder View settings to make them visible. I keep mine hidden just for this reason, I figure if I do need to find or change a system file I know what I am looking for and can use the command prompt to get to it.
Removing that file drops the zip size form 854k to 612k.
