For all those who have been waiting so patiently here it is. the Skin that Saldarus and I have been working on for the past couple months. We have it pretty close to complete but we want to give you all the chance to try it out and give us your feed back on how it is and what needs to be worked on.

The file will be uploaded soon. Looking to split the file up so that it will fit in here but for when it is here Following is the direction on getting it up and running.

Once you have the zip file(s) just extract them into a new directory under your "Everquest\uifiles\" directory. Remember the name that you give this directory because you will need it later when you are editing the ini file.

Now copy the "UI_Nechron_86.ini" file to your everquest directory. Check in the everquest directory for the current character ini file for your character and server. Rename your current ini file to something else. My suggestion is to rename it to the same thing and just add ".bak" or ".old" to the end. Now rename "UI_Nechron_86.ini" to the name of your character file.

For example if your ini file is named "UI_Enron_51.ini" then you would rename this file to "UI_Enron_51.old". You would the rename "UI_Nechron_86.ini" to "UI_Enron_51.ini"

Now that the file is named correctly you will need to double click on the file to open an edit window of the file. Scan down through the file and find the line that reads as follows:

UISkin=Default

Now you need to change this line to the name of the directory you extracted the files to that I asked you to remember earlier. For example say you installed the files to the following directory:

"C:\Program Files\Everquest\uifiles\Salchron\"

You would edit this line to read as follows:

UISkin=Salchron

Now Save and Close this file and start your EQ and bring up the character for this UI. The skin should come up correctly with all the windows in the right place. If the windows are not in the right place then most likely the cause is that you have named the ini files incorrectly. You will need to exit EQ and double check the ini file name to make sure that it is correct.

Sladarus and I will be monitoring this topic as well as the topic in the pre beta forum for those who have problems as well as for all your comments and suggestions.
