I had the same problem. Make double sure you have only 1 everquest directory under program files folder. In my case, I had original EQ installed at one time and then upgraded to EQ trilogy later. This created 2 EQ directories, one under:

C:\ProgramFiles\Everquest

and one onder:

C:\ProgramFiles\Sony\EverquestTrilogy


The original EQ folder didn't have the UIfiles yellow folder, but the Trilogy folder did. Everquestlive.com tech support recommends that you delete the outdated folder if you do indeed have two. I deleted mine, and made the changes under the new folder and mods/skins work perfectly now. Hope this helps.
