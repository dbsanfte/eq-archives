08-15-2002, 12:10 AM
As long as you don't replace any of the *.tga files when you copy things from one mod folder to another, then the color / graphic schemes should not be changed. The *.tga files are what tells the mod what you want the graphics and the colors to look like. The info in the *.xml files simply tells the mod where to find the graphics / colors within the *.tga files. But since, in this case it will be looking for something that's universal to the whole mod, there should be no chance of that belonging to any particular *.xml file you'd want to over-write.

~Lady Silver

<Gauge item = "Pet_HP">
<ScreenID>PetHP</ScreenID>
<!--<Font>3</Font>-->
<TextColor>
<R>240</R>
<G>240</G>
<B>240</B>
</TextColor>
<RelativePosition>true</RelativePosition>
<Location>
<X>16</X>
<Y>18</Y>
</Location>
<Size>
<CX>108</CX>
<CY>2</CY>
</Size>
<TextOffsetY>-50</TextOffsetY>
<GaugeOffsetY>0</GaugeOffsetY>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
<!--<TooltipReference/>-->
<FillTint>
<R>0</R>
<G>220</G>

</FillTint>

<R>0</R>

<Lines>A_GaugeLines</Lines>

</GaugeDrawTemplate>
</Gauge>
