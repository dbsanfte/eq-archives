Here it is, Luz1.0

This is my first Mod, and I made something that I personally love (but, of course, my opinion is a little biased) Here is the list of features:

*1280x1024 Resolution only (too much info for anything less, and I personally always played at this resolution)

* Just about ALL of the information is onscreen ALL THE TIME

* Nice, clean graphics that make the clutter seem much less cluttered

* Included, but not shown in pic: containers that include primary/secondary slots (all Bards rejoice!)

* Nice, large viewport that is in proper aspect ratio without any of the ui covering it up (except for the music player... will add this somehow at a later date)



*************************************
INSTALL INFO ***IMPORTANT***

1) Download and unzip into <everquest directory>/uifiles/luz

2) take the file named \"UI_Charname_servernumber.ini\" and rename it by replacing \"Charname\" with your character's name, and \"Servernumber\" with your server number (look in the main EQ directory, and you should see a file named this way\"

ex: My Character's name is Luzienne, and I play on Tarew Marr (server 80), thus I would rename it to: \"UI_Luzienne_80.ini\"


3) If you want to save your old ini file, save the one from the main EQ Directory

4) Move the renamed ini file into the main EQ directory (If you didn't backup the old one, it WILL save over it!!!)

5) Log on and have fun!!!

********************

Notes:

-The UI_charname_servernumber.ini file only changes your UI setup. There is a seperate file that saves your friends/hotbuttons/etc.

-please make sure you are playing in 1280x1024 resolution. This mod will not work properly otherwise
