You can change the size of the icons in the window in which they are being displayed. (Leave the definitions in the Animation file alone.)

In the buff window, look for the <DecalSize> tag. You can change this and it will scale the icons to whatever size you want.

Of course, if you are resizing the icons you'll need to reposition everything as well.
