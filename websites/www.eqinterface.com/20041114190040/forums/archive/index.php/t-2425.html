I hate editing RGB color values. Attached is a mod that provides the basic 16 colors predefined by name. Instructions are included inside the XML file.
This is untested and i'm looking for feedback on the following:

1) naming convention (consider point 4 as well)
2) are the colors correctly defined
3) usage of the alpha channel (I dont know much about it)
4) compatiblity with if a 256 color set were provided as well.


Instructions from the XML file:

<!-- Basic 16 Color Defintions -->
<!-- CS = Color Solid -->
<!-- CT = Color Transparent -->

<!-- usage - in the EQUI.XML add an include line to this file -->
<!-- <include>EQMOD_Colors16.xml</Include> -->

<!-- Use the color name where you set an RGB value -->
<!-- Original: <Textcolor> -->
<!-- <R>0</R> -->
<!-- <G>0</G> -->
<!-- <B>255</B> -->
<!-- <Textcolor> -->
<!-- With Mod: <textcolor>CS_blue</textcolor> -->
<!-- -->
<!-- both original and mod will set the textcolor to blue -->
