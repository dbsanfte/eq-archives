The warnings are fine and not really "warnings" so please ignore them.

The errors you have are the result of sony changeing the labels for the percentage tags back in May.

You will need to update the effected XML files using the information in this patch: http://www.eqgui.com/showthread.php?s=&threadid=8467

You will want to change the values in the patch code otherwise you will have 2 percentage lables for each place theres a percentage in your interface. You will need to change the <x>, <y>, <CX>, <CY> vaules to 0 (zero).
