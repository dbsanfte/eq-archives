[Sat Nov 01 12:40:46 2003] Error: Could not find child ClassAnim in window InventoryWindow

You are missing either:

<StaticAnimation item= "ClassAnim">
<ScreenID>ClassAnim</ScreenID>
<Location>
<X>338</X>
<Y>22</Y>
</Location>
<Size>
<CX>64</CX>
<CY>128</CY>
</Size>
<Animation>A_ClassAnim01</Animation>
</StaticAnimation>

or:

<Pieces>ClassAnim</Pieces>

or both.

If you don't want the class animation to actually show up, change the sizes <CX> and <CY> to 0,0.

Of course, the easiest possible solution is if the author of your UI skin has already made all these changes and provided patches for it, but if not, the info above tells you what needs to be fixed and where to get the stuff you need to do so.
