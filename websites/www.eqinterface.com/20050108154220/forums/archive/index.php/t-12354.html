Just make a text label using <label item> for each page:

<Label item = "SBW_PageTitle1">
<ScreenID>PageTitle1</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>100</X>
<Y>25</Y>
</Location>
<Size>
<CX>100</CX>
<CY>20</CY>
</Size>
<Font>5</Font>
<Text>Page Title #1</Text>
<TextColor>
<R>0</R>
<G>0</G>
<B>0</B>
</TextColor>
<NoWrap>true</NoWrap>
<AlignCenter>false</AlignCenter>
<AlignRight>false</AlignRight>
</Label>

<Pieces>SBW_PageTitle1</Pieces>
Something like this should work.
