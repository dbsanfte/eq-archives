Ok first the wolf issue is because your .txc files in your Everquest directory are corrupt. You need to delete the *.txc files from the Everyquest directory and that should be fixed. Don't worry about whethere this will break things. These files are dinamically created when you start EQ if they are not there. They don't get down loaded so they are not verified through the file check that the patcher does because they are strickly client side based. But if you delete them and then restart EQ your wolves should be good to go again.
