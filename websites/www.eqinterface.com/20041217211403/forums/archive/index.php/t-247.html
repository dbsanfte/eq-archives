From what I can see, it's not possiable.

We don't have option 1-100 for Hotkeys. We have option 1-10.

Each page of hotkeys is the exact same page. The only difference is that EQ reads your .opt file to find out what values are for what page.

Everytime you "flip" to the next page, EQ loads those values in.

Thus, giving you the illusion of 10 pages.

I bet VI would have gone 12-14+ if it wasn't for the fact there's only 10 number keys ( Shift-# to switch ).
