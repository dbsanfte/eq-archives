First, looking at your UIErrors file you have more to fix than just this one window.

Now, according to this line:
[Fri Jan 02 06:28:16 2004] Warning: file UIFiles\RE\compton_buffwindow.tga not found in directory UIFiles\RE\. Attempting to use file from Default skin.
The UI is looking for the compton_buffwindow.tga. The graphics file that the window uses. To add it into the file you need to know the image size of the graphics file, ie the H and W in pixels of the image. Once you know that you can add this code, but insert the H and W to replace the #H# and #W# here:

<TextureInfo item="compton_buffwindow.tga">
<Size>
<CX>#W#</CX>
<CY>#H#</CY>
</Size>
</TextureInfo>


Now after that you need to fix these files as well:


Hope this helps

SkillsWindow
TargetWindow
PlayerWindow
AdventureLeaderboardWnd

They may not be causing trouble now, but could potentionally if those windows are changed in future patches.
