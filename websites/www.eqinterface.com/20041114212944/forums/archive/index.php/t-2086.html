Haven UI v1.1

Screenshot (http://www.eqinterface.com/sdownload/preview.php?id=227)

Download (http://www.eqinterface.com/sdownload/download.php?id=227)

Haven UI is heavily based on the Quartz UI. The key differences that Haven carries are:

- Frequent updates to keep pace with Verant\'s UI changes
- A built-in graphical config utility
- Smart updates (part of the config utility)
- Ability to toggle the Help icon (especially useful for some plugins such as Timer) and pet information
- Ability to toggle buff names and character class configurations graphically
- A revised buff window
- A stable pre-built INI
- Executable installation (and patching in the future)
- Start Menu shortcut for the config utility and readme
- Revised (easier to read) server list (part of the config utility)
- 100% lit compass by default


New in v1.1:

- A revised installer that should work for everyone.
- Ability to toggle pet info.
- Config Utility v1.2 that works with the new installation format.

Ok... at least only 2 people seem to have this problem. That's not bad out of over 300 downloads ;)

Here's a fix for you two to start with:

1. Open the HavenUI.zip file and drag'n'drop Haven.exe to your desktop.
2. Right-click Haven.exe and select "Extract here"
3. Cut'n'paste the "haven" folder from your desktop to Everquest\uifiles.
4. Browse into Everquest\uifiles\haven\data and right-click on "hconfig.vbe", and "Send to\ Desktop (create shortcut)" (hconfig.vbe is the config utility)

That's all you need to do, and hopefully the config utility will work for you althohgh I'm concerned that if the installer didn't the config utility won't either. :\
