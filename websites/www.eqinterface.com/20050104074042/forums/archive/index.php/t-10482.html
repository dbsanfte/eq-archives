10-02-2003, 02:39 PM
Almost sounds like he is talking about the menu headings.

ie: Start->Program Files->Sony->Everquest

But the problem has to do with the installation directory/folder.

The original games installed into C:\Program Files\Everquest

the newer games install to C:\Program Files\Sony\Everquest

You need to copy all your UI files from the first install path to the new one. Then, if all works well, delete the C:\Program Files\Everquest folder and its subfolders to keep you from getting confused again later.
