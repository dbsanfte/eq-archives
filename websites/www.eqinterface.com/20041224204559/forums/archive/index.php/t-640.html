Not all Graphic changes require modifications to the EQUI_Animations file.

If you can modify the graphics within the current EQ size contraints, then you don't need to modify the file.

However; there is currently an open bug that doesn't load the wnd_bg_light_rock.tga and the window_pieces01-05.tga files from the custom directories. Until this bug gets fixed, it will require modifications to the EQUI_Animations file to make them work.
