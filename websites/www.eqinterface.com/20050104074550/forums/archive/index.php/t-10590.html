Unless your window_pieces01.tga file is actually custom, you don't have to do a thing with it, that is just there for "information" sake, the same as the rest of the messages (ie if it doesn't apply to you, ignore it).

If you do have a custom window_pieces01.tga file, you need to use a graphics editor (PhotoShop, PaintShopPro, Gimp, etc) to compare your custom one to the one in the default directory and make any applicable changes (or have someone else do said for you).
