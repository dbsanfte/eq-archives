On another post this has been answered, but I will add a little extra


If I understand you correctly, I believe you're looking for the /viewport command.

The syntax is :

/viewport <TopLeftXCoord> <TopLeftYCoord> <width> <height>

and the all-important /viewport reset.

For instance, if you're running in 1024x768 resolution, try...

/viewport 0 0 800 600

If you don't keep a 4:3 ratio between width:height, I'm not sure if the view gets clipped or 'smooshed'.


In addition to this if using above example: /viewport 0 0 800 600 it places the window in top left corner, as they stated "The syntax is:"

To move the same "port" around screen do so in small amounts like /viewport 15 0 800 600 - If you try larger numbers too much it may throw the window WAY off.
