Notes
-----
This UI is updated to the expected May 14, 2003 patch.

This UI is designed to be used with 1280x1024 resolutions, Do not use the included .ini if you use a different resolution. It uses a mix of mods that I use. I have further changed most of them to separate the graphics and animations out of the Window file and into a custom file for the creator.

This is unfinished, and I am still tweaking the Animation files to make them more independent and as unbreakable as possible.
