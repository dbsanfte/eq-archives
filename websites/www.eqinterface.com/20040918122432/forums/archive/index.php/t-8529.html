1) Enoks UI tking - Quick Note:

- To move windows:
Each window has an invisible edge on the left and right
Click on the edge (one pixel wide) to move the windows
The right edge is hard to find, it IS actually there
The inventory also has top and bottom edges
You can right click in the stats area to lock/unlock

2) Enoks UI tking - Instructions:

- Installation
Extract files to a subfolder of the EverQuest\uifiles folder
I suggest naming the folder enoktkde
I included the UI INI file that I use with my UI
The included INI file references the folder enoktkde

3) Enoks UI tking - Frequently Asked Questions:

Q: Why can't I click on links in the chat window(s)?
A: The "Sneeze Guard" prevents clicks on hitsmode chat windows; to
fix this simply click the little "locked" icon in the Main Chat
window's upper right corner, then links will be clickable

Q: Why can't I resize/move my chat windows?
A: You can, just hover the mouse over the top left and right
corners to resize, over the top middle to move

Q: Why is the title for my chat windows so messed up looking?
A: It just is, rename the chat window title to . (period) to fix

A: /shrug
