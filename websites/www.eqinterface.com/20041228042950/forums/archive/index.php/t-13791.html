<img src="http://www.eqinterface.com/images/sections/eqpatch.gif" align="right">A patch released on 5/19/04 changed the 5yr old login system to Everquest. The new login screens use the XML interface system and they can be modified. If you plan on using modified login interfaces you will want to read the authors install procedures as they are much different to install then the norm.

You can easily find the new login XML/TGA interface files since the file names all start with EQLSUI in the default folder. You can read more on what has been discussed here (http://www.eqgui.com/showthread.php?s=&threadid=13778).
