I've never used Vert mod but the standard practice when you add a casting gauge into another window is to modify the EQUI_CastingWindow.XML to resize the casting bar to 0 height 0 width, this way it doesnt show at all. Using this method when you hard-code a casting gauge into the new window, the casting gauge only appears in the new window.

My suggestion would be to look in your vert ui folder and delete "EQUI_CastingWindow.XML" if there is one present, as its likely just a 'dummy xml' as I like to call it, one that serves only the purpose of keeping a window from showing up.
