Nope, you cant copy the single addition into the default folder otherwise when you run the patch it gets overwriten again.

You have to create a seperate folder in the uifiles directory. You will see one called "default" well create one called "whatever" and copy all the files from "default" to "whatever". Then overwrite the file in "whatever" and log on.

Once inside the game use the command /loadskin whatever true

This will make your folder "whatever" as your skin directory and you should be good to go... Of course replace "whatever" with a name you want for your modified skin.
