10-08-2003, 03:03 PM
The problem is:

In vert 2.6 Window_pieces06.tga and window_pieces07.tga were renamed to solwindow_pieces06.tga and solwindow_pieces07..

That player window has Ui2DAnimation items with Window_Pieces06.tga and 07 as the textures..

EQ's looking for a TextureInfo item in the EQUI_Animations.xml that matches Window_Pieces06 and 07.. That texture info item is not there so it creates that vague warning..

I changed all the Window_Pieces to solwindow_pieces in the attached file.. I haven't been able to test it.. But it should clear up the issue..
