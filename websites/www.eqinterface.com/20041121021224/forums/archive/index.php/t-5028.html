RGB value stands for Red Green Blue value. The values are from 0 to 255. If values are set 0,0,0 you will get black text, if they are set 255,255,255 you will get white text. You'll be looking for the following flag in the EQUI_spellbook.xml :

<StaticText item="SBW_SpellName~number~">
with the ~number~ being from 0 to 15 for each of the 16 spells you have on a given set of pages.

below it, about 15 lines, you'll see:

<TextColor>
<B>0</B>
<G>255</G>
<R>0</R>

These are the 3 values that define the color of the text. You will need to change these values for each spell name, ie, 16 times.

Hope this helps.
