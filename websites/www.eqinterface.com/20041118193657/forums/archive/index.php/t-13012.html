There are multiple methods for changing chat window looks. One of which is very user friendly to just dropping a file into your UI and being done with it. I believe clean by dealig is one of those UI's and has very clean borders. Just remeber what affects the borders of your chat windows also affects every other border in your UI in most cases. The file thats normally used to "reskin" chat windows is window_pieces04.tga

You dont want to atempt to use chat windows created with any other method if youre unfamiliar with doing your own code.


So find a UI you like the chat windows in. Just add the window_piece04.tga from the UI to your UI folder. See how it looks. If its fubar. Delete window_pieces04.tga and look for something else.

Good luck,
