3Danalyze.
http://www.tommti-systems.com/main-Dateien/TOOLS/dontlinkthefile_3danalyzer-v22.exe

Input "4098" for the vendor ID
Input "20040" for the hardware ID
DO NOT check anything!

This will tell EQ that you have a Radeon 9800 card and you should now have the ring.
Since you are not forcing software TNL there are no artifacts.
This worked for me. It's not my idea btw, I found this over on the SOE tech forum.
Good luck,

Originally posted by Sariis
I have a 9500 with exactly the same problem, would this work for that card & if so, what codes would I require to enter please

I think the 9500 is supposed to work as is.
Do you have these lines in your eqclient.ini file:

UseTNL=1
Show3dTargetIndicator=1

If not add them.
Have you copied these two files from your default ui folder:

TargetIndicator.ini
TargetIndicator.tga

If not add them.
Do you have 3D target ring turned on in the in-game options window?
If you've done all the above and still no ring...then try the same values as in my original post. Shouldn't make any difference really, will just report the wrong video card.
