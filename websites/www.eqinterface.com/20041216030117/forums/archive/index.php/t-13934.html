TimerMod (http://www.timermod.com/) version 4.0 features a new type of timer with lots of customization options.

You can choose from 28 different windows in the UI for your custom timers. Also, an exciting new feature is the ability to reconfigure your timers quickly and easily without having to re-enter all the information. Just use the link provided in your dynamically-generated file package to launch the configuration wizard with your existing settings already filled in.

Timer Features
The marker can be set with a single click. Simply click on the location where you want the marker and it will automatically jump to the spot where you clicked.


Specify the name and duration for each timer to match the events you want to time. For easy identification, you can assign any spell/item icon in the game to each timer and choose from 12 colors for each timer bar.


Each timer's name and duration appears as a tooltip when you hover your mouse over the timer's icon.

http://www.timermod.com/image/screenshot/tooltip.jpg


To begin timing an event, just set the marker to the current position of the dot. The dot will continue moving and reach the marker's position again when exactly one duration of the timer has elapsed. As the dot approaches the marker, you will know that time's almost up.


For timing variable-duration events, tick marks on the timer bar make it easy to set the marker and see the amount of time left.


If you wish, you can add additional pages of timers so you can use different timer configurations for different situations.

http://www.timermod.com/image/screenshot/tabs.jpg


Include up to 8 timers in your configuration, or up to 12 timers divided between tabbed pages. You can configure up to 4 tabbed pages of timers.


The timer bar can be one of 13 different sizes. Choose a larger timer for better granularity or choose a smaller timer to conserve screen space.

http://www.timermod.com/image/screenshot/large.jpg http://www.timermod.com/image/screenshot/small.jpg


Customize the layout of your timers by specifying the number of columns and rows. The timers can be horizontal or vertical, and the icon can be placed on either end of the timer bar. If you wish, you can also specify the spacing between timers.

http://www.timermod.com/image/screenshot/vertical.jpg


For each timer, you can choose between two marker styles. A standard marker can be set by clicking anywhere on the bar, and is good for timing events that only have a single instance, such as abilities and some spells. A split marker contains two halves that are set independently of each other by clicking on the corresponding half of the bar. Split markers allow you to time two instances of the same event, for example, staggering costly buffs or mezzing a late add.


Choose from 4 different base skins for better integration with your existing UI.Visit the TimerMod home page at www.timermod.com (http://www.timermod.com/) for more information and to create your own custom timer configuration.

With regards to the images in this new excellent timer mod - will it be possible to use ones own images, instead of the default EQ ones?I'm not sure if I'm reading your question correctly, but if someone wants to use alternate spell/item icons (for example, T.King icons) instead of the default ones, this can already be done by simply placing the custom icon files in the same mod folder as the timer. The timer will automatically use the corresponding alternate icons.

If a player wants to create a few of his/her own custom icons for the timer, this can currently be done by modifying one or more existing icon files and then placing those modified files in the same folder as the timer. Of course, any place those icons are shown in the game, you will see the modified version, so if the custom icons are intended for just the timer, it would be wise to choose icons that are not frequently seen (like obscure items).

If there is a large demand for allowing timer-only custom icons (as opposed to custom spell/item icon sets that are used throughout the UI), I can add an option for the timer to use icons from a custom file. It would be up to the player to provide the custom file by placing it in the same mod folder as the timer.
