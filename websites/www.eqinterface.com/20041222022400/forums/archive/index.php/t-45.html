07-27-2002, 12:04 PM
It is not possible to add or remove buttons into a window. The code to manage the buttons is hard coded into the new EQ Interface. Removing a button causes UI errors on loading as the code looks for a button and doesn't find it. Adding a button doesn't do anything as the code to manage the button press isn't there. It is possible to add the other stuff you have requested.
