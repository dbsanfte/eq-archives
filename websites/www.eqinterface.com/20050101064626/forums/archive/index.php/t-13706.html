Quick and dirty version of the "How To Install mix and match UIs":

1.) Create a new directory in your uifiles directory. Name it whatever you want.

2.) Choose ONE modification (preferrably just one window) and copy the modified .xml file and all supporting graphics into the new directory.

3.) Fire up EQ. At the command line type "loadskin [dir name]". Make sure it works and that your new window shows up.

4a.) If it works and you wish to add new windows, go to number 2, choosing another modification.

4b.) If it doesn't work, go to your EQ folder and look for a file named UIErrors.txt. There will always be a long list of stuff. Look for the one that refers to the filename that you just added to your custom folder. The error message won't always be really clear, but it might help you figure out the problem. If you can't figure it out, remove the faulty modification from your custom folder and go to 3.

5.) Once you've added all the modifications you wish to add to the single custom folder, you can breath a sigh of relief and begin positioning and tweaking all your new windows.
