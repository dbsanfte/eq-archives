Sounds like you have a viewport set in your ini file.

Try setting it like this... in-game type:

/viewport 0 0 xxxx xxxx

The x's stand for your resolution. So if you play at 1024x768, the command would look like this:

/viewport 0 0 1024 768

That should get rid of the black area.
