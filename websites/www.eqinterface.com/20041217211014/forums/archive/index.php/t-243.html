I posted this in another thread about the TrackWindow, but as I can't find the link (site is too slow at the moment), I wil repeat it here.

The EQClient expects certain elements to be in the TrackWindow. The only way to get rid of elements is to 1) set their sizes to one pixel OR 2) set their coordinates outside the window to hide them. REMOVING the items will crash EQ or drop you to the default skin.

Hope this info helps.
