I've had the missing buff bar issue happen to me as well. Here's how I resolved it:

1. Type: /Load default (leave off the number 1, we want the boxes to go to the default locs)
2. Move the chat boxes and buff box to the the center of the screen.
3. Type /Load default 1
4. Type: /Load <whatever you named the UI folder> 1 (Be sure to use the number 1 at the end to lock the widows in place while the UI loads)
5. Move and stretch the boxes as desired.

Hope this helps :)
