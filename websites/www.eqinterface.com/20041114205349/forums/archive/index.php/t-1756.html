Here's my new targeting window, it started out quite a while ago done totally by hand in Photoshop and has gone thru countless revisions. The current incarnation of its look was done because the photoshop version was looking too "flat" and 2d. It still has quite a bit of polishing and tweaking needed, as well as some memory optimization that's just gonna be a major pain but won't get done 'till I'm completely done with the graphics. BTW, it's ANIMATED =)

http://www.datacore.org/dariath/preview.gif

Basically I'm posting to see if anyone would be interested in me finishing it up and making it into a real mod; or, if it's too large to be used by anyone but myself. As a cleric, I spend plenty of time looking at this gauge and I wanted to make something a bit more exciting than a red line. This is the only version of it I actually have packaged up and ready to distribute, and is kinda old actually.

By actually making it into a mod, I mean cutting down the 24 seperate frames of animation by half or so and combining them into one file, then coding all that into it. I also have about 5 different versions I've made that I might post to get input on which one people like best. We'll see...

As of right now some people may have issues running it but it works fine on my computer so I'm looking for some input on how it works on other peoples computers. If it doesn't work for some reason please let me know or email (dariathvaz@hotmail.com) me so I can try to fix it.

Click Here to Download (http://www.datacore.org/dariath/spiffy_target.zip)

Installation:

Just drop these files into whatever UI directory you like, it should replace the target window with my new spiffy target window. Please read the readme.txt
