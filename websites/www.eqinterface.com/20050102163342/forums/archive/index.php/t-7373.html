you can also achieve "full screen" mode after a bad alt+tab experience by doing the following:

navigate to your EverQuest installation directory (default is C:/Program Files/EverQuest

find the EQCLIENT.INI file

open EQCLIENT.INI with notepad

scroll down through all of the .ini keys, there is an entry in the file that looks like this:

WindowedModeXOffset=0
WindowedModeYOffset=0
WindowedMode=FALSE <---(might also say "TRUE" here)

TRUE = Run EQ in a window
FALSE = EQ is not run in a window

it may be a longer way to achieve full screen after an alt+tab experience goes awry, but it works 100% of the time.
