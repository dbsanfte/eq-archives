Xeatau,

I suspect your EQUI.xml file is custom so besides the three files you need to do the following from Caireen's post in another thread:

IF you have a custom EQUI.xml file, open using a text editor (Word, WordPerfect, NotePad, etc.) and:

Using the text editor's Search or Find function, find:

<Include>EQUI_StoryWnd.xml</Include>

Immediately after that line, ADD:

<Include>EQUI_BigBankWnd.xml</Include>
<Include>EQUI_BodyTintWnd.xml</Include>


BTW, you should delete those files from your custom ui folder. They only need to be in your default folder (at least until somebody here mods them!) If the patcher updates them in your default folder your ui might break again if you still have them in your custom folder.

--Syagush
