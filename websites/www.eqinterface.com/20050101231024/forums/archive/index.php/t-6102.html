Well, first you need something able to view and save .tga files.

Open it up, then select the inv_portrait.tga file (possibly Select All) and "copy" it.

Open your Paint program and make the white area fairly small (smaller than the .tga file).

Now hit "Paste".

If you couldn't paste, go back to the .tga file and copy it until you can paste into the Paint program.

Now, take the picture you want and cut it down to the size of the rectangular window that isn't white. Some programs can resize pictures, but you will have to cut down the edges of the picture most likely. I simply use a second Paint program window to cut off the edges.

Once the picture is acceptable in the Paint program, go to "Edit", then push "Select All". Now go to "Edit", then push "Copy". Now, go to your .tga viewer and "Paste" over the old inv_portrait.tga file. Save it over the old files name. With any luck, you should be able to see that in your inventory when you get into the game.


So you know, the viewable area in the inv_portrait.tga file is 124x206 pixels.

If you need a graphics viewer that can save .tga files, go to www.tucows.com for Freeware software. (more cows = better software)
