07-14-2003, 11:59 PM
If you're getting some sort of error message about files being in use when you're deleting the old folder, just reboot and delete the old soes folder.

When you download the 1.5 version, it should unzip to a folder called Soes15. Inside that folder is another folder called soes (actually, this will be the only thing in that folder). Copy THAT folder into your uifiles folder in the EQ directory.

Now, open the soes folder and find the UI_Yourchar_Yourserver file...open it. Select everything and copy it. Close that file and go back to the main EQ directory.

Find the appropriate UI file for the character you want to use this UI with (UI_name_##). Open that file, select everything and delete it. Then paste what you copied earlier into that file, save it and close it.

Start EQ and you should be good to go...no need to type /loadskin or anything. BTW, you're going to have some error messages about the UI because not every window was changed, so if the game can't find that window in the soes directory, it'll use the default window. Those error messages in the logfile are just telling you that it's using the defaults for those items.
