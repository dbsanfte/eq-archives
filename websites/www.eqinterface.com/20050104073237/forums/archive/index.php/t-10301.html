To get the Chat windows to work as you would like:

Go to your AYA File.

Inside you will find a folder "Optional XML and TGA"
Inside this folder you will find "Border Window"
Inside this folder you will find "EQUI Chat" and "UI_Character"
files.
Copy both to your AYA UI. Rename the "UI_Character" To your character name and server.

This is a GREAT UI. I have made some mods to it. But the lay out is Great.
