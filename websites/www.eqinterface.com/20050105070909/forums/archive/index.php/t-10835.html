A patch released on 5/14/03 added the percentages to the default interface. They renamed the labels in the XML for percentages and its not finding those new label names in the player and target XML.

Go here (http://www.eqgui.com/showthread.php?s=&threadid=8467) and preform the updates to the player and target xml. You will most likely want to set the <cx> and <cy> values to 0 in the patch code otherwise you will have 2 percentages for each guage in your player and target windows.
