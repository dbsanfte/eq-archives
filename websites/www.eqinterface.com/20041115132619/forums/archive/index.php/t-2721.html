Originally posted by aasimon
My questions concern choosing the Windows to overwrite with EQTC. If I place EQTC (any of them) in Friends, does my /who friends all no longer work? Same question for Target window...will I no longer be able to target if this is chosen?

If you are using the default interface, putting the Tradeskills Companion in the Target window will not stop you from targeting, but you won't see the target. Putting it in Friends will remove the functionality of the friends WINDOW only. /w all friends will still work, you just wont be able to call up your freinds list and edit it FROM THE WINDOW. ALl text commands will still function as before.

My suggestion, if you want to use the Tradeskills mods would be Help, Gems, Bug, Friends, Notes or Pet (if you aren't a pet class) Unless your using a UI that has adapted another window to reflect the seleced target I wouldn'ty use Target

Also, just a reminder. You never actually overwrite the default files. You create another folder and only place in it those files you are using. EQ will use the default files for anything not in that folder. If for some reason you don't like the mod or it isnt working for you all you need do is /load default and voila. Back to basics.

No it says only 1 window per skin.. so i guess if you want all 3 make 3 diffrent skins and do a load skin for the diffrent ones.

Actually yes you can have all 3 in 1 window you just have to do some moding of your own, and you'll have tabs out the wazoo. First download all the files. With one of them being the window you want the mod in. Then open up the EQTC_<Recipes, Locs, or Quests>.xml files. Whichever copy all but the list of pages part of the 2 files that are not already in the window you want, then paste them above the list of pages in the one that is in the window. Then Copy just the <page> (pagename) </page> part of list of pages from the 2 others and paste them into the one that is in the mod. And poof ya got them all in one window, but that's a lot of tabs.
