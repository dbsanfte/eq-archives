This is not something I made nor something I'm trying to take credit for but the current version of "avi-group2d.zip" group window mod is broken. The author completely removed all references to the invite/disband/follow/decline buttons in order to remove them. I downloaded this mod because I loved the layout but when I loaded it up I saw that fateful red errror message you get if you remove things VI requires to be in the code, saying "Your current UI will not work with EverQuest please load the default....".

If you use the uploaded version it will still load but you'll get that annoying error message every time you zone. I fixed the mod so that the the buttons are still there just not visible and this makes EQ happy.
