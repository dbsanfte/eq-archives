02-26-2004, 07:58 PM
that is the default map window.. you can add any modded "mini" map window really, but the EQUI.XML in my mod is specially set to only read the the modded windows I have in there.

if you wish to add a map window from someone else simple open the EQUI.xml in notepad and find the line
<Include>../default/EQUI_MapViewWnd.xml</Include>
<Include>../default/EQUI_MapToolbarWnd.xml</Include> - (this line is the little toolbar used for those who like to add labels and lines and such to their maps, not all mods will have this included, please be sure to change only the line that matches the modded XML files)

to read

<Include>EQUI_MapViewWnd.xml</Include>
<Include>EQUI_MapToolbarWnd.xml</Include> - (Ditto)
