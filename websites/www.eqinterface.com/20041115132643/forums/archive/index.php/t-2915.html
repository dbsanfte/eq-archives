Heh...trial-and-error is about the only way to learn :( try finding a window mod very close to what you want, then using SidlWidl to modify it so it's more like what you want- then try loading it in-game. Do this one window at a time, so if the UI crashes you know what caused it. And remember, certain functions are hard-coded to certain windows- for instance, the Autoequip can't be moved out of the inventory.
