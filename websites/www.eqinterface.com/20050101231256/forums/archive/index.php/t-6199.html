Hmm. Looks like old files causing mayhem.

Try this... delete or move to a backup folder the following files in the directory/folder you use for this UI.

EQUI_PetInfoWindow.xml
EQUI_OptionsWindow.xml
EQUI_LootWnd.xml
EQUI_SelectorWnd.xml
EQUI_BugReportWnd.xml

Once you move or delete those files, it will cause the UI to use the up to date files from the default UI folder. This may however change the look of the skin if any of those windows were customized. So again, you might want to just move them to a backup directory so you still have them in case you need them for further reference.

Once this is done, load up EQ and see how it turns out. If it's just old files you should be all set... or there might be other problems such as a customized EQUI_Animations.xml file.

This is a good first step to determining a solution however, as it just might be the only thing wrong. Also, the name of the skin you are using might help too!

Some users that followed the advice of 'copying the default UI files to your other mods folders' had this happen a whole lot.
