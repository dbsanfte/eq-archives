The new pet window is resizable horizontally but not vertically. This is hard coded the same way the loot window can be resized vertically but not horizontally.

The problem is that your .ini file is storing an old vertical size. To get the size to change to your desired default, you have a couple of options;

1) Edit your UI_char_server.ini file and manually set the dimensions of your window to the desired size in all resolutions.

2) Do a "/load uiname" without the 1 flag to make everything move to default positions and sizes, then move it all back where you want it.
