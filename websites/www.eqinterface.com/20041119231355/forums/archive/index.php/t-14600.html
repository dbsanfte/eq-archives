Quick question for you...

Did you create new graphics files from scratch, or modify the existing ones?

It's not a problem either way... but if you create new files you must remember to make the hieght and with an EXACT power of two. So if your window is 200 x 200 ( for example ) the graphic file you create MUST be at least 256 x 256 or it will distort when you load the UI.

As a side note, if you're using someone else's UI you should keep the graphics files EXACTLY the same size. Some UI modders will make the graphics files an odd size, then compensate for the distortion by playing wierd games with the size and position of the window pieces.
