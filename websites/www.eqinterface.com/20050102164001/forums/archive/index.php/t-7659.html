Please pardon me for an unsuccessful attempt when searching the forums for any existing information on this subject.

What I am looking for is a UI that looks like the old pre-luclin UI. Remember that beautiful, simple chat box? The one that looked like a thin rectangle drawn in MS-paintbrush? Yeah, I want all of my windows to look like that again. No fancy vines, busy colors, and no window names. I think everyone plays often enough to know which window they are looking at without having to read the window title as a reminder. Yes, I know you can just put a "." for your transparent window name and it's barely noticable, but for any opaque window the name area is just taking up space unnecessarily.

One day, a few months ago before the old velious UI was abandoned, I switched to the old UI for kicks and realized how simple and non-distracting it was. Don't get me wrong, I like the features of the new UI such as item links, multiple chat windows, the compass, the song window, and the map system on occasion, etc..., and I would like to retain these features, but every UI I have seen so far is cluttered, sci-fi, or artistically bold and I just don't have time to create a simple UI of my own.

I currently use the default UI that loads with everquest because I got sick of looking for updated UI's whenever a UI patch occured, however I would gladly do the extra work for a simple one.

Does anyone know where to find a bare-bones, user made UI that closely resembles the one from the velious era?
