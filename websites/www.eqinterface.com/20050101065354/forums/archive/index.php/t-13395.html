04-05-2004, 08:36 AM
Find one that you like, download it. Extract the zip to your drive:/.../everquest/uifiles folder. Each ui should have its own folder after extraction. Read any notes or readme files that are included. Once you have everything extracted and any UI_Player_Server.ini file changes made it should load automatically into the game. If it doesn't, open your options window, go to load skin and select the folder name of the UI. Then you should be good to go.
