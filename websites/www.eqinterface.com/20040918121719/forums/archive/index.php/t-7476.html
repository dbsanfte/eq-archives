Due to popular demand on many sites/forums I have decided to re-release my INI Copier, and I promise to leave it up this time. ;)

This will be the only place hosting it so please treat this site well (http://www.eqgui.com/donation.php).

Screenshot (http://www.eqinterface.com/sdownload/preview.php?id=1632)

Download (http://www.eqinterface.com/sdownload/download.php?id=1632)


This is a utility that uses an easy-to-follow graphical interface to copy the contents of one of your character's UI INI files to another character's UI INI file. This gives you the ability to quickly set up multiple characters to use the same UI with the same window settings.

It does not move or otherwise change your friends list, hotkeys, etc. and it also saves the UI INI you are overwriting before it writes any changes, making this utility risk-free.

Please see the Readme file included with the zip for usage instructions.


Enjoy!


P.S. It's a shame that the removal of this utility created more positive feedback than it ever got at release time or in the 2 months it was originally available. Think about that.
