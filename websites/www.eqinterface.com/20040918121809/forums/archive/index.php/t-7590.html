You will need to change both the inventory.xml and Hotbuttonwnd.xml files to put in your class animation.

The text you will need to change in OldCelt01 Inventory .xml is found under the "SELW_ClassAnim"
heading. The exact text you change is:

<Animation>A_ClassAnim06</Animation>

-where you put in the number of your class animation. example is to change from Druid to Cleric - change 06 to 02

<StaticAnimation item="SELW_ClassAnim">
<ScreenID>SELW_ClassAnim</ScreenID>
<Location>
<X>376</X>
<Y>24</Y>
</Location>
<Size>
<CX>56</CX>
<CY>118</CY>
</Size>
<Animation>A_ClassAnim06</Animation>
</StaticAnimation>
-------------------------------------------

The text for the hotbutton window in OldCelt01 is headed by "ClassAnim" where you again change

<Animation>A_ClassAnim06</Animation>

to what ever number you need for the class animation. (I have a list of the numbers in the instructions)

<StaticAnimation item="ClassAnim">
<ScreenID>ClassAnim</ScreenID>
<Location>
<X>150</X>
<Y>59</Y>
</Location>
<Size>
<CX>42</CX>
<CY>80</CY>
</Size>
<Animation>A_ClassAnim06</Animation>
</StaticAnimation>

This is all you have to do. Except save :)


As for the fuzzy text you may have the anti-aliasing turned on or smooth font.. I forget which is which but there are posts in EQ Interface Forums that deal with the problem. I run XP and have standard font smoothing turned on with no problems, but some people have had problems and so turn it off. It's found by rightclicking the desktop go to Properties- Appearance-Effects, and you will see "use the following method to smooth edges of screen fonts" you can see what you have checked (or not) uncheck it and see if that helps.
