As to the group window last player item. I will look at that right now. I guess i haven't been in full groups since i started using it hehehe. Course I havent even been playing much cause i been working on other resolutuions for this UI. I will be posting AGAIN on the mix and match board because I had to strip down the UI to get it to fit.

As to the right clickies well it is designed for druids those are the most common right clicky items for druids. To change this you will have to go into the EQUI_htbtn.xml file and change the inventory items to a different slot. This is a very long discussion, however there are threads on this web site that may help you with that. Feel free to mosify it to fit you.

As to the fontsizes in chat. You will have to do a /chatfontsize and set it to what you want. You can also change the colors etc. The stuff in the hitsmode window I would not change since it is sized like that to fit the interface. Obviously if you make it real big its gonna be screwed up because the UI doesnt adgjust to fontsizes. But your main chat window and the spell/fight spam window should size nicely.

Hope this helps
