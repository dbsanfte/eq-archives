Some of this could be done outside the scope of the UI, I think.

There is no way that I know of to list players in a raid, but you could have them all join a channel and list them that way.

You could write an external program that would scan your log file, searching for some key phrase like "AWARD DKP". So you would get your players into a channel, say "AWARD DKP START 1", then "/list raidchannel", then "AWARD DKP END". Your external program would read the names from the raid channel, read some external config file with everyone's dkp totals, and update the totals in the config file.

It could also scan for things like "SPEND DKP Luminarium 25" to track expenditures.

If I were doing this, I would put all the keyword messages in a special passworded channel to avoid potential false messages.

For viewing the information in game, you could have your external program add comments to your guild file or just format it in a story window file. It's probably easier to Alt-Tab to Excel or something.
