<a name="news">Latest News</a>
Per the EQ Live Developer's Corner (http://eqlive.station.sony.com/community/dev_view.jsp?id=53400), there are some major changes in store for the EQUI. The following are already on Test, and are apparently planned to be implemented after 01/01:
The old interfaces (both the original and the fullscreen version released with Velious) will be disabled permanently.
EQ will be playable in either the current Fullscreen mode or a new Windowed mode (similar to what has been possible through EQW; bannings related to the use of third-party windowing products have reportedly been reversed).
A second effects window will be added to the set, the 'Bard Box', "that would hold selected short-duration beneficial effects, most notably many bard songs. Initially, for tuning, the new effects area will allow six additional songs to be placed upon your character."
It's implied that there are other features still to be seen... discuss the upcoming changes in this thread (http://www.eqgui.com/showthread.php?s=&threadid=4953)!
