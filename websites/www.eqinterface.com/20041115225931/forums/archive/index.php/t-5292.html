I needed to get a free window for a timer so i combined the player and the target window.

I like my UI as simple as possible with only the absolutely necessary information displayed and this shoes in this window.

The target window must be replaced for this one to work properly. Replace it with the empty EQUI_TargetWindow.xml provided in the zip file or another window of your choice.

And you can see it here (http://www.eqinterface.com/sdownload/fileinfo.php?id=852)
