Features
-----
Based one the T.King "Celtic Broach(round)" Spellbar version 1.0.
http://www.eqinterface.com/sdownload/upload.php?s=&op=fileupload&catid=74

The center gem acts as a Combat Effect Time Remaining Gauge and button to open the Combat Skill Select Window, the gem order is as follows:

..2 3..
1.....4
5.....8
..6 7..
