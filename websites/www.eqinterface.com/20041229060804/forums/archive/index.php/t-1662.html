Thats is a extremely nice feature.

This will allow to make an UI where you dont loose anything of the viewable playscreen. For example, you could set a 800x600 screen in 1024x768 and use the black areas to put all your windows.

Warning: be sure to keep aspect ratio because if you dont, you will loose part of the image. For example, a 800x600 screen in 1024x768, you dont loose anything. But if you do for example 800x500, you will end up loosing 100 pixel on the y axis.

Again, this is major feature for UI design.
