Sorry, I was messing around with it both ways. You can switch back and forth easily in game with the /viewport command. To get the proper window in 1024x768 mode, type this command in game:

/viewport 80 0 864 568

At 1280x1024, use: /viewport 80 0 1120 840

At 1600x1200, use: /viewport 100 0 1400 990

The format is "/viewport x y width height" where (x,y) is the distance from the upper left corner of the screen to the upper left corner of the viewport.

1) If I want to use the normal EQ inventory box, do I just delete the inventory file in my xym folder?

You are both correct. Simply delete the EQUI_Inventory.xml file and you will get the default. Hopefully that should work for any window in the UI.

In general, if you want to try something like that, create a backup directory and move the files into that instead of deleting them. It's a lot easier to backtrack if you still have the files.

2) How can I revert back to the normal cursor? I am not sure what files to delete and if I need to edit anything in the animation folder.

You need to edit the EQUI_Animations.xml file. Find the definition of A_DefaultCursor and replace it with the version in the default UI:
<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>window_pieces01.tga</Texture>
<Location>
<X>230</X>
<Y>40</Y>
</Location>
<Size>
<CX>22</CX>
<CY>22</CY>
</Size>
<Hotspot>
<X>4</X>
<Y>1</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>
If you want to replace it with some other cursor, there are many examples in the Mouse Pointers (http://www.eqinterface.com/sdownload/index.php?op=showfiles&catid=50) UI category.

The black border around the sides of your screen are the result of a /viewport command. If you don't want a viewport, all you have to do is type in a new viewport in game:

/viewport 0 0 1280 1024

There is no setting that makes everyone happy. Luckily, it's easy to switch between them.

This will cut off the bottom of the inventory, but the window will at least fit.

How can I change the dimensions of the window?
Try removing the lines that specify the size in your ini file.

Look for the section headed [PetInfoWindow]. In that section remove the lines of the form:

Width1280x1024=###
Height1280x1024=###

This will cause the pet window to use its default dimensions, which should show all the buttons.
