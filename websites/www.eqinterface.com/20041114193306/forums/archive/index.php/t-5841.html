For those interested.. some dev. notes ahead :

splitting the UI in two directories, one with mana and one with non mana was certainly not practical, but was mandatory to get the UI display properly. That was due to a limitation of the UI not allowing to open the spellgems window when you play a pure melee class.. hence creating a 'hole' in the UI when playing the pure melee class.

So what Idid to make it managable was make a batch file (.BAT) that copies over the files from the classic-mana directory to my classc directory.. this way I only edit the xml files in one directory.

Only the selector window xml file I believe has the specific (small) differences betwee nthe two versions.. every other file is exactly the same in both directories.. silly isn't it ?

A little tip regarding the F10 key and this UI

You can make a macro with the following command and put it on a hotbutton :

/viewport reset

Then; in the keyboard options assign the F10 key as a 'secondary' key for that macro button (whichever one you have put the macro on).

Now when you press F10 to toggle off the UI the viewport will also go fullscreen.

Unfortunately to go back you 'll have to make another hotbutton with this command

/viewport 119 0 784 608

Place it on hotbutton page 2 for example and click it to restore the smaller viewport..

Illyanafan

howdy

For the spellbook edit the
uifiles/classic-mana/EQUI_CastSpellWnd.xml file

search for this line and replace the coordinates as such :

<!-- hide outside of window -->
<X>9</X>
<Y>309</Y>

then save the file and restart EQ

(the old coordinates were 40,300, which basically put the spellbook icon outside of the window and hide it)

Try it, and it should show up under the last spell gem. It wont look great but it should work.

As for the resolution, sorry it never was my intention to emulate the old school UI. My intention was to capture the look & feel of the old UI, but use modern resolutions.

Adapting this UI to higher resolutions would take a little work, adapting it to lower resolution (i.e. completely obsolete 640x480 would need to reowrk it all).

My account is closed and the UI is discontinued.

Hopefully someone using it may 'fix' it whenever the UI files have been affected by a new patch.
