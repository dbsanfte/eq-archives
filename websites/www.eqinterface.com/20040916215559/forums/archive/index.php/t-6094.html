This is Darlok V2.4 UI using a 780x585 viewport concept. This is written for a 1024x768 screen resolution. The rest is heavily modifed suitable to both casters and melee alike (casters looks better IMHO). Modified features include (sorry if I didn’t credit anyone for their original work that I may have used/modified):
--------------
Features
--------------

Modified 780x585 viewport originally written by Talyn (oldschool series)
Sokol’s Spell cast window modified for easy spell identification
Modified Cairenn/Grimmers 3tab pet window
Made new player/group/target windows showing all vital stats & experience
Cogmatrix AA window and modified Cogmatrix Inventory
Removed most titlebars for sleeker theme
Romelio compass modified to work with Bard spellbox icon
Allissiya selector window, resized
Modified Athee Hotbutton window
Merchant window showing all 80 items.
Athee loc wizard using /feedback window modified to include PoP.
Duel chat/combat windows to ease spam
Buffbox with labels for easy id

There are many more minor additions/changes.
--------------
Installation
--------------

Make a new Folder in everquest/uifiles
Copy all files from the Default folder to new one
Unzip into new folder replacing edited files.
Find your character’s ini file named: (UI_charname_##) in main EQ directory
rename this file by adding a “BAK” to the end (for backup)
rename the UI_charname_## file you unzipped to resemble the file you backed up
cut/paste this file to your main EQ directory
fire up EQ and loadskin
