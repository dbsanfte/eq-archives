I chose not to include the .ini file this time as it seems most people want to fiddle with the layout of the windows, and the fact that getting an .ini file to retain settings for 3 different reso's was a bit of a pain. The /viewport settings should be in the readme, and from there all you have to do is line up the windows around the edges of the viewport.

I believe the v3 zip has a LunaWrapv3 directory where all the files are, whereas the v2 zip didn't have the files contained in a dir. You may want to check that there isn't a 'LunaWrapv3' folder inside the custom dir you created for it. I know this is real basic, but since it is a change from v2 to v3 it could easily slip under the radar.

Other than that, I'm not sure what is causing your problem. Try a fresh dl and install, check your folders, and post your 'UIErrors' file, if there's anything in it. Anyone else having this problem?
