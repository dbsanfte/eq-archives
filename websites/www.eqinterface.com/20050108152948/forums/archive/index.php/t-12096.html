to add the spell names in the spell gems window, you would modify the following file:

EQUI_CastSpellWnd.xml

sample code looks something like this (I borrowed the code for this example)

<Label item ="CSPW_Spell0_Name">
<ScreenID>CSPW_Spell0_Name</ScreenID>
<Font>2</Font>
<RelativePosition>true</RelativePosition>
<Location>
<X>30</X>
<Y>3</Y>
</Location>
<Size>
<CX>65</CX>
<CY>30</CY>
</Size>
<EQType>60</EQType>
<TextColor>
<R>255</R>
<G>255</G>
<B>255</B>
</TextColor>
<NoWrap>false</NoWrap>
<AlignCenter>true</AlignCenter>
<AlignRight>false</AlignRight>
</Label>

*You need a section like this for each gem, numbering from Spell0 through Spell7.

*Modify Location and size numbers to fit your windows needs

*Make your spell gem window larger (increase X value) or there won't be enough room for these labels and they just won't appear.

*Be sure to add a line like this for each label (numbered Spell0 through Spell7): <Pieces>CSPW_Spell0_Name</Pieces>

add them in this section <Screen item = "CastSpellWnd"> along with the other pieces
