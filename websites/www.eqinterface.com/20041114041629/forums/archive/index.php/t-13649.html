- Sars 4 Window Selector v1.0 (http://www.eqinterface.com/sdownload/fileinfo.php?s=&id=3264)

Ripped out and modified from the Sars interface. This easy to read Window Selector bar is a dream for simplicity. Labels appear as the window is open or as your cursor hovers over the button. There is no modification of the EQUI_Animations.xml file to install this. Just unzip the files to your custom UI directory and load it up.

Thank you Sars for making such a great Window Selector !!!
