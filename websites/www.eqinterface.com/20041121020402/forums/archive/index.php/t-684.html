"TargetNameLabel" will return the true name for the target. Use it as the screenid. I made a simple mod with this last night.

http://members.cox.net/kcheyne/targetwindow.jpg

I actually prefer to have the two target names there. If you want it to have a more clean look with just the single name, you will have to remake the HP gauge because TargetHP includes both the name and the gauge. I suggest looking at Geddine's target window, he has done the same thing except made it bigger.
