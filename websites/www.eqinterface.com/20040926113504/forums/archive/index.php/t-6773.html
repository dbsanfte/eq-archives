My advice is make the border fit within the size of the default border, that way all you have to replace is the tga. The best way to do this is have the alpha channel visable while you edit the image. This is done in photoshop by selecting the layers tab in the paths window then cliking the eye next to the alpha channel. This makes an almost unbreakable mod since when the tgas are changed, EQ does not care that it puts blank stuff in the place of a picture.
