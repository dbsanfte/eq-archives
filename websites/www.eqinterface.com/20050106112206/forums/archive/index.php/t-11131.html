item names are case sensitive.

when an item is defined:
<Label item="Player_AttackNum">
<ScreenID>Playerexp</ScreenID>
</Label>
and then later a pieces uses it:
<Screen item = "GroupWindow">
<ScreenID/>
...
<Pieces>Player_AttackNum</Pieces>
...
</Screen>

The "item" name and the "pieces" name must match EXACTLY.

In your example, the Label tag is defined with an uppercase "N" ("Player_AttackNum"), but the Pieces tag is asking for a label spelled with a lowercase "n" instead ("Player_Attacknum").
