There should be a file called Sona_Filler_Blank.tga. This is the No-Character File. Open that file and save it as Sona_Filler.tga and the DE will disappear on the next load. OR, Open Sona_Filler_Blank.tga in a Paint Prog and take a copy of YOUR Char and paste it onto the file (Resize to fit the current image. Do not resize the Filler File.) Once you have got your toon on the filler_blank, save the filer_blank as Sona_Filler.tga (not _blank) then it will import the next time you load.
