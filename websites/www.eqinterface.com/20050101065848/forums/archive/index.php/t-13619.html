you don't have to copy the default folder at all. Just simply place all the new modifications that you want in one folder inside the "Uifiles" folder in the EQ directory ((the folder holding the "Default" folder. but not IN the default folder)) and name it. Then in the game load the ui under options screen what ever you named the new ui folder (like mystuff for example) under loadskin. EQ will pull from the default folder what is not in yours and TA DA your ui should be a modified ui.
