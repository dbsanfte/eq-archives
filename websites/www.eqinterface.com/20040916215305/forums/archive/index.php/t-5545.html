Well, I got my UI to a finished state and it's ready to download.

Screenshot:
http://www.thunderwalkers.net/logan/LoganUI.jpg

I wanted a UI that would allow me to have all the info I needed displayed and still have the maximum possible viewing area. I think I've accomplished those two goals and I'm really happy with the UI.

If you have any comments or suggestions, let me know.

Caster Version
http://www.thunderwalkers.net/logan/LoganUI-caster.zip

Hybrid Version
http://www.thunderwalkers.net/logan/LoganUI-hybrid.zip

Melee Version
http://www.thunderwalkers.net/logan/LoganUI-melee.zip

Instructions:
1.) Download the zip file of the version you'd like.

2.) In your EverQuest/uifiles folder, create a folder for this UI. Name it whatever you want. ex. LoganUI

3.) Unzip the file into the folder you created in step 2.

4.) Once you are logged into EQ, type /loadskin folder_name to load the UI. ex. /loadskin LoganUI
