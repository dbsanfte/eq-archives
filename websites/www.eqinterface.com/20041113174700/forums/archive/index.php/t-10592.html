Update: Patch for the changes to the Selector Window on 10 Feb 04 (GoD release) can be found here (http://www.eqgui.com/showthread.php?s=&threadid=12496). Might want to get it today, EQI is likely to be swamped tomorrow.

Patch for the changes to the Target Window on 5 Nov 03 can be found here (http://www.eqgui.com/showthread.php?s=&threadid=10974).

http://www.eqinterface.com/sdownload/p.php?id=971

Same as before, just updated to be current with all changes up to and including 9 Oct 03 (hence the d designation, as opposed to a new "version").

Resolutions 800x600, 1024x768 and 1280x1024 are all supported.

READ THE README FILE for details on how to install this skin. Instructions for changing the class animation as well as instructions for updating your files if you are already using the original version of this skin are all included in the README file.

Spell book is *not* included.

You may replace the inventory, with any of the LoY compatible ones from t.king/remelio if you wish.

You may replace the spell cast bar, spell icons 1-5 and gem icons 1-2 with any other t.king/remelio if you wish.

Credits to t.king and Remelio for the original graphics and most of the coding. I just tore things apart and put them back together again. Credit also to Vertebrae, Sokol and Kudane for a bit of code I borrowed.

Feedback welcome.
