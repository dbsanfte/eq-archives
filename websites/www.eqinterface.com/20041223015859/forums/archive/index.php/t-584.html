Sounds to me it got thrown off the screen.

There's still a bit of a bug in the UI when it will read cords completely odd.

To replace the Target window with hte default one: Go into your default2 folder and remove the file EQUI_TargetWindow.xml

Don't nesessary delete it. Just remove it.

Then load the UI again with /load default2 ( do not use a 1 at the end ).
[or just start upEQ if you're on one machine]

After that, you should see your window again. Once done, put the EQUI_TargetWindow.xml file back.

Should be fine then. If you understand X,Y cordinates, I would suggest opening that file up and make sure that 'Location' 'X' and 'Y' in the 'Screen' tags at the end aren't too far set for your screen resolution.
