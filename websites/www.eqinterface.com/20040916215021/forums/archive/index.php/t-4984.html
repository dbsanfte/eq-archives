The resolution in the screenshot is 1280x1024, the .ini file is only set up to have the windows in the right places for that reso. Like I said, I know ~very~ little about XML, and I think that to make the windows movable I'd have to have the sizeable option in the XML file set to false. Judging from the skim over the chatwindow XML I did in approximately 15 seconds, I think doing that would mean that the chat windows would both have to be the same size. Or they could have borders, but then they wouldn't be as pretty :p.

I'll see if I can get around to trying it at some lower resolutions, but I'm not sure how well it'll turn out.
