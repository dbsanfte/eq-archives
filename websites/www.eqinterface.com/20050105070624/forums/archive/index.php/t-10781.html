It really depends on where the custom textures are located.

If the custom textures are located in the xml for that window itself, then all you need to do is:

a) copy the PlayerWindow.xml

b) copy any of the .tga files associated with the Player window into the ui folder you want to use.

If they are in a custom animations xml then they should include what changes they made. If this is the case you will need to copy EQUI_Animations from the default folder into your ui folder and make any changes that they specified in a readme or their texture file. You will also need to have the .tga files in your ui folder as well.

If you're just using the default ui, just go to options, load ui skin button, and select the folder that this custom player window is in.
