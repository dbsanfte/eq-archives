*** ForryVP ***

Link to Picture (http://jason.forryweb.net\pet_hybrid_1600x1200.jpg)

The link above is to an example of what I'm trying to do with my new interface. What is shown is the interface that a hybrid class that has a pet would use. I'm not quite done designing all aspects of the UI, but would like any input before I start coding.

One thing I'm going to try to do is make each window independent...so if you just like one window, it should be real easy to add it to your existing UI. Basically this means that each window will have its own .tga file and all graphic definitions will be located in that window's xml file. (No messing with the Animations File!!!)

As of right now, I'm only planning on doing 1600x1200 resolutions!

Please let me know what you think, or any suggestions you may have. Thanks.
