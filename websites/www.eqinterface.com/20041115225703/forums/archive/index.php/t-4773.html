Hey Everyone!
I have decided to release a copy of Garryck's INI Copier. I take NO CREDIT for this, and all credit to Garryck for making such a great utility. Basically, all you have to do is copy the INI that came with the mod you want to use into the C:/Program Files/EverQuest directory and then load up the program. Then copy the file from the left side to the UI.Character.Server file that you want to use. Click copy and there will be a backup in your EverQuest folder in case it doesn't work right and you want to go back to what you had. Have Fun and happy modding!

To be fair Garryck had the file removed from the download section when he left everquest.

If you cannot get the INI copier to work.

1) Locate the UI_CharName_##.ini file you want to duplicate.
2) Right click the file and choose copy.
3) Paste the file in the same folder.
4) Locate UI_CharNameToBeModified_##.ini
5) Right Click to be modified and choose rename.
6) Add a .old to the end.
7) Right Click again and choose rename again.
8) Highlight the whole name EXCEPT the .old.
9) Right Click and choose copy.
10) Locate Copy_of_ UI_CharName_##.ini
11) Right Click/Rename
12) Right Click Paste and it should now show the original UI_CharNameToBeModified_##.ini as the name.
13) Click somewhere else.
14) All set.
