I've updated the UI Timer (http://www.horizonsoutpost.com/eq/timer.asp) XML generator to make all configurations compatible with the 5/26 patch. This affects the help window only. If you are using the help window for the timer, please generate a new XML file.

Slider Timer:
http://www.horizonsoutpost.com/eq/customize.asp

Dial/Digital Timer:
http://www.horizonsoutpost.com/eq/customize2.asp

Patch Summary
Here is a quick summary of timer versions needed for particular windows. You can find the version number that you are currently using in the comments near the top of the XML file. If you don't have at least the required version listed below, or if you have version 3.6 (which is no longer valid), you should generate a new XML file. Help window - version 3.15 or higher
Selector window - version 3.14 or higher
Player window - version 3.13 or higher
Target window - version 3.12 or higher
Actions window - version 3.11 or higher
Pet window - version 3.11 or higher
Group window - version 3.10 or higher
Bug window - version 3.8 or higher
All other windows - version 3.2 or higher
