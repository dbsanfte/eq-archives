- Open that UI's zipfile
- Locate the XML file for the cursor
- Locate the TGA file(s) for the cursor
- Extract the Selected Files to your custom folder

That is the correct TGA file. Copy that into your current UI folder, then open your UI's EQUI_Animations.xml file in Notepad or some other text editor. If your UI doesn't have one, then copy the file from your default folder to your UI's folder

Add the following to the top of the EQUI_Animations.xml page:

<TextureInfo item = "AYAepic.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>

Then search for (example is from default file):

<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>window_pieces01.tga</Texture>
<Location>
<X>230</X>
<Y>40</Y>
</Location>
<Size>
<CX>22</CX>
<CY>22</CY>
</Size>
<Hotspot>
<X>4</X>
<Y>1</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>

Replace that text with the following:

<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>AYAepic.tga</Texture>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>32</CX>
<CY>45</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>200</Duration>
</Frames>
<Frames>
<Texture>AYAepic.tga</Texture>
<Location>
<X>32</X>
<Y>0</Y>
</Location>
<Size>
<CX>32</CX>
<CY>45</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>200</Duration>
</Frames>
<Frames>
<Texture>AYAepic.tga</Texture>
<Location>
<X>64</X>
<Y>0</Y>
</Location>
<Size>
<CX>32</CX>
<CY>45</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>200</Duration>
</Frames>
<Frames>
<Texture>AYAepic.tga</Texture>
<Location>
<X>96</X>
<Y>0</Y>
</Location>
<Size>
<CX>32</CX>
<CY>45</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>200</Duration>
</Frames>
</Ui2DAnimation>

Save the EQUI_Animations.xml file to your UI's folder and you should have a new cursor the next time you load the UI :)
