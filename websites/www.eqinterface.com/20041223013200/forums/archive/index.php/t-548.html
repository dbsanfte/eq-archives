The main chat window (or the title of any of the other chat windows) cannot be changed at this time. These values are set by the EQ program itself.

There is an option in the popup menu for changing the title of the chat window. Every time I've used it, I crashed out to the Desktop. If these are changeable, then it follows that the titles would be stored in the character INI files where the window positions are stored.

[EDIT] I replaced a few of my files with ones from default - mainly the EditTextWin and ChatWindow and added the reference to EditTextWin to the EQUI file. Now the popup to change the window titles works.
