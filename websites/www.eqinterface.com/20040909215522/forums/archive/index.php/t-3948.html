Tyrnt, before I forget your name, here's the fix to your problem with the chat boxes;

1) Unlock the left box (with the exp bars, called the Main box herein) and move it elsewhere
2) Unlock and move the chat box the above covered. Make sure you set the Normal and Fade transparency both to 100% (temporarily)
3) Find the second chat box. If none exists (none did for me on initial install) then make one. Move that one as well, also making transparencies 100%
4) Move the main box (exp box) back into place and lock it
5) Move the first chat box into it's spot, enlarging it just enough so the chat bar on the bottom, scroll bar on the side, and the lines on the edge are outside the window on the main box. Lock it, make the transparencies 0%. It should be, for all intents and purposes, invisible
6) Repeat #5 with the second chat box. Now, set what messages go where. I suggest the main chat to be important messages, top left others spells/attacks, damage shields, etc and bottom left for raid/group

BTW, I LOVE this UI! Not only did it hold up to the patch, but it puts all the chats where I want them, in the importance I want them, as well as everything else. I hope you remake the Undead UI to something closer to this. I'm recommending this like mad to people on the Solusek Ro server
