Question(s): I'm really just wondering if we will see any of the following features return to the new upload script:

- The ability to edit the files and descriptions which you have uploaded when a new version or more information becomes necessary to update.

- The ability to name your mod, instead of displaying the generic file name (eq_ui_inventory_mod.zip); it's not very pretty. :P

- The number of files available per section, illustrated such as "Inventory [6]."

- The ability to list files by popularity, author, etc...


Suggestion(s): The following are suggested features which did not appear in the old script, but might be beneficial:

- Adding a "Popular Mods" section which automatically lists 25-50 of the most popular mods. An alternative would be to display the 5 or 10 most popular mods in their respective sections.

- The ability to add a thumbnail to your description, in addition to the clickable link to view your image, making it much more visually appealing. Possibly restricting the thumbnail file size and/or dimensions to cut down on bandwidth.

- Font Bug: The current script defaults all descriptions to the "Times New Roman" font, which doesn't go with the rest of the site's Verdana appearance.
