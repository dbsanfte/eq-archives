To make a window resizable, you'll need to important things.

First is a <style_sizable>true</style_sizable> in your Screen definition.

Second is a tag for <autostretch> which has to go in each of your elements that you want to auto-resize when you change the window size. I believe you also need to utilize these elements as well:

<TopAnchorToTop>
<BottomAnchorToTop>
<LeftAnchorToLeft>
<RightAnchorToLeft>
<TopAnchorOffset>
<BottomAnchorOffset>
<LeftAnchorOffset>
<RightAnchorOffset>

Basically, read the file called 'AboutSIDL.doc' in your default UI folder. Under the section entitled Screen Pieces. It briefly explains what those elements are used for.
