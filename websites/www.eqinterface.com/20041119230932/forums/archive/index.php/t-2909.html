As far as font sizes go, you have a range of 0 to 5, 0 being the smallest of course. However, the only things effected by font size in that UI file are <Label>'s and <Button>'s, and only them assuming that there is text which goes along with either. All of the letters and words which might appear on the individual Inventory buttons are actually done in the graphic itself, therefore you'd have to edit the graphic file if you wanted to change the text there. But, if you are wanting to change the text such as the stats, or on the buttons such as the 'Skills' button, then what you're trying is correct. Try going to a 0 (this would be quite small for most people) and see if that helps.
