Ripped out and modified from the Sars interface. This dynamic group bar grows and shrinks as your group grows and shrinks. There is no modification of the EQUI_Animations.xml file to install this. Just unzip the files to your custom UI directory and load it up.

Release Notes:

Version 2.0:

- Increased the height of the pet health indicator
- Cleaned up redundant animations code and combined it into one area
- Removed obsolete ripped animations code
- Changed variable declarations to remove future possibilities in code clashing
- Removed SarsBar02.tga file, was no longer needed with obsolete code removal
- Renamed sars_windowpieces02.tga to just bar_pieces02.tga
- Compatible with parent UI mod Sars 4 (no code clashing)


Thank you Sars for making such a great group bar !!!
