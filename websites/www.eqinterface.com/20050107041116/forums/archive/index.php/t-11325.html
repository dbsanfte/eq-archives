Hmm the warnings tell me that it cant find the interface files in the location you told it to look "player" in your UIFiles folder.

When you type /loadskin in game and the window pops up does it list your interfaces in there? If not then you must have two everquest folders on your computer due to a faulty install script on one of the expansions to the game. You'll need to hunt for the other one and move your interfaces to the uifiles folder in that one.

If you do see your interfaces listed in the /loadskin window then you must have a folder inside a folder. Make sure all the XML and TGA files your custom skin has are inside the custom folder and not inside a folder inside another folder. If that makes any sense to you.
