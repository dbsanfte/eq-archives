in the file EQGUI.xml (in your new mod folder)

find this line

<Include>EQUI_CastSpellWnd.xml</Include>

make it look like this

<!-- <Include>EQUI_CastSpellWnd.xml</Include> -->

that will "disbable" the default cast window

/bonk Kudane ... Modifying the EQUI will break when VI patches again, most likely.

Instead, since you're messing with the CastingWindow. Go into EQUI_CastSpellWnd.xml and locate the <Screen> and </Screen> tags. Comment it all out.

So it's more like:

<!--
<Screen>
........
</Screen>
-->
</XML>

This will disable it, and it will also prevent your mod from breaking if and when VI patches the EQUI.xml file, again.
