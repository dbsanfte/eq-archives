/// How to Edit Right Clickable slots ///

you can edit Hotbuttons.

first open inventry.xml and look it.

and search boots (Feet) slot. it said

<!-- IS_FEET 19 -->

right? it means, Feet slot is <EQType>19</EQType>

then Search Range slot. it is <EQType>11</EQType>

so, open Hotbutton.xml and Search <EQType>11</EQType>
(it is a range slot's EQType) and Edit it to <EQType>19</EQType>.

now you can edit Friendwindow,Hotbuttons,Container's Inventry Slots.

if anyone want to edit them, please read this.

aeluin
12-08-2002, 09:01 PM
If you copy any files you haven't modified from default to custom directory, you're just setting yourself up to have your UI break with EVERY patch. If it's having problems, there's something else going on- it *always* looks in default if a file is referenced in EQUI.xml but is not found in your custom.

The warnings in your UIErrors file are just warnings. They're harmless. You only need to worry about the contents of that file in three situations: 1, your UI is not loading correctly, 2 you see a line starting with "Error:", or 3 the warnings just keep looping infinitely because a needed file does not exist in either your default or your custom directory.
