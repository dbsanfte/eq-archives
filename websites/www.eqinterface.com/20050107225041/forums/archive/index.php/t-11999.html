Open up your EQUI_CastSpellWnd.xml file and search for CSPW_SpellBook. Inside that element you will find the description for your button graphics. It should look something like this:<ButtonDrawTemplate>
<Normal>A_ClosedBook</Normal>
<Pressed>A_OpenBook</Pressed>
</ButtonDrawTemplate>
Find the name of the Normal button image.
Search your xml files for that string to find the Ui2DAnimation definition for the image.
Make sure that definition is in a file that you are still including in your UI before your EQUI_CastSpellWnd.xml file.
Search for the TextureInfo definition corresponding to the Texture filename tag for that definition.
Make sure that definition is still being included in your UI.
Make sure the correct tga file is still in your UI directory.

If all these things seem correct, please post again.
