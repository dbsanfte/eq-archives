Adds a display to the compass to show the Norrath time of day.

You have to set the slider when you log in. Then it keeps accurate time until you log out (or switch characters or load a new UI). Never loses a second, even if you go to Luclin or you're a slow zoner.

This is a beta version, I expect to make improvements soon.

I'm no artist so it's not the prettiest, but it does work. If someone could make a new daynight.bmp file, at either of the shown widths, I'd appreciate it. Maybe the numbers could be integrated with the picture instead of hanging below. The width should match one of the scales (15 pixels per hour or 18 pixels per hour), but it can be any height.

The blue line that's supposed to point to the entrance to an instanced zone doesn't appear on the compass with this mod... does anyone know why? If I put this mod in any other window it wouldn't be an issue, but I'm still curious.

The main technology here is placing an animation in a scrollable subwindow. I believe this technology could be used to make a 'resettable' timer, where you would 'reset' the timer by adjusting the slider.
