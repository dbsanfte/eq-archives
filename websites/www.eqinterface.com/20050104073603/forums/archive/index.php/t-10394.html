Ok i had this problem too.
The Error is because the Base UI you are using has a custom EQUI_Animations.xml file and when you replace it wiht the one from the AYA pointer you erase all the custom code.
This breaks the UI

And here is the fix

1~open the EQUI_Animations.xml file for your TKing UI using notepad

2~Use serch function to find:

<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>window_pieces01.tga</Texture>
<Location>
<X>230</X>
<Y>40</Y>
</Location>
<Size>
<CX>22</CX>
<CY>22</CY>
</Size>
<Hotspot>
<X>4</X>
<Y>1</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>

And replace it with this :

<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>AYAepic.tga</Texture>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>37</CX>
<CY>45</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>200</Duration>
</Frames>
<Frames>
<Texture>AYAepic.tga</Texture>
<Location>
<X>37</X>
<Y>0</Y>
</Location>
<Size>
<CX>37</CX>
<CY>45</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>200</Duration>
</Frames>
<Frames>
<Texture>AYAepic.tga</Texture>
<Location>
<X>74</X>
<Y>0</Y>
</Location>
<Size>
<CX>37</CX>
<CY>45</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>200</Duration>
</Frames>
<Frames>
<Texture>AYAepic.tga</Texture>
<Location>
<X>111</X>
<Y>0</Y>
</Location>
<Size>
<CX>37</CX>
<CY>45</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>200</Duration>
</Frames>
</Ui2DAnimation>

Next find :
<TextureInfo item = "CS_Buttons.bmp">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>

And add Just bofore it:
<TextureInfo item = "AYAepic.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>


That should do it jsut close and save the changes.
Load your UI and check if it works.

Post if this does or does not work thanks and GL.:)


Oh and by the way NEVER EVER FOR THE LOVE OF GOD CHANGE THE DEFAULT UI FILES if you manage to mess those up and they do not get automatically patched you have to do a re-install of EQ ;)
