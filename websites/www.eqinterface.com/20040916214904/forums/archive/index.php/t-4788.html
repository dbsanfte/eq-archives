Hey guys, I'm wondering what you think of this set of windows.



I worked on that for some time, trying to create a simple, generic set of windows that is very usable by any class (and indeed helpful to any class).

http://www.eqinterface.com/sdownload/preview.php?id=711

I am definitely open for tweaks, etc, for version 4.x and ultimately version 5.0 of my window set. Just wondering what you think...

Briefly, let me explain my rationale behind these three setups.

First, all three of these windows are displayed at once-- these are not 3 different "versions" of one window. The above windows are the player, group, and target window collectively. I wanted to create a simple set of these three that can be moved around to be placed exactly where it best fits each player.

I added the large hit points (inspired by AYA's sets) so no matter what's happening, you can get a FAST view of your current state of health (and also get an indicator when certain HP buffs wear off). I inserted this large HP view in both the target window AND the group window so that ANY player would have quick access to it... clerics I imagine focus on the group window, but need to know when to hit that hot key saying they're being wacked on. :P Warriors don't really care about their groupies (lol!) but need to know their own HPs when they're targeting their mob....

Anyway, let me know what you think.
