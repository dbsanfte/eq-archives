What you need to do, then, is take the files out of the directory the UI does get extracted to and cut & paste them into the Monk directory. Then you delete the now empty directory the UI was originally extracted to and load the new UI using the /load Monk command.

If you need an explanation on how to cut & paste files from one directory to another, just let me know. Doing this won't hurt the UI, trust me. I've been extracting, cut & pasting, and renaming directories while playing with these UIs for a bit now. :D
