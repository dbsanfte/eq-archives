Look at EQUI_Inventory.xml to see what the XML For an InvSlot looks like. You'll be needing to add these to EQUI_HotButtonWnd.xml.

It's not really hard. You just:
- copy the InvSlot from EQUI_Inventory into EQUI_HotButtonWnd
- change the name of what you copy and pasted
- change the location to where you want it to be relative to the hot button box screen object
- add a new pieces to the hot button box screen object for the newly renamed InvSlot object

I'm sure someone who has done this can send you their EQUI_HotButtonWnd.xml, but that's not me.
