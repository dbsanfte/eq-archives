My problem is : How do I download a UI skin into a single folder? I've went to the FAQ's page and it shows that having a subfolder is 'wrong'. Put the downloaded zip file on your desktop. Unzip the whole thing to your desktop.

Open up the UIfiles folder in your EQ directory. Right click in the blank space next to the default folder, select "new" from the list, then select "folder".

Name the new folder whatever you want.

Go back to the desktop and open up the unzipped folder for the new UI. Open any subfolders until you get to the actual files. Select all the files (an easy way to do this is go to "edit" at the top and click "select all" (ctrl+A), or you can click on the first file, hold down shift, and click on the last file.) remember, select just files. No folders.

Now click on edit, and select "copy". (or just hit ctrl+C)

go back to that new folder you just made next to default inside the UIfiles folder in your EQ directory. Open it up.

Click edit, and select "paste". (or hit ctrl+V)

You should now have a new folder next to default with only the files inside.
