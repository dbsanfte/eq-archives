A custom EQUI_SelectorWnd.xml; and/or

If you have customized tga (graphic) files, make sure that you have the actual graphics that have been added (compare to the corresponding file in the default folder).

If possible, your best option would be to see if the original author of your UI skin has released an updated version/patch. Otherwise, you are either going to have to:

a) give up the customization of the .tga file and just run with the default, which may impact how other things "look" in-game; or

b) manually edit (or have someone else edit for you) your custom .tga file to include the additional graphics that it is currently missing.
