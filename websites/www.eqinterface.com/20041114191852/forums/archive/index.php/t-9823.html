More detailed information and download links available at my UI site (http://members.cox.net/katy-w/ui.html).
Download from here by clicking the link to my downloads on the left under my name.

A brief description:
This is a no frills, xml only mod, fully compatible with any graphics only mod. Does not use EQUI.xml or EQUI_Animations.xml at all!

Runs fine in most resolutions. I haven't tested with extremely low ones.

A number of windows have had their title bars disabled. Hotkey bank includes all bag inventory slots plus the main and secondary hand slots by default, but a no inventory option is included as well. Two tracking options are included, ranger options or non-ranger options. Uses a classic style inventory. And finally, there are 12 different player windows to choose from.

Several of the custom windows are also standalone mods in the downloads section. Click the link to my downloads on the left under my name to see which ones are available.

In addition, I've revived the old Vert 1.1 graphics and made it default animations compatible. The package, and update information on it, is available both here and at my website.

FAQ: Can you use this with TKing spellbooks? Yes, install this mod first, then install a TKing spellbook. You may need to use a modified Animations file.

Please don't hesitate to contact me if there is a problem with the mod or if you have a request for something specific to do with my mod.
