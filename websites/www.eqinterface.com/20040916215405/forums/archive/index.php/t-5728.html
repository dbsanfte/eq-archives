Nothing fancy here, it's finished for my purposes but if anyone shows any interest I'll slim the rest of the windows and call it a full set.

All window pieces except the compass and spell gems / icons use default textures. Drop the files in a new folder and load it up, no song and dance required. Works with no errors.

Most of the changes are apparent in the screenshot. Image taken at 1280x1024 but it works just fine at 1024x768 and is livable at 800x600.

- TKing/Remelio's Loc compass

- TKing/Remelio's Arcane spellbook

- TKing/Remelio's spell icons / gems

- An inventory window that I *cannot* remember the designer of since I've had it so long. It's not my work, let me know if you recognize it and I'll happily provide credit.

- Casting gauge added under Target HP bar.

- Mini Inventory and my personal clickies added to the Hotbutton window. Since I doubt this is a mod with mass-market appeal I'll offer to help with setting up your preferred clickies if you really want to use the mod.

Note that the Chat and Buff windows cannot be dragged, only resized. Just do a /loadskin cogmog 1 and save yourself a hassle.
