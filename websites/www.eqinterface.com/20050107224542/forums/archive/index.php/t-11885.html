This is a fairly common error which simply states that the file is either completely missing or corrupted. Most likely it's just missing.

A search on Microsoft.com revealed this link: Microsoft Download Site (http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=6F94D31A-D1E0-4658-A566-93AF0D8D4A1E)

and I hope it displays properly. Alternately, you should be able to find the file on your Windows install CD-ROM, depending on your version.

If you run the installer from Microsoft, all "should" be well when you finish. If you download the file from an alternate site and/or the CD, place MSCOMCTL.OCX into your Windows/system32 directory and then click Start->Run and type: REGSVR32 MSCOMCTL.OCX

This will cause your OS to recognize and properly register the file. I would post my own MSCOMCTL.OCX file for you, but it's too large (a little over 1mb, 504kb zipped)

RE: Where to place Remmy.exe... Remmy should be placed inside your customer UI folder and run from there.

If you have trouble locating the file, I will send you mine if I have an email address capable of accepting the attachment.

Good Luck!
