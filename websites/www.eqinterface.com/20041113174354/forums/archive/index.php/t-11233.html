You got 2 options:

1) Delete the Loot Window xml file, so you do not have to worry about having the Loot Window size conflict with your INI file.

2) Read the ReadMe file and use the enclosed INI file ~or~ edit your current INI file and remove the information from there that reflects [LootWnd] information. It will rewrite it correctly once you are in-game to the Loot Window xml it has loaded in memory (Default or Modified).
