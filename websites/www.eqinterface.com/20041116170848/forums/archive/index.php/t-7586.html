Well, I figured out a good color scheme and size for a vertical bar after a few weeks of testing different sizes and graphics for frame-rate.

Turns out, about the best possible accuracy, with the highest frame-rate in a vertical bar is (50) gauges in a single vertical bar.

I used some coloring to make it look a little more "vertical" for those times when a gauge is about half full on a single line.

In testing the only time my gauge didnt match the horizontal, my gauge actually appeared to be about 1%less, which IMHO is better than being more full :)
