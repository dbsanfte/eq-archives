Well, you can do 1 of 2 things:

1) Unzip the file into the Everquest\UIfiles directory (this is the parent directory). This would then create the necessary folder that the zip file comes with in the proper location.

or...

2) Unzip the file in the Pantha folder to your own separate directory, then move them all into Everquest\UIfiles\Pantha yourself.
