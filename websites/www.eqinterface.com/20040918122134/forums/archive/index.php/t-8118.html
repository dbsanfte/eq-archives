Hello!

I've done a couple of modifications to windows for Vert, and I thought some other folks might get some use out of them.

Firstly, I modded the Select window to be 2 horizontal rows of 7 buttons. This mod should be fairly universal, but I was working with a Vert copy of the file, so if you choose to use it elsewhere, beware of any "gotchas" that creep up.

Use the EQUI_SelectorWnd.xml file.

Next, I modded the Cast Spell window to make it a bit wider (allowing for spells with really long names to fit), and added the spell casting guage to the window. The spell casting guage is also a button for opening the spell book.

Use the EQUI_CastingWindow.xml and EQUI_CastSpellWnd.xml files.
