Burner Of Souls
12-18-2003, 04:12 PM
You need to copy the EQUI_CombatAbilityWnd.xml file from the default UI directory into your custom UI directory.

Then make a copy of window_pieces_06.tga and window_pieces_07.tga from the default UI and rename them to something like original_windows_pieces_06.tga etc.

Then modify your EQUI_CombatAbilityWnd.xml file by replacing all window_pieces_06.tga to original_window_pieces_06.tga and window_pieces_07.tga to original_window_pieces_07.tga.

Burner Of Souls

This was just intended as a temporay quick fix until the mod makers fix their version of their Mod.


You need to edit the EQUI_Animations.XML

(Modified from the 26 Aug 03 patch post)

Find:

<TextureInfo item = "window_pieces06.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>

ADD:

<TextureInfo item = "original_window_pieces06.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>
<TextureInfo item = "original_window_pieces07.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>

Originally posted by zugzugmika
Yes thank you very much. Now if I can figure out how to make that window bigger so I can see the text.

Down towards the bottom of the EQUI_CombatAbilityWnd.xml file you'll find

<Style_Sizable>false</Style_Sizable>

change to:

<Style_Sizable>true</Style_Sizable>

It doesn't change the button sizes, but it allows for dragging the window to the desired shape. Oh, and the fix with the windows pieces worked great for me. Thanks for the help!
