Info
Designed and tested in 1024x768, but useable in other resolutions.

The only differences between UIv2 and UIv3 are in the graphics. The XML files for the two are otherwise the same (with one or two exceptions to make allowances for things like gauge colors).

UIv2 does have some optional folders: one to change the base color of the UI from brown to blue, and one to get rid of the circle pattern seen on some of the window backgrounds.

All other pertinent info is in the readme files in the zips.
