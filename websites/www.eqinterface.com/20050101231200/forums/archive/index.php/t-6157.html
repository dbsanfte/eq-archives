You have to replace the UI_Bruklin_34.ini not Bruklin_32.ini..

Now you'll have to replace the UI_Name_##.ini and delete the Name_##.ini...

Edit: Spelled your name wrong :p

And more explaination:

The Bruklin_34.ini saves client options like screen resolution, hotkeys, socials, etc...

The UI_Bruklin_34.ini: The one you want to replace.. Saves the UI Options.. Like.. Which UI Bruklin is using, window locations, window fade levels, etc...
