Actually this is VERY easy to change and customize yourself. Let's see if I can explain it well enough for those you who are understandably a little weary about fooling around with your files :) First Right Click "Save Target As" and save it on your computer. Then right click it again and Select Open With and open it with a program like Notepad or Wordpad. Now open up the XML and scroll down until you run across this:

<!--<Text>PRIMARY SLOT</Text>-->
<InvSlot item="InvSlot30">
<ScreenID>InvSlot30</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>90</X>
<Y>20</Y>
</Location>
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
<Background>A_RecessedBox</Background>
<EQType>13</EQType>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
</InvSlot>

This is for the top little inventory Box. The ONLY thing you need to change is on this line

<EQType>13</EQType>

Just change the number 13 to whichever number of slot you want it to be as is listed on the first post. Let's say you want that box to be for your chest pieces you simply replace 13 with 17!


<!--<Text>INVENTORY SLOT PRIMAL 22-29</Text>-->
<InvSlot item="InvSlot31">
<ScreenID>InvSlot31</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>90</X>
<Y>52</Y>
</Location>
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
<Background>A_RecessedBox</Background>
<EQType>25</EQType>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
</InvSlot>

Here is the second one. Same thing as above. Don't let the Text "INVENTORY SLOT PRIMAL 22-29" Throw you off. Just replace 25 in the <EQType>25</EQType> with whichever number you want.

<!--<Text>EQUIP SLOTS 1-4 using 1-12 & 15-21</Text>-->
<InvSlot item="InvSlot32">
<ScreenID>InvSlot32</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>90</X>
<Y>91</Y>
</Location>
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
<Background>A_RecessedBox</Background>
<EQType>5</EQType>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
</InvSlot>

You've find this listed next and InvSlot33, InvSlot34, InvSlot35, are all listed together. Just do the same thing as above and change the number in the <EQType> </EQType> with whatever you want. Then save the file in its own folder in the character mod you're using, use /loadskin Foldername and you're set! Great mod! I love it by the way :)
