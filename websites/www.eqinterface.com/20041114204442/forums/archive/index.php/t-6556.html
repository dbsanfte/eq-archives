2/10 - Added variations below
----------------------------------------------------------

To change the animation:

find this:

<!-- CHANGE This (A_ClassAnim04) to your class -->
<StaticAnimation item="HTW_ClassAnim">
<RelativePosition>true</RelativePosition>
<Location>
<X>21</X>
<Y>10</Y>
</Location>
<Size>
<CX>60</CX>
<CY>124</CY>
</Size>
<Animation>A_ClassAnim04</Animation>
</StaticAnimation>


Change this line:

<Animation>A_ClassAnim04</Animation>

To whatever class you want displayed. The number in the above line (04) is all that you need to edit.
