**INCREMENTAL UPDATE .2 **

UPDATE:
The .zip file now contains several folders with color names. Those folders contain a replacement window_back.tga that is a partial transparent tileable color slate. By copying the window_back.tga from the color folder into the glass folder and overwriting the existing file, all transparent windows that use the WDT_HUD definition will change to that background.

The WDT_RoundedNoTitle borders now have the "glass" look to them, not much of a noticeable difference to that little 3pixel area, but enough ;)

Removed the Tracking Window XML. I will be writing a new version later, based on ranger input I have received.

Changed all modified windows to have a border, not be transparent and use the WDT_HUD definition. Any window I have modified will take on the appearance of the window_back.tga file now.

Moved all custom textures/animations/definitions to the file EQUI_Glass.XML and added it as an include in EQUI.xml. This follows the Statix idea of using ../default/EQUI.xml
