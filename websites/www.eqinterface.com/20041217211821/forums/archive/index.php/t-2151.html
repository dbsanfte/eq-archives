ok, I'll try...

So you found the declaration of the Player window, which is fine.
The declaration starts with the tags
<Screen item="PlayerWindow"> and ends with </Screen>
Sos basically you start defining an item of the type "Screen" that has an identifier that is "PlayerWindow".
Notice the declarations at the end of the screen definition:...
<Pieces>Player_HP</Pieces>
<Pieces>Player_HPPercentage</Pieces>
<Pieces>Player_HPPercHold</Pieces>
<Pieces>Player_Mana</Pieces>
...
<Pieces>Player_ManaPercHold</Pieces>
<Pieces>AltAdvExpPercentage</Pieces>
<Pieces>AltAdvExpLabel</Pieces>
<Pieces>AltAdvExpLabel2</Pieces>
<Pieces>ExpPercentage</Pieces>
<Pieces>ExpLabel</Pieces>
<Pieces>ExpLabel2</Pieces>
... This is where the items, that are going to be displayed on the screen are added to the window.
To make it possible to call a piece it must be defined before it's called.
For example the player HP gauge is called in the Player window by the line<Pieces>Player_HP</Pieces>
To make this possible, the gauge was formerly in your file defined by<Gauge item ="Player_HP">
...
</Gauge>
Somewhere between the Gauge begin and end tags should be a section that reads<Location>
<CX>some_x_value</CX>
<CY>some_y_value</CY>
</Location>
"some_x_value" gives the horizontal position of the upper left corner of the gauge while "some_y_value" gives the vertical position of the upper left corner.
In your case you only want to move pieces around vertically, so all you should do is play around with the value between the Y-Location tags.
E.g.: Your Player HP gauge should look something like this:<Gauge item ="Player_HP">
...
<Location>
<CX>3</CX>
<CY>3</CY>
</Location>
...
</Gauge>
The values might be slightly different in your XML file.
Now you want to move that HP gauge waaay down, so you might want to try to change it, so it looks like this:<Gauge item ="Player_HP">
...
<Location>
<CX>3</CX>
<CY>200</CY>
</Location>
...
</Gauge>
You'll see that the player HP gauge has moved quite a bit down but didn't move left or right.
Now go ahead and change the Y-loacation-values of all your displayed items till you got them arranged to your gusto.
And when you're done come back here and lemme know if it worked...;)

I hope this was understandable...
