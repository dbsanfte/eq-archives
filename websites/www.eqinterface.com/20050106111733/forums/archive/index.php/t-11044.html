I'd like to point out that the fonts feature can break many interfaces, if the user changes fonts.

For instance, you might leave just enough room for three digits of stamina. If a wider font is used there would only be room for two digits, so a stamina of 280 would instead display as 28.

And if a very tall font is used there might not be enough space to display even one line of it, so the field would just display as blank.

The only solution in general is to leave a little wiggle room. A little blank space around each and every word and number, and make every field and combo box and button a little too large.

My interface is 'minimist' and tries to save screen space so there isn't much wiggle room -- many fonts just don't fit. I've decided I like Arial font anyway, but it could have been a problem.
