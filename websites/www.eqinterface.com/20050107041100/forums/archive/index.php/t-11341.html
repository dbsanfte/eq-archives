If you have your skins loaded from any folder other than the default skins folder you will need to have a copy of the target indicator files in the folder you are using.

Copy the following two files from your default folder:
TargetIndicator.ini
TargetIndicator.tga

and put them in your custom skin folder that you are using. If you have already done this, then I have no clue as to why you can't see the targetting circle.

I thought SOE said they were going to fix this copy problem in a patch but I have not heard anything else about it since.
