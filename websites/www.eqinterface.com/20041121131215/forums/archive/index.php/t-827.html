This mod shrinks your spell gems by 25%, nice and unobtrusive.

(Was previously posted in the "resizing spell gems" thread, this one has an updated equi_animations.xml so that you won't crash in the bazaar)

Hmm, they changed the file size for attachments.

Well, I'll submit the new .zip to eqinterface.com

Meanwhile, all you have to do is download the current .zip from eqinterface.com and then modify equi_animations.xml and add this to the end of the file (right before the closing </xml> tag):

<Ui2DAnimation item = "A_DividerVertical">
<Cycle>true</Cycle>
<Frames>
<Texture>window_pieces03.tga</Texture>
<Location>
<X>0</X>
<Y>96</Y>
</Location>
<Size>
<CX>100</CX>
<CY>24</CY>
</Size>
<Hotspot>
<X>0</X>
<Y>0</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>
