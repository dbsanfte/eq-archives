The reason the chat windows in the second one look the way they do is because the author of that UI have made changes to the window_pieces01.tga file and possibly the EQUI_Animations.xml file as well. The only way you can have your chat windows look like that is to copy that (those two) file(s) over as well. However doing that will impact how the rest of your other windows look as well, if it doesn't outright conflict and make the whole thing crash.
