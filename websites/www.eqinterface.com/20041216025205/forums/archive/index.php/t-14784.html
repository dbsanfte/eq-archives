As long as they are in your ui folder. However, this is assuming that the ui creator used the same naming conventions of the default UI. If those files are in your ui folder, you can safely delete them and the default ones will load. If there are no such tga names in your ui folder, you will have to look in the EQUI_Animations.xml for the files that contain those icons. Never delete anything out of the default folder, unless you know exactly what you're doing.

Ok, open your custom UI folder and delete the following files:

gemicons01.tga
gemicons02.tga
gemicons03.tga
spells01.tga
spells02.tga
spells03.tga
spells04.tga
spells05.tga
spells06.tga
spells07.tga

Don't worry if your custom UI folder doesn't contain some of these files, just delete the ones it does contain.

Now when you load EQ, the UI will look for the missing files in the default folder.

As for finding you a UI with that specific information, you're better off picking one yourself. But... I'll plug my own offering...
