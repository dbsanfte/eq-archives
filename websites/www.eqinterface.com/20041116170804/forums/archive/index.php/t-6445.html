He didn't make it, he just updated it... but I agree it's awesome!

As for installing it, it's not ready to be installed yet... it needs the dye buttons and whatnot added to it since the 2/24 LoY Patch.

But to answer you question, you open your EQ folder (like where your screenshots are stored), look for a folder named "uifiles" and then open that. In there you will see a folder named "default" which you dont wanna touch. So in the folder that you're in (which is the "uifiles" folder) you make another folder and name it whatever you want (keep it one word and all lowercase). This folder you make is where you will be putting your modded UI Files. So say you named it "myui" you would put the inventory files from this thread in "myui" along with any other modified UI files. Then when you enter the game you can type "/load myui" and it will load it up.

That's it... enjoy! :)
