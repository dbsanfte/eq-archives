I looked at how the destroy button on the inventory window was set up, copied it, gave it a new button name and added the <Pieces> value to the loot window. Nope. Not possible. Buttons functionality is hard coded into place. You can put the button somewhere else. You can make it say Destroy. But you can't hook up the action "Destroy what is on the cursor" to the button. That action is hooked to the IW_DestroyButton of the InventoryWindow only.

Sorry.
