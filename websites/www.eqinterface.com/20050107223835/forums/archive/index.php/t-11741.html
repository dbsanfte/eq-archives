Actually solved it before I saw your reply hehe, friend in guild knew the answer.

Turned on "Style Size" option in EQUI_ConfirmDialogue.xml file (copied from Default since Vert didn't have it). Resizing with the mouse revealed my suspicion was correct, window was the wrong size, the yes and no buttons were further down =)

Oddly enough, resizing it by the numbers in the xml file didn't affect the size of the window at all...
