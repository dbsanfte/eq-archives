Either delete the target window from the custom UI to force default loading, or you can edit the custom target window to include the information about TargetBox.TGA.

Open the default EQIU_TargetWindow.XML and search for TargetBox.TGA - copy that section from default into your custom target window. (don't forget to include the name of the section in your "Pieces" section.)
