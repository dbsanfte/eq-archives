I wanted to make a UI that showed as much as possible on the screen, was as full featured as possible, stayed true to the default UI, and took up the least amount of visible space. It is not made specifically for any class, but the group window is very handy for healers. This is what I came up with:

http://goliath.frostburg.edu/rhenderson0/downloads/grampsmix.jpg

- The player window is a slightly resized version of TheWorks3 by blarz.

- The target window is an unmodified version of Tosevite UI - Target by Iare Tosevite.

- The group window is a modified version of nodent's Group-color-change mod (took out stuff that was redundant with the player window and resized it). AWESOME for getting your attention when someone's health is getting low.

-The mini inventory window was done by me, Gramp.

Here's a zip to try it out. If there is enough interest, I will request it be 'officially' released.

Gramp's Mix & Match (http://goliath.frostburg.edu/rhenderson0/downloads/gramps_mix&match.zip)
