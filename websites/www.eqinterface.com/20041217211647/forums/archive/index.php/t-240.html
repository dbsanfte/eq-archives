07-29-2002, 07:40 PM
Ok, installed this on 3 machines...here is the trick

You MUST follow these steps EXACTLY!!!

1. Copy your default folder and rename it Quartz
2. copy all the objects from the zip file into your new quartz folder.. over-writing ALL when prompted (It MUST be named quartz

3. in your main eq directory find UI_<charactername>_<server number>.ini (mine is UI_Kudane_91.ini for example)

4. RENAME THIS FILE adding _backup (for mine it would be UI_Kudane_91_backup.ini)

5. Copy the UI_Stratix_31.ini file into your default directory...

6 Rename the file _<charactername>_<server number>.ini (again for me.. UI_Kudane_91.ini)

again you MUST DO EVERY STEP.. (ok you dont need a backup of your INI but its better to have it)

If you dont like the mod later... simply swap the names on you backup and regular file...
