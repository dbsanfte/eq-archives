Nothing but edit. and sorry, no real "Web Page Tutorials" that I know of.

open up Notepad and open up a .xml file withing your Default directory ( Everquest\uifiles\default\ ) and take alook.

If you gonna edit it, first make a new folder inside of uifiles and then copy the files into that directory. Make sure the directory has no spaces, that's your UI's name, for the /load command.

Copy the file into there and edit it. Then when you start EQ, /load <name> 1.

Nothing more than just editing text.
