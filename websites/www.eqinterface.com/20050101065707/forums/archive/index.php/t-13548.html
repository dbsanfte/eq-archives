Most Ui's unzip to their own folder now. If not, all you need to do is go to your everquest/uifiles folder and make a folder, then unzip the UI that you want into that empty folder. As far as the .ini goes, not all UI's provide a generic .ini. If one does, just copy the contents of the UI_Player_Server.ini into the .ini for whatever character you want to use the UI for. The .ini files are found directly in your everquest folder.
