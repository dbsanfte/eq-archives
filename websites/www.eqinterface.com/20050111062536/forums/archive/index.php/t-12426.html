Used Kuvasie's workaround last night and it works fine.

1. Download the ani files.
2. ** Do not run Remmy patch at this point** Just close it
3. Find the animations xml in the directory you just copied to
4. Right click on the icon - Select "Open With" and use Wordpad
5. Save the file
6. Go back and Run Remmy (which is in the directory) and let it patch fully
