Perhaps more work than you fell comfortable doing, but to get transparent buttons on any mod is pretty easy. All you have to do is find each of the button objects and set the <Style_Transparent> tags to be true. This way you can look at all of the mods to find one you like and then simply edit it to be transparent. The button objects will start with <Button item="BW_Buff#_Button"> (where # is a number 0-14) and will contain a line saying <Style_Transparent>false</Style_Transparent> . Just change that to true and you are in business.
