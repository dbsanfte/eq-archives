Originally posted by Riksar

IF you've edited the .xml files correctly, then the other possible problem is with texture caching.

If everything esle is correct and you've been using texture caching to speed up zoning performance, then turning it OFF will cause EQ to properly load any new textures.

You can turn texture caching off BEFORE logging into the game, by using the button marked OPTIONS from the login screen. This will cause EQ to quit caching local copies of textures and graphics.

You can turn caching back on afterwards in order to restore any speed gain you recieved while using it.
