Edit your tga file.

Do NOT Create a background. Do NOT flaten your image.

When you're done editing it, it's time to create the transpartent background.

Select everything, I assume you have some basic knowledge of photoshop and how to use it.

Cntl-Click on your Finished Layer(s) in your layer window. If there's more than one, Shift-Cntl click on all of them.

Then goto Select -> Save Selection. Make sure that it says "New' in the second bar there ( I forget the name ) and there is nothing else in it.
Then click Okay.
If there's already an Alpha layer listed in there. Select that, and select the Replace option. Then click Okay.


Note - I'm not a graphic artist. I didn't know anything about Channels or how to create an Alpha Channel in PS before EQ's UI stuff. So I explained this as I have learned it.
