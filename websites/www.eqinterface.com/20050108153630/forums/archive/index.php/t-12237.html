First guess is to check your UI_Character_server.ini file, look for the entry for the window you are working on, if width and height are defined in this file it will override what is defined in the xml. You can delete the entire entry or just the width and height lines to make it use the default xml size. Also be aware that there are sometimes mulitple definitions for diferent screen resolutions, they they will take the form "width###x###" and "height###x###". Or if you are doing alot of work on the UI you can delete the whole file and it will be recreated the next time the character logs in and use the the position and size as defined in the xml.
