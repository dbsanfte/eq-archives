Lothay UI V 1.1 - RELEASE VERSION
An Interface Modification for EverQuest.
Inspired by Verina and her NOKUI

V 1.1 - Consolidated code and templates to EQUI_Templats.xml & EQUI_Animations.xml
- Fixed positions of various bars in player window
V 1.0.5 - Cleaned up some code, fixed misdefined buttons in several windows.

-- Features:
*Target Window:
- Large easy to read guage.
- Outer border is /consider color.
- Integrated Spell Casting Guage.
*Player Window
- Attack indicator
- Mana, Endurance, and AA Exp bars
- HP (Current / Max), AC and Attack values now displayed
* Group Window
- BIG health guages for easy health reading.
- F2 - F6 Labels for easier selection of group members.
- Pet health shows at top of group member's bar.
- Same height as a new chat window for easy alignment.
* Pet Window
- Tabbed interface.
*Hot Button Window
- 8 Bag slots.
- Compact, easy to read.
* Buff Window
- Aligned for positioning on right-had side of screen.
- Buff slots numbered 1 - 15.
*Short Duration Buff Window
- Aligned for positioning on right-had side of screen.
- Song slots numbered S1 - S6.
*General
- All windows now display with black-transparent backgrounds.
- Designed with a screen resoultion of 1024x768 in mind
works well at higher screen resolution settings.
- Works for casters, hybrids and melee classes.
- Modular. Many windows stack.
- Allows drop-in of any other mod that doesn't change the
EQUI_Templats.xml & EQUI_Animations.xml files.
