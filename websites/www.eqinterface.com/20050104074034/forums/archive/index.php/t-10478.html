I know exactly why it's freezing up on you. Every time EQ comes across a file that doesn't have proper syntax , it's going to do this to you.

Look in your slowPlanes.txt file. Toward the top you have this:

<br><c "#FF0000>Cazic Thule -- Immune</C>
<br><c "#0069FF>Dracoliche -- Magic</C>
<br><c "#0069FF>Dread -- Magic</C>
<br><c "#FFFF00>Enraged Golem -- Unslowable</C>
<br><c "#0069FF>Fright -- Magic</C>
<br><c "#00FF00>Terror -- Disease</C>
<br><c "#FF0000>A Tempest Reaver -- Immune</C>


See what's wrong here?? You forgot the ending quotes on all the color statements. Fix that, and the file should display correctly instead of freezing on you.
