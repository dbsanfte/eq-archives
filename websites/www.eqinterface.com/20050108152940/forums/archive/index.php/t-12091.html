If the target indicator you are missing is the new 3D Target ring, then first off make sure that your video card is a supported model. Basicly, if you cant see the default 3d ring, you wont be able to see mine. In order to check if you can see default, first make sure and use the /indicator on command to double check the 3d targeting is on. There is a list of supported video cards in the SOE Corner forum threads.

If you are refering to the Target Window itself, then try first to toggle the window by using (ALT + T). If this doesnt bring the target window up, then open your UI_Player file and find the section for [Target Window]. Make certain the x and y dimentions are > 0 so that it would be visible. There are detailed instructions on installation and frequent problems at my site http://sars.has.it as well as in the READ ME file included with the zip. Also remember that in sars 4, you will not see the target window at all until you click on a target or you cast a spell.

Lastly i am in alpha stage for an auto installation/configuration/patching program that will be available for beta as soon as possible that should eliminate 99% of the problems people are having with out of date files or incorrect installation problems. Look for more news on this in my forums as i progress. Hope this helps!
