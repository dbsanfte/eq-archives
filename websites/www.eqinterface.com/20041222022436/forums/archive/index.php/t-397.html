As you have posted in this forum I assume you have set up a new UI for the character if this is the case then it sounds like it has a bug. In this case you will need to go to the Everquest directory and look for a file called UI_XXX_99.ini where XXX is your characters name and 99 is some arbitrary number. Open the file and look for UISkin. Change this to "default" and restart. This will use the everquest skin.
