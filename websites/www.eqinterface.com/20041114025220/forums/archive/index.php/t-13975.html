To change the chat window backgrounds there are two files you need to replace in the UI. First, find a UI whose chat window backgrounds you like and copy the files in that UI named, "wnd_bg_dark_rock" and "wnd_bg_light_rock" to the UI folder in which this UI is named and that will change the background of your chat windows.

There is nothing else needed to do within the EQUI files.

Your other options.... which may be easier, is to just change the color of your text.. this can be done in game, by using the options menu (ALT+O) and pressing the "Colors" tab.

To close any chat window you will need to modify the EQUI_ChatWindow file as follows:

Open the EQUI_ChatWindow file using Wordpad or Notepad.

Find:
<Style_Closebox>false</Style_Closebox>

Change to:

<Style_Closebox>true</Style_Closebox>

Save and reload the UI... you will see the "close" X on each window... When you load the UI you will see why I chose not to have the close option on the Chat windows....

Downloaded the new PET-WINDOW here (http://www.eqinterface.com/downloads/fileinfo.php?id=3607) and get the following error after inserting it into my UI :

[Line:2927 Source:UIFiles\Qinadwen\EQUI_PetInfoWindow.xml] Schema error - Duplicate item PW_STRNumber

I use the caster version :)

Anyone else have this problem and possibly know how to fix it?

You have that problem because a file in your playerwindow is named the same as the one in the pet window.... go into the EQUI_PetInfoWindow.xml file and rename PW_STRNumber to something like PetW_STRNumber and then rename the <pieces>PW_STRNumber</pieces> to </pieces>PetW_STRNumber</pieces>.

That should fix your problem
