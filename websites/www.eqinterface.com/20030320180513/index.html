UI News & Updates



Latest News - Latest Updates - Patch Fixes - EQ Interface News
[ F.A.Q ]



HAPPY ST. PATRICK'S DAY!


Latest News
  • Virus Alert:
    Kudane's has posted about a "Trojan virus being spread around via a UI by the name of GlixUI. It is a keyboard recorder for 'stealing' EQ accounts + passwords." This UI is has NOT been in the Downloads area of EQ Interface/EQGui. Please carefully read the information posted.

  • Patch Message, 03/13/03:
    ** Interface ** - Added a command to allow you to toggle your target between your two last targets (unlike Tab which toggles between your character and your last target). To set the key that activates this feature, just open your options window and select the Keyboard Tab. If you select all categories, the entry will be the last one on the list, called Toggle last two targets. This was actually added with our last patch.

  • Video Lag?
    From Remelio: Bad Video Lag in some Zones? Try this: After you zone in, do the loadskin command
    /loadskin < modname > 1. This will re-load your UI and get your framerate back up to normal speed. This is NOT a problem with custom UI's as this also happens with the default UI. This is a problem with the way EQ loads UI's while zoning, and they have been made aware of it. Thanks Vaslin for pointing out this fix!
