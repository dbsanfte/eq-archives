Those class files contain the funky graphics for the classes (spinning sword over a shield for warriors for example) you see in the inventory window.

The wnd_bg_dark_rock.tga is the graphic for a windows background - in this case it's called dark rock.

The windows_pieces_xx are graphics used to draw the windows. Cursors, sliders, borders, etc.

All these graphics are saved in the Targa file format hence the .tga extensions.
