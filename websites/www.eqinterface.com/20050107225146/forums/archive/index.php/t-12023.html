/viewport x y width height

It changes the resolution , but leaves your UI at the current resolution.

So in 1280 x 1024

/view 200 100 800 600

will make the viewing area 800 x 600 but leave everything else (the UI) untouched, almost centering the "view".

/view 0 0 800 600 would stick the viewport up in the right hand corner.

Use "/view reset" to reset it.
