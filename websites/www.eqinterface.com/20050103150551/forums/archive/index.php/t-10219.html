I'm pretty sure the reason you are getting the error is because the comment lines are wrong in that file.

Here are the changes you need to make:

Find this line in your code: //added via patch 24 Aug 2003
and change it to this: <!-- added via patch 24 Aug 2003 -->


Then find this line: // end added via patch 24 Aug 2003. More below, see www.equi.com Forums: Patches
and change it to this: <!-- end added via patch 24 Aug 2003. More below, see www.equi.com Forums: Patches -->


I believe that should fix the error for you. Good luck! :p

One other note about your edit...(maybe I'm not taking what you wrote correctly) NEVER copy your default folder and insert the files in there!!! That is a big no no! Only the files that have been changed should be in your custom folder. Coping the default folder and making changes there will work for a little, but the next time Sony patches anything, it will most likely break the custom UI.
