First:
Right click the Window. Follow the menu in the following order -
Window Background Texture
That will turn off the texture (or turn it back on if it is off).

Then:
Right click the Window. Follow the menu in the following order -
Window Background Tint Color
Select the color you wish the background to be. :) No need to mod the window at all.
