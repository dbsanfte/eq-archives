I believe that this is because you have the EQ display attributes set to the the same screen resoultion that you have set for MS Windows. When running EQ in windowed mode, it adds the typical MS windows frame, which extends the size of the window beyond the limits of what MS Windows can display on one screen.

Try reducing the EQ resolution and it should fit within one screen.
