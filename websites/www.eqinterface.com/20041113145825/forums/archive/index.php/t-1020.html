First download the following two files, and install them into your current UI directory (overwrite files):
http://members.cox.net/jcs0001/EQUI_CompassWnd.xml
http://members.cox.net/jcs0001/window_pieces04.tga

Second open your EQUI_Animations.xml file that is in your current UI directory and,
REPLACE THE FOLLOWING CODE:

<!-- Compass art -->

<Ui2DAnimation item = "A_CompassStrip">
<Frames>

<Texture>window_pieces04.tga</Texture>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>180</CX>
<CY>24</CY>
</Size>
</Frames>
</Ui2DAnimation>

<Ui2DAnimation item = "A_CompassOverlay">
<Frames>

<Texture>window_pieces04.tga</Texture>
<Location>
<X>0</X>
<Y>24</Y>
</Location>
<Size>
<CX>98</CX>
<CY>24</CY>
</Size>
</Frames>
</Ui2DAnimation>

With this modified code:

<!-- Compass art -->

<Ui2DAnimation item = "A_CompassStrip">
<Frames>

</Size>

</Ui2DAnimation>

<Ui2DAnimation item = "A_CompassOverlay">
<Frames>
<Texture>window_pieces04.tga</Texture>
<Location>
<X>0</X>
<Y>24</Y>
</Location>
<Size>
<CX>35</CX>
<CY>24</CY>
</Size>
</Frames>
</Ui2DAnimation>

sorry if this isnt all neat and tidy, but my current UI is a hodgepodge of miscellaneous and custom UI mods.

following my instructions here will ensure that this mod works will ALL mods.
