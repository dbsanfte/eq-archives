Font size, yes. Within every EQUI file that has any reference to text, it says <Font>3</Font> .... the 3 indicates the size... it goes off the same sizing as it does when you do /chatfont 3 when you're in the game... so you can figure it from there what size you want. If you make it larger, you'll usually have to increase the
<Size>
<CY>15<CY>
that comes a little later in the code... just increase it by one or two - this will avoid "clipping" the letters... having half a "g" for instance.


As for the font itself, as far as I'm aware, EQ determines that. :( I wish we could change that. Maybe some day... /sigh


And you can probably find something with a larger / smaller font size... I think a lot of people alter those... but it's done per individual window/screen piece... not an overall thing.


~Lady Silver
