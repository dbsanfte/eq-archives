I explain this one all the time at work, and will try to give a full walkthrough:

1) Open the image

2) To make this walkthrough as simple as possible, DELETE ALL LAYERS THAT WILL NOT BE ADDED TO THE ALPHA

3) make sure that there are NO alphas already in the channels tab ( there should only be four... "RGB", "Red", Green", Blue" just like your ps7 screenshot)

3) in the layers tab, hold ctrl and left click on the first layer

4) still in the layers tab, hold ctrl+shift and left click onthe next layer

5) repeat step 4 till you have done this to all of the layers


Now, you should have a "selection" of all the elements in the image


6) goto the channels tab

7) hit the button at the bottom of the channels tab that looks like a white circle in a black rectangle (second from the left)

8) <in AOL guy's voice> You've Got Alpha!

PS 6 uses the alpha channel to mask items for display.. PS 7 applies this mask to the actually layer.

I have started using PS7, and the rule of thumb for it is..

anything showing will show in EQ
make sure you merge back down to "Layer zero"
save in 32bit

using those three things, i have had no problems with my images..

if using PS 6, follow steps listed above.. they are perfect..
