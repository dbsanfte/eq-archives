Changing the container windows is actually quite easy. The file name is: EQUI_Container.xml

You may also want to consider moving EQUI_Inventory.xml and EQUI_MerchantWnd.xml just to move all of those windows with a similar purpose and feel.

Just move the preferred versions of those files to the directory of the interface you are using. There are no other "hooks" unless Caeal has added something. If he has and that doesn't work, it will typically be an issue with EQUI_Animations.xml or EQUI_Templates.xml. Try moving both of those as well (but only do that if you have to - this can cause problems with patches if the files were not altered correctly).

Thanks for the feedback and good luck!
