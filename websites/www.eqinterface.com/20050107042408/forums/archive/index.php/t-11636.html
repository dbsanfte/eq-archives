You have experienced a corruption that is relatively common. it is not due to the UI, but rather the OPTION files in the Main EQ folder and possibly some dlls/exes

Delete eqmain.dll
Delete eqgame.exe
Delete eqgfx_dx8.dll
Repatch Full File Check.

In some cases, you need to go further.
Delete *.OPT.
Delete EQClient.ini
(you'll need to reset a lot of options after this one, but has led to a fix many times.)

Good Luck.
Note: If you prefer, rename the files. The patcher will d/l new ones for you, but some people prefer to be extra safe.
