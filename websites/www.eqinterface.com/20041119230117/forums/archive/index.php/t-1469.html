If the Tooltip reference tags looks like <TooltipReference /> then there isn't a way to change it, because the tooltip is done server-side. If you see <TooltipReference>This is a tooltip</TooltipReference> then just delete everything between the tooltip reference tags (you could, in reality, change it to look like the first one I showed, but then if you went back and looked at it you wouldn't be able to readily tell if that was a modification or if that was the default)
