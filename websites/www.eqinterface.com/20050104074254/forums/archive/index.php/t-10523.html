You can do this without any modifications.

This function is actually built into the default interface already.

If you right click on the titlebar if your original chat window (reguardless of ui, i think) it will bring up a menu, where you can select to create a new chat window.

Once you have your new chat window, right click on its title bar, and you can select to Filter specific things into it (which includes melee data).

Once you've done that, all regular chat will still go to the main window, but any battle damage, spell damage, or whatever you selected to filter, will go into the new window.

Unless I am mistaking what you meant, I hope that helps.
