*** 2 Working mods I designed ***

You can install either the Hotkey box mod, or the Inventory box mod, or both, but whichever you use, you have to install both the other files (EQUI_Animations.xml & window_pieces_large01.tga)

A single zip file with a readme is in the works, as well as instructions on what to do if VI patches EQUI_Animation.xml or any of the other files in the future. These work with all patches to 1st August 2002

(Use: --- Rightclick --- Save target as... --- for DLing these)

Download those files here:
http://home.btconnect.com/armoury/DemoMod_V1/EQUI_Animations.xml
http://home.btconnect.com/armoury/DemoMod_V1/window_pieces_large01.tga


Hotkey box mod:

http://home.btconnect.com/armoury/DemoMod_V1/DemoMod_Hotbox_V1.jpg

Download here: http://home.btconnect.com/armoury/DemoMod_V1/EQUI_HotButtonWnd.xml

Hotkey boxes numbered 1,2,3 along top, 4,5,6 middle, 7,8,9 bottom, and an extra large 10 for easier clicking. Slots on the right are (reading down) Pri, Sec, Bag7, Bag8 for easy midfight weapon swapping

Inventory bar mod:

http://home.btconnect.com/armoury/DemoMod_V1/DemoMod_Inv_V1.jpg

Hps are removed, since I use them in player bar, and at this time, only one instance of certain values will display the correct amount. If/when VI fixes it, HPs will show here too. All the green numbers are white when unbuffed

Download here: http://home.btconnect.com/armoury/DemoMod_V1/EQUI_Inventory.xml
