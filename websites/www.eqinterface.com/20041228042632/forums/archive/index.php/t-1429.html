This is a modified chat window used to demonstrate a few window tricks. Since the overlap settings on title bars don't work (a bug), it's not so easy to remove the title bar without messing up the graphics.

Removes the light grey border from around the chat output window (that's easy).
Removes the light grey border from around the chat input window (not so easy).
Removes the titlebar but the window can still be minimized, or moved by clicking on the top edge.
Removes the borders but the window can still be resized by clicking on the other three edges. (Also includes, for those interested, a WDT that leaves the borders, removing only the title bar.)

Includes only a single file, EQUI_ChatWindow.xml, zipped for your inconvenience.
