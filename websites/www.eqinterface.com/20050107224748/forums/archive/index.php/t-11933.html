For a target ring you need the following:
TargetIndicator.tga (or whatever .tga is specified in the .ini)
TargetIndicator.ini

If the .ini says:
Texture=wierdness
Then you need:
wierdness.tga not
TargetIndicator.tga

If you are in game after putting this in your folder type /indicator off followed by /indicator on.

If you are not in the game when you put these in your folder then when you start the game it should work.

In your EQClient.ini make sure u have the following:

[Defaults]
UseTNL=TRUE

[Options]
Show3dTargetIndicator=1
