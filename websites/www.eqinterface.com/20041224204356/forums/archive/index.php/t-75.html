Depends on what modifications you need to combine. Some of mine require changes to the EQUI_animations.xml file (where the graphics are loaded from), if 2 people modify the same file you could come into trouble.

But to put it simply you could copy the default directory rename it and the get the XML files for only the windows you want out of the downloaded file and extract them into the new directory. Once all are there load up and pray for the best. In no success you could try debugging it after that until it does work.
