11-10-2003, 05:35 PM
Then copied the default folder

Don't do that step. The only files that should be in your custom folder are files that have actually been customized. Otherwise, you are setting yourself up for a world of headache, as every time SOE changes any little tiny thing in any window, you will most probably have a UI that no longer works.

From the looks of the error message, there is an error in the window you downloaded. The author has (most likely) forgotten an opening or closing statement, or a partial statement (ie < > bracket missing, etc).

You can look to see if they have released an updated functional one, you can try going through the file yourself (start with the line number specified in the error log) or zip it up and attach it here and hopefully someone will have a chance to take a look at it, find and fix the error.
