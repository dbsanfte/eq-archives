/edit Okay, this is nuts. Some folks are getting one patch download, some folks are getting another ... soooooo, I'm gonna see if this universal fix will work:

IF you have a custom EQUI.xml file, open using a text editor (Word, WordPerfect, NotePad, etc.) and:

Using the text editor's Search or Find function, find:

<Include>EQUI_StoryWnd.xml</Include>

Immediately after that line, ADD:

<Include>EQUI_BigBankWnd.xml</Include>
<Include>EQUI_BodyTintWnd.xml</Include>

==============================================

Whether you have a custom EQUI.xml file or not, IF you are crashing out, unzip the attached files into your Default folder.

Edit 12:36 am EST: Added EQUI_CharacterCreate.xml file to the zip for those that got that other error, having recently discovered that there was a 3rd wrong push.
