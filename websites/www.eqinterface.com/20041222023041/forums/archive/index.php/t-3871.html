News - 11/09/02

Featured Skins
Sokol has Vert updates available - a fix to the EQUI_Animations.xml (http://www.eqinterface.com/sdownload/fileinfo.php?id=532) file that should reduce lag issues, and updates to Vert 2.2 CrystalMeth (http://www.eqinterface.com/sdownload/fileinfo.php?id=533) and Vert 2.2 Canabis (http://www.eqinterface.com/sdownload/fileinfo.php?id=534); both the latter changes involved temporary fixes for errors involving the BazaarSearchWindow, and also some lag reduction changes.

Buffs & Effects (http://www.eqinterface.com/sdownload/index.php?op=showfiles&catid=21)
Remelio (http://pub165.ezboard.com/ftranscendence39908frm31) has modified a Horizontal Buff Window (http://pub165.ezboard.com/ftranscendence39908frm31.showMessage?topicID=50.topic) by Cairenn (http://www.eqinterface.com/list.php?skinnerid=187) to include a top border for dragging.

Complete Sets (http://www.eqinterface.com/sdownload/index.php?op=showfiles&catid=22)
Sylverpaw (http://www.eqinterface.com/list.php?skinnerid=1398) has released a series of UI's for various classes: Monk, Wizard, Enchanter, Beastlord, and Druid.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- Each one has a basic 'Skin' version, ready for modification, and a 'Plus' version, which include window mods from other authors.
Achebrosh has created a RogueUI (http://www.eqinterface.com/sdownload/fileinfo.php?id=545), with various rogue-aimed window mods, and original art for the backgrounds.
Taniselvin has updated Crystal 1.1 (http://www.eqinterface.com/sdownload/fileinfo.php?id=547) to account for the recent Bazaar changes.

Hot Keys (http://www.eqinterface.com/sdownload/index.php?op=showfiles&catid=18)
Vixyn has released a Hotbuttonn Window (http://www.eqinterface.com/sdownload/fileinfo.php?id=549) that includes Inventory slots and all the gauges and stats.

Player Info (http://www.eqinterface.com/sdownload/index.php?op=showfiles&catid=20)
Aviendha has uploaded a Melee Player Window (http://www.eqinterface.com/sdownload/fileinfo.php?id=548), part of her Player/Group package. This window replaces the Class listing with ATK.

Spell Books (http://www.eqinterface.com/sdownload/index.php?op=showfiles&catid=25)
Manta has created the DeathAngel (http://www.eqinterface.com/sdownload/fileinfo.php?id=544) spell book, for those who lean towards the "evil" side.
