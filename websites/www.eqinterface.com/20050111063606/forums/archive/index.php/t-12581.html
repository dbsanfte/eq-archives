The .ini (which is short for initialization) file contains all the window postions and settings for the interface. It is named like this (using my own as an example): UI_Lothay_tunare.ini

The first part, UI, simply means that it is the initialization file for the User Interface.

The next part, _Lothay_, is the character's name.

The last part, tunare, is the server name. At one time the server name was not used. Server's were refered to by a number. This has changed.

If the artist who posted the UI you wish to use included a file like that, they just meant for you to change the character name and server name parts (in the example, the blue and green parts). The file contains all the window postions and settings.

I hope this helps.

The UI_Name_Servername.ini file exists for all UIs (including default.) So to find what the file name should look like, just look in your everquest folder. There will be a UI file there for each of your characters and it will have the correct name format for your server. (So there should be a file there UI_Otik_AntoniusBayle.ini. NOTE: AntoniusBayle, as used here, may not be the correct format for server name, but it will be something close.) So just follow Lothay's instructions and it should place all the windows in the correct positions.
