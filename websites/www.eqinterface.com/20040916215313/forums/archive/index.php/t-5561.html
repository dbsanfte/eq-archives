let's see. For each character you have a UI_charname_servernumber.INI file. This file holds the info for where each of your windows is posisitioned and such.

So you would simply find your chars .INI in your everquest directory, rename the INI that came with the UI you want to use and copy it over the one you currently have. Of course you might want to back it up in case you decided you don't like the UI and want your last one back.

Then just boot EQ and use the /load skinname 1 command.


Make sure you use the 1 in the command above it it will reset all the windows posistions
