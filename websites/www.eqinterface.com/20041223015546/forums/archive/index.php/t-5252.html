Patch coming this Thursday ... 1/9/03

New file:

EQUI_ShortDurationBuffWindow.xml - This is the heralded new window for bard song icons.

Changes to existing files:

Add LW_LootSlot30 to EQUI_LootWnd.xml - This is probably a brain fart. Someone forgot that 0-29 is 30 slots and added slot 30.

Add PIW_StandButton to EQUI_PetInfoWindow.xml - This appears in the same spot as the "/pet sit down" button, much like the player's Sit/Stand button.

Add SELW_SDBuffToggleButton to EQUI_SelectorWnd.xml - This is a toggle to open and close the new short duration buffs window.

These changes will not prevent a UI from loading, but they will display an error message at login and when you load a skin.
