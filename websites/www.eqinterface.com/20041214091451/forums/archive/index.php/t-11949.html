UPDATE: Due to popular demand you now have the option to display a con color indicator on your custom target window! Enjoy. :)

This doesn't have a specific file as it's totally custom made per user so bare with me here...

http://www.tritonguild.com/UI/

http://www.eqgui.com/attachment.php?s=&postid=62918

Basically what this script does is generates a custom targeting window based on your specifications. You can have 1-6 clickables (if people want more I can allow more) available on it. Once you decide how many you can then pick which ones you want (any inventory including bags). After that you click a button and a custom EQUI_TargetWindow.XML is generated specifically for you to download and place in your UI folder. It's universal and involves no external textures so it will work with any UI (I use Vert for example, but it works perfectly with default).

Anyway it's in it's final stages of testing so please mess around with it and tell me what you think. If you have suggestions on how to improve it please share. ;)

Many thanks in advance!

- Rombus
