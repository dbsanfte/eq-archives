Click on the Home button, located to the upper left on your current page.

To your left you see a frame like structure including Main and downloads. Click on complete sets and go to page 3 I think. There is the 2.0 version. Then get back to this thread and download the fix, voila, you have all files. :)

To resie the chatwindows, copy the file in the folder called *sizeable* located in the mods/yourresolution folder. Resize the chatwindow as desire and if you want to, copy the non sizable file in the same folder to your StoneAngel folder again. (The Sizes are saved in a different file). Basicaly, read the readme file. It's all there. :)
