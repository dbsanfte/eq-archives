Ok as for the cast time bar, it is commented out in the EQUI.xml file that resides in the directory where you have this mod. All you have to do for it is to open the EQUI.xml file up and remove the <!-- --> from around the line that says EQUI_CastingWindow.xml. That should allow you to see the T. King one. As for the added chat boxes, I'll have to look at this tonight when I get home but I think if you go into the EQUI_ChatWindow.xml file and change the border line to true instead of false you would be able to move the new ones. I don't think you can drag the whole window at once but you can stretch it until you get it where you want it.
