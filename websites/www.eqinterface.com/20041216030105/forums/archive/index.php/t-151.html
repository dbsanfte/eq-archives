From EQInterface.com (http://www.eqinterface.com)
Instructions for loading skins/mods in game:

1. Go to the uifiles directory in your Everquest folder, Right-Click on the DEFAULT folder there, and select "Copy"

2. Then Right-Click in the same folder as default (beside it) and select "Paste"

3. Re-name "Copy of Default" a new name, keeping it short

4. Download the file you want and un-zip them.

5. Drag all the items in your un-zipped file folder into your new folder, (you just named a short name) it will ask you "do you want to copy over...." select "YES TO ALL".

6. Log into Everquest

6. use /loadskin <folder name> (this is why you keep it short)

****If your using a SKIN Mod, you must then camp out and back into the game to see it****

Instructions for loading skins/mods in game:

First, you no longer have to copy the default, rename, paste over with the new mods. That takes up a lot of extra space and is no longer necessary. If EQ doesn't find the file it needs in your new one, it will look in the default. So you only need the new files in your new folder.

As for the error you're getting, right after it crashes, look in your Everquest folder for a file called UIErrors.txt ... that has information on why it couldn't load. Depending on the skin/mod you downloaded, it may not be compatable with the recent patch... I'm not sure if all the authors made their mods be compatable. I could be wrong on that. But first, find out what the UIError is so we can have a better idea of what to tell you to do. :)
