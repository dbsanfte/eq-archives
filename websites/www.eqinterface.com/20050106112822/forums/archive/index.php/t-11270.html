Works fine on all my tested UI's
And i test alot hehe.

Make sure you have the code on top just under:
<?xml version = "1.0"?>
<XML ID = "EQInterfaceDefinitionLanguage">
<Schema xmlns = "EverQuestData" xmlns:dt = "EverQuestDataTypes"/>

Add Pieces to buttom of xml file so it looks like this:

<Pieces>CWCasting_Gauge</Pieces>
</Screen>
</XML>
