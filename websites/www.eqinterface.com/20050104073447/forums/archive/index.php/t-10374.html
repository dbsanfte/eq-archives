I just did it in my player window.

Labeled the target area in text to show Target: below it shows 0%, then when a mob or a player is targeted shows player name below the Target: with the accurate hp %. Below are the elements i used. Youll have to paste them into the proper sections of your xml document, but you shouldn't have any trouble figuring it out. (Bold is the Target Label):

<Gauge item = "Player_Target_HP">
<ScreenID>TargetHP</ScreenID>
<Font>3</Font>
<TextColor>
<R>240</R>
<G>240</G>
<B>240</B>
</TextColor>
<RelativePosition>true</RelativePosition>
<Location>
<X>118</X>
<Y>17</Y>
</Location>
<Size>
<CX>108</CX>
<CY>24</CY>
</Size>
<GaugeOffsetY>16</GaugeOffsetY>
<TextOffsetX>8</TextOffsetX>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
<TooltipReference/>
<FillTint>
<R>240</R>
<G>0</G>
<B>0</B>
</FillTint>
<LinesFillTint>
<R>220</R>
<G>220</G>
<B>0</B>
</LinesFillTint>
<DrawLinesFill>false</DrawLinesFill>
<EQType>6</EQType>
<GaugeDrawTemplate>
<Background>A_GaugeBackground</Background>
<Fill>A_GaugeFill</Fill>
<Lines>A_GaugeLines</Lines>
<LinesFill>A_GaugeLinesFill</LinesFill>
<EndCapLeft>A_GaugeEndCapLeft</EndCapLeft>
<EndCapRight>A_GaugeEndCapRight</EndCapRight>
</GaugeDrawTemplate>
</Gauge>

<Label item = "Player_Target_HPLabel">
<ScreenID>HPLabel</ScreenID>
<EQType>29</EQType>
<RelativePosition>true</RelativePosition>
<Location>
<X>118</X>
<Y>29</Y>
</Location>
<Size>
<CX>60</CX>
<CY>12</CY>
</Size>
<Text>Playername123456</Text>
<TextColor>
<R>255</R>
<G>255</G>
<B>255</B>
</TextColor>
<NoWrap>true</NoWrap>
<AlignCenter>false</AlignCenter>
<AlignRight>true</AlignRight>
</Label>

<Label item ="Player_Target_HPPercLabel">
<ScreenID>HPPercLabel</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>182</X>
<Y>29</Y>
</Location>
<Size>
<CX>16</CX>
<CY>12</CY>
</Size>
<Text>%</Text>
<TextColor>
<R>255</R>
<G>255</G>
<B>255</B>
</TextColor>
<NoWrap>true</NoWrap>
<AlignCenter>false</AlignCenter>
<AlignRight>false</AlignRight>
</Label>

<Label item = "Player_TargetName">
<ScreenID>PlayerTargetName</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>118</X>
<Y>3</Y>
</Location>
<Size>
<CX>60</CX>
<CY>12</CY>
</Size>
<Text>Target:</Text>
<TextColor>
<R>255</R>
<G>255</G>
<B>255</B>
</TextColor>
<NoWrap>true</NoWrap>
<AlignCenter>false</AlignCenter>
<AlignRight>false</AlignRight>
</Label>

<Pieces>Player_Target_HP</Pieces>
<Pieces>Player_Target_HPLabel</Pieces>
<Pieces>Player_Target_HPPercLabel</Pieces>
<Pieces>Player_TargetName</Pieces>

Obviously the numbers will change according to what you wish to do with them but it works.

I made an empty gauge as was suggested and posted it above the code for the regular gauge. If it's below the regular gauge, it remains on top and it will appear that the target always has empty hp. Here's the code for the gauge I used:

<Gauge item = "GW_Player_Target_EmptyHP">
<ScreenID>TargetEmptyHP</ScreenID>
<Font>3</Font>
<RelativePosition>true</RelativePosition>
<Location>
<X>179</X>
<Y>31</Y>
</Location>
<Size>
<CX>108</CX>
<CY>8</CY>
</Size>
<GaugeOffsetY>0</GaugeOffsetY>
<TextOffsetX>0</TextOffsetX>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
<TooltipReference/>
<DrawLinesFill>false</DrawLinesFill>
<GaugeDrawTemplate>
<Background>A_GaugeBackground</Background>
<EndCapLeft>A_GaugeEndCapLeft</EndCapLeft>
<EndCapRight>A_GaugeEndCapRight</EndCapRight>
</GaugeDrawTemplate>
</Gauge>

I also added the Pieces tag above the Pieces for the main target gauge as well. You'll need to adjust the numbers as well, depending on the positioning you use in your target window.
