Dalink's 90 Minute Count UP, 90 Minute Count DOWN,
30 Minute Count UP, and 30 Minute Count Down Timer Set



Extract these files to whatever directory you use for your User Interface.

To restart the timer, type /loadskin <YOUR UI DIRECTORY> 1

The "1" switch is very important, this is what keeps your window positions.

The Ui Directory is the directory for whatever Ui Directory you extracted these files to.

Select the different timers with the tabs on the Actions Window.

When each timer reaches its limit, it will restart.

The timers do remain acurate through zones. In other words, they keep counting while yer zoning.

I hope you like it.
