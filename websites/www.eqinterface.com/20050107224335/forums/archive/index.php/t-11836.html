What resolution do you use?

Also setting viewport lets you adjust the screen inside the ui elements so nothing is overlapping and you can still see everything and none of the ui interferes.

/viewport x y height width

x= horizontal start point
y = vertical start point
height=size from top-bottom
width=size from left-right

/viewport 0 0 800 600 would set the screen at the upper left corner with a height of 800 pixels and a width of 600 pixels.
