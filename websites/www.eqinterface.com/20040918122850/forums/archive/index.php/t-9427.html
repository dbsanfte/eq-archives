Compton User Interface

Compton Home Page (http://moresoda.com/eq/ui/) | Current Screenshot (http://moresoda.com/eq/ui/current.jpg) | Current Version (http://moresoda.com/eq/ui/current.zip)

Note: this UI is designed for hybrids and pure melees. It uses the pet window in ways which would be maddening to pet classes. Sorry pet people!

Features:
* Custom interface graphics
* Unique buff / bard buff bars
* Friends window converted to invetory for quick weapon swaps
* Compact, clean yet informational layout of stats and data
* Includes installation guide and sample window configuration

Check the home page for frequent updates. Please submit any errors or ideas to fennin@moresoda.com, or post them here. Thanks!
