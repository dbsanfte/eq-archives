When you added windows that were not part of the original Tablets UI you replaced windows that had code in them that the rest of the Ui needed to load.

You will have to use the UI as it was intended or figure out how to get the code that the UI needs to run into another file so that it can still load.

To give you an idea of what I am talking about:
I might make a custom gauge for my UI and put the code for it in my player window. I used that gauge for every gauge in my UI, but it only needs to be coded once..and that code is in the player window.
If you remove that player window, then the rest of the UI would be looking for that gauge still, and would not be able to find it because you removed the code for it as well.
The UI would then give you an error like the ones you are seeing.
