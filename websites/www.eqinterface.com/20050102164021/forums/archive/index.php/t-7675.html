Although you don't have LoY, EQ still wants your files to be "current". You need to make the changes to your custom EQUI_Animations.xml file as outlined in the 24 Feb patch message in the Patch Fixes forum.

It isn't actually your inspect window (although if you do have a custom one, you should update it as well, same thread), but your Animations file that is causing EQ to gack. "FieldParseItemOfClass" almost inevitably indicates an outdated Animations file.
