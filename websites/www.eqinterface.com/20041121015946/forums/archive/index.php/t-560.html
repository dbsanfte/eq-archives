Don't copy the default directory. As Remelio has so nicely pointed out on his post at Tensendence, you do not need to copy this folder!

Actually, it's prefered you don't. Just cause of possabilities of new files, etc.

Read the thread for full detail:
http://pub79.ezboard.com/ftranscendence39908frm31.showMessage?topicID=9.topic


Doing this, files like EQUI_Animation, when you've *never* altered this file will no longer flank up in custom UIs.

Granted, if you've modified a file that got patched, then you'll have to look into merging the changes over. But not too hard, if the code's done and commented correctly in the modified file.
