The file Spell_Icons.tga has all the old icons. But the new GUI uses the files Spells01.tga, Spells02.tga... Spells05.tga. All you have to do is a little cut and paste to put the old icons over the new ones in those five files.

If you don't have a good image editor use Microsoft Paint to do the cut and paste. Paint won't save in tga format and it won't do transparencies, but you don't need either of those things for spell icons. Just remember that if you save them as bmp files, you need to go into EQUI_Animations and change all references to Spells01.tga to read Spells01.bmp.
