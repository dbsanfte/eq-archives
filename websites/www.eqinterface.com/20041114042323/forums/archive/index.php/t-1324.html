Wonderful interface for the mouse-user who wants everything readily available. I've been working on a UI for a keyboard-user that's more minimalist, so came up with something quite different. But you've done something very good.

Love the pet window and the combat tabs. Also the octagonal colors around the spell gems really highlight them well.

I put my casting time as a sub bar of the mana bar, rather than the target bar. Also, I removed the endcaps from all my bars -- they serve no function and don't match the colors. Something to consider :)

BTW -- I like that there's a gap below the chat windows for clicking on things. But I find I often need to click on the input bar in the main chat window in order to type a command while using the bazaar search window. It's easier to do if the input bar on that one window is flush with the bottom of the screen. But it's not possible to move the chat window down that far unless it's movable as well as resizable. I invented a window that's movable without a title bar (posted in released section), and it was worth it.
