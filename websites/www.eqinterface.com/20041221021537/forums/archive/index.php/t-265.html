Change the loc of <Button item = "BW_Buff1_Button"> to:

<X>48</X>
<Y>3</Y>

Also make sure the size on <Label item ="BW_Buff0"> are the same for 0 thru 14.

And another item(not an issue but to be thorough) Change

<DrawTemplate>WDT_Rounded</DrawTemplate>

To

<DrawTemplate>WDT_RoundedNoTitle</DrawTemplate>


The bug you're having while zoning is a result of Buttons 0 and 1 being coded with the same Loc.
