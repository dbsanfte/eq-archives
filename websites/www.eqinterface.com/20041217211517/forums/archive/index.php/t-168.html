Ok, to actually get new graphics to load they either need to replace and existing file, and be the same size, or there must be the proper tags in the files you have, designed to load the graphics.

The minimun number of files necessary to change to load the graphics are 2.

You must edit the EQUI_Animations.xml file to include the new graphic as a texture.

you must then edit the file for the window where you want the background to appear.
