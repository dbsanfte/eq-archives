Hi, just got your message. This problem is very rare but has been reported before, and is likely related to your browser. When you select the window from the drop-down list, the window name should be associated with the value of that form field, which the browser should then send to the server as part of the request. The name sent is then used during timer file generation. In your case, however, the name is somehow not sent/received so the generator is using "undefined" as the value. I'm not sure what causes some people to experience this problem, but here are a few things to try:

1) Try downloading the timer using Internet Explorer (5.5 or 6.0) if possible, if you're currently using a different browser.

2) If you have access to a different computer, try downloading it from that one, chances are it will work.

3) If all else fails, post or send me your desired timer configuration, and I'll be more than happy to generate the zip file for you.

Hope this helps. If there is any additional information (your browser version, etc.) you think might help pinpoint the problem, please post it here or send it to me. Thanks.
