You should delete the file you downloaded, and try downloading it again. It sounds as if the file got corrupted during download.

This is a fact of internet life, and happens from time to time.

If this happens consistently with only this one file, then there *may* be a problem with the file as stored on the site. If it happens to you often, with many files, there is something wrong on your end.

The laundry list of possible issues in that case is long, and I won't go into it here. Suffice to say that deleting the old copy and downloading a new copy should solve the problem.
