If you don't want the box to be any bigger, but you want it in the center of the target window...
Make a new graphic (.tga) that is the same size as your target window. Make the whole graphic transperant in the alpha channel... all except for the box where you want the color to show. If the graphic is the same size as the window, you can put your visible box anywhere you want in the window... and the graphic will still start at 0,0.

hope this makes sense.
