The deal with the Charm slot is that it is a graphic added to the window_pieces04.tga file. To get the graphic, you either need to delete the window_pieces04.tga file (simplest, but will make your compass the default one) or use the window_pieces04.tga file in this thread :

http://www.eqgui.com/showthread.php?s=&threadid=6440


As far as the Book goes, this zip file has two different book files. The one that unzips first uses a different book look. You will need to rename the spellbook01.tga to book01.tga through book04.tga of the T King spellbook you want to use for your book.

The second one that is included uses the same T King spellbook as the book as well.
