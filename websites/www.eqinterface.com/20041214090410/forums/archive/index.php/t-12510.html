<img src="http://www.eqinterface.com/images/sections/eqpatch.gif" align="right">10 February 2004 – Patch Day and lots of excitement. Gates of Discord goes Live today with a new playable class, lots of new AAs, more zones and the new TradeSkill interface! You can read the complete patch message here (http://www.eqgui.com/showthread.php?s=&threadid=12505).

Patch day also almost invariably means changes to the UI as well and today is no different. As always, I’ve done up a thread outlining the changes to the UI which can be found in the Patch Fixes (http://www.eqgui.com/forumdisplay.php?s=&forumid=36) forum.
