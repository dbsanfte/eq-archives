1) Enoks UI Sleek - Features:

- Target Window w/ Con Colored Target Gauge
- Compass w/ Loc Indicators (green = increasing, red = decreasing)
- Informative Player Window w/ Tick Timer and Sphere Gauge
- Divided Up Group Window (click on Pet: ###% to target pet)
- Buff Window w/ Color Coding and Spell Names (also Song Window)
- Container Window w/ Border but Moveable Only By Container Name
- Netstat Selector Window w/ Simulated Netstat Animation
- Compressed Hot Key Window w/ "Paranoid Anti Item Pickup"
- Minor Tweaks To Gauge Colors (and other things) In Other Windows

2) Enoks UI Sleek - Instructions:

- Installation
Extract files to a subfolder of the EverQuest\uifiles folder
I suggest naming the folder enoktkek
In game type /load enoktkek (yes, don't add the 1/true)
Reposition windows to your liking

3) Enoks UI Sleek - Important Note:

Most windows are hard to move. You can only drag them with one
pixel of the far left or right edge. This takes some minor
practice. The chat window is also hard to move, read the FAQ to
learn more about it.

4) Enoks UI Sleek - Frequently Asked Questions:

Q: Why is the title for my chat windows so messed up looking?
A: It just is, rename the chat window title to . (period) to fix

Q: Why do the experience and AA experience gauges in the player
window have a weird blue/pink bar below them?
A: That bar represents one fifth of one fifth of the total gauge;
basically, one whole blue/pink gauge represents one blue/lblue
experience bubble the same way that one whole blue gauge
represents one orange bubble of experience.

Q: How does the player window sphere gauge color coding work?
A: Green = Good, Yellow = Okay, Flashing Red = ACK!

Q: How does the group, target and pet gauge color coding work?
A: Green = 80-100%
Orange = 60-80%
Yellow = 40-60%
Red = 20-40%
Flashing Red = 0-20%
Dark Red = In Another Zone

Q: Why does the compass have black thingies on it?
A: I put the black thingies there to help me center myself on
cardinal directions (so I could walk in straight lines); if you
figure out how to use them to line yourself up, good

Q: Why can't I left-click to pick up items I have hotkey'd?
A: Because "Paranoid Anti Item Pickup" doesn't let you

Q: What is "Paranoid Anti Item Pickup"?
A: Something so items can be hotkey'd but not accidently picked up

Q: Why does your FAQ section suck so bad?
A: /shrug
