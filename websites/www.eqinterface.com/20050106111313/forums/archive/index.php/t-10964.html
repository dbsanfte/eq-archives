The default folder that is... you're just asking for trouble. What you should have on your hard drive is a direrctory "C:\Program Files\Everquest\uifiles". First part of that may vary, but the "Everquest\uifiles" should be the same regardless of any other variations. In any case, in the "uifiles" directory there should be a "default" directory. As I said.. DO NOT TOUCH this. If you want a custom ui, it should have it's own directory withing the "uifiles" folder... For example, the complete path would be"C"\Program Files\Everquest\uifiles\mycustomui", assuming mycustomui is the name of your ui. Then, you should be able to use the /loadskin command from within EQ to load up your custom ui.
