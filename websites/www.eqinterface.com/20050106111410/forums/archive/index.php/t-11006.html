You can adjust where the color appears by editing the TargetBox.tga file and creating a custom alpha mask that works for your target window.

Just know that when the game draws the color portion it uses the file specified in the:
<TextureInfo item = "TargetBox.tga">
<Size>
<CX>128</CX>
<CY>32</CY>
</Size>
</TextureInfo>


I was able to edit TargetBox.tga and make it big enough to cover my target window then create an alpha mask to only show the color around the target health gauge. Then I just adjusted the size info in the TextureInfo section to match my new graphic.

I do agree that it would be nice if this could be used in other windows though. I know a lot of the UI's out there have moved the target info to different windows and this will force them to either re-write or just not have the new functions.
