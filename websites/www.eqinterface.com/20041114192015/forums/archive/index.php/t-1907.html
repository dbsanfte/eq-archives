The container window resizes itself based on the location of objects in the window. But objects that are autostretch instead of at a fixed location don't count. So you can switch the done and combine buttons to autostretch to have more control. Make them a certain distance above the bottom of the window, similar to how the chat input window in the chat window works.

I've put a few container mods up here, you might want to look at them.

I'm not sure what you mean by the chatbox line. If you mean the border around the chat input line there is no way to turn it off. But you can make a new window draw template that doesn't have a border. Then change the chat input box from type WDT_Inner to your new one. I've posted a modified chat window that does that, you could look for mine.

If you mean sizable , such as to make the chat input window taller, I don't think that is possible. The chat input is only a single line of text.
