The only thing that would come close to being described as a "UI Maker" that I know of is something called Sidwidl, but it and its creator have apparently vanished. Also, it doesn't really "make" the interface - only serves as a tool (very buggy one since only an alpha version exists) to edit the components for an existing interface. It can't 'create' something from nothing.

This has been covered in the proper forum, but the only way you can create a UI currently is graphic design and a lot of XML coding. Pure 'mental labor', you could say.
