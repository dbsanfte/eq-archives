12-02-2002, 08:50 AM
Yes it is possible to put in a dofferent buff window, but you have to make the right changes to the EQUI_Animations file. You can usually puzzle this out by looking at what files are in the buff window you want. For example, whit Geddines window, you'll see the file buff.bmp. This is the graphic for the red and blue bars in the buff window. What I usually would do is open the Razor's animations file in note pad, then open geddine's animations file in another notepad. now in the one from Geddines, go to the edit menu, click find, and type in buff. It will bring you to the part of the code where the animations file looks for the buff.bmp image, which is under the heading "Blueiconbackground" and "Rediconbackground". Now on the Razor's animations, click edit, and find, and type in "rediconbackground" (or whichever on it is that comes first) and it will take you to the same part of the animations file. Now all that's left is to change this section of the razor's file to match geddines. You should have to change it so it looks for buff.bmp, and the x,y locations. click save in the file menu, and last but no least, copy the buff.bmp and EQUI_Buffwindow into your Razor's file. Make sure you don't copy over the animations file, or you'll put geddines over the work you just did, and nothing will load! If you did everything right, now you'll have a new buff window, which you can place wherever you like. :)
Good luck.

if you not sure which key if any is mapped to open you spell book you can find out and edit it if you like. Here are some simple directions:

In the window selector toolbar you can click on "options" looks like a nut, or press alt+o

click the keyboard tab.

in the "select category" box, use the drop down and choose "UI"

look down about 4 lines to Toggle spellbook window, it has two keypress options "Keypress" and "Alternate", whatever they say are the keys or combination of keys you need to toggle this window. If you wish to change it, click on either one and then press the desired key or combination of keys you would like to use.

*note* if you open the spell book while sitting, you will not automatically stand by closing the book, but if you open the book standing, you will stand when you close it. Also choosing to stand will close the book and stand. You can do this by using the sit/stand button or typing "/sit off"

hope this was helpful
