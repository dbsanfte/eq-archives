Join serverwide eqgui chat by typing

/join serverwide.eqgui
---------------------------------------------------------------------
if this is the only chat channel you have joined, then to speak in it,
type

/1 hello everyone
---------------------------------------------------------------------
To leave this channel type
/leave 1

To Leave all channels type
/leave all
---------------------------------------------------------------------
If you have joined more then one chat channel and dont know which channel number belongs to which then type

/list
---------------------------------------------------------------------
To find out the names of people in chat channel 1 type

/list 1


Hope this helps
