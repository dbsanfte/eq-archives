I had this same problem...In your UI directory there are TargetIndicator#.TGA files. These #'s need to be numbered 0 - whatever...mine are 0-7 now, used to be 1-8.

If you still have a problem with it, then in the TargetIndicator.ini file there is a variable for number of frames for your target ring, reduce that number by 1 and you should be ok to go.
