I'm not sure if this is the correct place to post this, but here is a fix for those getting a blank screen after logging in when using a custom load screen. This file was patched on Sat. Oct 2nd 2004.

Solution:
open eqlsplayerdata.ini in your EQ folder
change:
HasViewedWebOrder1=false
to:
HasViewedWebOrder1=true

This was true as of yesterday, but the problem is back again as of Oct. 4th 2004 eqmain.dll patch. The issue still however has to do with custom load screens and the "buy SOE stuff" window. If you want to get back in, just change your eqlsplayerdata.ini.file. CAUTION: This is as of 10/4!!! and may need to be changed again next time Sony wants us to buy stuff or releases some news.

[MISC]
HasReadSWarning=true
OmensFirstOffer=1084929848
OmensLastOffer=1095133933
HasViewedWebOrder1=true
WebOrder1FirstOffer=1096762630
WebOrder1LastOffer=1096851559

[UI]
XMLFilename=EQLSUI_custom.xml

[NEWS]
LastViewed=-814016356

If you want to avoid this all together you can revert your eqlsplayerdata.ini file to the default.This will no longer use your custom loadscreen, but will get you in game.

Change:
[UI]
XMLFilename=EQLSUI_custom.xml
to
[UI]
XMLFilename=EQLSUI.xml

and change this back to:

HasViewedWebOrder1=true
to
HasViewedWebOrder1=false

I got back in with this, so hopefully it will help get you back in as well.
