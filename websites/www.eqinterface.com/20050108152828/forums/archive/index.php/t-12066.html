the following file affects how all the chat /tell windows look.

EQUI_ChatWindow.xml

One of the things I do when making custom UI's is I remove the border, but this prevents me from changing size or location and I can't right click on it to modify it's properties. This may be why you are having the problem you are having.

Two things you can do to check/fix this is
1: rename the above file in your cusotm UI directory by putting an x in fornt of the name (xEQUI_ChatWindow.xml) this will make EverQuest look for the default chat window .xml file.

2: edit the above file. If it contains this line:
<Style_Border>false</Style_Border>
make it read like this:
<Style_Border>true</Style_Border>

make the changes you want, size, location additional windows, etc. then return the file to it's original name or chaneg the border back to false.

best way I would think to do this is to make the changes to the file before you log in, log on, adjust your interface accordingly, go into windowed mode, adjust the files back. enter game then /loadskin. Remeber to use a 1 after you loadskin o your locations will all go to default :P
