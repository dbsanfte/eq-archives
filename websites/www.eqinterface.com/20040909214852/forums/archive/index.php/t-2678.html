Originally posted by Molz
i like but I would like some kind of background one that i could fade if want is that possible? the names get lost a bit :(

There are two ways you can go about this.

The first is to open up the EQUI_BuffWindow.xml file in your custom directory. Scroll to the bottom of the document and find the line that says:

<Style_Transparent>true</Style_Transparent>

Change it to:

<Style_Transparent>false</Style_Transparent>

It will now use the background that you have on the rest of your windows (like chat windows, etc), and you can set the alpha channels (normal and faded) where you want them.

The second option is the optional one I've just made, that has the buff background colour (red or blue) behind the text:

http://members.rogers.com/khobson/images/buffwindow_optional.jpg
