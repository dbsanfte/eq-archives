I have had simaliar problems in the past. My fixes were:

Deleted all ini files
Ran docmem and found I have a memory leak Can be found HERE (http://www.simmtester.com/PAGE/products/doc/docinfo.asp) I had this problem after installing new ram. After taking the ram back the technitions at the computer store just pluged it into a mobo and said it was fine. Maybe this is what your computer store did. The leak would only happen when precise data was written in the right order to the ram.
Added a cooling fan(this only helped when the computer had been used intensivly for a while before playing)
Uninstalled Lucin graphics
Deleted all files associated with the zone the char I was trying to log on was in.


I'm sure you have tried some of these, but maybe someone else can come up with ideas.
