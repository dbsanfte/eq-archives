So let pretent you down load fubar.zip.

Open the Zip file.

Extract it to a folder named "fubar" in that folder you will see a lot of files some with "tga" and some with "xml" extention.

Move the folder to to ....\everquest\]uifiles

Now in the uifiles you should have two folders:
default
fubar

Run EQ; login into a char, type /load

You should see two items in your list which will match the two folders you have in uifiles

Select fubar, click load. If it does not load, the UI may not be compatable with LDoN. Other than that, it should work :-)
