Despite the name, the KTn-Fixed uis have NOT been fixed to work with LDoN. To my knowledge, no official fix has been made. But here is what I did to force it to work. The problem is with the custom EQUI_Animations.xml file that KTn-fixed uses. It is NOT compatible with LDoN. I simply renamed the file in the KTn-folder (which forces eq to use the file from the default folder). Doing this, I got an error message when EQ first loads up, but everything otherwise looks and works fine.
