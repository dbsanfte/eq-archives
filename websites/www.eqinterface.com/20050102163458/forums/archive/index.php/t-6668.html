Okay, first, they do all need to be in the same folder. Install one, go in game, make sure it works. Then install the next, go in game, make sure that they both work. If they do, make a backup folder with just them in it. Now install the third, go in game, make sure they all work. If they do, update your backup folder. Now the next one. Rinse and repeat until you have them all installed and working together.

When you come to the one that breaks it, go to your main EQ directory, look for the UIError.txt log file and post it here. Also let us know which mod(s) you are using.
