I'm not sure how you would add a gif animation to a window, but if you are wanting to use it as the BACKGOUND for a window, it may be fairly simple.

Try making a custom draw template (WDT) for that window and where the <Background>wnd_bg_light(or dark)_rock.tga</Background> tags are, plug in the file name of your animated gif in place of the normal background tga.

This is asuming you matched the X/Y size of your gif to match the size you want for the window.

I'm not sure your gif animation will work that way, but its easy to try at least.
