Okay, first, it sounds like you were making changes to your *default* EQUI.xml file. Don't.

You (normally) never make any changes to anything within the default folder. Last patch was a bit of an oddity, in that not all files required by *some* were being downloaded from the patcher, so I provided those files for folks to add to their default folder. There were no files to be *changed* in the default folder, only 3 files to be *added* Nontheless, this is *not* typical.

Any changes to files I list (ala "IF you have a custom EQUI_blahblahblah.xml file", etc.) is referring to if you have those files in your CUSTOM folder.

Next, where did you put the slide timer you downloaded?
