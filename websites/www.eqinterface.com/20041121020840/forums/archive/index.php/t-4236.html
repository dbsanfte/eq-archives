1. Right click the title bar or text entry section of the main chat window
2. Choose new chat window(or something like it) from the popup window
New window should appear
3. Right click in new window like first.
4. Towards bottom of popup select flyout menu labeled Filters
5. and then the melee flyout
6. choose My Hits
Now all of your hits will be displayed in that window you should get a message to that effect
7. right click the second window again. at the bottom select the Hits Mode flyout, then My Hits, then Number Only.

now your hits should be displayed in the new window only as numbers.

if you want to change the name of the window in the boarder
1. right click and choose change window name
2. enter your new window name. and click change or ok or whatever it sez.

good luck
