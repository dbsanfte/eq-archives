well not sure if your aware of this feature but you can save a spell gem setup by right clicking on the open spellbook button.

You basically set up the spells you want then save it with a name, then if you die or want to change back to the way you had the gems set up you just right-click on the book and load the set. It sits you down and mems the spells in that set without delay (except for the little mem bar)
