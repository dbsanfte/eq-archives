To create a digital 6-minute timer, you can go to Customize > Dial/Digital Timer from the timer home page. You could select the following options, for example:

1) don't include tick dial
2) include digital timer
3) duration 6 minutes (count up, or down if preferred)
4) don't include title bar
5) window: pick whatever you want to use
6) make window transparent (in advanced section, scroll down)

This would create a transparent 6-minute digital timer that continually counts each second. I personally would still prefer using the slider as it lets you mark when you used mend, but you can put this in a different window (or merge the XML, if you know how) if you want to use both.

While it is possible to put a tiny digital timer in its place, there would be no way to start/stop/pause the digital timer, so it would be somewhat pointless (for example, it would always read 0:00:00 when the bar is empty, and reach 0:06:00 when the bar is full).

You'll have to keep in mind that UI mods are not "programs", and are subject to many limitations. They can only alter the appearance of the UI, and cannot supply any additional functionality. Please read my post entitled "impossible enhancements" for more detailed information:
