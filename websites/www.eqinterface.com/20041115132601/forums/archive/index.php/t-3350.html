Sorry it took so long- finally bug-free, tested on 3 different computers, with and without PoP.

BlueFantasy is an attempt to give a strong fantasy feel to Everquest by replacing the modern-feeling new UI with a more elaborately decorative and roleplay-appropriate set of graphics. It is also designed to maintain and improve upon the functionality of the new UI, and to provide all possible 'bells and whistles'.

It's strongly geared to casters, especially those who use pets at least from time to time, but there's no reason you can't replace the pet window and remove the spellbar if you're a melee class. Upcoming releases will include a druid/ranger specific version and a melee version- but there's no reason any class wouldn't be able to use this UI right now.

Credits:
buff window by Cairenn, loc wizard by Atheenah, blue compass by Sylverpaw, gauges by Mynlaeni, spell icons, gem icons, and spellbook by t.king with modifications by Remelio. Group window is by Cairenn for KaoUI with some of my own modifications; inventory by t.king, shrunk by me. All other windows by me and/or Sony.

BlueFantasy is designed to run in 1280x1024; it won't fit in anything smaller, as the hotbutton window is 1280 pixels wide to accomodate your hotbuttons, clickies, general inventory slots, xp and aaxp bars, and your target. You should set your viewport to 0 0 1280 761. (this should reduce lag for those who can't run at high resolutions- and the windows look best opaque anyway, so you wouldn't be able to see what's behind them even if you didn't /viewport)

11-02-2002, 02:57 PM
http://www.amaprotu.net/eqtypes.html

go to the file in your custom directory labeled EQUI_HotButtonWnd.xml and open it with WordPad.

Search For InvSlot34,35,40,41,42,43, or 44. These are your rightclick boxes, from left to right; find whichever one you want to change.

the code should look something like this:

<InvSlot item="InvSlot34">
<ScreenID>InvSlot34</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>374</X>
<Y>0</Y>
</Location>
<Size>
<CX>30</CX>
<CY>30</CY>
</Size>
<Background>A_InvEar</Background>
<EQType>1</EQType>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
</InvSlot>

go to the above link and find the EQtype for the slot you want to make available. In the code find the line <EQType>number</EQType> (in bold above) and change the number to the one for the slot you want.

You'll probably also want to change the background for the current slot to the one for the slot you want. The backgrounds for inventory slots are named A_InvEar, A_InvNeck, A_InvHead, A_InvFace, A_InvChest, A_InvAboutBody (think that's the back), A_InvArms, A_InvShoulders, A_InvWrist, A_InvWaist, A_InvHands, A_InvRing, A_InvLegs, A_InvFeet, A_InvPrimary, A_InvSecondary, A_InvRange, and A_InvAmmo.
