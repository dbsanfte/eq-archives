The error-message has it all.

On line 1890 of your EQUI_PlayerWindow.XML file you have an error in the <Label> tag set. Specifically the </Label> (the tag that closes the tag set) doesn't match.

I find that when I get this error, I've usually forgotten to delete a line, or I've cut-&-pasted something into the wrong spot. You'll have to look at your file and the line in question and see if you have the close tag in the wrong order or what.

If you edit the file in NOTEPAD, under the Format menu, turn off word wrap. Then the bottom of the window will tell you which line you're on.

If you're using something else to edit your XML, most editors have a feature for showing you the line you're on.
