This is my first shot at a full UI skin, I did it with a few things in mind.
http://www.shadowedsoul.com/forums/gallery/data/500/96lil_ui1-thumb.JPG (http://www.shadowedsoul.com/forums/gallery/data/500/96lil_ui1.JPG)
1. Patch compatibility, EQUI.xml and EQUI_Animations.xml files are not changed in anyway so this should be a very stable UI
2. Low graphics overhead, very few custom graphics only added where it increases functionality
3. Compatable with all windows designed to work with the default interface, any mix and match piece that will work with default will work seamlessly with this.
4. Pleasant to look at :)

There's room for 5 clickies plus access to all 8 inventory slots in the hot button window and the Health of Target Target window is built in to the standard target window.

This UI is designed to be run at 1280 x 1024 and the included ini file is set up for that, but it should be able to run at any resolution over 800 x 600 just fine.

Please enjoy it and post any comments that you might have...

There are 5 parts to it. Each part begins and ends with a <InvSlot> tag. You need to change 2 parts of each one that you want to modify to how a different item. The Background part and the EQType.

Let's say you wanted to change the last one currently showing the Range item by default to another slot. For instance you'd like it to show your glove slot instead. Here is what you would do. Open up in notepad the EQUI_HotButtonWnd.xml file from your uifiles folder and make the following changes:

Change the line that currently reads:

<Background>A_InvRange</Background>


To read:

<Background>A_InvHands</Background>


Change the line that currently reads:

<EQType>11</EQType>


To read:

<EQType>12</EQType>


And save the file and reload your UI.
