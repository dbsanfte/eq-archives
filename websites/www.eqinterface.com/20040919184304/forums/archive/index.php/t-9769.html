08-31-2003, 09:53 PM
The problem is Solaria uses 512x512 pixel texture files... many older video cards do not correctly use textures over 256x256. They chop out the first 256x256 pixels then cut out slices from the texture as if it were still 512x512 (hence the reason the messed up areas are double the size and way messed up). This is why all default UI tga files are 256x256 in size. To fix this, Solaria's textures would have to be cut up and it'd be a lot of work.
