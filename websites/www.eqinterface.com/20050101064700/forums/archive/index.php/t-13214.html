In your UI folder you will have files named something like TargetIndicator.tga, TargetIndicator1.tga, TargetIndicator2.tga, etc.

Rename the one that is TargetIndicator.tga to TargetIndicator0.tga. That's a zero. Should work fine then.

Apparently Sony changed the way the target ring function looks for animation frames for the ring. No biggie, worked on my ring.
