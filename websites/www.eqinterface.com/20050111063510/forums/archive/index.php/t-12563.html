Cairenn's patch fixes can not always be followed word-for-word when patching a custom UI.

There are whole sections that are called to be deleted form point a to point b. In your custom file, there may be code added inbetween those two points that must not be deleted.

When I patch my UI I use Cairenns post as reference while I compare my old file with the new default and make the changes only after being sure that my code matches hers.

It is likely that those gauge pieces were deleted in one fo the blocks of delete form a to be and replace with...
