It's probably what Caleal said:
Open UI_ToonName_##.ini: (Example: UI_Hawkren_91.ini)
Look for:

[SkillsSelectWindow]
XPos1024x768=702 <--- Change all these XPos to 0
YPos1024x768=585 <--- Change all these YPos to 0
Width=200
Height=240
BGTint.red=255
BGTint.green=255
BGTint.blue=255
Fades=true
Delay=2000
Duration=500
Alpha=255
FadeToAlpha=128
Locked=false
XPos1280x1024=1040 <--- Change all these XPos to 0
YPos1280x1024=784 <--- Change all these YPos to 0
XPos1280x960=1040 <--- Change all these XPos to 0
YPos1280x960=720 <--- Change all these YPos to 0

Edit:
Do this before before you enter EQ..
