The best way I've seen to get this to work is to make a new folder in your UI directory and copy the files for the character creation screen into it. In your eqclient.ini file, there is a line that says UISkin=Default; change it to the name of the new folder. When you enter, it will show the character select screen by default. But you'll need to put the same files in your character's UI folder as well, or when you log out it will be back to default.
