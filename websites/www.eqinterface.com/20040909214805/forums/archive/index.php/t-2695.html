For modders out there that would like to use my animation splicer, here are some notes

* You must put the changes FIRST and additions LAST
* If you want comments included, put them inside the items instead of outside.
* If you want to add textures but no animations, or animations but no textures, do not delete the appropriate file, instead use a single comment line in the file
* There should be no trailing lines - the last line of your .txt file should be the last line of code - if you have empty lines at the bottom (even one) the program will crash.
* Be sure to put changes in the proper order

I think thats about it for now - please use this thread for feedback :) Hopefully this will make modding easier for those mods that need animation.xml changes.

I plan to go back and make the program a bit less picky sometime :)

Feedback GREATLY appreciated - let me know what you think.
