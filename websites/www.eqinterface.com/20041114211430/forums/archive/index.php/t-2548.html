* Documentation
Taken from my description in the downloads area:

A true mix of Vertebrae, T.King and Remelio, Drakah

I began working on this version (previous ones were only private works) when T.King and Remelio released their Standard Stone inventory. I originally started with Vertebrae's UI v2.2 as a frame. For this version, I added pieces from T.King/Remelio's works, Drakah's UI Mix v1.0a, and some of my own art and coding. They all deserve some credit for this.

This UI also comes with T.King's Standard spellbook (seamless version). His current spell buff icon files (v1.04) were too large to include in this zip file. You can find them here: http://pub10.ezboard.com/fmaximus94788frm10.showMessage?topicID=74.topic

Special thanks to Gendal and his excellent SidlWidl program (Everquest SIDL [xml] editor). It is unfinished and currently in its v0.2.9 Alpha stage, but still very helpful. You can find more info about SidlWidl at http://www.sidlwidl.com

* Important Notes
* The class animation in the EQUI_ActionsWindow.xml file included with the UI is set to 07 (Monk). If you aren't one: delete it. Then, open the /Alternate/ folder that comes with the UI. You'll find your class's file. Just re-name it to EQUI_ActionsWindow.xml and move it out of the folder. You'll also find one without a class animation if you prefer that.
* The long compass bar that is found at the bottom of the screen in the screenshot may not work in resolutions lower than 1280x1024. If you run the game in a lower resolution, I recommend using the EQ_CompassWnd-Small.xml located in the /Alternate/ folder. Just re-name it to EQ_CompassWnd.xml and move it out of the folder. You can preview this window at the top of the screenshot.
* I have included two optional setups in the /Alternate/ folder. The one from the screenshot is the caster setup. The melee setup just moves things around a bit to occupy the space that is left open from not having the spell casting window open. If you use mana, I suggest using the caster setup because the only mana gauge is located in the spell casting window. If you want to use one of these setups, rename the file to UI_Yourname_XX.ini where XX is your server number, and move it to your /Everquest/ folder. *Important: These setups are only for 1280x1024, they will not work otherwise*

* Edits
* Forgot to mention something about the hotbutton window. It might not be compatible with CCake, (the program that comes with Vert's UI) I havent tried. The slots for the two tabs on the right must be manually changed. A hotbutton window without extra tabs is available for download -> see October 1st news.
* Also, the friends window is in fact the pet info window. This is not accidental. I moved all the pet hotbuttons and the gauge to the friends window because of Arista's timers and other uses for the pet window that require tabs. Since the friends window is not compatible with tabs, you can now use her timers in the pet info window and still see pet stuff.

Have fun

Copy over your mouse.tga file from your default folder. Go into EQUI_Kaori.xml and change these lines:

<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>mouse.tga</Texture>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>22</CX>
<CY>22</CY>
</Size>
<Hotspot>
<X>4</X>
<Y>1</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>

Then add these lines:


<TextureInfo item="mouse.tga">
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
</TextureInfo>


Let me know if this doesn't work

To change the fill colors in the player window, open the EQUI_PlayerWindow.xml file in wordpad and go to 'Edit' 'Find...' and type in 'Player_HP'. It should warp you to the part of the code that defines the gauge. Scroll until you see:
<FillTint>
<R>200</R>
<G>0</G>
<B>0</B>
</FillTint>
Change the 200, 0, 0 all to 255's. To change the fill colors in the group window, open the EQUI_GroupWindow.xml file and repeat the process, except find 'GW_Gauge1', 'GW_Gauge2', 'GW_Gauge3', 'GW_Gauge4', and 'GW_Gauge5'. All the FillTints of these gauges should now be pure white.

Since there seem to have been so many problems with my mod ...

IF you are using my smaller mods of this UI:

1) Make a new subfolder within your custom folder, named anything you want.
2) Move ALL .xml files in your custom folder into it.
3) Download the attached zip file and extract into your custom folder.
4) Any other custom files (example a timer mod, map mod, etc) you had and want included, move back out of the subfolder you created in step 1 back into the custom folder.

There are two additional subfolders:

One is the ActionWindows folder. In it, you will find the EQUI_ActionWindow_CLASS.xml files for each class. Pick yours, rename it EQUI_ActionWindow.xml (take off the _CLASS part). Copy it into your custom directory.

The other additonal subfolder has two optional files in it. One is a Small Compass that is *just* the compass, without the long bar that has all the stats and whatnot on it. The second is a smaller Player & Group combo Window that does NOT include the stamina/casting, exp and aaexp bars.

This download does NOT include any .tga files other than those which are absolutely necessary for this to work. No spellbook, spellcastingbar or buff icons. No backgrounds. No custom spell book. It does, however, include a T.King Inventory, as that is required for the inventory tabs I have on the two tabbed windows (any T.King inventory can be subsituted for the one I've included).
