Finally after allmost 3 month of abcence from the Internet I have a new conection and can bring an updated Stone Angel to you. :o)

This is a *real* Beta release. Most of the code was written blindly (meaning i couldn't log on to EQ to test stuff.) and it's just roughtly tested. Don't expect everything to work as intendet. It does not include a .ini file, which means you will have to move to objects in place yourself. The Sizes are designed for 1280x960 and it probably won't run in lower resolutions then this because of the window size.

One warning, don't expect me to update all errors within the next few days. Im lagging behind my Guild ALOT and probably won't have much time to code.

Features:

- All files are independant of each other, including the Animation and Template file. I did this to make the UI compatible with other mods if people wish to only use some parts. It also makes it very patch friendly.
- A complete new button design for Hotbuttons and Pet Buttons.
- maybe more, I dunno.

IMPORTANT NOTICE:

Because of the file size and the fact that I don't have webspace atm I had to split the file in two halves. You have to download both for the skin to run completly.
