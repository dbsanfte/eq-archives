Yes it is possible. I have all the class animations installed in the UI I'm currently using. You don't need to edit your animations file by hand. Just download the animations from here (http://eqinterface.com/sdownload/index.php?s=&op=showfiles&catid=27&sortby=date&sortorder=DESC) and run the installers, making sure to select your UI folder when appropriate *and* telling the program to let remmy.exe run afterwards (that's what sticks the info in your animations file). I would suggest making a backup of your animations file (if you have a custom one) just to be safe.
