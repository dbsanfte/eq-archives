The statement about the target ring only loading once isn't totally true...If you are in game and have a target ring, but you want a different ring, you can make it change while in game. Here is the sequence that would be followed:

1. You're in game and have UI1 loaded and running...it has TargetRing1 loaded and you see that in game

2. You want to load UI2 that has a different target ring (TargetRing2) ... do the following command to change interface: /loadskin <UI2> 1

3. The interface has now changed, but the target ring is still the same!!! Now you have to type in the following command (this needs to be typed in... clicking the button will not work!): /indicator off

4. You will lose your target ring until you type in the following command: /indicator on

5. Target something and your new target ring should be visible!


Step 2 above is not actually needed. If you are in game and have a target ring loaded, then actually change the TargetIndicator.tga file that is located in your current UI folder, you can skip to step 3 without doing step 2.

EDIT: I just went in game and verified the above information to be true. This should work. If it doesn't, then there is something else wrong with the indicator.
