Hiyas . Since verant added the song buff window selector icons to windowspieces04 noone can use thier custom compasses ......
SOOOOOO



If you change your custom photo windowspieces04 to read windowspieces041 and go all the way to the end of your animations.xml file you see where it says to use windowspieces04 for your compass just change the number to read 041 and you can still use your custom compass.

Don't forget you also have to add this to the animations file to tell it that windowpieces041 exists

<TextureInfo item = "window_pieces041.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>
