[Line:10 Source:UIFiles\necro\EQUI_CastSpellWnd.xml] Schema error - Duplicate item TKingspellbar.tga

Translation:
On Line 10 of the EQUI_CastSpellWnd.xml, the item: Tkingspellbar.tga, is being defined (most likely as a new texture) and has already been defined either in another file (EQUI_Animations.xml perhaps) or just before this entry and Should not be defined a second time so like a tempermental UI, I will now close down.

Suggestion:
1) Make a backup of the file.
2) Open the file and locate Line TEN. If it is within a definition (TEXTURE) delete the whole section <Texture> thru to </Texture>
3) Save it and load EQ again.
4) Check for other errors if they come about.
