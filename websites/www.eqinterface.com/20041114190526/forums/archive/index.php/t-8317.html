Updated to work with May 14 Patches

Updated KatnTKBriar to KatnTKBriar-a..
Added separate folders for resolutions 1024 by 768, 1152 by 864 and 1280 by 1024..

You can download it at http://www.eqinterface.com/sdownload/fileinfo.php?id=1989

this is for Druids/Rangers and is using all TKing art...

http://www.eqguild.net/guilds/FRC/Uistuff/briar.jpg

All Art work in this UI is using TK/Rem art.

Included 2 alternate folders..

Ranger folder: Changes class animation on hotbuttons to the Ranger class animation.

Petinfo: Adds pet information to the clickies..for those druids that like to use there Bear pets.

Modifications:

Target window - added percentages to the target window
Player window - Added percentages, player stats, and hp..also made it longer and slimmer to fit at the base
Pet window - Added inventory slots
Hotbutton - player resists, and class animations
MP3 player - made it slimmer and longer to fit at the top of the UI.
groupwindow - added percentages, and backpack slots
Buff and short duration buff windows, added spell names..and made the icons smaller.

UI includes-
TK Briar spell book, forest inventory, Druid and Ranger class animations, G2 spell icons, and Tking compass/gauges.

Aslo includes Tking Arrow Pointer

TK Briar Spell Bar

Arcane Casting Gauge


**other graphics in this UI are made by TK/Rem I just basically cut and paste peices together.**

I'm not sure it will work because the pieces where made for 1152 width resolution but I'm sure someone out there can give it a go

what you do is type /viewport

You'll see like little directions come up

it first tells you what your at..for me I am at 1152 by 864 resolution so it would say

x=0 y=0 w=1152 h=864

w is width
h is hight

Y is for how much you want black in the top

X is how centered you want the screen

so say like you type /viewport 0 60 1152 600..your going to make it have some black on the top and black on the bottom..

if you change the 1152 witch is your width, to liek 800..it would make the look smaller

Its basically you gotta play around with it..hehe I'm not to great at explaining these things.
