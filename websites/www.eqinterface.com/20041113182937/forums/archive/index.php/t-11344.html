As i have been working with the Quartz UI alot and helped countless friends install the UI, i have come across a series of problems that kept comming up at every install.


Lack up basic Windows understanding
Lack of UI understanding


I've decided to make the UI install itself... I 've basically created a program that will take information about your character and install the UI in the appropriate place for you. This way, no mess, no fuss, just enter the required info and let the program install itself.

Download Here (http://www.darksights.ca/modules.php?op=modload&name=Downloads&file=index&req=viewdownload&cid=1)

NOTE: The program is written in C# and requires MS .Net Framework in order to work once installed. You can download the Framework at Microsoft (http://www.windowsupdate.com)


The installer is still in Beta, i would appreciate people from every server to relay on this thread the server name required at the end of their UI_<CharacterName>_<ServerName>.ini file.
