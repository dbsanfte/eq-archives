nope - currently restricted to 5 lines of code
there are a number of restrictions that go with this as well. Once a macro is started - there no way to stop it other than leaving game. ZONING does NOT stop a macro.

macros with adjoinging lines containg messages to the same chat channel may be accidently filtered as program confuses the

1: /ooc Hi all
2: /ooc Remember me?
3: /ooc Test line 3
4: /ooc Test line 4
5: /ooc just doing some silly stuff

in this example - lines 2 and 4 would usually be filtered out. However, the echo channel that you see DOES NOT filter it out and shows all 5 lines. This can be pretty confusing. If you're not sure what I mean - just work with a friend to monitor your output.
