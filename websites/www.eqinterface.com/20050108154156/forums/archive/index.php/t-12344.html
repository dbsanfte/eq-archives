I looked at the Solaria Cartography Map (http://www.eqinterface.com/sdownload/index.php?s=&op=showfiles&catid=56&sortby=date&sortorder=DESC) and it and Mortis Player Window (http://www.eqinterface.com/sdownload/fileinfo.php?s=&id=1589) can both be used together without an conflicts.

As far as mixing and matching differnt Ui components, there is no one answer as to how to do it.

Not every coder works the same.

Pick out the mods that you like, and start adding them one by one until you get an error. Then check your UIErrors.txt file and see what the conflict is. If you can't figure out how to modify the xml to resolve the conflict, there are many experienced modders here that are happy to help. Just make sure that you have read all the FAQs and READ MEs to be able to post the informatin we need to help you.
