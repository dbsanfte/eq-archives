I don't know the particular mods you are working with, but right now when people distribute "mods" a lot of the time they package up the whole skin directory. So if you unpack the skin directory into uifiles it will go into a new skin and you'd have to choose between the two.

Integrating the two skins is a simple process unless one or both have changed some of the shared files. In this case, you need to hand-weave the two together. If this is XML, that's not hard. If this is a tga file, it's a bit harder *grin*

I think once mod writers realize they don't have to include files that they didn't change because EQ falls back on the defaults in that case that we'll see mods that you can just drop on top of each other. But there's still the issue of the shared files like EQUI_Templates and EQUI_Animations that would need to be hand merged unless someone writes a mod installer that can read two XMLs and merge them.
