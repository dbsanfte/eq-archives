06-05-2004, 08:53 AM
I for one would love to see a default UI that doesn't suck. I remember when I first started playing EQ, pre-Kunark, when they still had the original stone-style UI in place. Sure, it was clunky and low-rez, but at the very least it was functional. Now, new EQ players who don't know anything about the UI modding scene are being greeted into the game by a default UI which is ugly, inexplicable, and utterly rudimentary. This isn't the way it should be.

Certainly, it's nice that they've given us the option to change everything from the base UI...but people shouldn't have to do that. Some basic default window positions for each common resolution would be outstanding. Right now, if I load up testing a new UI which doesn't work in my usual 1600x1200 resolution, the default UI will load up with uselessly small windows--especially the chat window--arranged loosely around the screen. It's easy enough to rearrange and resize these, but even that can take half an hour, an hour, just to get the UI functional enough to make playing enjoyable. I imagine there are are a lot of newbies still running around with loose, ineffectively arranged windows because they don't know how to make it any better. Please, give us an official default UI that's customizable *and* functional.
