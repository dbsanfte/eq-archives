Add a zero size element for each missing item. Copy the ids from the default UI. You will need to add the definition and the Pieces tag for each item. The default size is 0x0, so you don't even need to specify it. The new lines should be something like:

<Label item="GW_HPLabel1"><ScreenID>HPLabel1</ScreenID></Label>

<Pieces>HPLabel1</Pieces>

Repeat for all missing elements. Alternatively, if your UI has those elements but calls them something else, simply rename the labels to use the default names.
