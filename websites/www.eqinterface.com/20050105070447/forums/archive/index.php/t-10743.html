What I recommend.

Open each UI step by step. Don't smash them all in one folder. Make a folder for each piece. Make sure each piece works on it's own.

After you are sure that all the UIs work, start putting them together. Start with 2, then add the third after you are sure the first two don't conflict.

What I'm guessing...
1. You have one or more UIs that aren't working by themselves due to changes in the code
2. You have one or more UIs that are conflicting.

Yes, I know it's long and cumbersome, welcome to the world of custom UIs. And when Sony patches a major patch, expect your UI to "break".

When I had some issues as to exactly which UI skin was causing my UI not to load, I simply emptied out my UI custom folder and added a new piece (i.e., inventory UI skin) one at a time. Then I would loadskin'd it. If it worked then I Loadskin'd default, added another skin piece and repeated the process until I found the culprit that was causing my UI not to load. It takes a while, but now with the advent of EQ being able to become a window this process is much faster now. Just my take.
