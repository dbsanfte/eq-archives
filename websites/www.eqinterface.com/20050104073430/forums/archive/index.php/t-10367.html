actually with Quartz you cant move the windows. Some are designed to be permenently "stuck" where they are.

What you can do is go into the XML for the Action window and change:
<Style_Border>false</Style_Border>

to:

<Style_Border>true</Style_Border>

That will put a border on it and give you somethign to click on.

Then once in place change it back to false for the better fitting look.


Another solution that does not invole re-coding...you can switch to the default UI then move the Window then switch back to the quartz....I have done this myself with a few windows.
