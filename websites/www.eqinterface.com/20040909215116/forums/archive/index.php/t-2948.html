This is a skin for the wonderful timer mod, which can be downloaded here (http://www.horizonsoutpost.com/eq/timer.asp).

I love this mod, but I didn't really care for the plain graphics it uses. Thus I made my own:

http://members.telocity.com/~Tipop2000/timergraphics.jpg

As you can see, I prefer a more 3D look. I also made the left-right arrows invisible until the mouse rolls over them. I feel this gives the timer a cleaner look.

Note that to use my graphics you will need to change the text color in the xml to black (the default is white.) To do this, simply change every instance of:

<R>240</R>
<G>240</G>
<B>240</B>

to:

<R>0</R>
<G>0</G>
<B>0</B>

Originally posted by Tipop
Make sure you're getting the COMPLETE .xml file. I've had a few friends complain of the same problem and it turned out their .xml was cut off part-way through. I don't know why this happens sometimes.
This happens usually due to not waiting long enough for the complete XML to be downloaded. The file size of the XML will vary depending on the number and duration of timers used. While a single 1-minute timer will only be 30KB or so, for example, eight 2-hour timers will result in a file that is over 900KB. Since the dynamically-generated file is uncompressed, a file like that could take some time to download for people with slow connections.

Waiting until the browser is no longer busy before grabbing the XML source is the best way to make sure you get the entire file.
