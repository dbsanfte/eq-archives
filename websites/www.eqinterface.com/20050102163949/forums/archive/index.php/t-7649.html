OK

Lets say you want to download a UI you like. First things first, make sure the UI you plan to use is current and compatible with all the changes that SOE makes to the game. Any of them made after the launch of Legacy of Ykesha should be fine, and most older ones are posted as fixed or patched, when someone has updated it to work with the newer files. Lets take an example such as nightelf-loy which is a UI originally done by SPIRITWALKER and patched by Jethal. First you download the file (nightelf-loy.zip). Once you have downloaded the file, you unzip it. If you are using Windows XP this you can do it by right clicking on the file, or you can always use winzip to open it. You want to to unzip it into your everquest\uifiles folder. Some UIs require you to have a certain folder name so check and see if they have a readme file somewhere in the folder and rename it if you have to. So I unzip nightelf-loy.zip to a folder called nightelf-loy and make sure it is in my everquest\uifiles folder. Now AFTER i have read any readme files which give important info, I look in the folder and find the UI file i need to make sure all the windows end up in the right place. In this case the name of the file is ui_name_##.ini Now before I use this UI i need to change the name of that file to match the character i want to use it for. Example: UI_Methuselah_26.ini
the name is always the name of the character you want to use it with, the number is your server number. 26 happens to be the Tunare server. If you are not sure what your server number is, then look in your everquest directory for it. If you have played the character before, you will have a file like this already. So after you rename it, then copy that file into your main everquest directory and overwrite the one that is there. You might want to back the old one up, by renaming it or moving it to a different folder. Once you have the UI file in the main directory, you close all those windows and launch EQ. Depending on the UI some may start you off with everything in the right place, some you may have to use the loadskin command for. just type /loadskin (in game) and the window will pop up, pick th name of the UI from the list and leave the "use ini" button checked. When it loads, things should be in the right place. Hope that helps you out, most UIs have some instructions with them, but a lot do not. If you still have trouble let me know, and I can put together an example using screenshots and such. Have fun and good hunting.


also note, not all UIs have the ui_name_##.ini type of file, but if they do, you will see it when you unzip the folder.

Actually, I think I may see what you are asking...
Of course, I may be interpreting your request too literally.

1) Double Click My Computer.
2) Locate the C: Drive Icon and Double Click.
3) There should be a Folder that says Program Files - Doubleclick
4) You may not see anything. - Ther should be a link that says Show my files. if you cant see anything.
5) Now look for a folder named Sony or Everquest or Everquest Trilogy - Double Click that.
6) Do 5 over again until you have opend a folder called UIFILES.
7) Right-Click in an open area and choose NEW>Folder.
8) Name the folder VERT.

Now you can copy the files you need into the VERT folder then start EQ and when you get in, type /loadskin vert

Hope this helps.
