make sure you have the following things inside your player window:

<TextureInfo item = "AttackIndicator.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>

<Ui2DAnimation item = "A_AttackIndicator">
<Cycle>false</Cycle>
<Frames>
<Texture>AttackIndicator.tga</Texture>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>110</CX>
<CY>46</CY>
</Size>
</Frames>
</Ui2DAnimation>

<StaticAnimation item = "A_AttackIndicatorAnim">
<ScreenID>A_AttackIndicatorAnim</ScreenID>
<Animation>A_AttackIndicator</Animation>
</StaticAnimation>


<Pieces>A_AttackIndicatorAnim</Pieces>

If your ui is using a combination window to display information. Like taking info from the player window and putting it inside the target window; unless the player window is the parent, the attack indicator will not work.
