A web-page has been built to dynamically create Sars hotbutton.xml file, according to what slots you want.

The Hotbutton Configurator lets you choose between Melee, Hybrid or Caster and then configures each of your 20 hotbuttons. Then press Create XML and it will create your customized XML file. You then relabel the file created (it will be labeled as a string of numbers ATM) to EQUI_HotButtonWnd. Use this newly created & renamed file to replace your old one with the same name.

<remember you'll need to do one per Sars character type>

SARS 4.3 Hotbutton Configurator (http://www.midnightsojourn.com/sars/SARS_Config.html) - Thank You! Fobia
