I have made a mod using the base UI created by Vert. I used his version 1.1.

Grab it from my wesite:
http://users.adelphia.net/~locklear/id20.htm

Or from the Complete Set sections here:
http://www.eqinterface.com/sdownload/fileinfo.php?id=226

A few things I changed are:
Made some of the fonts bigger, because I play in 1600X1200
Moved the current HP's and Max HP's around and made them more visible.
Made the Stamina bar smaller
Included the weight in the hotbutton box
Got one of those kewl directional compasses that tell you what your loc will do

Included in the zip file is a file with the 1600X1200 window presets, and a .jpg of my UI so you can see how I have things set up.

This is a work in progress along with my website. I will be making updates as I can. Hopefully some of you will enjoy this UI as much as I have. Check it out, let me know what you think and see if there is anything you would change. Thanks
