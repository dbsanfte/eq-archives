Fuzzy? That would be a monitor setting. Play with that. Unfortunatly I can't figure exactly what the setting name is that deals with that aspect of the monitor.

If you want to increase the font size, you can generally find a commented out <Font>#</Font> tag in the XML file for each element in question. It's just a matter of increasing that number. It's from 1-5.

If there isn't any <Font> tag commented out and if you fell comfertable in the XML files, you can add them yourself.

From what I can recall, <Font> is a valid tag in most, if not just about every, element.
