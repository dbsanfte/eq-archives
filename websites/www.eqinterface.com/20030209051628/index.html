Solution that Worked for Me!

1. Opened My UI_Kudane_91.ini File

2. Found "[CastingWindow]"

3. Changed THIS:
XPos1280x1024=-485952
YPos1280x1024=27
TO:
XPos1280x1024=3
YPos1280x1024=27

The first number "X" is how far in from the left, the second "Y" is how far down from the top, as you can see, the bug moved my casting window to some where just outside Tulsa Oklahoma. This did fix it though.

As of 2/7/03 its been reported that no more windows should be moving, but any that have moved will need to be manually reset using one of the two methods shown above
