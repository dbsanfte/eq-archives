The simplest way would be to make a seperate eqclient.ini file for each skin, giving each one the name of that skin (quartz.ini, vert.ini, etc), including one for the default named default.ini THEN make a set of batch files that replace eqclient.ini with one of the other .ini files. Finally, you could make shortcuts to each of these batch files on your desktop or whatever, and run them before running EQ.
