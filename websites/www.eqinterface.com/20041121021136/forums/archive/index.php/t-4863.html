dont think you can do NEW chat windows in tabs. You could probably get your main one there and working but kinda lame - one tab.

EQ does give you the ability to create a second chat window (right click title bar - new chat window - then you have to go thru and assign all the filters). Its a pain to set up and if you accidently close the 2nd chat window you lose your settings. So you should probably make a backup copy of your character.ini file that contains alll the windows layouts. Easy to restore then.
