Right click on the chat window, this will bring up the chat window menu. Click on New Chat Window which will open a new chat window. Than all you have to do is set the channel (Say, Tell, Guild, Group etc...) that any text typed in that window to go to, and than set which channels you want to go to that window.

And that's all there is to it. You can have as many chat windows as you want. Personally, I have three. One for Guild chat, one "Main" window where say, tells, group, service and such go to, than a "Battle" window where all my battle text goes (you know such as melee dmg, non-melee dmg, and all that good stuff).
