In your MAIN EQ directory, find this file:

UI_YourCharacterName_YourServerNumber.ini

Open with a text editor.

Find this line:

[Main]
UISkin=YourCustomFolderName

Change it to be:

UISkin=Default

Save

Now find this file:

EQClient.ini

Open with a text editor.

Find this line:

UISkin=YourCustomFolderName

Change it to be:

UISkin=Default

That will *hopefully* allow you to at least log in and run using the default UI until you can get your custom one fixed.
