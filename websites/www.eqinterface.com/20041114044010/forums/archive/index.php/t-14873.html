One way to do this is to create a new xml file called something like EQUI_Deathbane.xml and have your custom graphic definitions in there. Then, if you look in the EQUI.xml file, near the beginning you'll see a bunch of <Include> lines for each of the other xml files. Just add an <Include> line for EQUI_Deathbane.xml. The format of the line should be very obvious. A number of custom UI's use this method. If you want a specifice example of one, Lothay 2.5 (http://www.eqinterface.com/downloads/fileinfo.php?id=3174) uses this method.
