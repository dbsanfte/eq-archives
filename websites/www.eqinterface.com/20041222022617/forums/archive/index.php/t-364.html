Each char has it's own UI ini file ( UI_<char>_<server>.ini ) in the EQ directory.

The mods are global, ie it's one directory. If you want a different one for another char, you need to create another directory for it and use that UI on that char.

If you want to copy settings over to another char, just copy the original char's UI ini file to the new char's one.
