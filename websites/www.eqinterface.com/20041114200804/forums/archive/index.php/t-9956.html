I finally decided to update to get rid of the xml compatability errors and release this UI.
It actually survived and worked flawlessly through the last 6 months of patches, without a single modification...

View it by clicking here. (http://home.comcast.net/~mbeiser/Images/Caleal2beta.JPG)


Download it here. (http://www.eqinterface.com/sdownload/fileinfo.php?s=&id=2437)


--------------------------------------------------------------------------------
A very "patch resistant" UI designed for 1280x1024 resolution with a /viewport 0 0 1143 840 setting.

Fully compatable with the default EQUI_Animations.xml file. If there is a patch that changes that file, just delete the custom one and your ready to play!

The zip file contains instalation instructions and a copy of a .ini file that can be used to insure the windows line up correctly.

The UI is primarily designed to be mouse user friendly, and for any class.

The hotbutton window has a tabbed section that gives full access to all inventory and armor locations.

The group window has large clickable areas for targeting self and group members. A pet lifebar will pop up under its owners lifebar if a group member summons a pet. The pet life bar is clickable to target the pet.

The pet window is intended to stay up at all times as part of the UI. It has 3 tab pages. The stats page displays full character stats and resists. The Player page shows Exp and AA exp gauges and %, as well as % allocated to AA, unspent AA points, encumberance level and character name/class/level/title. The Pet page contains all the functions of the normal pet window.

The buff and song windows are compact with small icons, buff names, custom icon backgrounds and numbered buff slots.

The "target" bar is very long and has 10 sections to it. There is a casting gauge under the target lifebar that also has 10 sections to it. I find it usefull for heal timing.

The "target" window is made from the player window, so it will have a flashing red outline around the target and casting gauges when attack is on. There is also a 60 second timer as part of the window.

The inventory window is very compact and has no stats in it. The autoequip area is the class animation.

Containers have the primary, secondary, ranged and ammo slots in them.


The chat window contains the template for its custom borders, as well as the templates for the borders for all other windows in the UI. All custom graphics animations used in the UI are defined in the xml code for the chat window.

The UI is fully compatable with the default EQUI_Animations.xml file. The custom one included with the UI is for the custom buff icon backgrounds. If a patch makes changes to the EQUI_Animations.xml file, you can just delete the custom one and the game will use the default backgrounds.

The modified xml files are:
EQUI_Animations.xml (only modified to use custom buff icon backgrounds, delete if it is patched)
EQUI_BuffWindow.xml
EQUI_CastingWindow.xml (reduced to size 0 0 so it is never visible)
EQUI_CastSpellWnd.xml
EQUI_ChatWindow.xml
EQUI_CompassWnd.xml
EQUI_Container.xml
EQUI_GroupWindow.xml
EQUI_Inventory.xml
EQUI_PlayerWindow.xml
EQUI_SelectorWind.xml
EQUI_ShortDurationBuffWindow.xml
EQUI_TargetWindow.xml (reduced to size 0 0 so it is never visible.)

Modified graphics files are:
window_pieces01.tga
window_pieces02.tga
window_pieces03.tga
window_pieces04.tga
wnd_bg_dark_rock.tga
wnd_bg_light_rock.tga
scrollbar_gutter.tga

Added graphics files are:
Caleal1.png (custom borders, buttons and gauges)

Custom graphics are adaptations of Raxthorn, Vertebrea, Lunakin, and TKing graphics.

This UI survived 6 months of patching and an expansion, with no updates, before I released it.

I fixed my containers by simply using cut&paste from this post by Remelio. (thttp://www.eqinterface.com/forums/showthread.php?t=14385) Was simple and easy.

I replaced the HP numbers in the tabbed player window with mana numbers in this UI. Mana isn't as much of a concern so having it there works out well enough. To do this, open your Caleal UI folder, open your EQUI_TargetWindow window with notepad (in Caleal UI the EQUI_TargetWindow file is the tabbed player window, don't ask me why).
Click the Edit button at top and use Find feature. (or use control+f) Find <EQType>70. Replace the 70 with 128. Save. That's it. The 70 stands for current/max hp, the 128 for current/max mana.
