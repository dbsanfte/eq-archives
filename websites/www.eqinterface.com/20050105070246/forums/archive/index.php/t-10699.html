The reason for the message is that the Hit Point Percentage Labels were added by Sony to the Default UI with the names HPLabel and HPPercLabel.

Many UIs already had these functions in them, using different names, so the modder didn't feel the need to go back and correct for this error.

You can fix this by MANULALLY editing your target window to assign the names the XML engine expects to the labels for these functions.
