You have a few questions in there.... the answer to all of them is the same:

Find an existing ui on this site that has the feature or features you want in yours, then open up the appropriate xml file in Notepad or Wordpad. At first it will seem like the file is written in another language (which technically it is), but you should be able to figure out how the various authors got what they did into their ui mods.

In other words, you can find the answer to your questions by closely examining the work of others.
