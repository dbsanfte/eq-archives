Hey guys. I currently use Drakah's UI, Mixed version, and I love it to death. Of course, this last patch screwed it all up so my Hotbuttons kept getting pushed up, Action/ Spell Bar window got pushed up, and Compass window was getting pulled off the side of the screen. So I went and messed around with it and found a fix that WORKS for those three items. Basically it resizes them and moves one a little so that they are inside the viewable area, and won't get pushed around anymore. Heres how:

Go to your Drakah's UI folder, and find the file named

EQUI_CompassWnd.xml

Right click and select Open With: Notepad

Use the find feature and search for "CompassWindow"

This should bring you down to the last line of code. Scroll down a little from where it says "CompassWindow" and you will see a line of code like this:

<X>0</X>
Change it to this:
<X>2</X>

Scroll down a bit more and you will see:

<CX>1290</CX>
change it to:
<CX>1280</CX>

Close notepad and it will ask you if you want to save, click yes.

Now open the file called
EQUI_ActionsWindow.xml

Again search for "Actionswindow" after opening this file in Notepad.

Scroll down and find the line that says:

<Y>25</Y>
And change it to:
<Y>2</Y>

Then down a bit more and find:

<CY>313</CY>
Change it to:
<CY>290</CY>

Click close and save the file.

Now find the file :

EQUI_HotbuttonWnd.xml

Open with notepad and search for "HotbuttonWnd"

Scroll down till u see the code:

<CY>140</CY>
And change it to:
<CY>100</CY>

Click close and save the file.

That should be all you need to do. You Should also be able to use the Action window fix for the Spell Bar window, if its having the same problem. Me being a melee, I don't use the spell bar, but I have my action window in the same spot, and I think they are the same size. As long as your UI has the Action Window/Spell Window at Coordinates:

XPos1280x1024=0
YPos1280x1024=728

Then it should work. Also, note that I did these fixes using the 1280x1024 screen res, so if you are using a different res, I don't know that it will work. (I haven't actually found another res that Drakah's UI works at, heh)

I hope this helps some people. You may see a little gap between your Hotbuttons window and your action window and the bottom of the screen, but for me, its better than having things moving all around.
