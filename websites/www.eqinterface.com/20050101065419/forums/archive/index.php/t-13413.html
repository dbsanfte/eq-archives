04-07-2004, 06:29 AM
Any storyline that shows up in the story window has a file in the /storyline folder whose title looks like this:

storyTitle.txt

(may just have the title storyTitle if you have the viewing of file extensions turned off)

Also make sure to scroll down in the story window... some mods have a modified or created date which is older than the SOE storylines and so will appear much farther down the list.
kernan
04-07-2004, 06:41 AM
I took a look at the storyLevel mod and when I open up the .zip file, the mod is down inside another folder named 'storylevel'.

You should look inside the /storylevel folder in your own /storyline folder and move the file named 'storylevel' and the folder named /levels to the /storyline folder so the structure is like this:

/storyline
...storylevel.txt
.../levels

not like this:

/storyline
.../storylevel
......storylevel.txt
....../levels
