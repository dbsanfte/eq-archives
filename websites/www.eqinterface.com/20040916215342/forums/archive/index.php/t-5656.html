SoesUI 1.3

- Based of Drakah's Drakahmix (Excellent work Drakah!)
- Optimized for 1280x1024 resolution but ALSO WORKS in 1024x768 resolution.
- Provides 100 more viewable pixels than Drakahmix.
- Designed for all classes (Casters, Hybrids & Pure Melee)
- Features Customized: Buff Window, Short Duration Buff Window (Songs), Player Info/Clickables/Pet Window combined into one, Spell Book, Warcraft 3 Style Spell Icons, Spell Cast Bar, T.King's Arcane Guage, Hotkeys, Group Window, MP3 Player, Compass, Target Bar.

*** PLEASE NOTE ***
The UI will come out mangled unless you copy the UI_YourCharName_YourServerNumber.ini file and rename it accordingly, for Veeshan players it would be like this: UI_Soes_35.ini where you would replace Soes with your character's name. Once you renamed the file, copy it to your BASE EQ DIRECTORY and make sure you overwrite whatever existing ini file might be there. IF YOU DON'T DO THIS THE UI WON'T WORK CORRECTLY!!!!

The crash is caused by failing to have the new tint XML file included in the EQUI.XML file in the \Soes folder. Simply add:

<Include>WhateverTheHeckTheDarnTintXmlFileIsCalled.xml</Include>

note: placeholder name for the actual name, which eludes me (I'm not at home)

at the bottom beneath all the other Include statements and then save the file.

I had the same problem and resolved it with this fix.

- Ash

Also, Darkrigel, I think you just need to change your game settings to the 1024x768 size. The Soes files have settings for both in them and should be cued from the client which set to use.
