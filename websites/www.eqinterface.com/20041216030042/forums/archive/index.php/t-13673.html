Anyway, the new version, 1.2, has been uploaded. It holds the following changes:

**** UPDATE TO 1.2 ****

History:
2004/07/16
Changed the EQUI_PlayerWindow.xml to display numerical mana in the player window. I left it out of the inventory screen because of no available space, and it is of more use in the player window I guess.
Be advised: some of you asked me how to remove the mouse pointer from the UI, and I sent them instructions. The updated version has this pointer, again, so if you want to remove it send me a mail, and I will send you the files to exchange them.

*********************

Concerning the patch-problem with Remmy: If you already used it in another UI, you cannot run it again (that's why it is greyed out). As it only changes the rate of animations in the EQUI_Animations.xml, you can run the UI without any problems even when not patched (only the class animation of the Paladin helmet looks weird if you never used Remmy).

Concerning the title "Player" in the window, I will look into it and fix it asap.
