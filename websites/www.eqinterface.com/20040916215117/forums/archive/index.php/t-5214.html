Thanks for the heads up on the item window in 1024, It seems the .ini file only remembers one value for that window position, whereas it will remember positions for multiple rezes on most windows, so it was using the value from the 1280 mod, displaying the window almost completely off the bottom of the screen. The very top border does appear along the very bottom of the screen, near the left corner, but it will blend in with the borders of any window open in that area, so shrink your chat window, right click an item, and watch the bottom of your screen for the border to appear, then slide your mouse off the item icon (still holding your right mouse button) and you've got a locked open item descrp window to drag to somwhere visible. Alternativley, you can load the new character.ini file I'm attaching.

Here's a couple of things to try:

NewUI=TRUE
