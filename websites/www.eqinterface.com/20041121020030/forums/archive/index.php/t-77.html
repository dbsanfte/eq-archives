Alshain
07-28-2002, 12:13 PM
You don't need the name. PetHP_Blah just needs to be a name unique to the screen. The ScreenID field in that label isn't really used but should also be unique - usually the same as the control's name. PetHP_Blah is only used in the <Pieces> list to get the control attached to the screen. Name it whatever you want. Using EQType 69 is what gets the label filled by the program with the pet HP percentage.
