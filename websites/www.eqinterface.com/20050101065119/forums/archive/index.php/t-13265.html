The contents of your uierrors.txt tell me that Everquest cant find your custom interface files.

This could be only due to 4 reasons.

1. You didn't follow the directions in either the readme.txt that came with the mod or the directions found at http://faq.eqinterface.com.

It sounds like your installing it correctly so you can ignore this. Only put the above incase others search on the same problem.

2. The files are inside a folder inside another folder in your custom ui folder explained by cromyr above.

Again it sounds like your checked this and the files are in the correct spot.

3. You have a space in the custom folder name.

Doesn't look to be the case here.

4. You have 2 or more Everquest directories because of a poorly written expansion install program. To make sure you are putting the ui files in your "active" everquest folder you need to right click on your Everquest Icon and select properties. The Target: field will tell you where your active everquest directory is located.

This sounds like the culprit. You can confirm this if you type /load and when the window pops up it doesnt show your new custom folder and only the default folder.

Hope that helps. :)
