If Alt-F1 isn't showing or hiding any windows, then he either has the window size set to 0,0 or he has moved it off the screen. Go into the petwindow.xml file, scroll to the bottom and make sure both LOCATION coordinates are NOT negative. Also make sure both SIZE coordinates are NOT 0.
