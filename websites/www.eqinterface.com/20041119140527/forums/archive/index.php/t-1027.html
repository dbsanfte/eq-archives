I don't know why its doing it, but here are my suspects:

1) Windows caching utility (you just thought you had changed it, Windows is using an older copy loaded into the file cache). Unfortunately, I don't know if there is a convenient command to force the file cache to reload.

2) EQgame.exe caching utility (you just thought you had changed it, Windows is using an older copy loaded into its file cache - aka the texture cache. I've been suspecting that a lot of stuff other than textures is getting dropped into that cache).
