Attached is a .zip file that contains what you need to build (yes, build) a custom bardsong spreadsheet.

If you're familiar with my spreadsheet, then basically, you just enter values into it, save it as a .CSV file. Then copy my feedback XML file to your custom directory. Then run the program which will insert the code into the XML file. Then you have your own personalized version of the spreadsheet.

You need Microsoft Excel to use this. (Sorry those who don't have it).

It's preliminary, so let me know what you think, etc.

Here is the documentation:

Installation
============

1) Setup the XML File
---------------------

Copy the "EQUI_FeedbackWnd.xml" to your custom UI directory. Note that the
feedback window is still fully operational! I've just added some new stuff to
it.

2) Generate a Bardsong CSV File
-------------------------------

Run the spreadsheet () in Excel. Enter your level, instruments, etc. Get it to
how you want it to look. Spreadsheet release 3.19 (and above) should work.
When you're done saving, save it before going onto the next step.

Make sure you are viewing the first page/tab (the page you can enter your level)

Do a "File->Save As".

For "Save as Type", select "CSV (Comma delimited)(*.csv)". This is very
important!

Press the "Save" button to save the file as a .csv file. It will warn you that
it can only save one page; Click "OK". (If you want to exit at this point, like
I do, click on "No" on the next prompt.)

3) Run the Program
------------------

Enter the name of the .csv file into the first line (or use the "..." button to
select it.

Enter the name and path of the EQUI_FeedbackWnd.xml file in the 2nd line (or use
the "..." button to select it.

Press the "Build XML File" to create the EQUI_FeedbackWnd.xml file. Note that
this will read the .csv file, and modify the .xml file.

To display the window run EQ and type in "/feedback"


Customization (Putting the tab somewhere else)
==============================================

When the program runs, it looks for two lines:
<!-- ReedInsertPoint:Start:Bardsong_Root -->
<!-- ReedInsertPoint:Stop:Bardsong_Root -->

The program code replaces everything between those two lines. You can move
those two lines to any XML file, run the program, select the new XML file, and
the code will be placed there.

The tab name of the spreadsheet is called Bardsong_RootTab_0. So you can add a

<Pages>Bardsong_RootTab_0</Pages>

as a page to any TabBox.

(I've never actually tried it... Someone tell me if it works ;)


Future Work
===========
It's a little messy to read. In the near term I want to put lines in to
separate out the cells. Also, I'd like to put in the "Song Details" spreadsheet
work as well. In the long run, I'd like to get rid of the Excel file and have
the numbers be program generated (I don't know when I'll get to doing that
though). Also, I'm always open to advice and new ideas.
