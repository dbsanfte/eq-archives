I didn't like what was out there so I made one. This is an uncomplete conversion of NewUI spells icons and gems to the Old UI style. It is uncomplete because I was only able to compare the old and new icons for a Druid. If you are a druid or something very similar like ranger or what not, than it should work well for you. Also someone icons are incorrect because certain spells shared an icon. Example of this is SoW and SoE, both use the same icon so it had to be either both a boot icon or both a tornado icon. Lastly, everyone feel free to take it and add on to it because it is only going to get more useful if it becomes more complete.
