Yes, it's possible. You can make a new gauge design and use it just for those two gauges by creating a new graphic file (for instance, big_gauge.tga) and having EQUI_Animations.xml point to that file, then copying the default gauge info (in the same file) and making a new set of gauge info by renaming all the parts (changing gauge to big_gauge for instance) and having them point to the file. You'll also have to enter the xy coordinates and the size coordinates for each part, based on how large they are and what pixel they start at in the TGA file.

Remember to leave the orignal gauge info alone unless you want to change the graphics for ALL gauges. Just copy the info to another location in the file (Preferably right below the default gauge info) and edit that.

Once you're done with EQUI_Animations.xml, you can add your new gauges to any window you want. I'd add them to the pet window if you want them seperate from everything else.
