Here's the scoop.

If someone wrote a complete UI, then each window might not be self-contained, and might be dependant on other files within the UI, meaning that window won't work when it is copied into another UI.

On the other hand, if you are talking about a window that is meant to be standalone, then it should work if you copy that file into the directory for the UI you are using.
