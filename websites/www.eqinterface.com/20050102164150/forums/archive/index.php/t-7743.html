If I have my UI set up just how I like it on my main character, is there a way to load the UI on an alt character so it's just the same as on my main? I'm talking about the window placement and fade levels and all. I know that if I type "/loadskin *name* 1" while I’m on a character, it will save the settings it currently has but I would really like to be able to load in the UI on all of my characters without having to set it all up on each one if I can.
Armengar
03-25-2003, 02:59 PM
In your EQ directory there is a file for each one of your characters with a name similar to

UI_CharacterName_##.ini

This file stores the UI data for that character. The "##" is replaced with the character's server number.

Make a copy of this file for the character where you have the UI the way you like it. Rename the copy to match the character for whom you would like to have the same setup. You will need to delete, overwrite or rename the existing file for the character you are changing.
