This would be my suggestion. Create a directory in the uifiles directory called Newskin(or NewUI or whatever you want). Copy everything from the defaul directory into that directory(Copy NOT Cut and Paste). Then for each piece that you like, copy that file into the Newskin directory, overwriting the file that is there. Once you have them all there, log in and type /loadskin newskin and you should be good to go.
