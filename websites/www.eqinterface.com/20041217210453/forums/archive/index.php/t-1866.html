Changes to Player window, Target window, Player "stats" window (converted Pet window), Group window, Containers, Inventory, Casting window, Chat windows and Confirmation box. Detailed descriptions of the changes are on the Screen shot.

For a "working" view, go here (http://members.rogers.com/khobson/ui/UIWorking.jpg).

For a screenshot which gives credit for all the work not my own, go here (http://members.rogers.com/khobson/ui/UICredits.jpg).

Originally made for a resolution of 1024x768, it will work in any resolution equal to or higher than that. It also works in 800x600 if you use a different buff box (eg, one that can be stretched across the top of your screen).

As a ranger, I find this incredibly easy to work with, and I think it would work well for all hybrids.

Full installation instructions, detailed descriptions of the changes made and detailed credits to other's work are all included in the README.txt file.
