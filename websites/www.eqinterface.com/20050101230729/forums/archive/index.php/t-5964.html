Did you actually *replace* the cursor information in the EQUI_Animations.xml file, or did you just add it in while leaving the old information there? Did you also make sure to include the texture information?

example:

I use t.king's arrow cursor, so I open my EQUI_Animations.xml file using a text editor, do a Search or Find function to find:

<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>window_pieces01.tga</Texture>
<Location>
<X>230</X>
<Y>40</Y>
</Location>
<Size>
<CX>22</CX>
<CY>22</CY>
</Size>
<Hotspot>
<X>4</X>
<Y>1</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>

I replace that information with:

<TextureInfo item="tk_arrow.tga">
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
</TextureInfo>

<Ui2DAnimation item="A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>tk_arrow.tga</Texture>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>


Notice that the new file is also defined? The <TextureInfo Item>.

Also need the cursor graphics file and a <TextureInfo> tag for the cursor file:

EX:
<TextureInfo item = "cursorsfilename.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>
