Fozzik
01-12-2004, 03:44 PM
What you have there isn't wrong, necessarily... but it's very incomplete. There are several things that have to be present in the pet window, the most important one is that the buff box in the pet window is like a separate window in and of itself. Another thing that you will have to overcome is that the pet window cannot be resized vertically. The only way to make the window taller or shorter is to change the size in the xml file, go into your ini file and remove the entries for pet window size, and then load the UI. Once it has saved a vertical size in the ini file, it cannot be changed in the xml.

Here's a window to get you started. It only displays the buffs, and should work fine to plug into your UI. Make sure you go in and remove the size entries from your .ini file before loading this one, I made it taller than the default. You can adjust the horizontal size in game.
