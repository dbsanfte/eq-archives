I've updated the TimerMod (http://www.timermod.com/) XML generator to make all configurations compatible with the June 16th patch. This change affects the Raid window only. If you are using the Raid window for your timers, please download a new file package.

You can click the Reconfigure Timers button in the "INSTRUCTIONS" file of your original download to launch the configuration wizard with your current settings already filled in. To configure a new file package from scratch, go to the "DOWNLOAD" section of the TimerMod site (http://www.timermod.com/) and click Configure Timers.

Patch Summary
Here is a quick summary of version numbers required for particular windows. You can find the version number that you are currently using in the comments near the top of the "Timer" XML file. If you don't have at least the required version listed below, you should download a new file packgage. Raid window - version 4.0.1 or higher
