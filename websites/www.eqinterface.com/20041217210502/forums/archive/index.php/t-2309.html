3. Buy a slightly better PCI card. Limit is probably $75.

Any suggestions as to which is better? I realize it's probably option 2, but option 1 is a quick fix, and to be honest I really don't care about the aesthetics as much as performance. I'm happy with the way the game looks now I just want it to run smoother.

Well, ask yourself this - Do you want to worry about this later on, again?

Option 2 - that way you set yourself up for the furture to be able to expand even farther, when you can.

Even if that means using your currect GF2 PCI card in the new MB's PCI slot until you can score a better Vid Card. You still have that AGP slot just waiting for a real upgrade.

1 and 3 are both hot fixes. Nothing more. They will work for a period, and then you'll be faced with the same troubles when the game upgrades it engines again. Not just EQ, but any graphicly intensive game.
