When you say disappearing, what do you mean? There was a bug in the last patch that was causing windows and gauges to displace, which *should* be fixed now... check this thread (http://www.eqgui.com/showthread.php?s=&threadid=6413) for info, to see if that fixes the problem.

If your reload the UI and a different spell bar is showing up, you may want to double check what folder you're telling it to load... the best way to reload your UI is the command /loadskin foldername; adding a 1 at the end will keep all windows in their current position (which may not help); leaving off the 1 will move all windows to their default position - you would have to move them all back where you want, but at least you'd be able to see them all.

Again, about the spellbar, it wouldn't hurt to unzip that downloaded file into your custom UI folder again, just make sure the right one's in there.

It's always a good idea to check your UIErrors.txt file; if you're still stumped, post it here.
