ok I figured it out and fixed this problem with mine. What I found the problem to be was that I had other ini files named for instance: UI_charactername_ini (not to be confused with charactername_ini). The UI_charactername was associated with my previous UI and it caused conflict. I deleted it out then logged on again and let it create an all new first time logging on that character on that UI ini file... all has been good since.

Hope this helps.
