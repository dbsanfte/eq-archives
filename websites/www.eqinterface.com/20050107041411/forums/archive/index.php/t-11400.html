I have a ATI Radeon 9600 this worked..

On the live tech page of sony there was a thread about "3dAnalyse" from:
Http://www.tommti-systems.com

and instructions
1. start 3d Analyse
2. At startup screen , next to the big number "1" click and find and chose "eqgame.exe"
3. in the 2. slot Chose the EQ directory
4. in 3 chose "Eqgame.exe"

5. VenderID I put in 4098
6. DeviceID I put in 20040
8. under DirectX Options I clicked
"Force SW TNL"

HIT the run button Makes 3 .dll files
d3d8.dll
d3d9.dll
opengl32.dll
..on patcher option screen pops up chose the 3d-analyaser
