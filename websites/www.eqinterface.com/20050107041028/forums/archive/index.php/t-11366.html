Xymarra
11-25-2003, 03:34 PM
Yes, you have to move the UI_charname_servername.ini file to your EverQuest directory. (If that is not described in the Readme.txt file, I will have to change it.)

If you put the UI in a directory named something other than "xym", you will have to either edit your ini file or load it in game with the /load command. The default for /load has been changed, so that "/load xym" is now the same as "/load xym 1". If you want windows to move to their default positions, you have to type "/load xym 0".

Please note that, due to limitations in the layout system, these default positions are not the correct positions. You can drag everything into the correct positions with the mouse, except for the tick timer. Because of the addition of the pet buff icons, that window now requires special manipulation. :P
