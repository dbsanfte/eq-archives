Also the reason for the four files is size.

With the restriction of the power of 2 rule in file dimensions the smaller the files the less wasted space you end up with.

The default image (sorry a little bit of guessing here) is about 380 by 280 pixels. This means an image file needs to be 512x512 in size which is about alot of empty space (over 50%). So by cutting it down into small chunks you can reduce this empty space by huge amounts.

Ok first the easiest - To make any bar only show when used use this in .xml

<GaugeDrawTemplate>
<!--<Background>A_GaugeBackground</Background>-->
<Fill>A_GaugeFill</Fill>
<!--<Lines>A_GaugeLines</Lines>
<LinesFill>A_GaugeLinesFill</LinesFill>
<EndCapLeft>A_GaugeEndCapLeft</EndCapLeft>
<EndCapRight>A_GaugeEndCapRight</EndCapRight>-->
</GaugeDrawTemplate>

They were just trying to avoid breaking the game for some of their customers.
