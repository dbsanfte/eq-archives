I'm not sure how many people have messed around with the Compass Window and to what extent, so if any of this info is a repeat then you can flame me later. However, I haven't seen any info about what I'm going to post:

In the compass window I've discovered the compass only works if it is contained in the <Screen item = "CompassWindow">. This also means that any other <Screen Item> inside the CompassWnd.xml that contains a compass will also work.

You're wondering, "Why do I want two compass in the same window?" It's simple really; because you cannot move the default compass around. This makes it very difficult to scale the compass window inside a ui without either resizing the overlay elements (which causes the compass to lose accuracy) or losing the visual integrity of the compass (having the compass animation bleed to the other side of the window). By putting a sizable window with a working compass inside the default window, you then solve both of these problems. You can scale the window however you like it to fit a ui, and keep the accuracy of the compass. The final problem remains. If you have one compass overlaying the other...you still see the one that bleeds . Which you then add another <Screen Item> to act as a filter and a background for the compass you want to display.

To demonstrate, I've taken a screenshot of a compass window I sized to 200 x 200 Pix using the template WDT_RoundedNoTitle. The compass is inset in the middle using the template WDT_Inner:

http://eq.onlinegamersguild.com/compass.jpg

Here are the main elements for the Screen Items:

<!--Background Filter-->
<Screen item = "CompassWindow2">
<ScreenID/>
<!--<Font/>-->
<RelativePosition>true</RelativePosition>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>400</CX>
<CY>100</CY>
</Size>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
<!--<TooltipReference/>-->
<DrawTemplate>WDT_RoundedNoTitle</DrawTemplate>
<Style_Titlebar>false</Style_Titlebar>
<Style_Closebox>false</Style_Closebox>
<Style_Minimizebox>false</Style_Minimizebox>
<Style_Border>false</Style_Border>
<Style_Sizable>false</Style_Sizable>
</Screen>

<!--Working Displayed Compass-->
<Screen item = "CompassWindow1">
<ScreenID/>
<!--<Font/>-->
<RelativePosition>true</RelativePosition>
<Location>
<X>47</X>
<Y>84</Y>
</Location>
<Size>
<CX>106</CX>
<CY>32</CY>
</Size>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
<!--<TooltipReference/>-->
<DrawTemplate>WDT_Inner</DrawTemplate>
<Style_Titlebar>false</Style_Titlebar>
<Style_Closebox>false</Style_Closebox>
<Style_Minimizebox>false</Style_Minimizebox>
<Style_Border>true</Style_Border>
<Style_Sizable>false</Style_Sizable>
<Pieces>CMPW_Strip1</Pieces>
<Pieces>CMPW_Strip2</Pieces>
<Pieces>CMPW_Overlay</Pieces>
</Screen>

<!--Default Compass-->
<Screen item = "CompassWindow">
<!--<ScreenID/>-->
<!--<Font/>-->
<RelativePosition>false</RelativePosition>
<Location>
<X>0</X>
<Y>24</Y>
</Location>
<Size>
<CX>200</CX>
<CY>200</CY>
</Size>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
<!--<TooltipReference/>-->
<DrawTemplate>WDT_RoundedNoTitle</DrawTemplate>
<Style_Titlebar>false</Style_Titlebar>
<Style_Closebox>false</Style_Closebox>
<Style_Minimizebox>false</Style_Minimizebox>
<Style_Border>true</Style_Border>
<Style_Sizable>false</Style_Sizable>
<Pieces>CompassWindow2</Pieces>
<Pieces>CompassWindow1</Pieces>
<Pieces>CMPW_Strip1</Pieces>
<Pieces>CMPW_Strip2</Pieces>
<Pieces>CMPW_Overlay</Pieces>
</Screen>

I hope someone finds this info as useful as I do.
