Not sure where else to post this - it seems appropriate for this thread (if not - pls feel free to move this to the appropriate area)...

As of the release of LDoN - Verant has apparently renamed the INI files that the game uses for loading UI's. To "fix" your UI, you need simply figure out what they have changed the file names to.

For example -

Old INI naming convention for characters on Bertoxxulous server were UI_<char name>_33.ini new naming convention is UI_<char name>_bertox.ini

In summary - they're using abbreviated server names instead of server numbers. Change the number to the server abbreviation and you should now find your UI "fixed".

Some additional info:

The INI files will automatically be renamed appropriately (based on whichever server the character resides) ONLY if you log on to that character. However by doing so - it will default your UI and reset window locations for that character.

On the brighter side of things, you'll only need to log on one character to figure out the naming changes - after which you can replace the server name with the server abbreviations (as stated above) for all of the rest of your INI files - so you'll only have to rebuild your UI settings for one character.
