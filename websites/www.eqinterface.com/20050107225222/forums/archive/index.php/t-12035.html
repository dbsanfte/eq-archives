I checked out your link... what you want would take some pretty extensive modding. There are many times additional files that would need to be included with the .xml file that you are trying to get.

For instance, my group window makes calls for graphics from two different .tga's... plus it uses the default guage which is defined in the animations file and uses window_pieces01.tga (this is the main picture file for a UI's theme). If you copied all those files into a custom folder, you might get the group window to work... but it would most definately break other elements that are also set to use their own animations file and window_pieces01.tga.

I appologize, I just didn't design my UI for windows to stand on their own. =(
I think you may run into the same problems with some of the other windows you selected, at least the ones from complete UI's.
