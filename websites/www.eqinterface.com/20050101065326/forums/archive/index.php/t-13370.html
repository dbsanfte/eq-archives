Originally posted by Inokis
go inside the pet window xml and adjust the size variables. If this doesn't work, you'll need to go inside your UI_Player_server.ini and adjust the variables for your pet window there.

Changing the XML of the PetWindow itself won't help. Once the window is defined as "resizable" the default values are no longer used. You have to go into the UI_Player_server.ini and change the values there, that's where is stores the current "resized value" of the window.

You have to do this while eq is NOT logged in though. If you attempt it while windowed, then reload you UI, EQ will overwrite that file with what it THINKS are the right values, and you lose your changes.

Also, the UI_Player_server.ini has settings for every resolution in there. You need to be sure to change the resolution you're currently using, or change them all. That will become more obvious when you look in the file.

HUGE pain in the butt, I know... I added a description of how to do this in my pet window mod.

Good luck.
