Ikabod's TKing Mix
This mod is based on TKing's latest Celtic release.

Changes/Modifications include:

1.) Altered the color of the overall UI to be less yellowish and more gray
2.) Added an inv from TKing that he had released prior to LoY, I like it because its small and well laid out. Added the charm slot, face pick and dye buttons to it.
3.) Remade a hotbutton window based on an old vert one that I liked. It does not use any animations or template xml file. Made up of just the EQUI_HotButtonWnd.xml and a tga file.
4.) Removed the titles from alot of the windows
5.) Changed the buff window to one that I was previously using (dont remember where I picked this one up but kudos to the creator).
6.) Slimed down the pet window a little bit more and moved the pet name farther away form the health bar so when I had a charmed pet with a long name it wouldnt overlap the bar.
7.) Added the player name to the player window (helps for when I'm running EQ in more than one window).
8.) Changed the compass to the one that shows + - for /loc
9.) Changed the spellbook to TKings Regency spell book.
10.) Made the target window show longer target names.

EDIT:
1.) Changed the buff windows to Drakah's buff windows (he did a sweet job on these)
2.) Changed the compass to Yasoldat's compass
3.) Some visual tweaks.

Enjoy.
