I posted this in the Help section that alot of people had seemed to have problems with Geddines Target Window not showing after last patch. So, here is what I have done and previously posted:
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=

I took out the little lines that had shown in middle of bar and I copy and pasted some stuff from Vert's mod to make the bar show up a little better. I also changed the font to be a slight bit larger and yellow to stand out that can easily be changed to any color you like.
In addition my background is bluish so that will show different on yours.

This is code to look for to change Target text color and such, and a link to code numbers to put in place. These can be used on many mods to change text color. If this is not in between the <font> and <RelativePosition> just copy and paste and test it out worked great on my Spellbook mod :)

<TextColor>
<R>240</R>
<G>240</G>
<B>0</B>
</TextColor>


Links for color changes:
Color Codes (http://a-zguides.com/HTML/htmlcolor.asp) or Another Color Code Chart (http://www.hypersolutions.org/pages/rgbdec.html) and yet another Listed by color name also (http://aggedor.freeshell.org/ccodes1.html)

I made the bar sizable if you do not wish to have this option change this way:


<Screen item = "TargetWindow">
<ScreenID/>
<RelativePosition>false</RelativePosition>
<Location>
<X>63</X>
<Y>240</Y>
</Location>
<Size>
<CX>424</CX>
<CY>60</CY>
</Size>
<Text>Target</Text>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
<TooltipReference>Your Current Target</TooltipReference>
<DrawTemplate>WDT_RoundedNoTitle</DrawTemplate>
<Style_Titlebar>false</Style_Titlebar>
<Style_Closebox>false</Style_Closebox>
<Style_Minimizebox>false</Style_Minimizebox>
<Style_Border>true</Style_Border>
<AlignRight>true</AlignRight>
<Style_Sizable>true</Style_Sizable>
<Pieces>Target_HP</Pieces>
<Pieces>Target_Name</Pieces>
<Pieces>Target_At</Pieces>
<Pieces>Target_Percent</Pieces>
<Pieces>Target_PercentSign</Pieces>
</Screen>


Change to this <Style_Sizable>false</Style_Sizable>
