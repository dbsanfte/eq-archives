Enoks UI - Features:

- Compass w/ Loc Indicators (green = increasing, red = decreasing)
- Borderless Note Window w/ Scroll Background
- Informative Player and Group Windows
- Buff Window w/ Color Coding and Spell Names
- Container Window w/ Border but Moveable Only By Container Name
- Borderless Spellbook w/ Animated Close Button (top center)
- Compressed Spell Gem, Hot Key, and Selector Windows
- Simplified Pet Window (same size as target window)
- Old Style Inventory w/ Retro Icons (old unused EQ graphics)
- Borderless Quantity Window w/ Retro Graphics
- "Sneeze Guard" Over Chat Windows, Prevents Accidental Clicks
- Minor Tweaks To Guage Colors (and other things) In Other Windows

I'm working on a second and final version after really really play testing my UI.

Here's the newest pic (I set everything to no transparency so they'd be easier to see in this pic):

http://www.eqgui.com/attachment.php?s=&postid=12792

Changes:

Player Window -
Added a "Tick Timer" to the bottom, the <-> arrow moves across the bottom of the window in six seconds (one EQ tick) and you can click anywhere on player window to target self.

Group Window -
Made it so you can click anywhere in the sectioned off areas to target a player, if you click on Pet: XXX% you'll target their pet.

Pet Window -
Click anywhere in window to target pet.

Container Window -
Resized Combine button (by two pixels) and changed it's graphic slightly.

Spell Cast Window-
Squeezed them in even tighter and added white circle around spellbook button (cause I like how it looks).
