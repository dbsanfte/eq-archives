Go into your EQ directory. You will find a file called
UI_<charname>_<server>.ini
(ie I would find one UI_weskat_tunare.ini)
In the zip file for the UI you want you will also find a file with the same format. (Note:depending on when the zip was made it may be UI_<charname>_<##>.ini, this is the old naming format) change the name of that file replacing <charname> with your character's name and <server> (or <##>) with the your server name as found in your EQ directory.
Then copy and paste the newly renamed file into your EQ directory.
Hope that helps.

Here goes
1- Unzip the UI you want to use (WinXP right click and extract all)
Note: remember where you extract them to
2- Open windows explorer and navigate to your EQ directory
3- Look for the UI_<charname>_<servername>.ini file (ie UI_Service_legends.ini, *the legends part is srictly for example purposes)
4- when you find that file, pay attention to and remember what the <servername> part is (exact spelling is required)
5- Open the folder that contains the UI you want to use
6- Find the UI_<charname>_<servernumber>.ini file
7- right click on that file and choose rename
8- Type your character's name in place of the character name <charname> that exists for that file
9- Type your server name (from step 4) in place of <servernumber>
10- hit enter, the file should now be named correctly.
11- Install the UI as directed in the readme file and check this FAQ (http://www.eqinterface.com/modules.php?op=modload&name=FAQ&file=index&s=) for help. (Note: the server numbers link in the FAQ is no longer valid.)
