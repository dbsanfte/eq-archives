There seems to be a misunderstanding, You CAN mix TGA and DDS files in the same UI.

For example Window_Pieces01.dds can be moded to a UI and still have Window_Pieces02.tga function correctly.

The confusion is if a file calls wnd_bg_light_rock.tga and you converted it to a DDS you must change all the files it's in.
IE. EQUI_Animations.xml
and EQUI_Templates.xml

Now if you have an exisitng Mod that still references it as a TGA it will crash.
So alot of the Custom Mods out there will be incompatible. That was why the warning was there.
