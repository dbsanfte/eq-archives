Wish list item for the SCREEN element in the SIDL

Style_Sizeable = true allows a window to be resized is already in the XML code.

I would like to see two new tags

Style_VSizeable = true (default)
Style_HSizeable = true (default)

these would allow windows not to be resized in one direction only if set to false. This way pull down type resizable windows could be done easier without width being messed up by mistake. I do this with my bazaar windows now and pull down as I need to see more window and I fix location near top of viewport.
