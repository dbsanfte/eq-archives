I assume you want this in your main player info window. If so, you'd want to edit EQUI_TargetWindow.xml.

If you weren't aware of it, (EverQuestDir)\uifiles\default has all of the XML files for the default setup. Any changes made to those files will be overwritten by the patcher, but it does let you see all the different things you have the ability to change. You need to put your modified files in another directory, like (EverQuestDir)\uifiles\mine, then use the command [B]/loadskin mine 1[B] in-game to have it look at the file(s) there. For simple tweaks like yours, it will probably work best to just copy the default files and edit them, rather than trying to rewrite the whole thing from scratch. I'm sure some people would try to do that...

Pretty much. You'll have to change the name on the copied section from "HPpercent" or whatever it's called (I didn't actually look) to "AApercent" or something, so that it makes a new item, rather than just having the same thing coded for twice. You'll also need to edit the location of the item, so that it draws in the right place. And you'll need to make sure you add your "AApercent" item to the window, usually at the end of the file. You'll see a listing of all the items that are in the window...

Just think of every single thing in a window as an item and you should be fine. So you want to copy the code for the HPpercent item, change the name and location of the new item, and tell the window that the new item is there.
