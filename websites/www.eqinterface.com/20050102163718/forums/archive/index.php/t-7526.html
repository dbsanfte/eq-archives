I have a feeling that you may have your ui folder in the wrong place. Not your fault though. I noticed that when I zipped up the original files I zipped them up along with my folder marked Old Celt.
If you made a folder and placed the folder named OldCelt inside it, then it won't work. You have to either take out the OldCelt folder (with all the files in it) and use that one- or take out all the files and place them directly into your folder that you made and throw away the oldcelt folder which should be empty if you moved those files.

I have an upgrade to this (final tweaking) called OldCelt01. It replaces OldCelt. This upgrade doesn't have the folder so if you extract that into your own folder placed beside the default one the files should all extract into it loosely .

Change the animation in both the inventory window and the hotbutton window. If you want the cleric animation that would be 02 for both of them. I found out that just changing the animation in the hotbutton window won't change the animation in the inventory since I made the upgrade OldCelt01.
