My UI is a snowball effect since the very beginning. I taught myself how to do XML modifications by looking at how the default UI files were coded - by defining all objects in the animations file and all templates in the template file. This makes it very easy for referencing and takes less actual coding to get the same results. The downside to this is that it is virtually impossible to remove a single aspect of my UI without major surgery, however you can still add other independant mods to my UI providing you dont remove the animations and templates files.

Had I known what I know now, i would have definately done it differently, and am currently in the painful process of extracting all of my modifications from the centralized animations and templates files and putting the definitions into their needed windows for a more mix and match friendly UI. You will be seeing a lot of independant window mods from me in the coming weeks as the surgery progresses.
