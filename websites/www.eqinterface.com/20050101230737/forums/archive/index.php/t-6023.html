To answer both of you YES. This is the reason that SONY has things set up the way they do. All you have to do is the following:

For each skin you are going to use have a different directory name under your uifiles directory. Put each of the different skins one in each directory.

Next login to your character, do the /loadskin skindir 1 where the skindir name is the name of the directory where you put the skin for that character.

Then you may have to rearrange the windows a little to get them where you want them and then logout.

Go to the next character and do the same thing only use the skindir name that you put the skin for that char in.

Repeat for last char.

That is all there is to it. You now have three different characters with three different skins.
