If you're looking for a warrior tweak of the popular Vertebrae Skin, look no longer...

Here's a list of what this package includes:

- Anything caster gone. Improved player buff window.

- Brand-new targeting/hp window; Designed it from scratch with our class in mind, it includes: A small fatigue/casting/breath gauge, a large HP gauge which now reflects the warrior frenzy state (30/45%). And a mob gauge below.

- Minor GUI trims, keeping the 800x600 player in mind (if you don't like the hotbutton setup edit the hotbutton xml file or reload the original one saved as .original).

- 3min duration Arista timer.

- And of course all the original Vert eyecandy you've come to expect.
