This post touches on the Diablo II UI by Ameril Dan`nodel for 1280x1024 resolutions (original UI base by Soes, customized extensively to fit my preferences).
This post will be edited as needed for updates to this UI.
This thread will be for suggestions, comments, complaints and special requests for this UI.

Take a peek at the screenshots. (http://www.eqinterface.com/downloads/showthumbs.php?id=3669)
Let's download this bloody thing already! (http://www.eqinterface.com/downloads/fileinfo.php?s=&id=3669)

Features:
-> Customized Inventory Window.
-> Customized (combined) Player/Hotkey Window...the health and mana *baubles* are the in thing, now!
-> Modded Party Window.
-> Modded Target Window.
-> Modded Chat Windows.
-> Two extra chat windows, one filtering Melee Spam, the other filtering Spell Spam!!
-> Customized Buff and Short Duration Buff Windows.
-> Customized Combat Ability Window (originally by Zutix).
-> Customized Tracking Window (originally by Valeanx).
-> Customized Alternate Advancement Window.
-> Customized Leadership Ability Window (originally by Vert).
-> Modded Pet Window (includes a tab for *your* equipment, a tab for your pet's health and commands, and a third tab for the buffs your pet is sporting).
-> Modded Spell Bar.
-> Customized Spell Gem/Icon set (gems and icons represented by Warcraft III pictures).
-> Modded Spellbook (credit to T.King)
-> Modded Window Pieces (credits to T.King and Drakah)
NOTE: All graphics pertaining to Diablo II are copyrighted by and property of Blizzard Entertainment and Blizzard North Studios. This UI only simulates the Diablo II interface by re-created graphics (from scratch or from screenshots...most of that credit belongs to Soes on an astounding job). You can check out Blizzard's Diablo II website HERE (http://www.blizzard.com/diablo2/).

I am, at this moment, assuming they aren't making any actual changes to the EQUI_AAWindow.xml file, since the Alternate Advancement Window's ability descriptions don't go into an extreme amount of depth for effects, but in the case that they do, here is how to quickly fix any errors the modded file might cause (make sure this is done AFTER THE PATCH *and* AFTER YOU HAVE RUN THE PATCHER AND DOWNLOADED *ALL* THE UPDATED FILES, otherwise this fix WILL NOT work if they updated the AA Window's XML file):
1) Delete the EQUI_AAWindow.xml file out of the custom folder.
2) Open Notepad.
3) Load the EQUI_AAWindow.xml file from the default folder.
4) Use the Find command (Ctrl+F...isn't that wonderful? Just like in EverQuest. Heh) and do a search for the following: FillTint
5a) Directly below that line, you should see the following:
<R>240</R>
<G>240</G>
<B>0</B>
</FillTint>
<LinesFillTint>
<R>0</R>
<G>220</G>
<B>220</B>
b) Change that to:
<R>220</R>
<G>150</G>
<B>0</B>
</FillTint>
<LinesFillTint>
<R>0</R>
<G>80</G>
<B>220</B>
6) Select the Save As... function under the File menu.
7) Save it with the same name into your custom folder.
8) All done...wasn't that easy?
