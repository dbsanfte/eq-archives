I've had that set up in the old interface. I still have to mess around with the new one. In my old interface I had:

/hitsmode my m2 112 748 10 0 0 0 300
/hitsmode me m2 612 748 10 0 0 0 300
/hitsmode other m2 362 260 10 0 0 0 300

This split up my hitsmode into 3 parts. The left window would be me hitting my opponent, my right window would be whatever was hitting me, and the top window would be other hitting other.

You'll have to fool around with the coordinates, as they're set up for a 1024x768 resolution. Good luck.!
