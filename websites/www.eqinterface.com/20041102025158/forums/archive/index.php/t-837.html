Dalink's 90 Minute COUNTDOWN TIMER

Extract these files to whatever directory you use for your User Interface.

This Replaces the Friends Window, To Display the Timer press alt+f on your keyboard.

To restart the timer, type /loadskin <YOUR UI DIRECTORY> 1

The "1" switch is very important, this is what keeps your window positions.

The Ui Directory is the directory for whatever Ui Directory you extracted these files to.

The timer counts down for 90 minutes, at which time the timer restarts.

The timer does remain acurate through zones. In other words, it keeps counting while yer zoning.
