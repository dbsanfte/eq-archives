Cairenn
02-02-2003, 02:32 PM
There is absolutely no reason to add an EQUI.xml file to your custom folder if there isn't one already there. If there *is* a custom EQUI.xml file in your custom folder, then you should only need to make whatever changes are required to make it current as per the Patch Fixed forum.

Copying *everything* from in the default EQUI.xml file into your custom EQUI.xml file, overwriting the existing information, merely turns your custom EQUI.xml back into the default one, in which case why bother having a custom one to begin with? All you've done is overwrite whatever had been customized about it and made a copy of the default one.
