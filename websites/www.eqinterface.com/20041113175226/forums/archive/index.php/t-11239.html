I didn't particularly like the twist factor of the default target indicator, so I made the following:

Type 1:
http://www.mythal.com/images/indicator1.jpg

Type 2:
http://www.mythal.com/images/indicator2.jpg

The two lines of runes move, one clockwise the other counter-clockwise. Color indicates con, as normal. I could have used more colors in the runes, but as there are already 12 frame files, using more colors would have meant a minimum of 36 files just for the target indicator, and I thought that was a bit excessive :b

The first type has slimmer runes, and I have to admit looks better. However, I saw a bit of fuzziness when the indicator got stretched (i.e. on larger-sized targets). I made the second type to be more visible, at the cost of some visibility on the outer-rim runes.

Let me know what you think please. Any suggestions welcome :)

You can download the files here:

Type 1 indicator (http://www.mythal.com/indicator1.zip)

Type 2 indicator (http://www.mythal.com/indicator2.zip)

Thanks :)
