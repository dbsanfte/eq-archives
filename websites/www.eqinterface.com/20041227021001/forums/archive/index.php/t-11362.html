<img src="http://www.eqinterface.com/images/sections/livechat.jpg" align="right">Alan VanCouvering hosted a live chat this evening on EQLive.

Here are the highlights I got from taking notes, while trying to save out the entire chat session.

Stamina Bar is going away. This will become Endurance
"Zing" type spells such as Acumen and spells that effected Stamina will not work to improve Endurance recovery
At this time there will be no Endurance recovery spells in the game.
Changes will be on Test soon, and added to Live Server on Dec 16th Patch
Changes in disciplines will effect all classes, new Melee Skills will be provided via Tomes for Rogue, Warrior, and Monks
Tomes will be vendor sold under 60, and partchment style for 61 and up.

The whole chat session can be read and commented on Here (http://www.eqgui.com/showthread.php?s=&threadid=11364)
