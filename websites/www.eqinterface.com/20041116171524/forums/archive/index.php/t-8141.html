Vert graphics don't lag the UI whatsoever since they're just replacement windows_pieces##.tga files. I used a look based off Vert because it cut a pixel off the borders of the windows and made info more easily viewable when compressed so much. If you'd like to remove the vert look (which is just 7 graphics files, delete all .tga files EXCEPT nvntry.tga and compass.tga, if you want the default inventory, delete nvntry.tga and EQUI_Inventory.xml).
