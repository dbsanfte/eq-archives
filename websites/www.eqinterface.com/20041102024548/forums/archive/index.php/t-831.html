Caution:
This theme is ONLY for the resolution 1280x1024

The installation is similiar to the Quartz theme, so just read the *.txt and folllow those instructions.

One tiny feature are the playerstats, I used the friendswindow to draw them, so hit "alt+f" to pull them out and see some stats monitored.


http://mitglied.lycos.de/bohorroth/industrial_v1.21.jpg

Download 1.21 here (http://mitglied.lycos.de/bohorroth/industrial_v1.21.zip), works with the recent patch

- Changed the group hp percentage colors to white

- Removed the playername

! Don't forget to overwrite the UI_playername_servernumber.ini again, because I changed the chat window positions.
