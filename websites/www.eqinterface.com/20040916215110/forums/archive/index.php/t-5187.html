This DOES work with the Jan 9th patch. It includes all 15 buff slots and 6 song slots. The only things this does not include are the Invite/Disband buttons and the Player_Window. I use hot keys for Invite/Disband which are available from the Actions window anyway. The Player_Window is currently 2 pixels high and is sandwiched between two windows to act as a red flashing combat highlight.

It has a modified Vert skin with Vert buff flags, Thangor's compass (updated for the Jan 9 patch), Heals Data's every merchant slot window with a couple of minor modifications and a modified Xirin's Castspell window. I started out with Enok's UI, but I am not sure how much of it remains. I set up every resolution from 640x480 to 1280x1024, but it starts to break down below 576 height. If you are using a *** x 480 resolution, use the small buff and small short duration buff windows or the .ini file will not work.

The main thing I wanted from this setup is the group health bars which include pet health bars that are selectable. I also got tired of digging for clicky items, so I just included the entire inventory in the hot button window. I fixed the graphics file and it looks fairly nice even in default mode. The viewport is nearly uncluttered except for the CastSpellWindow (spell gem bar) which is something you can look around to find a replacement.

Probably worth downloading just for the modified Vert window_pieces (complete with Song button in the windows_pieces04)).

The image is 1024 x 768 and has a viewport setting of 769 x 576. This is meant as a mod for more limited systems.
