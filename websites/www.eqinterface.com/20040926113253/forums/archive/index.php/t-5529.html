The easiest way to show a layout is to cut and paste parts of a screenshot into the window you want. Actually doing the code isn't that tricky really, I get by with just modifying the default code mostly (and grabbing any neat tricks I find in other's code). The layout of the window is the real trick. Get that right and the rest falls into place. Get a layout and post it (either for the window or the entire screen in a 16 color paint format that saves server bandwidth). Remember, all windows are rectangular, so any curves should be considered as being within a rectangle.
