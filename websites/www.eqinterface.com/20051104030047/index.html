  Keith Parkinson 1958 - 2005
Posted By: Cairenn - 11-01-2005 09:16 PM - Replies (1)
As I am sure most of you know by now, renowned fantasy artist Keith Parkinson passed away late last week.

I know many of you no longer remember me, but I had to post something. The problem is, I have been sitting here all day, trying to figure out what to post about this. I ... just can't come up with the words. I was at BlizzCon when I got the news he had passed on. The world tilted and turned black, and I found myself in a heap on the sidewalk with tears streaming down my face. The fantasy world has lost one of the truly greats, and even worse, the world has lost one of her best.

You can read the post by Brad McQuaid of SGO, where Keith was working on the upcoming Vanguard:Saga of Heros mmorpg, here.

Donna, Nick, our hearts and prayers go out to you.

Rest safely Keith.

  Halloween in Norrath
Posted By: Kudane - 10-26-2005 08:19 PM - Replies (0)
EverQuest Fans Get Ready for A Virtual Halloween

Starting this Friday (Oct. 28) fans of the original EverQuest can take a break from the latest expansion, Depths of Darkhollow, and enjoy a little Halloween-flavored fun. Players can look for an old man to teach them how to trick-or-treat and your practice will yield tasty candy treats. Meanwhile, a creepy bone collector will request help on collecting several rare bone artifacts and a story teller will spin some rather frightening tales, with the proper ticket of admission. In other areas, the Halflings are more hungry than usual and their craving for pumpkin will lead to a vampire mishap. For more information, players should seek out Zigand Ribshard in the Plane of Knowledge. He has all the details!

Fresh from the Norrathian News Wire!

  New Default UI pushed back
Posted By: Dolby - 10-26-2005 02:43 PM - Replies (0)
The new default interface that was scheduled to be patched into the game this morning has been pushed back a day. The patch is now set for the morning of October 27th.

Since this is a change to the default interface many mods that rely upon the default graphics may no longer work. If you are not interested in the new default interface you can make a backup right now of your default interface folder before they patch. You can read more detailed information here at the offical EverQuest forums.

As always with all interface changes I'll post a compare in the patch forum as soon as possible. However I'm headed to California for the weekend this Thursday so I wont be able to post it until I'm settled into my hotel room.

Upcoming Update

The next scheduled update is planned for October 27, 2005. Here are some of the things that are currently planned:
  • New Default UI Skin
  • Modifications to DoD Missions & Loot
  • EverQuest will now use the Operating System's mouse so players can easily go between applications while in windowed mode
  • Take a look at Kithicor for some holiday fun
  • Players will have the ability to make their helms invisible
  • Improved controls in the guild bank
  • New command for opening a web browser from the game command prompt
  • New Audio Triggers command
  • Fix for the Pet HP for items
  • Fix for the loss of mana/HP while zoning
Please join us on the Test server to see some of the changes.
