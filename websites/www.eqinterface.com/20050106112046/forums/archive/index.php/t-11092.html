The chances are that you don't have current .tga files. Those are the actual graphics, as opposed to the .xml files which are the coding files that define the various graphics.

If you go to this (http://www.eqgui.com/showthread.php?s=&threadid=6589) thread, it lists all the patches, and which files were changed in each patch. As you look at each of the patches, you will see that some have window_pieces0X.tga listed as having been changed.

You need to compare each of those to the one of the same name that you have in your custom file. Chances are good, from what you describe, that you will discover a lot of buttons on the ones in the Patch forum that aren't on yours.

What to do about it?

a) see if the author of your UI has released updated .tga files. If not;

b) you can delete the appropriate files from your custom folder, letting EQ pull them directly from the default folder; or

c) modify your existing .tga files yourself.
