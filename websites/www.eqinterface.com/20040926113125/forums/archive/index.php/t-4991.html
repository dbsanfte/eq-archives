Ok you will need to become comfortable with XML programming to do this but here goes.

With in the xml files for your windows that you are wanting to change there will be sections for the gauges. it will be different depending on the window you want to edit but as an example the one for the Groupwindow first group member would be <Gauge item = "GW_Gauge1">. once you find this section you need to find the entry in it that is labled <FillTint>. In this section is defined the RGB values for your color. You will need to find these values for the two that you want to switch the colors of and notes what the values are. You will need all three parts of the Red Green Blue values to make the colors you are looking for so make sure you get all of them for the olor you want. Next you just change the one you want ot change to the new three part color code and save the files. you will need to do this for all of the gauges that you are looking to change.

Not entirely sure whether you can actually change the color of the out of zone gauge color but if someone else knows about that part they can chime in here.
