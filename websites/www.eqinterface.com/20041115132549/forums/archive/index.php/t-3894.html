For all of you that are having trouble getting the fix for the Pet Window XML code that has reversed Exp and AAExp, please follow the procedures below.

Click the link from my post and the XML code will open in a window.

From that window click "File" then "Save As" and point the directory list to your uifiles/wc3whichever directory. Look for the file "EQUI_PetInfoWindow.xml and double click it.

You will be promted to overwrite the file as it already exists. Click "Yes" to overwrite it.

That should get you going.
