Latest Updates

Featured Interfaces
  • Thomus King has released a new Wizard (Ice) class animation, updated the Stone & Forest Inventories, and announced the G2 Spell Icons are now in the Supporter's Forum (screenshot is available at the linked page).
  • Tellgar has released Iceyblue v2, updated for LoY and including a new Pet window.

Complete Sets
  • Jethal has updated the NightElf UI, with window redesigns and some fixes.
  • Xen_Triton has updated Xen UI (for pure melee)to v1.1, and released Shinjuku UI for casters and hybrids.
  • Lerhos has created Beastlord UI for 1280x1024 resolution.

Mix & Match
  • Hortania has put together Horty, for all classes.
  • Tobin Itharel has put together LoY1280, a viewport UI.
  • Sanarie1 has put together Neptune and Cannbis Mix&Match, based on Vert.

Bazaar Window
  • Rath has created the Rath BazaarSearchWindow, with longer pulldowns and no Results tooltips.

Group Window
  • Jerdite has released Trex Group Window, set horizontally.
  • Kwoung has updated the LFG-Group Window to v2.

Hot Keys
  • Kaay ess has created Kaay Hotbutton WIndow, with inventory slots and player stats.

Inventory
  • Anatis has released more Inventory Portraits for T.King-style inventories:
    - Warrior Epic
    - Dragon

Miscellaneous
  • Jethal has created two new casting bars:
    - SwiftyCast, using Swiftwind-style graphic
    - SkeletonCast, with a seated skeleton graphic.

MP3 Player
  • Talestrike2 has released ColdFusion, based on the Winamp skin.

Story Window
  • Uuena Van Light has created ResearchPlus, from Ramsus original, including compnent types and WIS caster research.
  • Thong3320 has released quest walkthroughs for the Druid Epic and the Coldain Prayer Shawls
  • Adrastea has created a PoP Portals List, with locations and descriptions.
