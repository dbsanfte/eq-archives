Try opening your character's UI ini file (example, mine is UI_Cairenn_eci.ini), look for:

[Main]
UISkin=XYZ

Set that to Default instead of XYZ.

Now open your eqclient.ini file, look for:

UISkin=XYZ

Again, set that to Default if it isn't already.

That should hopefully make it so that you can at least log in.

After we get to that point, we can move on to next steps to try to get this sorted out with you.
