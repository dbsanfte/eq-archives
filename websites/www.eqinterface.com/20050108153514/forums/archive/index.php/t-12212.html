AYA Chat windows can be re-sized. Go to the "Optional" Folder. Then to the "Borderwndow" Folder. Copy the "EQ_Chatwindow.XML" File to your UI.
Log In and re-size.

01-19-2004, 11:56 PM
its also easy to go into the chat window xml and edit the following screen item variables to read:
<Style_Border>true</Style_Border>
<Style_Sizable>false</Style_Sizable>

You can now move them around.
