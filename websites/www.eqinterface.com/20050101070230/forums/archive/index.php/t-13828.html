Problem is that the EQUI_Animations.xml file is out of date. Specifically, this ui's version of the file only defines dragitemxx.tga up to dragitem25.tga. The current version of EQUI_Animations.xml has definitions u to dragitem27.tga. If you open up the ui's custom EQUI_Animations.xml file with Notepad, and search for "dragitem" and scroll down a bit, you should see what I mean. After dragitem25.tga is defined, add the following code to your custom EQUI_Animations.xml file

<TextureInfo item = "dragitem26.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>
<TextureInfo item = "dragitem27.tga">
<Size>
<CX>256</CX>
<CY>256</CY>
</Size>
</TextureInfo>

Save the file and exit, and then everything should be fine.
