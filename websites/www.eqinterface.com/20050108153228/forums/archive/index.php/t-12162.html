Hello Kutter,

it is right. There is some kind of "free" version of LoY.
With LoY had been introduced a map system of nearly all zones.
If you do not registered LoY itself you can use this map system only in Newbie Zones and Starter Cities. This is a feature to make it for new players easier to navigate. The skill Sense Heading had been set to 200, too.
Else you can use the map in nearly every zone (nearly, because in Plane of Hate it does not work and maybe somewhere else too).
So you have downloaded LoY completely but can only use the "free" map system as mentioned above.
The rest will work once you registered it.
