09-28-2002, 01:48 AM
The Ragebringer pointer may not work with this mod as is because I edited the cursor code to include animation changes for the shaman epic animated pointer which Remelio released. You need to edit it out and put in the original code that vert had for his pointer. The original code is:


<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>mouse.tga</Texture>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
<Hotspot>
<X>4</X>
<Y>3</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>

If you have problems switching out the code feel free to contact me and I will set you up with an animations file that will work for the Ragebringer pointer.
