1) Use a text editor (Word, WordPerfect, NotePad).

2) Open up the *modified* EQUI_SelectorWindow.xml file (the one you want).

3) Do a search/find for:

<Gauge item="Casting_Gauge">

4) Change that to read:

<Gauge item="SELW_Casting_Gauge">

5) Now do a search/find for:

<Pieces>Casting_Gauge</Pieces>

6) Change that to read:

<Pieces>SELW_Casting_Gauge</Pieces>

7) Save. You should be gtg now.
