Ok, VI is not making it easy for skinners/mod'ers. I was thinking that if you have 2 computers that perhaps you could simplify the process of the whole "Edit XML -> Run EQ -> Check custom UI -> Log off EQ -> Make changes to XML, rinse, repeat" procedure.

I'm at work at the moment and cannot verify any of this, but I'm thinking that if you logg in EQ on one machine, then on the 2nd computer connect to the 1st one, open up the XML files you need to edit from the EQ machine's hard drive, make modifications and save the XML files (again on EQ machine). Then on your EQ computer do a "/loadskin <your_modified_skin> 1"

Now this *should* work assuming that:
1) Your 2 computers are properly networked and can access each other's files;
2) EQ doesn't lock the XML files when it runs. Otherwise you will not be able to save your changes :( I hope that this is not the case;
and 3) Your changes don't make EQ crash, otherwise might as well edit the files on your EQ machine since you're on the desktop anyways :p

I'll check this tonight, but if anyone can try it in the meantime it'd be appreciated.
