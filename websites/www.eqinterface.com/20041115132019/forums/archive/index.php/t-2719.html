Thangor
10-11-2002, 02:23 AM
Those buttons are basicly invisible. I didn't remove the code for them because I thought perhaps at some later time I may want to add some kind of "done" button or "mem" button of some sort.

As for the colors in EQ... well it's not going to look as good. They are stored as 32bit colors, but I don't think EQ renders them quite right. My theory is that it has something to do with the gamma code they have in there. For some reason EQ boosts the gamma up way high. In my original photoshop file, I first draw the picture as I would like it to look, then I do a transparent black layer at about 20% or so to counteract the gamma when I save the TGA files. That's really my best guess as to why it doesn't look as clean in game as it does if you just view the files in photoshop.

I've heard there is code to control the gamma for a texture, however I'm not sure what that code is as I have not seen an example of it so far.
