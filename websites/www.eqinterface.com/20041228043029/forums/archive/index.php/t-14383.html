Amitan has started a thread with todays Patch Message (http://www.eqinterface.com/forums/showthread.php?t=14374)

for the UI Community, the big things are

UI Changes
-------------------
- It is now possible to display your mana as a numeric value on your inventory screen. *
- By mousing over your buff icons, you will now be able to see the buffs remaining duration. *

Both of these items were discussed in the Meeting I was in (and I am sure others) at the guild summit! its good to see the support still coming from SOE from the summit.

Neither of these changes "should" break the UIs, but watch for patches to your favorite UIs just incase!

UPDATE - It has been said that the container xml has been changed and could cause a crash if not updated. So if your UI changes bags/containers use cautiously
