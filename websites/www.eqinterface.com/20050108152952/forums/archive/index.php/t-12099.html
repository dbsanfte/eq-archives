You don't need server numbers anymore.

The format used to be ui_vitaee_33.ini for a character named Vitaee on the Bertoxolous server. The format is now ui_vitaee_bertox.ini for the same character and server.

In other words, use the server name and not the server number. This was changed a few patches ago. However, you must use the correct "nickname" for the server. (i.e, in the above bertox is used instead of bertoxxulous). Easiest way to find the correct server name is to have a character on that server, and check your Everquest directory for the name of the ini file corresponding to that server.

For instance, if you have a character named Whoami on The Nameless server, then you will have a file in your EQ directory called ui_whoami_xyzabc. In this case, xyzabc is the correct "nickname" for The Nameless (note, I have no idea what the correct "nickname" for The Nameless really is).

This file is created automatically for every character. To avoid confuison with existing characters, you might want to create a new character on the server in question so that you only have to deal with a fresh ini file and not any older existing ones.
