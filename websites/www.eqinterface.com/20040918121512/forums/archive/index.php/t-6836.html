So what to do if you wanna try it out :

1 - Aledark UI (aledark.free.fr/UI/Aledark.zip )
2 - Extract zip file to your uifiles folder which is in the everquest folder.
3 - Go to game and type /loadskin Aledark


if you want your screen to have the same viewport and window position as mine you can also download this :

[/URL] aledark.free.fr/UI/UI_Aledark_43.ini

put it in your Everquest folder but rename it under your character name.

for instance the file should be named UI_Solthan_43.ini if your character would be Solthan.

the number 43 is Morell Thule server number.
