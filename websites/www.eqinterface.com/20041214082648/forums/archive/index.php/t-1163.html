I want to customize my buffs and targeting windows. I downloaded the mods I want. Now where do I unzip them into? Maybe I screwed up, but I unzipped them into the default folder in the uifiles folder, but nothing happened when I logged in. Same old default skins. Help?
IshiroTakagi
08-18-2002, 01:11 AM
If you put them in the default directory, then EQ will "fix" them when you patch. You need to create a new directory, for example call it "newui". Put your files in there, then once you get in game, give the command "/loadskin newui" and it will load the new skin.
