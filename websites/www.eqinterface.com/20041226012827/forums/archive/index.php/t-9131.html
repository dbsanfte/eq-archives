These are the updated files for the infiniti2_3 interface which add the Shared Bank slots, as well as a DOT Damage filter, MouseWheelZoom toggle and so forth. These changes were part of the June 11 patch, and these files are up-to-date as of today (June 24th). To install them, just unzip the file into your infiniti2 uifiles directory (after backing up your originals).

Note: I have not made any changes to the XML code or the interface as a whole. I have only added the extra funtionality for anybody who wants to include it.

A big cheer for infinity9 who did all the real work and created a damn nice interface!!
