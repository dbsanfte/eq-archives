first go into you folder where you UI is and delete everything then just put in the folder the files you want to change. Now go into EQ and load up your skin in the options window under loadskin and it should work. You do not copy anything from the default folder. EQ draws from that folder what is not in yours automatically. Your folder is there to tell EQ what you want to overwrite from the default folder and that is it. Hope this helps.
