Quoted: The problem is that the new file EQUI_CombatAbilityWnd.xml is using the files window_pieces06.tga and window_pieces07.tga. That is the reason your combat window looks all jumbled.

You need to copy the EQUI_CombatAbilityWnd.xml file from the default UI directory into your custom UI directory.

Then make a copy of window_pieces_06.tga and window_pieces_07.tga from the default UI and rename them to something like original_windows_pieces_06.tga etc.

Then modify your EQUI_CombatAbilityWnd.xml file by replacing all window_pieces_06.tga to original_window_pieces_06.tga and window_pieces_07.tga to original_window_pieces_07.tga.
