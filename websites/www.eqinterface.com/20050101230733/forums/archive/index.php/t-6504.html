The spellbook had its own EQUI_Animations.xml file that came with it. That file defines what graphics files the UI uses. This file has been patched a lot by EQ lately, and you will need to do several patch fixes (found in the Patch Fixes forum) to fix this.

You might tell me which spellbook it is you are trying to get to work, I might be able to fix it so it doesn't need the Animations file at all. Personally, I prefer the T King spellbooks myself. ;)
