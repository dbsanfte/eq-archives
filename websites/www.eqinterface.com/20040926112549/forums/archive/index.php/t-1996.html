The bars are defined in the window_pieces01.tga image. Anything that is transparent in the TGA will not be displayed in the UI, so if you want (for example) a DNA helix instead of five bubbles, you can do that simply be replacing the image of the bubble-bar.

If you change the *size* of the bar, then you'll have to change the EQUI_Animations file to tell it where the new graphic is, what size it is, etc.
