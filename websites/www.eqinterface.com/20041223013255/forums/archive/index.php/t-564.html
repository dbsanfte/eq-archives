Sounds like you are not creating a separate directory for the necro skin in you "uifiles" folder in your "everquest" directory. Put it in there and make a copy of your original default directory place it in the same directory and name it "necromancer", then take all of the files from the mod folder and drop them into the new "necromancer" folder and say yes to all when it pops up. Delete the now empty folder that the mods came in. When you are done open your eqclient.ini file and scroll all the way to the bottem and make "UIskin=necromancer" then save that file and log into the game. Once in the game type /loadskin necromancer and camp back out then relog again into the game. Bingo new UI. Beware though this is causing people to crash to the desktop when in the bazaar and typing /bazaar.

If you just put it in the default directory as is the patcher will overwrite it every time.
