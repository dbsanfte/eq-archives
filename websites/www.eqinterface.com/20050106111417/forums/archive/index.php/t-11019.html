Since I had my player window using target window information, and only the target window can have the color change:

Back up the EQIM_PlayerWindow.XML and EQIM_TargetWindow.XML files
Copy the contents from one file into the other so that EQIM_TargetWindow.XML now contains all of the player information that used to belong in EQIM_PlayerWindow.XML, and visa-versa In each file, there is a Screen Item XML section near the bottom that contains all of the pieces sections. These need to be put back so that TargetWindow is in the EQIM_TargetWindow.XML and PlayerWindow is in the EQIM_PlayerWindow.XML.

Then your TargetBox con colors will work. You'll have to edit the TargetBox.TGA file to fit the dimentions of your TargetBox. I'm still working on that part of it so it does what I want it to do. I'll publish it for others to use when I'm done with it.
