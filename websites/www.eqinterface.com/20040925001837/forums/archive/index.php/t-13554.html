The addition of the /autoinventory command was very nice, but it would be nicer still if it could be bound to a keypress.

Some other actions that would be handy to have /commands and key bindings for are:
Dropping an item.
Combining a trade skill container.
Accepting a confirmation dialog.
Buying or selling from a merchant.
Accepting a trade.
Looting all items or the first item on a corpse.


Currently these actions all require a mouse click. Mouse clicks are slower and cause more wrist strain than keypresses.
