The problem is tied to missing buttons that are required in a bunch of windows. I'm guessing you started your work with files that were current before Ykehsa. Unfortunately, when VI adds buttons to the UI, you MUST add them to your XML files or it will error at the very least if not dump the UI or worse crash to desktop. To this end, go take a look at the default UI, and note the 5 new buttons on the window selector, the new LFG button in the group window, and then the charm inventory slot on both the inventory window and inspect window.

You can most likely copy the text blocks from the default files to your modified files and think nothing else of it.

Hope that helps,
Kirbol
