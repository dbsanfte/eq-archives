UI News & Updates



Latest News - Latest Updates - Patch Fixes - EQ Interface News




Latest News

  • The Legacy of Ykesha Cartography System has been announced! The maps are simple line drawings, but include basic EQ-generated maps plus the ability to add up to 3 layers of your own! It features various methods to draw lines; Text labels can also be applied. An interactive demo is included on the page showing how teh Height Filter works, and zoom support is included for mouse scroll wheels (those of us w/o wheels still have buttons for zoom). Bottom line: EQ Atlas it aint, but it's a nice addition. Authors, it looks like there are two new windows to mod or add to your UI's!

  • Theres been some updates to the EQInterface Home, including the addition of Cairenn, Drakah, Kanon (Aya UI), Telgar (IceyBlue), and Xymarra to the Featured Interfaces area!

  • Drakah has submitted a suggestion to SOE that they develop a XML editor for Everquest; read his post and the reply.

Errata  the Ranger turns Red!

Seems the News Ranger missed a few uploads back in January  my apologies to all the authors below.
Thanks to Sylphan for catching these and pointing them out- and please always let me know if I miss another!

Hot Keys
  • Katn released a SmallHotButton window, to conserve screen space
  • Hotharn created Hoth HBs, a Hotkey/Inventory combo based on Vert 1.1

Inventory
  • Weredragon released InvModel1, an Inventory window with slots arranged appropriately over a 3D model
  • Tobin Itharel created the 280x280 Inventory, with all player info removed  this window does include the new Charm slot
  • Fyste released ThinInv2, a modification of Waeolwens Thin Inventory
  • Kasca released Dolphin Inventory, the T.King Draconic Inventory with a Dolphin image in the center

Pet
  • Soontace created a Pet & Target Window, which includes target health, pet health, the casting gauge, and a one minute timer
  • Kuvasie created a T.King Pet Window, based on the T.King Celtic Spellbar
  • Thlayli released a Bard Pet Window, specifically for Bards needs
  • Earthscum updated his Necro Spectre Pet Window to v2.1
  • Ieatacid created a Horizontal Pet Window to conserve screen area
  • Onetwo1212 created a Melee Pet Window, replacing all pet info with Inventory and Weapon slots
  • Chronwaas released KireVertPet, a modification of the Pet tab of the Vert Player/Pet Window

Player Info
  • Canda created the Canda Player Window, designed with pet casters in mind
  • Shadrak released ShadrakPW, displaying all stats and resists
  • Dins created a Transparent Player/Target Window, with only the basic player and target info
  • Jolon-aislyn released a Melee/Pet Player Window, designed for melee pet classes, based on Vert v2
  • Chery Nobyl created two player windows, Transparent 1 and Transparent 2; 1 displays more information, while 2 is more streamlined
  • Tobin Itharel released a Player-Target Window, with both gauges displayed in the same window
  • Allissiya released T.King ColorChange, based on Kuvasie's T.King player window, where the gauge and label colors have been modified
  • Chronwaas created Elfy PW, a modified version of Avi's player window, to include T.King gauges
  • Kuthbert released PWindow. a modified version of Robynn Merrilin's, with larget numbers
  • Murtano Laorte released TheWorks v2, displaying "all the info on screen all the time"
  • Alisaunder created Corren's Player Info, using T.King artwork\
  • Chronwaas also updated ChronPlay to v2, adding additional info

Spellbook
  • Sylphan created the Mini Spellbook, displaying smaller icons and larger text
  • Nanan released a trimmed-down SpellBook Window, with a narrower profile and new black background
  • Talyn's created the Pocket Spellbook (later updated to v2)

Spell Gems
  • Aridale Belmont created the MTGSpells, spell and gem icons based on actual Magic: The Gathering cards
  • Sc2kmayor released v2 of Thangor's Ticker Spell Bar, to correct some graphics issues
  • Katn created the Box Spell Bar, in two vertical rows with the Book icon in the center
  • Sc2kmayor also updated the SonaPixie Spell Bar to v2, with graphics updates

Target Window
  • Kuvasie created a T.King-Based Target Window, using T.King's warding device and gauges
  • A_Nolan released a Player/Target Window, displaying both Player and Target gauges
