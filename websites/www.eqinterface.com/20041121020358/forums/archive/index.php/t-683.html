You specify the size <cx> and <cy> in the Animations file, as the size of the actual image. Ie/ An image 100x100 pixels would have a:
<Size>
<CX>100</CX>
<CY>100</CY>
</Size>

Now when you actually go to draw the image, you would use another wrapper object like a Gauge or Button. Here you specify a new size. To shrink the image down 50% you would use:
<Size>
<CX>50</CX>
<CY>50</CY>
</Size>

And to cause the image to be streached use:
<Size>
<CX>200</CX>
<CY>200</CY>
</Size>
