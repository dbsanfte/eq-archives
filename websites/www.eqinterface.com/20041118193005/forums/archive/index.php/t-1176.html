the help.html file is one of VI's files and is not really part of the UI. Therefore it will always get patched if your file doesn't match the CRC of the one on the patch server.

There are ways around this, however, as pointed out on other forums, circumventing the patcher to modify this file is technically breaking the EULA and can get you banned (although its unlikely they could ever tell you had modified the help.html file).

A better way to do this has already been done by others to add maps and whatnot as images in various windows of the UI. Check out the Map UI addon at http://eqtoolbox.not.net/modules.php?name=Downloads&d_op=viewdownload&cid=31

Basically though... you can't stop the patcher from writing over that file.
