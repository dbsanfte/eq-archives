Scurg,

I posted about this in response to Karmatic not too long ago, but here... of course, you need to use the windowed chat files also.

For the longest time I was trying to figure that out too... well... found out I had to do the following changes to the bottom of the
EQUI_ChatWindow.xml file in AYA's UI:

<Style_Titlebar>true</Style_Titlebar>
<Style_Closebox>true</Style_Closebox>
<Style_Minimizebox>true</Style_Minimizebox>
<Style_Border>true</Style_Border>
<Style_Sizable>true</Style_Sizable>

Make sure those lines say 'true' (without the quotes) and you will be able to close tell windows and minimize and move them around. Though, it does provide a little less lines of visible chat, but I think it is worth it if tells are going to separate windows anyway.
