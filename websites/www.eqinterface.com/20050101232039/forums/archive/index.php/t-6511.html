This might have to do with the size of your window. If the window is smaller than the width or height of the buff icons, the buff icons don't show up. The window must be 1 pixel bigger than the width/height of the buff icon. If you can't resize the window in the game, you will need to edit the UI_playername_##.ini file in the EverQuest directory for your character

[BuffWindow]
Width=180
Height=326

[ShortDurationBuffWindow]
Width=180
Height=135

These numbers are approximate, but should work. If not, add 10 to each number and tell me if there are any problems.
