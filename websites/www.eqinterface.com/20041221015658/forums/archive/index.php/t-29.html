OK to clear a few things.

the /loadskin command does load the coordinates stored in the XML files, so if you don't like where they go you can modify your mods coordinates (although there is still a bug with it, I explain in another thread)

No you don't need to log out to get your UI changes UNLESS they are of a graphic nature (eg. someone has actually modified the .tga files). Not sure about zoneing as I never tried to move zones while messing around with files but character selection screen seems to be far enough. The graphics seem to only be loaded once (at the point you log in).
