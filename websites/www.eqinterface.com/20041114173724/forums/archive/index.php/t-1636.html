The file is in the download sections, this is just a "Hey look" thread :)

http://www.personal.psu.edu/users/p/h/pht101/images/locationguide.jpg

The way it works is under each geographic location is an x, y location...

in the x y location is one of three symbols, 0, +, -

0, no change
+, Number goes up
-, Number goes down

So if you are at 0, 1000 and need to get to 1000, 500, then you need your X to go up, and Y to go down, looking at your compass you can see you need to head North East :)

This should help a lot of people who can't do it in their heads :)

Btw the locations are dead on, I did a lot of tweaking so that if you get the little arrow centered in the geographic location, the number labled 0 shouldn't move much.

Hope you like it!
