it's basically just a customized interface built on a quartz foundation with some t-king stuff. i think i changed the colours to some type of blue shade on nearly everything from the t-king inventory window to the text box windows. put some custom images in it also. brand new note window i made earlier today (chick in my sig is in it : D ), never seen a customized note window til now.


it was designed with a bard in mind - hence mask + boots slots in the bag and t-king spellbar replacing the quartz one.

special thanks to t-king + remelio of brell serilis for the great art + coding for the t-king work, atheen for the text-based locwizard (ouch it must have hurt to write all that ... ), cairenn of e'ci for showing me what i was doing wrong with the xml editing of the bag mod, and statix cinderblade of xegony for making the original quartz mod.

mod is based off the new quartz 1.5 which came out this week, so it should be good to go for quite a while.
