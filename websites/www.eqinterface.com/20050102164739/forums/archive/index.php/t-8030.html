I believe all I did for mine was to go into the EQUI.xml file and put <!-- before the <Include>EQUI_CastingWindow.xml</Include> and --> after it. So it will look like this.

<!-- <Include>EQUI_CastingWindow.xml</Include> -->

This worked for me as I have the breath window, target window and casting window info all in other windows.

Of course this is only going to work if the EQUI.xml file resides in your custom UI directory, otherwise it will just get overwritten when you patch to EQ if you are just using the one in the default directory.


Hope that helps
