Not quite sure which part of this you are referring to butI will try to answer with two responses

First if you are referring to the positioning and size of the windows reverting back to their original then you may want to try

/loadskin Orchids 1

This will keep the current layout.

If what you are referring to is the fact that when you do the loadskin the new .tga graphics file does not load this is because all of the .tga files load up in to the system when you first select the server you wish to play on at that time. You know that long delay you have after you select a server before it brings up your character select screen. If you watch carefully what is going on and interpret it a little you will find that this is the point that all of your .tga files are getting loaded. Because of this you can not change the .tga files while in the game by just using the /loadskin command. In order for the changed .tga files to take effect you will need to do the /loadskin command and then exit to at least the server select screen(but you may need to go all the way to the desktop) and come back in. At that point you should see the new .tga files in place.

Let me know if that did not make sense. sometimes my fingers go a little faster than my brain can spit out the information.
