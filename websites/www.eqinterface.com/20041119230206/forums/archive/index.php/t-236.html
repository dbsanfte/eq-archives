Nennayan
07-31-2002, 01:34 AM
Look for the <style> variables at the end of the file named Titelbar, closebox, minimizebox, borders, and sizable.

If you turn off everything, you'll also want to turn off sizable so that you can still move the window by dragging. This means you'll have to play around with it a bit to make it look the way you want.

Theoretically you could make a set of completely round windows, or even pyramid windows or any other shape.

One other thing, you'll probably want to either edit window_pieces01.tga or make your own file for the window borders etc, to make sure they're totally invisible.
