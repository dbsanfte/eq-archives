The Hotbuttons are actually 3 different parts in the hot button window. Each button 'slot' contains a 'spellgem' part, a 'inv slot' part and a 'button' part. This is because a button in the hotbank can actually look like any of those 3 things. If you want to mod those boxes, you would want to make your changes to all the 3parts of each button (so you'd have to make your changes 30 times) Clicking on a hotbox button in SidlWidl only shows one of the 3 parts. Use the left side menu to find the other parts of the button so you can change those as well. I think this is the info you are lookin for.
