OK, for both of you it sounds as if you try to load your custom ui, and when you do, the game pauses for a second or two as if it's trying to load something, but then it comes back with just the plain ordinary ui, correct?? If so, scroll back in your chat buffer and you'll see a message saying that it was unable to load the custom ui, so it's using the default instead.

Here's what you both should do:

[list=1]
In your main everquest directory, find a file called UIErrors.txt and delete it
Log into everquest, and try to load your custom ui the way you mentioned above.
Log out of EQ, and you should have a new UIErrors.txt file. Post a copy of this file in a new message.
[/list=1]

Doing this will give those of us on here an idea of what is wrong. Somebody will most likely be able to help you.
