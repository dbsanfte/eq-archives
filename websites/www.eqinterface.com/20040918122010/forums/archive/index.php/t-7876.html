Shadrak1280-1024
hello
This is a ui mod i made for myself using the wrap around method of window placement. this mod is made for casters/hybrids. It uses a screen resolution of 1280x1024 with a /viewport of 100/0/1055/800.
Custom Spell Icons are not included.
Windowspieces not included.
Playerwindow and target window are switched to give flashing red thingy around the target window.A few other changes made as well.

**uses-
Cairenns buff.sdbuff windows
Lunakins gauges
Tkings spell gems
also a few tking class animations.

the rest was made or edited by me.
