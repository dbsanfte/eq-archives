well this is what i do,

1. Download the UI, put it in your desktop or an easyily accesable place
2. Un-Zip all files in the UI to a folder in your everquest directory under UI files and under UI files put it in the name the UI tells you to
3. If the UI has an UI_BlahBlah_00.ini file, copy that and paste it into your main eq directory, (copy it from your UI files, or extract from the files u downloaded) If there is no .ini file, in game /loadskin _______ <--- name of UI
4. Log on and have a GOOD time :) :nana:
