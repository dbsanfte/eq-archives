Probably bad luck, since it automatically saves to your UI_CharName_ServerName.ini file. (ie, mine is UI_Cairenn_eci.ini)

The other possibility is that said file has been set to read-only for some reason. Right click on the file name, go down to "properties" and make sure that the 'read-only' option is not checked.
