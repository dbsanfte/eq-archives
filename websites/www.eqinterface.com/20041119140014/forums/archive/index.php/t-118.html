Filtering 'my hits' and 'me getting hit' to two different windows can be done, and I believe enrage and rampage are also filterable, so they can be directed to a different window than hits, I have them going to my spells window. The hitsmode option on the bottom of the chatwindow right click menu lets you change the damage messages to full message, abbreviated message, and just the damage number.

The window properties are pretty much set, and they affect all windows. But, you can find some interesting ideas to cheat this in this thread:

http://139.81.52.39/forums/showthread.php?threadid=2328

All you need to do is create 2 new chat windows,
filter the melee hits to one and the other,
filter the disciplines and spells and wahatever to yet another window if u like

Change the size of messages in the menu options

Then just set the fade to / normal to 0 percent, the chat text stays on, the chat input box and window frame disappears

Size your window or move it so the chat input bar disappears out of the screen, and stack your 2nd window on top , since the window frame will be invisible anyway

then Lock them and rename title to ' ' a space if you like

Also use ' always chat here' on the main woindow and you're set
