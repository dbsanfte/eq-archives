Let me start off by saying I'm no xml expert. I'm learning how to play with these just by doing.

As I understand it, the only way to see the output of these xml and image files is through EQ. The xml and image files we're manipulating is only part of the information needed to be able to conjure up the results. Some of the key information is hard-coded into EQ. Thus without EQ, we can't see our changes take effect, let alone be able to manipulate it as Front Page does with html.

Maybe as things progess, someone will be able to create what you're looking for but for now it isn't possible.

Hope that made some sense.
