after following the suggestion by VI to ONLY put the Modified files in my "newskin" folder, I discovered something about this download. When I extracted it into my "newskin" folder, it created a new folder for itself. So my directory tree looked like this:

UIfiles
--default
--Newskin
----compactspellbar (with the XML file inside)

I copied the XML file into Newskin and IT WORKED!

before leaving the normal files in default, Newskin was so full, I did not notice the folder. hehe.
