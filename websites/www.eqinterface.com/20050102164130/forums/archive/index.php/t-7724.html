Find the .ini file they want you to use.

In your main EQ directory, find the current .ini file (eg for me, it's UI_Cairenn_70.ini).

Change the name of the new one to match the existing one.

Copy the new one out of the custom folder to the main EQ folder.

It'll ask you if you want to overwrite it. Say yes.

Open the .ini file using a text editor (Word, WordPerfect, NotePad, etc) and make sure that this line:

UISkin=CustomFolderName

is set to whatever you named your folder.

That should be it.
