Thanks for trying out my UI.

If you are using version 2.0.2, that was made before the 2/10/04 patch so it doesn't have the new pieces they added like the "Change" button in the Bank Window and the "Combat Skills" button in the Action Window.

In my testing, the 2.0.2 version would run fine after the patch, but it does give that error message ("Your XML files are not compatible with current EverQuest files, certain windows may not perform correctly.") since it can't find the new pieces that were just added in the patch. Like I said, the UI appears run fine, but obviously you won't have access to the new buttons they added.

Once you load the 2.0.3 version you won't get that error message any more and you'll have access to the new fuctionality (buttons, etc.) they added in the patch.
