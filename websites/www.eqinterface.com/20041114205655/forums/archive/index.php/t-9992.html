***********************************************
* Zeremus Windows: Breathe
* Designed by Zeremus Honorfyre, Ayonae Ro Server
***********************************************

I work too many hours, I took a break, sometimes you have to breathe.

Added:
-Mana/End numbers to player window

Changed:
-New 3D Targeting Ring (spniy ?)
-Updated Buff and Spell Windows (Drakah)
-Compatible with OOW

Steps:
1. Download Zeremus Windows: Breathe and extract it into your main EQ directory.
2. Make a backup of your current character's UI INI file (ex. UI_Chet_ayonae.ini -> UI_Chet_ayonae.bck)
3. Rename UI_zw1280_servername.ini (or UI_zw800_servername.ini) to whatever your character's UI INI file was.
4. Log in and play.

NOTE: Zeremus Windows: Breathe is designed for usage with 1280x1024 desktops.
