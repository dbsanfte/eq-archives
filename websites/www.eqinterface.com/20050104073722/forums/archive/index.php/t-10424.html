It should be fairly easy to load. The UI is modular (meaning you can arrange things the way you see fit) or you can use the provided .ini file to have things arranged for you the way you see them in the preview shot. As stated in the instructions, if you use the .ini file you must unzip the contents to a directory called 'blue' (IE: ..\uifiles\blue) unless you open the ini and change that.
