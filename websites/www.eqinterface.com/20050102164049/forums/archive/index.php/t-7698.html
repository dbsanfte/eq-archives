if you look in your EQ directory there should be a file towards the bottom called UI_charname_servernumber, what you want to do either rename it (like what I do is put a exclamatio mark at the end of the file name) and then copy and paste or cut and paste the UI_Statix_31.ini file into the main EQ directory, and rename the file to your chars file, so the renaming looks identical to the file you just moved. So change UI_Statix_31.in to UI_Charname_Servernumber.ini, to whatever your charsname and servernumber maybe.
