Here is a UI that I just finished working on that I call RTUI. It was designed for 1024x768 screens but should work fine on any other resolutions.

Some of you may notice some of the Vert and Canopy windows/gauges design. I love them myself. The spellbook is from T. King and Remelio. The Help window has locations of all zones and the Actions window has the EQToolbox maps.

I have changed the Pet window into the inventory/equipment window you will see in the top right of the screen. On the top left corner you will find all your resists,stats,HP,Mana,Exp,AAExp, ATK, and AC values.

At the bottom left is the Hotkey window which has a built in group window with your own HP Gauge ( makes it easy to select yourself if you dont use F1). There is also the Primary/Secondary/Chest/Inv Slot 1 & 2 built in for quick swapping items.

The old Group window still contains the Invite/Disband buttons but more importantly it is now the Cast time window. (I nestle it close to the new combo hot button/group window.

The Spellgem bar is place close to the combo window for ease of casting spells without taking up your precious hotkeys. (More room for macros and such without swapping pages). It also has a mana line gauge there so you can see how much mana you have without looking elsewhere.

I have shrunken the Inventory screen since everything you normally need from there has been moved to a new window.

Download (http://randytam.home.att.net/Files/RTUI.exe) my RTUI Everquest Skin.

The file is a self extracting zip file that will install all files into the RTUI folder in the default Everquest directory.(C:/Program Files/Sony/Everquest Trilogy/uifiles/ ).

Once done just log into EQ and type /load rtui to load up the UI and then rearrange the windows to the positions in the screenshot. I force the windows into locations based on a screen size of 1024x768 because of compatibilty issues.

Download (http://randytam.home.att.net/Files/SpellIconPack.exe) the T.King spell/buff icons.

Also in a self-extracting zip file that will install to the above folder within my rtui skin folder. Just run it and play. It also comes with an unistall file in case you don't like the new graphics. Not always fun to re-learn the new spell icons. It might also be better to install the actual mod before this as I have not tested doing it backwards.
