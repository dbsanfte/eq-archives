If I could find a way to bring up and select a spell set with the keyboard, I would be in heaven.The /mem command (short for /memspellset) allows you to load spell sets using the keyboard. The syntax is:

/mem < INDEX | PARTIAL NAME >

You can use the index or partial name of a spell set. For example:
/mem 2 (loads second spell set)
/mem buffs (loads spell set named "buffs")
/mem buf (also loads "buffs")

If a partial name matches more than one spell set it loads the first match. With this command, you can create a hotkey bank full of buttons where each hotkey mems a different spellset, or combine this command with the /pause and /cast commands in a hotkey to mem a particular spell set and then cast a particular spell, for example.

Of course, you'll still need to right-click the spellbook button in order to save/delete spell sets. But if you have them set up already, you can use this command to load them without using the mouse.
