Take a look at this:

http://www.thunderwalkers.net/logan/EQ000102.jpg

Basically, I made a web form that you can paste any text into and it will spit out a EQUI compatible XML file. All you have to do is type /note to bring up the window. Any kind of text can go into the window. The above example is our current point list as of last night.

Here's the link:
http://www.thunderwalkers.net/cgi-bin/tw-yabb/AddText.pl

Just save the xml directory into a subfolder in your UIFiles folder. Then type /loadskin x 1. x is the name of that subfolder. Once the UI is loaded, just type /note to view the file.

Next step is multiple tabs.... prolly done in a day or two.

Let me know what you think!
