Well, the problem is that it looks like that group window isn't meant to be a stand alone. It was a fix for the Quartz UI.

You can try editing the XML, but I have no idea how it's going to look, I've never played with Quartz.

The line you would look for is near the bottom. It looks like this

<DrawTemplate>WDT_Group</DrawTemplate>

change it to

<DrawTemplate>WDT_RoundedNoTitle</DrawTemplate>

And then see if it loads.

Again, I didn't look at everything, it's possible there will be something else in there that relies on other files found in the Quartz UI, but that will fix that particular error by changing it to a standard border.

Hope that helps.
