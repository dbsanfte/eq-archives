Features
*New square guages. Very easy to read.
*Target window indicates /consider color and integrates casting time meter.
*Group window includes health for all party members, including self. Large easy to read guages.
*Hot Buttons Winodw includes Primary, Secondary, Ranged, Ammo & 8 inventory (bag) slots as clickies.
*Player Window displays Endurance, Mana, Exp and AA Exp. Also shows Stats, Combat Stats, & Resists.
*Way Cool attack indicator - Sword flashes red when you're attacking.
*Vertical Combat Abilities Window (not shown) with 8 buttons.

Known Issues
* Weight values look a little out of alignment.
* Player Window is huge.

Removed from this version:
*Spell Casting Window (you can not just drop-in any of my previous versions).
*Merchant Windows - Many available out there. I didn't want to reinvent the wheel.

This release is complete and as of March 26, 2004 generates no errors.

Credits
Veirna - Original art/theme, Target Window
Cairenn - Original Buff Window code

This is a release because I'm sick of futzing with it. The UI is complete and code error free.

It doesn't accomplish everything I'd set out to do, but is fine in it's current form. I hope you enjoy it. :)
