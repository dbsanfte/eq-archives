Had same problems... including the suggestion in the above linked post not help me at all.

I'll spare you the gory details of how I figured this out, but this is what I did to solve the problem. First, make sure you're completely logged out of EQ. Find the UI_charname_servername.ini file for the character in question. Open that file in notepad. Hit the F3 key to open up the Find dialogue. Find the phrase "popdialog". (without the quotes). When notepad takes you to the matching text in the file, look at the lines immediately below that. Within a few lines you should see a pair of lines that look like this:

Width1024x768=274
Height1024x768=200

The values at the end of each line should be different. In fact, I'm willing to bet that in your case the values are less than what they are in the example above. What you need to do is to change the values of these lines in your file so they are at least 274 and 200 respectively as in the above example. Then, search the file again for "popdialog". You may have a [PopDialog0] section, a [PopDialog1] section, etc. Make sure all the sections in your file have been edited as per this paragraph.


Go back in the EverQuest and see if you have the same problem. (BTW, easiest way to test this is to try and drop something).

If you want the nitty gritty details as to what's going on with this, reply here and let me know.

If you have any more difficulty, let me know as well.
