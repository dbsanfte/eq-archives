Hello,
Here is how I understand it works. EQ looks for the files that activate the target ring only once when you first log into the game. So, if you copy those targetindicator files to your UI that you go into when you first go into the game they will work. If you copy them after you have logged in and reload your UI they will not work. If you only have them in your default directory they will only load if that is the first skin you log in with. It sounds as if you may have logged in with the right skin, then changed it or something. So try copying those files to the UI directory you use and it will prolly work. Make sure you have two things in your eqclient.ini:

1. TNL=TRUE
2. Show3dTargetIndicator=0

With these both in your .ini make sure you have both the:
a. targetindicator.ini
b. targetindicator.tga
in whatever custom uif folder that you're using.

Once you have the game up and running type /indicator off.

Exit the game completely.

Restart EQ. Once you are in the game type /indicator on.

These steps worked to get my rings working.
