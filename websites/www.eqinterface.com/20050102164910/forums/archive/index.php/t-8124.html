Hmmm.

My own XML knowledge is very basic, so I might be overruled by someone with more skillz... I think what you're suggesting is possible, but I'mn not sure if it's worth the effort. Here's what comes to mind:

A) The only programmable buttons I'm aware of are on the Hotkeys Window. Buttons in the UI aren't create-able or transferable between windows, so you'll probably have to use the Hotkeys.

B) This should work, as long as you make the size of the button and the size of the image the same; I don't know which .tga file you'd need to replace though... and I'm sure it's used in other windows, so you'd have to specify the new image file in your XML.

Of course, by using the Hotkeys Window, you make it usuable for other things, so... someone with real XML skill may have a better way, though.


EDIT: Don't let me discourage you! I've seen plenty of UI additions that I never dreamed possible! Take my info as a place to start from, and see if you can't make it happen! :D
