Patch for the 12/18/03 UI changes. The only file you need to update is EQUI_PlayerWindow.xml.

Please note: even if you don't need either of the new (hybrid and pure melee) versions of my player window, you need to download this zip anyway, or you will get error messages when you use my UI. Some new code was added to the window for the attack indicator.

I decided that the best way of reincorporating the endurance bar into my player window was to make three versions: pure caster, pure melee, and hybrid. That way, people don't have to look at gauges they don't use.
