Just take your modded window_pieces01.tga and rename it, then add a new texture definition to the EQUI_animations.xml, as well as new definitions for the new gauges, having them point to the new file, but using all the same dimensions as the old gauge types.

This should work as long as you don't change the SIZE of the gauges.
