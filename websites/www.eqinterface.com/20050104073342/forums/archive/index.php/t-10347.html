That solution above is not very clear. This is what you do. Make sure you are logged out of the game!


Click Start.
Click Search/Find
Make sure set to search all local hard drives
Search for files *.txc
When done searching, select all, that is highlight all the files it found
Delete all these files


Additionally I suggest the following before doing the above search. Make sure nothing in your EQ folder is set to Read Only.


Right click Start button
Click Explore
Look for your Everquest Folder
Right click Everquest Folder
Click Properties
Clear any checks you see in the Attributes section.
Click OK.
Another window may pop up with options - have set to Apply to this Folder and all subfolders yadda yadda yadda


Then do the Search and Delete above.
