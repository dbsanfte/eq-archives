The problem (in part) is with your Inventory Window. Using a text editor (Word, WordPerfect, NotePad, etc.) to open the file (EQUI_Inventory.xml) in your custom folder.

Using the Search/Find feature, look for:

<StaticAnimation item= "ClassAnim">
<ScreenID>ClassAnim</ScreenID>
<Location>
<X>338</X>
<Y>22</Y>
</Location>
<Size>
<CX>64</CX>
<CY>128</CY>
</Size>
<Animation>A_ClassAnim01</Animation>
</StaticAnimation>

(sizes and location may be different)

and also look for:

<Pieces>ClassAnim</Pieces>

If either of those aren't in your file, add them. If the ClassAnimation isn't supposed to show, change the size in the first part to 1,1.

Also, *where* did you put the files from 28 Jan 03 - UI Changes - UPDATED ONCE MORE 12:36am EST (http://www.eqgui.com/showthread.php?s=&threadid=6012) ?
