Thank you all for the kind words =D I'm glad you like what I've done so far...

Lisimba - I'm guessing by 'inventory window' you mean the window with all the inventory slots and the stats. This is actually the Target window, and can be opened and closed with ALT+T.
There are 2 versions of target windows, one has small inventory slots on tabbed pages (this is in the original Translun download) The other target window is bigger and doesn't have tabs (that's the one in the Transwrap download) The two target windows are interchangeable, and can be switched if you like. If neither of these suit you, post a slightly more in depth description of what you want the target window to look like (tabs or no, keep big fonts on hp/ac/atk or scale them down, more of a horizontal or vertical shape, ect) and I'll see if I can't put something together for you.

Perhaps if you put the inventory into a different window, instead of one that closes when you zone.

Essentially, that's what I did. I took all the parts out of the inventory that I could, and placed them in the target window so they wouldn't close everytime you zone. These parts included all the inventory and bag slots, as well as all the character stats. Unfortunately, one of the many things about the EQ User Interface that you can't change ( things you can't change detailed HERE (http://www.eqgui.com/showthread.php?s=&threadid=2267) ) is : You cannot move buttons/coins/autoequip into any window except the ones they usually appear in. That means that you HAVE to keep the coin slots, all the buttons (destroy, face, tint, skills, aa, and done) and the autoequip box in the Inventory window, and that widow in HardCoded to close each time you zone.
