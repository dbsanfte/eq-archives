Enoks UI - Features:

- Compass w/ Loc Indicators (green = increasing, red = decreasing)
- Borderless Note Window w/ Scroll Background
- Informative Player Window w/ Tick Timer
- Divided Up Group Window (click on Pet: ###% to target pet)
- Buff Window w/ Color Coding and Spell Names
- Container Window w/ Border but Moveable Only By Container Name
- Borderless Animated Spellbook w/ Page Turn Animations
- Compressed Spell Gem, Hot Key, and Selector Windows
- Simplified Pet Window (use bottom edge to resize)
- Old Style Inventory w/ Retro Icons (old unused EQ graphics)
- Borderless Quantity Window w/ Retro Graphics
- "Sneeze Guard" Over Chat Windows, Prevents Accidental Clicks
- Minor Tweaks To Guage Colors (and other things) In Other Windows
