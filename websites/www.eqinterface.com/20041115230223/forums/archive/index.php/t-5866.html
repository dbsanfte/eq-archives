Not all windows have been altered, what has been done is shown in the screenshot. With the exception of the Social Edit window, which I forgot to bring up before taking a screenshot.

-

T.King "chipped edge" spell gems
T.King buff icons
T.King spell icons
T.King / Remelio Arcane spellbook 1.0

Remelio's loc compass included.

Inventory has been resized and reorganized. If you remember the old "full screen" UI you'll recognize this. Original author unknown, has since been modified by yours truly.

Player Window includes hitpoints (max / current), AC as well as gauges for Hitpoints, Mana, AA Exp, and Stamina.

Hotkey bank includes complete mini-inventory and slots for three clickies. Set currently are Primary, Head and Chest. Because that's what I click.

Target window includes more of the target name and an enlarged percentage readout. Included directly below the target HP gauge and visible in the screenshot is the Casting Time gauge.

Group window slimmed down and percentages for groupmember health added.

AA Window buttons moved from the side to the bottom and wasted space eliminated.

Chat window titlebars removed, resize the window to move it.

Buff / Song box titlebars removed, resize the windows to move them.

Skills, Actions, Load Skin, Social Edit, Options, Breath, Quantity and Raid windows have all been slightly modified as well.
