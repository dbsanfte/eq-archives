Ok, I was able to get my problem fixed. Here's what I did. :)

First, I completely removed any trace of the old zwnitrain files/folders. I also renamed my UI_Kyrnik_Brell.ini file to UI_Kyrnik_Brell.old.

Once that was done I deleted eqgame.exe and also the uifiles folder from the Everquest directory. I then ran the EQ patcher to pull down clean files from the server. (Had to run the patcher twice to get the files to download)

I then went into C:\program files\everquest\uifiles\default and copied all the default UI files. I created a new folder in c:\program files\everquest\uifiles named zwnitrain. I pasted all the copied files to that folder. I then downloaded a fresh version of zwnitrain and extracted the files to c:\program files\everquest. When prompted to overwrite files I chose 'Yes to all'. Once that was completed I went into c:\program files\everquest and renamed UI_zw1280_servername.ini to UI_Kyrnik_Brell.ini.

I then loaded up EQ and the Zeremus UI came up as my default. The only problem I had was that the font was set to Georgia. Under the options tab you can change the font back to Arial.

Well, that fixed it for me. Not sure if it'll help anyone else but I thoughtI I'd at least share my findings.

Good luck,
