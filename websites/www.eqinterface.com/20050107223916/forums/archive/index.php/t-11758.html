Have you unplugged it from your internet connection? Especially if you have high-speed (cable) that is always on... UNPLUG it. Besides the annoying things that the virus is doing to your computer, it may also be communicating with the internet, accepting commands from a hacker, or even harming other people's computers and networks.

Try booting in safe mode. If you have windows 2000 or xp, hit F8 during the startup process, and select boot in safe mode (not with networking). See if you can get to the task manager and add-remove programs that way. If you can you can start looking for the virus files or folders. You may even be able to start anti-virus manually and run a scan. If you still can't access any system tools in safe mode, you probably need to start thinking about reformatting. If you back up files, remember to scan the backup before you get it anywhere near your new install.

12-21-2003, 08:35 PM
If your going to re-install windows2000 or windowsXP just boot off the install CD and when it asks you what drive and partition to install it on choose to remove the partitions and then re-create one and then choose to format. Then continue with installing.

If your going to re-install win98 boot off your win98 floppy and at the command prompt type fdisk /mbr (http://www.google.com/search?hl=en&lr=&ie=UTF-8&oe=UTF-8&q=fdisk+%2Fmbr) (wait a few sec) it will finish with not much of a prompt. Then format the drive and install win98.

Now if your really paranoid you could try doing a low level format (http://www.google.com/search?hl=en&ie=UTF-8&oe=UTF-8&q=low+level+format).

Then the first thing to install after you’re up and running would be NAV or some up to date virus scanning software.


Just fyi with all of the above options all data currently on your harddrive will be lost. I just want to make sure you understand that.
