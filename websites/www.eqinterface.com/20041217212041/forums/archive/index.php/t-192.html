Well, I refuse to get used to the new icons, and I'm making good progress on my own icon conversion. It's not a literal translation back to the original icons, although I'm using them as much as I can. When a spell uses an icon that doesn't resemble any of the original icons, I just simplify it a bit, to make it more quickly recognizable.

For example, insidious retrogression uses a graphic of a shoe with an hourglass over it. This icon is nearly identical to another icon with a slightly larger shoe, (/boggle) While Shackle of Spirit (Undead only, slows attack speed) uses an icon of an armored arm holding a sword, apparently swinging the sword, with an hourglass over it. I honestly couldn't tell what this icon was supposed to be until I looked at it for a good long time. This icon is inexcusable VI!

In my conversion, both icons are replaced with a simple hourglass, imposed over the old boot graphic and the old arm graphic, respectively. Other spells, like resist fire, use the old shield graphic with the old fire graphic resized and placed in the corner, giving it the same style as the new icon but more easily recognizable for an old fogey like me. Later I'm going to go over the icons again, giving the icons that I know are strictly for beneficial spells a blue background, and strictly bad ones a red background.

Kinda sucks that pact spells, counteract disease, etc don't use the "heal" icon anymore :\
