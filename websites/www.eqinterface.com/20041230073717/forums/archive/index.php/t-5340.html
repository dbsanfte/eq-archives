Some extra Tips From Alan (Absor) at EverQuestLive.com (http://eqlive.station.sony.com/community/dev_view.jsp?id=53589) on the EQWindows...

The default key command ALT-SHIFT-R is the command to release your mouse to the desktop when you go into window mode... to return simply click inside the EQ Window.

Also this command can be remapped in the option menu, (I reset mine to CNTRL-ALT-R, just familar to me.... hmmmmm wonder why)
