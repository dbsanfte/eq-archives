Current Version: Solaria Version 1.1
(Production IE. NON-Beta)

EDIT: Changed some file names for TGA's that caused lag on soem video cards. If you experienced no lag you do not need to upgrade to 1.1

==================================================
Color Packs:
==================================================
Untill we add a program to auto install color schemes, you will manualy
have to goto the color packs directory, and choose which one of the 9 color schemes
you want to use.

DEFAULT = Mercury
To use, just copy the TGA files into your Solaria directory
over writing existing files.

==================================================
Options Folder:
==================================================
Located in the options folder are the following:

- Melee Player window (Removes the mana gauge)
To use, just copy the XML files into your Solaria directory
over writing existing files.

- Caster Player window (Adds the mana gauge) <--DEFAULT
To use, just copy the XML files into your Solaria directory
over writing existing files.

- Cast Spell Window without the names (Removes spell names
from the default casting window)
To use, just copy the XML files into your Solaria directory
over writing existing files.

- Cast Spell Window with the names (Adds spell names
to the default casting window) <--DEFAULT
To use, just copy the XML files into your Solaria directory
over writing existing files.

- Organized BigBank window (Adds catagory labels above bank slots
divided into various sections)
To use, just copy the XML files into your Solaria directory
over writing existing files.
NOTE: To remove this mod after installing, simply delete the
EQUI_BigBankWnd.xml file, and Solaria will load the DEFAULT Bank window.

==================================================
Ccake Manual:
==================================================

1) Copy the EQUI_HotButtonWnd.xml from the Solaria directory to the Ccake directory
(Do this only if your using a Hotbutton mod compatible with the Ccake program)
(The Default Solaria HotButton is already included in the Ccake folder)

2) Run the ccake.exe choose the changes you want.

3) Click inject.

4) Save it to the solaria directory overwriting the hotbutton window there.
(DO NOT I REPEAT NOT over write the one in the Ccake Dir via this method)

5) Reload Solaria, and enjoy!

10-21-2003, 01:41 PM
It's trivial to change, which I did even though I'm a major pet-user -- everything in the pet window is duplicate info anyway.

Open the EQUI_PetInfoWindow file in the Solaria directory with notepad.

Go to the bottom of the file and scroll up until you see:
<TabBox item="Pet_Subwindows">

At the bottom of that section you'll see:
<Pages>Pet_Page</Pages>
<Pages>Player_Page</Pages>

Swap those two lines to read:
<Pages>Player_Page</Pages>
<Pages>Pet_Page</Pages>

Save the file. You're done.
