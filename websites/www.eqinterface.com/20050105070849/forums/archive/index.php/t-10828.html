4. Am I able to make changes between characters as to how the UI works, or should I have a UI for say my magi and my bardi?
Personally, I use the same UI for almost all my characters. The only special case is a separate UI for pure melee since they lack the spell casting window, which is an integral part of the layout.

You can keep different windows up for your different characters and you can lay out the windows differently for each character. Message filters are per-character as well. If you want the UIs to differ more significantly than that you will need to maintain a separate UI directory for each character.
