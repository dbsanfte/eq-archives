Cragzak
01-14-2003, 05:06 PM
Ok, In your "Everquest" folder this is another folder named "uifiles" and in that folder you have another folder named "vert" which has the "cragzak" folder in it... that's your problem. You can only go as deep as to have a folder in your "uifolder" with a name. So you need to pull out the "cragzak" folder from the "vert" folder and put it next to the "vert" folder so they are both in the "uifolder". Get it? It should read:

C:\Program Files\EverQuest\uifiles\cragzak

or

C:\Program Files\EverQuest\uifiles\vert

depending on where you want to keep all the files.

Then you have the issue of having changed the command in that text file to UISkin=vert. This might end up being a problem too (unless you have the stuff it needs in your vert folder too) if you are going to keep everything in a folder named "cragzak". So if you are keeping it named "cragzak" and you move it into your "uifolder" next to vert... then to make sure EVERYTHING works fine and dandy i would suggest you change that to say UISkin=cragzak. You should be good now... lemme know! :)
