I'm not sure a set time will do it. Some of these skins, (like the one I just fixed recently), worked fine for 6-7 months through various SOE updates.

What I think is very much needed is a new table of SOE Versions (which updates everytime SOE sends a new patch) and a author/admin only feature to let everyone know if the version of their skin works with the latest patch.

Something like:

My Skin Version 3.0
SOE Patch 1/1/03 | SOE Patch 2/1/03 | SOE Patch 3/1/03
Functional | Functional | Broken. Please check My Skin Version 4.0 at this URL...

It's pretty difficult now to know if any of the skins (even the ones updated within a few weeks) to work with the latest patch without downloading and testing.

Then if a month or so goes by without a fix or an acknowledgement by the owner, to send an email to the author asking if someone else can take it over.

As for a vote, I'm also not sure. The past month has been a pretty bad experience for me in EQ without my favorite skin. I'm thinking the first person who has a fix simply submit it making sure that they give credit to the original author. I think ownership should go between the original owner and the the person asking to take it over. It would kinda conflict with other websites hosting skins and having different authors. Can of worms I'd rather not get into. :)

I'm also curious if there is a reason to having older versions of skins available for download knowing they don't work. There are quite a few 1.0's, 2.0's 3.0's and so on. Is there a reason why 1.0 is still available?
