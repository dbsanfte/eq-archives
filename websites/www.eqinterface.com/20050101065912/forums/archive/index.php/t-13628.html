I'm not sure what you mean by "UI for fonts."

It is possible to change your default font in the Options window. This will change the font itself, and not the size, for all the text in the game.

As far as changing the font in the chat window, this can only be done with the /chatfontsize command that you have already used.

Also, if you would like the font changed for the labels in your windows, those must each individually be changed in the window coding. Time consuming, but not too difficult.

I agree that there is too broad a jump between 4 and 5.
