IF you have a custom EQUI_OptionsWindow.xml file, open using a text editor (Word, WordPerfect, NotePad, etc.) and using the Search or Find function:

Find:

<!-- Load a new UI skin -->

Immediately before that line, ADD:

<!-- Allow/prevent the game from making your character duck to squeeze into small spaces -->
<Button item = "OGP_AllowAutoDuckCheckbox">
<ScreenID>OGP_AllowAutoDuckCheckbox</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>10</X>
<Y>275</Y>
</Location>
<Size>
<CX>150</CX>
<CY>20</CY>
</Size>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
<Style_Checkbox>true</Style_Checkbox>
<Text>Allow Auto-duck</Text>
<TooltipReference>Toggle whether you will let the game make you duck to fit into small spaces.</TooltipReference>
<TextColor>
<R>255</R>
<G>255</G>
<B>255</B>
</TextColor>
<ButtonDrawTemplate>
<Normal>A_BtnNormal</Normal>
<Pressed>A_BtnPressed</Pressed>
<Flyby>A_BtnFlyby</Flyby>
<Disabled>A_BtnDisabled</Disabled>
<PressedFlyby>A_BtnPressedFlyby</PressedFlyby>
</ButtonDrawTemplate>
</Button>

Find:

<!-- Load a new UI skin -->
<Button item = "OGP_LoadSkinButton">
<ScreenID>OGP_LoadSkinButton</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>10</X>
<Y>285</Y>
</Location>

CHANGE:

<Location>
<X>10</X>

</Location>

<Y>305</Y>
</Location>
