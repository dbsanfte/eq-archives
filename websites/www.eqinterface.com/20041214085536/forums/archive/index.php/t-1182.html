Ok, if you copied all of your Default folder and renamed it, then:

1. delete all of the files inside the new folder.

2. Now, copy all of the files you want changed in your UI Mod into this new folder.

3. Type /test <whatever_name_you_chose>

The UI now looks into your skin folder and uses those files over those in the default directory, and pulling from the default folder, those not included in your mod folder.

When you run the patcher, if it downloads updated UI files, it saves in the default directory, but if you still have old files saved in your new folder, it won't work properly.

Hope that helps.
