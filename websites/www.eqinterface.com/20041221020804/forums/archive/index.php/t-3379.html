News - 10/25

Thomus King (http://pub10.ezboard.com/fmaximus94788frm10.showMessage?topicID=74.topic) has made the Black Marble background (used in the new Celtic Inventory) available for download; in addition, the artwork for the Celtic Inventory has been "tweaked" a bit.

Garryck has released a new program: INI Copier (http://www.eqinterface.com/sdownload/fileinfo.php?id=471), a "graphical utility to help you copy the contents of one of your character's INI files to another character's". This should help folks wanting to use the same UI setup for multiple characters do so, without doing a lot of manual changes.

Earthscum has created the PacMan Mana Muncher (http://www.eqinterface.com/sdownload/fileinfo.php?id=474)... "Yes, now you can watch Pac-Man chomping your mana up!"
The News Ranger will refrain from comment... as he doesn't want his mana eaten...

Xymarra has released UI v2.1 (http://www.eqgui.com/showthread.php?s=&threadid=3343), a viewport mod with very nice original artwork.
You can download the UI via attachments in the thread linked above.

Sokol has uploaded the final Vert 2.2 Animations fix (http://www.eqinterface.com/sdownload/fileinfo.php?id=468); this should repair all issues related to the 10/21 patch.
