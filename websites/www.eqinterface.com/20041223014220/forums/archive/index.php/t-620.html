I started the game again and typed /loadskin modern and it came up all messed up again.

That's cause your forgot the use 1 again.

/load modern 1

1 Means Leave my Windows alone. Actually it means don't over write my UI_<char>_<num>.INI settings.

The settings that the 'non'1 /load uses is "hard coded" into the UI files. You don't want that. Always use 1.

If you have to, make a hotkey for it.
