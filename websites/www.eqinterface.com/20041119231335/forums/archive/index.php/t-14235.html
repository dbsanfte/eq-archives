I am currently using Adobe Photoshop CS as my graphics editor.

Does anyone know how to save images as .TGA and retain the transparant layer?
forry
07-04-2004, 10:37 PM
When using a .tga file, I don't think you use transparency. Select everything in your .tga file that you want to be able to see in game, then save that selection as the "Alpha" channel. Anything that isn't included in the "Alpha" channel will be transparent in EQ. Hope this helps.
