I've found that's the most hassle-free way of doing it. Unless it's a patch specifically for a certain version, I back up the previous version install (usually with WinZIP or WinRAR), then start fresh using the same folder name as before and copying in all the proper files as needed, like altered components from the mods folder, or other spell icons or target rings.

That way none of the old problem files can influence the new version. Same kind of thing if you ever have to reinstall Windows due to major crashes, just start fresh so that there are no lingering problems waiting to jump out at you.
