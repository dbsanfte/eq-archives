Okay, I'll dispell this right now, and no more will needed to be talked about. :)

I hear rumor of a program that can identify what a MoB is carrying in game BEFORE you kill it... anyone know?

SEQ can tell you one thing about the mob's "loot".
That is - What graphic the NPC has in hand.
This is exactly what you see if you where standing 5 feet from the beast, or if it was whaling it upon your soft skin (or scaley) head.

It can not tell you loot. It can not tell you what the X sword/weapon is. Loot is all server side handled and not even decided until you kill/loot the beast. This is how VI was able to implement the 'Trival Loot Code' they had a while back (and still active on the FV server).

SEQ is illegal to use and VI will terminate your account w/ou warning if you're cought using it.

I hope I cleared up a few things for you. ;)
