ok since I don't seem to have the Note_locsV1.1 in the downloads section I am going to Place the file here.

If you choose not to use the EQUI.xml file included, please make sure you have the following line added to the EQUI file so the player_notes window will come up.

<Include>EQUI_PlayerNotesWindow.xml</Include>

This should be following the rest of the <include> statements.

the bottom of the file should look something like this.

<Include>EQUI_VideoModesWnd.xml</Include>
<Include>EQUI_TextEntryWnd.xml</Include>
<Include>EQUI_CompassWnd.xml</Include>
<Include>EQUI_PlayerNotesWindow.xml</Include>
</Composite>
<Schema xmlns = "EverQuestData" xmlns:dt = "EverQuestDataTypes"/>
</XML>

Notice the addition of the PlayerNotes line.

See ss above on how the note_locsV1.1 should look although the text is now white on a black background.

Have fun.
