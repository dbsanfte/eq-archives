Welcome to my, uh, updated TrilZW3 UI. I removed the cursor for good, promise! All mods were NOT made by me, except the slight change to the inventory (switched the done and destroy buttons around and make the destroy button larger for all your tradeskillers with mass combines. if it's annoying enough, I will post the original version). If I don't mention names, chances are, it's because I forgot, but ALL of the mods can either be found in Zeremus' pack, or at http://www.eqinterface.com.

I've also cleaned up the zip file, removing all unnessecary files, but it's still a very hefty <3meg download, I'm afraid =(

Note that this UI is for 1280x1024 ONLY, and is very caster friendly!

-=Features=-

Based on Zeremus Windows v3.0 [found here (pub142.ezboard.com/fayonaerofrm1.showMessage?topicID=3064.topic)]
Streamlined NPC-Give window (Not shown in the screenshots)
Streamlined container windows (the little space on the bottom of each container is where the 'combine' button would go. the borders have been removed and the only place to move the container is the title bar -- perfect for you smiths doing mass combines!)
t.king's beautiful arcane v2.0 spellbook, and v4.0 spell gems
Athee's 'expanded' friends list, featuring big AAXP and XP gauges for those who want to see EXACTLY what they're getting in that Plane =P
Kathara's Tradeskill Mod, combined with the actions window (ctrl+o in game)
Athee's Loc Wizard (combined with the feedback window)
Much more visible, numbered buff window
Also features a fully modifiable UI, allowing nearly all parts (except the target window) to be switched, ripped, and changed at your leisure
Streamlined item-info window, with a smaller font to see all the effects of that new loot you just picked up has =P
Other goodies I probably forgot to mention =P Just explore

Arista's timers are NOT included, and are there for show because I use them =P They can, however, be found here (www.horizonsoutpost.com/eq/timer.asp)

This UI was meant for nearly everything to be turned off, so you only get the information you need, when you need it, and to save you those precious frames-per-second during that NToV raid. Remember, chat windows are also 'click-through' and can be use as a flota--er--and are made transparent so you can see what's going on, on all sides of you.

I hope you enjoy the UI, I've gotten make compliments from it so far, and I'll continue to update it as people want it! Enjoy =)
