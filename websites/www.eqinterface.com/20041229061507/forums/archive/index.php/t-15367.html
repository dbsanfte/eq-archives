<img src="http://www.eqinterface.com/images/sections/directx.gif" align="right">Nvidia has released a new texture based compression tool with the extention .dds. The Microsoft DirectDraw Surface (.dds) is part of DirectX 9. You can read more about the format it self at msdn (http://msdn.microsoft.com/library/default.asp?url=/library/en-us/directx9_c/directx/graphics/reference/ddsfilereference/ddsfileformat.asp).

You can find the photoshop and other plugin info at Nvidia's site HERE (http://developer.nvidia.com/object/nv_texture_tools.html)

Sokol and I have done some experiementing and found that you can use DDS files in your UIs.

Upside
Files are 10% of the file size of a TGA. (TGAs that are 1meg become 100k)

Possible Downsides
DDS UIs are not cross compatible with TGA UIs. Trying to merge them in experiements caused crashes

If you want to see DDS UI at work, you can download the Solaria test HERE (http://www.eqinterface.com/downloads/fileinfo.php?id=3740). Its a stand alone version of our UI to test the DDS files.

We have added a category in the forums for DDS discussion (http://www.eqinterface.com/forums/forumdisplay.php?f=60).
