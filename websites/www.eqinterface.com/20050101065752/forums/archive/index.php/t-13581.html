Been 323 days since i had to reformat my hard drive. Learned some tricks an havent had to reformat since.

Read about virtual memeory an partitioning the hard drive into sections.So i partitioned mine into 3 sections, 1st one was 5 gig just for windows an programs that work closely with windows (like ontrack system suite 3 or nortan stuff).2nd one was smaller 2 gig section just for virtual memory.3rd section was for rest programs eq included.

Best part about this set up is if/when windows acts up an cant be fixed i just nuke c drive , reinstall windows an drivers an my registry.The rest stay right on the hard drive .

This set up also helps prevent hard drive thrash ( which wears out hard drives fast), but also cuts back fragmentation alot.

Recomend partition magic if any one is gona try this , an system suite 3 (not 5 the new one ) for win 98 se .Both programs have helped me alot. Hope this helps some because reformating is a pain.
