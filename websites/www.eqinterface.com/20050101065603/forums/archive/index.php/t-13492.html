04-14-2004, 07:26 PM
You may have changed the key binding to open that window. In general, if you want to find what key opens and closes a window, you can find it under the Keyboard settings in the Options window.

However, I don't think that is your problem here.

- You may have an error in your custom mp3 UI window. You can try deleting or renaming it and see if the default one works. If the default works, come back here for more help to get your custom one working again.

- The window might have gotten positioned off screen. You can force all windows to default positions with "/load <ui> 0" or you can edit your UI_character_server.ini file and move just that window to some visible location, like (0,0).

- You might have turned off sound completely from the Options window on login.
