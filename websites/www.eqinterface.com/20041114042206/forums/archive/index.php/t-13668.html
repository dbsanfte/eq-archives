- Sars 4 Spellbook v1.0 (http://www.eqinterface.com/sdownload/fileinfo.php?id=3266)

Ripped from the Sars 4 UI by Request. This is a Stand-Alone, Compact Spellbook. The upper left and right page corners animate to a fold-down for when you are about to change pages. This will not modify your books for recipes / stories.

Just copy the files to a custom UI directory and use /loadskin to load it. No messy modification of the EQUI_Animations.xml file. That easy.

This should work with most UIs. If it does not, please PM me with the name of the UI you are using and / or the error messages from your UIErrors.txt file.

Thank you Sars for making such a great Spellbook !!!
