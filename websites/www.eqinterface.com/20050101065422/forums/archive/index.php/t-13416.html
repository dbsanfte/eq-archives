1) You can't delete SONY storyline files. (well you can, but the patcher puts them back anyway)

2) There is a limit to the number of stories you can have. Also, each one you use adds more resources to the game, so consider that when using stories, especially if you're on a PC that's not top-of-the-line. I have sent this issue in as a /bug but Sony doesn't seem to care at this point.

3) If the files you are using are not named story____.txt they will not show up in EQ. They also have to be in the \Everquest\storyline directory.

4) The Story window displays the newest created stories at the top. It's sorted by creation date (descending). This is an annoying "feature" and has made me resort to making the first line of all my user-created stories have an _ character in them so they will be easier to spot in the list.

... I hope some of this helps.
