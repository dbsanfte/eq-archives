There are currently two WYSIWYG EQ Interface Editors.

Sidl Fiddle
This editor is made by an EQInterface member that goes by the name Ambros (http://www.eqinterface.com/forums/member.php?userid=11471). The development of this program is still active. This program is written in Java and requires that you have 1.3.1 or greater installed.

Offical Webpage & Download:
http://www.barbre44.com/
http://www.barbre44.com/downloads.htm

-----------------------------------------------------------------

Sidl Widl
This editor is made by an EQInterface member that goes by the name Gendal (http://www.eqinterface.com/forums/member.php?s=&action=getinfo&userid=637). The development of this program is dead at the moment but the program still works well. This program requires you install the Microsoft .NET Framework.

Webpage & Download:
