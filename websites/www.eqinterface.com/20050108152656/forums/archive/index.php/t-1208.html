Muse
08-19-2002, 01:07 AM
Try this then. Am I right in guessing that this is a mod that you got or set up prior to the last patch, or maybe hasn't been corrected? Sounds like with that error message, it's looking in the EQUI_Animations.xml file for those items and not finding them. If you have an old EQUI_Animations.xml file in your mod folder, that will happen. Just delete that as well.

As a rule, only include the files that you absolutely need (i.e. were changed for the mod) in the mod folder. If the game wants it and can't find it, it will look for them in the default folder and nothing bad happens. If you try and keep them all in the mod folder, the game could patch them and then cause crashes because it's reading the old one out of the mod folder and not the new one that it needs. You should never need to copy files from default to the mod folder to get it to work correctly.
