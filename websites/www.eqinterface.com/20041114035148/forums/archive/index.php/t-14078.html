Ok.

Download the Zip file from downloads section of Eqinterface.
Link:

http://www.eqinterface.com/sdownloa....php?s=&id=3392


save zip file to Desktop
Extrace All Files to Desktop call the Folder Vert.
You should be able to Double click the Folder and see all the files in side it. There shouldne be a folder in a folder.
In that folder is a UI_YOURNAME_YOURSERVER.INI file
Rename that folder replacing the Your name with...well your charactors name and Your Server with well you guessed it...your server.
Once thats done Right click that file and Copy it Directly Into your Everquest directory..
Now go back to your Desktop and Right Click the Vert Folder and Copy that into your Everquest/Uifiles Directory.

If you did that right you should see in the UI files Directory a folder called Default and the Folder you created with the XML files INside that.

Start EQ and you should Be GTG.
