When in doubt, the best thing is probably to pick up the latest version and start fresh.

I usually rename the old UI install (like "TabletsBlue" renamed to "TabletsBlue2") and then reinstall fresh to the original folder name (creating a new folder with the name you normally use, like "TabletsBlue" in my previous example). That way if the new install of the UI works even less well, you can always ditch it and rename the old folder back to the original name and get back what might have at least been working somewhat before.

But going with a completely fresh copy of the UI can sometimes rule out inconsistancies with older files or other files that conflict.

So far my Tablets UIs are working (knock on wood), and I'm just waiting for Talyns to patch the updates in. :)
