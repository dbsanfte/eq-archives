Avalon 3.4 beta readme
EQUI.XML modding

if you wish to add your own custom windows into Avalon, normally any modding isn't supported but the process to allow a new window is fairly simple.

if the window does not allready exist in Avalon open EQUI.XML with notepad and find the line that matches your file name* <Include>../default/yourwindow.xml</Include>
then simply remove the ../default/ from infront of that line, you may now add your own window. :-)
