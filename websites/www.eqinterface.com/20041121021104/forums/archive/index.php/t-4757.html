I don't want to clutter up the board with pictures and such, but I can describe it (if you not familiar with using Notepad, stop reading, you not gonna get far)

One good step to learning what to copy and paste is using something like SidlWidl that picks out the Tags, every buttong, number, guage etc has a tag. XML is Extremely similar to HTML, just different tags.

As for seperation from XML files, not sure what you mean, Copying and pasting is one cheap way to get all the fields of the desired itemcopyied to a new window (Like Inventory slots on the Hotkeys window) Although as mentioned (and I forgot hehe) You cannot copy things like buttons and the Autoequip area.

The name and screenID are both inside the XML file itself and defines it as a window (the name tag is actually the indicator, such as Button, InvSlot, or guage)

It would help you immensly to open up an XML file with notepad (right click adn chose edit) from your default directory, or download one you think is interesting. There is just a lot to cover regarding the fields used and I hope there is a sticky post somewhere that points em out(I havent looked over all the boards yet)
