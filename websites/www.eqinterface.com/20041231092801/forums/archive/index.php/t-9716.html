Thanks to an invite from Hammerfel, I was able to take a tour of SOE Studios today.

I have posted a more detailed transcript in the Chit-Chatsections. But I wanted to post a small blurb about it and what I learned here are the basic highlights here.

Everquest
Next patch Aug 27th will add two new TGA's. Window_Pieces06.TGA & Window_Pieces07.TGA If you use these Names for custom TGAs in your UI, you will want to change the name of them Before Aug 27th. A long list of XML changes are coming in the 27th patch, they are merging the two journal windows, adding the Shared Plat Slot, and several other tweaks (more info when Absor post in Dev Section)

Everquest II
No official release date mentioned, game is on schedule and the team promises lots of info coming soon, and Yes it will have a customizable UI just as EQ 1 does.

More in the Chit Chat Section can be read here (http://www.eqgui.com/showthread.php?s=&threadid=9720)

Originally posted by cryingwolf
Hate to sound stupid. lol but whats a TGA?
You don't. It is an image file saved in TARGA format. It is less popular than Windows Bitmap (.BMP) format but offers nearly the same file sizes and picture quality. You can edit these files using Adobe Photoshop, JASC Paint Shop Pro, convert and view them using IrfanView, ACDSee and many other software including AVIedit.
