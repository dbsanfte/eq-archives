OldSchool v2.6
-----------------------
This skin is designed to sorta resemble the Pre-Fullscreen Interface.
It uses a 1024x768 resolution with a 768x576 viewport.
It should be good for any class to use!
Now in 5 colors!!
Blue, Green, Grey, Purple, and Red

**Remember to follow the readme when installing.**

Current to:10-09-03 patch
------------------
Notable Features
------------------
Uncluttered viewport.
Most info readily available without opening extra windows.

Compass loc indicators. Tells you if loc numbers will increase/decrease
Red is Decrease, Green is Increase

Two tabs under the compass.
Tab I. 8 Inventory Slots, Pri, Sec, Range, Ammo
Tab II. All other Inventory slots (Clickies)

Vert style inventory slot icons.

Sure you can move the spell gems.. It's actually quite easy:
1. Right click the gem window and a popup menu will appear
2. Click Lock
3. Move Gems
4. Repeat 1 and 2 to relock it

For a spell gem window with names there are quite a few available for download that you can copy to your oldschool folder and use:

Mafoo's:
http://www.eqinterface.com/sdownload/fileinfo.php?id=114

Chronwaas':
http://www.eqinterface.com/sdownload/fileinfo.php?id=1059
http://www.eqinterface.com/sdownload/fileinfo.php?id=1058

Plus many others here in the spellgems download section:
http://www.eqinterface.com/sdownload/index.php?catid=19

But, If you choose another one from there make sure it doesn't contain a EQUI_Animations.xml or it will mess up the skin.. The 3 I listed don't contain a EQUI_Animations.xml :)

Moving the buffs window around is a little tougher but they can be put anywhere too.. To do that:
1. Type /loadskin default 1
2. Unlock the window by right clicking the titlebar
3. Move and/or shape.
4. Relock the window
5. Reload oldschool skin by typing /loadskin oldschool_purple 1 (for the purple one) or /loadskin oldschool_grey 1 (for the grey one)
