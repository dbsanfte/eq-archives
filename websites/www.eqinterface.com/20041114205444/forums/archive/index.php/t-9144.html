Version 1.02 - updated 7/3/2003

Since SOE did not release the subset of HTML tags which work within the Story files, it has been up to player experimentation to find them. - further edits will be made and incorporated into this document.

Sourcing for this guide includes the review of over 30 published storys, the default HELP and TIP files and info from posts on this board.

In particular Jentraal (http://www.eqgui.com/member.php?s=&action=getinfo&find=lastposter&threadid=7961) has provided alot of insight based on his experimentation.

Important note:
EQ creates an index of the story windows. The way this is done results in EQ placing a file lock on all the main story window files (storyXXXX.txt). So the more story windows you have installed, the more system resources are required.
Sub-files to the story windows are opened as-needed, one at a time and minimally add to the system resources required.
