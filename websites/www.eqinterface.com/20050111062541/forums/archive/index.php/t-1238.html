Originally posted by RagnosTroll
Do I need to download the T. King spell gems in order to use a T. King Spell book?

Nope. If you don't install the spell gems, it will use the default ones.

Also, if I download both the T. King spell book and the T. King spell gems, do they get saved in the same folder?

Yes. Place them in a directory you create in uifiles. If all of the mods you wish to use are in this directory, all of those mods will work when you select that skin. More info on this is documented elsewhere on this site.
