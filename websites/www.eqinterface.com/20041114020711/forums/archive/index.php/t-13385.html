For those that enjoy using custom spell effects, I've made a Rainbow Colored Find indicator:

http://eq.onlinegamersguild.com/find-rainbow.png (http://www.eqinterface.com/sdownload/fileinfo.php?id=3205)

Instructions:
The files will extract to the folder find-rainbow.

After you extract the files, copy them to your:
Drive:/.../EverQuest/SpellEffects folder.
Select yes to overwrite.

*Do not copy the find-rainbow folder into your everquest/spelleffects folder, just the files.

To use these files you must skip the patch process or the files will be overwritten. Change your everquest shortcut icon target to read:
Drive:/.../EverQuest/eqgame.exe patchme

In most cases all you should need to do is change:
EverQuest.exe
to
eqgame.exe patchme

I suggest using two seperate icons:

1. Drive:/.../EverQuest/eqgame.exe patchme
(for skipping patch)

2. Drive:/.../EverQuest/everquest.exe
(for running patch)

Enjoy.

Some info about the path.tga:

If your .tga is in a vertical format, such as lines, loops will form in succession for the length of the indicator.

If your .tga is in a horizontal format, streaks will form for the length of the indicator.

If your .tga is in a diagonal format, it will form a "tube wrap" for the length of the indicator.

Frames can be used in a similar method as the Target Indicator.
