Without seeing that file, I can only guess that the problem is with the EQUI_Animations.xml file. The readm file that came with the beastlord pointer gave clear instruictions on how to edit an existing EQUI_Animations.xml file if your current UI has a custom one. Don't know for sure, but I believe that the Drakah UI you're using does have a custom file. Since you made no mention of having edited that file per the instructions in the readme file, I'll assume that you did not do so.

Try editing the EQUI_Animations.xml file in the Drakah UI folder per the instructions in the beastlord pointers readme file. Please make sure to back up the EQUI_Animations.xml file before doing this. If it still doesn't work, post a copy of the UIErrors.txt file. Posting a screenshot of what your screen looks like might also help.
