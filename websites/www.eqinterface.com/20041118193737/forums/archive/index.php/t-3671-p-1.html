Download the 3.0 version.
Download the 1024x768 patch and overwrite the files.

1) How would I go about changing the pointer. I find that I have troubles seeing yours during heavy combat situations.
You would need to edit the EQUI_Animations.xml file. Find the block of code that looks like this:<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>Deco.tga</Texture>
<Location>
<X>160</X>
<Y>0</Y>
</Location>
<Size>
<CX>25</CX>
<CY>25</CY>
</Size>
<Hotspot>
<X>0</X>
<Y>0</Y>
</Hotspot>
<Duration>10000</Duration>
</Frames>
</Ui2DAnimation>Change the Texture name to the name of the file with your cursor image. Change the location and size tags to match the location and size of that cursor.

Cursors are the only elements that actually use the Hotspot tag, so make sure that is correct as well. The hotspot is the location within the cursor that is the active point; (0,0) is the upper left corner.

Moving the side panels from left to right is very easy.

To get a left handed mel&eacute;e version:

In a fresh directory,
unzip the basic XymUI
unzip the mel&eacute;e mod, and
install the lefty mod.

Now open up EQUI_PetInfoWindow.xml, the only file that was changed by the mel&eacute;e mod and search for this line:

<DrawTemplate>WDT_Left</DrawTemplate>

Change the word "Left" to "Right", since you want that panel to appear on the right hand side of your screen, as such:

<DrawTemplate>WDT_Right</DrawTemplate>

You will have to drag that panel to the proper location, but otherwise you should be good to go.
