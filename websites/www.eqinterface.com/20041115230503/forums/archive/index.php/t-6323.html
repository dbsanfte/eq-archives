Copy the storyTinkering.txt file to \EverQuest\storyline. Then all you need to do is press Alt+N in-game to bring up the Storyline window and select Tinkering Recipies.

The format of the recipes is:

Animated Bait
Trivial: 50
Cost: 0.003
Cork, Spriket

I got most of the information for the recipies from the Clockwork Tinker's Union and EQTraders. Be sure to visit their fine webpages!

I just posted storyAlchemy and will probably do more when time permits. Both of these are now under the Download\Misc\story... on the bar on the left side of the home page.

As for creating your own, it's very easy. Just open notepad and start typing. The only thing you need to remember is to end each line with a <br>

The color code is standard HTML, here's a real good site with example codes: http://www.htmlgoodies.com/tutors/colors.html
The code tag is <c "#yourhx">colored text</c>

The first line in the text file becomes the Title in the Story Window, so you may want to leave it unformatted.
