Chronwaas
01-29-2003, 07:31 PM
Ok now start EQ and bring up the character you want to use this skin on.

Next just type in the following:

/loadskin playerwindow

Hit enter and after a couple seconds it will load up what ever mods you have in the playerwindow directory.

If you are looking to have multiple mods to be used as a single skin for one character you will need ot have all of the mods in that one directory.
