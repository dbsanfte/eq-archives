A caster version of my Rynoa UI is now available.

It can be downloaded here:

http://www.eqinterface.com/sdownload/fileinfo.php?id=1445

You can view a picture of it here:

http://www.eqinterface.com/sdownload/preview.php?id=1445

The melee version can be found here:

http://www.eqinterface.com/sdownload/fileinfo.php?id=1390

And can be viewed here:

http://www.eqinterface.com/sdownload/preview.php?id=1390

Remember these only workin 1280 x 1024 resolution. Both come with an INI file to auto arrange the windows and set the viewport.

The two chat boxes on either UI can be customized to display whatever text you want. They are set with an opacity of 0 so to access the menu on them simply right click near the right edge of the black area. It may take a few tries to find and it may help you to up the opacity temporarily while working with them.

Let me know what you think! You can PM me if you have any questions or simply leave a reply here.
