CamarUI v2.0

Edit 5/28/04: Updated the Bazaar Search Window to include the "Hide Traders" button and extend pull-down menus.

All files are updated for the GoD Expansion and patches as of 5-25-2004.

This UI is designed for someone running EQ at a 1280x1024 resolution. The layout was inspired
by an old UI that Grimthorn created when PoP was first released. I've made a few changed to the
appearance from v1.0. Buffs are now in a vertical list on the right of the screen, and the player
and group windows have changed a little as well.

Also included is a folder called Alternate Windows. In there I have the following windows for those
that want to use them. Just copy the file to the CamarUI folder, delete the original file there,
and rename it taking the -hybrid, -Melee, -Border, or -NoBorder out of the name:

EQUI_CombatAbilityWnd-Hybrid - a smaller Combat Abilities window only 4 buttons high for use by
Hybrid Melees.
EQUI_CombatAbilityWnd-Melee - a Combat Abilities window 8 buttons high the size of the Spells window
for Pure Melees.
EQUI_TargetWindow-Border - a Target window with a dark background and a gold border.
EQUI_TargetWindow-NoBorder - a Target window with no border or background. Same as pictures in the
screenshots on EQInterface.com.
