I dont normally play with the animations file mainly because it is high maintenance, but here is a suggestion that works for me..

If you get any error, example " Could not find child BIGB_BankerName in window BigBankWnd", then I give it to it to make it happy, edit it as needed.

Open up the default BigBankWnd file, find the BIGB_BankerName tag, copy it, then paste it into your customized BigBankWnd file. Other than the "code" portion, be sure you also copy over the <Pieces> portion that corresponds to that from the default to your customized as well.

Do this for the other windows that it is requesting.
