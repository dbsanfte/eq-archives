Here's the easiest way to change font size:

On any chat window, right click the chat area and select "Always chat in this window." Don't worry this is temporary and you can always make the main chat window the "Always chat in this window" later.

Then type /chatfontsize n

Where n is a number (1, 2, 3, 4, or 5).
