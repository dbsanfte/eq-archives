What are you using to hold the image you want to shrink down? If it's Ui2dAnimation, there's a size element within the Frame child of it. If you're using WindowDrawTemplate you probably want to use WindowDrawTemplate: FrameTemplate: Ui2DAnimation: Frame: size. ScreenPiece has a size element within it. The point is there are a few places for size, depending on what exactly you want to resize. Look in the about SIDL.doc on pages 9-13, they have the "extensive list of elements." Within that, find the containing element for what you are using, then look within that for either a size Parameter (1st column) or for a Type (2nd column) that has a size Parameter. The Type's refer to other elements within those pages.

Hope this helps.
