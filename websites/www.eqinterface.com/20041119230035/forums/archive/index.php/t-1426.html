Well this is what I do.

Put everything in the window you want. Add the player hps last in the file. Make the tile the size of the whole window. Use the offset for the gauge and name to put it where you want.

It seems the files use the graphics as they get them. Meaning whats last will be ontop. If the piece you put in the window is the same size as the window everywhere you click will select it. Play with it and you will get the effect you want.

I think he means the size. The hp bar is usually set to be around 108 by 40. If you make it size 150 by 200 it'll still look the same -- all the rest of the space will be just transparent so the other bars will still show through. If 150 by 200 is the full size of your player window, then you'll click on the hp bar no matter where in the player window you click.
