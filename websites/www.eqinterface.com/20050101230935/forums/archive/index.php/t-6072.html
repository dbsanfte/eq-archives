First of all make an empty file in the uifiles directory. Next all you need to do is unzip all the files from EQInterface into that folder. You do not need to copy the whole default files into the folder!
EQ will use the files in the default folder automatically.

Now, Since there have been some patches, and I am sure not everyone has fixed their files with what needs to be, so more than likely your files will not work, but I suggest trying out the UI once to make sure. If it does not, open the uierrors.txt file in the EQ directory. At the very bottom it will show you what file is not working ((example: EQUI_Animations.xml ))
Go here (http://www.eqgui.com/showthread.php?s=&threadid=5314) and here (http://www.eqgui.com/showthread.php?s=&threadid=6012) and open up the file that is not working in Notepad and fix the problem you find. Carien has easy to follow instructions, but not everyone finds it too easy so...

If you are not comfortable with doing this yourself or do nto understand anything, go
here (http://www.eqgui.com/forumdisplay.php?s=&forumid=30) or here (http://www.eqgui.com/forumdisplay.php?s=&forumid=16) and request assistance by telling them what file is not working and what uierrors says and if anyone will do it for you or explain how to do so.
