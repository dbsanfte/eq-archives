Download all the UI bits you want.

Install the druid UI normally. (See the FAQ for help on this.)

Load it up in game to make sure it works.

Alt-tab to your UIs directory.

One by one, install each UI to the same directory and use /load to check if it worked in game.

The bank window, container, merchant and trader mods should work fine, since they appear to be only xml. The inventory portrait probably will not work at all, since it is just a piece of artwork to replace artwork that is only used in TKing's inventory window.
