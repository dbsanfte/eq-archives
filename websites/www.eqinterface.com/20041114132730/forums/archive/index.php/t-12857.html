-==< ************ >==-

THIS set of changes fixes the Container files, and adds target CON color to the target window again. The target window is overlayed on the player window, so the center of the player window changes color based on target con.

Also, the INI file is correct now :)

This means the 36 minute timer MOVED WINDOWS. It's not on the Target window anymore, it's on the MP3 player window. That's Alt-K by default. If you don't want it there, remove EQUI_MusicPlayerWnd.xml from the UI folder and replace it with whatever you want.

The narrow container doesn't work as well as I like... yet. I'll look at it later. Next priority is fixing the spell icon colors. That will come in the next update, as well as updated screenshots.

For now, trust me ;)

Enjoy !

From the readme:

-==< ************ >==-

With this update, I've added a new Map window ( Rax's 3D Map Window by Raxthorne ) and a new Pet window ( by me ). Also, with the changes to the Tradeskill containers I've changed the default container window from the "Tradeskill" ( borderless, small ) container BACK to the default bag that came with the Sars 3.0 UI. If you want the small container window back, just copy it from the "ContainerWindow" folder and overwrite the default one.

The pet window I added uses a tab box to switch between buttons and / or buff icons. The ATTACK and BACK OFF buttons, as well as pet health gauge are always visible. The rest of the buttons switch back and forth with the pet buff icons. The default state of the window is to make all the buttons visible and hide the buff icons. If you use the buff icons alot and want that to be the default, you can change it. The instructions are in the readme.

EDIT: My 36 Timer no longer has con-color around it, because I switched to the new vertical timer. WAY cooler and fits better on the side of the melee screen. Although he took out the marks that made the other one nice, that made 6 nice divisions... I put them back in so it's more useful for Mend timing.

Last major thing I did was to add a 12 second timer on the Friends window. This is specifically for the Monk Phantom Wind Lull, and if you don't need it you can ignore it or delete the EQUI_FriendsWnd.xml file and remove that timer from your UI.

Minor Stuff:
Cleaned up the flashing box on the PlayerWindow. Now uses the red animation" rather than the old flashing red box. This was more to clean up an error than for any functional reason, although it looks alot better. ;)

Updated the UI_Playername_ServerName.ini with new settings.

-==< ************ >==-

This update completes all the little changes I've been looking at through the last few patches. Melee Abilities window is in and functional, Grey and Blue skins have been updated as of the last patch, pet window now supports buff icons AND actually looks like it belongs in this UI, and a few annoying button alignment problems have been cleaned up.

Ooooo planning on buying GoD finally. If there are requests for specific GoD features, let me know!

Enjoy, and post any problems, comments, etc here.

Thanks! :)
