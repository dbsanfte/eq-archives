12-23-2003, 04:19 AM
the error is a very generic error message. But the relation is that it's looking for a TGA file that's within the default UI but not within yours.

The solspellbook01.tga is the bug in the current Vert mods. I've already PM'd Sokol and Kudane about it. Simple fix for Vert mobs is to add spellbook01 - 4.tga into your Animations file.
(I think the solspellbook01 was renamed to spellbook01 in the later Vert UIs)

However, this error will only cause 3 or 4 files of blank warnings. Any more generally means the UI is looking for another graphics file that's not referenced.


The best bet is to make sure all TGA files within your default directory are referenced within your custom Animations file. That will remove all blank warnings as listed above.

And believe your me, the way SOE did this just plain sucks. Instead of being able to make your UI "patch proof" by adding custom graphic files and removing old "unused" files, you now need to have all files referenced. This either adds more to memory (add your custom files and the defaults) or you have to "overwrite" the defaults making your file prone to patch problems.
