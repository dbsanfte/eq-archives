Aldeneb,

love the OldCelt UI ... it's so very sweet.

On your window selector window, the Help button is missing. I played around with the xml file and discovered the cause; the window isn't long enough.

old code:

Screen item="SelectorWindow">
<!--<ScreenID/>-->
<RelativePosition>false</RelativePosition>
<Location>
<X>135</X>
<Y>0</Y>
</Location>
<Size>
<CX>325</CX>
<CY>41</CY>
</Size>

new code:

Screen item="SelectorWindow">
<!--<ScreenID/>-->
<RelativePosition>false</RelativePosition>
<Location>
<X>135</X>
<Y>0</Y>
</Location>
<Size>
<CX>350</CX>
<CY>41</CY>
</Size>

The ideal size maybe slightly longer than 350, but 350 worked well enough for me to see the help button and click it to bring up the Help window.

Normally I don't need the Help window, but I was installing the UI Timer and wanted to use the Help window for this ... but I couldn't get the Help window open and there's no command to make it appear.
