If you want to just add a section to go to your tadeskills page, then add a link in the Help.HTML file.

If you want to replace the help alltogether, then rename the origional help to help.bak, then rename your custom file to help.html

Note however that this would probably get overwritten eachtime you open Everquest. (Just tested and verified.)
