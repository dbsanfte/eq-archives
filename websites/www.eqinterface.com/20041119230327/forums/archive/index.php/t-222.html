I want a celtic knotwork border around my UI and ran into the same problem you did. What I am having to do is use a very long thin graphic for the middle pieces of the border.

You will have to make a custom border template and frame template for each window.

Since the "corners" do not get streached, you can use the same TopRight, TopLeft, LeftTop, LeftBottom, BottomLeft, BottomRight, RightTop and RightBottom, graphics in the frame templates for each window.

For the Top, Left, Right, and Bottom pieces, you will have to determin the exact hight and width of your window, and make a border template for each window that defines the lenth of the graphic you need to fill in the space between the graphics for the corners.

I believe you can make one border template that includes all the graphics for the corners and place it in the first modified window your UI loads. Then for each modified window, you can make a border template for the top, left, bottom and right pieces for that window. Then make a frame template template for each window to pull it all together.

I hope my explanation is understandable, hehe. Its really not that dificult if you already understand how to make border and frame templates.
