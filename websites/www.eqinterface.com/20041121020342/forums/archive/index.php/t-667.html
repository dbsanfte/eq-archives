Senwiz,

as explained in several threads, several files are now "cached" during start up... they load these from the folder listed in the eqclient.ini file under

UISkin=Default

If you change this setting to the folder that has the images you want (say you downloaded the purple to a folder called Senwiz) you would change it to

UISkin=Senwiz

then it would cache the images in the Senwiz folder.

Remember:
You can NOT change objects in the default folder
You HAVE to remake your custom skins folders from the new default to fix the bazaar file, and chat windows. (otherwise you crash to desktop when you type /bazaar)
You only have to do the /loadskin command once. after than anything you put in the folder loads with the character

expect further changes to the UI since it still in "beta 2" and have to remakes skins a few times...

hope that helps
