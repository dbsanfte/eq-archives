Garryck's INI Copier

A graphical utility to help you copy the contents of one of your character's INI files to another character's. This will help you greatly in setting up multiple characters for use with a UI (once you get one character's interface set up the way you like use this utility to copy the settings to another character).

You retain each character's individual hotkeys, friends list, etc. and each "touched" file is backed up and restorable via the utility so this is a no-risk tool. :)

For installation see "Install-Readme.txt" and for use see "Readme.txt" in the "INI Copier" folder on your Start Menu (after installation).



Enjoy!
