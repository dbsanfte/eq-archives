This is the UI that im using. I have put it together from pieces from Tking / Remelio (I have been told its ok to post the links =) ).

Ive modified the info windows to match a window that Remelio originally made ( that I modified, than created to match).

This is a compact, but information laiden ui that I hope you will enjoy and give me suggestions on.


If you want your window to look like mine is also, enter this command to get the proper "black space" top and bottom for the windows:

/viewport 0 47 1280 735

This is assuming your running the same resolution I am, 1280 900 (approx). You can use a small resolution, just dont use my /viewport numbers.

Also, yes, all the windows are seperate. The only one that is combined is the Pet window + Pet icon buffs.

So you can mix n match / move em all over to your hearts content.
