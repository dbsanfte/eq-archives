/// What is AYA UI? ///
This is a Custom UI File for EverQuest.

with....

Large Target WIndow with Your Mana and Cast Bar and GROUPING.
Small Spell Book / Inventory / Container / Right Click(Friend Window).
Custom Player window,Group,SpellGem,Chat and Hot Buttons.

All Your Status and Equip in Pet Window.

/// How to Use... ///

Thanks for DownLoad. This UI .ini works for 1280 x 1024 mode.

1) Open Zip file.
2) Make New Folder. Name AYA in your UI Folder. (i.e. C:\ProgramFiles\EverQuest\uifiles\AYA )
3) NEW STEP >>>>>> Copy All Default Skin File to AYA Folder. <<<<<< NEW STEP
4) Copy and Overwrite All AYAUI File (TGA and XML) to AYA Folder (Step 2).
5) If 1280 x 1024 mode , Rename UI_CHARA_40.ini > UI_(YOUR CHARACTER NAME)_40.ini (i.e. : UI_Kxes_40.ini)
6) Copy 4) File to Your EQ Folder. (i.e. C:\ProgramFiles\EverQuest )
7) Login and Enjoy.

If You find a Bug, Plz Let Me Know. I'll Check it =)

/// How to Use Mod ? ///

1) If You dont like a Ball Garge, Delete EQUI_PlayerWindow.XML and Rename EQUI_PlayerWindowOLD.XML
to EQUI_PlayerWindow.XML.then Login and Set Alpha 100%.

2) If You need a Version 3.10 Spell Book, Delete EQUI_SpellBookWnd.XML and Rename EQUI_SpellBookWnd_OLD.XML
to EQUI_SpellBookWnd.XML.

3) If You need a Pet Life, and Pet Button, Delete EQUI_PetInfoWindow.XML and Rename EQUI_PetInfoWindow_PET.XML
to EQUI_PetInfoWindow.XML.

4) are You Pure Melee Class? I made Large Target Window for Melees. Delete EQUI_TargetWindow.XML and Rename
EQUI_TargetWindow_Melee.XML to EQUI_TargetWindow.XML

5) You need XP Bar and AA Bar in Player Window, Delete EQUI_PlayerWindow.XML and Rename EQUI_PlayerWindow_Xp.XML
to EQUI_PlayerWindow.XML

6) If You dont need a Inventry Slot in Pet window, Delete EQUI_PetInfoWindow.XML and Rename EQUI_PetInfoWindow_Noinv.XML
or EQUI_PetInfoWindow_NoInvNoPet.XML to EQUI_PetInfoWindow.XML

/// Note ///

1) ALT + F (Friend Window) = Right Click Window

/// Knowledge Bugs ///

1) DON'T Make a HotButton from Petwindow. Its not a AYA UI's Bug. Its a EQ Client's Bug.
