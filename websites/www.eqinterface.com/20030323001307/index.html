UI News & Updates



Latest News - Latest Updates - Patch Fixes - EQ Interface News
[ F.A.Q ]



Latest Updates

Featured Interfaces
  • Kanon has updated AYA 6.50 to the Plus release.
  • Drakahhas re-released Drakah Tking UI - Minimized with multiple changes, fixes, and... hair?
  • Xymarra has released a Stand-alone Inventory Window for use with other UI's.

Complete Sets
  • Jethal has released NightElf 2, with player window sets for Rangers and Bards.
  • Tinkfu has created the Fu Melee Version, for pure melee classes.

Mix & Match
  • Salgurdar has released Salgurdar Player Window, updated from another Player window mod.
  • Anaa Mortalita has put together RoamieWiz, a T. King-based UI.
  • Shoggoth haas put together Shoggoth UI, a viewport UI using various windows.
  • Zeremus has put together ZWFUCR, the "Crayola Release" with color-changing HP gauges.

Compass
  • Kilamon has updated Ysoldat's EQUI_Compass to be compatible with LoY.

Group Window
  • Salgurdar has created Salgurdar Group Window, with Group and Target info, plus room for the Compass.
  • Kuvasie has released TK_Kuv_GroupLeft, a reverse-image version of her earlier Group Window mod.

Hot Keys
  • Kuvasie has released a number of Hotbutton versions, in various combinations:
    - Kuv_TK Hotbutton/Inventory/Animation
    - Kuv_TK Hotbutton/Inventory
    - Kuv_TK Hotbutton SidebySide
    - Kuv_TK Hotbutton SidebySide - Reversed
    - Kuv_TK Hotbuttons Only
    - Kuv_TK Hotbuttons/Animation
    all with the choice of blue, green, or black buttons.

Inventory
  • Anatis has released Inventory Portraits for the following classes, to be used in T.King-style inventories:
    - Bard
    - Cleric
    - Druid
    - Paladin
    - Ranger
    - Rogue
    - Shadowknight
    - Warrior
    - Wizard
    - Enchanter

Mouse Pointers
  • Kungpo has created Aven Cursor, in blue, and a little longer than normal.

Pet Window
  • Kilamon has released KilaPet, inspired by Katn's mod.

Spell Books
  • Kungpo has released an Angelbook Fix, correcting an Animations file error.

Story Window
  • XeroWing has created a Vertical Story Window, putting the list on the left, display on the right.
  • Nostrilman has released Tinker's Blueprints, with info from EQ Traders and the CTU.

Target Window
  • Lunakin has created an Oval Target Window, with custom gauge graphics.
