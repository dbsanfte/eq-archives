Notepad is good enough. All you need to do is change the EQType for the InvSlot you are talking about.

If you can be more specific about what exactly you need, we can help you more. You just need to find the exact InvSlot you need to change and change one number around.

Range is InvSlot EQType 11. Legs are InvSlot EQType 18.

All known EQtypes are in a post in the Info Vault here:
http://forums.eqinterface.com//showthread.php?s=&threadid=536

Hope that helps a bit.
