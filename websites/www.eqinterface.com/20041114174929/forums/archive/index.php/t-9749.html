08-29-2003, 11:16 PM
If you have trouble moving a window - rule of thumb is a 5 pixel border below all graphic windows in which you can position the window. If you cannot get that window to move, try right-clicking to make sure that window is not locked. If it is locked, unlock it and try to move it again.

Feel free to use my group window, but you will need to add a lot of other files as well as modify a lot of XML to make it work. I suppose I could recode my UI to make it work more with other UI's but that is a ton of extra work and I basicly keep adding on to my UI since the first version. I doubt I will ever get to recoding my UI, unfortunately.

Patches and alternate downloads will be available through both my webpage at Sars.Has.It and through the Featured UI download page here at EQInterface.com. If the patch completely breaks my UI, I promise it will be patched immediately. Else I will get the patches out as time permits.
