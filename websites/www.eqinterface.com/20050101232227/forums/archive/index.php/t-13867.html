Actually, the pulsing Delagated ring that you saw was the buggy version. The new, static Delegated ring is the way it was meant to be.

It's very easy to change the color of the Delegate ring. I've changed mine to a purple-pink color, which shows up very well against all con-colors of mobs. Follow these directions which I originally posted to the concert hall and eqdruids forums:

the default assist and Mark circles are not supposed to look like spastically flashing, undecorated concentric rings. This is a bug that happens if you are using a custom target circle which hasn't been updated to include the new definitions that SoE introduced with GoD. They actually have a runic device, and are non-animated. If yours is flashing, then you need to edit the TargetIndicator.cfg file in your custom folder to include the new xml definitions for the Assist and Mark rings. :)

To fix this, you need to copy the files "assist.tga" and the files "mark1.tga" to "mark6.tga". (7 files total) from the default UI folder into your custom UI folder.

Then open both the TargetIndicator.cfg files in your default folder and your custom folder, and compare them.

Stuff that is different in the two files include 5 new definitions for "AssistMarker" and "Marker0" to "Marker3".

Copy that section of the code from the Default file into the custom file.

Also note that each "Marker#" has its own opacity and height settings, so a few edits are needed if your custom circle has file definitions at the top that SoE went and customized in the Mark/Assist area of the file. (in my case, i had to remove the following from the top of the file:
FadeStart=0.0
FadeEnd=2.0
OpaqueStart=0.0
OpaqueEnd=0.1
InitialLength=1
GrowSpeed=1
FloorOffset=0.0

and paste it into EACH con-color definition that follows it, "VeryEasy" to "Deadly". In your case, the number of definitions you need to copy down may be different.)


Important: if you edit the file in the Default folder and leave it there, the patcher will just overwrite it next time EQ starts up.

After you do that, you just need to restart EQ, and the flashing should be fixed.

While editing the file, I found it helpful to also edit the color of the assist ring to some other color than white (255,255,255) so that the ring showed up better against even-con mobs. I changed my assist ring to pink (Red=255, Green=102, Blue=255). That helped a lot. You can also change the color for each of the Mark rings too.
