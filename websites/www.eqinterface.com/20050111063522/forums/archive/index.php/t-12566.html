Basically the error is saying that you have a Ui2DAnimations that is not complete.

An Example of correct would be:
<Ui2DAnimation item="A_AttackIndicator">
<Cycle>false</Cycle>
<Frames>
<Texture>AttackIndicator.tga</Texture>
<Location>
<X>6</X>
<Y>2</Y>
</Location>
<Size>
<CX>116</CX>
<CY>28</CY>
</Size>
</Frames>
</Ui2DAnimation>

This same code would give that error is the last line was not there or mispelled.
As you see it must start with <Ui2DAnimation . . . > and end with </Ui2DAnimation>
