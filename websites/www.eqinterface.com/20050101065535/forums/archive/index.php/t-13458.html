what I do when I try out a new UI is this. ( I know it is backwards way but it works for me hehe )

1. open up the zip file where you downloaded it to. For me in a file called "everquest UI's"

2. next open up the "UI files" in everquest. So I am looking at two windows.

3. make a new folder in the everquest "UI files", calling it whatever I want (usually test or a name of a toon I am going to try it with )

4. in the zip file I highlight all the files I need to use minus any preview or "read me" files and then left click + hold. Move them to the new folder in the everquest "UI files". Letting the computer complete the trasfer.

5. I make a copy of the "UI_ini_file" if needed and paste it to the desktop. Close the zip file.

6. back out of the new folder in the everquest "UI files" and into the main directory of Everquest look for the character that is going to try my new UI and check the name and server. Making sure of it I rename the desktop one the same as the one in everquest and then move the file into everquest. Saying "yes" to the overwrite.

7. done. close the Everquest directory and log on the game and have fun with the new UI.

Works everytime unless the UI is old or I like to see if this mod will run with this UI and they break each other. I mix and match a lot. :D

Hope this helps. also if you type /viewport reset. It get rid of the black bars you said you have at times.

My method to installing new UI's.

Create a directory in c:/program files/sony/everquest/uifiles for my new ui. In this case, we'll call it "test".

-- NEVER COPY FILES FROM THE DEFAULT FOLDER -- You will have major problems when sony releases minor patches.

Unzip the new ui file to c:/program files/sony/everquest/uifiles/test. When you are prompted to overwrite files, you must select "yes". If you do not get a message telling you that you are overwriting files, the files did not go to the right place which means the creator of the UI has probably packed them inside a directory. In this case, go to c:/program files/sony/everquest/uifiles/test and look for a directory folder with the name of your UI. Open that folder, select all, copy, and paste all those files back into your uifiles/test directory.

If the file came with a UI_CharName_Servername file I copy that to desktop, rename it to UI_Mycharname_Myservername and save it to c:/program files/sony/everquest. Load up game, click on options, click on load ui skin, load ui.

Note: Of course, you're exact Everquest folder location may or may not be the same as mine.

Hope I wrote this so it's understandable, it's easy for me to think about and do, but seems to be a pain to explain. :)
