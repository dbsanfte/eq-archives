So, if you use a mod that you are condensing space (such as the spellgems) and you DO want the tooltips but they were getting too big, you now know how to change them.

okay, basicly, try this as an example. You'll see I'm not poking my nose in the schema too far :)

Go into your EQUI_CastSpellWnd.xml file (this is your spell gems window).

scroll down to this section:
<SpellGem item="CSPW_Spell0">
<ScreenID>CSPW_Spell0</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>1</X>
<Y>1</Y>
</Location>
<Size>
<CX>30</CX>
<CY>24</CY>
</Size>
<SpellGemDrawTemplate>
<Holder>A_SpellGemHolder</Holder>
<Background>A_SpellGemBackground</Background>
<Highlight>A_SpellGemHighlight</Highlight>
</SpellGemDrawTemplate>
<SpellIconOffsetX>3</SpellIconOffsetX>
<SpellIconOffsetY>0</SpellIconOffsetY>
</SpellGem>


now, simply insert the text in bold below


<SpellGem item="CSPW_Spell0">
<ScreenID>CSPW_Spell0</ScreenID>
<Font>1</Font>
<RelativePosition>true</RelativePosition>
<Location>
<X>1</X>
<Y>1</Y>
</Location>
<Size>
<CX>30</CX>
<CY>24</CY>
</Size>
<SpellGemDrawTemplate>
<Holder>A_SpellGemHolder</Holder>
<Background>A_SpellGemBackground</Background>
<Highlight>A_SpellGemHighlight</Highlight>
</SpellGemDrawTemplate>
<SpellIconOffsetX>3</SpellIconOffsetX>
<SpellIconOffsetY>0</SpellIconOffsetY>
</SpellGem>


save the change and load up the game. press and hold Alt and notice the size difference in the tooltips. I know, nothing spectacular, but it's a start at finding something we may be able to do with those.

Originally, when I found this, I thought you might be able to do it for the entire window with one font tag placed in the screen item definition (see below)


<Screen item="CastSpellWnd">
<!--<ScreenID/>-->
<!--<Font/>-->
<RelativePosition>false</RelativePosition>
<Location>


See, I thought if you could set that font tag you could change the font for the whole window's tooltips. This, I assure you, does not work.

My next trick will be to find out if I can remove the blue background from the tooltips, or change it's color conditionally (such as in the buff widow). There are some windows where I find them useful :)
