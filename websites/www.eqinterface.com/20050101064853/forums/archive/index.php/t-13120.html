In the Delta 2.3 package, there is a file named "EQUI_PlayerWindowgood.xml" which you can replace over "EQUI_PlayerWindow.xml". But the indicator shows incorrectly. I made a tweak to it with the following...

<?xml version="1.0" encoding="us-ascii"?>
<XML ID="EQInterfaceDefinitionLanguage">
<Schema xmlns="EverQuestData" xmlns:dt="EverQuestDataTypes" />
<TextureInfo item = "AttackIndicator.tga">
<Size>
<CX>128</CX>
<CY>32</CY>
</Size>
</TextureInfo>
<Ui2DAnimation item = "A_AttackIndicator">
<Cycle>false</Cycle>
<Frames>
<Texture>TargetBox.tga</Texture>
<Location>
<X>589</X>
<Y>10</Y>
</Location>
<Size>
<CX>270</CX>
<CY>5</CY>
</Size>
</Frames>
.......

The change I made are the location and size. Now it looks the same as the Target Window.
