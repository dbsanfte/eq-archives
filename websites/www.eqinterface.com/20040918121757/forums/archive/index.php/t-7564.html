My UI is VERY gauge heavy. I don't know of any UI that uses more, it has probably a little over 250 gauges (100 from a vertical casting gauge, probably 80 or so gauges in the group window). Note that slowdown due to a complex UI is NOT LAG, slowdown CAN cause lag, but it isn't the same thing. Here are my framerates:

All framerates are clip plane all the way down, staring at the ground in Gunthak. 1.5 GHz CPU, Geforce4 Ti 4200, 512 MB SDRAM.

UI OFF (F10) - 115 fps
Default UI - 70 fps
Enoks UI (My UI) - 50 fps

So, as you can see the framerate drop isn't MAJOR, it's significant but not unbearable, and it doesn't get much worse with more gauges. I wouldn't be too worried about how this affects performance.
