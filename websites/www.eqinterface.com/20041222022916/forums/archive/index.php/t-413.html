Well, as far as Eqclient.ini is concerned, the window position settings for various resolutions do not apply to the new UI, so if you switch back, you'll still have your old settings.

All the window size and position settings for the new UI are saved in files (named UI_charname_server#.ini) that are unique for each character, allowing each one to have a completely different skin and window setup.
