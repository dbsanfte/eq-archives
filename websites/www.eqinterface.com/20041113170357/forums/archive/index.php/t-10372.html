I am posting my release of what I am terming as TymUI. It is a mix and match UI which is based on the default UI so when patch day comes, stuff does not break. I included 2 sets of icons for spells. You can use the WC3 icons which I use or the TKing icons which other people who use my UI have requested. Just copy the files over the base spell icons. I have written some of these xml files but not all, most are a mix and match and the best of the best found on this site for usuability.
