I'm assuming that you're using the "new" map feature that Sony implemented, if not, then you can pretty much disregard this whole post :)
First, check to see if the height filter is on. One thing that I have noticed when using my maps is that when the height filter is on, I can't see diddly on my map, except for the little marker showing where my character is. Secondly, check to see how "high" up the view is. Personally, I prefer to see an entire zone as opposed to one teeny tiny portion of it. When the height filter is off, you should be able to adjust the view to whatever you feel most comfortable with. It does seem to turn the height filter on and off at the games leisure, and I'm not sure why this is or if there is any fix for it. :(

Any way, hopefully this helps a bit.
