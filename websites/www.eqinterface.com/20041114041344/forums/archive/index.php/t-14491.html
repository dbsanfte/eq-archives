CamarUI v2.1

Download Here (http://www.eqinterface.com/downloads/fileinfo.php?s=&id=3474)

All files are updated for the GoD Expansion and patches as of 7-16-2004. This update adds
mana totals, endurance totals, leadership xp percentage, and a full window flashing attack
indicator to the player window, a hide trader button to the bazaar search window, and makes
the containers compatible with the new system. Also have removed the EQUI.xml file, making
this UI much harder to break and easier to modify with other windows. Removed a few of the
UI pieces from other authors to shrink the download size.

This UI is designed for someone running EQ at a 1280x1024 resolution, but is workable at most
resolutions 1024x768 and up with some moving around. The layout was inspired by an old UI
that Grimthorn created when PoP was first released.

Installation:
1. Unzip the CamarUI21.zip file.
2. Copy the CamarUI2.1 folder into your Uifiles folder.
3. Rename the UI_character_server.ini file with your character name and server name.
4. Copy the renamed file into your main Everquest directory.

Also included is a folder called Alternate Windows. In there I have the following windows for those
that want to use them. Just copy the file to the CamarUI folder, delete the original file there,
and rename it taking the -hybrid, -Melee, -Border, -NoBorder, -Titled, or -ReSizable out of the name:

EQUI_CombatAbilityWnd-Hybrid - a smaller Combat Abilities window only 4 buttons high for use by
Hybrid Melees.
EQUI_CombatAbilityWnd-Melee - a Combat Abilities window 8 buttons high the size of the Spells window
for Pure Melees.
EQUI_TargetWindow-Border - a Target window with a dark background and a gold border.
EQUI_TargetWindow-NoBorder - a Target window with no border or background. Same as pictures in the
screenshots on EQInterface.com.
EQUI_ChatWindow-Titled - Adds a title bar to chat windows, making them dragable and closeable.
EQUI_MerchantWnd-ReSizable - Makes this window resizable for people wanting to shrink the window.
