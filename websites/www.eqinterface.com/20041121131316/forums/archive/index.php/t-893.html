On any errors where the default is loaded, check the UIErrors.txt file in your EQ folder to see why.

One common problem is copy/pasting the exact code from one window into another. If items with exactly the same names exist twice, the UI will often error stating that duplicate items exist. In this case, simply change the <Gauge item="[name]"> and <Pieces>[name]</Pieces> tags in one of the windows.
