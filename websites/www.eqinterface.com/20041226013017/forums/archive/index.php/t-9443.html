I cant find a way to toggle the display in game yet, but through some experimenting I found out how to at least change it before loading EQ.

If you look in your Warcraft ui folder, you'll find 3 files: EQUI_HotbuttonWnd, AAXP of EQUI_HotbuttonWnd, and EXP version of EQUI_HotbuttonWnd. By deleting the basic file and copying the EXP version, I was able to switch the display to EXP after renaming the copy and removing the 'copy of EXP version of' part of the file name so that it was simply called EQUI_HotbuttonWnd.

Hope this helps, and if someone knows a better way of doing this, please let me know.
