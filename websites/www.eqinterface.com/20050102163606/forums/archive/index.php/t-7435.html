If you open EQUI_ChatWindow.Xml ..located in The abatooirs Ui Folder ..at the very end of the file you should see the lines:

<Style_Border>false</Style_Border>
<Style_Sizable>false</Style_Sizable>

I Included Both line for the purpose of having a visual aid to have something to grab on to with pointer to make the window Sizable and Movable ... But if you change those 2 lines above to "true" the window should be movable and sizable again. Changing them back To "false" will make it back the way it was =) ...But should atleast make it so your able to Move the window where you wish it to be =)

Hope This Helps =)
