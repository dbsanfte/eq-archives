when i try to go into my note pad it says..*windows can not find Notepad.exe* type in the executable file to be used instead...what do i type?

Notepad is a Windows file. It is a protected system file and cannot be removed. (at least not easily)

It should be located in your Windows or WinNT folder, as well as your System32 and dllcache folders.

If it is giving you an error when trying to open it, then put in your Window CD, then delete notepad.exe from all three directories and let Windows restore it.
