That parse error means that the xml was incorrectly typed out on line 128 of your Selector Window file.

Find line 128 of the EQUI_SelectorWindow.xml file in a text editor and a) check the syntax yourself OR b) post the paragraph block that line 128 belongs to for someone here to correct the syntax.

Most likely what happened is that a bracket got opened and did not get closed (there's a "<-- text here ", but no " -->" to close it out).

I'm sure I just confused you (and anyone else reading this), but it amounts to this: the xml for that Selector Window was not properly coded. For some reason, line 128 has an error in it and it needs to be corrected. Which selector window did you download, who coded it, and where did you get it from? Answer those questions and someone can probably get a fix up for you (or tell you where to look to get a fix).
