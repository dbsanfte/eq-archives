I could very well be misunderstanding your question, but if not the solution is very simple.

All you have to do is create a specific modded UI folder for EACH toon or desired UI setup.

For example, I have a specific UI setup for each of my toons. My monk's folder is called "monk". My mage's UI folder is called "mage", and so on.

In your "uifiles" folder in the EQ main directory you will have the default folder and your Ranger's modded UI folder. Just make an additional modded UI folder for your SK.

To do this, make a copy of your "default" UI folder and rename it whatever you'd like. "SK" comes to mind. Then just copy and paste whatever modded UI files you're wanting to use for your SK into the SK folder, log onto your SK, type "/loadskin SK" and that should do the trick.

You just have to create a specifically modded UI folder in your uifiles directory for each different UI you want to use.

Hope that answers your question :).

All correct, except for the part about copying your default folder into your custom folder. Do not do this. The only files that should be in your custom folder(s) are the files that are actually custom. EQ will automatically go and get any files it needs out of the default folder if it doesn't find them in your custom folder. Copying your default into your custom merely means that many more files that can "break" every time SOE patches something.
