I've never used this UI before, but I know that one of the first problems is that you copied the default folder!! NEVER COPY THE DEFAULT FOLDER TO MAKE A NEW ONE!! Just create an empty folder named Oceanspray (or whatever) and unzip the changed files into that folder. Then try adding the patch. This may help, but I'm not sure.

By the way...copying the default folder will work, but the next time Sony patches, it will probably break. EQ is set up to grab the files from your custom folder, but if it can't find a file there, it will automatically pull it from the default folder. So no need to do the copy. Hope this info helps. ;)
