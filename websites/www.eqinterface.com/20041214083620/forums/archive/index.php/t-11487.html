Copy all of the enclosed files into the ../everquest/uifiles/EQPDA directory
Rename your UI_(yourcharname)_(yourservername).ini file for a backup in case you wish to revert to your old UI.
Copy UI_Dreamlife_Xegony.ini to the ../everquest directory and rename it to UI_(yourcharname)_(yourservername).ini.

This UI contains 2 custom folders for your own customization options.

../EQPDA/Pure Mele
../EQPDA/Caster Hybrid

First choose pure Mele or Caster versions. Copy the EQUI_HotButtonWnd.xml and EQUI_GroupWindow.xml files from the ../EQPDA/pure mele OR ../EQPDA/caster hybrid folder into the ../EQPDA folder.

UI_Playername_Servernumber.ini, Must be included for placement of windows. If you do not use this file the UI will function but will look like a puzzle waiting for help. All the windows are locked in place by the ini included with this UI. If you choose not to use the included ini or want to move the PDA from its location you will need to unlock and move each window individually. You WILL notice if they are not lined up perfectly.

AFTER YOU HAVE COMPLETED ALL OF THE PREVIOUS STEPS.....AFTER... Log into EQ. If you used my INI you're done. If you did not use my INI, at the chat prompt type /loadskin, select EQPDA and have fun moving windows!
