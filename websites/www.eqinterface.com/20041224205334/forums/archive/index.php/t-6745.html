I attached a copy with the correct alpha channel..

If you want to learn how to make the alpha channel: Do a search on this site for Alpha Channel.. It has been explained a few times, for a few different graphic editors..

Quick Explaination for Paintshop Pro: Load the file and select everything you want to be displayed.. Then click selections.. Then Save To Alpha Channel.. Then Double Click New Channel (Or the existing alpha channel).. Then ok.. Save the file and you're good to go..
