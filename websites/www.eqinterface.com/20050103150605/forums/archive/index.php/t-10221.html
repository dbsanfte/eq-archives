SoE changed the label names to show percents back on May 14th 03. You can fix it here: http://www.eqgui.com/showthread.php?s=&threadid=8467

Doing this will add another % to many gauges so you will want to set the position and size to 0 in the code given in the thread. Or if you’re really ambitious you can remove all the old % labels and use the new ones.
