EtasiUI is available for download in the Complete Sets section (pending admin approval), and the Read Me file enclosed in the download contains all of the necessary information.

Couple things:

1. The spell icons shown in the attached screenshot aren't included in the download (would need to get permission from a few people before I could make them available to the public) but they may be made available for download in the future if anyone shows interest in them.

2. The custom class animation shown in the screenshot isn't available for download because I just haven't felt like modifying it for all of the classes. I'll post the .tgas if anyone wants to mess with them on their own.

3. Due to the fact that my UI actually uses two backgrounds, the brown (default) one shown in the chat boxes, and the reddish patterned one shown in the player/group windows, most of the windows can't be exported to other UIs without the inclusion of a few other files (detailed in the read me).

4. There's a folder called 'options' included in the download that contains a modified AA window, bazaar search window, raid window, raid options window, and a vertical version of the inventory (this is the one shown in the screenshot, but the horizontal version wound up in the main folder of the zip on accident).

That should be all the necessary info. Let me know if you like the mod or if you have any problems with it.

Enjoy! :)

Edit: A few posts below this one is the zip file with the inventory window files in the proper folders (uses vertical by default).
