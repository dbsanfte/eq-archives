The issue seems to be a change that was done to some people's ini's yet not to all. (It happened on one of my pcs yet not the other for the same character).

Edit your characters INI file (ui_character_server.ini), do a search for "SkillsWindow", change the values for the width/height. (example below)

Width1280x1024=0
Height1280x1024=0
change to.....

Width1280x1024=300
Height1280x1024=240

ok I had this problem myself. The way I fixed it was to edit the UI_charname_servername.ini file.

Find the section SkillsWindow and you should see something like this:

[SkillsWindow]
XPos1024x768=300
YPos1024x768=300
Width1024x768=400
Height1024x768=300
BGTint.red=255
BGTint.green=255
BGTint.blue=255
BGType=1
Fades=true
Delay=2000
Duration=500
Alpha=255
FadeToAlpha=255
Locked=false
XPos1280x1024=300
YPos1280x1024=300
Width1280x1024=374
Height1280x1024=414

You need to make sure that the position and size values are not zero for the resolution that you are using (mine was before I changed em). Set these to sensible values and the window should display again.

Hope this helps
