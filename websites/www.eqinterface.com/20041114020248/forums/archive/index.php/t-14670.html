Quartzlike Player Window Update v1.3
by DarkSir23 (Moggan Proudfoots)

NOTE: If you're using Quartz 1.7 or better, you don't need this Player Window. It is included in
the new versions of Quartz.

This is a quick revamp of the Player Window from the Quartz 1.6 UI by Statix Cinderblade.

This originally started out as a personal project for myself and my wife, but it came out so
well, that I thought I'd share it.

INSTRUCTIONS: If you are using the Quartzlike Player Window with Quartz 1.6, just copy all
the files into your Quartz directory. For standalone usage (Or with another UI), copy the following
files into your UI directory: EQUI_PlayerWindow.xml and wnd_bg_player.tga

Version History
v1.3 - Gave a bit more room for the HP, Mana, and END numbers. Should be room for 5 digits now.

v1.2 - Added Attack Indicator box around window.

v1.1 - Minor Update -- Allows for Standalone usage.

v1.0 - Initial Release:
* Squeezed the status bars together, added a Group Leadership XP Bar
* Rearranged the current/max HP indicator, AC and ATK score
* Added Current/Max indicators for Mana and Endurance.
