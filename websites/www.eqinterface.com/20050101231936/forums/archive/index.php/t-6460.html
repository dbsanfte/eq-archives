If the cursor you are talking about is this one :

http://www.eqinterface.com/sdownload/fileinfo.php?id=94

Then what you need to do is to change the EQUI_Animations.xml file in directory where your UI is to what are in the "Changes to Animations.txt" file.


If you have a custom EQUI_Animations.xml file, you will have to keep it current with the Patch fixes. If you DON'T have a custom EQUI_Animations.xml file, copy that file from the default UI to your UI and add the following. ******



Note: This file is only if you are mixing with other mods - if it works fine, you don't need to do anything ;)

- Remelio Gemweaver

Changes to Animation.xml are as follows:

<TextureInfo item = "naturdef.tga">
<Size>
<CX>64</CX>
<CY>64</CY>
</Size>
</TextureInfo>
<Ui2DAnimation item = "A_DefaultCursor">
<Cycle>true</Cycle>
<Frames>
<Texture>naturdef.tga</Texture>
<Location>
<X>0</X>
<Y>0</Y>
</Location>
<Size>
<CX>41</CX>
<CY>57</CY>
</Size>
<Hotspot>
<X>1</X>
<Y>1</Y>
</Hotspot>
<Duration>1000</Duration>
</Frames>
</Ui2DAnimation>
