Downloads

The following files are available for download:

  • GrpXpFul.zip - the GroupExperience Calculator program with required DLLs.
  • GrpXpDll.zip - the DLLs in a separate file.
  • GroupExp.zip - the GroupExperience Calculator program without DLLs.
  • Exp.zip - Excel 97 Spreadsheet version of GroupExp (includes graphs).

Remember, if you download GroupExp.zip, be sure to read the Readme file to ensure that the program runs properly. If you have never downloaded GroupExp, you probably need GrpXpFul. If you have downloaded before, you're ok.
