Wednesday, May 1, 2002

Uploaded another development version of EQini. This one fixes a bug with the Texture Quality setting. I had the values for High and Low reversed. Minor Update only. Big thanks to Evalin of Tallon Zek for pointing this out to me.

Saturday, March 23, 2002

Uploaded another development version of EQini. This one adds support for the new User colors (colors 35 - 51), and the ShowGrass option. It also corrects a small bug in the support for keymaps. I had the default for "KEYMAP_AUTOPRIM" set as "NUMPAD+" instead of "a". I've had it set to NUMPAD+ for so long I'd forgotten that I ever changed it. I'm working on adding support for the /hitsmode settings, TargetGroupBuff setting, and the new channel settings.

Friday, January 18, 2002

Uploaded another development version of EQini. This one supports all the SOL options in the eqclient.ini file (at least all the ones in my eqclient.ini but since I don't have SoL there might be something missing). Many thanks to Gavin Schneider for information on "ShowNamesLevel". Anyone using "DefaultLanguage"? As far as I know this is the only option that I haven't got supported (to some degree at least).
