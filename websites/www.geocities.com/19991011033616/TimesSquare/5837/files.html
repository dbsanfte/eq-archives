The More Than Complete Character Record Sheets
These character sheets were originally created on the Macintosh in PageMaker 6.5 and are compatible with the Windows version of PageMaker. These sheets were designed with the 'official' typeface in mind—I used the typeface that TSR used for the AD&D 2nd Edition Logo (which, if you have the Adobe catalog is called "Friz Quadrata"). If you don't have this font, just use standard "Times." This file comes in a variety of different formats, available below. Note: if you don't have PageMaker, Photoshop or Corel will open the EPS file, but be warned: compressed these files are 260k, uncompressed they near 5 MB.

Character Sheets (PageMaker 6.5, BinHex, 260k)
Character Sheets (EPSF-Encapsulated PostScript, BinHex, 120k)
Character Sheets (HTML Format, 20k)

Pools of Darkness Saved Games
This file is an archive that contains saved games in the AD&D computer game Pools of Darkness. I played the game on the Mac, but I received a file for the IBM so those with PCs can download that archive. In any case here's the package deal you get when you get this file: 6 characters, with minimum 190 HP and minimum AC of -14. Most weapons have at least a +5 bonus to hit and damage. If you're having problems with the game, download one of these files below.

POD Saved Game (Macintosh BinHex)
POD Saved Game (IBM/Compatible ZIP)

Dark Queen of Krynn Saved Games
This archive is a stuffit archive that contains some saved games in the AD&D computer game the Dark Queen of Krynn. I played the game on the Mac (as always), so if you've got an IBM, you can download them—though I don't think they'll work. In any event, these are not characters to be messed with!
