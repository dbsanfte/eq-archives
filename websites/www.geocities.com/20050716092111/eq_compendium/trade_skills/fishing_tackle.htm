Fishing Tackle

General:

          Fishing Tackle making is an extremely new skill.  It seems to involve combining various items into a tackle box.  Again, this is an extremely new skill so most of the information about it has yet to be discovered.  If you have any other information about making Fishing Tackle, please contact me.

 

Things to Note:

·        The main components for this skill, including tackle box can only be bought in the Ocean of Tears, on the sister island

·        Tackle making seems to use the Fishing skill that all characters have
