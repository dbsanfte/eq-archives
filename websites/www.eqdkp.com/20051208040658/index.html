March 16, 2005
EQdkp 1.2.0 Security Update (submitted by tsigo at 02:10pm EST)

A security vulnerability was recently discovered in all previous versions of EQdkp, up to and including 1.2.0. A patch has been made available to fix the vulnerability and bring the version up to 1.2.1 that is highly recommended for all users to install.

To Upgrade:
  • First ensure that EQdkp 1.2.0 is installed. This patch was only applied against the latest 1.2.0 code and may break earlier installations if applied.
  • Upload the selected files from this patch.
  • Because 1.2.0 is already installed, delete the install.php file on your web server.
  • From a browser, access the upgrade.php file and select 1.2.0 from the Previous Version drop-down.
  • Delete the upgrade.php file from your web server for added security.
If you encounter any problems please post in the support forums.
