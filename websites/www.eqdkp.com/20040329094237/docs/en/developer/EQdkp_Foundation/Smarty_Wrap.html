Wrapper for Smarty class

Perform common output-related tasks and provide a better interface to the Smarty base class. Also allows for assigning blocks iteratively, rather than only all at once.

method dyn_entry [line 494]

mixed dyn_entry( string $menu_action)

Create a dynamic menu entry.

Allows a menu row in the database to call a PHP file to create a dynamic menu row on the final page output. Applications include creating a list of the styles on the menu so that the user can easily preview the page with a different style.

method set_vars [line 259]

bool set_vars( string|array $var, [string $val = ''], [append $append = false])

Assign values for class vars.

Two special cases:

  1. If var is "template_file", '.html' will be replaced with our current $output_type in val.
  2. If var is "display" and val is true, page_header() is called automatically.




Tags:

access:  public


Parameters:

string   $val   Value to set var to
