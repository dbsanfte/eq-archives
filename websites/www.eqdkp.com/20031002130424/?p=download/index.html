EQdkp CVS
EQdkp developers utilize SourceForge's CVS (Concurrent Version System) server to manage EQdkp's core code. Below are instructions to download the latest EQdkp code. Anything downloaded shouldn't be considered stable and should not be used in critical environments. 
cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/eqdkp login
Press enter when asked for a password
cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/eqdkp co eqdkp
