method array_csort [line 421]

	

    

    

		array array_csort(
array
$array, mixed
$array_args)
    

    


	
		Perform an SQL-like column sort on an array

Allows us to sort DKP data stored only in an array.  $array_args must be  an array in the format array('field_name', SORT_ORDER) or a string in the  format 'field_name SORT_ORDER', where SORT_ORDER is SORT_ASC or SORT_DESC.  field_name MUST exist as an array key in each array value in $array.

method uri_self [line 303]

	

    

    

		void uri_self(
[bool
$escape = true], [string
$skip = ''])
    

    


	
		Return a string that contains the name of this page as well as any  requested input var=val pairs.

Loops through $input and appends the input variables to a URL  string that begins with the filename of the current page.  If the input  key is found in $skip then that variable isn't appended to the string.  

  Example:
       Current page is "listmembers.php?s=&rank=Member&filter=Plate"
       $eqdkp->uri_self(true, 'rank') results in "listmembers.php?s=&amp;filter=Plate"
       $eqdkp->uri_self(true, 's,filter') results in "listmembers.php?rank=Member"





Tags:





  

    access:  public
  












        
Parameters:

    

    

          

        bool  
        $escape  
        True: use '&amp;', False: use '&' in query string
      

          

        string  
        $skip  
        Comma-separated list of input keys to skip
