ADJUSTMENT ENTRY SYSTEM ENHANCEMENT REQUEST

This is a request I have for the adjustment entry screen after using EQDKP 1.3.0 for about 2 months. It would make my life so much easier for maintaining and completing the DKP for our raids.

1) Please change the adjustment entry screen so that we can paste lists of adjustment names like creating a raid does instead of using CONTROL CLICK to select multiple people.

2) An even better way to help us maintain this information would be to have the pastable list box similar to the Raid Entry screen, but have a button to click which will transfer all the names of the intended adjustee listing to a row by row listing which I could then adjust all of the names and their adjustment reasons and values individually and separately.

There would be another button to PROCESS the list of rows and automatically generate separate adjustment entries per person.

There would need to be a few extra entry boxes to handle default values. Default Description, Default Value that should be set before the user clicks the >> Convert to lines button transfering items from the multi-line paste box to separate rows.

Example

Multi-line paste box for the Names:

Default Description: Raid Adjustment

Default Value: 0.50

Names:

Bob

Frank

Nick

Clicking >> Convert to lines converts the list of people entered into separate lines to be saved to separate rows in the adjustments table.

Bob,Raid Adjustment,0.50

Frank,Raid Adjustment,0.50

Nick,Raid Adjustment,0.50

And now I can just go down the list and adjust the description or amount as desired for the certain people.

Bob,Raid Adjustment,3.50

Frank,Raid Adjustment,0.50

Nick,Raid Adjustment,0.50

PROCESSING INTO SEPARATE ADJUSTMENT DETAIL LINES

Clicking >> PROCESS would create INDIVIDUAL adjustments PER person not a GROUP adjustment with multiple people with all with the same value.

You could also add this as an option in the EQ configure system, to use NEW or OLD style system to give the old timers flexibility to use old system.
