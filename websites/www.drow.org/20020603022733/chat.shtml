Useful IRC Commands

Command Description
/clear Clear the chat message screen
/nick 'nick_name' Change your current nick name to a new one
/join #channel_name Leave your channel and join the new one
/msg nickname message Send a private message to another user
/action message Display message in EMOTE format
/topic #channel new-topic Change the channel topic to the new one
