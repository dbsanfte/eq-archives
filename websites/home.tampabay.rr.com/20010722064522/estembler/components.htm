A plug-n-play component which encapsulates the functionality, logic, and design of the classic Breakout game.
Play online (requires Microsoft's Internet Explorer)

CDPlayer
A component which provides CD player functionality.  Utilizes the WinMCI APIs internally.
SystemBitmaps
A component which provides dynamic access to the Windows OEM bitmaps.
FolderLocations
A read-only component which provides access to Window's system folder locations (ProgramFiles, Personal etc.).
BitmapMetafile
A component for dynamically converting Windows metafiles to bitmaps.  The bitmap version of the metafile takes up less memory and takes less time to display.
Wizard
A control which implements the Windows "wizard" paradigm.   This component was used system-wide by Sitech, Inc. in their SIPAC applications.
DBQuestionGrid
A data-aware question grid control.  This component was used system-wide by Sitech, Inc. in their SIPAC applications.

ClientFormController
A component for controlling/managing client forms.  Client forms are dynamically created using a metaclass, and  can be limited to a specified number of instances.
CaptionBar
A caption bar control like the one used by Microsoft Outlook.  A corresponding image can be assigned to the control.   This component was used system-wide by Sitech, Inc. in their SIPAC applications.
ProblemDialog
A "problem" dialog component, which displays a user-friendly error dialog box.  This component was used system-wide by Sitech, Inc. in their SIPAC applications.

ProgressDialog
A progress dialog component, which displays a customizable progress dialog box.     This component was used system-wide by Sitech, Inc. and TRAC'R Software, Inc..

SysInfoButton
A custom button control, which displays Window's System Information when clicked.  Usually placed on an About box.  This component was used system-wide by Sitech, Inc. and TRAC'R Software, Inc..
ReportX
An ActiveXForm control used by TRAC'R Software, Inc. in their Tax Trac'r 97 product.  The control provided access to all of the application's reports and encapsulated all of the reporting (Crystal Reports) functionality.  It was used in all of their programs which contained access to reports.
