Dedicated Servers - Technical Details



Powered by Cobalt Networks RaQ server appliance, The SimpleNet RaQ Dedicated Server includes: root access, FTP, Web, email and virtual domain support.

Each machine can concurrently handle over 140,000 emails, 50,000 file transfers, and over 250,000 web page requests in one day. If being used only as a web server, each machine can serve over seven million web page requests per day.

Cobalt RaQ Standard Features

All Cobalt RaQ Dedicated Servers include the following standard features: Linux OS, Apache Web Server, FTP Server, SendMail , Majordomo List Server, FrontPage Support, Name-based Virtual Hosting, Easy to Use Web-based Server Administration, and SimpleNet's DNS Administration Tool.
