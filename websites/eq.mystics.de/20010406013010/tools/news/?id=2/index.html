 GUTE NACHRICHTEN FÜR TDSL USER [ 19:07 MET • Andragon • Internet ] 
Eine kleine Sensation bahnte sich auf der gestrigen CeBIT an, die vor allem für die Onlinespieler unter den T-DSL Benutzern interessant sein dürfte. Im Rahmen des Gesprächs mit der Initiative "Flatrate-Protest" wurde von der Telekom bekanntgegeben, dass man "in naher Zukunft Fastpath einführen wolle".

Onlinespieler bei denen ein paar Millisekunden mehr oder weniger schon den Unterschied zwischen Punktgewinn und -verlust ausmachen können werden sich über die Neuigkeit besonders freuen. Interessant ist Fastpath aber nicht nur für Spieler, auch für "Telearbeiter", die mit Telnet, X-Windows oder Terminal Server auf entfernten Rechnern arbeiten, fällt die Latenzzeit ebenfalls ins Gewicht.
