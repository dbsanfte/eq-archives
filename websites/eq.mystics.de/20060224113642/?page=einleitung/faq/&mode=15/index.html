15.01. Wird das Starter Kit außerhalb der USA verkauft?
Das Starter Kit wird von einigen Händlern in den folgenden Ländern verkauft werden:
  • Canada
  • UK
  • Germany
  • Australia
  • Austria
  • Scandanavia (Sweden, Denmark, Norway, Finland)
    Wenn Ihr in einem dieser Länder lebt, kontaktiert Euren Händler und erkundigt Euch, ob er das Starter Kit verkauft.
    • 15.02. Wird der Starter Kit in Frankreich erhältlich werden?
    Es wird eine sehr begrenzte Zahl von französischem EverQuest II preorder Kits geben und die werden in orten wie Micromania, Scores Games, Armageddon und Ubi Online Store. Diejenigen die das Preorder Kit unbedingt haben möchten werden empfohlen so schnell wie möglichst vor zu bestellen so lange der Vorrat reichet.
    15.03. In welchen Formaten wird EverQuest II in Europa erhältlich werden?
    EverQuest II wird als CD, DVD und Limited Edition erhältlich werden.
    15.04. Was beinhaltet die Limited Edition?
    Wir können noch nicht bestätigen ob die Limited Edition in-game Objekte beinhalten wird oder nicht. Wie schon bekannt beinhaltet die U.S version in-game Objekte wie paintings von EverQuest II scenery/screenshots, sprechende Staturen von Antonia und Lucan, and an exklusives Babydrachen Haustier. Diese Gegenstände jedoch werden nur in Häusern der Spieler eingesetzt werden können. Es werden weitere Informationen zu der europäischen Limited Edition auf EverQuest II in den kommenden Wochen verkündet.
    15.05. Wer hosted die britischen, französischen und deutschen Server?
    SOE wird alle Server in Nord Amerika und Europa hosten.
    15.06. Wo werden die UK/FR/DE Server physikalisch lokalisiert?
    Die europäischen Server werden in Amsterdam stehen.
    15.07. Wer wird die Kundenbetreuung für die Server zur Verfügung stellen?
    SOE wird alle Kundenbetreuung in Nordamerika und in Europa führen.
    15.08. Wird EverQuest II komplett in französisch und deutsch übersetzt?
    Ja. Der ganze Ingame Text und die Sprachausgabe wird sowohl in französisch als auch in Deutsch übersetzt. Jede Version wird komplett in ihrer Muttersprache zuspielen sein, wenn man sich zu einem französischen bzw. Deutschen Server verbindet.
    15.09. Können Leute mit den europäischen Versionen des Spiels auf U.S Servern spielen?
    Ja, unser Plan ist es einen Patch bereit zustellen, der es Europäischen Spielern erlaubt auf U.S. Servern zuspielen.
    15.10. Wenn ich mir die Französische oder die Deutsche Version von Everquest 2 kaufe, und mich auf einem U.S. Server einlogge, in welcher Sprache werde ich dann das Spiel spielen.
    Um auf einem U.S. Server zuspielen werden sich Spieler mit französischem oder deutschem Account sich einen englischen Sprachpatch downloaden müssen. Wenn sich diese Spieler dann auf einem U.S. Server einloggen, wird aller Ingame Text auf Englisch sein. Wegen der Größe der Sprachdateien werden wir diese nicht als Patch für andere Sprachen zur Verfügung stellen. So werden französische und deutsche Spieler auf U.S. Servern zwar den ganzen Text in Englisch haben die Sprachausgabe aber in ihrer jeweiligen Heimatsprache.
    15.11. Können Spieler mit der U.S. Version auf europäischen Servern spielen?
    Ja, unser Plan ist es einen Patch bereit zustellen, der es U.S. Spielern erlaubt auf Europa Servern zuspielen. Aber wie schon oben erwähnt, die Sprachausgabe wird nicht mit gepatcht. Also werden U.S. Spieler auf Europa Servern zwar allen Text auf Französisch oder Deutsch haben, aber die Sprachausgabe weiterhin in Englisch.
    15.12. Kann die gleiche EverQuest II subscription benutzt werden um auf einem U.S. und auf einem Europa Server zuspielen?
    Ja, Du kannst auf jedem nicht-premium Server in den USA oder Europa mit einer EverQuest II subscription spielen. Aber die Sprachausgabe wird immer in der Sprache der jeweiligen Landesversion sein.
    15.13. Weist die UK Version von EverQuest II Unterschiede zur U.S. Version auf?
    Die UK Version von Everquest 2 ist identisch mit der U.S. Version. Wir fordern aber UK Spieler auf, auf den lokalisierten englischsprachigen Europa Servern zuspielen, weil das die beste Entscheidung für optimalen Spielspass ist. Aber natürlich können die UK Spieler auch auf den U.S. Servern spielen wenn sie möchten.
    15.14. Werden Europäer die U.S Version Gesetz mäßig kaufen können.
    Es wird Händler geben die das Spiel auch in U.S Version liefern, aber es wird empfohlen die lokalisierte Version zu kaufen um auf den Europäischen Servern zu spielen und somit ein optimalen Gameplay zu erhalten.
    15.15. Wird es ein Server in Australien geben?
    Nein, es ist derzeit nicht geplant ein Server in Australien auf zu stellen.
