SaviorrI have a question as to what these Tooths (teeth) are for?
ThistleburrGreetings!

These teeth are are used for the portal spells to those zones. You need to have a tooth for the appropriate zone to be teleported there.

The order of teeth is:
Cobalt Scar (CS)
Wakening Lands (WL)
Great Divide (GD)

If you have a CS tooth you can be ported to CS, WL, or GD. If you have only a WL tooth you can be ported to WL and GD. If you have only the GD tooth then you can be ported to GD. Iceclad portal requires no tooth.

Once you get a CS tooth you can throw away the other two.
