TaseMix - UI Mod

My UI is heavily based off Drakah's UI. You may find you prefer that one rather than mine.

First off, the disclaimer: This is an advanced UI. It's also a very personalised UI - I customised it for my play style, and don't currently have any plans to make changes based upon user requests. Sorry.

I don't use the pet window at all, so it's been converted into a player stats window. Also, the timers are currently hard coded into the MP3 window and are not easily customisable. The skin is currently for 1280x1024 resolution only. The casting gauge appears underneath the mana gauge when a spell is cast.

You can view a screenshot of the skin here.

Generic info for installing skins can be found here. Specific information, including how to install the INI file for correct window placement, can be found in the MUST_README.rtf file included in the main zip.

Update: 31/1/03, version 1.2
Just a few minor fixes and modifications.

More timer options are now available.
  • The default for the main UI is a set of generic timers (5, 25, 36, 60 minutes), also included is a set of timers for bards (36, 12, 1, 60 minutes). You will need to rename the EQUI_MusicPlayerWnd_Bard.xml file appropriately.
  • The melee UI has a set of timers suited to a monk/rogue as defaults (6, 36, 25, 60 minutes), also included is a set of timers for warriors (15, 3:30, 10:30, 7:48 minutes). You will need to rename the EQUI_MusicPlayerWnd_War.xml file appropriately.
Unfortunately, you will need a seperate UI directory for each different set of timers you wish to use.

Some slight graphic changes to the MP3/Timer window.

Moved to a modified version of Keiz's buff windows. These feature a number down the side for each buff, the song window is positioned at the bottom and simply acts as buff slots 16 - 21. The old style buff windows are included if you prefer them, as EQUI_BuffWindow_Default.xml and EQUI_ShortDurationBuffWindow_Default.xml. Rename them appropriately if you wish to use the plain list style (without numbers, left-aligned, with borders) for your buffs.
Update: 9/1/03, version 1.1
New version released, with a couple of minor changes. Pet window is back! Now appears as a tab behind the player stats for all those people who like using it. Also fixed a couple of small text alignment issues, changed the actions window slightly, and the melee version now sports a mend timer which may please some monks. The new short-duration buff window is also included.

I haven't had a chance to test the new version much since the 9/1 patch (the live servers aren't even up as I write this), but there shouldn't be any major issues.

The caster and melee versions are now split into two completely seperate downloads. This makes for a little more download time, but it should be much easier to install (and maintain). They both include instructions for installing, so just pick the one you wish to use and download it.

Changes from Drakah Mix, and credits:
Inventory Window
Since I moved all the stats and resists off into the old pet window, I reclaimed the space from the inventory window. It's styled after the old layout of the inventory, with buttons moved around and a larger destroy button since I find myself destroying an awful lot of crap these days.
Mp3/Timer Window
The timers are courtesy of Arista, more info can be found at the UI Timer Homepage. Since I could find no other way to get the timer window to stay up all the time, I attached them to the Mp3 player. While you can customise your own timers from the homepage, getting them into the Mp3 window can be a pain and is not recommended unless you know your XML.

The Mp3 player itself is a modified version of the Vert Mp3 player, rearranged to fit in nicely. You can find the original version here.
Player Stats Window
This is the old pet info window. HP/Max HP, AC, ATK, all stats and saves, plus weight are listed here.
Hotbutton Window
Added some more space for clickies, and made it a little wider to fill up empty space.
Spell Gem Window
Removed the permanent labels next to the spell gems. The tooltips that appear when you hold down ALT usually end up hidden behind other windows; hitting ALT-S twice can fix this but I find it only mildly annoying. Spell gems and the cast bar are courtesy t.king and Remelio
Item Display Window
Added a Lotus Cult logo to the background. Most people probably don't want this, so deleting the EQUI_ItemDisplay.xml file will change it back to the default.
Tracking Window
Simplified tracking window, suited to a druid or bard. Rangers will likely prefer the default tracking window.
Buff Window(s)
Modified version of Keiz's buff windows. I stretched the blue/red backgrounds to fill the entire buff slot, and made them slightly wider.
Compass Window
Small graphic tweaks to make the letters more visible.
Containers
Remelio's container mod. This removes all the space between container slots, making it more difficult to accidentally move the container around. Large done button at the bottom, with the combine button appearing over the bag icon. You can move bags around by clicking between the minimize and close buttons and dragging.
Spell Book
Is once again thanks to t.king and Remelio. More spellbooks are available on t.king's page if you prefer one of his other designs.
