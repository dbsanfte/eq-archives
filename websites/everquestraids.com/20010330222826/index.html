EverQuestRaids.com News
There are 6 news items posted within the last 7 days.
Features Added: Delete Character and Delete Guild 3/26/01 5:59:01 PM
Users now have the option to delete their character on the Edit Profile page or delete their guild (if they are the leader) from the Edit Guild Profile page.
-Rordin Firewalker
Bugfix: Some guild-locked events not showing as locked 3/26/01 4:59:29 PM
Thanks to Xandir of Bristlebane for pointing out that some guild-only events were not showing up as locked on the Event List page but were locked on the Event Details and Event Calendar pages.
-Rordin Firewalker
Bugfix: No link from Event Calendar even if event host 3/26/01 10:45:40 AM
Thanks to Azlinon Seflarian of Terris-Thule for pointing out that even if you are the host of a guild-locked event the Event Calendar was showing a * instead of being a link. This bug was due to type-casting of variables to type varient instead of integer. The problem has been resolved.
-Rordin Firewalker
Guild database bugs 3/26/01 10:27:54 AM
Thanks to Namcidem of Erollisi Marr for pointing out a problem with invalid guild data. I'm in the process of cleaning up guilds now. If some of you have been removed from your guild, that's because the guild had no valid guild leader. This happened mostly because the guild leader did not log in in 3 months and was automatically deleted from the database. The guilds should have been deleted as well and all other characters removed from those guilds but that didn't happen as expected. As a result we got a lot of bad data which caused other errors.

I'm in the process of writing code to prevent these types of problems again.
-Rordin Firewalker
Bugfix: Search Profiles page 3/26/01 8:39:22 AM
Thanks to Nardok of Tunare for pointing out that the search profiles does not return character profiles who are not in a guild. I've fixed this bug.
-Rordin Firewalker
Feature Added: EQMaps links added to Event Details page 3/26/01 1:53:00 AM
I have added a link on the Event Details page which will take you to EQMaps.com website's zone details page for that zone. This should help many people who are first-time in the zone to get familiar with the zone or print out the maps for that zone before going on the event.
-Rordin Firewalker
Acomsolutions DNS table crashed 3/25/01 8:00:10 AM
As most of you know I pay Acom Solutions to host EverQuestRaids.com. Well tonight at around 12am PST their DNS table got messed up. This caused EverQuestRaids.com to not resolve to any actual website. Instead it gave a DNS lookup error for some people or no page at all for others. Despite my attempts to email them, I got no replies. Well after around 4 hours of being unavailable it's back up now. It would seem they fixed the problem. They haven't replied to my emails yet however.

I appologize for any inconvenience this caused any of you. I am indeed sorry the site was down for so long. Please accept my appology.
-Rordin Firewalker
