GUIDE DE DÉPANNAGE  
Version 1.1

Ce document contient des informations permettant de résoudre certains problèmes que vous risquez de rencontrer en jouant à Everquest.

Note : Ce document évoluera fréquemment au fur et à mesure que des solutions aux différents problèmes seront découvertes. Consultez-le régulièrement si vous rencontrez des problèmes, il sera mis à jour aussi souvent que possible.

Sections

III. Problèmes techniques généraux
IV. Problèmes de carte vidéo

I. Configuration requise
Windows® 95, Windows® 98 ou Windows® 2000

Pentium 166 (ou plus) (PII 200+ recommandé)
(Note : les anciens processeurs Cyrix ne SONT PAS COMPATIBLES avec Everquest.)

Pentium 200 (Pentium II - 266 Mhz et plus recommandé)

64Mo de RAM (128Mo recommandés)

8Mo de RAM pour la carte vidéo

Carte son compatible Direct X

X2 (X8 et supérieur recommandé), avec des drivers Windows compatibles 32 Bits

Direct X 7.0 (inclus sur le CD-ROM d'installations)

Connexion internet indispensable

600 Mo d'espace libre sur le disque dur

II. Problèmes de compte

Problème : J'ai oublié mon nom d'utilisateur de Sony Station et/ou mon mot de passe. Que puis-je faire ?
Solution : Sony Station peut vous envoyer votre nom d'utilisateur ou votre mot de passe grâce à l'un des formulaires en ligne. Le lien permettant de retrouver votre nom d'utilisateur est : http://www.station.sony.com/services/forgottop.jhtml?forgot=station

Le lien pour les mots de passe est : http://www.station.sony.com/services/forgottop.jhtml?forgot=password Si vous n'arrivez pas à utiliser cette méthode ou si votre adresse e-mail a changé, vous devrez contacter le service consommateur de Station à l'adresse accounts@station.sony.com ou appeler le 1-801-579 (USA).

Problème : J'ai perdu la clé de mon compte ou elle ne marche pas. Que puis-je faire ?
Solution : Contactez le support technique de Sony Station au (858) 537-0898 (USA).

III. Problèmes techniques généraux

Si vous lancez Everquest pour la première fois et que vous rencontrez des problèmes de son et de vidéo, assurez-vous que vous utilisez les pilotes de périphériques les plus récents disponibles sur les sites web des fabricants. De plus, nous vous recommandons VIVEMENT de jeter un coup d'œil à la FAQ Cartes Vidéo. Assurez-vous également que DirectX 6.0 ou plus soit installé sur votre système. DirectX 6.0 est inclus sur le CD d'Everquest. DirectX 7.0 est disponible en téléchargement sur le site de Microsoft.

Problème : Lorsque j'essaie de me connecter au World Server ou en cours de partie, le jeu plante et je retourne sous Windows.
Solution : Vérifiez que vous disposez d'au moins 100 Mo d'espace libre sur le disque sur lequel vous avez installé Everquest. De plus, vous devrez laisser Windows gérer la mémoire virtuelle ET disposer d'au moins 500 Mo d'espace libre sur le disque sur lequel Windows est installé (généralement le lecteur C:).

Problème : Une boîte de dialogue Windows me signale une opération illégale lorsque je clique sur le bouton « Play Everquest » de l'écran de sélection du serveur.
Solution : Il n'y a pas grand chose à faire si vous recevez ce message. Il est probable que vous utilisez un ancien processeur Cyrix. Ces processeurs ne sont pas compatibles avec Everquest.

Problème : Je n'arrive pas à télécharger le patch, il plante sur le fichier
« eqgame.ex_ ».
Solution : Il est fort probable que votre système soit infecté par le virus CIH. Vous devez impérativement vous débarrasser de ce virus pour de nombreuses raisons (Everquest en est une très bonne). Nous vous recommandons d'aller sur le site http://www.avp.com et de télécharger Anti Viral Toolkit Pro. Ce logiciel nettoiera votre système. Veuillez noter que ce virus NE PROVIENT PAS d'Everquest. Vous l'avez probablement récupéré en téléchargeant des programmes sur le Net. Problème : Pendant l'exécution du patch, un message me signale une "DNS Error". Solution : Ceci signifie que vous n'êtes pas connecté à Internet. Vérifiez vos paramètres de connexion et réessayez.

IV. Problèmes de carte vidéo

Problème : Quand j'essaie de lancer Everquest, le jeu plante pendant le chargement et un message d'erreur apparaît : "VXD error, expecting 1.4, found 1.X", ou une fenêtre avec le mot "Error" apparaît.
Solution : Suivez les instructions qui suivent : cliquez sur le bouton "Démarrer" dans la barre des tâches de Windows. Cliquez ensuite sur "Rechercher", puis choisissez "Fichiers ou dossiers…". Tapez "Glide*.*" dans le champ "Nommé" puis cliquez sur le bouton "Rechercher maintenant". Une fois la recherche terminée, supprimez tous les fichiers "Glide*.*" de la liste. Si vous utilisez une carte à base de chipset 3dfx, vous devrez installer les pilotes les plus récents disponibles sur le site du fabricant. Si votre carte n'est pas à base de chipset 3dfx, ce n'est pas la peine de réinstaller les pilotes de votre carte vidéo.

Problème : Lorsque j'exécute Everquest sur une Voodoo 1 en mode Glide, les icônes des objets, les éléments en surimpression et les dialogues sont entourés d'une bordure rose.
Solution : Ce problème a été résolu dans la dernière version des pilotes 3dfx. Vous pouvez visiter le site http://www.3dfx.com pour récupérer ces pilotes, ou lire la section FAQ Cartes Vidéo pour obtenir un lien direct vers ces pilotes.

Problème : Lorsque j'exécute Everquest avec une carte à base d'ATI Rage 2, Rage 2+, ou Rage Pro, les icônes des objets, les éléments en surimpression et les dialogues sont entourés d'une bordure rose.
Solution : Ceci signifie que vous n'avez pas installé les pilotes les plus récents. Consultez la section FAQ Cartes Vidéo. Notez également que les chipsets de la série Rage 2 ne sont pas officiellement pris en charge par Everquest.

Problème : Avec une carte à base de 3dfx Banshee, je ne peux pas lancer Everquest en mode Glide.
Solution : Pour le moment, le mode Glide d'Everquest n'est pas compatible avec le chipset Banshee, il ne vous sera donc même pas proposé. La raison de cette incompatibilité est un problème connu de Glide 3 sur lequel 3dfx travaille actuellement. Bien que de nouveaux pilotes censés résoudre le problème viennent de sortir, nous vous recommandons d'éviter l'utilisation du mode Glide avec un chipset Banshee.

Problème : Avec une ATI Rage Fury, de nombreux problèmes graphiques surviennent.
Solution : Consultez la section FAQ Cartes Vidéo et téléchargez les pilotes les plus récents.

Problème : Le jeu se bloque après avoir cliqué sur « Play Everquest » dans l'écran de sélection du serveur.
Solution : De nombreuses erreurs surviennent à ce moment, mais la plupart sont dues à des pilotes de périphériques dépassés. La plupart des cartes basées sur le Riva TNT plantent à ce moment si vous ne disposez pas des pilotes les plus récents. Assurez-vous de disposer des pilotes que nous avons testés avec Everquest en consultant la section FAQ Cartes Vidéo.

Problème : Avec une Voodoo2, le jeu plante quand je change de résolution.
Solution : Il est probable que vous ne disposez pas des derniers pilotes. Les derniers pilotes pour Voodoo 2 de 3Dfx résoudront ce problème.

Problème : Everquest est très sombre et les réglages de gamma n'ont pas l'air de fonctionner.
Solution : La correction gamma ne fonctionne pour l'instant qu'en mode Glide. Si vous utilisez D3D, utilisez le centre de contrôle de votre carte vidéo pour rendre le jeu plus clair.

Problème : Everquest est vraiment très lent sur ma machine.
Solution : Everquest est une application très complexe. Nous vous recommandons en premier lieu d'augmenter votre quantité de RAM. Assurez-vous ensuite de posséder une carte 3D récente : une Voodoo 2, une Riva TNT ou mieux. Désactivez les effets de particules (dans le menu Display Options du jeu). Diminuez également la valeur du paramètre Sound Realism afin d'augmenter la vitesse d'affichage.

V. Problèmes de son

Problème : Je n'entends pas la musique du jeu.
Solution : Everquest est uniquement compatible avec les cartes son capables d'utiliser la technologie Sound Font. Ces cartes sont la Sound Blaster Awe 32, la Sound Blaster Awe 64, la Sound Blaster 128 PCI et la SB Live.

Problème : Je n'entends aucun son pendant le jeu.
Solution : Assurez-vous de disposer les derniers pilotes pour votre carte son. Pour obtenir ceux que nous avons testé avec Everquest, consultez la section FAQ Cartes son.

Problème : Je n'ai pas de musique avec ma SB Live.
Solution : Allez dans le panneau de contrôle Sound Fonts et augmentez la taille du cache Sound Fonts afin qu'il fasse 3 Mo ou plus.

VI. Problèmes de connexion

Si vous utilisez un modem, vérifiez dans Panneau de contrôle -> Modems -> Propriétés -> Connexion -> Avancé, que le Contrôle de flux est réglé sur Contrôle matériel.

Problème : Je n'arrive pas à rester connecté plus de 15 minutes. L'écran "You have been disconnected" est ensuite affiché.
Solution : C'est malheureusement un symptôme d'une mauvaise connexion au jeu. Si cela se reproduit fréquemment, contactez votre fournisseur d'accès. On vous dira généralement que "tout marche très bien"…ce qui est faux dans la plupart des cas. Considérez l'éventualité de changer de fournisseur d'accès.

Problème : Lorsque j'essaie de lancer "Everquest.exe", une erreur survient : "ERROR Host not found, no DNS entry for host"
Solution : Vérifiez que vous êtes connecté à votre fournisseur d'accès Internet. Si vous êtes effectivement "on-line" et que vous continuez à recevoir ce message, contacter votre fournisseur d'accès. Les utilisateurs d'AOL doivent également vérifier qu'ils utilisent le Master Login et que tous les paramètres de contrôle parental sont désactivés.

Problème : A chaque fois que j'envoie un "Ping" ou un "Tracert" aux serveurs d'Everquest, je reçois le message "Request Timed Out".
Solution : Nous avons bloqué le trafic ICMP auprès de notre fournisseur d'accès et vous ne pourrez donc pas nous envoyer de Ping ou de Traceroute. Vous pouvez faire un Traceroute jusqu'à la dernière jonction vers nos serveurs afin de détecter d'éventuels goulots d'étranglement. Ceci devrait de toute façon suffire pour détecter la plupart des problèmes.

Problème : Mon réseau utilise un logiciel de Firewall/Proxy, quels "ports" doivent être ouverts par mon administrateur réseau ?
Solution : Donnez les informations suivantes à votre administrateur réseau : Application de patch :
Une connexion TCP est initialisée sur le PC du joueur (port arbitraire>=1024) vers patch.Everquest.com, port 7000.
Client Everquest :
Des datagrammes UDP sont envoyés vers/depuis le PC du joueur (port arbitraire>=1024) depuis/vers chat.Everquest.com, port 5999.
Des datagrammes UDP sont envoyés vers/depuis le PC du joueur (port arbitraire>=1024) depuis/vers l'un des serveurs d'Everquest (port arbitraire>=1024). Les adresses IP du serveur Everquest sont sur les sous-réseaux : 206.19.150.*, 208.236.12.* , 63.65.235.* , 192.215.54.*, 192.215.37.* et 192.215.33.* Veuillez noter qu'Everquest ouvre un port de données aléatoire sur le PC du joueur à chaque fois qu'Everquest est lancé. Il est donc nécessaire que tous les ports >= 1024 soient disponibles.
