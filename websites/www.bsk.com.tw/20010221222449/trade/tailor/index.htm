裁縫(Tailor) 補充說明

這個技能需要花上相當多的時間(打獵), 後期則是錢. 有謠言說帶相關的書在身上可以增加成功率...也許只是謠言...... 如果真的怕麻煩, 可以和朋友分開練鐵匠和釀造

在一開始建議你最少在公會導師身上砸10點來練習, 如此可以減少失敗率,也節省時間, 先去買一個sewing kit 接下來在技能10點左右, 你大概只有一樣東西會成功, 把殺掉蜘蛛得到的2個蜘蛛絲(spider silks) 注意 spider silk 和 spiderling silk 不同 放進sewing kit中,再按下Combine, 成功的話會出現(swatch), 記得先留下來 一直練到差不多快20點時, 你就可以開始作Patchwork(簡稱PW)

PatchWork
製作PW還需要各部位的patterm, 在大部分的裁縫店都可以買到 為了練點數, 大部份玩家都是帶著Sewing Kit和護喉(gorget)的patterm去打獵, 因為做出來的護喉最輕
PW所需的毛皮是Ruined類毛皮
Wolf類-小型PW----小型身材種族可穿
Cat類-中型PW----中等身材種族
Bear類-大型PW----大型身材

再繼續練到快30左右 可以開始作Raw Silk 此時你需要買Large sewing kit
Raw Silk Armor 也很簡單, 將蜘蛛絲合成的Swatch 和各部位的pattern放進再結合即可. 不過作pants和Tunics則需要兩個 Swatch

Studded Armor:
要製作 studded 防具 首先請將 raw silk 練到熟練 , 接下來你必須要練習鐵匠技能 將 1 water + 2 ores (ores 必須分開) 放進熔爐內, 再按下 combine 成功會得到 metal bits (鐵匠技能練到21時才算熟練), 真的要練就做它2,3百個 (用來練studs....非常非常累....) 另外再做幾個 files ( 可重覆使用 ) 接下來將 1 water + 3 metal bits (3個一樣必須分開放) + 1 file 放進熔爐內, 再按下 combine 成功會得到 studs (鐵匠技能練到4,50左右才算熟練) studs 就是你所須要的.
接下來去買 large sewing kit 並四處收購或自己去打一堆 Medium Quality 的 Pelt 將 studs + medium pelt + pattern 放進並 combine即可做出 studded armor 但注意每個部位所需的 studs 數量不同, Mask 只需要一個,其它都在2個以上 所以如果是純練技能,只做 Mask 就好

注意不同種族要不同動物的皮
Barbarian, Ogre, Troll Large size : Bear Pelt
Barbarian, Elf, Erudite, Half-Elf, Human Medium size : Cat Pelt
Dwarf, Elf, Halfling, Gnome Small size : Wolf Pelt

Studded Boots = Medium Quality Pelt + 4 studs + Pattern
Studded Bracer = Medium Quality Pelt + 2 studs + Pattern
Studded Cap = Medium Quality Pelt + 3 studs + Pattern
Studded Gloves = Medium Quality Pelt + 4 studs + Pattern
Studded Gorget = Medium Quality Pelt + 2 studs + Pattern
Studded Mask = Medium Quality Pelt + 1 stud + Pattern
Studded Pants = Medium Quality Pelt + 4 studs + Pattern
Studded Sleeves = Medium Quality Pelt + 3 studs + Pattern

將 studded 練到熟練之後(約 6,70 吧) 再開始做 cured silk

Cured Silk Armor:
做cured silk 需要先練習釀造 (Brewing) 技能, 將 2 kiola saps + 1 bottle + 1 flask of water 丟進釀造桶內, 可做出 Heady Kiola (釀造技能必須要到4,50左右才能熟練的做出) kiola 在很多地方都可以買到,像 Freeport 或 Neriake !!

關於釀造,如果你只想練到能做出 Heady Kiola
1).將 1 snake skin(殺蛇後loot到的,買比較快) + 1 water + 1 bottle 丟進釀造桶內可做出蛇酒 (Bog Juice), 失敗會剩下bottle, 除了 bottle 外各買50個左右可讓你技能升到21
2).將 1 fish(一樣用買的較快) + 1 water + 1 grapes + 1 bottle 丟進釀造桶內 可做出魚酒 (Fish Wine) (bottle買100個,其它各買200個左右可讓你技能升到60,不過升到40左右也就夠了) 殺它個幾百隻 giant spider, 將 spider silk 組合成 swatches 將 silk swatches + Heady Kiola + pattern 放進 large sewing kit 內即可做出 cured silk 防具 同樣,每個部位所需的 swatches 和 Kiola 數量不同 純練技能就做 Mask, 因為只需要各一個 當你能熟練做出各種cured silk時,就是你回本的時候了

Cured Silk Cap = 1 silk swatch + 3 Heady Kiolas + Pattern
Cured Silk Cloak = 2 Silk swatches + 2 Heady Kiolas [brewing skill] + Pattern Cures Silk Gi(Robe) = 3 silk swatches + 2 Heady kiolas + Pattern
Cured Silk Gloves = 2 silk swatches + 2 kiola + Pattern
Cured Silk Gorget = 1 Silk swatch + 2 Heady Kiolas + Pattern
Cured Silk Mask = 1 Silk swatch + 1 Heady Kiola + Pattern
Cured Silk Pants = 2 silk swatches + 3 Heady kiola + Pattern
Cured Silk Mantle = 2 Silk swatches + 2 Heady Kiolas + Pattern
Cured Silk Sandals = 1 silk swatches + 2 Heady kiola + Pattern
Cured Silk Sash = 1 Silk swatch + 2 Heady Kiolas + Pattern
Cured Silk Sleeves = 2 Silk swatches + 2 Heady Kiolas + Pattern
Cured Silk Wristbands = 1 Silk swatch + 2 Heady Kiolas + Pattern

Cured Silk到熟練後, 接下來則是最後的 Reinforced Armor 系列和像10 slots的背包,Monk專用的魔法手套(市價各約50pp)等
