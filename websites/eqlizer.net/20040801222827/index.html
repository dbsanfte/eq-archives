2004-07-31 Posted by : J.C. Smith
First Loots From Tacvi, Spell Database Functional!
We have a very special update for you today. This update features the first loots from Tacvi, the GoD end zone! It also includes a few new items from Txevu's boss.

But wait, there's more... Matt has our Spells Database in to a functional format now. The spell data is pulled directly from the game, and allows for advanced searches, or quick class searches. The full details page allows for comments, and contains information on items which have this effect. We'll have the scroll locations listing on that page very shortly, which should help explain all of the Spells we entered last week. I'm still keying in spell locations for many of the older spells, but GoD, PoP, LoY and LDON should be fully documented. If you know a spell location that we don't have listed, feel free to leave a comment and we'll get that information in there ASAP. As always, you can send any suggestions, questions or comments to us by clicking here.
