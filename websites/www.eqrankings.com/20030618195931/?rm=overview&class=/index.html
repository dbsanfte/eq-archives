17.02.03: Guilds and stuff..

17.02.03: How to remove your magelo profile from the list
If you have an old duplicate profile or for some other reason don't want your magelo profile on these lists,
you can add <!--nolist--> to your magelo bio, then add/update it once and it we be removed.

11.02.03: Counter
Counter is back again, in a new version.
It wount show number of views in the profile, but hits will be counted and displayed on this list soon.
Just insert the following code in your bio:
<IMG src="http://www.ascendingdawn.com/cgi-bin/magelocounter.pl">
Note: This will insert a 1x1 transparent gif, so you might want to put it at the bottom of your bio if you have any other stuff there.
