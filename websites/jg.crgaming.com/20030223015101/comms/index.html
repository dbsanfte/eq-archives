COMMUNICATION
| Receiving Messages | Sending Messages | Macro Messages | Commands | TRI Booths |

An integral part of playing JumpGate is communication. The standard TRI communication window will be located at the top of your screen in flight, and in the upper left-hand corner  while docked.
   

Receiving Messages

Each line of text will contain several different attributes which will help you to discern the purpose of each message. The first thing you will notice is the different colors of each message.
   
Grey Grey text is an echo of your communication inputs, if you are on a channel that does not already come back to you (such as a booth).  This is only broadcast to yourself.
Red Red text is the Octavian faction's private broadcast color. To see this text, you must be an Octavian. A message broadcasted in this mode can only be sent between fellow Octavian's in the same sector.
Green Green text is the Quantar faction's private broadcast color. To see this text, you must be a Quantar. A message broadcasted in this mode can only be sent between fellow Quantar's in the same sector.
Blue Blue text is the Solrain faction's private broadcast color. To see this text, you must be a Solrain. A message broadcasted in this mode can only be sent between fellow Solrain's in the same sector.
Aqua Aqua text is reserved for private squad broadcasts. To see this text, you must be a member of a squad. A message broadcasted in this mode can only come from a fellow squad member, but crosses sector boundaries across the entire galaxy.
Orange Orange is the general public sector communication color. A message received in this mode may be coming from any pilot in your current sector, regardless of race, religion, or sex.
White White text lines denote whisper mode communications. White messages can be received from any pilot anywhere in space. Whisper messages are sent to only one player at a time privately.
Lime Lime text is received from your currently selected booth. Booths are player defined, cross-sector channels, which may be occupied by any player. They are semi-private in the respect that you must know the name of the channel to join, and anyone who does join is announced (except for newbies in the help booth). Beware of spies and intelligence gatherers who may be snooping.
Purple Purple denotes an administrator or system broadcast. These broadcasts are seen by every active pilot in game, and usually contain important system information such as server status and imminent downtime.
Yellow Yellow text means a system message directed only at you. System messages consist of reports and warnings sent from the client or server depending on your current condition. Pay attention to these messages, as they often contain important directions or error warnings.  Honor Guards receive their Beacon reports in this color.

Macro Communications

Macros are programmable communication keys which can be used to store and send commonly used messages. Five macro keys are provided (F6 - F10).

Macros can only be set while docked in a station.

To record a macro, press Shift+F key (F6, F7, F8, F9, or F10). This is the F key that the macro will be recorded to. You are now in record mode. Now type which channel you want the message to be displayed in, dropping the F. For example, if you want it to play in F3, type '3'. Type your message exactly as you would if you were using the channel normally.

For example, a macro set to F6 that displays the message "Greetings Sector Pilots." in your faction channel (F1), would look like this:

1Greetings Sector Pilots.

Alternatively, a macro that would whisper the message "Hiya." to player Scorch would look like this:

4:Scorch Hiya.

To play a macro, simply press the key which it was mapped to. If you pressed Shift+F7, press F7. If you pressed Shift+F8, press F8, etc. 
   
Communication Line Commands

Certain commands to JOSSH can be sent via the communication line. The following chart gives a brief description of each communication line command.
/bounty Displays your maximum possible bounty (this value may vary depending on who is attacking you).
/cargo Displays your current cargo loadout.
/credits Displays your current amount of credits.
/forgive If a pilot has shot you and no one else, you can target them and type /forgive to eliminate their bounty. (will take up to 6 seconds to clear bounty.)
/give Transfers funds while in flight. Target a pilot, type '/give x' where 'x' is the number of credits you wish to transfer. You will not be asked to confirm, so be sure the amount is correct.
/mission Displays your current mission details.
/exp Displays your current amount of experience.
/ignore [callsign] Adds callsign to your local ignore.txt, preventing any further text messages from that pilot.
/q Quick logout. Counts as a death if you are not docked, i.e. in space.
/time Reports how long you have until your pilot account expires, in Days, Hours, Minutes, and Seconds.
/voice [callsign]
or
/hear [callsign]		 Removes callsign from your local ignore.txt to allow hearing from them again.
   
TRI Booth Channels
help The default channel all new recruits join automatically. This channel should be used for general help questions and responses.
new This is an alternate help channel, used when there is too much traffic on the default :help channel. This channel should be used for general help questions and responses.
newb This is an alternate help channel, used when there is too much traffic on the default :help channel. This channel should be used for general help questions and responses.
chat This is a general chat and conversations channel. This can be used for all forms of general discussion.
sol This is an alternate help channel, used when there is too much traffic on the default :help channel, or for help specific to the Solrain faction. This channel should be used for general help questions and responses.
oct This is an alternate help channel, used when there is too much traffic on the default :help channel, or for help specific to the Octavian faction. This channel should be used for general help questions and responses.
quan This is an alternate help channel, used when there is too much traffic on the default :help channel, or for help specific to the Quantar faction. This channel should be used for general help questions and responses.
flux This channel is used for intelligence on current Conlux infestation. Communication should be limited to requests for Conflux extermination, information on current Conflux whereabouts, general Conflux hunting strategy, or anything Conflux related.
jobs This channel is used for offering services and hiring assistance for all manner of odd jobs. Chat should cover any jobs being offered by individuals or squads, as well as responses to those offers. This is the channel to join if you are looking for work from other pilots, or if you need to hire another pilot to help you.
roid This channel is used for asteroid farming intelligence. Conversation should surround locations of valuable asteroids, mining tips, and anything asteroid related.
