Configuration minimum

OS: Win 98/ME/2000/XP

Processeur: 800 MHz Pentium III, AMD Athlon, ou équivalent

RAM: 128 MB ; pour la version XP : 256 MB obligatoire

Carte Video: 32MB VRAM compatible Direct X 8.1 avec hardware T&L

Carte son: compatible DX 8.1

Direct X version: 8.1 ou supérieur (Version 9 inclue dans le jeu)

CD-ROM: 16x ou supérieur (non recommandé avec les CD-RW)

Hard Drive: 2 GB minimum

Souris compatible Windows

Connection Internet pour jouer en mode multi-joueurs sur Ubi.com (minimum 56k modem)




Configuration recommandée

OS: Win 98/ME/2000/XP

Processeur: 1.4 GHz Pentium 4, AMD Athlon ou équivalent

RAM: 256 MB, pour la version XP: 256 MB obligatoire

Carte Video: compatible Direct X 8.1 avec hardware T&L 128 MB

Carte son: compatible DX 8.1

Direct X version: 8.1 ou supérieur (Version 9 inclue dans le jeu)

CD-ROM: 16x ou supérieur (non recommandé avec les CD-RW)

Hard Drive: 2 GB minimum

Souris compatible Windows

Connection Internet pour jouer en mode multi-joueurs sur Ubi.com (Haut débit)
