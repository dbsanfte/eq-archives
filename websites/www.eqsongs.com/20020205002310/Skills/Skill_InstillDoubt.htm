MELEE COMBAT SKILL - Instill Doubt

Available at Level: 26

Effect: Single target fear.

Maximum skill: 100

Quantified: N/A

Analysis by Rokenn Swiftsong, Ayonae Ro server.

Comments: This skill is also known as Intimidation (this is what you see when getting a skill increase).  This skill is very slow and difficult to raise.  There is only one trainer in the game that will train this skill for bards.  He is located in the Freeport Bard guild on the second floor.

Strategies:

This skill is useful when using the Fear/Chain method of attack.  The fear effect from this skill seems to have a longer duration then Angstlich's Appalling Screech.
The skill goes up much faster after skill level 50, so if you want to avoid frustration, you may want to drop in enough skill points to get to that level.
You can practice this skill on anything you can target, party members, corpses, pets, other players, NPCs, boats, canoes, etc..  Do not worry about range you will still get a skill check even if your target is on the other side of the zone.
When the fear spell from Instill Doubt casts runs out you will get a message saying that is ended.

Cautions:

DO NOT USE THIS IN DUNGEONS.  Unless of course you love to start trains :)  The feared target will bring back all the friends it can find to show them the scary bard.
The fear effect from ID does not stack with Screech.  If you successfully ID while the target is feared you will still do the token kick damage, but the fear effect will not stick. But if you have ID'd a target then Screech the effects will stack
