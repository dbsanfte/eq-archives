Discipline: Shadow Knight



Discipline を使うには、拡張パック Scars of Velious が必要です。

/discipline と打つと使える Discipline と再び使えるまでの時間が表示されます。

レベルが上がるにつれて Charge Time は短くなっていきます。

なお、discipline コマンドは /disc と略せます。



Lv.51
Name RESISTANT
Command /disc resistant
Effect すべての耐性を10上げる
Duration 5min
Charge Time 60min （51min at Lv.60）
Lv.54
Name FEARLESS
Command /disc fearless
Effect Fearを100％防ぐ
Duration 11sec
Charge Time 60min （54min at Lv.60）
Lv.55
Name UNHOLY AURA
Command /disc unholyaura
Effect Harm Touchのダメージを1.5倍、Disease属性でLife Tapと同じResist率にする
Duration Instant
Charge Time 72min （67min at Lv.60）
Lv.60
Name LEECHCURSE
Command /disc leechcurse
Effect 与えたダメージの分だけHPを吸収する
Duration 20sec
Charge Time 72min （72min at Lv.60）
