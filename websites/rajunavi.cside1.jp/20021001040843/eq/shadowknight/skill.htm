Bash Level: 6
Skill Max: (Level + 1) x 5
Skill Cap: 175 (Lv.34) / 200 (Lv.59)
Shield を装備している時、その持っている Shield で攻撃します。
主に Stun 効果を得るために用いられ、若干の Knockback もあります。
大型種族（TRL/OGR）が使う Slam は Bash と同じ扱いで、
Slam を使うことで Bash のスキルが上昇していきます。
相手が MAGIC ITEM しか受けつけない場合、
Shield も MAGIC でないと Bash は当たりません。
Slam の場合は Shoulder が MAGIC かどうかになります。
スキルが高いほど命中率・ダメージが上がります。

Riposte Level: 30
Skill Max: (Level + 1) x 5
Skill Cap: 175 (Lv.34) / 200 (Lv.59)
相手の攻撃をかわしつつ反撃します。自動的に発動し、
成功すればダメージをまったく受けず、さらに反撃の一撃を加えることができます。
スキルが高いほど発動率が上がり、回避率・反撃の可能性が上昇します。
反撃した時の命中率は、あくまで使っている武器のスキルに依存します。
なお、後ろからの攻撃に対しては発動しません。
