An Official RP server is needed in EQ. Making an official server in no way deminishes the RPing and hard work that others have done on other servers but it does give us an Official home as opposed to us hopping from one server to the other and trying to claim it for RP. It also allows new players to the game to know exactly where they can go in order to find RP.

When creating the next unofficial server the word never gets to all the RPers and as a result we've ended up with several unofficial RP servers now. The RP community ends up being spread out into pockets on all these servers with powergamers and d00ds always being the majority.

We're not advocating anything but a server designated as for role-playing by Verant.

Please read the threads and lend your support. With a truly united RP community Verant will need to listen and for once we will not be ignored or brushed aside.
