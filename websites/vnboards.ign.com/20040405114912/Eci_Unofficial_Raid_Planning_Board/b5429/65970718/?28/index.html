Making a few updates, including the arrows note and Gate potions note.

I am not including Invisibility note, because the ability to invis is so common. A raid cannot succeed without enough diversity to be able to cast this on every group member. The only ones that cannot cast it are rogues (who can sneak anyway), warriors, monks, beastlords, clerics, and paladins. Would any single group REALLY be composed only of these classes? If so, is that a mistake on the raid leader's part?

EDIT: Oh, I also made it so the links on that page stayed the same color. That might help in readability.
