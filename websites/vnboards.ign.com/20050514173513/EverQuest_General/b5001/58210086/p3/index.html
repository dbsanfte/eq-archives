Date Posted: 12/2/03 11:47pm Subject: RE: The only MMORPGS see in the NEAR future to give EQ a run for it's money is....
A factor not mentioned is what approach SoE will take. EQ is an old, stable and reliable cash cow. In the short term SoE will want that to continue. These days it is also a very slick game in many, many ways. The impact of Everquest Evolutions can only be beneficial as well by the way attracting older players back to the fold and being a 'good deal' for new players. Graphically the old zones would probably have to be redrawn from scratch to utilise the new, more powerful engines.

EQ2 will be new. It will take advantage of the latest graphics engines, be able to run on newer servers, the coding language may be more up to date (more object orientated maybe, more visual perhaps). It will also be fresh start - for better or for wose - from a game mechanics point of view. As far as storyline goes of course it won't be new new - having the advantages that EQ1 brings to it.

So why would SoE invest in a 'new' game? I suspect its a trade off. It wants to avoid existing players who have 'seen & done' all of EQ1 from gradually drifting away to newer games. Like an earlier poster said if 10 games each take 3% that represents a big chunk. And if the player base shrinks more people may leave etc. I suspect that there will also be current cost benefits as far as code development goes, what type of servers the game can be run on etc - and possibly some of the existing servers may need replacing anyway. It is also a good time to strike back at some of the 'second generation' games like DAoC (which is currently in decline with huge problems with its latest add-on)

In the long term SoE would probably like everyone playing EQ2. One game means one set of code to maintain, one set of servers etc., one set of players to provide that critical mass that sheer size brings etc Yhat said i could see EQ1 running for years and years so it is possible that they will simply allow EQ2 to grow without poaching (can't see it though and I think we are already starting to see a strategy aimed at getting people to move.)

So what's the medium term strategy. some months ago I suggested a large beta test. Well the beta test is going to be big. I suspect that it is also going to last a while. There could even be an alpha test - which would be unusual in software terms but not completely unheard of. And if there was a formal delay to evaluate and fix things found in th ebeta this would a) allow word to spread about how good EQ2 was (an assumption I know) and b) suggest that they want EQ2 to be bug free from day 1.

How do they get from EQ1 to EQ2. Shrug. It may just be a case of giving people a discount or even the option to move. Maybe at some stage there will be an EQ1 expansion pack in which temporal statsis spells can be discovered - or a cryogenic chambers blown backward in time.

Time will tell
