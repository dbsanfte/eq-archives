Isn't going to happen. Wine is not an emulator of x86 processors. In fact, the name Wine stands for "Wine Is Not an Emulator" (self-referential, but that's what you get with programmers). It supports the windows API's, so that x86 programs that run in windows might run in linux.

You'd need an x86 emulator for it to work on a G4. I think such exists, but have no experience with them.
