NM found it.

1. Quit out of EQ entirely.
2. Open up Windows Explorer
3. Find your eqclient.ini file (usually in c:\program files\sony\everquest)
4. Edit that file with notepad
5. Find the entry for windowed mode
6. Change that entry to FALSE
7. Save the changes to the ini file
8. Restart EQ
