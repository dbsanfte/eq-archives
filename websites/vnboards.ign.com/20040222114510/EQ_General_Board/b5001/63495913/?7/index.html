Is anyone really even interested in this class? It sounds sort of like a gimpy monk paired with the party buffing of a very sub-standard bard. If there is anything EQ classes have in abundance it's those whose sole purpose really is adding damage per second, I just don't see any need or anything very vital this class would bring to a group.

I know they scrapped that "opening" new combat style system (whose sole purpose was to make the mana-less melee classes more interesting), but it was my understanding there is some more basic replacement in the works, I thought originally the beserker was supposed to be the combat "style" master class which was the whole point of adding the class but without the style system I fail to see why they are going ahead with beserkers.

If there are any new classes or races to be added to EQ I'd like them to bring some totally new ability or gameplay mechanic to the game rather then be watered down merged versions of what is already available.
