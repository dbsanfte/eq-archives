Firiona Vie is the official "Roleplaying Preferred" server, and they have a few different rules than the regular servers.

From http://eqlive.station.sony.com/library/sz_rule_sets.jsp

Firiona Vie (A roleplaying preferred server)

The Firiona Vie server is a roleplaying preferred server. To motivate players to take advantage of the roleplaying atmosphere, the following feature list was implemented.

This is a PvE (Player vs. Environment) server, NOT a PvP (Player vs. Player) server. In other words, it's a standard "blue" server with the same rules and features regarding PvP as the vast majority of our servers. it is not one of the "Zeks".

Beneficial buff spells will not work on characters 20 levels lower than the caster.
The Common language will not exist. A Human language will be added and only made easily available to certain races.
Only one character can be created on each account.
The Trivial Loot Code will be in effect in all zones.
Very few items will be No-Drop. Exceptions will include things such as newbie notes and epic weapons.
Bind Affinity will be restricted for all characters to locations where melee characters can bind on other servers.

Trivial Loot Code

Briefly, the Trivial Loot Code works like this: If a characters kills an NPC and that NPC does not give them experience, then the Trivial Loot Code takes effect. This can happen when either a member of your group is too high in level to receive experience from killing the monster, or when a member of your group is outside of the level range of the group for receiving experience from the kill. If any member of the group does not get experience for a kill, the Trivial Loot Code will take effect. The Trivial Loot Code prevents magical, lore, and no-dorp items from appearing on the corpse. Normal items and coins will appear on the corpse in all cases as usual.

The main Firiona Vie community can be found at http://pub94.ezboard.com/bfirionavie - they're a great source of information and support.

If you're looking for a regular rule-set server that is more into RP, there are a couple that seem to have a bit more than most, but FV has the best that I've seen so far.
