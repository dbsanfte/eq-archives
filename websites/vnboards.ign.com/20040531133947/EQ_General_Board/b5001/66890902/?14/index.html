a temp fix that someone posted:

goto your eq directory and locate eqhost.txt open it with notepad and change the information to look like this

Code:

[Registration Servers]
{
"eqlogin1.989studios.com:5999"
}
[Login Servers]
{
"eqlogin3.989studios.com:15902"
"eqlogin3.989studios.com:15901"
"eqlogin3.989studios.com:15903"
"eqlogin3.989studios.com:15900"
}



Now goto start>run and type in cmd or command to bring up a command prompt.

direct it to your eq directory
something like
cd c:/Program Files/Sony/Everquest

once in the EQ main directory type this

eqgame.exe patchme

and press enter

This will bypass the patch server and log you directly to the login server using a different log in address. I got in instanly after this fix.

FYI this is the same way you would log into an EQEMU server.
