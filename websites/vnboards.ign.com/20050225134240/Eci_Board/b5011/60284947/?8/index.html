With NVidia it is always a good idea to set your video to standard vga, shut down and restart before attempting to install a new driver. On 2 computers I have owned going to standard vga mode before installing new drivers was the only way I could update drivers.

Look for Detonator RIP program or Detonator Destroyer, these are programs that help clean up stray registry entries that nvidia drivers can leave behind even when you update drivers. I havent dealt with an NVidia driver update in over a year so I am not sure if these programs are still available and if they are upto date for the new NVidia drivers. www.guru3d.com is where I used to go for them.

My suggestion is...

1. Uninstall the drivers from the Add/Remove Section of Control Panel and shut down.
2.When the machine has restarted, Install the display adapter as "Standard VGA"
3. Restart when prompted
4. Install the new display drivers
5. Restart again

If that doesnt work then look for an NVidia clean up tool.

I finally managed to nuke the drivers in their entirety. As a general note you can't force install "Standard VGA" drivers in an xp system if there is a compatible version of the vid cards driver installed. The detonator destroyer will remove the drivers that are in the windows folder but won't clean the entire hard drive of them, which is what I eventually had to do.

I went through the registry and there were several keys I missed in the first pass, that seems to have done the trick. I then downloaded version 45.23 of the video drivers and that seems to be stable. I haven't crashed yet with those drivers.
