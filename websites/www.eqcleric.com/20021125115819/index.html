Welcome to the Typo3

If you see this message it appears that you succeeded in installing Typo3. At least some of it. You should browse these pages, read them and check various things in order to verify the install.


FIRST TEST: Does the menu show?


To the left you should see a menu with three items exactly like the image to the right shows.

- Do you see this menu? If you do, this means that GDlib and FreeType works in your installation.

 

Now move your mouse over the menu.

- Does the items shift to white text when you roll the mouse pointer over them? If they do it's even better, because this means that ImageMagick is properly installed. If they do not (appear blank - which is a very common error) ImageMagick hasn't worked well. Please check the Install FAQ on the Typo3 website.

 

? Why would there be a difference?

! Because Typo3 features an option, "niceText" when rendering text on images. This is a hack made to simulate better antialiased text than the freetype library has been able to produce. The trick is to generate the text in twice the size as a black/white mask which is then scaled down to 50% and finally used to overlay the text. "niceText=0" needs only GD/Freetype (is set for the regular menuitems), "niceText=1" utilizes ImageMagick (is set for the rollover-items).

The example also shows how much you gain by using the "niceText" feature, doesn't it?
