Download the file somewhere like your desktop where it's easy to find.

Go to My Computer and browse to your Everquest folder. Go in and open the UI folder. Make a NEW folder called EnjuMods or something like that.

Go back to the downloaded file and unzip it into your new EnjuMods folder.

Log in.

Type /loadskin EnjuMods 1
Your screen will freeze up for about 5-10 seconds. Then the new stuff should be in. (I guess if it's character select, you'll need to go back to the character select screen to see if it worked.)

Lucretia
Don't copy the default folder into a new folder. It's unnecessary. Create a new folder in the same directory as your default folder (so if you click on the directory you see default and your new folder name under it). The UI will pull any files you don't have in your current UI directory from the default folder.

What does it matter, you ask? When SoE patches the ui, the files are downloaded into your default directory. They are not however, downloaded into any of your custom directories. So whenever the UI is patched and you have files copied into your custom directory, those files will be broken. You can copy them over, but it's easier just to leave them out of your new directory.

Hope that's not too confusing!
