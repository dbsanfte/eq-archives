Insurance
Alas, I have never purchased a 'mount' for any of my characters. To be honest, I don't see them used much by clerics in raid situations. It is very cumbersome to try to 'duck' a cast in a CH rotation while mounted. Most critical rotations won't give you enough time to make it advantageous. However, for exp work without a bard or enchanter it would really come in handy. If you're doing a lot of this lately then I would suggest you consider this. Personally, I'll spend my cash on gear before I'll purchase a mount. I'll save up the cash later for the best one I can buy afterwards.

I believe the bridle quest in Befallen is still active if you'd like to try out the effect just to see if you would like it. It's a horsey though. Single use, no rent, lore bridle that pops you a horsey to play with. Allakhazam's should still have it listed somewhere if you'd like a walkthrough.
