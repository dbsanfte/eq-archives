This is very easy to fix. It's related to the cached textures in your EverQuest folder. Most likely you have texture caching turned on and tons of megabytes in so-called TXC-files.

Enter your EverQuest folder and find every file with the .TXC extension. Easiest way is to open Windows Explorer (WindowsFlag - E), browse to the folder where Everquest is installed, and click on the "Type" field above the file listing to sort files by type.

Make sure you're viewing files as "Details". (On the View menu select "Details").

Now go ahead and delete all those files. (Do it quickly using Shift-Del combo on your keyboard).

Now textures should be fine. You'll likely notice a slightly longer delay when going to a zone for the first time again, as it's re-caching the textures, but after that it's all good :)
