Pomaikai
I've tried... The only voice recognition software that EQ supports is Dragon Naturally Speaking 4.0. No, it does NOT work with any later versions. No, it does not work with any OTHER versions. Version 4.0 is it. It does not work with ANY other voice recognition software. Period.

There is one other alternative. I have been really interested in finding one of these things... It's called Voice Commander. This device does have EQ templates, and does supposedly work with EQ. Pretty nice just to be able to say "cast 1" to cast the spell in your first spell gem. If anyone uses Voice Commander please give us a review. I'm interested.

But for the most part, voice recognition was only implemented in a very half assed way early on in EQ's development. It was never finished, and never ever worked on for compatibility with multiple software solutions, nor kept up-to-date.
