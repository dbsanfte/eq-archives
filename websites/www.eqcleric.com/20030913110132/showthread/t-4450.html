The Spell System is now LIVE! Yeah, that's right...it's working! John's (Sepodati) been doing some serious work on the code, and it's only been getting better and better. We decided to open it up for you guys to try so we can get some feedback about functionality and the sorts.



Hope you like it. Of course it's not completely done yet, there are some features and bonus candy we haven't completed yet, but it's not vital to test driving it. The commenting feature is active on all spell details. You have to be a registered member and logged in to post comments, so if you're not registered yet run and do that :)

Also we're not entirely happy with the way the system handles AA spells/abilities, but we're working on a better way to filter it.

Have fun and enjoy!
