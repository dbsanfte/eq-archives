If you don't know the DI info... it goes sort of like this:

50 CHA = ~30% chance for DI to fire
120 CHA = ~50% chance for DI to fire
200 CHA = ~70% chance for DI to fire
280+ CHA = ~96% chance for DI to fire

These tests were run by EQClerics, fairly extensively. If you want the link I can provide it.

Edit: I should point out that these CHA values are for the WARRIOR, not the cleric casting the spell.
