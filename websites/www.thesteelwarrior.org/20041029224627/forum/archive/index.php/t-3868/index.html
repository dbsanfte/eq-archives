Originally posted by Krorghar
BTW guys those weaps that allready got a proc, and then u aug em with another proc stuff like anger III...

Does em proc twice? or does em do a first proc check and then if failed a second proc check?. As a result of this they proc the same times or more?


It first checks to see if the main proc goes off. If the main proc doesn't go off, then it checks the augment proc. So both procs never go off simultaneously on the same swing.


Even so, because the chance to proc on any given swing is so low, your augment proc will go off only very slightly less than your main proc (a percent or two maybe). For practical purposes, the two procs will go off about the same number of times per minute.
