wandor
The type has to match the slot. A slot can be of any single type, and an item can have any number of slots. "Slot" is just a term used to determine how many augments an item can receive.

Type 1 = these are also know as type all - they fit in any augment slot.
Type 2 = there are augments with multiple stats on them.
Type 3 = these are effects
Type 4 = these are pre-LDoN weapon procs
Type 5 = LDoN weapon augment
Type 6 = LDoN weapon augment
Type 7 = these are pre-LDoN single stat augments
Type 8 = Time loot augments only
Cily
"Type 7 = these are pre-LDoN single stat augments"

There are a few effects on type 7, like some vengeance augs and such.

As for Type 1, they really do not exist. SoE made it pretty confusing by putting "type 1" slots on gear but not having any type 1 augs.
