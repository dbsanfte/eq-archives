I'm dabbling with Javascript to create an Hate calculator. Looking for some relevant formula's.

I use:

HD = delay / ( haste / 100)
SPM = 60 / HD
HPS = DMG + DB
HPM = SPM * HPS

Legend:
HD = Hasted delay
SPM = Swings per minute
HPS = Hate per swing
DMG = damage
DB = damage bonus
HPM = Hate per minute

I'm looking for ways to factor in the following:
Dualwield skill
Double attack skill
Tripple Attack ability
AA's which affect the number of swings per minute.
Offhand hate (Brodda Thep proved that offhand hate is reduced?)

Leaving procs out of this, since the purpose is to calculate melee hate and to compare weapons.
