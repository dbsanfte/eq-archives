For weapons, get a Copper hammer and a sword of the bloodsworn. Also pick up a Greatstaff of Thunder, you'll be able to use it next level anway.

As for everything else, I would get some cultural for now and start leveling up. With the amount of plats you have, you can get some pretty good gear if you watch the bazaar. There are a few slots that are hard to fill with good items. Try to get those first, the rest you can fill in with ornate drops or cultural gear.

As for the slots you need now -
Rings - Valorium rings will last a while
Legs - go with cultural for now, then upgrade to ornate of Tak legs later
Mask - grimling contraption is pretty good
Neck - either a necklace of flame marked links or an amulet of mummy wrappings will do for now
Helm - Helm of RZ is as good as it gets pre-Elem. On my server it will set you back 150k or so. You might consider making an LDoN haste helm if you can find some other rare tradeables like crystalized acid bracers.
