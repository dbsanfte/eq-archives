Haven't done it from EQ2, but I'm pretty sure you need to put in "eq1" and not just "eq".

To send them from EQ1 to EQ2 you have to use a square bracket instead of the /. For example:

[tell eq2.unrest.lelgollas Hi!

You can also join EQ2 channels from EQ1 by using [join EQ2.server.channel but you have to talk in it using the "[" which is very annoying since it doesn't automatically open the text bar and you have to hit 'enter' first.
