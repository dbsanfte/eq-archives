If he tried booting up with a OS "boot disk" then it isn't refresh rate, although it does behave that way. Boot disks use a default display driver with a basic refresh rate.

I would point to something getting wacked in the bios, like maybe it got set to 4x when the card supports 2 only (just an example) or something. Or maybe power save was set on, or the "wake on lan" or something.

Try getting into the bios, and double checking the settings. Can even flash the bios if your MB supports it.

Think about what has changed, hardware or software wise, since the last time it booted up normally. If nothing has changed, it could just be a bad MB, especially if you can't even get into the bios.

Good luck
