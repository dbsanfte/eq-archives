The one other thing that I know of (but it's very situational)

you can use "/tar a", which will target virtually any trash mob assuming there is no PC around who's name starts with "A". This of course won't catch many named mobs either. Only works since almost all trash mobs names start with "a xxxx xxxx". This can also be very handy for looting lost corpses.

/tar a
