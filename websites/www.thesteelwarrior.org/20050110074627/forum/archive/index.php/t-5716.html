From the EQ Tech Board -

Hello everyone,
Like many others, my wife and I were forced into a cycle of 1017 errors tonight after zoning into/out of GFay on Solusek Ro.

After much frustration, I hit upon a potential workaround. Now this workaround does not stop you from crashing while zoning. It may, however, get you back in the game MUCH faster than waiting for the server to boot you.

1. Restart EQ (and/or get back to Server Select screen).

2. Rather than selecting the server you were just booted from, select ANY other server and hit the "go" button ("enter world", whatever, you get the idea).

3. A message window should pop up telling you "You already have an active character on <your server name here>! Would you like to continue playing as the active character on <your server name here>?" And it will give you a YES/NO option.

4. Tell it that you want to play your active character, and it *should* start loading that character. Worked for us anyway.

Hope this helps get some of you back in the game quicker.
