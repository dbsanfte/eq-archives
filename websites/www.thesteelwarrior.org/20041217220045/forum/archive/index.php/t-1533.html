We are currently all spread out. Everyone having their ideas, everyone agreeing and disagreeing...
How could we ever assemble all these diverse ideas into one realistic fix for warriors?

Trying to get a fix here and there for warriors is asking too much imo.
If we want to see something happen, it has to be focused.

When casters ask a fix, it's simpler for them, because they know it's Spell related.

It's a bit more complicated for us, because we need a fix, but don't know where exactly. (everyone has their own opinion)


So I'd like to start a poll, and see in which domain the majority of the community would wish to see a fix happen.
We could then concentrate on this domain, create an armory of ideas all within this domain.

We would then have a Solid, Realistic Upgrade to warriors.
Rather then scattered ideas that give the impression that we want a complete Overhaul.

So here are the "Domains" in which warriors could seek a change, that i've thought of.
Please say so if I missed any possible domains.


Disciplines and "Stances" (New/Upgrade disciplines, as well as a new sort of permanent discipline called "Stances")

Statistics (the base of the warrior. Hp, AC, Skill caps etc..)

Skills and "Combat Moves" (More tools to the warrior. Be it simple skills, or more elaborate Special Moves.)

Itemization and Focus Items (weapons, effects and armor. New ones, upgrade some, make more warriors only ones etc.. As well as bringing in a plethora of new Focus items for warriors)

Knights (would the problem be knights only? would nerfing them fix everything?)



I've put in all possible domains, even those I don't agree at all with :p... it's up to the community to decide.

Yes, this isn't a poll yet, I want to be sure I got all domains possible, before starting a poll.

And please don't make any posts about how this Domain is useless or that Domain is completely moronic... the poll will show that :p
