It does work without MQ or EQmu as I had stated. It is similar to posting with unicode characters.

This is my INI string for incoming messages. I have used it for as long as EQIM has been out because it shows the true ascii string for links. The link itself is unimportant , it is the BOLD PURPLE color that get's the attention that I use it for.

You can use this or make your own link or create a new one, I don't care.


Page1Button9Name=ATK
Page1Button9Color=0
Page1Button9Line1=/g 0028071-00001-00001-00001-00001-000015ED87FFCATTACK ~~~> %t
Page1Button9Line2=/rs 0028071-00001-00001-00001-00001-0000157FED8FCATTACK ~~~> %t


Obviously you'll want to use a different hotkey button or move it to were ever you want to use it. You can change anything at the ATTACK and beyond and make your own message as long as it stays with in the "" and contains 35 (7-5-5-5-13) hex characters.
