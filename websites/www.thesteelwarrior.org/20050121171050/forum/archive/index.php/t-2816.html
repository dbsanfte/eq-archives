Instructions: Works with all versions of windows.

1) Open Charmap.exe Start--> Run--> Type "charmap" hit enter.

2) Make sure its set to the Arial font(EQ font). You can use any symbol you see there. Some look better ingame than others.

3) Open your Charactername_servernumber.ini file using notepad. In my case its Gear_22.ini

4) Find the bind you wish to add symbols to. You must have made the bind ahead of time. Like "Incoming heal to %t"

5) Copy the symbol you want to your clipboard. Just hit copy if you dont know.

6) Paste it into your bind. When you are done doing this with your binds you want to save the ini file using UTF-8 encoding. You select "Save As" from the file menu and change the encoding. You also want to select "All Files" under file type.

7) it will ask you if you want to overwrite your old one and you say yes. Bingo you are done. If it didnt ask you to overwrite you didnt save it properly.

On a side note everytime you save this file using UTF-8 encoding you nuke your friends list. They are still listed in the ini but you have to manualy re-add them ingame to get them to show up again. Enjoy =)

Please dont be to obnoxious with this.
