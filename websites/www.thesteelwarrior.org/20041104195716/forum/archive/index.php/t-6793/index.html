Sassinak
Enraging Blow (and all other pure hate procs aside from epic) is not resistable, but if it was, yes it would still generate the same aggro. Resists never spare you from aggro, period. Anyone who's played a class with a mana bar understands this. It works this way as a detriment to spellcasters; but it works to the advantage of proccing warriors.
