Well, I'm WIZ and not WAR, but it seems to me that above thing is exactly like they always said taunt works, at least according to post in your Library (http://www.thesteelwarrior.org/forum/showthread.php?postid=32130#post32130)

What they say there is that your Total hate has two parts:

Total_Hate = Hate + Hate_Bonus

Hate part is one that you get by meleeing or casting spells, or using Taunt. Hate bonus is part that you get by being first one to pull mob (First contact aggro, Bi), or sitting etc...

What they showed in that post is that you will need always 4 successful taunts to get aggro from person if that person was one who first aggroed/pulled mob. But if person who has most aggro at time you are doing taunt is not one who first aggroed/pulled mob, then only one successful taunt is enough.

Although that post in your Library was started and maybe even ended before latest taunt changes, in my recent tests for wizard aggro (http://www.graffe.com/forums/showthread.php?s=&postid=275928#post275928) I also tested taunt, and it appears to behave still as it did before. I also found numerical values for Bi and Ti. More specifically:

a) you still need exactly 4 successful taunts to get aggro from initial puller
b) value for initial aggro Bi is around 100
c) therefore value added for successful taunt is between 25 and 33, lets say Ti=30


Few examples:
--------------------

1) SK pulls mob and start tanking it. After he stop, aggro is like:
Total_SK_Aggro= SK_Tank_aggro + 100(pull)
Total_WAR_Aggro= 0 + 0

War 1st succ. Taunt: Total_WAR_Aggro= ( SK_Tank_aggro + 30) +0 -> mob still ok SK
War 2nd succ. Taunt: Total_WAR_Aggro= ( SK_Tank_aggro + 60) +0 -> mob still ok SK
War 3rd succ. Taunt: Total_WAR_Aggro= ( SK_Tank_aggro + 90) +0 -> mob still ok SK
War 4th succ. Taunt: Total_WAR_Aggro= ( SK_Tank_aggro + 120) +0 ==> mob turned to WAR !


2) WAR pulls mob and SK start tanking it. After he stop, aggro is like:
Total_SK_Aggro= SK_Tank_aggro + 0
Total_WAR_Aggro= 0 + 100(pull)

War 1st succ. Taunt: Total_WAR_Aggro= ( SK_Tank_aggro + 30) + 100 => mob immediatelly turn on WAR !


3) MNK pulls mob and SK start tanking it. After he stop, aggro is like:
Total_SK_Aggro= SK_Tank_aggro + 0
Total_MNK_Aggro= 0 + 100(pull)
Total_WAR_Aggro= 0 + 0

War 1st succ. Taunt: Total_WAR_Aggro= ( SK_Tank_aggro + 30) + 0 => mob immediatelly turn on WAR !


As you can see, there are regular situations where you need 4 successful taunts to get aggro, although usually you will need only one.
