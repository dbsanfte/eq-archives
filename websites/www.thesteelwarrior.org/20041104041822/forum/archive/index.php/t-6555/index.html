I use OSoW primary with a2 aug, and would not replace it with a GBoS with a2 if it were free. The OSoW's innate proc is enough aggro that, when used with incite, will pull aggro after a slow. (I've tested this many time duo with my shammy). It's been parsed that slows are ~700 aggro, incite is 400 aggro... so that leads me to believe the OSoW proc is right around 350ish. The GBoS proc is pretty commonly accepted at around 400 hate, so I think the increased proc rate (and the dead segzy look) make the OSoW just as good as, if not better than, the GBoS. Both procs are enough to pull aggro off pretty much anyone combined with incite, so again, the increased proc rate is nice. The only downside is that since its a debuff, the 30dd portion cannot be crit'ed =/
