richardkent
Nomryh,

Depending on your role for tanking, which augs you'd go with. Since Anger2 has been upgraded to 400hate (vs 200 a while back), it appears that it generates more aggro than ykesha (125dd + 0sec stun). Folks were guessing that ykesha was around 300ish hate as it seemed to compare favorable to the anger3 at that time.

If you're going to be MT or want certain aggro, go with 2 x Anger2 (preferable anger3) procs.

If you want some DPS while keeping decent aggro, go with 2 x ykesha or ykesha + anger2. I'd think if you want DPS, then stick with the ykeshas. Keep in mind you will get resists on the DD.

Would doing ykesha and anger2, one for each weapon, work well? Yes. So would ykesha on both. It's just more of what you want and how many points ya think you're gonna be getting in LDoN. If ya think 1150pts is easy, then experiment a bit and go for the higher end procs.

Just don't forget when you remove the aug from a weapon it is gone forever. Keep in mind anger2/3 is unresistable and a fixed 400/600 hate. Ykesha DD can be resisted, but that still adds hate.
