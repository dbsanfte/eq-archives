Keep all skills up. As you level the levels continue to take longer so you have a lot longer to get skills up. You never know what will drop in your lap:) Also I know a lot of Warriors 50+ use a Windblade (2hs) for agro when MTing. So you won't always use 2 weapons.

As for keeping 1hb and piercing up, get a decent one and put it in your off hand. Keep it there. Always get dex buffs.

As for 2hs and 2hb, I'm not sure how warriors get these up if they are main tanking. I would imagine that you would just use these in groups that you aren't MT in. My main is a beasty, I'm sure I'll run into the same issues that you have when I level my warrior up.
