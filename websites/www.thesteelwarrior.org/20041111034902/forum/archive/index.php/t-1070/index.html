EQ Companion
Sounds like you have not had logging turned on in EverQuest. EQ only logs the messages into the log file if you have logging turned on. Just type "/log on" in your console window after you have signed on to EQ (I.E. not in YALP, but in EQ).

The file name is something like 'eqlog_xxxxx_nn.txt' where the 'xxxxx' is your character's name and the 'nn' is a number that identifies the server you play on. For example 'eqlog_Gilador_85.txt' is the log file name for my shaman named Gilador on Tribunal server (85).
