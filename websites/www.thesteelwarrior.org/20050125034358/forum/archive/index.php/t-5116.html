IRC stands for Internet Relay Chat. It's one of the first internet chat protocols and is still very popular. If you're intimidated, don't be - it's just multiplayer notepad.

First off, you will need an IRC client. There are many to choose from. mIRC is popular. Trillian has a good IRC client as well. I hear even Mozilla has a built in IRC client, not to mention the several web-based IRC clients. Point is, pick one that is comfortable to and accessible by you.

Once it's installed, you'll have to join the server. This isn't too hard. Look around the menus or the FAQ included with the program to find the exact way to do this. In mIRC, you can click File > Select Server or Alt-E, then go to Connect > Servers. You may also have to add a new server, which is self-explanatory once you reach the menu.

In the server box, type "irc.thesteelwarrior.org" w/o the quotes. The port box should already read 6667, but enter that if it's not already there. Type whatever you want in the description; I use "TSW." You can leave the password blank for now. You'll also want to choose a "nick," which is a handle or username. Then, just hit connect!

Once you are connected, you'll want to type "/join #thesteelwarrior" in the server window w/o the quotes. Simple as that.

A little technical jargon -

channel - what you probably think of as a chat room. channels usually have the pound sign # in front of them.

op - an admin of the channel, stands for operator. they usually have a @ sign in front of their name.

voice - granted to channel users, it allows them to speak when the channel is "moderated."

moderated - a channel status that silences anyone who isn't an op or doesn't have voice.

kick - a temporary boot from the channel by an op. usually a warning.

ban - pretty self-explanatory =) there are all kinds of bans. when someone is banned, you'll see a message with +b in front of their username or ip address or some such.

idle - when a user doesn't type anything for a while. lots of people with perma net connections will stay in a channel 24/7, so don't be put off if they don't say anything to you at times.

On to Nickserv!

Nickserv is a service that lets you keep a nick on a certain server. Once you register your nick with a password, you can get the nick back any time as long as you have the password and have logged in within the last 30 days.

To register your nickname, type "/nickserv register passwordyouwant email" without the quotes. Don't worry about spam - I've not gotten any from registering. Now, you can enter this password in the server menu and it will be entered when you log on.

If your nick is stolen, or if you are disconnected but come back to find your nick is still online, type

/nickserv recover yournick yourpassword

afterwards, type

/nickserv release yournick yourpassword

then, just type /nick yournick and you're good to go.

Incessant leaving and re-entering a channel is considered to be bad etiquitte. I got chewed out pretty good the first time I did that =)

Anyway, come hang out with us. =)
