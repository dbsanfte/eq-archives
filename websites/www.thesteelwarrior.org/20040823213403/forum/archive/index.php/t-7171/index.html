I use Qbox2 (http://www.jaranya.com) and really like it. I run 1280x1024 res and use the /viewport command to put all boxes in a black background. The actual command at my resolution is /viewport 0 0 1130 700

This puts a 700px black bar across the bottom and a 150px black bar up the right side for the spell effects window... I like the look of it, it's easy to use and very easy on lag...
