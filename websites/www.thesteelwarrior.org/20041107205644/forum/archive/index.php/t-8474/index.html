OK, math time - formulas are a combination of those in the Library thread and what I've been told/figured out on how mitigation components work, and reworked em to solve based on given information

Basic Formulas and Definitions
Set Y as DB Mitigation, Shielding as the % number (ie, 21 for 21% shielding), Innate as the innate class bonus mitigation as the % number (5 as far as I know), Y1=Mitigation with Spell boost, currently Vie is 10% boost

Y=(100-Shielding-Innate)/100

This yields Y=0.74 for the above calculation

Y1=Y*(100-Spell)/100

This yields Y1=0.666

Set Z as DI Mitigation, Defensive is currently 45%, Innate is 5% once again, as the innate 5% affects both DI and DB to my knowledge. Z1 is Mitigation with spell boost, currently Vie is 10%

Z=(100-Discipline-Innate)/100

This Yields Z=0.5 for defensive

Z1=Z*(100-Spell)/100

This yields Z1=0.45 for defensive+vie

Now, using D1 for min damage, and D2 for the second step up, we have:

DI= D2/Z-D1/Z for defensive numbers, or
DI= D2/Z1-D1/Z1 for defensive+vie numbers

Now that DI has been calculated,

DB= (D1-(Z*DI))/Y for defensive numbers, or
DB= (D1-(Z1*DI))/Y1 for defensive+vie numbers.


Damage spreads are simply
DMG = (Y*DB) + (Z*DI*X), where X = 1-20, for defensive spread
DMG = (Y1*DB) + (Z1*DX), for defensive + vie spread.
