Doggabone
You want a named anchor tag, I think.

In cell 2, you'd have <a href="#cell3">Link text</a>. In cell 3, you'd have <a name="cell3"></a>. I'm pretty wore out atm, so you may wanna double check that for syntax. ;) If you just want to slide cell 3 into view, then the named anchor should do the trick.

If the intent is to load new content into cell 3, you could do that with an IFRAME. <IFRAME name="cell3" ...></IFRAME>. And link with a target of *cell3* will load in the IFRAME, just as with a FRAME.

To do that layout in nested frames, you could use a nested frameset - one frameset that slices the window across the middle (a top and bottom half). The bottom half would be a frameset with a left (cell 2) and right (cell 3) half. You can nest framesets ad infinitum, tho it gets easy to lose track of them the more you do. :eek:

Check out http://www.htmlcodetutorial.com/, it's a pretty good site for design and code tips, tricks, and refreshers.

< frameset rows="20%, *">
< frame src="header.htm" name="header">
< frameset cols="15%, *">
< frame src="nav.htm" name="nav">
< frame src="main.htm" name="main">
< /frameset>
