I have CDG3, and I love it. Is it something I would get early on? No, but it's nice when you get all the useful skills. I have it go off several times per LDoN adventure. The mob has to be fleeing, so it won't go off if there are other mobs in camp, or if it is undead (undead do not flee). So if you are in Guk, where most are undead, and the eyes do not flee (immune to changes in run speed), then you will not see it go off.

But go to Ruj or Tak, and you will see it go off quite often when you have a single mob start to run. It has saved my ass many many times.
