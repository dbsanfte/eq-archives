At lvl 65 there are two Sta mods:

0-255sta = lvl/10 = 6.5 HP per STA

256-305sta = approx 2-3 HP per STA

Both of these figures are on tanks with no ND/PE/PD, so nothing is further modifying their sta to hp. Simple calculations could re-weight these values depending on your aa's. (ie. x1.10% with ND3)

In short, there are serious diminishing returns on the hp gained from sta that exceeds 255.

using magelo formula :

HP = TRUNC[ 5 x ND ] + TRUNC [ Lvl x LM x ND ] + TRUNC [ TRUNC [ ( STA - TRUNC [(STA-255)/2] ) x Lvl x LM / 300 ] x ND ]

the TRUNC[(STA-255)/2] value being =0 if sta <255 (that part just mean in fact any STA point after 255 give only half benefits).
with ND multiplier due to AAs (=1 without hp-giving AAs) and LM based on class and level :
Warrior:
Level 1-19: Levelmultiplier = 22
Level 20-29: Levelmultiplier = 23
Level 30-39: Levelmultiplier = 25
Level 40-52: Levelmultiplier = 27
Level 53-56: Levelmultiplier = 28
Level 57-59: Levelmultiplier = 29
Level 60-65: Levelmultiplier = 30

so HP per sta is LVL*LM /300, = lvl/10 for lvls 60 to 65, or 6 to 6.5hp /stamina point. not counting AAs like ND or PE

for example, at 65 with ND3 ( +10% hps ), 65*30/300*1.10=6.5*1.10 = 7.15 hp per stamina point.
