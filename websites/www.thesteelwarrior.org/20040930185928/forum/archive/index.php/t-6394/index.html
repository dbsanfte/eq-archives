Enchanter Code v0.2

-= Instructions =-

The Enchanter Code consists of several categories. Each category is labeled with a letter and some qualifiers. Go through each category and determine which set of qualifiers best describes you in that category. By stringing all of these 'codes' together, you are able to construct your overall enchanter code. It is this single line of code that will inform other enchanters the world over of what a great enchanter you actually are.

Some of the qualifiers will very probably not match with you exactly. It is impossible to cover all possibilities in each category. Simply choose that qualifier that most closely matches you.

After you have determined each of your qualifiers, you need to the construct your ENCHANTER CODE BLOCK. Instructions are provided on how to do this towards the end of this file.

Also, pay particular attention to case-sensitivity, there can be a big difference between a 'w' and a 'W'.

-= Variables =-

Enchanters can seldom be strictly quantified. To facilitate the fact that within any one category the enchanter may not be able to determine a specific rating, variables have been designed to allow this range to be included.

@
for this variable, said trait is not very rigid, may change with time or with individual interaction.

()
for indicating "cross-overs" or ranges.

>
for 'wannabe' ratings. Indicating that while the enchanter is currently at one rating, they are striving to reach another.

$
Indicates that this particular category is done for plat.

?
Unless stated otherwise within the specific category, the ? is placed after the category identifier and indicates that the enchanter has no specific knowledge about that specific category.

!
Placed BEFORE the category. Unless stated otherwise, indicates that the person does not participate in this category. This is unlike the ? variable as the ? indicates lack of knowledge, while the ! indicates lack of participation.

-= How to Display Your Code =-

Now that you have your ratings for each of the above categories, it's time to assemble your code for displaying to the world. Take each category you determined and list them all together with one space between each one. If you run out space on one line, continue it on the next. When completed, it will look something like the following:

E65.125/L h- 2.45khp 5.54km ft11 G++>+++<3 e+++@ o- T+++ L(+++)++ H++(--)@ f++ g++ b- c++++@ t+ !D X@

If you are going to place your Enchanter Code into your .signature or .plan file (highly recommended), it is also recommended that you include a link to the version of Enchanter Code you're using. Here are a couple examples of acceptable bbcode (see how it they look at the bottom of this post):
