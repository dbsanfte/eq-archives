Sassinak
I used to use a combination of /assist and /shield. What I found was that it too often gave me a person who really didn't need to be shielded, and left me with no opportunity to judge the value of shielding them.

Nowadays, I just type the first couple letters of people's names. It takes me maybe 2 seconds to type "/shi Arw" for example. If they don't survive that short delay, then shielding probably wouldn't have saved them anyway.

Brutul
I still use the /assist /shield key. If somebody pulls the mob off the person you're shielding before it lands (another tank for example) it's no more wasted shielding the tank than it would be shielding an enchanter who no longer has aggro. I use shield a lot personally. Any time someone who's not supposed to be tanking stays on the target's target window for more than a second or two I hit my macro.

Some other buttons that are great are the "toggle last two targets" and "cycle through nearest NPCs" buttons. Both have tons of good uses once you start playing around with them. I have them mapped to 'Q' and 'W' and use them all the time.

Cuthbadth
I use the shield key in xp groups, not even on my raid toolbar...for groups I typically set it up for the enchanter or shaman, if the slower is a beastlord, they can take a few hits, so then I set it to a healer. I don't bother with assist mob, etc., because in a group if they are getting hit I want agro back fast, in which case I don't want to miss one round on the mob if I can help it. If someone else gets agro, then I'll type out /shield name manually if it's not a melee class.

My raid toolbar generally has (keys are not in order listed here):
Main Assist (if I'm the MA, this is set to assist puller)
Secondary Assist
Call Assist (if I am not MA I change it to: Offtanking << %t >> , please assist MA).
Taunt (use if MA or to taunt mob while chanter mezs or re-charms)
Kick (since I'm a short race)
AET (regardless of tank order...I've had to use it on Bert when 4 tanks died).
Range Attack (use if MA or if too many melee on a small target)
Rampage DMG (/rs I have RAMPAGE DAMAGE!!!)
...I also have two open slots

For my disciplines, I have incite, furious, fellstrike (this is generally the better offensive disc), defensive, fortitude, evasive, fearless and the flashy dps (forget name for some reason) which I only use when I want to spam some crits (flash) or I am low health and want the chance to stun the mob a lot from crippling blows.

Oh...and I should mention I'm not elemental geared, so I can't say what tanks use in the 'big guilds'.
