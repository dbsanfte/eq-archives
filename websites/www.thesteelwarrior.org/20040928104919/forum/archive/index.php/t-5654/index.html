FIXED!

To get the ipconfig /flushdns command to work, I had to reenable the DNS Client in the Start -> Control Panel -> Admin Tools -> Computer Management -> Services and Applications -> Services -> DNS Client which had somehow gotten disabed. Once I turned it back on and flushed the dns using that command (I tried before re-enabling the DNS Client and it didn't work) everything worked fine.
