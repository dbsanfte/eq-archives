What Absor is saying here is that SOE doesn't recognize a class balance issue and create an expansion to fix it, or withhold a fix for the next expansion. Balance is a by-product of boxed content, not a feature. You can see this reading past patch messages. After all, while some developers are working on content others are working on class issues, and while there will be cross-over there will not always be synchronicity. They try to release expansions that are balanced for the classes, they don't try to balance classes by releasing expansions.

Valeris hits the nail on the head that balance cannot help but be affected by expansions. Class balance, done perfectly, means that every class has an equally viable role in the game and equally viable avenues of progression.
