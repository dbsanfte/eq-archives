Best way i can think to search for warrior items with skillmods would be to go to http://www.cardplace.com/eq/config/ and set all the values to zero except for the skillmods you want to find. For me the script has a problem with resetting some fields to non-zero values, so I just set the skillmods to 1000 weight so that any item with a mod comes out on top. It's sort of a pain in the butt too that you can only look up one slot at a time, but once i got used to it I found items pretty quickly, for example:


Boot of Kylthorn - 8% riposte
Sandals of Contemplation - 8% kick
Expeditious Boots - 7% dodge

Mace of Precience - 5% dodge
Bloodfrenzy - 15% riposte
Snowchipper - 3% parry
Axe of the warmaster - 5% parry

There's many more than these, just a few examples.
