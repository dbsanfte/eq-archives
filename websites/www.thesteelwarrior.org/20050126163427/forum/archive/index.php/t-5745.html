As said above they are on same timer so having both in a hotkey will do no good unless it works as mentioned above about illusions.

ONLY the pause command can be used on the same line as another command, the pause is first on line and happens after the fact i.e. /pause 20, /do 1
which would doability 1 and then pause for 2 seconds

As for pause NEVER use it in any combat ability as it locks up your ability to do other things for said amount of pause
