you can add the following key to the registry instead of downloading that big 7 meg program. Easy way to do it is described below.


1. Paste the following into notepad:

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\AppCompatFlags]
"{e05bd30e-07df-4054-a343-7ec90de46dd2}"=dword:00000001

2. Save the file as <filename>.reg (example: eqalttab.reg)
3. Right mouse-click on the file and choose "merge" from the context menu.

No reboot is required, you're finished at this point.
