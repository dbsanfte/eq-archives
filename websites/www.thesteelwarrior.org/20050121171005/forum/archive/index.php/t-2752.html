To be 100% your memory is not screwed, boot up, go into the BIOS, there should be an option under the General or Advanced BIOS that is called: Quick POST (Power On Self Test). Turn it off and reboot, it will make a long memory count, it does it 3 times, if no error jumps there, your memory is fine :)

I use Win98SE and never had a problem with EQ. Also, run 'msconfig' from your 'Run' button on Start, and check the Start tab, there is a list of things that get loaded on the OS startup, uncheck the ones you think should not be loaded, thats a good way to stop working troyans too fyi :)
