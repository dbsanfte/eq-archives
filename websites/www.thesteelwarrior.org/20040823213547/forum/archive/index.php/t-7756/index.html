So we've been debating how good/bad the changes in Ikkinz are. Most people have said that they hate the change, but not many alternatives have been posted.. So......

I started thinking about how they could retune the encounter so that, warriors could keep defensive, yet still have knights be able to tank the boss. I thought about the "raid encounters" that knights can tank, and had this idea.

When a Knight is in a group when the expedition is created, it creates a named that has a chromatic resist fear proc every tick. This would make a warrior fairly useless because - we can still be feared, whereas Knights are immune. It would make a warrior a poor choice to tank that version. If there are no knights in the group, then we you get the pre-patch version. Warriors can then useless defensive to tank in their version, which would be too difficult for a knight to tank, and the knight version would have an effect which would be too difficult for a warrior to tank.

That's my first idea. What are yours?

Please keep this thread to *ideas only* - leave the debating of the change to the other thread please.
