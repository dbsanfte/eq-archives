1(auto attack)3(kick)33333333333333333333

This sequence is quite reliable for about 95% of the stuff you kill in a group.

In WoW we'll assume you pull 3 elite mobs and win fairly easily to simulate the equivalent generic 1 trash mob versus group deal in EQ.

Mobs come in, you charge(1), demo shout them (2), drop Sunder on some mobs (3), maybe taunt something (4), put a dot (5), maybe use some AE attack (6), snare runners (7).

Without loss of generality we'll assume the combination 123334335673357337 works. If you're not convinced you can always take an example where you've won such an encounter and change the #s appropriately (e.g. maybe your #6 is concussive blow, or whatever).

Now we'll assume you're a good tank and your group won without anyone dying. After all this is a pretty generic pull in the dungeons that you're expected to win. Now you come across another set of 3 similar elite mobs. Do you have any reason to believe the combination 123334335673357337 won't work this time? No, because the game is deterministic. You wouldn't want a game where what worked the first time suddenly didn't work under the same situation. So it'd be pretty safe to assume that repeating what worked earlier should enjoy the same success (assuming of course everyone in the group is still doing the same thing as before of course), and you'd be wise to repeat the same thing that worked before rather than reinventing the wheel again.

So in the end, the EQ sequence of 1333333333333 is same as the complicated sequence of 123334335673357337 once that sequence is known. Both works 95% of the time and both are the sequence of choice for a vast majority of encounters.

The same can be applied to soloing. Go to any post where people talk about how great they solo. I can guaranteed you they did mostly the same thing every time they solo. For example the one that talks about how MS + Overpower works great on soloing would roughly look like:

M12MM12MM12MM12M

where M = macro to switch weapons, 1 = MS, 2 = Overpower

Once the optimal sequence is known, you'd be stupid to do anything but repeating it over and over again, and that means no matter how complicated, it's exactly same as autoattack. The best you can do in both game is done via repeating the same things over and over again, but WoW happens to require to push more buttons. Before the optimal (or just winning is sufficient) sequence is discovered yes you have a lot of options to try, but once you find one that works, there's no reason not to stick with that one.

Now you come across another set of 3 similar elite mobs. Do you have any reason to believe the combination 123334335673357337 won't work this time? No, because the game is deterministic.
