Some tips for targeting mobs other then MA's:
- set up /assist MA key, hit it on incoming, will see what target the MA has, and can change to another.
- Talk to MA, see how he selects mob, he takes first to camp? He takes target from puller?
- Some camps where you are clearing fast, set a left and right rule. Every time 2 mobs come into camp, MA will take left, SA right.

Tips for dealing with charming chanters (I don't cover everything, let them deal with mezing, I just want to have some control over mob selection as MA).
- Have them call charm (nothing worst then hitting mob for 2-3 seconds then gets charmed).
- If / When charm breaks, DON'T ATTACK THERE PET. Can Incite, or taunt / area taunt to help, but don't do any damage.
- Con mob if you are not sure if a pet (pets will be indifferent).

Charmed pets can be worth the added DPS for some encounters, but might pass on some encounters that you can win easily, don't want to loose from a pet getting loose after killing it's owner.
