Bonequest
The key to the upper level of city of mist goes on keyring, (a stone key), all the keys to ToFS go on it. Also there are keys for BoT ( goes on keyring only), BoT towers (goes on keyring), Agnar area (will not go on keyring), PoEb (no longer an actual key but is on keyring), PoV dragon door (not sure if goes on keyring), CoD alternate access key (nice instant click item is key), Tactics alternate access key, the Arbitor area in PoEa (a gem etched key--will not go on key ring), plane of time (Quintesessence of elements), Factory door in PoI is a flagged key, castle doors in PoD are a flagged key, Sol Ro area in Sol Ro tower is a flagged key gained by completeing tower bosses, and probably a lot more I missed. Of course there are also zones like KT where it is technically a flag but no piggybacking is allowed so it is really a key but there isn't much difference really between a key and a flag other than one usually takes up bank space.
