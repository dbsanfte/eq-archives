Best idea I've seen is to make the disc setup to do the following:
1. After using a disc, re-use time for that same discipline is the normal re-use time we have now.
2. However, after using a discipline (any discipline), we only have a 2-3 minute timer (can be extended a bit or lessened) before we can use a different discipline.

This means you can use the auto-riposte to save your life if needed in an emergency, and only have 2-3 minutes before you can use a different discipline. However, if you want to use auto-riposte again, you'll have to wait 56 minutes (the normal current re-use time on the current situation) before you can use that same disc.
