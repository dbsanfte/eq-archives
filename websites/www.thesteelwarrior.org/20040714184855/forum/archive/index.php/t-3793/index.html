Controlling for time, that looks like anger 3 is proccing at about the same rate as the regular proc, which is to be expected.

31/24 = 1.3 ppm
137/41 = 3.3 ppm / 2 procs = 1.67 ppm average for each proc

That's a difference of about .37 ppm, which is probably within simple random variance given the sample size.

Both numbers seem a little low, really, especially the unaugmented one, but I guess that's because the time number is adventure time, not just melee time, so the period between pulls is included and watering down the rate.
