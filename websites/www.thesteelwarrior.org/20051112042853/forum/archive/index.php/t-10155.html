Each point of stamina from worn items is worth 10hp. Each point of stamina from your base stats is worth a lot less, but the exact formula is not obvious.

2) How does defense work (not AC, the + to defense skill stuff) and does this have diminishing effects or a cap? (What I'm trying to ask is how good is this stuff, is it worth sacrificing stamina/AC for it and going all out defense gear or is it not that great..?

Each point of defense adds 0.04 to dodge, block and parry and decreases the chance for a mob/player to crit you by an unknown amount, which is presumed to be the same as the normal level difference. ( I.e. a level 20 mob will crit you as if you are level 20 yourself if you are level 19 with +5 to defense ). What that is worth in terms of STA/AF is a personal decision, much the same as deciding if 12STA is better than 1% dodge.

3) How does enchanting work? I can see that you can add certain enchantments to items but do all items have an enchantment slot? Also it says that some items already have enchantments and you can apply a new enchancement by overwriting the old one. How can you tell if an item already has an enhancement and what it is?

Enchanters use components created by destroying green (good) quality items to add enchantment effects to items. All items can be enchanted, however most items only have one enhancement slot, so enchanting an item will mean that you can't use other enhancement items, such as armor kits or shield spikes, on it. The enchantments are fixed by the type of item, so for example enchanters will get a spell that enchants a bracer by +5 stamina. The better enchantments require items that are difficult to get, usually only obtained by destroying blue (rare) or purple (epic) quality items. Enchanters can enchant soulbound items through a special trade interface.

Enchantments on items will show up under the item description as a line of green text and if you attempt to enchant an item that already has an enchant, it will popup a dialog box asking the enchanter if they wish to replace the existing enchantment with their new one.
