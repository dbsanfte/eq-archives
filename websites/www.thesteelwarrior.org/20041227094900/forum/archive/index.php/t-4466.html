beg/disarm is a skill - if you pull up the skills tab you should be able to see those skills. cant rememebr if you have to train them if you have never used them before.

you can then drag those two buttons over to your hotbank and use accordingly.

there jurys still out on how much hate you actually get from using them, but for what its worth i guess theres no harm having a couple extra buttons to press.

as for using the taunt key, well, the first thing is dont mash it. use it when you lose aggro. not sure what level you are but only if you need to train up your skill then mash the taunt key.

the way i try and set things up in exp groups is to stay close to the sham/slower and preferably having them opposite me. i then watch for the cast messages.

for shams, normally they will malo first. then slow. watch the slow to cast, cast time is 3 seconds.

once slow lands, get ready to taunt. if aggro sticks to you then fine, but if not thats the time to taunt.

normally if you time it right,and on a successful taunt, you can pretty much ensure the sham avoids even a single hit.
