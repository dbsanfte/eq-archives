Let X be the aggro from a 8/20 per swing.
Let I be the initial aggro, A be the aggro from the first force of Akera, B from the second and C from the third.

Then what you have proved is:
5 X <= I <= 6 X
67 X <= I + A <= 68 X
131 X <= I + A + B <= 132 X
192 X <= I + A + B + C <= 193 X

First question: is this consistend with A = B = C = Force of Ak aggro
Let us assume F is all of A B and C. Then

1) 5 X <= I <= 6 X
2) 67 X <= I + F <= 68 X
3) 131 X <= I + 2 F <= 132 X
4) 192 X <= I + 3 F <= 193 X

1 & 2 give:
61 X <= F <= 63 X
1 & 3 give:
62.5 X <= F <= 63.5 X
1 & 4 give:
62 X <= F <= 62.667

Together we get
5) 62.5 X <= F <= 62.667 X

now, 2 and 5 give us:
4.333 X <= I <= 5.5 X
with 1 this becomes
5 X <= I <= 5.5 X

Plug this into (3) and we get
62.75 X <= F <= 63.5 X
which together with (5) provides us with a contradiction.

Thus proving our aggro model is flawed or incomplete. There is an additional source of aggro wandering around that isn't accounted for.

Dispite this additional unknown aggro source, it was really damn close to linear.

I'll take as a rule of thumb that
raw stun aggro =~ 62.5 offhand swings of an 8 damage weapon.
Initial aggro =~ 5.5 offhand swings of an 8 damage weapon.

which gives us
108 DD component of QWoS =~ 17 offhand swings of an 8 damage weapon. (+/-)

Let Y be the aggro from a 9 damage weapon in offhand.

Lets assume that the offhand swing has some "base" aggro and some aggro that is purportional to the weapon damage. The base aggro might be zero.

So,
Y = B + 9 * D
X = B + 8 * D

The above test says:
61 Y <= I + F <= 62 Y
The test with the 8 damage in offhand weapon says:
67 X <= I + F <= 68 X

Rearranging we get
61 Y <= 68 X
67 X <= 62 Y

Using the B + # * D equations, we get
61 B + 549 D <= 68 B + 544 D
67 B + 536 D <= 62 B + 558 D
or
5 D <= 7 B
5 B <= 22 D
0.71 D <= B <= 4.4 D

Ie, aggro from offhand swings is not quite just purportional to weapon damage. There is a small additional component, between .71 raw-damage-aggro worth and 4.4 raw-damage-aggro worth.
