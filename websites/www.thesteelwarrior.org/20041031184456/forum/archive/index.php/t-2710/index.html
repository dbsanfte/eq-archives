wold what ranger board u use ? go to The Ranger's Glade (http://forums.interealms.com/ranger) for a ranger board.
archery depend on DEX the SAME way melee depend on STR... and as well as dex as no effect on melee, str has no effect on archery. (special : procs and crits, based on dex for both melee and archery)
It is a known thing that has been tested / parsed, and if there is no more data on it, it is mostly because noone wanna do it again.

for more , read varaho post

about crits : crits rate on archery for war and rngs = crit rate on melee for war iirc (we had parses about that on the glade, must dig a bit to find them)

about damage :
the easiest way to know archery damage : like any weapon, a bow + arrow will have a damage/delay : total damage is sum of bow damage and arrow damage, and the same thing apply for delay (well, in fact delay = bow delay because until now, all arrows have 0 delay).
The fact is damage calculation does not work the same on archery and on melee, but the difference is simple : with a given weapon damage, archery will do 2 times less damage than melee.
Meaning a 20damage bow with a 2 dmg arrow will do the same damages than a 11 dmg weapon (with a 0 DB). fact is delay will prolly be more important, and u will lose the melee DB.
Meaning also that all formulas u use for melee will apply to archery if u use ((bow dmg + arrow dmg) /2) instead of (wpn dmg) and DB = 0

so for answers to the 1st post :
1) Is it exactly like melee damage, even though archery damage is non melee?
yes, but main stat is dex and not str, and the formula to know real damage, based on weapon damage, is a bit different.

2) Is Atk the main determinant, ,and therefore max str important?
1st part : ATK is exactly the same importance for archery than for melee. but str doesn't change archery atk, only dex does.
(REMINDER : the ATK value shown in EQ is the MAIN HAND MELEE ATK only, and does not apply to offhand nor to archery. off hand and archery atk are calculated the same way than main hand atk, but use their own values for skill ( slashing/piercing/archery) and for stat ( str used also for off hand, but dex for archery). yes, +atk item does influence archery atk.)
so the answer to the 2nd part is : no , max str not important at all, but max dex will be (and will also give more procs and more crits, and more skillups)

3) I'm pretty sure dex affects crits in bow usage
yes, same way it effects crits in melee

4) Does haste affect bow delay?
same way it effects melee. But archery uses one more kind of haste, stacking with the others : quiver haste (QH). it follows the same rules than all the other kinds of haste : only the best QH is used if more tahn one apply. To have a QH, u just need to have a Weight Reducing quiver in the inventory.
QH = WR of the quiver / 3.

Sylviania
wold : no, that mean, like i previously said, that the ATK number shown under EQ is only the one for MAIN HAND MELEE WEAPON. It does NOT influence your off hand or range attack.
best proof is to play with something that is not a weapon in main hand : H2H skill is used to calculate shown ATK value, but off hand and range attack are not at all effected and u still hit as much with them.

There is not 2, but 3 separate atk : the main hand, the off hand, and the range

main hand and off hand atk follows the same rules : based on offence, weapon skill, and str.

range atk is calculated the same way, but instead of using the STR it uses DEX.
That mean everywhere STR is used for calculation of melee atk, u use dex for range atk

atk buffs and +atk items of course effects the 3 values

for the parses, I am not even sure they are still there ( big cut through our boards a few monthes ago, a lot of things were lost), and i am too lazy to try to find them. I am just giving u the result : for range, str has no effect on damage output, but Dex seems to have the effect str should have.
