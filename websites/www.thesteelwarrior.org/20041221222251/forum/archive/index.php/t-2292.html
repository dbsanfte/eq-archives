From the discussion on HPs from the Magelo board, this is the formula it uses and its pretty close

Just to make a quick recap of the hitpoints and all, we have seen the following in the mean time for hitpoints due to Stamina and level:

HP = TRUNC[ 5 x ND ] + TRUNC [ Lvl x LM x ND ] + TRUNC [ TRUNC [ ( STA - TRUNC [(STA-255)/2] ) x Lvl x LM / 300 ] x ND ]

Where the ND is the multiplier bonus they use (Natural Durability, Physical Enhancement, etc ...) and where the term TRUNC [(STA-255)/2] is always 0 or greater (i.e. when STA is below 255, it is considered equal to 0). LM is a class and level dependent multiplier as indicated below:

Warrior:
Level 1-19: Levelmultiplier = 22
Level 20-29: Levelmultiplier = 23
Level 30-39: Levelmultiplier = 25
Level 40-52: Levelmultiplier = 27
Level 53-56: Levelmultiplier = 28
Level 57-59: Levelmultiplier = 29
Level 60-65: Levelmultiplier = 30

Sure, ill see if i can explain it.

I'll use myself as an expample.

Lets establish the variables first:

Lvl = Level = 61
LM = Level Modifier = 30 (60+ Warrior)
STA = Stamina = 228
ND = Natural Durability Modifier = 1
(I have no ND or PE, ND1 would be ND = 1.02, ND2 = 1.05 and ND3=1.1. Not sure what the numbers for PE are.)

So lets take it a step at a time,


TRUNC[ 5 x ND ] = TRUNC[5 x 1] = TRUNC[5] = 5

...TRUNC[x] just means to take off the decimal place, TRUNC[19.6] would equal 19

TRUNC [ Lvl x LM x ND ] = TRUNC[61 x 30 x 1] = TRUNC[1830]= 1830

The Third part is a bit more daunting

TRUNC [ TRUNC [ ( STA - TRUNC [(STA-255)/2] ) x Lvl x LM / 300 ] x ND ]

My STA is less than 255 so the "TRUNC[(STA-255)/2]" part is just 0 for me, so i can get rid of it, leaving me with:

TRUNC [ TRUNC [ STA x Lvl x LM / 300 ] x ND ]

TRUNC [ TRUNC [ STA x Lvl x LM / 300 ] x ND ] = TRUNC[ TRUNC[228 x 61 x 30/300] x 1] = TRUNC[ TRUNC[417240/300] x 1] = TRUNC[ TRUNC[1390.8] x 1] = TRUNC[1390 x 1] = TRUNC[1390] = 1390

No add those there numbers together to get your HPs from STA:

5 + 1830 + 1390 = 3225

If you add the 870 HPs from +HP items to that you get my total HPs, 870 + 3225 = 4095
Which is correct except I dont have my PoP charm in my profile, so my real HPs are 4111, but 4095 is correct with out it.

I hope that clears it up a bit, if not let me know and ill help some more if i can.
