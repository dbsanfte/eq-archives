Anker steadfast
AS far as I know, INT / WIS doesn't govern how fast you get combat skills in anyw ay, Offensive OR defensive. Instead it is governed by DEX and AGI .... DEX for offensive skills sich as Offense, Weapon Skill, Dualwield .. etc. And AGI for the defensive combat skills, such as Defense, Dodge, Parry, Riposte, Safe Fall.

INT / WIS governs Spell skills, and tradeskills however (Though smithing has a Strength bonus, and Fletching a DEX bonus, if I remember correctly). And also those skills not already covered, such as Forage and swimming. Used to govern skill ups for Sense Heading too, but they gimped the skill instead.

However, it can never hurt to have a KEI buff there if you have room enough for the buffs.


Offcourse, things can have been changed since I played.
Frawdo
