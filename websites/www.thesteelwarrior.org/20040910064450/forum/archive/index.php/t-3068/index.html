Retnaburn
You can send tells to other servers.

;tell servername.playername

Sometimes it uses a short version of the servername. Like Solusek Ro is just ;tell Solusek.playername. If you have a question you can see if I'm on.

;tell Stormhammer.Retnaburn

Also, there is a serverwide warrior channel you can join.

/join serverwide.thesteelwarrior

You can use /autojoin instead of /join if you want to automaticly join it when logging in. To speak in that channel type /1 or /2 etc. whatever number that channel is. Type /list to see what number that channel is set to.
