using Breal's hate formulas epic1.5+2xA3 should have aproximatly the same aggro generation than EoE+A3
but epic 1.5 has much better stat, so go for it.


EDIT : calculated for mainhand.
please note that breal's formula : Hate_per_Swing = DM + DB * mod1 / mod2
mod1/mod2 = 1.4 for warrior without hate mods (works for 1Her, don't remember if it's the same for 2Her). let's call DBM = DB * mod1 / mod2 for the one hander.
assuming they will both swing x times each round.
round per minute being Rpm = (1+Haste%) * 600 / DL
we get Swing_Hate_per_minute = x * (DM + DBM) * (1+Haste%) * 600 / DL
x and haste% being equal for each weapon (they are in the same slot and of the same type), we get the ratio between EoE (weapon 1) and epic1.5 (weapon 2) is : ( (DM1+DBM) * DL2 ) / ( (DM2+DBM) * DL1 )
Numerical application :
main hand : ( 26*(20+21) ) / ( 20*(28+21)) = 1.09
off hand : ( 26*(20+0) ) / ( 20*(28+0)) = 0.93 (breal parsed SS as being pure DPS but having no impact on aggro).
thus EoE has a bigger hate generation from swings than epic 1.5 oin main hand !
but time snaps being a stun it's weaker than A3 thus it should balance each other depending on your AA and buff.

Note : as soon as DBM is bigger than 7 the advantage is for the EoE (on swing hate).
