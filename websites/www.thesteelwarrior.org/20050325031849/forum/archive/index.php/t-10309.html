I don't see how this would work. From my experiences on playing 3 vastly different computers, most of the graphical "lag" comes from the game trying to frantically load everything it needs into RAM. You turn and you need to see something else, so the game tries to load it. You only have so much RAM, and eventually the game needs to start using virtual RAM, which is just a file on your hard drive. HD Access is significantly slower than RAM access, and when you start using that, your frames go down the toilet until the game and windows agree on some middleground where everything WoW needs is immediately accessible in RAM, and the stuff that can wait or doesn't need to be changed that often sits in your virtual RAM.

I haven't tried it yet, so it just *might* do something, but every "miracle" tweak i've ever seen has never worked, and only people who don't seem to know much about computers, or have no real way to parse differences in performance seem to claim they work at all. Maybe theres somebody out there with a boatload of RAM and a slow processor that this tweak might work for, but I doubt even that.
