Yes, its playermade -> you better get yourself a GM smith for this (quite hight trivial, don't know for sure ... ~ 282)

The armorparts (for each slot) are dropping in MPG from the lightning warrior guys.
A part of these parts have to be transformed into the basic components and rest is needed as mold.

The result - if the smith don't fails :) - is the magnetic armorpiece.


If you wanna make charged pieces you'll need lightning cores as well.
And you'll need a lot of them !
-> e.g. a charged magnetic BP needs four cores.


The main problem is to get the cores these drop extremly rare....
We got 4 sets of complete plate armor but only 5 cores over all so far.
