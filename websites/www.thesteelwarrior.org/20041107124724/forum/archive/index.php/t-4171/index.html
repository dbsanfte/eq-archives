Hi all-

Here's the concept: how about suggesting some melee "re-balancing" given the following assumptions:

1). changes must be consistent with existing game mechanics, including extensions of existing code, but precluding massive recoding.

2). any item changes should be algorithmically determinable (is that a real word? ;) ) as opposed to requiring individual human review, with very very few exceptions.

3). Changes should not unbalance the game further than it already is; for instance, improving warriors to where knight classes are now just makes a broken situation worse.

4). Minmal changes are >>>> massive nerfs.

Those groundrules sound reasonable? Any other suggestions?

Anyway, some ideas, mix/match any of the ones below:
-make +hp/+AC/+mana items work less well for knight/hybrid classes. Code exists that trims down stats on items (recommended level) and checks for class (check to equip an item); modify this to either remove either HP, AC or mana from items when used by hybrid classes. Depending on implementation, may require item updates, which would need to be automated.

What this does: corrects the imbalance from itemization where the same item helps hybrid classes significantly more than parent class. Items which have +hp +ac +mana should have been hybrid ONLY, and rare. They weren't. This fixes it.

Note: IMHO this imbalance was perhaps the BIGGEST screw up in EQ over the last few years, and I'm quite unhappy to say I TOLD YOU SO VERANT!!! (see archives at eq stratics warrior board from, oh, about when knights first got "un-nerfed". YEARS ago. This was TOTALLY AVOIDABLE).

-make hybrid classes have channeling = to pure melee (i.e. none) Easy fix, code exists for this (i.e. war/rog/monk).

What this does: removes ability to reliably chain cast during melee, eroding the aggro lock ability which destabilizes the game.

-remove all aggro generation from cast spells which are resisted due to immunity. Coding change, unknown complexity but conceptually simple.

What this does: limits aggro generation by chain casting stuns or similar fully resisted spells. May (negatively) impact some weapon procs as well...

-set hybrid mana regen = to bards (i.e. one point + any FT). Code exists (i.e. bard regen).

What this does: Makes spell casting ability a UTILITY instead of a STAPLE of hybrid classes, making hybrids much more melee centric than currently, and thereby "balancing" them better vs. pure melee (as well as pure caster). Has minimal impact on "boss mob" fights, but has much greater impact on "grind" situations.

-Make warriors a sub-class of bards, able to use any self or group only (no AE) and voice-only (no instruments) songs, perhaps limited to only combat-oriented ones (i.e. no charms/mezzes/etc). Code exists, but *major* changes needed.

What this does: nothing for the other "pure melee" classes, but gives warriors some significant utility. And more buttons to push. ;)

-Make distance modifier/multiplier for aggro higher for warriors; coding change, unknown complexity.

What this does: makes holding aggro easier for warriors via proximity, offsetting some of the hybrid aggro lock advantage.

Any more ideas? There have been tons of threads about this, so I figured one more with more explicit guidelines/assumptions wouldn't hurt any...

If you happen to do slightly less/take slightly more dps while tanking because you can't cast while getting hit, it's more than compensated by the ability to generate a nice chunk of incoming aggro, and the ability to "snap aggro" mobs back if they get pingponged, let alone the other utility your spellbooks provide. You also DO lose the ability to permalock aggro, a Good Thing for the game imho. Tank or cast, pick *one*. You might also note that if you AREN'T getting hit, your spell casting ability wouldn't be impacted at all.

With regards to reducing mana regen:
All the sks and pallys I have asked (elemental level toons) run at 80-100% mana while MTing hard LDON missions. That means they are able to aggro lock + patch heal/debuff/etc. while tanking for about an hour WITHOUT ANY MED BREAKS, due to regular mana regen + FT + mindbuffs + focus items, without even having bard songs. Basically same issue as above- tank or cast, pick one- if you want to cast, then you should have to do it selectively OR take the downtime needed to med up. Effectively infinite mana pools (due to cheap spells + full caster level mana regen capability), plus the constant baseline ability to do non-trivial dps via melee, is broken.

Warriors are able to do everything we *need* to be able to do already; to me that says they don't need to be changed much. Perhaps reduce some annoyance issues, or add some "fun factor", but not more capability, unless they plan on re-tuning an awful lot of encounters...
