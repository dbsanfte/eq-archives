Yes EQw is against the EULA but it's about as offensive as jaywalking.
You'll find a quote from a recent fain faire stated by an SOE rep about this subject on the EQ Windows web site. He essentially said that, yes, it is technically illegal, but that it "isn't on our radar screen at this time."

It gives you no real time advantage in game, such as a speed hack or plat duping, or warping, instakilling mobs, etc. would, and it facilitates people paying them extra money to run multiple accounts on one machine, so I think they are highly unlikely to come after you for it.

To your original question, yes.... EQ Windows in combination with EQ Playnice, have been godsends for me. You can customise the keyboard settings to switch sessions, minimize, etc. however you like. And it's fast. And with Playnice running in the background, EQ eats up so few resources that you can run a few sessions on a modest EQ capable machine and still have browser, etc. running in the background. I've set it up so I can quickly Alt-Tab though whichever accounts I have running, or use another setting to just show the desktop if I want to go check something in the browser, look at e-mail etc.

It's worth a look.
