First, try this

/viewport reset

When I first installed, the bottom portion of my screen (behind the UI windows if you push F10) was black. So I did the reset, and now my entire screen is viewable.

Also, another thing I just noticed, is you said you changed the name in the ini file to SOES. I never had to do that, so that might be the prob. I would suggest deleting the SOES folder (that you should have) in your UIFiles folder in your EQ directory.

Unzip the SOES UI again. Make sure the path is c:/Program Files/Sony/Everquest/UiFiles/Soes (at least, that's what it is on my PC) And all the UI files should be in the Soes folder. Make sure there isn't another SOES folder inside it.

From there, look in the SOES folder for the UI file named UI_<character>_<Server>

From there, rename that file to UI_Kogur_Maelin

Then right click that file and copy. Paste it into your Sony/Everquest directory, overwriting whatever else was in there.

From there, don't go changing anything else in any INI file. Just log back into EQ. Hope this helps and I hope it works for ya.
