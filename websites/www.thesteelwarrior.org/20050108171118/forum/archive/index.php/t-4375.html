This was originally /Feedback'ed to Verant pre-velius in the hope of it being in Scars of Velius expansion.
And that was 2 years ago it was released.
I have updated it slightly, but it's basically the same.
I'm a bit stumped that the suggestions could still be considered useful.



I think the changes needed for warrior should be rather fundamental and something they start out with.
I have 3 moderate suggestions on improvements to warriors to make them more interesting to play with.

First - Anger Skill :

... A skill akin to Taunt, but instead it adds a fixed amount of hate, based off of Taunt skill.
... A good amount would be 3 times the Taunt skill, for a total of 3 x 230 = 690 hate.
... Warrior only skill. (knights can already do it with spells)
... Cannot fail or resist, range 50 or so.
... 30 second refresh. (approx the same time as between regular procs. except we can control this one)
...
... The refresh time on Anger will help keeping this from being overpowered, but will help with initial aggro.
... Also, since it's locked to the Taunt skill, which is level dependant, it will scale nicely as you level up.
...
... Available at Character creation.



Second - Taunt Refresh :

... For every 50 skillpoints in Taunt skill, 5% of Taunt refresh time gets shaved off.
... (Note: Anger skill would be unchanged)
... So you get :
...
... Skill 0 ----- 100% refresh time
... Skill 50 ---- 95% refresh time
... Skill 100 --- 90% refresh time
... Skill 150 --- 85% refresh time
... Skill 200 --- 80% refresh time
... Skill 230 --- 75% refresh time
...
... While skill 230 isn't strictly 50 points more than skill 200, it's our max cap, and it seems prober.
... Also, since it's locked to the Taunt skill, which is level dependant, it will scale nicely as you level up.
...
... Available at character creation.


Third - Basic Combat Stances :

... I would like to see 3 basic combat stances added, Aggresive, Neutral and Defensive.
... This would be 3 buttons, where only one could be "active" at any given time.
... If another is choosen, the previous will not be used.
... Like a radio decks toggle buttons.

Aggresive Stance
-This would increase all damage given by 10%, but also increase all damage taken by 10%
-You deal more out, but also take more.

Neutral Stance
-This Stance would give no positive or adverse effects and you would be just like we are now.
-Can use Disciplines in this stance.

Defensive Stance
-This stance would decrease all damage given by 10%, but also decrease all damage taken by 10%
-You deal less, but also take less.

... The Stances should be toggable while moving and in combat, though I think there should be a 5 second delay a Stance shift.
... This would be much like Spellcasters have a Cool Down time on spells before they can cast another.
... If you are ready for a stance Change, the change itself should however be instant though.
...
... In order to avoid overpowering Disciplines, the only stance to allow Disciplines should be the Neutral Stance.
... Stances would automatically scale with equipment, offensive/defensive skills and the environment (how hard MOBs hit).
...
... Available at character creation.



BTW, the reason have Taunt and Anger Seperate is because they are used differently.
Anger would be a preventive aggro method.
Taunt only works after things have already gone bad and someone else have aggro.
This is also the reasoning behind decreasing the Taunt skill, to help save someone if Taunt fails first time around.

Thoghts and comments on my suggestions are highly encouraged.
