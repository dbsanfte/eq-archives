You will read a lot of different thoughts on this but there are a few facts (at least as near as we can get to one in this game).

#1 the proc rate is the SAME FOR all but a few weapons, every one you listed has the SAME PROC RATE, you can take this to the bank.

#2 the proc rate is 2 per min main hand and 1 per min oiff hand, for both 1 hand and 2 hand weapons.

So with this info you can see that with the Copper/Blood you will get 3 procs per min and 2 with the GSoT.

In my exp both the GSoT and the Copper/Blood give about the same dps so if you want aggro there is only one choice.

If you take the proc rate to the max, anger augs and WA5 you get 9 and 6 procs a min, thats 3 more procs per min. Then when you are told to use the staff because of riposte ask to see a log parse on rispote damage. I have done the logs and I get 6-7 dps risposte in LDoN normal and 12-13 in LDoN hards and 25 on Agnarr using 1 handers.
