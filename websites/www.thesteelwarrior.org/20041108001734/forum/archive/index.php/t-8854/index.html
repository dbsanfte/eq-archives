Brael
Damage+(DB*(mod1/mod2)). The threads were linked elsewhere in this thread where all the work was done to discover it, although they don't quite cover everything since I didn't post everything I've done with this.

I'll break the formula down into 4 parts.
Damage - This is very simple, it's simply the damage on your weapon, bane damage is counted for this as is Elemental damage. When using Bash it's the AC on the shield and 25 for shields with 26+ AC.

Damage Bonus - This is quite literally the dmg bonus shown on your weapon. When it comes to offhand weapons the DB is 0. Bash has a DB of 0 as does Kick. AA's that increase Bash/Kick/Offhand damage while behaving like a DB for the dps portion have no effect on aggro, they're quite literally aggro free dps (just like crits are).

Mod1 - This is a class based aggro mod. For SK's it's a base of .84 for Warriors it's somewhere between 1.3 and 1.4, I haven't done enough work to get it down exactly (and I would suspect mods have a level adjustment too so it's very possible these will change at 70).

Mod2 - This has a base of 1, aggro mod effects will alter it. If you have a decrease of 20% to aggro (SCS) you will add to it so the new mod would be 1.2. If you have Echo of Anger (15% more) you would subtract making the new mod .85. If you have Echo of Anger and Voice of Thule it would be 27% more making the mod .73 (I think, stacking on it's never been proven conclusively or how it works).

This formula has worked perfectly every time it's been used so far so I'm pretty confident it's correct.

So for a SSoY in the hands of a Warrior without Echo of Anger it would be 8+(13(1.4/1)) and a BoC would be 15+(13(1.4/1)). I don't see where you're getting 27% from though, I'm seeing 64.97% more melee aggro over time and 16.25% more total aggro over time.
