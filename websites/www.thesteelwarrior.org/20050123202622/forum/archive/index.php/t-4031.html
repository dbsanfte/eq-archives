Errrm ...
Will just answer the question no coments :D

In a type 4 slot u can only add type 4 augments. Most 1h weapon have a slot 4.

In a type 8 slot u can only add type 8 augments. (dejavu?) Most 2h weapon have slot 8.

Now each piece of equipment can have up to 3 slots that are named slot 1,2 and 3 if present. Every "old" (aka pre LDoN) equipment will have only one slot while purchased LDoN equipment usually have 3.

Wow, that was some pretty fast misinformation. The types must match, for sure, but type 8 is Plane of Time items, NOT 2 handed weapons. All pre-time weapons, minus a few special items (primal for one), are type 4. The slots are just the specific spot for an augment of that type. IE a weapon could have slot 1 type 4 and slot 2 type 6. So basically, yes, you're screwed for the hate augment. You need to go to Everfrost and run missions there for anger 2.
