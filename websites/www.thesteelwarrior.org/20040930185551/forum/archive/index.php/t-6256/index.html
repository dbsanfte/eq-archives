It's possible to take a compiled map like an S3D file and extract the geometry, but if it follows the same rules as most other 3D engines that has no curved surfaces, it will be an ugly and unmanagable mess. If you ever mapped in Quake I/II you should know what I mean. However I doubt anyone has made a map extractor for EQ, let alone a 3rd party map editor... but once you have the raw data you should be able to load it into something like 3D Studio.
