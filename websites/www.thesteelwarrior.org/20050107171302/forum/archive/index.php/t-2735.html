Yes wolves respawn. I think timer is 12 minutes. No they do not run. Do not need snare.
Drop twisted longbow, diamonds, raw diamonds, blue diamonds and peridots. 61-62 parch also drop.

The birds are on some other screwy timer that i never bothered to figure out but its much longer than 12 min.

Zone into PoN, levi to hedge. Chat to Thelin to zone in

Working from memory here, from Thelin run forward and take the first left. Keep left and you will see a wolf. I call him mob 1. Kill him. If you have a CC/FD crew, you can go farther in this direction and kill a bunch of mobs. If not, then go back to Thelin.

From Thelin, take the first right. Then keep left – if you take second right you’ll wander around a bit but find nothing but dead ends. If memory serves there is a big straitish section. If you hunt around down there you’ll find three more wolf spawns within about a 1 minute run from each other.

I never found any other mobs down in this section. Probably with Clr/War duo just the 3 spawns will keep you busy in the 12 minute respawn window. The mobs are not side by side so you have to run from camp to camp, but they are all single pulls unless someone has triggered the maze event in which case some of them are doubles of a mob with the same model but diff name. This is quite rare that someone would leave the maze in this state but you should have a gate pot/mechanism that you use in this case. The mob you want is A Nightstalker. They all conned 60 to me although Allak lists them as 59-60.

Move slowly and make a spaghetti map with LoY until you have the spawn locations memorized. If you move too fast you might round a corner and get surprised.
