Not sure if this is the correct place for this or not, but here goes...

Using my guild leader's Planar Progression Flowchart, I created a little program that will allow you to keep track of your progression through PoP. It's fairly easy to use; all you have to do is click on the Tier you wish edit, then click the Plane you wish to edit, and then click the flags that you have completed in that zone. You can enter your character name in the field provided, save the file, and reload it at a later date if you wish.

The link is: Planar Progression Manager (http://mannhec.home.insightbb.com/PoPProgress.zip)

Feel free to download it and distribute it to friends and whatnot. If you have any questions, post here or email me at waeolwen@yahoo.com

If you are getting errors when attempting to run the program that say something about missing files or whatnot this is how ya fix it:

If the "missing file" is COMDLG32.OCX go to this site and follow the directions therein:
Missing COMDLG32.OCX File (http://www.ascentive.com/support/new/support_dll.phtml?dllname=COMDLG32.OCX)

For various other "missing files" download the VB6 Runtime files (the program was created in VB6). Go here and follow the directions:
Visual Basic 6 Runtime Files (http://www.microsoft.com/downloads/details.aspx?FamilyID=ba9d7924-4122-44af-8ab4-7c039d9bf629&DisplayLang=en)
