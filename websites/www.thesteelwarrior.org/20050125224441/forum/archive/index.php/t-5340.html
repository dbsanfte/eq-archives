01-22-2004, 05:24 PM
Miraculous Visions

Slot Description
3: Increase Attack Speed by 30%

Vallon's Quickening

Slot Description
2: Increase AGI by 52
3: Increase DEX by 33
4: Increase ATK by 41
5: Increase Attack Speed by 68%

Because VQ doesn't have haste in the same slot as MV, vallon doesn't overwrite the MV icon, but the higher hate % takes precedence, and if you have VQ your Miraculous Visions buff won't do anything.

VQ will also stack with shaman haste, but you still just have 68% haste.

Talisman of Alacrity

Slot Description
1: Increase Attack Speed by 50%
