If all mobs had equal AC, it would be possible to plot Modal as a function of ATK and level.

As is, there are 3 factors in determining modal value at any time, ATK, AC, and Level. ATK and level can be determined, but AC cannot.

Any work done to plot modal as a function of ATK would have to be done on a mob by mob basis.
