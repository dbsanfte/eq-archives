Coray
It's always been 72 people. And yes an instance is yours, no one else barring bugs can enter your instance. It takes a bit to get used to how things work so it varies by guild how long it takes them to progress.

No one knows of any limit to the number of total instances and I doubt there is one, it's tracked per person, not for raid/guild. Your last question is sort of weird and I think you need to understand how the instances work.

First they aren't instances, it's more like flags, and like I said its per person. PotimeB is set to the 'instance' of the first person to zone in. So if they have are flagged as having been there for phase 3 in the last week or so, when they zone in, it's setup so that after phase 1 is cleared, phase 2/3 will also be empty and you can run right in. Anyone who has been flagged as going farther than phase 3, say they were there for a Terris kill would not be able to zone in.

Each 'flag' resets in around 6 days, and they're tracked seperately, so it isn't the case that it all respawns every 6 days.

And because it isn't instanced, only one raid can be in timeb at a time.
