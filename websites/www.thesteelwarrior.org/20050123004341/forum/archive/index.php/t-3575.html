Originally posted by Brutul
Try a search, there's lots of threads on this topic. The short answer though, is the procs are independant, and you will get a lot more procs from a 2-proc weapon than you did before it was augmented.

Actually, that's a little misleading. The procs on a two-proc weapons aren't independent - it's not a 50/50 chance which proc you get.

Instead what happens when you swing is:

1) The game checks to see if the weapon's main proc goes off.

2) IF the main proc doesn't go off, then it checks to see if the augment proc goes off.


So the augment proc only goes off if your main proc missed. Therefore the augment proc only goes off a fraction of the time the main proc does. And the main proc itself is totally unaffected by whether or not you put an augment proc on.
