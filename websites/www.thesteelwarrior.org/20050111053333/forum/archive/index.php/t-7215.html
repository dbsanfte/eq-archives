my first guess would be to go to your options window and toggle the "auto-duck" feature. This feature was added very recently, and I could see how this might introduce problems on zoning in -- if the computer senses you need more room, it forces you into a ducked position, and this is probably running afoul of the buggy zone geometry/collision detection code they have in place right now.

If you have the feature turned "on" (this is the default when the added the feature I believe), turn it off. If you have it "off", you might try turning it on and see what happens.
