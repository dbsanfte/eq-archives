Screenshot:
http://home.covad.net/~sunwoo/EQ/Yun_UI/EQUI_CombatAbilityWnd_Screenshot.jpg

Download:
http://www.eqinterface.com/sdownload/fileinfo.php?s=&id=2890

This displays the Combat Abilities lined up horizontally and without the names truncated. This feels more natural to me and reminds me of playing with a spell bar. The "S" button gets hidden by the current active discipline, and if you click the current active discipline icon it deactivates and the "S" button reappears.

I've only tested this with the warrior ability tombs so far, I hope rogues and monks don't get super long named abilities hehe... :)

Note: Some people are worried about space. You can resize the window in the .xml so it only shows X number of buttons or no buttons at all. This value is way at the bottom of the file under Screen item = "CombatAbilityWnd" then Location then Y (lower this value to make the window shorter). You can turn EQ into windowed mode and use "/load (UIFOLDER) 1" to tweak this.
