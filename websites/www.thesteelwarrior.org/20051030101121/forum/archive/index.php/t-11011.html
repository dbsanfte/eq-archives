How do you enable port forwarding?

There is a guide on the blizzard forums some where. When I patched I clicked on the help link and found the guide. I only know how to do it for a router though.

Router:
1) Make is so your PC gets the same IP address from your router. Most routers you can reserve an address for a mac address.
2) Find the port forwarding option for your router ( this is all in the configuraiton menu, probably connected through an internet browers ) and set the ports that blizzard specifies to be forwarded to the IP address that you reserved for your PC.

That's it. These are pretty simplistic instructions but there is a full guide for lots of different specific routers and firewall options on the blizzard support site.
