You might want to go into your bios and and see what the AGP apariture size is set at. I'm willing to be that its still set to 32mb. From my experience, and what I've read this number should be set to 2x the video card ram. Which in your case would be 256mb. I've had my 64mb card set to 128mb in the bios since I got it and have had 0 problems. If this doesn't fix it, there could be a whole host of other things going on like a bad AGP slot or video card. Sometimes they can be shipped defectively.
