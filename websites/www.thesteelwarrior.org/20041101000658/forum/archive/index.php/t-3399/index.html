From Safehouse http://pub146.ezboard.com/fthesafehousethebackroomsecretsspoilers.showMessage?topicID=13555.topic :

Augment Levels:
Type 1 - Single stat enhancements (+5 intelligence, or whatnot)
Type 2 - Multiple stat enhancements (+15HP/Mana, +3 sv FR/MR/CR, etc.)
Type 3 - Worn effects (Haste, Regeneration, FT)
Type 4 - Weapon Procs
Type 5 - Single Weapon Damage additions (+2 Fire dmg, or +3 dmg)
Type 6 - Multiple Weapon Damage additions (+2 Dmg with + 2 cold dmg etc)
Type 8 - Time Loot (I've never seen a Type 8 augment) i heard but dunno if its true that this involves spell shield, accuracy etc and high level foci
