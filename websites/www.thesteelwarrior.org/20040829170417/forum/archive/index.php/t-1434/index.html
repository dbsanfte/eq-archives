Lorre
Data Transfers refers to the size of your data link from computer/laptop to hub. Im assuming you're getting a wireless router that acts as a hub right?

Via RJ-45 or CAT-5 cable, the data speed can go up to 100 base. Wireless max speed is only 10~11 base. That means that Kazaa files will download slower thru the wireless than the wired computer.

My suggestion is to get a router that does both. Linksys wireless routers run for about $70-$90 depending on where you are at. They have 4 RJ-45 connectors and you can connect a ton of wireless devices.

Wireless is great for if you have to move around a lot. I used my laptop in my kitchen, tv room, and even bathroom (yuk). If you go to college and you're college has a wireless internet system, you can use it almost anywhere. Downside of wireless is that you have to buy a wireless ethernet card that costs a lot more than a regular Cat-/RJ-45 ethernet card.

Anyways, try to get Cable Modem instead of DSL. DSL uplink is only modem speed.

Lorre
I have run EQ on wireless before. You can run it and its all good and dandy. Wireless is still about 5~10 times faster than 56k modem and people play at modem speed all the time. The good thing about wireless is range and not needing an actual cable. A good example is in your school courtyard in the middle of the field (if you're school has wireless lan).

Download speeds arent too hot though. You would need to buy a wireless card which will run you MORE than the freaking router. Thats the main problem with wireless, the cards are expensive.

Well Im assuming that you would only use wireless on a laptop (thus you can move around). You would want to use it on a desktop if you dont want to run wires or drill holes, but 8 computers is gonna kill you money wise.

You can connect hub/switches so that a cluster of computers can be connected vai cable and you only have to run one cable from the router to that switch. For example, from your DSL modem to your router you would run a cable from the router to you're room of computers. In that room you conect a Hub/switch that splits that internet conenction.

As for price, routers run from $70~$100 depending on what kind. 8 port hubs run about $30~$40. Wireless cards for PCMCIA laptop slots run $120~$140 EACH! Thats where wireless is gonna kill you (esp with 8 computers). Its cheaper to go cable.
