earrings - orc fangs, if not than invest in some black sapphire electrum ( i think thta's the name ) 2AC 35 hp's a pop..depending on prices on your server some tempest earrings might be valid ( the 7Ac 35 hp +12 FR CR MR ones )

neck - ry'gorr collar...if not that than shoot for a 14AC 35 hp neck ( fine heraldic or something similar ) .. good necks basically are no drop so don't go blowing a lot of coin here IMHO

face - crystal spider eyes .. if not than invest in a death mask of the eleysans..or a grobb hide mask

wrist - lose hero bracer, grab crustacean shell bracer

legs - lose them pants, grab some crustacean shell pants ( 16 sta is nice )

waist - get a chitinous belt or start saving for a heavy rubi belt


That should jack you up a bit in hp's; plus not losing any of the heavy AC from your cobaly pieces..worse case get some chitin armplates and use cobaly arms just for buffs.

Also, any visible armor pieces I suggested could be negated if you find some ornate mq's for sale ( for kizrak's ) but they might be pricey
