I have had similar problems. Mainly, I dual wield, and I want to switch to my 2 hander when I do an overpower etc, then switch back. After some research, I found out a few things.

When a macro is run, all of the commands are run at the same time. There is no way to have one command wait until after another command has finished. So when you run your macro, the system is trying to perform a shield bash at the same time that it is still swapping weapons. Then, if you try to include switching back, the system is trying to equip the weapons that you already have equipped at the same time that you are trying to equip the shield.

If you use cosmos, there is a command for macros to wait so many seconds before performing that command, but this is still very limited. This command only works with chat text or emotes, and will NOT work with casting, weapon equips, and I think items.

Somebody please correct me if I'm wrong, but this may be what is screwing with your macro.

I think the FAQ is at this link:
http://forums.worldofwarcraft.com/board.aspx?fn=wow-interface-customization

What you may have to do is have a macro to swap weapons:

/equip weapon_name
/equipoffhand shield_name

then manually execute shield bash

then have another macro to switch back to your 2 hander

You could use the WeaponSets (http://www.curse-gaming.com/mod.php?addid=111) AddOn, and use it's keybind to cycle through two sets.
