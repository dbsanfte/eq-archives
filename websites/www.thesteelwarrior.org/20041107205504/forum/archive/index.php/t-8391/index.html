A few notes before I get to the numbers. I will list the skill name and then nine or ten values after each. The first value represents the max skill you can have at 51st level. The second value represents the max skill you can have at 52nd and so on. All skills reach class max at 60th but there are a few that max at 59th and some even earlier. You of course can still increase skills that are not at the class max beyond 60th but if you max your skills at each level like I did you will not be gaining any additional skill-ups after 60th. Maybe the new expansion will allow higher skill caps. Here's hoping.

For me Hand to Hand (100), Disarm (200) and Swim (200) all max'd out at or before level 51. As I stated earlier I did not really start tracking it until then so I'm not certain at what level these three skills can be max'd. Also a few of the bind wound scores below are estimates (183, 187 and 195). It was the only skill that I struggled to max at each level before advancing.
