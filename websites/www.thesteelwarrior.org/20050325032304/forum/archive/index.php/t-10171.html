if you 9k with just conviction then with Wunshi/Brells/misc you right around 10k, which makes it doable for sure. Try someplace like Blind/Task with no lightning guys and i bet you be just fine.

Wanted to add this also; make sure that everyone in the group understands the situation and how to make it work. Make sure cleric isn't trying to start a CH when mob already on you and hope to land it. The healer is gonna have to start CH on inc and/or use fast heals whether they like it or not (some clerics will complain because their fast heals aren't as mana efficient as CH). If you have a bard pulling, bringing em in pre-slowed can be a big help. Some of these guys need to be debuffed before slows work. Make sure people are on it pronto. Any class that can heal can throw a small heal or two on you while the mob is being debuffed/slowed or if it looks like cleric heal wont land in time. I guess what im sayin is that you have what it takes to tank there as far as HP/AA/gear as long as rest of group is smart too. Get everyone else in the group doing all they can to make it work and you should have a good time there.
