/join serverwide.thesteelwarrior

To talk into the channel just type /1 then message if that is the only channel. Or you can make it your defualt chat channel by doing.

/channel chat serverwide.thesteelwarrior

This will make it so that you do not have to type /1 or the number of the channel. All you have to type a regular message and it will post it in the channel.
