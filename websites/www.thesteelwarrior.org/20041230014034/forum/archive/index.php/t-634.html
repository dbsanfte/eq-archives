Donnel is correct, EQ Companion purposely leaves the old fights in when you open a new log file. The reason for this is that you can open multiple log files and use the 'combine fights' feature to combine many fights agains the same mob to get more accurate average DPS readings.

I have, however, added a new menu option to the upcoming 0.8 version which clears all the fights if you want to start from scratch so you do not have to quit and restart the program. 0.8 is not released yet, though...

I'm curious, though. Why would you want to clear the old fights? Is it because you are looking at the stats in the 'Stats' page and when you open multiple log files, the stats are all messed up? If this is the reason, you really should not bother with those stats. To get better stats, just select all the fights and combine them. You will get a lot more accurate stats that way. Even better yet, combine fights that are against comparable mobs to get even more accurate stats.

Donnel is correct on the other problem as well (thanks!). If you run the installer program, it will install all the required Visual Basic componentd, like this MS Flex Grid control that it was complaining about. The installer installs an older version (0.6), so you should download the ~200kb ZIP file for 0.71 as well and extract that on top of the one the installer installed executable and you should be good to go with the latest version.
