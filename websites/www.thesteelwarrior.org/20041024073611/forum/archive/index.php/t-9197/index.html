First, none of the 1.5 quests (all epic for that matter) are MQ'ble. Primarily this is because you have several pieces cannot be looted unless you have reached that point in the quest. The other reason is that Korbuk won't talk to other classes and returns the items they turn in. so you have to loot all items.

Second, if you are unable to zone into Vexxd you need to get someone who can to get the event. They can piggy back you into the event.

However, before that you have to realize that the quest code is set up that you do everything in order. I assume because you haven't done step 5&6 you don't need to go to Vxxed because Blackfall won't be up even if your group could zone in. And besides, if he did spawn and you killed him you couldn't loot the item.

Third, you don't need to be level 70 to get into MPG. The zone level restriction only applies to Riftseekers zone and that is level 69 so far in this expansion.

They, SOE, set this quest up pretty well (except for somethings I'll discuss another time). You have to do everthing in order. So don't try to take short cuts.
