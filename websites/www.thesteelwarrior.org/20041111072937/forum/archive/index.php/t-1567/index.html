Zukobim
The problem with warriors vs knights as I see it, is the warriors lack of a good fast agro lock.
This new skill might provide just that without making the game trivial.

The new skill could be called "Charge!".

Charge! would work as a two component spell.

Component 1: root, that is unbreakable by melee damage.
Component 2: spellshield.
Duration: 1 tick (6 secs)
Resists: very low, but not entirely unresistable
Reuse: 30 secs

The effect would make debuffing of a mob possible on the way in possible, because the warrior would be able to "root" the mob for 1 tick with Charge!
Once the mob is rooted with Charge! it is not possible to use spells on it, because the spells would be absorbed by the spellshield and thereby using it.
Once the spellshield component is used up it will cause the effect to wear off.
A proc would function like a spell.

This will mean a warrior can lock a target for 6 secs and build agro with the side effect that all casters would be "disabled" for the same period of time.
The risk is the effect being cancelled out by a weapon proc or a trigger-happy caster. The other classes would actually have to get skills in order to function in groups and not just blast away.

The skill would be pretty much useles once the fight is over the initial period, because nuke, procs, taps, dots would all break it.

The code is already there, so it should not be hard to implement it.
