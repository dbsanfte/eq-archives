Quato
I use a ED Primary and SEM secondary. Both have 125 hp Ykesha augs. This set up is nice for damage and decent for aggro. It is no raid MT set up but the SEM is a nice secondary weapon which will do fine holding aggro for most group situations. I chose the Ykesha augs over anger augs because unless I am MT for a raid target I don't want to switch out my weapons. The rune II is a handy effect but I don't think it generates much if any aggro at all. Still as an offhand weapon I am sure you will be happy with it for quite some time.
