There's no big secret. Have decent DPS, don't ping-pong, don't waste any time between mobs, and you'll be ok.

The last trial I did was a 65 SK tanking, a 65 monk and a 61 ragebringer rogue for DPS, an enchanter doing CC, a 55 cleric (my bot) healing, and a 54 cleric nuking. Hardly a super-group. The monk was the only one in the group with uber (had the 2-hander from Aerin Darr so it's not like he's using time stuff) gear. Right before that we did it with my warrior tanking, the same enchanter and monk, a ranger with epics, the 54 cleric healing, and a 57 druid doing DS/healing. Both groups won on the first try.

It's all about killing as fast as you can, but don't think you can't win without three 65 wizards or something. Switching targets quickly is really important if you are the tank. I usually target and start hitting the next mob when the other mob is at 5% so the DPS can assist the second the other mob dies. Make sure your cleric/enchanter stay close to the group when they cast, but get back so they don't get mobbed when the waves spawn.

Good luck.

Way I have had most success is with AE groups, but have successfuly done it both ways. The group you list should have no problem at all getting thru DPS wise. A few things to help, some already mentioned.

1: Don't have chanter mez, have him chain stun. Have everyone in middle so mobs are all nicely compact, and stun, stun, stun.

2: DS potion, plus any other DS you can get. Make sure the chanter uses DS potion as well, as mobs resist the stuns alot and he is going to get hit more than you. You mention a mage, so getting DS should be NP.

3: Fast targeting. I rebound the f8(target closest NPC) key to the delete key on my keyboard, found it handier there for me. Getting next target quickly is important, so everyone can assist and start doing damage fast(When doing trials with my mage, I am pushing /assist as mob drops, soon as a target is up, pressing /pet attack...my pet does from 70 to 90 DPS, so getting it on target fast helps). Unless you are going to be really close on finishing a mob, go ahead and change targets a little early and let pet(s) and other melee finish mob.

4: With the group you listed, I would have mage start out each wave with a rain spell(hits all 4 targets, very efficient, tho resist rate sucks). If there are rocks, you can have mage banish one after casting rain, just make sure not to banish mob you are fighting, that is a waste. If not banishing, have him single target nuke as needed to finish off mobs(done trials alot with my mage, am very good at them). Mage/wizzy combo's rule for trials btw, both casting a rain to start, wizzy does another AE while mage switches to single target, will be scarey how fast mobs drop.

5: When Prime Executioner spawns, engage and /disc Defensive. With chanter AE stunning, your cleric should have plenty of mana left to get you thru.

6: Almost forgot, since you have a mage, make sure everyone has a rod with at least 2 charges at start of trial, casters use first charge at end of first wave, and again when Prime Executioer spawns.

There are a lot of different ways to win trials, the above ideas are not the only way, but I have had good luck with them. I think if you are tanking trials, the biggest thing you need is practice. I went 0 for 30some when I was trying to get flagged(back when you needed flag to get to PoV/PoS) in pickup groups. I noticed as I got faster at targetting and more comfortable with trials, groups where doing much better. I eventually did pass in a pickup group, and if it can be done with a pickup group, a group used to working together should be able to get thru it.
