Brutul
Moving the chat windows is a little cumbersome, but you only have to do it once. If you really need to move your windows a lot for some reason, there's instructions on the installation page on sars site to turn the title bars back on.

Somewhere near the bottom of the very last cluster of code and above all of the <Pieces> entries is a line that says <Titlebar>false</Titlebar>. Change that to <Titlebar>true</Titlebar>, save the file, and then press your reset hotkey and voila! If the window is not resizable, you may notice some clipping at the bottom. To correct this you will need to locate the <Size> portion of the same cluster of text at the bottom of the page for that window and add 21 to the <CY>value</CY>.
