I have the main chat window and 4 others

Main chat window, bottom of screen, slightly to the right of center with any conversation going into.

Immediately to the right of the main chat window I have another window in which all spell casting information goes. I have this one set to near the same opacity as the main chat so I can see when mobs are casting (and either ghetto bash or just kick - now that i'm over 55 and kick can stun) to interrupt them

In the upper right corner with opacity set to 0 (so I don't see the box) I have an abbreviated window showing the monster hitting meet (shows about 8 lines worth of hits, so I know what is hitting me, etc.)

To the right of the me taking hits box I have a very narrow, 0 opacity chat box (so I don't see the chat box iteself, just the numbers) of all my hits. This is pared down to just the numbers. Sure I don't see when I miss, but that is pretty obvious to tell by watching as I do swing animations but no hit numbers show up.

In the upper right, again with opacity set to 0 so I don't see the actual chat box, I have spell damage and damage shield numbers. This way I can see if the lazy ass wizard is actually nuking, which of my damage procs went off, how much my group mate's procs are doing or when the damage shield is off (or how much it is hitting for).

This generally keeps my chat window cleaned up so I can read what is going on.

This is all on the standard EQ interface. I couldn't imagine having everything in one chat box. The spamola would be HELL!!!
