Ezedriel7th Ham
We all know that one of the biggest arguementsabout class balance is itemization and the difficulties involved in going back, by hand, and tweaking every single drop of every single encounter, something SoE is far from willing to do.

We also know that itemization is a major contributor to the off-shoot of class balance. If a paladin can derive exactly the same benefits and more from a piece of gear than a warrior then he is that much more powerful.

So my idea is this:

What about Recommended Class?

Basically code it the same as Recommended Level, which is extremely easy to code. It would be something like:

If class does not = warrior, then -X Hit points, -Y Armor class, -Z STA

and so on. They can add this to a specific series of equipment that seem to be causing the most grief about class balance, it's not that hard to code, and paladins would no longer acquire greater useage from gear.
