In the default options menu there is a command to adjust mouse sensitivity, but it isn't what Kavak is more than likely needing to adjust.

In the eqclient.ini file there is this line:

MousePointerSpeedMod=1

make it = MousePointerSpeedMod=2

This should make it move alot faster.
