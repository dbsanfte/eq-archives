There is agro relative to how much damage you could have done, not including crits, so a 10/40 weapon won't generate as much agro as a 30/40 weapon. This is generally referred to as DPS agro.

Whether the Sinister Strikes bonus is considered for agro generation or not is unknown, but my guess is that it would give a tiny bit of additional agro.

Remember, we aren't the only class that gets Sinister Strikes and the rogues would hate it if it gave a lot more agro.

Hmm, now that I think about it, you should probably check with the rogues since that's probably the first GoD AA they bought.
