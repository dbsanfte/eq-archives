rogan
Some tests i did last week...

----------------------------------
Allegedly avoidance didn't stack, which i found weird, so i decided to do my own test.

Basically i spent several nights with 5 or 6 Pickclaw Foeslicers hitting me in a corner, with my back to them ; I then only considered their regular attack (HIT, not BASH or KICK). I did 3 sets of tests, one with 0 avoidance, 1 with 25 avoidance, and 1 with 55 avoidance.
Here are the results:

AVOIDANCE 0 - 1633 AC STATS
hits: 22407 - misses: 94732 - damage: 158154
connecting blows ratio: 19.12 % - error radius: 0.292 %
18.83 % < cbr < 19.42 % with 95% certainty
average dmg / connecting blow: 7.058

AVOIDANCE 25 - 1630 AC STATS
hits: 26034 - misses: 115644 - damage: 185934
connecting blows ratio: 18.37 % - error radius: 0.265 %
18.10 % < cbr < 18.64 % with 95% certainty
average dmg / connecting blow: 7.141

AVOIDANCE 55 - 1635 ACSTATS
hits: 10962 - misses: 51128 - damage: 81479
connecting blows ratio: 17.65 % - error radius: 0.401 %
17.25 % < cbr < 18.05 % with 95% certainty
average dmg / connecting blow: 7.432

Conclusion:
Avoidance stacks with about 97% certainty

Method criticism:
- not sure of goblin levels, 3 level range and multiple goblins involved
- not sure avoidance works for greenie mobs the same way as for challenging mobs
- i didn't compensate for the visible AC increase from avoidance ... which results in the average hit getting higher with increasing avoidance (total AC being about the same)
