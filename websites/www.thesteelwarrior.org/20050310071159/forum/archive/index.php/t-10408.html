If Alt-Tab isn't working for you, you must be using WinXP. Somehow, SoE greased Microsoft's palms enough to disable Alt-Tab. Luckily, you can override the default settings. Go to the Microsoft web site and search for the Windows Compatability Toolkit. It will let you turn on Alt-Tab for EQ.

Alternatively, you can use third party applications such as WinEQ and EQWindows. I think both let you set up a hot key to switch between two or more instances of EQ.

On the left window, click on "Applications"
On the right window, find the "Everquest" entry , click on it
