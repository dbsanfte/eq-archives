Bridgecrusher
I did some analysis on the magelo formula and came up with changes that seem to match a majority of the posts in this thread.

Here is the basic formula from the magelo laboratory:

HP = TRUNC [ 5 x ND ] + TRUNC [ Lvl x LM x ND ] + TRUNC [ TRUNC [ TRUNC [ STA - Post-255 ] x Lvl x LM / 300 ] x ND ]

TRUNC means Truncation occurs for the term between [ ] that follows it.

For the term Post-255 we have

Post-255 = MAX [ 0, ( (STA-255)/2 ) ]

So Post-255 will only be non-zero when STA exceeds 255.

ND is the multiplier bonus they use (Natural Durability, Physical Enhancement, etc ...). LM is a class and level dependent multiplier as indicated below:

Level 1-19: Levelmultiplier = 22
Level 20-29: Levelmultiplier = 23
Level 30-39: Levelmultiplier = 25
Level 40-52: Levelmultiplier = 27
Level 53-56: Levelmultiplier = 28
Level 57-59: Levelmultiplier = 29
Level 60-65: Levelmultiplier = 30


If you convert that formula to the following:

HP = TRUNC [ 5 x ND ] + TRUNC [ Lvl x LM x ND ] + TRUNC [ TRUNC [ TRUNC [ STA - Post-255 ] x Lvl x LM / DIV ] x ND ]

where LM is defined as

Level 1-19: Levelmultiplier = 22
Level 20-29: Levelmultiplier = 23
Level 30-39: Levelmultiplier = 25
Level 40-52: Levelmultiplier = 27
Level 53-56: Levelmultiplier = 28
Level 57-59: Levelmultiplier = 29
Level 60-69: Levelmultiplier = 30
Level 70: Levelmultiplier = 31

and DIV is defined as

Level 1-69: DIV = 300
Level 70: DIV = 297

You end up with values very close to those reported in here. Try it out on an excel spreadsheet and you should see comparable values.

Example 1, going from level 69 to 70 with varying values of chaotic potential gives:
CP0 = 270 hp
CP1 = 281 hp
CP2 = 275 hp
CP3 = 282 hp
CP4 = 275 hp

Example 2, going from level 65 to 70 with no points in chaotic potential gives:
65->66: 83 hp
66->67: 90 hp
67->68: 84 hp
68->69: 94 hp
69->70: 270 hp

You might test other values to see, but it seems pretty accurate.
