Raaj
Please select no more than FIVE options. They would all be nice - but we need some way to narrow things down.

Remember that a few of these are fairly broad. "New AA skills" for instance. Should it happen to get the most votes, then a discussion can be started about what the specific aa skills should be.

Don't take this as a final draft. It's easier to work towards a solution if we narrow it down bit by bit.

Improve Taunt - This would include any changes to Taunt or AE taunt. Including: pass/fail messages, increased effectiveness,shorter re-use, larger ranges, etc.

Increased innate Hp/AC/Mitigation/Avoidance - Includes changes to defensive tables. Assumes these changes would not be gear dependant, but a class change.

Itemization - Includes anything to do with items. More weapons/armor, different stats, better placement of items, etc.

Discipline Changes - Includes releasing timers, Giving us new disciplines, lowering/increasing timers, or totally reworking existing disciplines.

Using 2hs with a shield - Giving warriors the ability to do this. Via aa, a new skill, or any means.

Melee Focus Effects - Deals with the introduction of effects which "glow" like caster focus effects, yet give positive effects to melee classes.

+Hate on Hits / Crits / Procs - Via AA or not, increased hate on crits procs or hits for warriors.

New Attacks - Includes any new attacks (much like the arsenal of monk attacks) which have different effects on the mob being attacked. Via AA or not.

Change buffs to increase %'s instead of hard caps - Gives warriors an advantage with buffs since they have a higher base hp than other classes.

Permanent/Semi-permanent Shield/Protect skill - The ability to permanently shield a person, range dependant.

Increased "stacking" of plate tanks - Something to make plate tanks stack better in groups as well as on raids.

Skills/Auras (via AA or not) which increase an entire groups effectiveness - Think about the warcries from diablo 2. Very similar.

New AA skills - Self Explanatory.

Increase proximity agro for warriors - Allow warriors to gain more agro from being close to a mob/

New Class only Skills - Summon, Double bash, any new ability given to our class.

Armor / Weapon specialization - The ability to increase skill caps on certain weapon skills, or to better utilize your armor dependant on specialization.

Increase Skillcaps on important warrior skills - yeah.
