Everytime you visit the website IE will check the page you look at against the old one it got saved in its cache.
Clear the cache and it should stop bugging you and at the same time lower the cache size to something more real.
If you are using a fast connection, like ADSL or better, then set it to a number around 15-20MB.
If you are using a dialup, like Modem or ISDN, then use a number around 50MB.
Too high a setting (usually the case on bigger harddisks) makes windows search through several thousand files each time you visit a page.


To clear the cache: Tools -> Internet Options... -> Delete Files...
To set the cache size: Tools -> Internet Options... -> Settings... -> Enter 20MB in input box
