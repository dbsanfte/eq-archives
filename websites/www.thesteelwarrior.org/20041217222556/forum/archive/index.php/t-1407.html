Ok so this is not even remotely original but here goes anyway


Skill name: Grapple

Effect: Root2 ( works on immune to run speed changes mobs: see below ) while grapple skill is being used.

Penalty: User cannot attack and also has run speed changed to 0. User cannot use the /disc command or have a /disc up while using grapple ( see /shield )


Restrictions: Cannot work against targets of phenomenal strength or size ( I.E. dragons, gods and other boss type mobs ).


Reasoning to allow root on otherwise immune mobs: The immune tag is to prevent people from attacking said mobs with lessened risk of getting damage from the mob and seeing as how a rooted mob will attack whatever is in melee range, the restriction of having a PC in melee range negates the use of root to completely avoid mob damage.


You may now pick it apart and do whatever other nasty things you like too it
