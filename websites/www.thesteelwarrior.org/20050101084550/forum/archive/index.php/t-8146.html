usually skill and disc timer are linked to the local machine clock. thus having a difference between server and client clock value (not time) make bad result in timers like having to wait a week before using def, or 10 years before looting a corpse, ...
so if you find the warrior you have grouped with ask him if his time is correct or if it has a difference with his local time (usually it's few minutes), if yes correct it and it should be good.
