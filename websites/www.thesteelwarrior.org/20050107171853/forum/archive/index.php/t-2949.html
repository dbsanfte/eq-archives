Sassinak is right, you need to parse both weapons main hand to have a more accurate comparison. In this case, this is the test I would do -

Parse 1 hour with BoC main, then switch weapons and parse 1 hour with GBoS main.

Then I would count 2 things - the number of total procs with each weapon mainhand and the number of total swings for each weapon mainhand. This will not only give us a raw # of procs to compare, but also a # of procs/swing for each weapon.

You will need to make sure that you have the same buffs for the entire duration of both tests. I would recommend NOT using an avatar proc because it is one more variable that can skew results. If you must have avatar, get it from a shaman and don't let it fade.
