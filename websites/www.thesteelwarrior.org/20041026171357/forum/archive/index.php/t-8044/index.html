It used to be in order to corner a mob you'd need a perfect corner and a big enough difference in mob size vs your size to do it, since the mob would warp past you and still hit you if not. Now that they have some real collision, you can in theory "corner tank" in any small little notch in the wall.

We used to set up down the ramps from the arena though, gave the pullers more room to work with.
