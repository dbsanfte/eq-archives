OK, I've posted a couple of times on the SW site. I had an epiphany last night. How about an ability/skill/discipline (however it comes) that a warrior could turn on and off? Call it a nastier fighting technique (which paladins would refrain from using) where his objective is to cause more pain than damage by hitting the mob in more sensitive areas, thus getting the mobs attention? It would effectively halve the warriors damage and would drain double the normal amount of stamina while it is on (and would turn off if there is no stamina left), but double the hate generated with each blow (or add a fixed +hate amount), or it would allow for the warrior to generate procs equivalent to an Enraging Blow with whatever weapon(s) they are using (or if they are using EB weapons, it would add an extra chance for an EB proc). Turning it on or off would allow the warrior to control when he is generating agro, and halving melee dps and doubling stamina drain while it is on would reflect the amount of focus a warrior is spending on causing more pain than damage. To make this less than overpowering, make certain mobs immune to this effect so a warrior has to rely on normal agro generation techniques so as not to trivialize the raiding portion of the game.

EDIT: Just wanted to add, if this will be disc based, the reuse timer is only a trivial count (like 6 seconds) and use length is stamina dependent, but if any other disc is used (ie, defensive) then this fighting style cannot be used again until the timer is up and a new disc can be used.
