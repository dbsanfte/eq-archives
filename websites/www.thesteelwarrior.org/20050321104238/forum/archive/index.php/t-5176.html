I made this (http://pub146.ezboard.com/fthesafehouseloungegeneraldiscussion.showMessage?topicID=29326.topic) post on The Safehouse regarding an idea for changing EQ's discipline system.

Here is the thread quoted, let me know what you think.

Proposal for discipline system:
=======================

*Remove duration and re-use limits from all disciplines.

*Make all disciplines stack. (with penalties, see below)

*Add new low-cost +ATK disc

*Add new high-cost +ATK disc for burst DPS (much more +ATK)

*END use scales with player actions (less used while sitting, more while in combat)

*Special attacks/abilities (i.e. Flying Kick, Bash, Backstab) instantly remove an amount of END based on a fixed number (unique per ability) times END use per second at the time that ability was activated.


Ideas to change current discs:
=======================
note to SteelWarrior: my primary character is a Rogue

*Focused/Healing Will should work in-combat.

*Remove Twisted Chance's "deadeye" component and raise crit rate to 100%

*Discipilnes should not break hide/sneak.


Conclusion / Summary of ideas:
=======================
*Pure Melee will be able to 'throttle' DPS/tanking, reducing dependancy on high-uptime scenarios

*Pure Melee have true 'combat' advantages over other classes.

*Melee power and these new abilities can be balanced very easily by adjusting END per second costs of discs.

*Disc stacking can provide impressive 'variable' power to Pure Melees, while easily balanced via END/sec cost penalties which increase per disc active beyond 1.
Thanks for reading! Please consider /feedback if you like these ideas.

I think the key to balancing it is in the exponentially-increasing END cost penalty for stacking multiple discs. For example... let's say a Warrior can use "defensive" for 5mins before he is OOE and he can use "fellstrike" for 45seconds. If there were no penalty, stacking these 2 disciplines would probably burn up all of your END in about 30-35 seconds. WITH a penalty, it would burn all of it up in ~20 seconds.

So sure, you can stack aggressive, riposte and fellstrike all at once, but you're going to burn all of your END in just a few seconds... So yes, you have the ability to be a "hero" and really go into an ass-kicking berzerker rage, and you can do so without being ridiculously overpowered, as the short duration limits the "overall" impact on the group/raid.
