Uh, you said exactly what I said :) The only difference is if they do all this in integer math, and round the DI calc down before multiplying by the interval part. Then you'd see a difference of up to whatever the random interval value was between the two.

But, with floating point math,

DB + 20 * (DI * .95) is the same as
DB + 20 * (.95 * DI) which is the same as
DB + (20 * .95) * DI since multiplication is both commutative and associative, and since 20 * .95 is 19, all that equals

DB + 19DI

The same holds for the other 19 values.
