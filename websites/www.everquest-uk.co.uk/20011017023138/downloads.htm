Utilities
 

Evermon is one of the most useful programs for Everquest. It is developed by Hydra Computers to monitor server status.
Everguild (Note link same as above as same launcher program) allows guild masters more managment scope of their guild. It is developed by Hydra Computers to monitor server status.

EQu is THE best ini file editor Ive seen to date. It allows you to change all the basic settings within EQ. Text colours, screen layout, defaults etc. In addition there is a log file parser and a server monitor program. Get it now.
