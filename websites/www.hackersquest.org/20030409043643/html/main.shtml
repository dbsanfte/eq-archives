EthernalQuest 0.6.1 Alpha is out! :: 7.3.2002
Heres the changelog:

Version 0.6.1 Alpha

* bug fix: client was crashing when dealing with packs and books
* bug fix: client was crashing when mobs/players deleted from zone
* bug fix: client was disconnected after you stay in one zone for exteding period of time
* mobs will now follow you when attacked
* /consider should work properly
* new # commands
#spawn findskill [ skillPattern ] - list all skills that match pattern
#spawn findlanguage [ laguagePattern ] - list all languages that match pattern
#spawn skill [skillNameOrIndex [, skillValue]] - show or change skill values (player only)
#spawn language [languageNameOrIndex [, languageValue]] - show or change language abilities (player only)
* guildmasters will function properly
* you will be able to hit for more then 1 hp. Formula is a bit more real then one posted in forum:
STR / 5 + level + [ 0 .. level ] + 2. You will hit for about 80 to 140 at level 60.
* you can now scribe / memorize and cast spells. Do not expect any server side effects.
All client side effects should work fine.
* it should be easier to attack something
* Spell viewer/editor is included as part of EMU
* EqDb version 0.3.0003 included

Get the EthernalQuest / Src packages from the filetable above!

Thanks Zordon, for another high quality release :)
