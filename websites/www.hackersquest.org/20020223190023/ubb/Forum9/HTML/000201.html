This tool creates a window that sits in the bottom right corner of the screen, and display all relevant details on the missions given by a mission terminal (location, cash, item reward, item to be found), without having to manually shift-clicking each mission and scrolling to the bottom of the description to find out what you're getting.

New in 1.3 ======
* Displays all the reward item icons. (The background color can be modified via the new 'Options' panel to suit your personal taste.)
* Displays the item to be found for item retrieval missions. (Note: It is not possible to display it's QL or Icon, as these details are not passed to the AO client.)
* Added a 'Description' button to display the mission assignment without having to use the in-game interface.
* Revamped the mission packet parsing code; it should be a lot more reliable now.
* Fixed window resizing. It can be enlarged horizontally to allow more room to display mission details.
* Fixed numerous crash bugs.
* Unnamed network adapters will no longer show up blank in list, leading to confusion. The title "Unnamed Adapter" will be shown instead.
* Ported the remainder of VB code to VC++. This makes the tool a small download, and runs a lot faster.
* Incorporated debug logging into the release code.

It is available at http://muzzleflash.da.ru

--------------------------------------------

Note: it has only been tested on my computer... Please reply if it no longer works for you.

You can help me out in finding problems by running the tool with the command line option '/debug:all', and sending me the debug files "Debug_AllPackets.DAT" and "Debug_MissionData.DAT" to muzzleflash@emaildownunder.com
