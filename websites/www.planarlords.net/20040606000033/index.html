News: Website Changes

I made a few changes to the website today.

Now there are usergroup abilities throughout the website and not just in the forums area. This means that unregistered users and people that are registered but are not guild members will have limited access to the system.

The reason for this implementation is to keep Magelo profile information resonably personal and to allow a bit more control over the system in general.

I also reactivated the Guild Chat module. I don't know how many people will use it, but the function is there should you ever want to communicate with each other while out of game.

A calendar system will be implemented before too long so raid and meeting days can be posted. I put a poll up in the forums section concerning raid/meeting days. Please check it out and leave your thoughts on the matter.
