- Erstes Einloggen -

Wenn Sie EverQuest zum ersten Mal mit einem Charakter betreten, werden Sie eine Vielzahl von Fenstern sehen.

In der oberen linken Ecke befindet sich das Alarm-Fenster. Dieses Fenster ist nichts anderes als ein Timer, der Ihnen helfen soll, die Zeit, die Sie in EverQuest verbringen, im Auge zu behalten. Sie können eine Zeit festlegen, nach welcher ein akustischer Alarm sowie ein Fenster ausgelöst werden, die Sie darauf hinweisen, dass die angegebene Zeit verstrichen ist. Durch anwählen der Box vor "Dieses Fenster nicht mehr anzeigen" können Sie dieses Fenster fürderhin unterbinden.

In der oberen rechten Ecke finden Sie den Tip des Tages. Durch aktivieren der Box vor "Keine Tips zeigen" wird der Tip des Tages beim nächsten Einloggen nicht mehr angezeigt.

Die EverQuest InGame Hilfe ist der erste Anlaufpunkt, den Sie benutzen sollten, falls Sie im Spiel auf Probleme stoßen sollten. Lesen Sie sich die InGame Hilfe beim ersten Einsteigen in EverQuest aufmerksam durch, Sie finden dort viele Hinweise zur Benutzung und Steuerung des Spiels. Sie können die Hilfe, nachdem Sie sie geschlossen haben, jederzeit wieder aufrufen, indem Sie im Window-Selector am oberen Rand des Bildschirms, auf das " ? " Icon klicken.

Wenn sie das erste Mal Ihr Inventar öffnen, wird ein "Inventory Tip" angezeigt. Diese Tips enthalten interessante Informationen rund um Ihr Inventar-Fenster.

Wenn Sie sich das erste mal setzen, öffnet sich das "Sitting Tip Window", welches Näheres über die Vorteile des Sitzens und die damit verbundenen Gefahren enthält.

Wenn Sie mittels " C ", "/einschätzen" oder Druck auf die rechte Maustaste zum ersten Mal einen NSC einschätzen, öffnet sich das "Considering Tip Window". Dieses Fenster enthält Informationen über das Einschätzen-System und sollte aufmerksam gelesen werden.

Wenn Sie das erste Mal bei einem Ihrer Gildenmeister sind, um Fertigkeiten zu trainieren, öffnet sich automatisch das "Trainer Tip Window". Hier finden Sie Informationen über das Trainieren von Fertigkeiten sowie.
