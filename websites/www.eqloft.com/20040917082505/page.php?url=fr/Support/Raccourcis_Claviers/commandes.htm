/chat set (canaux) /dial permet d'accéder à une série de canaux de communication
<canaux> se compose d'une suite de définitions de canaux: <canal> [:mdp], <canal> [:mdp], ...
<canal> est le nom ou le numéro d'un canal
<mdp> est le nom (optionnel) du mot de passe du canal
exemple:
/chat set naggychefs:secretofficiers, naggygroupes:secretchefsgroupe, naggyraid
     création de 3 canaux:
     - 1 canal naggychefs pour les officiers avec mot de passe secretofficiers
     - 1 canal naggygroupes pour les chefs de groupe avec mot de passe secretchefsgroupe
     - 1 canal naggyraid pour les membres du raid, sans mot de passe

/hitsmode <my|me|other> <on|off|m2|m3|m4|m5> <X> <Y> <>NbLignes> <TailleCar> <Titre> <Fond> <Largeur> <Forcé>
<my> mes attaques, <me> mes coups reçus, <other> attaquesdes autres
<off> supprime l'affichage de tous les messages de combat, y compris dans les fenêtres de chat normales
<on> (par défaut) affiche les messages de combat, comme dans les fenêtres de chat normales (par défaut)
<M2> toutes informations de combat
<M3> forme réduite, indique: type de dommages, dommages de la cible, grandeur des dommages
<M4> uniquement la quantité de dégats infligés
<M5> uniquement "mes attaques", type et quantité de dégâts infligés

le choix de M2 à M5 entraine que:
     <X> <Y> sont les coordonnées de la fenêtre invoquée
     <NbLignes> est le nombre de lignes de la fenêtre (entre 1 et 10)
     <TailleCar> est la taille des caractères du texte (entre 0 et 2)
     <Titre> la valeur est soit "1" soit "0", et indique si le titre de fenêtre est affiché
     <Fond> la valeur est soit "1" soit "0", et indique si le fond de fenêtre est affiché
     <Largeur> est la largeur de la fenêtre
     <Forcé> la valeur est soit "1" soit "0", 1 force l'affichage, même si la fenêtre est vide

/who all friend : liste vos amis en ligne
