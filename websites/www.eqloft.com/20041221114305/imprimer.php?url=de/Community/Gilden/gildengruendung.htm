Gilden-Informationen

Alles, was Sie wissen müssen, um eine Gilde ins Leben zu rufen, einer Gilde beizutreten oder eine Gilde zu leiten, finden Sie in den folgenden Abschnitten.

Gründung einer Gilde

Wenn Sie der Meinung sind, dass keine der bestehenden Gilden dem entspricht was Sie suchen, können Sie Ihre eigene Gilde gründen. Dazu müssen Sie einen geeigneten Gildennamen aussuchen.

Bitte beachten Sie dabei neben den normalen Namensgebungsrichtlinien folgende Regeln:

  • Der Name darf nicht mit einem Artikel anfangen (also der, die, das). Der Artikel kann dann von einem Spielleiter im Spiel hinzugefügt werden.
  • Der Name darf nur folgendes Sonderzeichen beinhalten: `
  • Der Name darf nicht mehr als 27 Zeichen beinhalten

Alle Gildenamen die nicht diesen Regeln entsprechen werden abgelehnt.

1. Nachdem Sie einen passenden Gildennamen ausgesucht haben, besuchen Sie bitte folgende Webseite:

http://eqlive.station.sony.com/guild_requests/de/intro_view.jsp

2. Folgen Sie den dort angegebenen Anweisungen

Klicken Sie auf den ersten Link: Click here to submit a guild creation request

Geben Sie falls notwendig Ihre Login Daten ein.

Geben Sie im ersten Feld Ihren Gildennamen ein: Proposed Guild Name

Suchen Sie im zweiten Feld den Server aus: EQ Server

Geben Sie im dritten Feld den Namen Ihres Charakters der Gildenleiter werden soll an: Guild Leader (Must Be Associated With The Above Station Name)

Geben Sie im vierten Feld eine eMail Adresse wo wir Sie kontaktieren können an: Email Address

3. Sie können den Status Ihres Antrages jederzeit auf folgender Webseite ansehen: http://eqlive.station.sony.com/guild_requests/request_view.jsp

4. Wenn der Name angenommen wurde erhalten Sie per E-Mail eine Bestätigung, dass die Gilde nun gegründet werden kann.

5 . Nachdem Sie per E-Mail die Bestätigung erhalten haben, dass die Gilde genehmigt wurde, müssen Sie einen Spielleiter im Spiel mittels dem /petition Befehls kontaktieren, um den Gründungsvorgang abzuschließen.

 

Gilden beitreten und verlassen

Wenn Sie zum Entschluss gekommen sind, dass es Zeit für Sie ist, einer Gilde beizutreten, sollten Sie Ihre Weggefährten in Norrath um Auskunft bitten. Sie können anhand der Gildenwappen, die über ihren Köpfen angezeigt werden, erkennen, zu welcher Gilde sie gehören. Fragen Sie nach den Gilden, informieren Sie sich und entscheiden Sie, ob die Gilde Ihrer Persönlichkeit und Ihrem Spielstil entspricht. Die meisten Gilden unterhalten eigene Websites, auf denen Sie weitere detailliertere Informationen finden können.

Nicht alle Gilden akzeptieren neue Mitglieder. Über die Mitgliedschaft in einer Gilde und wer sie bekommen darf, entscheiden einzig und allein die Gilde und ihre Mitglieder selbst.

Wenn Sie in eine Gilde aufgenommen werden, wird Sie der Gildenführer oder ein hochrangiges Mitglied der Gilde (ein sogen. Offizier) mit dem Befehl /gildeneinladung einladen. Wenn Sie diese Einladung erhalten, können Sie sie entweder annehmen oder ablehnen.

Wenn Sie sich entschließen Ihre Gilde zu verlassen, können Sie dies jederzeit mit dem Befehl /gildeentfernen tun.

Gilden-Befehle

/gildeneinladung <NAME> <RANG>- Laden Sie eine Person ein, der Gilde beizutreten. Wenn diese Person ein Offizier sein soll, fügen Sie das Argument (wie Oskar) für den Rang hinzu. Der Befehl kann nur von Gildenführern und Offizieren benutzt werden, wobei nur Gildenführer Offiziere ernennen dürfen. In dieser Funktion können Sie ebenfalls ein Mitglied zu einem Offizier befördern.
/gildeentfernen <NAME>- Entfernen Sie ein Mitglied aus der Gilde. Nur der Gildenführer kann Offiziere von der Gilde ausschließen. Benutzen Sie den Befehl für Ihren Charakter, um die Gilde zu verlassen.
/gildelöschen - Lösen Sie Ihre gesamte Gilde auf. Nur der Gildenführer kann diesen Befehl verwenden.
/gildenstatus - Liefert Informationen darüber, ob die genannte Person ein Mitglied, Offizier oder der Gildenführer ist.
/gildenführer <NAME>- Überträgt die Führerschaft über die Gilde an <NAME>.
/gildentext - Schickt eine Botschaft an alle Mitglieder der Gilde.
