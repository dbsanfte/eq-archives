- Bug Reports -
Beta-testing isn't all about playing EQ for free. Even though they don't have to, we would like our beta-testers to file in bug reports, which is the whole point of the test... Here's the procedure:

You can send in a bug report with the /petition and /bug commands. If you use the first, please describe the problem you encountered in as much detail as possible so a Guide or GM can find out exactly what it is when he reads your petition. He may also have to contact you for more details. This is thepreferred mehtod for quest bugs.

The second command, /bug, is for everything else: crashes, stability issues, freezes...).

Please don't expect bugs to be solved by the folowing patch. Some take a very long time to fix - but you already know that :)
