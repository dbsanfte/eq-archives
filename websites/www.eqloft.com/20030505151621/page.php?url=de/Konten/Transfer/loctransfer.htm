Seite 1 - Präsentation

In Zusammenarbeit mit UbiSoft haben wir kürzlich drei neue Server in Europa eröffnet (Venril Sathir in UK, Sebilis in Frankreich und Kael Drakkal in Deutschland). Die Reaktionen auf unser Angebot, für einen kurze Zeit 1000 Spieler gratis auf jeden dieser Server transferieren zu lassen, war aussergewöhnlich gut, und unglücklicherweise gab es weniger freie Plätze als interessierte Spieler.

Obwohl der Gratis-Transfer beendet ist, können jene die auf die europäischen Server transferieren wollen, dies nun mit dem Standard Transfer Service machen. Dieser Service ist unserem traditionellen Service abgesehen von 3 Punkten ähnlich:

  1. Alle Charaktere auf dem eigenen Konto, werden von einem einzigen Server auf einen neuen bewegt.
  2. Sie behalten alle Gegenstände und Münzen.
  3. Der Sullon Zek Server ist ausgeschlossen von diesem Transfer Service.

Die Charaktere werden ohne Gewähr von Sony Online Entertainment zu UbiSoft transferiert. Sobald der Transfer beendet ist, werden Sie UbiSoft Kunde und sind nicht mehr länger Kunde von Sony Online Entertainment, in Belangen die Buchungen und Kundendienst betreffen.

Ausserdem, wenn ein Charakter-Transfer von einem SOE Server auf einem UbiSoft Server stattfindet, müssen Sie es erlauben ihre persönlichen Informationen, zusammen mit den gültigen Kreditkarten Informationen (Global Collect Daten nicht eingeschlossen) mit UbiSoft zu teilen. Das Weitergeben der Konto-Informationen erlaubt es UbiSoft ein Konto auf einem UbiSoft Server zu erstellen. Sobald Sie zu einem UbiSoft Server transferiert sind unterstehen ihre persönlichen Daten den Datenschutzrichtlinien von UbiSoft. Persönliche Informationen und Kreditkarten-Informationen die bei Sony Online Entertainment bleiben unterliegen weiterhin SOE's Datenschutzrichtlinien. Des weiteren ist diese Dienstleistug nicht für Abonnementen vom Legends-Server verfügbar. Falls Sie sich entschliessen Ihre Charaktere mit diesem Dienst zu transferieren wird Ihr Konto, für Buchungs und Kundendienst Angelegenheiten, an UbiSoft übertragen, selbst wenn sie andere Charaktere auf dem selben Konto haben welche nicht auf den neuen Server wechseln. Falls Sie dieses Angebot nutzen, wird Ihr Konto ein "UbiSoft Konto", Sie werden Ihr Konto nicht mehr zurück zu SOE wechseln können.

Alle Transferanfragen benötigen die Zustimmung von SOE und UbiSoft.

Für weitere Informationen darüber wie Sie dieses Angebot nutzen können, lesen Sie bitte unsere Richtlinien.

Alle Bedingungen und Konditionen der EverQuest User Agreement and Softwae License (EULA) und der Rules of Conduct gelten für jeden Transfer oder Transfer Versuch. Sie können nicht Charaktere auf ein Konto transferieren, welches nicht Ihnen gehört oder nicht ursprünglich auf Sie registriert war.

Richtlinien und Restriktionen

  • Alle Charaktere werden mit Geld und Gegenständen von einem US Server auf einen der neuen europäischen Server transferiert. Dies bezieht sich auch auf Gegenstände in der Bank des Charakters und auf die Zaubersprüche im Zauberbuch. Es wird empfohlen keine Gegenstände vor dem Transfer zu zerstören oder weiterzugeben, da es möglich ist das Ihr Transfer Antrag abgelehnt wird. Gegenstände welche, aufgrund eines bevorstehenden Transfers, weitergegeben oder zerstört wurden werden nicht zurückerstattet. (Hinweis: Charakter-Leichen können nicht transferiert werden. Bevor Sie um einen Charakter Transfer ansuchen, stellen Sie bitte sicher, dass alle Ihre Charakter Leichen gelootet wurden und sich keine Leichen der Charaktere im Spiel befinden.)

  • Gildenzugehörigkeit: Jede Gildenzugehörigkeit wird bei dem Transfer gelöscht.

  • Leider können können wir keine Charakter Transfers für Konten, welche sich noch in der 30 Tage Testphase befinden durchführen.

  • Alle Anfragen werden, solange es angemessen und durchführbar ist bearbeitet. SOE und UbiSoft behalten sich das Recht vor jede Anfrage, nach ihrem einzigen und alleinigen Ermessen, aus beliebigen Gründen oder ohne Gründen abzulehnen.

  • Das Anfragen-Formular kann nicht benutzt werden, um 2 oder mehr Charaktere auf verschiedene Server zu transferieren. Falls Sie Charaktere auf mehrere Server transferieren möchten, müssen Sie mehrere Anfrage-Formulare ausfüllen.

  • Jede Anfrage die abgeschickt wird, erlaubt es alle Charaktere von einem einzelnen Server auf einen europäischen Server zu transferieren. Um Charaktere von einem zusätzlichen Server zu transferieren, müssen Sie ein neues Anfrage-Formular ausfüllen.

  • · Falls Sie einen Charakter-Transfer versuchen, welcher gegen diese Richtlinien, irgendwelche Gesetze, die EULA oder die Rules of Conduct verstösst (zum Beispiel beim Versuch Charaktere von einem Konto einer anderen Person zu transferieren, oder von einem Konto welches Sie oder jemand für Sie gekauft hat, oder eines Kontos welches Ihnen geschenkt wurde und von dem Sie nicht der original registrierte Besitzer sind), werden diese Charaktere nicht transferiert, Ihre Anfrage wird nicht bearbeitet oder genehmigt und wir behalten uns weitere disziplinäre Massnahmen gegen Ihre Konten vor.

  • Charakter-Namen können bei Servertransfers nicht garantiert werden. Falls der Charakter-Name schon auf dem Ziel-Server vergeben ist, wird ein "X" am Ende des Namens angehängt.

Falls Sie fortfahren möchten Charaktere von einem US Server auf einen der neuen europäischen Server zu transferieren, füllen Sie einfach das folgende Antragsformular aus.
