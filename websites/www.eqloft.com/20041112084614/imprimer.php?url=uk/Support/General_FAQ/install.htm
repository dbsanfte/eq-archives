- Installing the Game -
Before installing EverQuest, make sure that your PC specifications meet or exceed the minimum requirements.

Insert the EverQuest game CD into your CD-ROM drive. If your CD-ROM drive has the AutoPlay function enabled, the installation will begin automatically. If AutoPlay is not enabled, Open the "My Computer" icon on your desk top, double-click on the CD-ROM icon, then double-left-click on "setup.exe". The installation process will then commence.

EverQuest requires at least 600MB of free disk space to install.
    1. Select a directory in which you would like to install EverQuest. The default directory is C:\Program Files\EverQuest. If you would like to install the files into a different directory, change the path and proceed. If the directory does not exist, create a directory name (Example: C:\Program Files\Eqrules), then proceed.
    2. Install the DirectX drivers:
    After the installation of EverQuest, an option to install DirectX to your hard drive will be given. Select "Yes" to proceed, then follow the on-screen instructions. If you already have a better version of DirectX installed, you will be told that no update is possible.
    3. After completing the installation, you will be prompted to restart your computer, select "Yes" and allow your system to reboot.
DirectX 8.1 or Higher Required

With the release of EverQuest: The Shadows of Luclin expansion pack, we require DirectX 8.1 or higher. This is true not only for those purchasing Shadows of Luclin, but for everyone playing EverQuest. All details regarding this change can be found here:

http://boards.station.sony.com/ubb/everquest/Forum1/HTML/000348.html

To determine what version of DirectX is currently installed on your system:
    1. Click the Start menu.
    2. Use the Run.. command..
    3. In the Open field, type "DXDIAG" then click OK. This will launch the DirectX diagnostics tool.
Your version of DirectX will be listed at the bottom of the System Information area. For example, version 7.0a is listed as "DirectX 7.0 (4.07.00.0716)"

Please do the following prior to playing EverQuest. It is very important that your drivers are installed in the order indicated to avoid potential problems:
    1. Go to https://www.microsoft.com/downloads/release.asp?ReleaseID=27741and install the latest DirectX drivers for your operating system.
    2. Go to the website for your 3D accelerator. There are a number of different manufacturers, and it is preferred that you visit the website for the manufacturer of your card. At your option, you can go to the chipset manufacturer site. Most will be able to use either www.3dfx.com (for Voodoo cards), www.atitech.com (for ATI cards) or www.nvidia.com (for TNT/GeForce cards). Download and install the latest drivers appropriate for your card and operating system. See our Supported Video Card list and Driver Installation Guide for more details.
    3. Go to the website for your sound card manufacturer and do the same. If you do not know what brand of sound card you have, you can right-click on "My Computer", select "properties", click on "Device Manager", and open the "Sound, Video, and Game Controllers" section. Your sound card will be listed there. If you use a SBLive sound card, you can run "LiveUpdate" from the Windows Control Panel off of your start menu. View our Supported Sound Card list and Driver Installation Guide for driver locations and more details.

If you experience any problems, please do the following:
    1. Run DXDIAG.EXE from your Start menu (click start, run, then type "dxdiag")
    2. Click the "Save all Information" button at the bottom of the program window.
    3. Save the file as "dxdiag.txt"
    4. Email that file to eqtech@soe.sony.com along with a complete description of your problem, the "dbg.txt" file from your EverQuest directory, the text from any error messages that are presented, and if you are willing to assist our test department over the phone, a phone number where you can be reached.
