- Commandes -
%T, %S, %O, %P, %R
Ces 5 commandes vous permettent de connaître : le nom, le genre (respectivement he, she, it / him,her,it / his,her,its), la race (seulement pour les personnages joueurs) de votre cible.

/adriver
Vous permet de retranscire automatiquement les données de votre carte son dans un fichier text.

/afk
Vous permet signaler que vous n'êtes pas derrière votre clavier. Si l'on vous envoie un tell, la personne recevra le message suivant " Player is temporarily Away From Keyboard ". Par ailleurs, un tag [AFK] apparaîtra à côté de votre nom et dans les /who. Enfin, vous pouvez personnaliser votre message d'absence en précisant un texte de réponse automatique au tell. Pour ce faire, il vous suffit de taper /afk [texte]

ALT (1 à 8)
Vous permet de lancer manuellement les sorts mémorisés dans les gemmes 1 à 8.

ALT B
vous permet de fermer ou d'ouvrir la fenêtre Buff/Spell Effects Window

ALT P
vous permet de fermer ou d'ouvrir la fenêtre Party (Group) Window

ALT C
vous permet de fermer ou d'ouvrir la fenêtre Chat Window

ALT S
vous permet de fermer ou d'ouvrir la fenêtre Spell Gem Window

ALT H
vous permet de fermer ou d'ouvrir la fenêtre Hotbox Window

ALT T
vous permet de fermer ou d'ouvrir la fenêtre Target Window

ALT Y
vous permet de fermer ou d'ouvrir la fenêtre Self-Status Window

ALT M
vous permet de fermer ou d'ouvrir la fenêtre Menu Window

/anon ou /a (ON ou OFF)
Vous permet de vous mettre en mode "anonymous" ou de sortir de ce mode. Lorsque vous êtes anonyme, les autres joueurs ne peuvent pas connaître votre niveau, votre classe, race ou l'endroit où vous vous trouvez.

/assist <nom_du_joueur>
Vous permet d'aider une personne pendant un combat. Très pratique lorsque vous voulez taper sur la même créature que vos amis par exemple.

/attack
Vous permet de déclencher votre attaque de mêlée manuellement. Il faut préalablement cibler la créature.

/auction
Vous permet d'utiliser le canal de vente aux enchères, indiquand que vous souhaitez vendre ou acheter un objet. Ce message est vu par toutes les personnes présentes dans la zone.

/audiodebug (on/off)
Vous permet d'enregistrer les informations de votre carte son dans un fichier texte.

/autosplit
Permet de partager la somme d'argent trouvée sur le cadavre d'une créature.

/book
Vous permet de méditer et d'ouvrir automatiquement votre livre de sorts.

/bug ou /b
Vous permet de signaler un bug (faire un « report »).

ALT
Permet d'afficher le nom du sort à côté de la gemme de mémorisation ainsi que la liste des sorts d'enchantements (buffs) actuellement lancés sur vous.

/camp
Vous permet de sortir du jeu de la manière suivante : si vous tapez /camp server vous reviendrez automatiquement à l'écran de sélection des serveurs ; si vous tapez /camp desktop vous arriverez directement sur votre bureau.

/channel
permet de définir votre channel privé par défaut qui servira à discuter notamment lors de raids avec différents groupes de différentes guildes. Ce dernier est sauvegarder dans vos settings. de nombreuses commandes ont été ajoutées dans ce mode de chat privé. Il s'agit de créer vos propres channels de discussion et vous trouverez toutes les infos relatives dans l'eqmanual_supplement.doc. Sommairement, ces commandes à utiliser sont les suivantes et permettent la mise en place de passwords pour ces channels.

/join (par exemple /join casters:seekrit)
/chat (texte)
/leave

/cast (1 à 8)
Vous permet de lancer manuellement les sorts mémorisés dans les gemmes 1 à 8.

/charinfo
Vous permet d'afficher des renseignements de base sur le personnage, notamment le point d'anchrage.

/chatfontsize (0 à 5)
vous permet de modifier la taille de la police dans la fenêtre de chat, la valeur par défaut étant 1.

/clearchat
Efface tout le texte dans votre fenêtre de chat.

/clearhits
Permet de nettoyer vos hits dans les modes M2-M5 de la commande /hitsmode. Vous trouverez plus d'explication dans l'eqmanual_supplement

/consent <nom_du_joueur>
Donne la permission à un joueur de "draguer" votre corps jusqu'au bord de la zone où vous êtes mort.

CONTROL + clic gauche
Vous permet de sélectionner un seul élément d'un stack dans votre inventaire.

/consider ou /con
Même résultat qu'un clic droit de votre souris. Permet de connaître le niveau moyen d'une créature ou d'un joueur et son attitude envers vous.

/corpse
Vous permet de trainer votre corps ou celui d'un joueur ayant fait un /consent à votre nom. Cette commande est très utile dans la mesure où elle fonctionne à une certaine distance et vous permettra donc de récupérer votre corps même si ce dernier est entouré de créatures ou dans la lave.

/duel ou /d
Permet de provoquer en duel un autre joueur. Le combat est toujours à mort. Vous perdez si vous changez de zone ou campez. Vous ne gagnez aucun XP et ne pouvez pas récupérer le moindre objet (sur un serveur "normal"). Vous ne perdez pas non plus d'XP, et dans ce cas précis de mort vous n'avez que 5 minutes pour vous faire ressuciter.

/decline
Permet de refuser un duel.

/disband
Vous permet de quitter le groupe dans lequel vous vous trouvez.

/discipline
Permet d'afficher la liste des disciplines disponibles pour votre classe ainsi que de savoir si vous êtes prêt à en utiliser une (il sera indiqué dans combien de temps dans le cas contraire).

/doability (1 à 10)
Permet de déclencher manuellement vos compétences.

/dopropertinting
Permet d'activer/désactiver le "DoProperTinting" de l'eqclient.ini.

/duck ou d
Pour que votre personnage se plie en deux. Surtout utile aux ogres et autres créatures de grande taille qui veulent rentrer dans un passage étroit.

/dynamiclights
Permet d'activer ou de désactiver les effets particulaires dynamiques autour de vous.

/em (texte) ou : (texte) ou /me (texte) ou /emote (texte)
Vous permet d'écrire une phrase d'attitude. Par exemple "regarde les mouches voler" permet de signaler aux autres que vous vous ennuyez :) (voir aussi les "Emotes"). La zone d'action est la même que lorsque vous parlez.

/exit
Vous permet de quitter immédiatement le jeu. Votre personnage sera alors LinkDead et pourra être attaqué pendant quelques minutes avant de disparaître du serveur. A n'utiliser que dans des cas d'urgence extrême, comme l'arrivée d'un collègue de travail dans votre bureau.

/fastdrop (on/off/never)
Vous permet de choisir si vous devez confirmer ou non quand vous jetez un objet sur le sol.

/feedback
Vous permet d'envoyer les derniers textes de vos messages directement à un GM (utile si vous avez un problème avec un autre joueur).

/filter
Pour activer ou désactiver le filtre dédié aux insultes et autres termes du "bad langage".

/follow
Cette commande nécessite que vous soyez dans le même groupe que celui que vous suivrez. Il vous suffit alors de le targeter et de taper /follow, vous le suivrez automatiquement. attention néanmoins aux zones avec de la lave, des précipices ou dans les forêts, son usage y est dangereux...

/friend (nom)
Vous permet de bâtir une liste d'amis :
/friend -> Affiche la liste de vos amis.
/friend (nom) -> Ajoute ou retire un ami de votre liste.
/who all friend -> Vous permet de savoir si l'un de vos amis est en ce moment sur le serveur

/ftell (message)
Vous permet d'adresser un tell à tous ceux qui se trouvent sur votre liste d'amis.

/gems
Vous permet de lancer un tetris like... sans commentaire.

/gsay ou /g
Vous permet de parler uniquement aux personnes qui sont dans votre groupe. Vous pouvez automatiser cette fonction en faisant un clic droit de souris sur la zone de texte et en sélectionnant le mode "groupe". Les textes en mode groupe s'affichent en bleu clair.

/GUILD
Vous permet de voir la liste des membres de votre guilde dans la zone où vous vous trouver.

/guild...
Vous trouverez ici l'ensemble des commandes de guilde. pour plus d'information sur ces confréries de joueurs, suivez ce lien.

/guildremove <nom du joueur> Quitter la guilde (un membre peut quitter la guilde de lui-même, mais seul le Maître peut faire quitter la guilde à des membres ou des officiers).
/guilddelete ou /guildd Met fin à la guilde (maître seulement).
/guildstatus <nom du joueur> Affiche le grade d'un personnage dans sa guilde : player, officer ou leader (tous les membres).
/guildleader <nom du joueur> Nomme un nouveau maître de la guilde (maître seulement).
/guildsay Envoyer un message à tous les membres de la guilde, où qu'ils soient (membres de la guilde).
/guildwar <nom du maitre de la guilde ennemie><durée> Déclarer la guerre à une autre guilde.
/guildpeace < nom du maître de la guilde ennemie > Proposer une trêve et mettre fin à la guerre.
/guildinvite <nom du joueur> Inviter quelqu'un à rejoindre la guilde (maître ou officier).
/guildinvite o <nom du joueur> Faire de quelqu'un un officier de la guilde (maître seulement).
/guildmotd Affiche le message du jour de la guilde (seuls les Officiers et le Leader peuvent en spécifier un).

/help
Affiche la liste de toutes les commandes dans votre interface de chat.

/hitsmode
Permet de modifier la vision des infos relatives au combat. Tout est expliqué dans votre eqmanual_supplement.doc

/ignore (nom)
Permet d'ignorer une personne, vous ne recevrez plus ses messages.

/inspect on | off
Permet d'activer ou de désactiver la possibilité d'inspecter les autres en cliquant droit sur eux.

/invite
Vous permet d'inviter un joueur dans votre groupe. Seul le leader du groupe peut inviter et un maximum de 6 personnes.

/lfg
Vous permet de signaler que vous recherchez un groupe. Le sigle LFG apparaît à côté de votre nom et vous apparaissez dans les tables de /who et /who all comme recherchant un groupe.

/loc ou /l
Vous donne votre localisation sur la carte selon les axes X pour le Nord/Sud,Y pour l'Est/Ouest et Z pour la hauteur. Pour vous diriger efficacement, il vous suffit de prendre en compte les X et les Y sur la base suivante : plus le chiffre est est grand, plus vous êtes proche du Nord ou de l'Ouest. Attention, ces coordonnées changent pour chaque zone.

/log on ou off
Permet d'enregistrer automatiquement ou non votre fenêtre de discussion dans un fichier txt de votre répertoire EverQuest.

/loot
Permet de looter un corps qui se trouve dans la lave ou une autre zone dangereuse.

/lootnodrop
Vous permet de spécifier si vous lootez automatiquement les objets no drop, si vous devez à chaque fois spécifier si vous lootez ou non ou si vous ne les lootez jamais.

/mcicontrol
Vous permet d'écouter un CD musical tout en jouant à EverQuest. La marche à suivre est la suivante :

- insérer un CD dans votre lecteur avant de lancer EQ
- Une fois dans le monde de Norrath, tapez /mcicontrol open cdaudio
- ensuite vous tapez /mcicontrol play cdaudio

/motd
Permet d'afficher le message du jour.

/mousespeed
Permet d'augmlenter ou de diminuer le paramètre MousePointerSpeedMod.

/note
Il vous suffit de taper cette commande puis d'entrer une phrase qui restera dans votre journal personnel (fichier notes.txt).

/pause (1 à 600)
Vous permet d'articuler différentes actions avec des pauses variables d'une durée de l'argument en dixième de secondes (un argument de 100 indique une pause de 10 secondes).
Exemple :
/sit off
/cast 1
/pause 100
/sit
Le personnage se lève, lance le sort qui se trouve dans le premier slot de sorts, s'arrête pendant 10 secondes et s'assoit. Vous pouvez également taper la suite de commande comme ci-dessous :
/sit off
/pause 100, /cast 1
/sit

/ooc
"Out Of Character". Si vous "jouez" votre personnage, le mode OOC permet de signaler que c'est le joueur et non votre personnage qui parle. Attention : le /ooc affiche les commentaires dans toute la zone. Si vous désirez parler à une personne en OOC, vous pouvez le faire en tapant "ooc: salut" par exemple.

/particledensity
Permet de visualiser ou de faire disparaître les particules des sorts sans passer par le menu des options

/pet as you were
Ordonne à votre familier de reprendre à sa position initiale.

/pet attack (cible)
Ordonne à votre familier d'attaquer la cible.

/pet back off
Ordonne à votre familier de revenir. Attention : si ce dernier est attaqué, il peut retourner se défendre :).

/pet follow me
Ordonne à votre familier de vous suivre.

/pet guard here
Ordonne à votre familier de monter la garde à un endroit précis.

/pet guard me
Ordonne à votre familier d'attaquer toute créature qui vous attaque.

/pet get lost
Ordonne à votre familier de se suicider.

/pet sit down
Ordonne à votre familier de s'assoir et de ne rien faire.

/pet stand up
Ordonne à votre familier de se lever.

/pet target
Cible automatiquement votre familier.

/pet taunt
Permet d'activer ou de désactiver le taunt de votre familier en combat.

/pet who leader
Ordonne à un familier de donner le nom de son maître.

/petition (message)
Demander l'assistance d'un GM en cas de problème avec EverQuest. Attention les GM ne sont la que pour vous aider en cas de problèmes "hors jeu" ! (par exemple si votre corps à réellement disparu d'une zone).

/played
Vous donne le temps que vous avez joué à EverQuest avec ce personnage. Ne pas montrer à votre copain, copine, ou parents. :)

/random (x)
Génère un chiffre au hasard entre 1 et X. Très pratique pour savoir à qui va aller un certain objet après une bataille, par exemple, car le résultat est affiché pour tous.

/reply ou R
envoie automatiquement un tell à la dernière personne vous ayant envoyé un tell. il ne vous reste plus qu'à taper le message.

/report
Vous permet de petitionner en envoyant directement le texte qui se trouve dans la partie visible de votre fenêtre de chat à un GM.

/reversesound
Permet d'inverser les canaux droits et gauches de vos enceintes.

/roleplay ou /role
Permet de signaler que vous êtes en mode roleplay. Votre nom apparaîtra alors en une couleur différente selon le type de serveur où vous vous trouvez et vous serez anonymous sauf en ce qui concerne votre guilde qui apparaîtra toujours à côté de votre nom.

/rtarget (/rt)
Permet de targeter la dernière personne à vous avoir tellé.

/safelock
Vous permet de bloquer entièrement votre interface, à savoir les clics souris, les raccourcis clavier, les flêches directionnelles et la plupart des /command. L'utilisation de cette commande est simple, vous tapez /safelock password pour bloquer et vous retaper /safelock password pour débloquer.

/say
Pour parler dans la zone qui vous entoure immédiatement.

/serverfilter on
Vous permet de filtrer plus efficacement le transfert des données et de diminuer le lag notamment en cas de gros combat ou de connexion lente. Vous filtrerez automatiquement les messages de sorts et de chants, les messages de combat, les différentes catégories du chat (auction, ooc etc...).

SHIFT + Clic gauche
Vous permet de sélectionner en une seule fois tout un stack dans votre inventaire.

SHIFT+ flêche directionnelle gauche ou droite
Vous permet de déplacer le curseur dans votre ligne de texte.

SHIFT+Retour chariot (Home) ou SHIFT+Fin
Votre curseur se retrouvera directement au début ou à la fin de votre ligne de texte.

/shout
Permet de hurler quelque chose qui sera entendu par toutes les personnes dans la zone.

/showspelleffects [on/off]
Permet de filtrer les effets particulaires des sorts incantés afin de réduire le lag de certaines situations ou de certaines machines.

/showgrass
Permet de switcher le radial flora (Luclin only) sur on ou off

/shownames (ON ou OFF) et /shownpcnames (ON ou OFF)
Vous permet d'afficher ou de masquer les noms situés au dessus de la tête des créatures ou des joueurs.

no parameter = settings de base.
off = les noms des personnages n'apparaissent pas
1 = seulement le prénom des personnages
2 = seulement le prénom et le nom des personnages
3 = seulement le prénom, le nom et la guilde des personnages
4 = seulement le prénom, le nom, la guilde et le titre des personnages

les tags LFG, AFK et LD continuent d'apparaître, sans crochets, sauf si l'option est off. LFG apparaît enfin au même titre que LD et AFK et n'est plus indiqué par une * en début de nom.

/showspelleffects [on/off]
Cette commande filtre les effets particulaires des sorts incantés afin de réduire le lag de certaines situations ou de certaines machines.

/sit
Vous permet de faire assoir votre personnage.

/split # # # #
Permet de partager un certain nombre de pièces avec le groupe. Le premier chiffre correspond aux platinium, puis à l'or, à l'argent et enfin au bronze. Par exemple

/split 1 2 3 4
Permettra de partager 1 pièce de platine, 2 d'or, 3 d'argent et 4 de bronze avec votre groupe. Tous les chiffres doivent être présents, mettez un "0" si vous ne désirez pas partager un certain type de pièces.

/stand
Permet de se lever...

/stopsong
permet aux bardes d'arrêter leur chant.

/surname
Vous permet d'écrire votre deuxième nom quand vous devenez niveau 20 (20 caractères maximum).

/target (nom)
Permet de cibler directement et manuellement la personne ou la créature indiquée

/targetgroupbuff ou /tgb
Pour l'activer vous devez taper /targetgroupbuff on ou off pour le désactiver. Quand elle est active, cette commande permet de faire bénéficier de vos buffs de groupe tous ceux qui sont groupés avec la cible de votre sort.

/tell ou /send (nom du joueur)
Permet d'envoyer un message à une personne précise.

/ttell
Permet maitnenant d'envoyer directement un tell à votre cible sans devoir taper son nom. La commande est également fonctionnelle sur les corps des PJs.

/time
Vous donne l'heure actuelle de Norrath.

/trackfilter (red/yellow/white/blue/green)
Vous permet de filtrer le /con quand vous utilisez votre compétence tracking, par exemple en n'indiquant pas les mobs verts... c'est à dire que si vous tapez /trackfilter red, les créatures que vous /con rouges n'apparaîtront plus dans la liste de vos track.

/trackplayer
Vous permet de garder en mémoire les traces suivies avec votre compétence track malgré le passage d'une zone ou un /camp.

/tracksort (normal/distance/con)
Vous permet de sélectionner le type de track utilisé.

/usercolor
Vous permet de personnaliser la couleur des textes en jeu. Vous devez taper la commande /usercolor et préciser les données de la manière suivante. Vous tapez /usercolor usercolornumber RVB, usercolornumber étant le numéro de couleur/texte indiqué dans votre fichier EQClient.INI et RVB les composantes rouge, vert et bleu. Les changements seront automatiquement enregistrés dans votre répertoire.

/vrdelay
Vous permet de modifier le delay dans l'utilisation des commanbdes vocales afin d'améliorer les performances.

/who ou /who (nom du joueur) ou /w
Permet de lister tous les joueurs ou une personne précise.

/who corpse
permet de lister tous les corps qu'une personne possède dans une zone.

/who all ou /who all (nom du joueur ou guilde) ou /who all (classe) ou /who all (level)
Permet de lister les joueurs selon certains paramètres. Par exemple, /who all ogre warrior vous permettra de connaitre le nombre d'ogres warriors actuellement présents dans Norrath.

/whotarget
Permet de connaître le nom de la personne qui vous cible.

/yell ou /y
Vous permet de faire savoir à tous ceux qui sont autour de vous que vous avez besoin d'aide. Ils verront le message suivant " (name) yelled for help from your (direction) ". Il est conseillé de vous faire une Hotkey avec cette commande.
