Wie erstelle ich einen neuen Charakter?

Nachdem Sie sich mit dem Server Ihrer Wahl verbunden haben, gelangen Sie zur Charakterauswahl. Klicken Sie auf "Neuen Charakter erstellen", und Sie gelangen zur Charaktererstellung. Wählen Sie hier zunächst die Rasse, der Ihr Charakter angehören soll. Nähere Informationen über die Rassen finden Sie auch hier.

Wählen Sie als nächstes die Klasse, und somit den Beruf Ihres Charakters. Unter diesem Link können Sie zusätzliche Informationen über die Klassen erhalten.

Wählen Sie nun das Geschlecht Ihres Charakters aus. In der Welt von Norrath sind Frauen und Männer gleichberechtigt, und das Geschlecht hat keinerlei Einfluss auf die Attribute.

Ein Klick auf "Gesicht" lässt Sie das Aussehen Ihres Charakters verändern. Sie können diese Änderungen selbst vornehmen, oder über den "Zufall" Knopf EverQuest die Wahl für Sie übernehmen lassen.

Wählen Sie als nächstes den Namen, den Ihr Charakter in EverQuest tragen soll. Auch hier können Sie sich durch EverQuest helfen lassen, indem Sie auf "Namen gen." klicken. EverQuest wird Ihnen eine, für Ihre Rasse, typischen Namen vorschlagen, den Sie jedoch nach belieben ändern könne. Beachten Sie bitte, das gewisse Namen durch die Namensgebungsregeln nicht erlaubt sind. Genaueres zur Namensgebung finden Sie hier.

Als nächstes verteilen Sie die Attributspunkte, die Ihrem Charakter zu Beginn zur verfügung stehen. Dieser Link bietet Ihnen nähre Informationen über die Attribute.

Entscheiden Sie sich nun für eine Gottheit, die Ihr Charakter anbetet, oder dafür, als Agnostiker durch die Länder zu ziehen. Diese Wahl kann nicht mehr rückgängig gemacht werden, wählen Sie also mit Bedacht. Sie finden Informationen über die Götter in der Enzyklopädie unter dem Eintrag "Götter" oder unter diesem Link.

Die letzte Auswahl, die Sie treffen müssen, betrifft Ihre Geburtsstadt. In dieser Stadt beginnen Sie Ihre Abenteuer in EverQuest. Viele Rassen- / Klassen-Kombinationen sind auf eine Stadt beschränkt, bei einigen dürfen Sie jedoch zwischen mehreren Möglichkeiten wählen. Informationen über die Städte finden Sie hier.

Zu guter Letzt wird Ihnen eine kurze Zusammenfassung Ihres Charakters präsentiert, die Ihnen Hintergrundinformationen über Ihren Charakter sowie eine Einschätzung der Schwierigkeit, diesen Charakter als Neuling zu spielen, bietet. Nachdem Sie ein letztes Mal auf "Weiter" geklickt haben wird Ihr Name überprüft. Wenn dies geschehn ist, steht Ihrem ersten Abenteuer in Norrath nichts mehr im Weg.
