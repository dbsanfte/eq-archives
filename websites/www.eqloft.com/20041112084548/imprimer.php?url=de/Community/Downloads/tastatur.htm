- Tastaturbelegungen -

Diese Seite sollte Ihnen helfen, EverQuest mit verschiedenen, internationalen Tastaturbelegungen laufen zu lassen.

Zunächst sollten Sie herausfinden, welche Tastaturbelegung Ihr Computer benutzt:

Klicken Sie auf Start
Klicken Sie auf Einstellungen
Klicken Sie auf Systemsteuerung
Klicken Sie auf Tastatur
Klicken Sie auf die "Sprache" Karteikarte
Sehen Sie nach, was unter "Belegung" eingetragen ist.
Anschließend laden Sie die Datei herunter, die Ihrer Tastaturbelegung entspricht:

Laden Sie die Datei aus der unten aufgeführten Liste herunter, die den selben Namen wie die Eintragung unter "Belegung" hat.
Kopieren Sie die heruntergeladene Datei in Ihr EverQuest Verzeichnis.
Führen Sie die Datei aus. Dadurch wird eine keyboard.txt Datei für Ihre Tastaturbelegung erzeugt.
Spielen Sie; das war's schon!
*Anmerkung - Um zur standard Tastaturbelegung zurückzukehren, löschen Sie einfach die keyboard.txt Datei aus Ihrem EverQuest Verzeichnis.
