D) Questfragen, Fragen bezüglich FF (AA) Punkte, Zaubersprüche, Flag Probleme

1) Was sollte ich beim Questen beachten?

Bitte beachten Sie die folgenden häufig vorkommenden Fehler von Spielern:

) Wenn Sie sich beim Quest unsicher sind dann fragen Sie unsere unsere Community auf unseren Diskussionsforen. Spielleiter können jedoch nicht bei Questauflösungen helfen.

) Kontrollieren Sie alle Gegenstände doppelt und dreifach bevor Sie sie dem NSC geben. Fragen Sie erfahrene Spieler die diesen Quest schon gemacht haben ob die Gegenstände stimmen.

) Wenn Sie die falschen Gegenstände abgeben, dann behält sich der NSC in den meisten Fällen diese und Sie verlieren sie.

) Sie müssen alle geforderten Gegenstände auf einmal abgeben. Wenn Sie nur einen Teil der Gegenstände abgeben die der NSC von Ihnen verlangt, dann behält er sich diese. Sollten Sie die restlichen geforderten Gegenstände bei sich tragen, dann geben Sie diese sofort ab, um den Quest zu vollenden.
Sie können den Quest jedoch nicht zu einem späteren Zeitpunkt fortsetzen. Es ist also normalerweise nicht möglich am nächsten Tag die restlichen Gegenstände dem NSC zu bringen.

) Kontrollieren Sie Ihr Ansehen (Faction). Bei den meisten Quests erwartet der NSC, dass Ihre Ansehensstufe hoch genug ist, um den Quest zu beenden. Sie können auf dieser Tabelle die Liste der übersetzten Ansehensnachrichten nachlesen. Fragen Sie erfahrene Spieler die diesen Quest schon gemacht haben ob Ihr Ansehen stimmt.

) Überprüfen Sie, dass die Gegenstände nicht übereinander (gestackt) abgeben. 4 Knochensplitter sollten also aufgeteilt in jeweils ein eigenes Feld des NSC-Handelsfensters gelegt werden.

) Geben Sie die genaue Anzahl an Goldmünzen ab (kein Platin und kein Silber). Wenn Sie auch nur ein Goldstück zu viel abgeben, dann ist das ganze Gold verloren und dann kann auch im Normalfall kein Spielleiter Ihnen helfen. Sie können allerdings sofort nachdem Sie die falsche Anzahl an Goldmünzen abgegeben haben, die richtige Anzahl an Münzen abgeben. Der Quest NSC sollte Ihnen dann die Questbelohnung geben.

) Sollte trotz allem ein Quest nicht geklappt haben, dann geraten Sie nicht in Panik. Schreiben Sie eine Petition und erklären Sie einem Spielleiter wahrheitsgetreu was Ihnen passiert ist. Wenn Sie wissen, dass es Ihr Fehler war und Sie erklären können was falsch war, dann helfen Sie dem Spielleiter und erhöhen die Chancen, dass Sie Ihre Gegenstände zurück bekommen.

Verheimlichen Sie keine Details, da dies dem Spielleiter und Ihnen Zeit kosten wird und die Chance vergrößert, dass die verlorenen Gegenstände nicht zurück erstattet werden können. Wenn Sie wollen, dass ein Spielleiter Ihnen hilft, dann müssen Sie ehrlich zu ihm sein.

E) Spielleiter kontaktieren, Bugs im Spiel, verlorene Gegenstände

1) Wie kontaktiere ich einen Spielleiter?

Sollten Sie im Spiel ein Problem haben, dann können Sie einen Spielleiter kontaktieren. Spielleiter können jedoch keine Fragen die Ihr Konto oder die Bezahlung betreffen beantworten.

Schreiben Sie im Spiel eine Petition und geben Sie einen Petitionsgrund an. Machen Sie dies indem Sie zum Beispiel folgendes eingeben:
/petition Ich habe Aliana Sturmwind in Felwithe den falschen Gegenstand abgegeben

Beschreiben Sie so genau wie möglich den Grund Ihrer Petition. Dies stellt sicher, dass die richtige Person Ihre Petition erhält. Wenn Sie eine Petition wie
/petition hilfe!!! schreiben, dann kann es passieren, dass es viel länger dauert bis diese beantwortet wird.

Sie können Spielleiter auch per eMail kontaktieren. Schreiben Sie dazu eine eMail an gms_de@ubisoft.fr .

Hier sind einige Regeln die Sie beachten sollten:

) Schreiben Sie falls möglich die Petition von dem betroffenen Konto. Es ist egal von welchem Charakter, solange er sich am selben Konto befindet.

) Fragen Sie keine Freunde für Sie zu petitionieren. Wir können nur Petitionen vom Betroffenen beantworten.

) Wenn ein Gegenstand verschwunden, ein Charakter weg oder Ihnen ein Fehler passiert ist, dann geraten Sie nicht in Panik. Schreiben Sie eine Petition und erklären Sie einem Spielleiter wahrheitsgetreu was geschah. Wenn Sie wissen, dass es Ihr Fehler war und Sie erklären können was falsch war, dann helfen Sie dem Spielleiter und erhöhen die Chancen, dass Ihnen geholfen werden kann.

Verheimlichen Sie keine Details, da dies dem Spielleiter und Ihnen Zeit kosten wird und die Chance vergrößert, dass Ihnen nicht geholfen werden kann. Wenn Sie wollen, dass ein Spielleiter Ihnen hilft, dann müssen Sie ehrlich zu ihm sein.

) Sollten Sie multiquesten (zwei Personen machen einen Quest zusammen und jeder gibt einen Teil der Gegenstände ab) dann kann ein Spielleiter Ihnen nicht helfen. Es werden keine Gegenstände zurückerstattet.

) Sollten sich das Problem, weswegen Sie petitionierten von alleine gelöst haben, dann löschen Sie bitte mittels /petitionlöschen Ihre Petition.

) Petitionen werden automatisch einige Zeit nachdem Sie ausloggen gelöscht. Bitte schreiben Sie wenn Sie wieder online sind eine neue Petition.

) Sie sind für Ihr Konto selber verantwortlich. Es ist aus zahlreichen Gründen gegen die Richtlinien, dass Konten mit anderen Personen geteilt werden. Genauso ist es verboten Gegenstände oder Charaktere gegen richtiges Geld zu verkaufen. Es passiert oft, dass Freunde des Kontobesitzers Gegenstände oder Charaktere löschen oder generell gegen unsere Richtlinien verstoßen. Erinnern Sie sich: Sie sind für die Konsequenzen selber verantwortlich und die Aussage, dass jemand anderes auf das Konto Zugriff hatte wird nicht akzeptiert.

Fragen bei denen Spielleiter Ihnen nicht helfen können:

Fragen Sie bei den folgenden Problemen andere Spieler um Rat:

) Questauflösungen
) Fragen ob Zauberspruch XY geändert wurde
) Fragen wieso NSC X nicht erscheint
) Fragen wo man NSC X oder Gegenstand Y findet
) Flagprobleme
) Custom User Interface (UI) Probleme

3) Ich habe einen Bug gefunden!

Fragen Sie sich bitte zuerst, ob es wirklich ein Bug ist? Fragen Sie Spieler der gleichen Klasse und Rasse ob sie das gleiche Problem haben. Posten Sie es auch auf unserem Technischen Forum. Wenn Sie sich absolut sicher sind, dass es ein Bug ist, dann können Sie diesen direkt an die Entwickler bei Sony Online Entertainment weiterleiten indem Sie den Befehl /bug verwenden.

Sollte es ein Bug sein, der mit der Übersetzung des Spieles zusammenhängt, dann können Sie diesen auf unserem Forum posten oder einen Spielleiter im Spiel mittels /petition kontaktieren.

 

Zurück zum Index

4) Ich bin für 20.000 Schadenspunkte gefallen!

Wenn Ihnen während des Spiels etwas außergewöhnliches zustößt (Zum Beispiel ein Spieler beleidigt andere, oder sie fallen durch die Decke und nehmen 20.000 Schaden, um nur einige zu nennen) haben Sie einen sehr nützlichen Befehl, um uns genau mitzuteilen was passiert ist.

Der Befehl /bericht (/report) sendet uns einen Schnappschuss desjenigen Chat Fensters in dem Sie den Befehl eintippen. Scrollen sie, so dass sich die Textzeile die Sie uns schicken wollen in den ersten 20 Zeilen des Fensters befindet. Tippen sie dann /bericht <SPIELERNAME> oder /bericht <IHR NAME> , wenn es wegen Fallschadens ist, ein.

Nachdem Sie dies gemacht haben schreiben sie eine Petition für Hilfe während des Spiels. Scheuen Sie sich nicht den /bericht Befehl zu benutzen
