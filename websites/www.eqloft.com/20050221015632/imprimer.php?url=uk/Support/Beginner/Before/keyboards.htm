- Keyboard Layouts -
This page will help you set up EverQuest to work with various international keyboard layouts.

First, determine which keyboard layout your computer uses:

Click on Start.
Click on Settings.
Click on Control Panel.
Click on Keyboard.
Click on the "Language" tab.
Look under the heading "Layout".
Then, download and use the file that matches your keyboard layout:

Download the file from the list below with the same name as is listed under "Layout".
Place the downloaded file in your EverQuest directory.
Run the file. This will create a keyboard.txt file for the specified keyboard layout.
Play the game; that's it!
*Note - To revert to the default keyboard setup, remove keyboard.txt from your EverQuest directory.
