Erstellen eines neuen Kontos

1.

Starten Sie nach der Installation von EverQuest den Patcher und laden Sie alle notwendigen Dateien für EverQuest herunter. Achten Sie darauf, dass Sie die Version 1.1.7 des Patchers besitzen, um EverQuest auf Deutsch zu spielen. Sie finden die Versionsnummer auf der linken, oberen Ecke der Patchsoftware.

Sollten Sie diesen Patcher nicht besitzen, dann können Sie ihn auf folgender Webseite herunterladen.

 

2.

Sie können während Sie das Spiel patchen Ihr EverQuest Konto erstellen. Gehen Sie dazu auf folgende Webseite: https://eqreg.station.sony.com/launchpad/registration.jsp?language=de

Füllen Sie alle Felder des Formulares aus und teilen Sie uns bitte Ihre eMail Adresse, Namen, Geburtsdatum und Telefonnummer wahrheitsgetreu mit.

Der Station-Name ist der Name Ihres EverQuest Kontos. Überlegen Sie sich den Kontonamen gut, da Sie diesen nicht wieder ändern können. Das Passwort, das Sie auswählen, muss aus mindestens vier Zeichen bestehen. Als relativ sicher haben sich Passwörter mit etwa sechs bis acht Zeichen erwiesen, die eine Kombination aus Groß- und Kleinbuchstaben sowie Zahlen oder Sonderzeichen sind.

Um fortzufahren müssen Sie die Geschäftsbedingungen akzeptieren und auf "Ich akzeptiere die Geschäftsbedingungen" klicken.

 

3.

Die Registrierung wird nun bestätigt und Sie werden eine eMail mit Ihren Registrierungsdaten erhalten. Klicken Sie auf "Melden Sie sich jetzt an" um fortzufahren.

 

4.

Sie werden nun nach Ihrem Kontoschlüssel gefragt. Sie finden diesen in der Bestätigungsemail, falls Sie ein Gratis Trial erhalten oder auf Ihrer EverQuest Installations-CD oder Verpackung falls Sie das Spiel im Handel erworben haben. Klicken Sie danach auf "Abschicken".

Beachten Sie, dass Sie bei der Eingabe des Schlüssels V V nicht mit W und 0 nicht mit O verwechseln.


5.

EverQuest überprüft die Richtigkeit der von Ihnen eingegebenen Daten und wird Sie auf eventuell bestehende Fehler hinweisen. Wenn keine Fehler vorliegen werden Sie zurück auf den Hauptmenü Bildschirm gebracht. Gratulation. Sie haben so eben ein neues EverQuest Konto erstellt.

Sollten Sie eine Probezeit zum Ausprobieren von EverQuest haben, dann sind Sie mit Ihrer Registrierung fertig. Sie können nun EverQuest starten und sich mit Ihrem Stationsnamen und Passwort einloggen.

Sie können Ihre Konto-Informationen jederzeit ändern. Gehen Sie dazu auf folgende Webseite oder klicken Sie im Spiel auf "Konto":

https://login.station.sony.com/login/launchpad/everquest/login2.jsp?lang=de&returnURL=https://eqbl.station.sony.com/launchpad/index.jsp?language=de
