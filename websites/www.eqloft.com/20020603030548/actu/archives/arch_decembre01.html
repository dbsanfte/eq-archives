PATCH D'HIER 13 Décembre

Message de Rackhyr

Le patch d'hier a un descriptif assez réduit, mais un développement assez long concernant les temps de zoning et de chargement...
Donc, pour faire court et aller à l'essentiel, le patch d'hier a amener les changements et corrections suivants :

- le chemin d'Acrylia Caverns a été corrigé
- le bug graphique des visages d'érudits a été modifié (en changeant la couleur des cheveux on changer le type de visage)
- le bug du %T a été corrigé, il faisait crasher certains joueurs
- plusieurs bugs provoquant un crash ont été corrigés

Concernant les temps de chargement et de zoning, le dernier patch vous permet maintenant de mettre certaines textures dans votre cache machine. Ainsi, quand vous zonez il ne faut pas charger à chaque fois les textures de la nouvelle zone qui se trouvent dans vos RAM. Cette option, si vous l'utilisez, demande un espace disque de 1.5 Go au maximum, plus vraisemblablement 500 Mo au moins. A sa première utilisation le temps de chargement sera plus long dans la mesure où les textures seront installées mais plus rapide dès la seconde utilisation.

Pour activer cette option, vous avez deux possibilités :

- dans le menu d'options du jeu vous pouvez cocher " Use Texture Cache "
- dans eqclient.ini du répertoire EQ vous pouvez ajouter la ligne suivante : " TextureCache=TRUE "

Notez bien que cette option est surtout utile aux utilisateurs disposant de 256 SDRAM et de beaucoup d'espace disque. Plus vous aurez de SDRAM, moins l'option en question aura d'utilité pour vous.

Message de Rackhyr

Si jamais vous rencontrez de problèmes de connexion aux serveurs de jeu après la fin de ce patch, voici peut être une solution...
La configuration de vos dll graphiques dans le répertoire EQClient.ini doit afficher "GraphicsDLL=EQGfx_Dx7.dll" quelque soit le serveur sur lequel vous jouez et quelque soit la version de directX que vous utilisez...
