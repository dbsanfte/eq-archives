  PATCH CLAVIER (QUERTY > AZERTY)
 

Marre de taper "sqlut cq vq" au lieu de "salut ca va" ?! Alors téléchargez ce patch
(32ko seulement! ) qui va convertir EverQuest au format de votre clavier! Veuillez choisir votre pays :

  • Clavier Belge
  • Clavier Français Canadien
  • Clavier Français
  • Clavier Suisse (Français)

Installation : téléchargez le fichier qui correspond à votre pays, placez le dans votre répertoire EverQuest, double cliquez dessus.... c'est tout! Ce patch va juste créer un fichier "keyboard.txt" ou le modifier s'il existait déjà avant.
  UTILITAIRES
 

Profitez pleinement de cette petite liste d'utilitaires pour EverQuester confortablement installé dans votre fauteuil ! EQLoft vous assure qu'ils sont tous reconnus par Verant et que vous ne courez donc aucuns risques si vous les utilisez.

  • EQClass, le programme amusant pour savoir quelle classe/race de personnage jouer selon les réponses données à un questionnaire.
  • EQEditor, pour changer la couleur des textes dans votre interface de chat (fait une modification dans votre fichier EQClient.ini).
  • EQHell, un utilitaire pour calculer le nombre de mobs à tuer pour finir votre Hell Level...
  • EQNumbers, un programme qui permet de calculer sa mana et ses dommages maximums avec n'importe quel type d'arme.
  • iSiloW32, ce petit programme vous permet de convertir les fichiers docs en fichier palm... Indispensable pour se construire une petite BDD personnelle !
