- Zahlung -

Diese Seite wird Sie durch den Zahlungsvorgang von EverQuest führen.

Sollten Sie trotzdem generelle Probleme mit der Anmeldung haben, dann fragen Sie auf unserem Technischen Forum nach. Falls die Probleme jedoch Ihr Konto betreffen, wie zum Beispiel die Meldung, dass Ihr Konto auf Provisional Expired gesetzt wurde, dann kontaktieren Sie bitte unseren Technischen Support.

Wenn Sie ein neues EverQuest Konto erstellt haben werden Sie bei dem Versuch sich in EverQuest einzuloggen den Hinweis erhalten, dass Sie noch kein gültiges Abonnement haben. Um ein solches einzurichten, klicken Sie einfach auf "Jetzt".

 

Ein Hinweis erscheint, der Ihnen mitteilt, dass Sie EverQuest jetzt auch in Ihrer Landeswährung bezahlen können, wenn Sie in einem der aufgezählten Länder leben. Bestätigen Sie diesen Hinweis mit einem Klick auf "Anmelden".

 

Ein weiterer Hinweis erscheint, und nachdem Sie auf "OK" klicken, wird sich EverQuest schließen und Sie werden auf die Internetseite von Station.com weitergeleitet.

 

Auf der Station.com Seite werden Sie aufgefordert, Ihren Stationnamen und Ihr Passwort einzugeben. Dies sind Ihr EverQuest Login Name und das von Ihnen bei der Kontoerstellung gewählte Passwort. Klicken sie anschließend auf "Sign in".

 

Auf der nächsten Seite sehen Sie alle Station.com Spiele aufgelistet, bei welchen Sie Konten haben. Klicken Sie hier auf "EverQuest", um fortzufahren.

 

Sie sehen nun den derzeit gewählten Zahlungsplan. Sollten Sie gerade ein neues Konto registrieren, wird standardmäßig "1-month" angegeben. Fahren Sie mit Klick auf "Subscribe now" fort.

 

Wählen Sie auf der nun folgenden Seite die gewünschte Währung aus und den Zahlungsplan, der Ihren Wünschen am ehesten entspricht. Geben Sie anschließend die e-Mail Adresse an, die Sie auch bei der Erstellung des EverQuest Kontos angegeben haben. Bestätigen Sie Ihre Eingaben mit Klick auf "Change subscription".

 

Ihnen werden nun die Nutzungsbedingungen von Station.com angezeigt. Sie müssen diesen Nutzungbsbedingungen, nachdem Sie sie gelesen haben, mit Klick auf "Accept" zustimmen, um fortfahren zu können.

 

Ein weiterer Klick auf "Accept" bringt Sie auf die Internetseite von Global Collect.

 

Hier können Sie in der linken unteren Ecke die von Ihnen bevorzugte Sprache wählen.

 

Falls Sie EverQuest per Einzugsermächtigung bezahlen wollen, klicken Sie auf "Einzugsermächtigung" oder "Direct Debit" und geben Sie anschließend die erforderlichen Daten ein. Mit Achtername ist Ihr Nachname gemeint. Bestätigen Sie Ihre Eingabe mit Klick auf "Nächst".

[Bild vergrößern]

 

Sollten Sie EverQuest mit Kreditkarte bezahlen wollen, wählen Sie durch Klick auf das Pfeilsymbol neben dem Kreditkarte Knopf Ihre Kreditkarte aus. Geben Sie dannach die nötigen Daten ein und bestätigen Sie sie mit einem Klick auf "Nächst".


[Bild vergrößern]

Folgen Sie dannach den weiteren Anweisungen auf der Global Collect Seite, um Ihr Abonnement erfolgreich abzuschließen.
