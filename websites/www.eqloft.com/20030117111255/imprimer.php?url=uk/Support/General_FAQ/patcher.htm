- EverQuest Patcher Help -
Important Note: If you are unable to download files with the new patcher, download this file to your EverQuest directory (default location is C:/Program Files/EverQuest/) and run it. You must have Internet Explorer installed on your system for the EverQuest patcher to work properly.

There are two things that are vitally important about the EverQuest Patcher. One, is its functionality. Not only do you have the ability to customize the patcher for your personal preferences, this patcher takes advantage of differential patching technology. Say for instance that we created an addendum to a certain file. Let's use EverQuest.exe as an example. In the past, we would have downloaded to you a brand new EverQuest.exe file. Now, with differential patching, we are only downloading the appended part of the EverQuest.exe file thus shortening the download process.

And the second thing vitally important, this new patcher is much better looking than the previous one. As an added feature in the near future, the EverQuest Patcher will allow users to create their own skins or download some special skins that we create to customize the look of their patcher.

This page has been created to help everyone get accustomed to the new patcher. Below, you will find some information on the new options and how they work with each other.


Checking Files
Like the previous EverQuest patcher, once you double click the EverQuest shortcut on your desktop, this new patcher launches and begins scanning for files that you need.


Patching
Once the patcher determines that your game should be updated, files are downloaded to your computer. The experience bar indicates how much of the file downloading process has occurred. The blue line will tell you how much of the current file has been downloaded, and the orange line will keep you informed of the overall status of the entire download. Once a couple of files have been downloaded to your system, a numerical time will appear below the experience bar estimating how long it will take to complete the download process.


Play now enabled
Once all files are downloaded, you can click the Play button to enter the game. If you notice the AutoPlay button, you will see a check box. Put a check in this box and you will not have use the Play button. Once you receive any needed files, you will be automatically taken to server select to begin playing. If Autoplay is selected and you wish to deselect while files are still being transferred to your system, simply hit the Cancel button. The transfer will be paused so that you can deselect Autoplay. To resume downloading, just use the Download button and the process will begin where it left off.


Download button enabled
The Download button will be enabled depending on your choices under Advanced options. Keep reading for information on using Advanced options.


Advanced Dialog Box
The Advanced dialog box is accessible by clicking on the Advanced button. This will allow EverQuest players to modify their downloading options. The first frame, "Package Downloads" indicates what EQ expansion packs you currently have installed on your system. Occasionally, a patch is offered that is specific to a certain expansion and can only be downloaded by choice. For example, say there are some new sounds added that are specific only to Kunark. To obtain these new sounds, you will need make sure the box for Kunark is checked so that these files can be received. By putting a check in the appropriate boxes, you can stop wondering if you received the most current patch for an expansion. Click on the " i " next to each package to read the most current patch message for that package.

The "Environment" frame will allow you to choose where you want to play. You can choose to play on the Standard servers like Karana or Xegony, or you can choose to play on the Test server and help us check out some new features that might one day find themselves on the Standard servers.

Also in this frame is the option to auto download files. Choosing this option will save you the extra step when launching the game by automatically downloading any files you may need. Deselect this option and manually download needed files by clicking on the Download button when it is activated.

Once you change any option, you can Save those options or cancel and start over. If you think one of your EverQuest files may be corrupted, then use the Complete File Check to run a complete scan of every file in your EQ directory.

General Issues

All anti-virus software should be disabled before running the patch.

Internet Explorer Issues

Set Dial Up Settings
  • Under tools in the menu, click on Internet Options

  • Click on the Connections tab

  • Highlight your dialup and click the "Settings " button

  • At the bottom of the Settings page, uncheck "Do not allow Internet Applications to use this connection"

  • Click OK

  • Uncheck "Perform system security check before dialing"

  • Click OK
Set Security to "Medium"

All Internet Explorer Security settings should be set to "Medium." To do this, follow these instructions:
  • Under Tools in the menu, click on Internet Options

  • Click on the Security tab

  • Click the "Custom Level " button

  • In the pull down menu under "Reset custom settings," choose Medium

  • Click Reset

  • Click OK
Use HTTP 1.1

Follow these instructions to use HTTP 1.1:
  • Under Tools, click on Internet Options

  • Click on the Advanced tab.

  • Under HTTP 1.1 Settings, check "Use HTTP 1.1" (unless you are running a proxy).

    • If you do not see this option, you should update your version of Internet Explorer. This can be done through Microsoft Windows Update (http://windowsupdate.microsoft.com/).
