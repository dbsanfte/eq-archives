- Alt Tab sous Windows XP -

EverQuest n'est pas prévu, à la base, pour permettre de sortir du mode plein écran. A l'origine, Verant Interactive avait décidé qu'au nom de l'immersion, c'était une bonne idée. Au bout de plusieurs années les joueurs ont préféré avoir l'option activée, et SOE avait décidé de leur accorder. Cependant, pour des raisons techniques que seul un Gnome Technicien comprendrait, le raccourci "Alt + Tab" qui permet de passer rapidement d'une application à une autre, ne fonctionne pas sous Windows XP (uniquement XP).

Microsoft a diffusé un patch à ce sujet. La page pour le télecharger est la suivante : http://go.microsoft.com/fwlink/?linkid=9513. Le lien direct pour le patch est le suivant : http://download.microsoft.com/download/a/6/f/a6f42077-eae3-4d4c-8df6-7ac402594127/act30pkg.exe.

Une fois le patch installé et votre PC redémarré, il faudra lui indiquer qu'il faut autoriser EverQuest a faire du alt-tab. Pour cela allez dans le menu "Démarrer > Programmes > Windows Application Compatibility Toolkit" et lancez le lien "Compatibility Administrator Manager". Vous vous trouverez devant la fenêtre suivante.

 

Une fois le logiciel lancé, il faut rechercher dans la base de donnée le logiciel "EverQuest". Pour se faire allez dans "Search / Query database", ou tapez le raccourci CTRL+F. Une nouvelle fenêtre s'affichera. Tapez "everquest" dans le champ approprié puis cliquez sur "Find now". Le résultat devrait être semblable à l'image ci-dessous :

 

 

Il faut maintenant double cliquer sur l'entrée (surlignée en bleue dans l'image ci-dessus) pour pouvoir accéder à l'entrée dans la base de donnée. Ceci ramènera la fenêtre principale au premier plan, avec l'entrée EverQuest dans la partie droite du logiciel. Il faut maintenant faire un clic-droit sur l'entrée, et cliquer sur "Disable entry", comme indiqué sur l'image ci-dessous :

 

 

Ceci activera la possibilité de "Alt Tab" dans Windows XP. Pour quitter le programme allez dans "File > Exit".
