- Fenster -


Die Benutzeroberfäche von EverQuest besteht aus einer Vielzahl von Fenstern. Im Folgenden werden die Fenster, die Sie beim ersten Einloggen, und ihre Funktionen kurz beschrieben. Klicken Sie einfach auf eines der Fenster um nähere Informationen darüber zu erhalten.

Eine vollständige Liste aller Fenster finden Sie unter diesem Link.

Das Aktions-Fenster

Über das Aktions-Fenster können Sie alle Befehle in EverQuest festlegen und ausführen. Sie können es mit ALT-M anzeigen oder verstecken. Das Aktions-Fenster enthält vier Karteikarten.

Die erste Karteikarte enthält die wichtigsten Standard-Funktionen.

  • Der Wer-Knopf listet alle in der Zone befindlichen Spieler auf.
  • Über den Einladen-Knopf können Sie andere Spieler in Ihre Gruppe einladen und mit dem Auflösen-Knopf Ihre Gruppe verlassen.
  • Mit dem Lager-Knopf können Sie die Welt von EverQuest verlassen, vorausgesetzt Sie haben sich zuerst gesetzt.
  • Der Hinsetzen-Knopf erlaubt es Ihnen, sich zu setzen, um so schneller Trefferpunkte und Mana regenerieren zu können.
  • Mit dem Gehen-Knopf können Sie zwischen Gehen und Laufen umschalten. Laufen erlaubt es Ihnen, sich schneller fortzubewegen, in gefährlichen Gegenden ist es allerdings schwieriger, sich fortzubewegen.

 

Die zweite Karteikarte enthält die Fertigkeiten Ihres Charakters. Sie können über diese Karteikarte allgemeine Befehle, wie Orientierungssinn, Verstecken, Fallen entdecken oder ähnliches aufrufen.

Klicken Sie mit der rechten Maustaste auf einen der sechs Knöpfe der Karteikarte und wählen Sie aus der Liste die Fertigkeit aus, die Sie mit diesem Knopf verbinden möchten.

Die dritte Karteikarte enthält kampfspezifische Fertigkeiten. Die obersten Knöpfe sind vorbelegt für Nahkampfangriff und Fernangriff, die vier Knöpfe darunter können Sie mit Fertigkeiten wie Wunden verbinden, Treten oder Ähnlichem belegen.

Die vierte Karteikarte enthält eine Liste mit Verhaltens-Knöpfen. Eine nähere Erklärung über diese Verhaltens-Knöpfe finden Sie weiter unten auf dieser Seite.

Das Chat-Fenster

Das Chat-Fenster ist Ihre Verbindung zu anderen Spielern und zu den NSCs. Sprechen Sie mit anderen Spielern, indem Sie <ENTER> drücken. Geben Sie dannach den gewünschten Text ein und bestätigen Sie erneut mit <ENTER>. EverQuest verfügt über verschiedene Chat-Kanäle - eine genaue Auflistung finden Sie in der Ingame-Hilfe.


Wenn Sie mit der rechten Maustaste in die Befehlszeile am unteren Rand des Chat-Fensters klicken, öffnet sich ein Menü, in dem Sie die Sprache, mit der Ihr Charakter spricht festlegen können, den standard Chat-Kanal festlegen oder ein neues Chat-Fenster erstellen, in welches Sie bestimmte Nachrichten filtern können, ein Beispiel wäre ein eigees Chat-Fenster für Kampf-Nachrichtenn.

Das Effekte-Fenster

Das Effekte-Fenster zeigt die Zaubereffekte an, die auf Ihren Charakter wirken, wie beispielsweise Zauber, die die Trefferpunkte Ihres Charakters erhöhen.

Wenn keine Effekte auf Ihren Charakter wirken, ist das Fenster transparent.

Wenn Effekte auf Ihren Charakter wirken, sehen Sie die dem Zauber entsprechenden Symbole im Effekte-Fenster.

Wenn Sie den Mauszeiger über eines der Symbole bewegen, erscheint ein Hilfetext mit dem Namen des Zaubers.

Wenn Sie einen Effekt, den ein anderer Spieler auf Sie gewirkt hat, entfernen möchten, klicken Sie einfach mit der linken Maustaste auf das entsprechende Symbol im Effekte-Fenster. Sie können nur positive Effekte aufheben, negative Effekte, wie beispielsweise "Tollwut", können nur durch Heilung oder abwarten der Wirkungsdauer aufgehoben werden.
Mit ALT-B können Sie das Effekte-Fenster aktivieren oder deaktivieren.

Das Gruppen-Fenster

Das Gruppen-Fenster zeigt Ihnen die Namen Ihrer Gruppenmitglieder sowie deren Trefferpunkte an.

Sollte sich eines der Mitglieder Ihrer Gruppe nich in der selben Zone befinden, in der Sie sich aufhalten, so sehen Sie zwar den Namen des Mitglieds, nicht aber seine Trefferpunkte.

Sollte einer Ihrer Kameraden im Kampf gefallen sein, so sehen Sie dies ebenfalls im Gruppen-Fenster.

Sollten Sie nicht in einer Gruppe sein, haben Sie die Möglichkeit, anderen Spielern mitzuteilen, dass Sie auf der Suche nach einer Gruppe sind, indem Sie auf die kleine Lupe in Ihrem Gruppenfenster klicken. Dadurch öffnet sich das Gruppensuch-Fenster.

Sie haben im Gruppen-Fenster auch die Möglichkeit, weitere Spieler in Ihre Gruppe einzuladen (wenn Sie der Anführer Ihrer Gruppe sind) beziehungsweise aus einer Gruppe auszutreten.

Sie können sich Ihr Gruppen-Fenster mit ALT G anzeigen lassen beziehungsweise verstecken.

Das Helfer-Fenster

Im Helfer-Fenster befindet sich eine Trefferpunkte-Anzeige wie im Spieler-Fenster. Außerdem finden Sie dort die Befehle, die Sie Ihrem Helfer geben können.

  • Angreifen läßt Ihren Helfer das gewählte Ziel attackieren
  • Folgen befiehlt Ihrem Helfer, Ihnen zu folgen
  • Verspotten weist Ihren Helfer an, den Gegner zu verspotten und so von Ihnen abzulenken
  • Hier lässt Ihren Helfer den Ort, an dem er sich gerade befindet, bewachen, bis er andere Befehle erhält
  • Setzen lässt Ihren Helfer die Position ändern, sodaß er seine Trefferpunkte schneller regeneriert
  • Zurück befiehlt Ihrem Helfer, den Angriff abzubrechen
  • Geh weg schickt Ihren Helfer an den Ort zurück, von dem Sie Ihn herbeigerufen haben.

Das Helfer-Fenster erscheint automatisch, wenn Sie einen Helfer herbeirufen. Sie können es alternativ über den Window-Selector oder die Tastenkombination ALT-F1 aufrufen.

Die Ingame-Hilfe

Die Ingame-Hilfe bietet Ihnen detailierte Informationen rund um die Welt von EverQuest.

Sämtliche Aspekte, von Bewegung bis Zaubern, finden Sie darin erklärt. Durch die Ingame-Hilfe können Sie wie durch eine Internet-Seite navigieren. Sie können die Ingame-Hilfe aufrufen, indem Sie auf das " ? " Symbol im Window-Selector am oberen Rand Ihres Bildschirms klicken.

Der Kompass

Der Kompass zeigt Ihnen die Himmelsrichtung an, in die Ihr Charakter blickt. Die Zuverlässigkeit des Kompasses ist abhängig vom Wert Ihrer Orientierungssinn Fertigkeit. Je besser Sie diese Fertigkeit beherrschen, umso weniger schlägt der Kompass aus. Haben Sie die Fertigkeit gemeistert zeigt Ihnen der Kompass immer exakt die Richtung an, in die Sie blicken.

Sie können den Kompass mit der Tastenkombination ALT-D aktivieren oder deaktiveren.

Das Schnelltasten-Fenster

Mit den Schnelltasten, die in dem Schnelltasten-Fenster angeordnet werden können, können Sie die Fertigkeiten Ihres Charakters einsetzen.

Wählen Sie einfach eine der Fertigkeiten aus dem Aktions-Fenster aus, halten Sie die Maustaste mehrere Sekunden gedrückt, bis sie auf Ihrem Mauszeiger erscheint und klicken Sie dannach auf einen der leeren Plätze im Schnelltasten-Fenster.

Ihnen stehen zehn Mal zehn Schnelltasten zur Verfügung.

Sie können die Schnelltasten mit den Tasten " 1 " - " 0 " auf Ihrer Tastatur aktivieren, 1 steht für die Taste oben links, 2 für oben rechts und so weiter, bis zur 0 für unten rechts. Mit SHIFT + " 1 " - " 0 " können Sie zwischen den verschiedenen Seiten des Schnelltasten-Fensters durchschalten.

Sie können das Fenster über den Window-Selector oder die Tastenkombination ALT-H aufrufen.

Das Spieler-Fenster

Das Spieler-Fenster zeigt Ihnen alle wichtigen Informationen über Ihren Charakter an.

Der rote Balken repräsentiert die Trefferpunkte Ihres Charakters. Der blaue Balken steht für die Manapunkte des Charakters (ein Charakter, der über kein magische Begabung verfügt, wird hier keinen Balken angezeigt bekommen). Der gelbe Balken schließlich steht für die Ausdauer Ihres Charakters, die durch anstrengende Handlungen sinken wird.

Des weiteren sehen Sie Ihren eigenen Namen beziehungsweise Ihren Nachnamen, falls Sie Level 20 erreicht haben und sich einen Nachnamen ausgesucht haben.

Sie können das Spieler-Fenster mit der Tastenkombination ALT-Z akivieren oder deaktivieren.

Die Verbindungs-Anzeige

Die Verbindungs-Anzeige, auch Lag-O-Meter genannt, gibt Ihnen auskunft darüber, wie gut Ihre Verbindung zu dem Server ist, auf dem Sie sich derzeit befinden.

  • Die erste Zahl ist der sogenannte Ping. Dieser gibt die Verzögerung zwischen Ihrem Computer und dem Server in Millisekunden an.
  • Die zweite Zahl ist der Datenverlust in Prozent auf dem Weg von Ihrem Computer zum Server.
  • Die dritte Zahl ist der Datenverlust in Prozent auf dem Weg vom Server zu Ihrem Computer.
  • Die vierte Zahl gibt Ihnen die Anzahl der Pakete an, die vom Server zu Ihrem Computer gesendet werden. Je mehr dargestellt werden muss, desto mehr Pakete werden gesendet. Wenn Sie sich also in einer Gegend mit vielen Spielern aufhalten ist es nichts ungewöhnliches, wenn diese Zahl auf 1500 oder höher steigt.

Sie können die Verbindungs-Anzeige mit der F 11 Taste aktivieren oder deaktivieren.

Der Window-Selector

Der Window-Selector, der sich am oberen Rand Ihres Bildschirms befindet, erlaubt es Ihnen, Ihre Benutzeroberfläche nach Ihrem Geschmack zu verändern.

Klicken Sie auf eines der Icons, um das entsprechende Fenster zu aktivieren und ein zweites Mal, um es wieder zu deaktivieren.

Halten Sie den Mauszeiger für ein paar Sekunden über eines der Icons, so erscheint ein Hilfetext, der Ihnen anzeigt, welches Fenster mit diesem Knopf aktiviert wird. Der Hilfetext enthält außerdem das Tastaturkürzel, mit dem Sie das Fenster aktivieren können, beispielsweise ALT O für das Optionenfenster.

Das Zauber-Fenster

Im Zauber-Fenster befinden sich die Zauber, die Sie vorbereitet haben.

Wenn Sie sich keine Zauber eingeprägt haben, sind alle acht Plätze des Fensters leer.

Um sich Zauber einzuprägen können Sie auf das Zauberbuch-Icon am unteren Rand des Zauber-Fensters klicken. Dadurch öffnet sich Ihr Zauberbuch. Wählen Sie einen Zauber aus und klicken Sie dann an die Stelle im Zauber-Fenster, an der Sie den Zauber einprägen wollen.

Alternativ können Sie sich Zaubersprüche auch über das Zauberspruch Kontext Menü einprägen. Klicken Sie dazu mit der rechten Maustaste auf einen der leeren Plätze im Zauber-Fenster.

Es öffnet sich nun ein Fenster mit den verschiedenen Kategorien an Zaubern, die Sie besitzen und in Ihrem Zauberbuch aufgeschrieben haben. Bewegen Sie den Mauszeiger über die entsprechende Kategorie, so öffnet sich ein weiteres Fenster, in dem Sie die Auswahl weiter einschränken können und schließlich den Zauber auswählen können.

Sie finden neben dem Namen des Zaubers auch die Erfahrungsstufe, die erforderlich ist, um ihn zu sprechen. Die Farbcodierung der Namen ist mit denen der weiter unten erklärten Symbole identisch.

Eine weitere Funktion des Zauberspruch Kontext Menüs sind die sogenannten Zauberspruch Favoriten. Klicken Sie mit der rechten Maustaste auf das Zauberbuch Symbol, so öffnet sich ein Fenster, in welchem Sie die bestehende Auswahl an Zaubern als Favoriten speichern können, einen bestehenden Favoriten laden oder löschen können.

Sie können bis zu zehn verschiedene Zauberspruch Favoriten speichern.

Wenn Sie sich Zauber eingeprägt haben, erscheint ein kleines Symbol an dem entsprechenden Platz. Um einen Zauber zu sprechen, klicken Sie einfach auf das entsprechende Symbol.

Halten Sie den Mauszeiger über eines der Symbole, so erscheint ein Hilfetext mit dem Namen des Zaubers.

Der Hintergrund der Symbole bestimmt, auf welches Ziel dieser Zauber gesprochen werden kann.

  • Gelb bedeuted, dass der Zauber nur auf Sie selbst gesprochen werden kann.
  • Rot bedeuted, dass Sie den Zauber auf ein beliebiges Ziel sprechen können (Zauber die Schaden verursachen, können Sie nur in speziellen Gebieten auf andere Spieler wirken).
  • Violette Zauber wirken auf die gesamte Gruppe
  • Blaue Zauber wirken auf eine Fläche um den Zaubernden herum
  • Grüne Zauber wirken auf eine Fläche um das gewählte Ziel herum
  • Orangene Zauber wirkenauf eine Fläche um das Ziel herum und fügen in drei Wellen Schaden zu.
Das Zauber-Fenster können Sie über den Window-Selector oder die Tastenkombination ALT-S aufrufen.

Das Ziel-Fenster

Im Ziel-Fenster sehen Sie eine Trefferpunkte-Anzeige des aktuell gewählten Ziels.

Mit F1 können Sie sich selbst als Ziel wählen, mit F2 - F6 können Sie Ihre Gruppenmitglieder von oben nach unten anwählen.

Drücken Sie eine der F-Tasten zweimal, so wählen Sie Ihren Helfer an, oder den Helfer eines Ihrer Gruppenmitglieder.

Mit F7 wählen Sie den nähesten Spieler an und mit F8 können Sie den nähesten NSC anwählen.

Mit der Tabulator-Taste können Sie zwischen sich selbst und dem zuletzt gewählten Ziel hin- und herschalten.

Das Ziel-Fenster können Sie mit der Tastenkombination ALT-T aktivieren oder deaktivieren.
