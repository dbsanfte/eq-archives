- Système d'attaque -

Le système d'attaque est relavivement simple. Nous allons présenter dans cette partie les différentes méthodes d'attaque : auto-attaque des mêlées avec les armes et les sorts (mémorisation et lancement de sorts).

Attaque de mêlée (avec armes au contact et à distance).

Pour attaquer un ennemi avec vos armes au contact, vous devez le cibler en cliquant sur la cible et le monstre que vous devez attaquer. Vous pouvez maintenant cliquer dans la fenêtre d'action sur l'icône représentant les deux armes croisés. Le combat est représenté par la touche avec l'arme que vous portez et s'appelle attaque de mêlée. Vous pouvez déclencher cette attaque automatiquement ou en cliquant par défaut sur le bouton A Votre personnage attaquera la cible sans s'arrêter. Si vous désirez cesser de taper vous devez reappuyer sur la touche A ou sur le bouton d'attaque de mêlée. Il vous faut tout de même être proche de votre cible pour pouvoir l'attaquer.

Vous pouvez également attaquer un adversaire à distance. Pour cela, vous devez avoir une arme de jet (comme un arc) et des munitions. Le bouton pour attaquer avec des armes de distance est appelé Attaque à distance et est aussi dans la fenêtre d'actions. Notez que vous devez être à une certaine distance pour exécuter une attaque à distance. Trop près, vous n'aurez pas l'espace nécessaire pour lancer vos projectiles ou vos flèches. Trop loin, vous ne verrez plus la cible ou votre arme à distance ne peut pas tirer si loin.

Attaque à l'aide de la magie (lancement des sorts).

Pour mémoriser un sort, vous devez ouvrir votre livre de sorts et chercher la page du livre sur laquelle le sort que vous voulez a été mémorisé. Puis cliquez gauche sur le sort en question pour le sélectionner et déplacez l'icône qui apparaît sur la barre d'incantation sur la gauche de votre écran. Vous avez dans cette barre 8 gemmes sur lesquelles vous pouvez mettre un sort. Cliquez gauche sur l'emplacement où vous voulez mettre le sort.

Pour lancer un sort, vous devez d'abord sélectionner une cible puis cliquer sur la gemme approprié du sort. Une petite fenêtre avec le temps d'incantation apparaîtra. Pour lancer le sort une deuxième fois, vous devez attendre la fin de l'incantation du premier sort.

Les gemmes de sorts ont différentes couleurs. Vous trouverez ci-dessous l'explication de chaque couleur :

* JAUNE - Ce sort ne sera lancé que sur vous-même.

* ROUGE - Ce sort peut être lancé sur n'importe quelle cible (ennemie ou amie). Ce sont soit des sorts bénéfiques, soit des sorts de dégâts.

* VIOLET - Ce sort renforcera toutes les personnes de votre groupe et qui sont à votre portée de lancement.

* BLEU - Ce sort fera des dégâts sur tous les ennemis qui sont dans un environnement proche de vous.

* VERT - Ce sort fera des dégâts sur tous les ennemis qui sont dans un evironnement proche de votre cible principale.

* ORANGE - Ce sort fera des dégâts sur tous les ennemis qui sont dans un environnement proche en utilisant 3 vagues de dégâts en général.

Les sorts faisant des dégâts ne peuvent être lancés sur les autres joueurs que dans les zones Joueur contre Joueur. Les sorts de type bénéfique ne peuvent être lancés que sur les joueurs et les serviteurs.
