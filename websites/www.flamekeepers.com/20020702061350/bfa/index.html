Sneak a peak at the Bazaar traders function
Posted by: ghazban on Friday, June 28 @ 14:24:47 CDT
Just in case anyone wants to know, the bazaar is looking far cooler that I expected it to with the following functions I pulled from the EQUI_BazaarSearchWnd.xml file off the test server... Query the Bazaar for a list of items that match your search criteria. Search by partial item name. List all items for sale by selected player. List all items that can be used in this slot. List all items that have this statistic. List all items that can be worn by this race. List all items that can be worn by this class. Sort your list of items. No running around and looking at what people have to sell apparently, it queries the zone for the items YOU want! None of this is official of course, but this is definitely the way it looks =)
