To submit your mod to our site, please e-Mail Kluz!

Instructions for loading skins/mods:

1. make a directory under \everquest\uifiles. Name it whatever you wish.

2. Copy everything from \everquest\uifiles\default to \everquest\uifiles\(your new folder)

3. Right click on which ever link below that you want and save it to \everquest\uifiles\(your new folder) Replace the file that is there with the new one.

4. Log on to the test server

5. use /loadskin (folder name)
