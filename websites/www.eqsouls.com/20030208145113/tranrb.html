Transparent Table Backgrounds

Its accualy really easy to do this. I have seen some tutorials to where you use a checkerboard transparent .gif, those really didn't work for me. I know there had to be a better way to do it then that. The light bulb went off and here it is!. Just use this code in your table after your background color or background image and there you have it.

style="filter:alpha(opacity=60);-moz-opacity:60%;"

You can get creative with that and make some nice tables. You can use it in the table or table data tags. here is an example of one and the code I used to make it.

<table background="./images/info-bg.gif" valign="top" align="center" width="400" height="200" border="2" cellpadding="5" cellspacing="5" bordercolor="000000">
<tr>
<td valign="top" bgcolor="505050" style="filter:alpha(opacity=60);-moz-opacity:60%;">
<p> And there you have it. It really is that easy, and looks much better then a checkerboard transparent .gif!</p>
</td>
</tr>
</table>
