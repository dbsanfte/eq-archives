Noctix
I recently built a new pc but up until about 2 weeks ago, I was running 2 instances of EQ every night on a 1ghz 640mb ram machine with an OLD hard drive and a 32mb geforce 2. So your machine will work fine.

Here are some pointers:
1) Create a seperate eq folder for each instance. You can install once and then just cut and paste the entire folder to a new location for each instance. If you have multiple hard drives on seperate channels, then put each eq folder on a seperate hard drive.
2) Put a file called "nosound.txt" in all except your main instance. Having sound coming from multiple instances causes problems. You can do the nosound thing for every instance if you don't care about sound.
3) Start out with the abolute minimum graphics settings. Go into options before log in and turn off everything that uses system resources. Turn off all new character models. Then, in game, turn off all particle fx, 3d sky, and lower your clipping plane as far as you can tolerate. Once you make sure you can run with these minimum settings, add the things you want back in one at a time to see what you can handle.
4) WinEQ and EQW are both widely used. I have tried both and both work well for running multiple instances. However, they often break with new patches and you have to do most of your own technical support. EQ comes with a default windowed mode and yesterday's patch increased the performance of the default windowed mode. I suggest giving that a try before bothering with 3rd party software. It works well for me.

That's the main stuff I can think of right now, other than using macros and stuff for controlling 2 characters. If you have other questions though, please ask. I 3box every time I play and know several others who play multiple characters fulltime.

Noctix
There is a command to release your mouse from an EQ window. I want to say "Ctrl-shift-S" but that might not be correct. If you look in your keyboard settings in the options menu, you will see mouse release as one of the options. I can't remember the default because I have my mouse release mapped to the ~ key. As an alternative, you can hit your windows key to open your start menu, this will also release your mouse.

EQW and WinEQ both have their own mouse release commands. Check their documentation for details.

I believe WinEQ is currently more popular than EQW and the message boards are more active. Of the two though, I prefer EQW because I've had fewer issues with patches breaking it.

I am currently using the default windowed mode though. I used it last night after the patch and it worked great.
