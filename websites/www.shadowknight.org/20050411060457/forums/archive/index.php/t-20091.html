While solo'ing using the kite method (either fear or aggro style), I often founnd myself guessing when to recast my darkness spell. I created a hotkey that might be usefull to others

/cast 1 - (spell slot you have darkness mem'd)
/pause 450 - (this will time 45secs)
/1 45 seconds left (You must creat a private channel for this command to work)
/pause 300
/1 CAST DARKNESS NOW!!!

Rinse and repeat

*Note - while using this all other hotkeys will become disabled untill the last line (/1 CAST DARKNESS NOW!!). If you fizzle you must click the spell gem, reclicking the hotkey will not work
