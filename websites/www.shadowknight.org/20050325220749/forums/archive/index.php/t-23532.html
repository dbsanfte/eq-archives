Nafarious,

I've actually played around with this issue a bit more and have found that more RAM may be greater then anything else in improving the performance of EQ2 (I know I said previously that this wasn't the case, but whatcha gonna do:-). It seems EQ2 is leaking RAM every time you zone and not refreshing the RAM it has already used, which causes the amount of RAM used to grow the longer you play. I tested this by turning off my virtual memory and got much better performance out of the game (well until I ran out of RAM and the game shut down:-)....Once the RAM starts to get used up, the PC turns to virtual memory which is god awful slow cause it's reading from the hard drive. Only way I can think to improve this is to get a seperate high speed HD (10K) and dedicate the VM to it alone or put about 2+ gigs of RAM in your system, shut off VM and hope you don't use up all the RAM. BTW, I ctrl+alt+del'd to task manager in game to check my memory usage (when the game started running poorly) and found that I was using 900,000 physical memory and 1,800,000 virtual memory, which is utterly insane....but when I first log in, I use 300,000 physical and 300,000 VM (for EQ2 alone that is).

So, a new vid card probably isn't going to fix performance issues unless you upgrade all your hardware, but a good vid card makes this game look far better and, IMO, is worth the $.

Edit - I O/C'd my vid card to about where a 6800 ultra would be (410mhz 1100mem) and only got about 200 extra on my benchmark and minimal, if any, difference in game and since I have only the 2 stock fans on the 6800GT, I decided to stick with 370mhz 'cause I don't want to piss away $350, especially considering BFG offers the best warranty in the business(lifetime unless o/c'd):-)
