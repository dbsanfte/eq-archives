Inferna
Turns out it was not a bug but a feature


I have attack keyed to Z, but I use ~ for the "toggle between last two targets" key. I use it a lot on my cleric as I melee a lot (and it's a lot faster than /assist if I know who's got agro), and couldn't figure out what was wrong when it wouldn't work.

Find the "Target nearest enemy" key (should be F8 normally), and you'll see the ` (AKA, the ~ key) is set as the alt key. Click on the alt key spot and hit Esc...you'll be back to normal.


Remapped F8 and no more problems
