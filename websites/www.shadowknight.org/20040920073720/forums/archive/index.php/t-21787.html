Hello shadow knights,

The Community team is looking to set up a correspondent for each class. We would like to give you the opportunity to let us know if you think this is something you would be interested in doing. The class correspondents would be responsible for helping the Community team collect information about the class to pass along to the development team. This could range from bugs found, class balance issues, or general questions about the class.

If this sounds like something you would enjoy doing and you have a good feel for how your class works please send me a PM with your character’s name, the server you play on, the level of your main character, the name of your guild and its web page address (if applicable), the names of any other EQ fan sites you regularly post on and the handle you use on those EQ fan sites. Please also give a brief description on why you would like to be your class’s correspondent.

Thanks!!
