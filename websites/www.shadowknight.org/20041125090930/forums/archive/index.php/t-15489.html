Sarrasis Darkmist
Got this from FoH boards...

Sorry for being quiet on this for so long.

We've got a potential fix for her that we're going to try to roll out tomorrow.

Long story short is that there was a pretty bad memory corruption bug that occurred as a result of some optimization that had to happen as a part of Lost Dungeons.

If you leave one or more of the set of three adds alive, the event won't crash the zone. Some memory is being overwritten on the server when the last of the duplicate creatures dies. Any time after that, the zone is primed to crash.

Thanks to all the folks who mailed in their experiences that helped nail this one down.

- Scott Hartsman
Technical Director, EverQuest
