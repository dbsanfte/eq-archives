If an admin with the correct access wants to help you out and let you know who has you on Ignore, they can go to their admincp / Import and Maintenance / Execute SQL Query and paste the following:
SELECT DISTINCT user.username FROM user, usertextfield WHERE user.userid = usertextfield.userid AND usertextfield.ignorelist = 233 ORDER BY user.username DESC;

This will produce a list of exactly what you are looking for.
