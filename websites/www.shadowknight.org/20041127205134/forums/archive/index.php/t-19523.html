I somewhat ineffectively two and 3 box at times (all on seperate machines). I know one person who can 4 box better than I two box.

Good Luck and enjoy. I suggest you start in a zone you know you can solo to learn timing and positioning etc.

Actually I just recently started dual boxing on the same machine, and I first tried it using the default windowed mode for EQ, but it was way too much of a pain changing windows. Now I use EQWindows and it's a simple key press to switch windows, and it's near-instant too, which makes it much more manageable to me. It is best to copy your EQ folder to another one though, and set your secondary box down to the lowest possible graphics settings, and turn off sound. Then set your primary one to whatever your computer can handle. Good luck.

Your absolutely wrong about ONLY being able to Alt-CTRL-S between EQW. Alt - Tab is the only thing needed. Directions for the Windows XP Alt-Tab fix are as follows:

Step One: Download the Windows Compatibility Tool Kit from

here (http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=09DAC794-B803-4B92-83C9-6A9E8B801434)

Step Two: Install it

Step Three: After installed, start the program from your start menu.Windows Application Compatibility Toolkit then in that sub menu. Compatibility Administration Tool

Step four: Select "Search" from the top bar

Step Five: Select Query Database

Step Six: In the box Search for application type in EVERQUEST

Step Seven: In the bottom box you should see the everquest search result. Double click it.

Step Eight: In the right window pane you should see at the top eqgame.exe. Right click it and select Disable Entry

Step Nine: Close out the program and start up EQ as you normally would.
