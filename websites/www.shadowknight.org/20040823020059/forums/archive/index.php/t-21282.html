Could be any number of things.


First things first, update all of your drivers. Motherboard, video card, sound card, everything possible for your computer.


Get all the latest updates for whatever OS you are currently running.


Next, uninstall EverQuest completely. Then do a fresh install of EverQuest, repatch and try again.



If that doesn't work, well if could be a BIOS problem, a hardware problem, or a software problem.
