With the introduction of DoN came the Bandolier, which allows us to switch weapons with a hotkey. This came in handy for use for swapping out Primal/Prismatics, switching between 1 and 2 hander configs, etc.

But... if you are a player without racial slam and without the 2hand bash AA and like to use a 2 hander weapon primarily but would like the ability to bash -- there is a way without using 3 hotkeys. It can be done with one hotkey, centered around the /bandolier command.

First, you need to set up your bandolier with at least 2 configurations, one for 1hand and one for 2hand. I won't go into how to do that, I'm sure everyone has played with the bandolier.

Secondly, you have to set up the bash hotkey in the Actions menu. Yes, you can create a Bash hotkey from the skills menu but in order to use bash within a macro you must set it up in the Actions box in either the Abilities tab or the Combat tab. Find an open slot, right click on it, and choose Bash. Note which number you just put the bash hotkey into. I won't go into detail on how to set that up either, but for this example, let's say it's occupying slot #1 in the Abilities bank.

Syntax: /bandolier activate name

Let's say by default you use your 2 hand weapon, that you've created a bandolier config named "2hand". We'll also assume /attack is already on when you use this macro. And we'll assume you've created a bandolier config named "1hand". Plus as mentioned above, you've set up your Bash ability as #1 on the Abilities tab of the Actions box.

Okay, now create a new hotkey macro to contain the following lines:

/bandolier activate 1hand (switches to your 1hand weapon and shield)
/pause 5 (pauses for half a second, might not be necessary. YMMV)
/doability 1 (activates the ability in Actions tab in slot 1, in this case: Bash)
/bandolier activate 2hand (switches back to your 2hand weapon and bags your 1hander and shield)

Using this macro or any other macro relying on a timed ability means you have to either be aware of the refresh time for Bash, or have the bash key itself visible on your hotkey bank, otherwise bash might not have refreshed when you press the hotkey (same with HT and other activated abilities).

I did a search for '/bandoleer' (and later '/bandolier') and nothing came up so I'm assuming it hasn't been previously discussed. I also can't take credit for discovering how to do it, a guildie played around with it (for another reason entirely since he can't bash) and figured out the syntax.

Hope this might be helpful to some.
