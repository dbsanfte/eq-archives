Fheyaran
I too just got this nasty browser hijack yesterday. The replys you have received are all good suggestions, but they need to be done in the proper order to remove this nasty bugger.

1) Download Hijack This! (http://www.spywareinfo.com/~merijn/files/hijackthis.zip) and CWS Shredder (http://209.133.47.12/~merijn/files/CWShredder.exe)
2) If using XP turn off system restore
3) Boot into safe mode
4) Run Hijack this and remove any lines that look similar to the following:
a) R1 - HKCU\Software\Microsoft\Internet Explorer\Search,SearchAssistant = file://C:\DOCUME~1\Username\LOCALS~1\Temp\sp.html
b) R0 - HKLM\Software\Microsoft\Internet Explorer\Search,SearchAssistant = file://C:\DOCUME~1\Username\LOCALS~1\Temp\sp.html
c) R1 - HKCU\Software\Microsoft\Internet Explorer\Main,HomeOldSP = about:blank
d) and any .dll in the Hijack This! log (not the dir) that is not associated with a known program and is showing up directly in the path X:\WINDOWS\system32\xxxxx.dll (the dll is randomly named by the malware)

***Username above will be the currently logged on user and substitute the appropriate drive letter for X above ***

5) Delete the appropriate .dll from the C:\WINDOWS\system32 directory.
6) Run CWS (CWS found that the hijack had actually modified my Windows Media Player and I had to remove it and reinstall it)
7) Reboot into Windows normal and make sure browser loads normally
8) Run a virus scan
9) Re-enable System Restore

This should get rid of the hijack for you.

If you are still having problems though head to the Broadband Reports security forum (http://www.dslreports.com/forum/security) and I am sure they can help.
