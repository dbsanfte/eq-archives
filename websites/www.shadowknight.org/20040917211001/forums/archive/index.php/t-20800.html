Depends on the spell, example:

Stun
Slot Description
1: Stun(4.00 sec/55) <---- Stuns up to lvl 55


Quellious' Word of Serenity
Slot Description
1: Stun(10.00 sec/65) <---- Stuns up to lvl 65
2: Decrease Hitpoints by 108
Thakalas
Stuns that don't have a level listed (they haven't added that info to everything) cap at 55. So, for instance.

Ykesha
Slot Description
1: Decrease Hitpoints by 75
2: Stun(0.00 sec)

Stuns up to 55, same as if it said Stun(0.00sec/55).
