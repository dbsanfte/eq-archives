Bolas
You'll need begging skill. When you get level 66, you'll get 5 more points with which to train, no doubt. July 27 for new expansion.

But you should be able to raise it without spending training points. You'll only raise it on mobs other than green con. So when you're in combat, practice turning off attack, hitting "beg", turning attack back on. Can't beg while attacking. That will help it go up. Pretty sure on some char I got begging "1" without spending any training points on it.

-Bolas
