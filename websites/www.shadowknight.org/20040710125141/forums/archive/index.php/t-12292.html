Linksys I assume?

Open up the HTTP config screen. (Point browser to 192.168.1.1. username admin, password admin unless you've changed it.)

Advanced Tab -> Forwarding Tab -> Port Triggering

Enter...

Everquest 1023 7000 1023 7000

Click Apply and Close....


Viola.

Krazaak
The router is configured using your browser from any PC behind it.

With a default installation, if you open Internet Explorer (or whatever) and enter http://192.168.1.1 it will bring you to the configuration tool.

Username : admin
Password : admin

There you can configure a great deal of settings. The port triggering info above should be all you need to play EQ without the constant LDs.
