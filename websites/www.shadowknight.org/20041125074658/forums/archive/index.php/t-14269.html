It would be VERY nice if we had that same ability applied to what's in our bank.

This gets pretty old after awhile and takes a considerable amount of time. If we had the ability to dump the bank content info to a file, even if it was just the item name, it would be a great time-saver. For me, anyway; (but I'm sure I'm not alone on this one).
