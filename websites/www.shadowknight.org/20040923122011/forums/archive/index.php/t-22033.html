Siilvir
- Added taunt state saving for Suspend Minion

- Added pet hold state saving for Suspend Minion

- Made it so Suspended Minions update their visibility after they are unsummoned.

- Reset bards meditate skill back to 1. Having skill > 1 didn't effect mana regeneration.

- Created four new label types which can be referenced in UI files:

EQType 128 = Mana / Max Mana

EQType 129 = Endurance / Max Endurance

EQType 130 = Group Leadership Exp %

EQType 131 = Raid Leadership Exp %

- Added checks when selling items back to adventure merchant to handle augments in the item

- If any augments have a point value, the merchant will tell you how much you will get for them

- When you try to sell an item with augments, you will see a confirmation dialog; if you accept, you will then receive the total points for the item and the augments

- If any augment does not have a point value, the merchant will not accept the item from you

- More AA Fixes
