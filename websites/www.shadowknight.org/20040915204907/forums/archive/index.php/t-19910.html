Unholy Steed / IHT / SA 3 / ToL 3 / SCF 3 / UT 3

level 1: 1.5xdam (SoC = 97)
level 2: 2.0xdam (SoC = 130)
level 3: 3.0xdam (SoC = 195)

Level 3 will make Shroud of Chaos proc for 195 pts each time.

w/ SCF 3 ... you will score crits on this for 290 pts
w/ ToL 3 ... you will score exceptional heals on this

on rare occasions you can proc a crit hit and an exceptional heal ... thus increasing your max crit ( 290 ) to a ( 580 ) pt lifetap proc.

Now, according to what the AA description is ... Weapon Affinity does NOT effect Shroud of Chaos .... but hell .. you never know what SoE's done so oh well.

WA is supposedly only supposed to increase the proc rate on weapons with their own " proc " ... So, buyer beware.
