Originally posted by KereithisFV
Should I just copy the eq directory (so as to not have to patch)? If so- where to copy to...
You can, but it just takes up extra harddisk space and forces you to download each patch twice. And you could copy it wherever you wanted to. Which directory you install(or copy) EQ into has nothing to with anything. As long as you remember <i>where</i> you install/copy it, so you can find and run it, you're fine.

Easier to use EQwindows than built in feature? if so- how do I set that up?
Some say yes, some say no. You set it up by going to <a href="http://www.eqwindows.com/">the EQW site</a> and installing it into the EQ directory you wish to use it with.

Personally, I didn't bother with copying my EQ directory. I just have the one directory, and it does have EQW installed in it. When I play, I run the standard EverQuest. If I need to box another character, I'll keep the one instance of EQ running, and fire up EQW. Works perfectly for me.

If you run into any problems with not being able to alt+tab between windows, follow the instructions on the second post of <a href="http://www.shadowknight.org/forums/showthread.php?s=&threadid=16520">this thread</a>.

Assuming Windows XP here...

Download windows compatitility toolkit...instructions here: http://www.shadowknight.org/forums/showthread.php?s=&threadid=19523
