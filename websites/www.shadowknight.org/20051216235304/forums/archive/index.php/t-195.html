HTML code...

replace every ** with a "<" and every && with a ">"


**a href="http://www.magelo.com/eq_view_profile.html?num=85354"&&Xarshaaz Dreadbones**/a&&


Then just paste that badboy in and it'll look like this


<a href="http://www.magelo.com/eq_view_profile.html?num=85354">Xarshaaz Dreadbones</a>

Also if you would rather clickin on it bring up a new window than open your profile in the current window you can add something to do that. Borrowing Ghob's notation it would look like this ...

**a href="http://www.magelo.com/eq_view_profile.html?num=85354" target=new&&Xarshaaz Dreadbones**/a&&

Which would give ya this ...

<a href="http://www.magelo.com/eq_view_profile.html?num=85354" target=new>Xarshaaz Dreadbones</a>
