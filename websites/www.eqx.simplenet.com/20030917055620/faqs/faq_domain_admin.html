1. How do I add additional domains?
If you would like to have a domain name added to the DNS servers, please email:

   servers-support@simplenet.com

with the following information:

  • IP address of server.
  • Domain name to be added.
  • Administrator Password

By default, the following records will be set up for the domain:

A record - domain.com
A record - www.domain.com
A record - ftp.domain.com
A record - mail.domain.com
MX record - mail.domain.com

If you would like any other records set up, please include them in your response.

2. How should name servers be configured for my domains?
If you are setting up a domain name as a virtual site on your dedicated server, you will need to transfer the DNS modifying the name servers.

The primary and secondary DNS servers are:

sns1.simplenet.com 209.132.1.20
sns2.simplenet.com 209.132.2.20

Please allow up to 48 hours for the domain name transfer to take place.

3. How do I look up information about my domains (whois)?
You can verify which name servers and registrar your domain name is currently hosted with by using the tool available at:

http://whois.melbourneit.com.au/

At this web page, enter the domain name you wish to check and click "Enter" or click on the "Whois" button.

Please be aware that SimpleNet has no jurisdiction over domain names registered with other services. Only you (the registrant), your current host, and your domain name registrar have the ability to modify your current domain name information.

4. What domain name information is public?
The personal contact information you provide as the registrant and administrative contact is used to register your web address. As required by ICANN, this information is publicly available and is distributed to bulk data subscribers. You can opt-out of bulk distribution lists, but you can't opt-out of the public database.
