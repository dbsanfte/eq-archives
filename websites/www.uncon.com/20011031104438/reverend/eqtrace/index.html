  • There are some issues with the Traceroute routines, if the program crashes, it's because it's trying to trace to a server whose URL has changed. To fix this problem, make sure you are using the most up to date serverlist.txt file. If this doesn't work, then open the serverlist.txt file using notepad, locate your server, and change the URL to the proper value. Server URLs are in the format "eqworld-##.989studios.com" where ## is the number of your server. The number of your server can be taken from your character's current log file, it's filename is in the format "char name_##.txt".

    • Why EQTrace?

    I wanted to teach myself about traceroutes and pings, and I was having problems connecting to EverQuest one day, so I thought to myself "Now is a good time to start that program...". So here it is!

    How to use EQTrace

    Installation

    Unzip the files into a directory of your choice. Then double-click on the Setup.exe and follow the onscreen instructions.

    Operation

    After you open the program, you should see a window that looks like this:



    Menu

    File
    Exit: This will exit the program.
    Edit
    Copy: This will copy the contents of the information area to the widows clipboard. This does nothing right now.
    Options
    Advanced: Does nothing right now.
    Settings: Allows you to change a couple of different options regarding the way the program works.
    Update Server List: As new servers are added to EverQuest, I'll update the master server list. Use this option to make sure you have the latest version of the server list.
    About: Shows you information regarding this program, and it's maker.

    Button Bar

    Link Test: After choosing a server, press this button to perform a 30 ping test of the server and it's 11 zone servers.
    DNS: After highlighting an item in the list that has an IP but no Domain Name, you can press this button to re-check for a DNS entry.
    WhoIs: After highlighting an item in the list that has a Domain Name, press this button to perform a WhoIs lookup.
    Ping: Select an item in the list, and press this button to perform a ping.
    Trace Route: After selecting an server from the list, press this button to perform a trace route.
    Ping all Games: Press this button to ping all game servers.
    Poll: Press this button to continuously poll all items that are listed.
