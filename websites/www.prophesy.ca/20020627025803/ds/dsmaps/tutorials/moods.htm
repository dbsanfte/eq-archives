So you want to make it so it's snowing outside and NOT snowing inside? Well here is how you do it. This will give you a basic idea of how moods work.

First, create two new trigger_change_mood boxes, found in game objects/gizmos/triggers.

Next, right click on the trigger inside the door and select properies. This will be the mood trigger that turns the snow off. In the value field beside "Mood Name", which is in the second big box in the middle from the top, enter the name of the snowless mood, which is "multiplayer_world_snow_town_2". The default bounding box is too big, so let's make it smaller. On the first big box in the middle from the top, there are 3 fields, "half_diag_X" "half_diag_Y", and "half_diag_Z". In the Value field beside each box, put 0.5 for X, 1 for Y, and 1 for Z. Repeat this for the box outside the door, but put the mood "multiplayer_world_snow_town_1" instead. Now move the boxes so the inside one is just inside the door and the outside one is just outside the door, like this, rotating them as necessary.

That's it! Snowing outside, not snowing inside!
