This is where all maps and programs go. I have selected a few programs you may find useful, and have put my maps I've made on as well. ENJOY!

MAPS - Download the COMPILED ones into your "my documents/maps" folder to play em!

Download the UNCOMPILED ones into your "my documents/bits/world/maps" folder and open in Siege Editor! For learning purposes only! DO NOT USE ANY OF MY MAPS AS TEMPLATES FOR YOUR OWN!

ATHESIA, and ATHESIA RESOURCES - This is the beginnings of the Land of Athesia. Take a look! You'll need to put the map in "my documents/dungeon siege/maps" and the resources in "dungeon siege/resources". Have fun, and let me know what you think!

MIKETEST2 COMPILED and MIKETEST2 UNCOMPILED - My first map in Siege Editor. It is in NO WAY complete, but shows some demonstrations for moods, fade triggers, and some neat landscaping.

COLOSSEUM PVP - A map I put together in about an hour, consists of a simple bowl using the Canyon node set, 8 start positions, and some other misc scenery, for effect. Intended for PvP battles!

PATCHES - Install em! That's all there is to it! Fixes a bunch of funky crashes and crap.

Dungeon Siege Update Beta v1.09.2 - Fixes a bunch of bugs, read the readme that is included. Requires you to connect to Zonematch after the install is complete.

PROGRAMS - Install em to make em work. I suggest keeping em in their own subdirectories in your Dungeon Siege folder

SIEGE EDITOR v1.09.2.318 - The one we've all been waiting for, use this nifty program to make your own worlds!

TANKVIEWER - Lets you rummage through the contents of tank files (dsres, dsmap, dsparty, etc.) and view/extract the contents.

DSMCONVERT - Allows you to run your map in Dungeon Siege without the patch! Instructions are HERE.

GAS VIEWER - Auto-formats .GAS files to make them readable and easier to work on. You can open any .GAS file with a text editor such as notepad, but it's rather difficult to interpret.

TANK CREATOR - Self-explanatory, allows you to make your own Tank files (dsres, dsmap, dsparty, etc.)

RAW 2 PSD - Command line tool that is essential if you want to make your own skins. Converts the RAW format skins in the tank files to Adobe Photoshop PSD format. If you don't have photoshop, find a program that converts PSD to a format you can edit.

RAW 2 PSD GUI - Makes a nice and easy to use GUI to go along with RAW 2 PSD. Makes using it a LOT faster.

MISC STUFF - Things like skrit and gas files

NECROMANCERS.GAS - My first template, this is for the base of the 6 Necromancer bosses. I included documentation so that you can see how I did it. Also included is the DSRES file, so you can see what it does, however you will need to place one of the Necromancers in your map before you can see them. Theya re located in game objects/actors/regular/evil/a. Includes a custom skin for the Necromancer Female, and a few other things.
