A few people requested this one, and it's very quick and easy, so here it is :-D

First off, let's create 2 rooms that connect with an arch, like so. Just for fun, we're gonna add some defenseless maiden on the other side of where we want our door, screaming "Help me! Help me!" as you strain your mind as to what could possably open the door and set the maiden free. They didn't teach you how to use a lever at Warrior School, so this is a new problem for you. But fear not! Gravity is with you, young knight, so stumble backwards and fall into the switch and set her free for some crazy nights of boozing and unbridled lovin'. Or something.

Anyways, it looks like this before we do anything:

Now let's add a switch on the right, on the side that we start at, and a door to seperate us from our true love. Select the door, right click, and select properties. In the Template Prooperties tab there is a value in the main area called "can_use". The default value is "true", so we want it set as "false". Why? Becuase then you wouldn't need the switch to open the door. "Now I see said the blind man as he pissed into the wind, it's all coming back to me now". Anyways, copy the SCID of the door (top middle of the Object properties window, the long HEX number beside "Scid:") by highlighting it and hitting "CTRL-C" on your keyboard, or highlighting then right clicking and selecting "copy". Ok? OK! Close that window, open the object properites window of the switch, and scroll through the template properties tab till you find the value "On_Scid". beside it is "0x00000000". Paste the SCID of the door, which you copied earlier. That would be "CTRL-V" on your keyboard, or right click on the numbers and select "paste". You have just set it up so that when the switch is used, it sends a "we_req_activate" message to the door, telling it to open. If you did it right, when you exit the object properties window of the lever, there should be a fun animated green arrow pointing from the switch to the door. YAY!
