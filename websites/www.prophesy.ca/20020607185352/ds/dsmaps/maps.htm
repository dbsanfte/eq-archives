Colosseum PvP - Development Complete

A map I put together in about an hour, consists of a simple bowl using the Canyon node set, 8 start positions, and some other misc scenery, for effect. I also added health and mana shrines, but they don't quite work yet. There are some low level items there for level 0 character battles. Me and 7 other people on Zonematch tried this all using level 0 characters, and DAMN WAS IT FUN! I have removed the old screenshots for now, and will update them ASAP.

Intended for PvP battles!
