Class Chooser! Posted Tuesday, April 18, 2000 by Lenenae McGillicuddy
This is entertaining, I took the story from EQVault, I hope they don't mind too much. It's a Class/Race Chooser for EverQuest. Answer 20 questions and it will assign points to classes and races based on your answers. At the end it'll tell ya what each question means and the point distrabution. Even includes the Iksar Race.

For your information it choose Dark Elf Wizard for me. Here's the post and link to the download:

EQClass is a free Windows program that asks you 20 questions with multiple choice answers. Each answer adds points to the various classes and races you can be. At the end, the results are tallied and the most suitablerace/class is displayed!
Iksar is included!

Version 1.1 includes status bars for all races and classes so you can see the impact of each question as you go. It also includes the completeroyalty-free source code in EQClassSources.zip.

As usual, I would have liked to done more with this program, but I feel that it is complete enough to be released and never be worked on again. The data is read from a script file (script.txt), it's well-commented if you feel the need to tinker with the values.

-Yappie-
