AutoCard windows

　なんか書いてあるけど、いったいどのカードのことなのか解らないよ。
　読んでてそう思った事は無いですか？　まぁ解脱しちゃった人なら雰囲気で大体解るだろうから問題無いんですが。逆に書く側からすると、正式名称なんざぁろくに覚えてないんで、浮かんだ略語を適当に使っちゃうわけです。例えば

　パタパタ　→　《金切るときの声/Battle Screech(JU)》
　ティム化　→　《霊力/Psionic Gift(OD)》
　優雅　→　（C)ナック

　と、こんな感じに。で、そんなあやふやだった部分を、解りやすくする方法が開発されたっていうか一般開放されたのです。

[1] 下のソースを、使用したい html ファイルの上部にコピぺ。
<script language="Javascript">
function WizardsAutoCard (cardname) {
  windowName = "WotCWindow";
  params = "toolbar=0, location=0, directories=0, status=0,
   menubar=0, scrollbars=0, resizable=0, width=450, height=400";
  win = window.open
("http://www.wizards.com/magic/autocard.asp?name="+cardname,
   windowName, params);
}
</script>
[2] リンクしたい部分をこんな感じに
<a href="javascript:WizardsAutoCard('Battle Screech');">
パタパタ
</a>

↑を使うと、こんな感じに　パタパタ

※ WizardsAutoCard の後に書く正式名称を間違うと、ちゃんと動きませんので注意
