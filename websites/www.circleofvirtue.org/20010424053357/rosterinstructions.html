Step 1 -- Creating a User Profile
Click Here to Begin (this will open a new window)

You should see a screen like this when you go to create your User Profile. This step is important and neccesary as it allows GuildMagic to track who owns each character.

Step 2 -- Adding Guilds to User

When you click this button you be given a list of Servers, select The Rathe. Now you'll see a list of guilds, select Circle of Virtue.

Step 3 -- Creating Characters

 When you click this button you will be brought to a character creation page, from there you will create your character.

Be sure to select our guild for each character you create! Once this is done hit Create Character. From my Administration page I will add this character once you create it. From then on its all under your control on when you update it!
