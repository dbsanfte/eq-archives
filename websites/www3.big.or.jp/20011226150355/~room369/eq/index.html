§ Alternate Advancement

まだ最初なので何に振っていいか分からない人も多いと思いますので。とりあえずお勧めは 'Innate Run Speed' です。これにまず 3pt つぎ込んでよいと思います。次は 'Innate Regeneration' がよいでしょう。この２つあわせて 6pt でいい感じです。

ここで一つ、Run Speed の速度比較を。'RUN' で 21sec かかる場所で計測してみました。

- Walk Mode 0:31.81
- Run Mode 0:21.00
- JB & TB 0:15.53
- Sprit of Wolf 0:13.57
- Run Speed Level 1 0:18.88
- Run Speed Level 2 0:17.59
- Run Speed Level 3 0:16.13

Skill Level 3 で J-Boots に近い速度が Indoor でも出せるようです。実際これだけの速度があると Pull や移動はかなり楽になります。

§ New SoL Skills

こんな長い訳したの久しぶり。なので ranger 関連のみ（といってももうほとんどだけど）。他のクラス → The Safehaouse

General Ability - 全クラス （Level 51〜）

--- 各 level 毎に 1 skill point 消費
  • Improved Natural (Status名) : (5 skill levels) - 各 status をレベルごとに 2pt ずつ上昇させる。
  • Improved Natural (Resistance名) Resistance : (5 skill levels) - 各 resistance をレベルごとに 2pt ずつ上昇させる。
  • First Aid : (3 skill levels) - Bind Wound できる上限を 10%、20%、30% と上昇させる。
  • Improved Natural Run Speed : (3 skill levels) - 移動速度増加。但し移動系 spell と効果は重複しない。
  • Improved Natural Lung Capacity : (3 skill levels) - 呼吸量を10%、25%、50% と上昇させる。
  • Improved Natural Metabolism : (3 skill levels) - 食料消費量を 10%、25%、50% と減少させる。
  • Improved Natural Regeneration : (3 skill levels) - HP の自然回復能力を、1pt / 6sec 毎に付加する。
Archetype Ability - Caster、Melee、Priest （Level 55〜）
【General Ability に 6 points 以上消費している事】

Ranger は Melee、Priest の Archetype Ability を使用することが出来る（一部を除く）

--- skill level 1 : 2 skill points 消費
--- skill level 2 : 4 skill points 消費
--- skill level 3 : 6 skill points 消費

- Melee
  • Combat Agility : (3 skill levels) - 攻撃回避能力を 2%、5%、10% と上昇させる。
  • Combat Fury : (3 skill levels) - critical 率上昇。skill level 3 で skill を持っていない WAR と同等の能力を得る。
  • Combat Stability : (3 skill levels) - 受けるダメージを 2%、5%、10% と軽減する。
  • Fear Resistance : (3 skill levels) - fear 耐性、fear からの復帰率を 10%、25%、50% と上昇させる。
  • Finishing Blow : (3 skill levels) - HP 10% 以下の敵を一撃でしとめる確率を 1%、2%、5% 上昇させる（要 Combat Fury、WAR を除く）。
  • Improved Natural Durability : (3 skill levels) - HP を 2%、5%、10% と上昇させる（装備、Buff なし状態）
  • Natural Healing : (3 skill levels) - HP の自然回復能力を、1pt / 6sec 毎に付加する。（Improved Natural Regeneration、種族による自然治癒力と Stack 可）
- Priest
  • Channeling Focus : (3 skill levels) - ダメージや移動による spell interruption 率を 5%、10%、15% と減少させる。
  • Healing Adept : (3 skill levels) - 即効性 healing spell の回復量を 2%、5%、10% と上昇させる。
  • Healing Gift : (3 skill levels) - 即効性 healing spell の critical 率（2倍の効果）を 3%、6%、10% と上昇させる。
  • Mental Clarity : (3 skill levels) - Mana の自然回復能力を 1pt / 6sec 毎に付加する。
  • Spell Casting Fury : (3 skill levels) - direct damage spell の critical 率を 2%、4%、7% と上昇させる。
  • Spell Casting Mastery : (3 skill levels) - specialization skill の成功率を 5%、10%、15% と上昇させる。（hybrid class は specialization がないため選択不可）
  • Spell Casting Reinforcement : (3 skill levels) - buff の効果時間を 5%、15%、30% と上昇させる。
Class Ability - 各クラス （Level 59〜）
【Archetype Ability に 12 points 以上消費している事】

- Ranger
  • Ambidexterity : (1 skill level - cost 9 skill points) - dual wield の成功率を上昇させる。
  • Archery Mastery : (3 skill levels) - 弓によるダメージを 30%、60%、100% と上昇させる。
  • Body and Mind Rejuvenation : (1 skill level - cost 5 skill points) - Mental Clarity と Improved Natural Healing の skill level を 1つ増加させる（Natural Healing と Mental Clarity がともに skill level 3 必要）。
  • Double Riposte : (3 skill levels) - riposte 成功時にさらに riposte する確率を 15%、30%、50% と上昇させる。
  • Endless Quiver : (1 skill level - cost 9 skill points) - 無限に矢を撃つ能力を与える。
  • Innate Camouflage : (1 skill level - cost 5 skill points) - 効果時間無制限の invisibility タイプの skill。Lay Hands タイプで再使用時間は 1sec （ゲーム時間で 1/20 sec）。
  • Mass Group Buff : (1 skill level - cost 9 skill points) - group buff を 周辺の group にも効果を与えるようにする。cast するには 通常の mana 量が必要だが、cast 後には2倍 mana を使い果たす（要するに 1 cast 分の mana しか残っていない状態で使用するとお得って事）。再使用時間は 72min （ゲーム時間で 1日）。
  • Physical Enhancement : (1 skill level - cost 5 skill points) - Natural Durability、Combat Agility、Combat Stability の skill level を 1つ増加させる。
