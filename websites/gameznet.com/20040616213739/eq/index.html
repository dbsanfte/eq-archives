06/14/2004

Clearing Some Things Up On Renovation
 4:08 EST - J.C. Smith

I've received a few emails asking this question so I figured I'd clarify this all on the main page. We've mentioned that the site is going through a major renovation currently and that we will have some big changes soon. You have likely already noticed the AAXP, Bestiary, Faction, Quest, Quest Components, Zones and the accompanying By Zone listings which have appeared since that time. All of these features can be considered in beta form, although they are functional for the most part. They have to be considered beta though for two reasons. First off many quest rewards or related items and many mobs drops are showing up as blank. This is because most of that information had to be keyed in manually in the equipment database. That transition is still underway and is currently in the Hs, going alphabetically by zone. Feel free to use these databases, and feel free to tell your friends to use them in the current form. If you find any bugs then let us know about it. Be aware though that zones I-Z are still not converted so you may find some bugs or missing information in those zones.

Also be aware that with the new site design we are also completely rewriting all of the layouts for the site. This means many new features will be implemented once the new design goes live. Some of those may be seeing their way in early, as our Quest Component database did, but for the most part we'd have to write the code twice to do so, so you will have to wait for the new design and PHP to be written. If you have any feature requests for site, now is the time to send them in. We've been packing in everything we can think of, but we're still open to new ideas. Also we are lost a lot of our bestiary screenshots during the merger, so if you see a mob with a missing picture feel free to email it to us for inclusion. Last but not least this entire site has been opened up to the public for free. Flashlink subscribers are able to view the site ad free of ads, but there should be no more protected pages here. If you find any protected pages that bring up a login prompt, please let us know so we can remove them.

There are some exciting changes just a couple weeks away. I hope that you enjoy all the new features of the site, and feel free to tell your friends that this site is now open to all for free. Be aware though that this is not the new EQ'Lizer yet. We're still a couple of weeks away.
