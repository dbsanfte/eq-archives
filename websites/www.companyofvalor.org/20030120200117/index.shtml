Quellious is UP

Friday, January 10, 2003 2:45:42 PM
First, as everyone knows there was a patch yesterday. One of the files changed was EQUI_Animations.xml which is part of my Buff Window customization. I have updated my version of this file to be compatible. Without this new file, your skin probably will not even load. Please download my newest Buff Window if you haven't already done so. I have also already started customizing the new Song/Short Duration Buff Window to be similar (small graphic on left with text description to the right). I should be done with this customization later tonight. I mostly got it working last night.

I have re-programmed how I am getting the status of the Quellious server and the Everlore news links. They now have a timeout factor. If it takes more than a certain amount of time to get the information (which could happen if say Everlore was down or just really slow) it will cause a timeout and stop waiting to get the information and state that it timed out. This should solve the problem of this page taking forever to display sometimes. Thank Kuadine's complaint to me that the main page took forever for her for this and the change in how the newest members list is made hehe. *HUGS* - Mysstie
