Well.. got to fiddling around with the main page here to try something out. To view any of the older news items, click on the link below and it will expand out to reveal the information. As you can tell on the right side, several pages of the website have been changed. The Guild Vault should accurately reflect any donations made up until around 3:00am. Thanks to all those who have donated! :)

I have been working on a new Calendar system also. This is a single script that generates the calendar on the fly (hence it will always be the correct month) instead of individual files for each month. It's still in the development process, but you can view what's there so far. Appearance wise it will look overall the same as the current one; however, the current day will be highlighted with a different color to make it standout. The actual events are stored in a database which means once it's all done, officers and/or other designated persons will be able to add/modify/remove events from the calender. Also, they'll be able to setup re-occuring events (such as the BGM event that is currently there). It will be awhile before it's all done but once it is it will be cool. :) *HUGS* Mysstie
