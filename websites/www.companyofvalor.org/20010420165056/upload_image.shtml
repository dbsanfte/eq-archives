Upload Image

Before uploading an image, please convert the image to a .jpg or .gif before submitting it. The EQXXXXX.BMP files that Everquest Saves are quite large. Also, I would recommend renaming any file with your character name in the filename. For example, I would name my files mysstie_something.jpg or mysstie_thisthing.gif and so on.

After browsing and the pressing upload, please be patient. It will take awhile for it to transfer. It will tell you when it is done and display the image. It will also tell you how to reference your image.
