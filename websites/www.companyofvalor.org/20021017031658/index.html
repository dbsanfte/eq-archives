CoV Hyperchat Chat Server is Offline.
Thursday, October 03, 2002 12:54:34 PM
Hiyas! :). Well, it seems as if EZBoard is down.. they had a hardware failure on at least the server that our board was on and maybe even the main EZBoard server itself. I'm sure you've seen their message if you've tried to go there recently. We appreciate your patience and understanding during this time. We expect to be back up Thursday evening (8 PM Oct 3, 2002 PDT). That would be, 10:00pm Central.

One thing.. I don't know if everyone was aware, but the multi-colored upcoming events there on the right and on the Calendar itself are actual links you can click on to go the Sign-Up/Information thread for that event on whatever message board it may be on. When you click on the link it will spawn a new webpage.

The Guild Vault has had some fall cleaning performed on it to make some room. I would like to thank Viron and others (Viron gave them to me, I know he wasn't the only one at that raid) for the recent spell additions to the vault. :)

Right now I am deciding the fate of the CoV HyperChat Server. The main reason for me getting HyperChat in the first place was to have a logging feature for meetings. At the time you could not log the EQ Chatrooms. Well, now you can and any future meetings will probably be held in an EQ Chatroom. Also, some folks have problems running HyperChat. The only time I usually start the server now is when I know EQ is down for a patch or something.. and maybe that is reason enough to keep it around.

One last thing, Brubell and Necratis Elderwolf have been in the guild exactly one month (both joined around September 3rd) and have been promoted to full membership! :) - *HUGS* Mysstie
