CoV Hyperchat Chat Server is Offline.
Use our EZBoard chat: Join Live Chat!

Currently, we are no longer using HyperChat. My initial reason for using it in the first place was for having meetings and being able to log them. Since I can log stuff in EQ Chatrooms now, that is where meetings will be held. I'll still keep HyperChat in case of emergencies.. like the EQ Chat Server goes down or something. For general online chatting, use the chatroom that we have available with EZBoard. Join Live Chat!

I've looked around and this seems like a pretty good chat system to hold our meetings in and for just chatting. It is freeware, with no ads or anything. It also seems to be a clean installing program and it's uninstall routine (accessed via Windows' Add/Remove Programs) cleans everything out it looks like (although it may leave a C:\Program Files\HyperChat Client folder which can be manually removed). According to them it works in Windows 9x, NT, and 2000. It also has a log feature. :)

  • HyperChat Client Program: Download And Install this program first. Please install the program into the default directory.

  • Company of Valor Chat Customizer: This is a self-extracting ZIP file containing the configuration file needed to connect to the CoV Chat Server. Once downloaded, run it, click on unzip, and then close it.

  • If you get an error message that a DLL is missing or required to run the program, then download and install the HyperChat Support DLL files (MFC42.DLL and MSVCP50.DLL).
