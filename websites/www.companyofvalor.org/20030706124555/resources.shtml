Tips

Commands
  • To check who is on in the guild:

    / GUILD all (GUILD must be all caps)
    / "company of valor" all (this works if char is in guild or not)

General
  • If you want to select a single inventory item from a stack, press the CTRL key as you left-click on the item with the mouse. This makes tasks such as fletching arrows much faster. It was quite awhile before I knew this trick.
  • To select all the items on a stack at once, use the SHIFT key as you left-click. This also now holds true if you want to buy a whole stack of items at once, hold down the SHIFT key as you click on Purchase.
Navigation
  • Clouds always move from East to West.
  • If you drop a sword on the ground, it's blade will point North.
  • Learn the location system and how to use it. Make a hotkey for /loc. For a detailed explaination, go to The /loc Guide at EQAtlas

Third Party Tools

  • Winproxy - WinProxy is a shareware program that allows multiple computers access to the internet. Some CoV members use this to play EQ with more than one computer at the same time. Download page.
