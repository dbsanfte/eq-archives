Whew.. been busy today. Doing alot of work on here. First off, the website is back up! YAY!!!! Well, let's get to the changes. Lordy I'm tired. Think after I get done typing this I am going to bed LOL. Okay... lot of changes to the website backend. Pretty much everyone will have to re-login to the website.
