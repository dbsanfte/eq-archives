    Mobile Quake anyone?
    Reported by: sands   0 Comments | Add  

    Nokia, Ericsson, Motorola and Siemens, 4 giants of the mobile communications world, have gotten together and formed the Mobile Games Interoperability (MGI) Forum. The aim of MGI is to define a mobile games interoperability specification for network-based servers. This means that game developers will be able to produce and deploy mobile games that can be distributed across multiple game servers and wireless networks, and played over different mobile devices.
