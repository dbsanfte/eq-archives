Fletching Tools
Tool Type Notes
None Rough None
Whittling Knife Carved Whittling knives can only be used on Elm or better bows
Planing Tool Shaped Planing tools can only be used on Ashwood or better bows



The base statistics on a Rough bow using hemp string of each type are:
Type Damage Delay Range Trivial
Hickory 10 50 50 35
Elm 13 51 75 80
Ashwood 16 58 100 126
Oak 21 65 125 185
Darkwood 25 68 150 202



Each possible component included is additional modifier to the bow,
reducing the damage and delay (range is only determined by which staff you use)
String/Tool Damage Delay
Hemp -0 -0
Twine -1 -4
Silk -2 -8
Whittling Knife -1 -4
Planing Tool -2 -9



So making a shaped oak is no more difficult than making a rough oak.
However the difficulty is affected by tool and the cam.
Cams have -1 Damage and -5 Delay.
Cam Types Notes
None Recurve None
One 1-Cam One cam may be used on Oak or better bows
Two Compound Two cams may be used on Darkwood bows



Cams give the best bonus but also add to the difficulty of the bow.
