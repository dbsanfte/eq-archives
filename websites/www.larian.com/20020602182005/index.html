** Demo Update **

It’s come to our attention that some people have problems starting up the demo of Divine Divinity after they’ve installed it. This has been isolated as a bug with the installer. To solve it, download the file divdf1.zip and unzip the executable that is in the zip file to the folder in which you installed the demo of Divine Divinity. Next run the executable instead of going through the menu, and the startup problem should be solved.
We apologize for the inconvenience.
