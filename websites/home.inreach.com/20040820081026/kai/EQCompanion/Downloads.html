EQ Companion / Downloads

EQ Companion is freeware, you are welcome to download it and use it freely. It is written in Visual Basic 6.0 and requires some VB 6.0 run-time libraries. If you do not have VB 6.0 installed on your computer, you should download the ~2.8MB installer file and use it to install EQ Companion. It will install all the required run-time libraries.

The installer program installs version 0.6 (build 651) of EQ Companion. I would recommend downloading the latest executable as well. It is ~200kb ZIP file with the executable only. You should extract the executable on top of the file the installer installed (typically in "C:\Program Files\EQ Companion" -directory).

EQ Companion optionally uses Microsoft's TTS (Text to speech) engine to allow it to speak. Windows XP has this already built-in but if you are running older versions of Windows, you can download the package from Microsoft's speech SDK web site. I have a direct link to the 68MB file below as well.

NOTE! - EQ Companion requires that you have at least version 5.5 of Internet Explorer installed on your computer. If you do not, you will get 'Run time error 429 Activex componet can't create object' error message on startup.

NOTE2! - Version 0.791 beta uses a new type of text box called Rich-text box. If you get an error message like "Component RICHTX32.OCX or one of its dependencies is not correctly registered", download RichText.zip file below and extract the 2 files in it to your EQ Companion folder.
