Wednsday March 17, 1999                     EQ Arrives Day 2!

EQ Updates - Sacrates
Ok lets get you guys up to date with the EQ news going on. Smedly, John that is was on the boards and had these few juicy morsels for you to jaw on. First up is a current update.

Ok.. we found and fixed a problem with our patch servers (we actually have 6 different patch servers that are dynamically selected to do the patching). 2 of the machines we just put up had a configuration problem. We are fixing it and it should be done within 10 minutes of this posting.

Top 2 registration problems and what to do:

1) Invalid Account Key - You MOST likely are mistaking an O (Oh) for a Zero.. try different combinations. If not, call our tech support at 1-888-Games02. You will need to read the key to us to insure you actually purchased the product, but we can help you at that number.

2) Bad Station Name or Password after registering (when you log in for the first time) - 90% of the people that call in with this problem have forgotten their password or mistyped their station name. We can help you with this - just call our tech support at 1-888-Games02.

One other comment - If you are having ANY sort of trouble registering for EverQuest PLEASE call our tech support number at 1-888-Games02. We are here to serve you.

Then he stops by to give an update on the Server problems recently.

To those of you who got the message "can't connect to chat server" between 6:30 and 7:30 PST tonight - The chat server WAS up during that period (we had plenty of people in) those of you getting that message were unfortunately not able to connect due to a router problem at UUNET.

I want to clear up the misconception regarding the chat server once and for all.. the chat server is merely a gateway point that we verify you do indeed have an account, so bypassing it isn't an option. The Chat server itself didn't go down today at all... with the exception of the one time late this evening we brought it down to put in some additonal code (and then it was down for less than a minute).

If you get the message : Can't connect to chat server, it isn't because the chat server is actually down (it will automatically restart itself in under a minute in the unlikely event it ever does go down).. it's actually because you can't physically connect to the chat server itself.. which in any event will mean you would be unable to connect to the game servers as well.

We do apologize for that bad period this evening, but as has already been explained, a router went down upstream from us on ONE of the ISP's we use (we use multiple ISP's for redundancy and lower latency). We don't expect that kind of thing to happen often (except that Murphy's Law dictates it should have happened on opening night).

We take downtime VERY, VERY seriously here. You are our valued customers and you payed for the right to play this game, so we need to make sure you can!!!

BG and Roger Wilco - Sacrates
Now here is some groundbreaking news for you to chew on. Looks like Baldur's Gate will be getting a modified version of Roger Wilco integrated. This is something that I expect to be hearing from in many more games very soon. Check it.

Black Isle Studios, the role-playing game (RPG) division of Interplay Entertainment Corp. (NASDAQ: IPLY) and Resounding Technology announced from the Game Developers Conference today that Black Isle Studios has licensed Resounding's Roger Wilco(tm) Internet voice communication software and will include it in the Baldur's Gate(tm) adventure pack add-on, Tales of the Sword Coast(tm).

Roger Wilco enables gamers playing over the Internet to talk each other, eliminating the need to type messages to other members of their party. This allows players to strategize, react to situations in real time and interact on a more personal level while adventuring through Baldur's Gate and Tales of the Sword Coast. With Roger Wilco, messages are immediately broadcast to other gamers, making it possible for them to communicate with each other without interrupting gameplay, audio or game performance.

Roger Wilco requires a microphone or headset for the PC which easily connects to almost all standard soundcards and a uses a minimum 28.8K modem. The experience is similar to using walkie-talkies. Unlike Internet phone programs, Roger Wilco is specifically designed for to run with Internet multi-player online games without requiring an additional soundcard. In addition, the software supports 128 players simultaneously per channel without requiring a dedicated server.

Howl like the Wolf - Sacrates
Looks like Gamecenter had to go and leave their scent all over the place and poke up a Preview of Werewolf. About time we started to get some concrete plans behind this gem.

ASC and DreamForge are classifying Werewolf: The Apocalypse as an action-adventure with role-playing elements. Of course, that can mean a lot of things. First and foremost, the game is a third-person 3D action game, based on a modified version of Epic's Unreal Engine, which DreamForge has now coined Unreal Plus.

Like any 3D action game, Werewolf: The Apocalypse offers a level-based single-player game and a collection of multiplayer options. In all, the game will have 20 single-player levels, which begin in rather conventional real-world locales, such as office spaces and foreboding city streets, and eventually mutate into unnerving supernatural labyrinths with their own laws of physics, perverted by the ways of The Wyrm
