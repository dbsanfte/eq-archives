HOW TO RENAME A FILE ?


ENGLISH

Some people seem to have difficulties to change a file's name under Windows.
Here is a quick answer for those of you on valium ;-)

1. Open Windows Explorer.

2. Go to "Tools" and select "Folder Options".

3. In the "View" tab, uncheck "Hide file extension for known file types". Click on "OK" (you now see all file extensions such as .exe, .doc, etc.)

4. Go to the folder where you saved the files.

5. For each file, right click and select "rename". Change the extension.

 

 

COMMENT RENOMMER UN FICHIER ?

 

FRANCAIS

Certaines personnes semblent avoir des difficultés pour changer les extensions de nom de fichiers.

Voici une méthode rapide pour nos amis handicapés en informatique ;-)

1. Ouvrez l'Explorateur Windows.

2. Dans le menu "Outils", sélectionnez "Options des Dossiers".

3. Dans l'onglet "Affichage", décochez l'option "Masquer les extensions des fichiers dont le type est connu". Cliquez sur "OK" (les extensions telles que .exe ou .doc sont à présent visibles).

4. Ouvrez le repertoire dans lequel vous avez sauvegardé les fichiers.

5. Pour chacun des fichiers, faites un clique avec le bouton droit dessus et choisissez "Renommer". Changez l'extension.
