3.2: Trolloc Tribe Names

The Trolloc tribes seem to have been named after various mythological creatures:

  • Ahf'frait = Afrit
  • Al'ghol = Ghoul
  • Bhan'sheen = Banshee
  • Dha'vol = Devil
  • Dhai'mon = Demon
  • Dhjin'nen = Djin
  • Ghar'gael = Gargoyle
  • Ghob'hlin = Goblin
  • Ghraem'lan = Gremlin
  • Ko'bal = Kobold
  • Kno'mon = Gnome
