MUSH Information General Resources
Available
We've put together a list of resources that include MUSH/MUD lists, newsgroups, game help files, and other useful resources. If you have comments or suggestions, please feel free to @mail Melkir online.
General
The following are some general resources that may be of interest to the MUSH player.
Amberyl's MUD Resource Collection
A comprehensive collection of resources for the players of MUSHes and similar games. Includes FTP archives, lists of MUSHes and similar games, FAQs, and links to scholarly articles on virtual enviroments.

The Journal of Virtual Environments
The Journal is an electronic periodical that publishes academic articles related to MUDs or the use of MUD environments. (Previously called the Journal of MUD Research.)

Electric Soup
Electric Soup is a MUD community board that features reviews, essays and discussions on topics ranging from administrative needs to softcode to player thoughts on game affairs. It is intended as a general resource, unattached to any particular game. This website is somewhat offline and outdated, but still has some useful information.
Newsgroups
There are several newsgroups devoted to discussion of MUSHes and similar games. Topics range from announcements of new games to in-depth discussions of role-playing, programming, etc.
rec.games.mud
This is a general purpose newsgroup for all players and admin of any MU* game, and topics vary widely.

rec.games.mud.tiny
This newsgroup relates specifically to MUSH and other games in the "Tiny" family - MUCK, MOO, etc. Common topics include new game announcements, some coding, etc.

rec.games.mud.admin
This group is designed for the use of MU* game administrators, though all are welcome to post to and read it. Common topics include staff wanted ads, security issues, technical problems, etc.

rec.games.mud.announce
This is a moderated group, submissions must be approved by the moderator to be posted. Common topics include announcements of new games, address changes for established games, ads for MU* lists, etc.
MUSH Lists
The MUSH and MUD lists below can help you find games to explore and enjoy.
The MUD Connector
The MUD Connector provides a searchable list of MUDs/MOOs/MUSHes that includes an admin-submitted description of the game, game link, and link to game Web page if one exists. More than a MUD list, the Connector also includes links to resources for MU* players.

MUSH Warehouse List
This list only includes MUSHes, MUXes and similar server games. The contents can be listed by server type or genre. One of the more up-to-date lists currently available.

Book-Themed Science Fiction/Fantasy MU*s
This site is an index of the MU* games based on science fiction and fantasy novels, and includes authors such as Terry Goodkind, Robert Jordan, JRR Tolkien and many others. The author also has an index of games based on television series, films and comics.

The Almost-Complete List of MUSHes
Only MUSHes and MUXes are indexed on this site, which includes game addresses only. An extended list which includes links to game Web pages is also available, as well as an automated list. The automated list includes information on game uptimes and number of characters connected in the past 24 hours. Please note that all three of these lists have not been updated since December of 2000, so information may not be current.
Help Files
Information on all the commands and functions of MUSH and its programming language are available online by typing help or help <topic>. We have been unable to find current TinyMUSH 3.0 help files online. If anyone knows of a website, please @mail Melkir with the URL.
Web version of MUSH Help Files
WWW version of the help files internal to the TinyMUSH server, for version 2.2.3.

MUSH Helpfile Search Interface
Cauldron of Chaos MUSH provides this Web page, which allows you to do a search on the MUSH help files. The search tool supports several servers (TinyMUSH, PennMUSH, MUX and Muishy) and allows you to search either player or wizard help files.
MUSH Source
For those that are looking to set up their own MUSH, there are a number of server code bases that are used. The most popular of the group are listed below. Each person has their preference and, to some people, it is almost a major war between the major code bases as to which one should be used.
TinyMUSH 3.0
This is the source code that Shadowrun Seattle runs off of.

TinyMUX 2.0
This is another very popular code base that is similar to TinyMUSH 3.0.
