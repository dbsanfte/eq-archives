New Player Reference Guide
  
 

So you're new to the MUX, chummer? Well, here's where you can find links to all the info to get started!

Seattle MUX FAQ - Answers to most of the basic questions new players to the MUX will have.

Commands cheat sheet - This is a quick reference to some of the most commonly used commands on this MUX.

Required Reading - All of the information files that every player on the MUX is required to read before stepping onto the grid.

Character Generation Info - These files are present in the character generation rooms, as well as here. This is a quick reference to access all of them at once.

Character Types We Need, Character Types We Don't - SR-Seattle MUX has some restrictions on what you may play. Please read here before deciding on a character type.

Softcode Library - Repetitive commands can be a pain in the butt. Here are some softcode items to make roleplaying a little simpler, and let you do complex tasks with a single command.
