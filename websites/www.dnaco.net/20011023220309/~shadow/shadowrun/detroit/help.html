Help:
Need a lil help? Maybe a lot? heh.. This is the place to look. To the left you can find all the help one could ever need for playing the Shadowrun Detroit MUX. HTML files were created and taken right from the MUX's matrix help files and make for good reading or even just quick refrences. Covering all aspects of the game, media, and even web related pages. Also including the global commands help file from the MUX of all basic commands for users (perfect for you newbies out there). Feel free to click the downloadable files and save them for offline reading. They are in plain vanilla text so almost all users should have no problem viewing them.

For any questions or comments please send feedback to <rle@alder.nesc.epa.gov>.
