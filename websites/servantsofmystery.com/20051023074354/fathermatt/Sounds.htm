Hello everyone, finally after about 2 weeks of working and testing the new version of the Sound Pack has been released, it is fully compatible with DAoC v1.49 and up and this time around the file is available from Game Spy's File Planet (special thanks goes out to Fargo for helping out getting the file to actually show up in File Planet) which is good since the new version is 19.6mb - in it you will find 269 new sounds which replace almost all of the game sounds  with new ones,  a lot of tweaks and new sounds have been added since the first release of the sound pack 3 months ago, also included is an optional set of Stat Bars graphics to change the color of your Health, Stamina and Mana to 3 different choices!
 
Download the Insomniax Dark Age of Camelot Sound pack v1.1 here :
http://www.fileplanet.com/index.asp?section=1124&file=87304
 
 You can find also find all the information you need in the very lenghty read me file which you can find at http://www.insomniax.net/files/daoc/ix_daoc_sound_pack_v11_readme.txt  or inside the Zip file along with the sound pack.
 
Please also note that this sound pack is NOT supported in anyway or form by Mythic Entertainment as this is a fan created add-on for the game which does not change DAoC's gameplay in anyway or form.
 
If you need to find updates or news related to the sound pack please visit our DAoC message board at  http://www.insomniax.net/forums and let us know what you think, Enjoy!
