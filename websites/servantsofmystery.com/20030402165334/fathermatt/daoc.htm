Other Useful Links
If a category has * after it, there are additional sites listed for that category in the chart above.
Abbreviations See the Camelot Vault Player Lingo Guide, and DAOC Abbreviations and Slang by the Icemen and the Camelot Stratics MMMORPG Lingo and Abbreviation Guide.  
Armor For everything you want to know about how armor works, see Bound by Honour's Technical Guide to Armor.
Attributes Warcry provides a Character Stats page that describes the attributes and show the attributes for each race.  
Bestiary Allakhazam offers Illia's Bestiary, which you can search by zone or name (and which has pictures), and the DAOC Catacombs Bestiary allows you to search by name, level, realm etc.--useful for determining which creatures to hunt.   
Bonuses As Messenger of Albion says:  "Now a word about experience. If you pull 2 social monsters together, you will get a nice group experience bonus from killing them. If you pull from a designated "camp" of monsters, you can get a camp bonus of 10% per mob group member (though what is defined as a camp and what is not does not seem logical to me.) Bonus experience and additional loot of up to 30% more can be gained by killing monsters that have not been killed recently. So finding an out-of-the-way spot or a camp is quite beneficial."
Character Builder Classes of Camelot presents a Character Builder available for download.  
Class Guides* Paradoxic Camelot provides Class Guides and Camelot Crusades provides Class Guides.  
Class Progression Camelot Stratics' Class & Progression shows the skills each class gets at each level.  
Classes* DAOC Catacombs has class information in the left-hand column that provides detailed class characteristics charts.  Also see The Classes of Camelot for information about classes, and don't miss the chart for class-specific sites below!
Combat Styles* DAOC Catacombs has information on combat styles in the left-hand column under "Styles".  Camelot Vault provides a Combat Styles page, and Inside Camelot has a Weapon Styles page.   Also see the Classes of Camelot combat styles pages for Albion, Hibernia and Midgard.  
Commands Allakhazam has a list of the Commands, and Camelot Stratics has a Commands page as well.  Camelot Vault lists the Commands (which include instructions on making macros) and Inside Camelot has a Commands/Emotes section.  
Compass Window Use Shift-c to activate, change and remove a compass window.  It's small and very useful.
Darkness Falls Here's info from the Camelot Herald on Darkness Falls, and a map of Darkness Falls from Kirstena's Dark Age of Camelot Atlas.  Darkness Falls can be entered through the relic fortresses and through the following: 
     Loc for Midgard Vale of Mularn entrance: 7150, 29075
     (If anyone has the Hibernia (Connacht) and Albion (Camelot Hills) entrance locations, please e-mail them to me!)
Equipment* Rising Shadow offer a nice list of special equipment and stats.  DAOC Catacombs has  information on items in the left-hand column.  
Events See Camelot Players Event Group (PEG); be patient, it's a bit slow to load.  
Forums* MMORPG offers a General forum, where you can access its other forums as well (see the "Change Message Board" box).  In addition, there is the DAOC Catacombs Main Forum, and the Camelot Vault Main Boards, though also check the left-hand side of the Camelot Vault main page for its vast list of forums.  Warcry also has myriad forums (see the left-hand column).  Camelot Classes has extensive Forums as well.  
Grouping Camelot Stratics provides Grouping tips.  
Guides/Tips/FAQ* Camelot Vault provides misc. Guides.  Camelot Vault also offers a Camelot FAQ page, complemented by Warcry's Unofficial FAQ (with a clickable outline).  Camelot Classes has The FAQ, which "is intended to replace all of the Dark Age of Camelot Game Manual and most of Prima's Strategy Guide, because they were obsolete even before the game was released."
Guild Software Guild-Builder and Guild Boss offer software to manage a guild; there is a fee for use.  Player Vault is free, though it is much more basic.  
Interface Warcry provides a page explaining the game Interface.  
Links* The official page has good links with descriptions; scroll down to "Fan Sites".  Also see the Camelot Warcry Links (by category; click on the rectangle at top), the MMORPG Links and the Camelot Vault Links.  Camelot Classes also provides Links by category and with descriptions.  
Macros Camelot Vault has a list of the Commands which includes instructions on making macros.   
Manual The Camelot Herald offers the DAOC Manual.  Camelot Classes has The FAQ, which "is intended to replace all of the Dark Age of Camelot Game Manual and most of Prima's Strategy Guide, because they were obsolete even before the game was released."
Maps* For maps, see Camelot RPG Maps (maps are rated and have /locs) and the DAOC Catacombs Maps page (again, the maps are rated and have /locs).  MMORPG has a maps section in the left-hand column (scroll down); not all areas are included, but MOB locations are shown for those that are--which is very useful for hunting.  
Kirstena's Dark Age of Camelot Atlas provides maps with MOB locations!  Not all areas are included, but these are extremely useful for hunting!
From the Dark Age of Camelot Mapper we have Albion, Hibernia and Midgard maps.  
Some of these maps won't print on a single page unless you use separate software.  One simple and free program is PicPerk.  
Special note:  Krieger Online's interactive frontier maps (with /locs) are truly wondrous!
Newbie Guides Check FatherMatt's own Tips for Beginners.  See also the Newbie Guide from Allakhazam, the First Timers Guide by Camelot Realm and the Beginner's Guide to Camelot by Camelot Lore.  Finally, see Warcry's Unofficial FAQ (with a clickable outline).  
News* MMORPG has its Current News and Camelot Vault has a Front Page.  Also see the official site's Camelot Herald, and Warcry's News and the Camelot Classes News.  
Finally, if you'd like DAOC news e-mailed to you, check out DAOC News, where you can sign up for an almost daily newsletter.  
NPC's Casters Realm offers an NPC List; this is useful for locating trainers, though it has several gaps regarding vaultkeepers.
Official Site Here's the Official site.  
Packet Loss Window Use Shift-p to activate or close a packet-loss window.  
Patch History (Updates) Camelot Stratics offers Patch History and Casters Realm has its Patch History.  Warcry's Patch History has useful summaries.  
Quests* Warcry offers a Quests page that allows searches by class, level, quest name and NPC name; Inside Camelot has a Quests page with quests divided by town.    
Races* See the Dark Ages of Camelot Article Section for race guides.
Relics For information on the Relics, see Casters Realm.  
Realm Points and Abilities See the Camelot Herald's Realm Abilities to see how to spend your realm points on realm abilities.  See the Camelot Herald's Realm Points page for a bit on info on how Realm Points are distributed.  
Realm v. Realm (RvR) See Krieger Online's interactive frontier maps (with /locs).  
The Camelot Herald provides Realm war status.  
There are new battlefield areas for levels 20-35.  See FatherMatt's own parchment on these battlefields, which comes with a map.  
Sanguis Draconis OK, here is the board for Sanguis Draconis, my DAOC guild (Midgard) on the Palomides server...though this link is mostly for those guild members who keep forgetting the board URL.  =)  (There are some public posting areas here too.)  
Search Engine To search for DAOC-related sites, feed in a key word at the Hibix Search Engine.  
Server Status The official site provides server status.  Inside Camelot has a Servers page that shows the average number of players for each server.  
Server Usage To see when usage is highest (or lowest) on each server, see the Camelot Reporter.  
Siege Weapons Allakhazam provides a page regarding Siege Weapons.  See also the DAOC Tradeskills page Siegecraft HOW TO: A Guide To Pointing and Shooting and the Messenger of Albion Siege Warfare page.  
Skills/Abilities DAOC Catacombs offers abilities information in the left-hand column.  Also see the Allakhazam Skills page.  Classes of Camelot provides a Skills section containing General Info, Weapon Skills, Armor Skills, Combat Skills and Magic Skills.  (For tradeskills, see Tradeskills below.)
Sound Interested in adding some additional sound to DAOC?  Then check out the Insomniax DAoC Sound Pack v1.1.  
Specialization Camelot Stratics provides a rundown on Specialization.  
Spells* Inside Camelot shows what level you get the next upgrade in a given spell line.  Classes of Camelot has Spells by spell line; see "Spells" on the left-hand side.  
System Requirements System Requirements for DAOC can be found on Camelot Stratics.  
Tradeskills For tradeskills, see the Dark Age of Camelot Tradeskills site, and Camelot Lore's Tradeskills and Tradeskills Forum.  Inside Camelot also offers a Trade Skills page, and the Allakhazam Skills page covers trade skills among other things.  
Utilities If you are looking for utilities, try the Camelot Log Analyzer, SkillCalculator.exe, the Stratics Utilities links page, and the Red Dragons' Utilities Links page.
Vault/Bank Here is a list of vaultkeepers for Hibernia.  In Albion, there is a vaultkeeper in Camelot (in building #27), and a vaultkeeper in Adribard's Retreat (building #1).  For Midgard, there are vaultkeepers as follows:  in Jordheim in Midgard (in building #22), in the house at Gna Faste, in the tower near Nalliten, and in Svasud Faste up above the north gate (so you can back while waiting for the teleport!).
