ACVault adopts Microsoft Approach to uniform login information

Effective immediately, All existing Vault Network board users must merge their existing account (username/password) with a new or existing my.ign.com account. We have pre-loaded our my.ign.com user database with your old Vault network username/password. However, we had to prefix your username with "vn_". For example, if you had an account on the Vault boards with a username of "conan", your new IGN username would be "vn_conan". Your password would be unchanged. PLEASE NOTE: You must first login to my.ign.com as "vn_[your old username]" before you can use the merge form below as we will need to collect vital registration information (birth date, email, etc). You DO NOT have to use this pre-made IGN account to merge with your Vault account. You can simply create a new username/password at my.ign.com and use them below to merge your accounts. This process WILL allow you to keep your existing user profile, posting history, stars, private messages, etc. You can only use the tool once so make sure you are using the new my.ign.com username/password that you want. Not all users will be able to keep their existing username as there are millions of users already registered in the my.ign.com user database. We apologize for this inconvenience but this was the only way we could merge Vault users with IGN users and not lose all the messages AND it will allow us to eventually turn on IGNinsider related features for Vault boards. So far the results have not been favorable and there has been many problems during the transition
