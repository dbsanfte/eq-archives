29/01/2003 - Patch the patch, and then patch it!


The last patch seemed to have caused many problems for a lot of us. I can't guarantee this solution will fix them all, bit it made the game a whole lot more playable for me! My machine crashed every time I zoned. This seems to be the solution, and I hope it helps any of you with similar problems.

1. Go to the folder where Everquest is installed.
2. Find the file TestEverQuest.exe (should have an EQ Shield picture over the name)
3. Run this version of EverQuest! - it will start patching the patch program!
4. Just to be safe, delete the texcache.txc file. Don't worry - its gonna work!
5. Now, run EQ as normal. - if you use EQ Windows - make sure you run the patcher for this time!
6. Now, another safety thing - you might want to perform a complete file check. So, de-select the AutoPlay option - and when available, click the Advanced option. Click on the "complete file check" option.
7. Enjoy your newly working EQ!

Texture Fix!


Greetings fellow Wolves!
After the last patch, it seems that many of us (including me) seem to be suffering texture problems. I played around a little, and have sorted it out. Its something I've had to do before to sort out a slightly different graphical problem. Worth noting for future.
In the EQ Game Options, there's a "Texture cache" option. Remove this option.
In your EQ Game install directory there is a file called "texcache.txc". Delete this file.
Now try the game. Should work perfectly now. The next time you play, you can turn the texture cache option back on. Everyting will be fine now! *smiles*
