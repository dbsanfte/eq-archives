Dictionary

最終更新：98/12/30
記載順序：数字→アルファベット→日本語

989 Studios

EverQuestチームはここに所属。http://www.989studios.com/
改称前の名はSISA。

Beta Test

Closed test → Closed Beta(Phase 0.5〜3) → Open Beta(Phase4;Public Beta-98/1)→ リリース(99/3)

Brad McQuaid

EverQuestのプロデューサー。989Studiosの中でも掲示板やニュースグループで最も活動的に発言している。

Class

クラス。Bard, Cleric, Druid, Enchanter, Magician, Monk, Necromancer, Paladin, Ranger, Rogue, Shadow Knight, Shaman, Warrior, Wizard.
種族により就けるクラスの制限がある。

Closed Beta

Open Betaの前に行われるテスト。段階(Phase)を経ながら参加人数を増やしてゆく。

Closed test

ベータテストの前に行われるテスト。

Command

"/"で始まるコマンド。"/who""/tell""/group"など。

First person view →View

GM

Game Master。ゲームを管理し、バグやユーザの苦情に対処したり、時にはイベントを起こしたりする。

Guide

GMのような役目を担うが、989Studiosの人間ではない。UOで言うところのカウンセラー。

ISP

インターネット・サービス・プロバイダ。つまりプロバイダのこと。

MUD

MUD(Multi-User Dungeons/Dimensions)は、ここ数年のテキストベースオンラインRPGである。それらのMUDは単にパラグラフをうまく使って書かれているのに対して、EQではリアルタイム16bitカラーの織り成すグラフィックを使って徹底的に描かれている点を除いたら、EQはMUDに近い。
Ultima Online, Dark Eyes, Middle-earthなどの多人数オンラインRPGのこと。

Name Filter

キャラクターの名前はハイ･ファンタジーという設定に合う必要がある。そして名前は唯一のものでなければならない--同じサーバに同じ名前のキャラクターは創れない。移動先のサーバに同じ名前の人がいる場合は後から来た人が名前を変えることになる。

NDA

Non-Disclosure Agreement(非公開協定;守秘義務)。テストに参加するプレイヤーは、原則的にゲームに関する情報は公開してはならないという協定。Phase4(パブリックベータ)にはこの制限はない。

Norrath

EverQuestの世界。Norrathには、Antonica, Faydwer, Odus, Kunark, Veliousの5つの大陸がある。

NPC

Non Player Characterの略。コンピュータが操作するキャラクター。アイテムの取引やクエストの提供などに利用されることもある。

Open Beta

NDAなしのオープンテスト。すでに募集は締切り。EQでは通常Phase4と呼ばれる。

PKスイッチ

PKスイッチオンのプレイヤー(+PvP)は他の+PvPを殺すことができるが、他の+PvPからPKされることもある。PKスイッチオフのプレイヤー(-PvP)は他のPC(+PvP/-PvP)をPKできないが、PKされることもない。また、+PvPは+PvPをルートできるが、-PvPは他のPCを基本的にルートすることはできない。

PvP

Player vs. Player。+PvPはPK、-PvPは非PKを示す。

PC

Player Characterの略。人間が操作する一般のキャラクター。

Race

種族。Barbarian, Dark Elf, Dwarf, Erudite, Gnome, Half-Elf, Halfring, High Elf, Human, Ogre, Troll, Woodland Elf.
種族同士にはそれぞれ相性があり、また邪悪な種族や特殊能力を持つものもある。

SISA

Sony Interactive Studios America。Play Station XのRally CrossやNFL Gameday '97、オンラインゲームのTanarusなどが代表作。現在は989Studios（Nine-Eight-Nine Studios)と改称。

Skill

技能。

Third person view →View

Top down view →View

View

First person view(一人称視点)やTop down view(三人称視点、Third person view)を状況に応じ的確に選択する必要がある。

一人称視点 →視点

三人称視点 →視点

視点

一人称視点(First person view)や頭上からの視点(Top down view、三人称視点:Third person view)を状況に応じ的確に選択する必要がある。
