Monsters

外見

BlackWidow,Giant None
Brown Bear 雑食性で温暖な地域に生息し、性質は荒い。空腹時や怒った時には攻撃してくる。
Drake None
他にSmall。
Fire Beetle None
Fire Eｌemental None
Gelcube ゼラチン状で地底やダンジョンに生息し、小さな有機体を消化するスカベンジャーである。彼らが這いずり回っている時に邪魔になるようなことがあれば、大きな災難に遭うかもしれない。
Giant Piranha 淡水にも海水にも棲息し、超攻撃的な性質で有名である。群れになれば数分で大きな生物から肉を全て噛み取ってしまうこともできる。
Giant Rat None
Giant Snake 全長数十フィートに及ぶ。人間を数分で死なせるほどの猛毒を持つ。体の小さな亜種とは違い、獲物を狩ったり追跡したりする。
Lava Dragon None
Plague Dragon その巣の周囲は腐敗臭で覆われ、招かれざる者には苦痛となる。鉤爪と牙で引き裂かれるのを祈ろう。というのもその吐息に晒されてしまえば、ゆっくりと苦しみながら死を迎えることになるからだ。体が黒変し水疱ができ、骨から肉が剥がれ落ちてゆくことになるのだ。そう、爪と牙で即死するほうを願いたいものだろう。
Shadow Wolf None
Skeleton 命を吹き込まれた人間の残骸。不死の存在理由は知られていない。
Skulking Brute 獲物を求めてNorrathのまさに地殻を掘り進む、容赦無しの飢えた生物。その強烈な拳と鋭い歯で倒された勇敢な冒険者は多い。どこから来たのかは知られていないが噂は数多い。発狂したEruditeの魔術師による創造物だという話もあるし、Brell Serilis自身により創られたという話もある。だがこの点は皆が認めるものだ：Skulking Bruteがいない所が一番良い。
Spider, Large None
Void Beast もしこの生物が存在するなら、空間、つまり大虚空(the Great Void)の狭間にいて、狩をする時のみそこを離れるのだ、という議論が起こるだろう。だが人間はその虚空では生きていられないので彼らの生息地は不明のままである。伝説によればその魔法生物は夜になると獲物の元へとテレポートし、触覚で突き刺し毒すという。犠牲者はまだゆっくりと動くことはでき、この猫科のモンスターの玩具となり、ついには無論、餌となるのだ。
Wolf None
Wyvern Dragonの遠縁で外見上は似ているが、より小さく普通腕を持たない。多くの生物よりも知性は高いが、同族の竜の知性には遥か及ばない。喋ることも、時には魔法を使う能力を持つDragonとは違い、喋ることのないただの戦士である。
