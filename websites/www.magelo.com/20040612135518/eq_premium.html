Magelo EQ Premium Services

Becoming a premium member will give you full access to Magelo.

The most obvious benefit is the automatic synchronization of your in game characters using Magelo Update. Pretty much everything is synchronized (exceptions are Flags/Keys but it does not impact your character stats).


Currently, the synchronization provides the following :

  • Create / update characters on the fly
    • Name Surname Race Class Deity Server Level XP AAXP Money
  • Update AAs and unspent AA points
  • Update skills & languages
  • Update equipments (including augments and charm)
  • Update spellbook


Moreover, when you synchronize your character, you will see this seal on your profile.
It certifies that your profile is authentic not a fake one, it will of course vanish if you manually alter your profile.





Note: The synchronization is limited to a total of 3 EQ accounts, deleting all the characters from a given EQ account will not free a slot. Once Premium, you will have the option to upper this limit (See settings webpage).

Beside the synchronization, as a premium member you will discover some new features on the website. Most of the premium features will let you access the magelo item database one way or another. The Magelo item database is kinda unique because nothing is entered manually like other databases you can find on the net, it means no human errors and no fakes, accuracy is guaranteed.
If an item is not in the database, the owner has to run Magelo Update to add it, same thing to update an existing item.


Quick overview:
  • List of the latest 100 new/updated items.
  • List of haste / hp regen / mana regen items which can be filtered for your class
  • Armor section that let you quickly compare armor set for a given class
  • Advanced item search and really advanced one that let you create pretty fine grained query because you can search over a specific spell component / focus component whatever spells/focus got it :
    • I am interested in items which have an instant right click effect which works indoor
    • I am interested in items which proc an effect that reduce attack speed
    • And so on, you got the idea, the possibilities are really endless
  • Spell search
