Features

The goal of Magelo is to provide an easy and elegant way to build and manage a community on internet.
General Features

  • Easy-to-use content management system
  • Fully customizable layout
  • Supports users and ranks roles
  • High level of security granularity
  • Single sign-on
  • Supports for Timezones and Daylight saving times
    • All date/time in all components are in local time of the user
Content Management Features
  • Full featured forums management
    • Expanded / collapsed view
    • Flat / Thread view
    • Tags and emoticons supported
    • Personal photo and signature supported
    • Ability to set a quota over the size of the personal photo and the signature
    • Ability to mark forum read and to show new topics only
    • Powerfull search function
    • Support true moderation (message need to be approved to be able to be seen by other users)
    • All admin options : delete, lock and move message / topic
  • News management
  • Polls management
  • Events management
