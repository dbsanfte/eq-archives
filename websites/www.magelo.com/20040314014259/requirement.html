Requirement

To use Magelo, you will only need a simple internet space where to install your skin.
An home page can host a skin for example.

A default skin will be provided as an example of how to use Magelo.
You can use this skin, customize it or develop your own from the start.

To be able to build your own skin, you only need some basic knowledge in Html and in Javascript.
No need of Php Asp Jsp or other language to provide dynamic content.

Building and managing a community has never been so simple.

At the moment, the community management system is still a prototype and far from ready.
There is no ETA.
