Magelo Update



Magelo Update is a windows program that will let you upload your everquest items and mobs to the Magelo database. Run it in background as often as you can, it will help greatly to keep an updated database.

Thanks in advance and dont forget to read the readme.txt file before using it for the first time.

Files
  • Magelo Update
  • MFC42.DLL if you dont have it yet (see the README.TXT for more info)
