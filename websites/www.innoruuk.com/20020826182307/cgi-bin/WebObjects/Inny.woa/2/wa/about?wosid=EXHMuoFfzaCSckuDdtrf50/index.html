About innoruuk.com

Please send any comments about his web site to admin@innoruuk.com.

This site is not intended to be a revenue generator and has no advertising. This site is not intended to be an EQ infobase or character repository and while forum software is implemented, we decided not to use it and keep the EZ board forums in place to avoid yet another move.

In return for the donations from individuals and companies that make this site available for free, we thought we would give a little recognition blurbs in the hope of getting even more free stuff.

Programming for the Inny application is being done primarily by Skyal and Wolfypaw with additional coding help from Dayala.
The computer that hosts innoruuk.com lives in the machine room of illumineX , inc.'s Minneapolis office. illumineX also provides the network connection without charge, because they are fine people.
The computer that runs the site uses RedHat Linux using the Apache webserver and the Postgres database.
The application server to make the site dynamic is WebObjects 5, running on Java 1.3.x. The development is done on Macs using Apple's unbelievably nice operating system Mac OS X 10.1
The web site was designed by Skyal using lots of feedback from Wolfypaw ("the colors suck", "you need more dragons", "get me a sandwich", etc.). Dreamweaver and Fireworks by Macromedia were the tools of choice along with his Titanium G4 Powerbook running Mac OS X 10.1 (which he loves way more than is healthy).
