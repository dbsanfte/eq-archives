November 10th, 4:30pm
------------------------------
*Patch Scheduled for November 14th*

We will be patching the code from the Test server on November 14th. This change will include the DirectX upgrades mentioned in the portions of the patch message below.

For information on installing DirectX and updated drivers, please read EQROKManual_Supplement.doc (or .txt) in your EverQuest directory. You MUST have these updates installed in the prescribed order in order to continue playing EverQuest after Tuesday's patch. You can test your installation by running TESTEVERQUEST.EXE and creating a character on the Test server. As a note, the character you create should have a name different than any character you currently have on a Live Server.

Please check the Scheduled downtime page off of www.everquest.com for further information.

-The EverQuest Team
