Hasten/Zephyl in Rathe Mountains
by Chris - 05/18/00 13:58 EST
Anyone who has been trying to do the J Boots Quest knows that there has been a problem with Hasten not spawning, because Brother Zephyl wouldn't despawn (Zephyl is a level 50 monk, and killing him also carries some nasty faction hits).

We've figured out the problem with these two NPCs not popping/depopping. It's being updated on Test now, and will go to production when next we patch zone data, which will probably happen in a day or two, as a complete server outage isn't required.
-Gordon

Lord Grimrot status
by Chris - 05/18/00 13:54 EST
This will probably turn out to be another thing that is broken, after Abashi looks into it more closely. I've talked to a large number of Paladins and Clerics on my server, and read the message boards, and nobody has seen him spawn in 2 weeks, on any server that I have heard about.

Grimrot is spawning, he's just got a fairly long spawn time. I believe this was adjusted a few patches ago because he was spawning too frequently.
