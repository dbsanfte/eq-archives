New EQ client changes
by Chris - 08/08/00 02:10 EST
All good changes. Excellent in fact.

Hello all,
We're currently testing some new changes on the **Test Sever** that players should enjoy. I thought I'd come give everyone a sneak peak at what we are working on:

- Text messages that wrap will be indented on subsequent lines. This means that if a person or NPC says something that is more than one line long, all subsequent lines will be indented a few spaces. This should make it a lot easier to pick out separate messages.

- /trackplayers [ON|OFF] command. This will allow rangers (and only rangers) to remove or include other players on their tracking display.

- New file, EQCLIENT.INI in your EQ directory will allow you to set some interesting options: 1) Color of all text messages (by setting RGB values for each defined text type: text types listed below), 2) Default "RUN", 3) Default "Overlay" view mode.

The EQCLIENT.INI setup info follows:

EQCLIENT.INI documentation

If the file does not exist it will be generated by eqgame.exe with default values. If an entry in the file is missing it will be added by eqgame.exe with default values.

The [TextColors] section contains entries for various text colors in the client. Here is a list of user color number to what it changes in game:

1 - Say
2 - Tell
3 - Group
4 - Guild
5 - OOC
6 - Auction
7 - Shout
8 - Emote
9 - Spells (meming, scribing, casting, etc.)
10 - You hit other
11 - Other hits you
12 - You miss other
13 - Other misses you
14 - Some broadcasts (duels)
15 - Skills (ups, non-combat use, etc.)
16 - Unused at this time
17 - Unused at this time
18 - Default text and stuff you type
19 - Unused at this time
20 - Merchant Offer Price
21 - Merchant Buy/Sell
22 - Your death message
23 - Others death message
24 - Other damage other
25 - Other miss other
26 - /who command
27 - Unused at this time
28 - Unused at this time
29 - Unused at this time
30 - Unused at this time
31 - Unused at this time
32 - Unused at this time

All colors to be set with Red, Green, and Blue values, in decimal. Valid settings for each color range between 0 and 255. "Black" (0 red, 0 green, and 0 blue) will be white in overlay mode, and black in normal mode.

The [Defaults] section contains some default setting for the client. Here is a list of valid settings:

RunMode - TRUE for run, FALSE for walk.
ScreenMode - NORMAL for the default window view, FULL for the full view with overlays

-Gordon

Ranger offence vs. defence
by Chris - 08/06/00 15:18 EST
If you wondered why ranger's don't receive any more skill in any defensive skills, here is why:

Though the rangers didn't get an increase in defensive skills, you'll notice that they got larger increases in offensive skills than other classes. From 50 to 60, the balance swung in favor of being more offensively oriented. Where one area received no increases, another area received more increases than normal.
This isnt' related to the hybrid mana issue.
