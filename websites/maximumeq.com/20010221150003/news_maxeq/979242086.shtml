Whoops! Archery inadvertently changed
by Geoff - 01/03/01 15:40 EST
Alan mentioned that Archery was mistakenly changed. It will be fixed in the next patch.


I have noticed that no matter what bow you use or what arrows, you cannot hit for over 90 non-melee damage, non-crit, non-disc/trueshot, non-snared/rooted/moving mobs. This used to NOT be the case, but in the last few days (since the last patch i think) I cannot hit for more than 90 double damage, when I used to be able to hit for more.

Yes, bow damage was unintentionally capped recently. We'll be changing it back when we patch next.

The reasoning behind it this mistake is pretty complicated. Basically, there was a damage cap already in place, but it wasn't being used. When we were setting up stuff for the new discipline we activated that cap (that's how it was intended to work when first designed, I guess).

Basically, we'll be putting it back as it was soon. I'm sorry about the mistake.

Alan
