EQ ServerScan by Erik Elamson (as used by EQ Vault)
EQ ServerScan is a program that shows you the status of the game servers for the popular online game EverQuest.

The program also shows you the number of users playing on each server. Furthermore, you can set the program to signal when a certain number of users are logged in to one or more servers. This will insure a true indication that the server(s) are up if the servers have been down for an extended time. You can also have EQ ServerScan start the EverQuest program. This version has automatic patching.

Features:

  • Compact, uses few resources.
  • Easy to use.
  • Can sound alarm through Internal PC Speaker (great for playing another game or music while you wait for the servers to go online).
  • Multiple alarm trigger settings (makes sure you get a true indication the servers are up when the alarm sounds).
  • Sortable server display.
  • Displays the EverQuest Chatserver banner text.
  • Autodownload and patching of new program versions.

BattleCom Version 1.3 (release) 2,149k
BattleCom is the first real-time voice communications package designed and optimized to work with multiplayer games. The BattleCom Client is the download that most users need, and will allow you to talk to up to 64 of your friends at once while playing your favorite games.

System Requirements:

  • 28.8K modem or higher (ISDN, DSL, Cable modem, LAN, etc.)
  • Pentium 133 or faster computer
  • Full duplex sound card
  • Speakers and Microphone
    a good quality integrated microphone/headphone headset combo is recommended
  • 32MB of memory
  • Windows 95/98/2000
  • DirectX 7.0 or higher.

The BattleCom client is now FREE.
