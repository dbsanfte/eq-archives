I learned it when I was a few levels before Canni, and I saw someone else sitting and standing constantly, and I was like ummm... what the heck? And he explained, so I set up my macro and used it since 24.

I've found myself using it a lot less though, even indoors, as I simply hate waiting for each tick. Seems I get a good deal more mana if I simply chain cannibalize and then Q back up. Plus, I really hate squinting and waiting for each tick. I used to do it based on large jumps in HP with regen/fungi, but as my hp and mana pool rose, that became harder to do.

Also, most of the times I'm indoors these days, I'm moving constantly, and I simply stick my canni's in between clickies, or hit them whenever I have a second, and never really have time to sit anyhow.

So I set up a macro for hitting Canni IV multiple times, with /cast # and /pause. I use that a lot more than dance, nowadays. Then, if I'm outdoors, I do the same thing, only to a much higher efficiency thanks to horsey med.
