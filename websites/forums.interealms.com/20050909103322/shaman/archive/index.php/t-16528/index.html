I won't get into the whole Norton vs Mcafee argument, since we all know Grisoft's AVG is superior anyways. Regardless, alot of virus checkers will flag viruses that Windows has put in a restore profile, but it cannot remove them because they are Windows protected. You don't mention your OS, but for the Windows XP family:

1) Right click "My computer"
2) Select Properties
3) Select the tab marked "System Restore"
4) Check the "Turn off system restore" box
5) Dismiss the properites sheet (Click ok)
6) Acknowledge the dialogue box confirming you want to turn off System Restore
7) Reboot - all system restore files are wiped - turn it back on if you want.

-Good luck

That one does not definetly sound like a virus to me, you might just have hidden files turned off. To remedy this, do the following:

Right Click on Start
Left Click on Explore
Left click on the Tools menu
Left click on Folder options ... (on the bottom of the tools list by default)
- A window will open
Left click on the view tab
Now check the boxes "Display the contents of the system folder" and "Display the full path in the title bar" these 2 options are turned off buy default.
Finally move the dot from "Do not show hidden files and folders" to "Show hidden files and folders"
Left click Apply then OK then close all windows.
Open up that C:\_restore folder and it should show you ALL of the files within no mater what ... failing that may i recomend Norton System Works 2003 ;)
