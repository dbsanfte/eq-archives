Thank you a ton! I had this happen to me and it really is a pain. If the virus does infect you and RPC keeps terminating causing you to restart heres what i did (Have windows XP):
1) Start up the computer and quickly hit the start button then hit Run.
2) Type msconfig and hit enter.
3) Go under the BOOT tab and hit Safemode.
4) Restart the computer and it should be in safemode
5) Then run scan disk this will take a long time (6 hours here)
6) Go to msconfig again and uncheck safemode and restart computer.
This fixed it for me hope it does for you aswell.
