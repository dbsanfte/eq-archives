Whatever you do stay as FAR away from Windows Me as possible. It is literally the red-headed stepchild of the 9x line. It blows donkey balls. If you go 98se then you can only run 512megs of ram. If you go Winxp then you can run the full gig.

You have alot of processor. If I were you I would get a much cheaper cpu and spend the money on video card instead. The benefits will be far greater for eq.
