Jilariz
Removing the title bar off chat windows is very very easy.

Find the file "EQUI_ChatWindows.xml".

Scroll down to the bottom till you find this line:

<Style_Titlebar>true</Style_Titlebar>

And change it to this:

<!--<Style_Titlebar>true</Style_Titlebar> -->

Basically comments that line out. You can do this for any window, just have to find the right file.

Note: Don't modify the file in the default directory.
