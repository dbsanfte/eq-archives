Dandi
You can also do a search for items with an Effect. It's one of the options you can filter by, I think down towards the bottom. So I'd limit it to iksar shaman and then to items that have an effect. You'll get lots of things with other effects, but anything with EB should be in there. The list won't be so long that it'll take terribly long to check each one. You could *try* doing a search for enduring breath to see if the text search looks at effect names as well as item names, but I'd be surprised if that works. There are also different names for the same effect (Enduring Breath, Breath of the Dead, etc.) so you'd still wind up doing multiple searches or missing things again that way.
