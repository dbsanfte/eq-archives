Valgoth Ulv'Sagart
Had the same problem with one of my old accounts I had setup for my brother MONTHS ago. You'll have to remember the accounts login name and password since the CD Key is already activated/associated with that account info. Once you do that it will tell you "You are not currently subscribed to EverQuest. Would you like to subscribe now?"

Just log in using the accounts login name and password. That's all. The system will walk you through the rest (resubscribing).

If you don't remember the password, but the email on the account is still active, there is an "I forgot my password" on the Station's website. I used it to retrieve my old password.
