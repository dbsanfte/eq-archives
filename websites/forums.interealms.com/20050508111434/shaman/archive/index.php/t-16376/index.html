There are help files in game to tell you how to do this.

Take item, and NPC purchased solvent, put the two inside the same container you use to put the augment in in the first place, hit combines. Augment removed, augment destroyed.

At present, you can only remove Type 1/2/3 augments. Types 4-8 are PERMENANT, unless the devs add in solvents to remove them.
