Caanus Grey
Dragons will always assist each other unfortunately, as long as you are fighting one of them, or dragging a corpse your faction is meaningless.

However, if you are not directly engaged with a mob it seems that the aggro assists work a bit differently. It is a lot safer to run away from a recent pull if you are ally then if you are KOS. So tag the mob with slow/malo then run it over to a place where nothing will path near you (Over the cliffside down on the beach near boat is where i normally use).
