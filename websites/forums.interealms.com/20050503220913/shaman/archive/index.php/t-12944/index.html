Zhorn
When running 2 EQW's there are two main things you need to be aware of: CPU usage and Memory usage.
Ideally you reduce both as much as possible so that you don't lag horribly or crash.

There are many, many good ways to run 2 EQW's better on the EQW forums. But here are the main ones.

Turn off all SoL models (except your model and horses IF you own a horse)
Disable sound PROPERLY. This means making an empty text file in your EQ directory called "nosound.txt". Turning off sound in the options menu doesn't fully stop EQ from wasting CPU time generating the sounds. Make that file and it will.
Get a memory defragger like Memturbo. This can help once you get lots of memory leaks/fragmentation from zoning lots. Can save you a reboot. Or not!
Yes turn off all particle effects, dynamic lighting, etc etc.
Turn down clip plane on the chars not pulling. Also turn off PC names and NPC names.

one thing that REALLY helps ( besides turing everything off ) is installing EQ into a different hard drive. its amazing how much faster it works that way. Also, just turning off sound on one ACC will greatly improve performance.

here are a few tips

dont zone at the same time unless you have to

hotkeys are your friends =)

try to focus on one acc and run the other off of hotkeys. it just makes things so much easier then alt tabbing all the time.

put the windows on top of eachother, this REALLY helps with video lag. example. if you're using to say 1024x768 run the 2nd account at 800x600 and place it underneath your first, but running both in the same res works just fine too, just make sure they are directly on top of eachother.

The thing about EQW, their switch window mode and mouse release are all you need.
Cntr + Alt + X
Cntr + Alt + Y
Or whatever they are set at... this is what I did, rather than hitting that every time, get yourself a Intell-mouse with the 2 side buttons and affix EQW's cntr alt etc to them. Then its just a click of 2 seldom used buttons on your mouse. That takes care of the alt tab issue. You do have to change your EQW.ini to a new command for switching windows and releasing the mouse. But thats easy to do. Then open up your mouse manager program and assign the same 2 new commands to the 2 side mouse buttons. Makes 2 boxing on one PC a whole lot easier.

Also on the issue of running 2 accounts on only one install. From what ive heard this is really bad to do, its putting twice the work on your hard drive partitian and could fry it. Its best to have 2 different installs on 2 different partitians. Then just set up to desktop icon shortcuts with one labeled Main account and Secondary account.
