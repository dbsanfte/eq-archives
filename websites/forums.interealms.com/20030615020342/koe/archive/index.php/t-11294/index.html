I also wanted to mention that I created a very simple story window mod that enables you to look up basic spell info ingame without needing to alt-tab or run windowed. I think Syndaen may have mentioned it once in guildchat (he asked me about it and then started typing heh). I updated it this morning with LoY spell info, so if you downloaded it prior to this afternoon, you might want to grab the updated version. It's only an additional 2 spells per class, but it save the few minutes of work from adding them yourself. The following is the description from EQInterface.

***
Story Window Spell Lists
created 02.17.2003 | updated 02.24.2003


This mod simply puts spell lists in the story window for easy access. Just unzip the .txt files into your \EverQuest\storyline folder. They will be available no matter what UI you are using, and you can edit the spell lists using any text editor if you find errors or wish to add notes to what is already there. Alt+N brings up the story window ingame.

The spell lists now include the spells added in the Legacy of Ykesha expansion (2 spells per class).

Note: The spell information was taken directly from the spell lists at EverQuest Casters Realm (http://eq.crgaming.com)...please send errors and corrections to them, not me. :)


Hope you enjoy it!
