The turn help syncs to sync the client and server. As when your running in a straight line the client/Server tends to try to save bandwidth and not update/sync with each other near as much. Soon as you change directions turn/look left right. They resync with your new current heading. I'll help when you run / sit, canni / sit, or run / cast but making sure the server knows you have stopped running and are now sitting / casting.
