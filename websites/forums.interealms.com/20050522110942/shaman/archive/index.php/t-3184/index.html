I think the coefficient weightings will vary by circumstance (e.g. For me at level 52, they'll be different than an end-game shammy like Bini.)

Things I'd love to see:

Ability to select between levels (from level 51 to 54, for instance)

Add focus effects as modifiers (I know I swapped out gear for some focus effects)

Have a way to get back to the first screen with your choices intact (annoying to have to put them in over and over when I want to change one little thing- a two frame thing might help here.)

Multi-select of certain fields (class, server, etc.)

A way to narrow searches by attribute (e.g. End game shammies generally have more than 2000 HP, so I might want to search only those profiles with more than that.) Since you won't want to add another selection section for each attribute, you might want to add a few "open" selections:
{Dropdown with attribute names} {Dropdown with Greater/Equal/Less than} {Text box where value is entered}

On the result screen, show filter selections at the top instead of giving them their own column (e.g. I don't need to see the "Server" column if I selected Karana, just put Karana in the header at the top of the table.)

Lose the "Profile_Type" and "Magelo_ID" columns (if it's hyperlinked anyway, you don't need the magelo ID as a separate column.)

Pretty up the selection screen- don't use the database column names (ATTRIB_CHA) - this one is obviously less important.
