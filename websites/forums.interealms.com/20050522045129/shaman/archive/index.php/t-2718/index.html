Nektar
Ok, well I'm level 60, a lot of stuff in Hate I can slow fine, but I'll try to break it down for you.

Clerics of Innoruuk are hard to slow, normally can be done after malosi and tash. If someone can malo/mala, that's a definite plus, but if not, let the mage try malosi'ing if there is one. Mages normally have an easier time landing malosi on the same mob. If not, wait till the chanter tashes and try malise/malisement (keep both memmed so u can chain/alternate between the 2). Once malise landed, then you can probably land slow.

Anyways, the point is Clerics of Innoruuk have fairly high resists and have to have their resistances debuffed before trying to slow.

Spite Golems, forget about them, they're immune to slow.

Ahsenbone drakes, they're fairly resistant, and can generally be slowed after tash but... you have to be in melee range to be able to land something on them.

The rest on first floor, except for nameds, is pretty easy to slow imo, like spectres, rats, evil little imps, chests, kira-whatevers, revs, ghouls, etc.

Second floor is a nightmare to slow. It can be done, but mobs have extremly high resists. Need chanter to tash (level 56 tash if possible), bard for occlusion (level 55 mr debuffer song lasting 18 seconds so you might wanna let him chain it until slow sticks), shaman for malo or mage for mala. Once tash, occlusion, and malo/mala have landed, then you have to chain malosini till it sticks as malosini does not generate much aggro. If succeeded, then you can slow. By that time, even if the slow is resisted, you shouldnt get aggro as the MT should have pissed it off plenty during the time it took to land all the prior debuffers.

Oh yeah, Innoruuk is slowable too :)
