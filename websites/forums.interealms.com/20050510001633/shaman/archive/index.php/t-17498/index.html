Zhahz
I have recently reintroduced myself to EQ after a break of a year. Does anyone have a link or any instruction on how to use the map window?

Usually you want to disable the height filter so you can see everything.

You can use buttons or your mouse wheel to zoom in and out (while your cursor is over the map).

You can click and drag on the map to 'scroll' it around if you're looking for something in particular.

At the bottom right hand corner is a button that'll open up tools for the maps...like if you wanted to add a label for a wayfarer camp in SRo that's hard to find or something.

Those are the basics that I use...mostly the zoom and scroll stuff.

Dink around with the other buttons and stuff to find out if there's anything else that might be helpful to you.

You can also edit the map .txt files directly. Generally the last 3 numbers on a line are the RGB values for the line. Most of the light colored lines are 155, 155, 155. Just Find/Replace these with a color of your choice like 0, 0, 0 (black).

Personally, what I do is set the faded opacity of the maps to 0 (right click > backgroud > alpha > faded level), which is almost what it seems like you're supposed to do for those maps from eq-toolbox. They usually have light colors for the lines like grey.

With the opacity of the window at 0, it doesn't affect the lines or words on the map, so they're totally clear and you can still see what you're doing.

(I also have my fade delay set to 0 in the options menu so I can really quickly mouse-over the map if I'm having a hard time seeing something for whatever reason)
