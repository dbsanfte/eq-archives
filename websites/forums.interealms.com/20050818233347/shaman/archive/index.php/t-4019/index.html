I posted the files!

Go to your EQ folder and open it, there will be a folder called Uifiles. Open it. There will be a folder called default. Don't ever mess with the default folder.

Next to the default folder, create a new folder named something easy to remember, like skin or mod.

Open my zip folder and put all those files into your newly created folder.

Load up your character, type /loadskin (name of your folder). Move the windows around the way you like. Voila.

If you're looking at other people's mods, you can borrow just bits and pieces that you like. If you look in your new folder, you'll see some .xml files named stuff like:

EQUI_BuffWindow.xml - buff box
EQUI_PlayerWindow.xml - player window
EQUI_SpellBookWnd.xml - spell book
etc.

Each of the EQUI files correspond to a window in the new UI. If you find an inventory or spellbook (for example) you like better, you can simply cut and paste them into your new folder and it will save over the ones there.

Send me a tell in game if you ever have questions. I don't know how to create the stuff myself, but I sure know how to borrow.
