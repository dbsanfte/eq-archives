If you care at all about dollars, don't go with a Pentium. The Athlons are cheaper and faster.
This is what you want for an ultimate and still affordable system:
All prices from www.pricewatch.com from a vendor I used for Hammer's new system.

Base System: AMD Athlon 1800+ XP with and Asus Motherboard (I'd recommend the Asus A7V266)

Memory: 512MB NONECC SDRAM (2 DIMM)
For memory you can go top end and get Corsair memory CAS2 or for a bit less cash go with Micron. You can get 256MB CAS2 Corsair chips for $57 each.

Hard Drive: 20 GB EIDE 7200 RPM Hard Drive
Dont bother going under 60GB for a HD, you'll only save about 20 bucks. Western Digital 7200RPM drives are great and have a great warranty. Just make sure you get the ATA/100 model.

Monitor: 19" Grey Dell M991 (18.0 vis) Don't go Dell. Viewsonic is a great monitor and much more affordable.

Video: 32 MB ATI Radeon w/VGA Cable
GAH NOOOO!. Get yourself a Gefrorce card, that is what the SOL upgrade is being designed for. Best card is the Geforce 3 Ti500, great power at about $300 US. Next step down is the standard Geforce 3, for around $225 to $250. For those wanting to save some cash but have great perfomance go with the Geforce 3 Ti200 @ $180.

Sound Card: Soundblaster Live! Value 512V
Go with the new Audigy Sound Card from Creative Labs. The SBLive is old technology and you might as well get the newer card. Go with the low end Audigy card though for around $60. You won't need the Midi crap or other hookups with the more expensive versions.

Primary ROM Drive: DVD-CDRW Combo
There is no DVD and CDRW combo drive that I've ever seen. If you aren't going to create CD's then go for any DVD drive you like the look of. There is no real difference to the ones out there that I've seen but don't get a cheapo model, it's not worth the hassle for a few bucks.

Total for a complete new system that is the best of the best I've been able to put it together for around $1300 US (was $1400 last week) plus shipping. That is a complete system with Windows XP and all hardware required except a mouse and keyboard, better you pick that out at a local store with the "feel" that you like.
