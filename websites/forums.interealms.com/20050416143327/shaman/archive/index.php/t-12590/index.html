Last Update: December 01, 2003


When posting updates please only include personal first-hand confirmations of slowability.
-FULL NAME OF MOB
-NAME OF ZONE
-SLOW SPELL USED (MAGIC - TURGURS or DISEASE - GRUMMUS)
-IMMUNITY OR MITIGATION MESSAGE


If you are reporting a "contested" mob is slowable, screenshots are appreciated! It is assumed that magic/disease debuffs are cast - Shaman Malos line (or Insidious Decay), Enchanter Tash line and/or bard debuffs like Harmony/Occlusion of Sound.

I'll be periodically cleaning up this thread to aid in readability, while I update the tables on the web. If your post doesn't include full mob names, zones and slowability info that is used to update the tables it will be removed. All posts used to update the table will be kept for future reference.
