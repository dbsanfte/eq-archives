Joril
changing icons is easy if you have a graphics program that can load .tga files. paint shop pro is a good one for this.

the graphics files are located in the uifiles directory, then under the appropriate subdirectory (ie: default, or whatever you named your new one)

there are 3 sets of files:

gemicons01 - 02.tga - this is the spell gem icons

spell_icons.tga - this is the old icons for the spell gems, the effect window, and spell book

spells01 - 05.tga - this is the spell effect and spell book icons

all of the icons are listed in these files. all you have to do is cut and paste between files. for instance, the icon for focus is that ugly dream catcher. find the dream catcher icon in the gemicons and spells files and cut them out (if you dont know how to do this, then you shouldnt try altering anything here, you can crash the skin which will crash EQ to desktop). then find the old shield icon in the spell_icon file and copy it, then paste it into both the gemicons and spells files.

if you have all three files open you can paste the copied icon into the other two with the "paste into selection" option. that way you dont have to worry about resizing anything, it will automatically fit it to the appropriate size.

this can be done with any icon to replace any other icon. i liked the new orange eye icon, thought it would be good for acumen, so i copied it and put it instead of that weird shield like thing acumen had.

also icons are used for several differen spells, so when you replace the icon one spell it will replace it for all of them. for example talis of epuration uses the same icon as talis of jasinth. i found a icon in the new set that had the shield with the poison and disease items, though it was perfect for talis of epuration instead of the what ever ity had before (i dont remember). now all my resists have the same shield with a time for the appropriate resist. problem is talis of jasinth, which is disease only, has the icon for disease and posion resist. i didnt think this was a big problem since they dont stack anymore and i wont be using jasinth. this is just an example of spells using the same icon. usually they are all in the same class so its no big deal, but sometimes its not. i dont think anyone will run into anything strange, but you might....... if you do and dont like it, can always change it back >:)
