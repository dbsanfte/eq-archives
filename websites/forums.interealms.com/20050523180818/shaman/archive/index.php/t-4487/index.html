Try another CD like the guy above said but if that don't work consider this.

Save all your character files, any files with your characters names on them, UIfiles and save your eqclient.ini to another folder then uninstall your old EQ via add/remove programs. DELETE the EQ folder after uninstalling. Reinstall EQ, Kunark, Velious and then SoL, then add your saved character files, DON'T add the eqclient.ini let it make a new one those settings are easy to change in the new UI, you can use the old to compare with the new if you have problems. DO NOT log on until the whole mess is installed, log on and patch up, enter codes, should only need to enter new SoL code, and get the game playable using the old interface. Don't forget to use the advanced log-in to get the optional zones and files, i.e Warrens, Stonebrundt and Jagged Pine, do this after a successful log-in. Then you can switch to the new and set that up.

Set-up for SoL will ask if you want to re-texture the old world and expansions, this sounds like where you had the problems, a new install will make sure the expected files are there.

If don't have broadband or don't feel comfortable re-installing EQ, do like the other guy said and try another CD or take it back for another. I like re-installing junk so that's always my suggestion. It can't hurt.
