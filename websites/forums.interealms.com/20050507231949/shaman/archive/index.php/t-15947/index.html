For IE6, make sure under Internet Options/ Advanced, Use HTTP 1.1 is checked, I also have Use HTTP 1.1 thru a Proxy checked as well.
Kamidari
Go to Tools....Internet Options, then to the Advanced tab. Scroll down a bit, and make sure both 'Use HTTP1.1' and 'Use HTTP1.1 through proxy connections' are checked. That should do it!
