Cappi Aprile
The nice thing about alchemy is that it sells really well. Potions are expendable, so there is always demand, and at every level of the game if you price it right. If I ever need cash, which is all the time since I buy a lot of toys I just spend a few hours daisy picking. It's not in the crazy amounts a deathstriker will bring you, but without the headaches.

If you have a partner, any build will do. 2 Rogues together are just plain evil. If you get improved sap it's all the better, you will be able to take on greater numbers at higher levels. I'd actually suggest both going with subtley to offset timers between you two, but really anything would work.

The only real downside, is that if it's only the two of you, daggers might be hard to come by at higher levels. Still, your build is great for instances so finding a pickuo group shouldn't be too hard.
