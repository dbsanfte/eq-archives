The macro is good. If you are concerned about losing time from having a macro for several canni's in a row and one failing then set up a macro to stand, cast canni, sit. Sometimes canni fails or sometimes the zone is lagged and i can't cast or sit, respectively. This is not a problem for lost time because my pause time is set fairly short and I can immediately sit myself by pressing sit.

I highly suggest using the macro, it is well worth it and saves much typing.

Play around with the pause time if you want until you are satisfied.
Bigguss
To include fizzle redundancy, have your macro look something like this:

/sit off
/pause 2, /cast 3
/pause 2, /cast 3
/pause 14, /cast 3
/sit on

(stole this from soupypoopy on this thread http://forums.interealms.com/shaman/showthread.php?s=&threadid=161)

See how there's three casts there? That's bascially in case there is a fizzle or two. If the first one (or second if the first fizzles) succeeds, the others will just not cast. This gives you three chances to cast in each cycle. Pressing the hotkey will stand you up, cast cann once (even if it takes three times to get a successful cast), then sit you back down. You just need to press it every few seconds... or create another hotkey to press this one over and over. :)
