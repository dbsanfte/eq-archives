Make sure the EQW files are not in your eq directory (sub-dir is ok)

Look for that file in you everquest directory... If it's not there run the patch and make it get that file.

If it is you are probably not setting up your EQW shortcut right... make sure that you use fully qualified paths for both eqw and EQ itself in the Target field of the shortcut... for example: Target: ["c:\program files\everquest\eqw\eqw.exe" "c:\program files\everquest\egame.exe"]. you need the quotes around the paths because program files has a space.
