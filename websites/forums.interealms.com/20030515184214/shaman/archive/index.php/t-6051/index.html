I've added a better buff window. Not moveable, although you can stretch it around to put it where to like.

Using Remelio's mod for T.King Spell Books so it looks better.

I cleared up a LOT of screen. So much infact that I can now use a second chat window, just for the hell of it lol. :)

I had replaced the pet window with a location guide a while back. I'm un-sure if I want to change it back to the original pet window, then modify it to make it look better/be more practicle or keep the location guide. :)

Actually, this brings me to my next update. Kind of a good bad situation if you know what I mean. I went around looking at this nice stuff people made for their mods. I thought, 'wow, I could really use that' and 'that would make my interface look nicer' and so on. Well, I decided to make another folder in this interface folder that would allow for more customization. Well, it doesn't reeeally add more than was before, but I put the mods I liked all into the customize folder. So now all you have to do is look through and if you see something you like, right click on it, copy, go back to the original folder and paste it there, overwriting the previous file. So I could put the location guide as a customizable feature and put in a proper pet window.

Now for the bad news. Including this customize folder, the whole mod totals out to be 9mb :( BUT, a bit of good news is that's before I put them in winzip so it may be less due to the compression in .zip files. Anyway, I might have to just include them in two different .zip files.

I'm using the old compass in this UI, just because I rarely look at the compass anymore anyway. But included in the customize folder are some more compass mods.
