The first obvious change when you enable usb is that irq's get changed around. USB controller requires one so when you enable it more things get shared, etc. I would try this. Get an usb mouse and disable the ps/2 port. that will free up irq 12 so it should be the same as before with usb disabled. If disabling ps/2 port isnt in bios then booting w/o a mouse in port should disable it.
