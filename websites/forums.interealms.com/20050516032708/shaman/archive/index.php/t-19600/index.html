Regarding the hardware aspect of this setup, macros would do no good as the KVM switch will only send keystrokes/video to and from one system at a time. Now your issue sounds more like something one of the external macro programs would cover. You would run the program to catch keyphrases or certain criteria regarding the tank that would set off a specific macro on the shaman. I dont know the "legal" implications of using say EQMacro in the game. Although using a macro program to catch say keyphrases or criteria to initiate a macro might be a warning or bannable offence. This assumption might be way off base as I havent dealt with macro programs in game and how GM's deal with their use.
