well, here's the interface I use at the moment, I've posted a link to it at the bottom of the screen. It's several mods and 1 skin all put together, and it utilizes the viewport command to position the viewable image so that I don't have to deal with all the fading windows.

http://pages.sbcglobal.net/serax/gorck_screen.jpg

Any questions, just post and I'll see if I can answer :) You will have to move the windows around a bit to fill in the spaces, but at least this will give you an idea how I run around. I like it quite a bit :)

Video size is 1280 by 1024 and the viewport setting for this mod is :

/viewport 40 0 1150 766

Just unzip this into your UI directory of choice and load it up, also, in order to get the blue border, be sure to add the line

UISKIN=<uifiles directory of choice>

or that part of the mod won't work. EQ recently changed how it loads the images, and now needs to be told which images to cache.
