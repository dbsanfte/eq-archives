Woodsman's staff gained incredibly good press when a level 58 ranger got to the last round of the BOTB using it instead of epics. Maybe someone has actual parsed dps to report, but here's some rough calculations.

Approx weapon damage by X * dmg + bonus. Assume EC bonus is 11, SW bonus 1 (offhand min), and WS bonus 20 (maybe low, don't know).

Epics:
(x * 13 + 1)/21 * 70% + (x*14 + 11)/24 =
.49 + 1.0... * X (in damage per tenth of second)

WS
(X * 31 + 20)/35 = .57 + .89 * X

Not that as X -> 0, WS outdamages epics
Find crossover point:
.49 + X = .57 + .89 * X
X = .73

If your average damage is 43 or more with the woodsman's staff, then epics will outdamage it. This is even before accounting for the haste, stats and DOT on the epics.
