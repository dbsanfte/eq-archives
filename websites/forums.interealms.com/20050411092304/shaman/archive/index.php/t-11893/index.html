Tuga
Question, did the error message specify whether it was RAM or virtual memory? If it was ram and it worked before check to see if anything new is loading at startup and eating ram. If you're using Win 95/98/ME use Ctrl-Alt-Del and look to see what's running in the background. After boot up all you need running in the background is Explorer and Systray. Also you might want to run an updated virus scan to make sure you don't have a worm or virus running in the background using resources. If it's virtual memory check to see if the disk is close to full, if it is windows can't allocate enough disk space for EQ to run and you need to clean up the disk a bit. Good luck. BTW the website Gruog mentioned is excellent and highly recommended for checking RAM, just be prepared to leave it running overnight, software checking of RAM is really, really slow. :)
