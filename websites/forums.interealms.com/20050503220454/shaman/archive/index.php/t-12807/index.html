Jenina Icemyst
When I've done trials with a shaman we've usually used a different approach. You need decent gear for AC and HPs but basically the shaman AE slows and then tanks the mobs that the melee aren't working on. Replace the enchanter with another DPS class and the shaman takes over crowd control. You know how easy it is for you to hold agro when you want it. Use that and don't try to be dps. Be the one holding the agro off everyone else so that the cleric only has to heal you and the MA. If you have torpor keep it up all the time - you shouldn't have to move much anyway. You want to be near the MA so that he can grab the next mob and the rest can assist him asap without having to run around. Keep a nuke up and have the cleric do the same. If the executioner gets to the stage, nuke. Otherwise you don't need to unless you have extra mana (this really depends on the dps in your group). If you save enough mana to slow the end mob, that is all you need. A good cleric and warrior can duo the end mob with it slowed if the cleric has some mana left.
