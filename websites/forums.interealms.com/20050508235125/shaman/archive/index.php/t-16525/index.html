Jinheim
With my guild, we just set up an MT order. First tank, secondary tank, third tank, etc... usually if it goes past the third tank we're going to lose anyways so it doesn't matter.

If the first tank goes down, clerics start healing the second tank. He AE taunts to take aggro and hopefully gets enough blast heals to stay alive until CH's start landing. If you have to wait for a tank to call out "I have aggro, heal me!" then that tank is probably going to die. People need to start healing before he even has aggro.

On most raids I blast heal. F8 + assist has always worked for me. I have never had trouble with things being too far away for F8 to target them, either. You can be too far away for /assist, but never F8. I guess if F8 just doesn't work for you, you could make an "/assist Mobname" hotkey. It doesn't have to be the mob's full name, just the first word (or maybe just the first few letters? I'm not sure). I have only used a hotkey like this once, and it was just "/assist Dain" and it worked fine.
