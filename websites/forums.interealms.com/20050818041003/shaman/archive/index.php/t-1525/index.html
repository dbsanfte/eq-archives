Alternative Advancement which you can't do till L51 is what they were talking about.

For each AA point you spend you get 1 x-tra hp, so with 3 you regen 3hp more per tick than you previously did. However those classes that have natually higher regen were getting 2 points hp regen per 1 point spent in AA exp.

They changed it (fixed it) so that it would only be one.
