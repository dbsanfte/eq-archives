Other upgrades:
Velium Fire Wedding Ring for finger, ditch the gem encrusted ring unless you are root rotting. 6ac 65 hp on Wedding ring

1924 hps from canniV.
You need 2565 HPs to not get low HP aggro from casting canniV at full health.
2565 - 1924 = 641
(641 / 2565 ) x 100 = ~25% (24.99025)
If you are 62 then you have soul + boar.
Soul is 544 hps and if you don't go over 255 sta you should get 3hp per sta so boar = 195hps
So the min unbuffed HPs you would need to safely use CanniV while self buffed would be 2565 - 544 - 195 = 1826.
Throw a virtue into the mix and you can do it with only 421 HPs unbuffed :)
