XP has two very bad issues, as far as EQ goes. Number one, if you use a dial up connection, stay very,very far away from XP. As in garlic-wreaths-and-wooden-stakes far away. XP has a problem with using ICS (Internet Connection Sharing) in a gaming environment. Doesn't matter if you havve ICS turned on or not, it's part of the operating system. Basically, XP opens a port for EQ on the first network device it finds, which 99% of the time is the network card, and not the modem. You will get constant refusals of service from the game servers and horrid zoning times.


The other issue just cropped up. The "optimization" for sound that was done in the last patch has created serious issues with SB-based cards, especially SBLive! cards. If you have an SB card of any flavor, XP or not, disable sound in the options screen before log-in and I guarantee your disconnects on zoning will be cut in half.
