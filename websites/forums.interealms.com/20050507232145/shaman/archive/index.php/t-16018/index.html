*Changes = Full time +50 CR is likely unbalancing, so thats gone. The attack is lowered also, it won't make much of a difference but it cuts down on the 'omgs so much attack' whinge likelyhood. Double attack and haste element lowered to limit the potential dps against low ac mobs to a more reasonable level.

The innate proc should be a disease based DD (NO dot element) with NO resist adjustment (IE it lands with the same frequency as BoUltor), this gives more use to Malicious decay/insid malady so we can get the most out of this ability. The reasoning for this is that if we can land poison fine, we are more likely to use BoSaryn/Nife, hence it would be an alternative source of damage when our poison dots and cold nukes simply get resisted too much.

The slam skill level is quite low, but that is to prevent it from being too effective with such a large atk element. It appears to be possible to have a buff give the player a buttoned skill (Ogre/troll/barb illusion grants the player Slam, even if they don't have it in there normal form), although how flexible this is is unknown.

It should stack in much the same way as Rabid bear, but also block voice of the berserker and form of the bear/greater bear/other illusions (To prevent getting around the size issue of being a bear, this would be a fair limitation of using the buff imho).

Its base duration should mirror the mage elemental form line with a 36 min duration and 15 minute recast time on the ability.

Visually 1 of 2 approaches is best, if global model load is an issue, simply replace the existing old bear model with the bear from PoStorms, the PoS bear is MORE complete than the old world bear as it also includes casting animations. Otherwise as suggested, if global model space is no longer an issue, perhaps using the Velious kodiak would be more fitting?

All in all, the buff is basically a combination upgrade of Voice of the berserker, form of the great bear and Rabid bear all rolled into one and brought into line in a way that would actually make it worth casting in the pop environment without it being overpowering since it still has limitations (Size issues and buff slot issues).
