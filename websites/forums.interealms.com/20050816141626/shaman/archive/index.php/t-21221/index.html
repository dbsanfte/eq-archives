this "warning" messages used to be efficient when viruses were really sent from your own computer. but new viruses are just spoofing emails headers "from" with any email adress they can find in an infected computer.

what is happening is that the postmaster tools are still using this "from" header to notify you, spoofed or not. this is why you are receiving this notification.

you are most likely not to infected if:
- your antivirus is updated on a regular basis
- you are using anti-trojan tools like "ad-aware" and/or "spybot"
- you are using a firewall with correct settings (use shieldsup at https://grc.com/x/ne.dll?bh0bkyd2 for test)

and the rate goes up to "99% not infected by new virus" if not using IE and Outlook :) (go to www.mozilla.org) (or using linux instead of windows)
