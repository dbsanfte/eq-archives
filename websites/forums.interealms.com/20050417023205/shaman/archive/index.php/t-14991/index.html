Well the run there is a long one, but basically its this: go to Butcher Block, take the FV boat, hop off early and swim to shore and run along the edge of the beach and get into warsilks woods and run across there and into field of bone. You can also just beg for a port there.

Now that I think about it the game is loads of fun being to travel easily with PoP. Highly recommend dropping the money for the expansion if you are serious about playing and having some good times.
