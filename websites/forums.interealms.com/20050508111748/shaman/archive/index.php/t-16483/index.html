weissblau
Note that if you have a hotkey that invokes Canni5 it won't stay depressed after you use it, so it can be hard to see when it's usable again. What I did was put Radiant Cure and Canni5 on their own hotkey page, with the hotkey set up in the left column and the raw AA key set up in the right column. Then I switch to that page to check to see what AA abilities are up - if the key says Canni5 is available, I hit the hotkey that makes the announcement and invokes it.
Kianor
What I do for Canni V and Radiant Cure is put the raw AA hotkey on my main page, so I can see when it pops, and put the macro hotkey on another page. That way, I don't have to keep checking a different page to see when I can use the skill, I just switch when I see it has popped.
Piglet
I do the same as Kianor, also means that if you're soloing or in a group where you don't need to annouce a C5, you don't have to switch anywhere to do it.
