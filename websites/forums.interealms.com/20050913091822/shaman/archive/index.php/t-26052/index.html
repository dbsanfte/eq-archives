I posted this on my guilds forums, but thought some here may find it of use:

Slouken, one of the UI designers for Blizzard, has acknowledged a problem with the way the client saves some of its window/ui frame positioning data. While a permanent fix is on the way (not until the next content patch, which there is no telling when that will be), below is a temporary fix that has worked very well for me.

1) Open Windows Explorer (not Internet Explorer).
2) Navigate to your World of Warcraft game directory.

or

1) Right click your World of Warcraft game icon on your desktop.
2) Select "Properties", then click the button that reads "Find Target".

3) Double click the "W T F" folder. (spaces added so it's not censored).
4) Double click the "Account" folder
5) Double click the folder that is named the same as the account name you use to log in.
6) Double click the folder for the character you're experiencing video lag/low fps with.
7) Double click (edit) the file named "layout-cache.txt".
8) Look for instances of the parameter named "FrameLevel:"

Any of those instances with a numeric value higher than 50, edit the numeric value back down to the number "1". Do this to ALL instances that have a value over 50!

Save and close the file.

Restart World of Warcraft.

While this may not resolve all low video frame rate issues on all systems, it has worked very well for me when I notice video lag or low FPS.

Hope it helps.FYI, "50" is just an arbitrary threshold I decided on personally. Usually the ones causing problems have values in the hundreds of thousands and are easy to identify.
