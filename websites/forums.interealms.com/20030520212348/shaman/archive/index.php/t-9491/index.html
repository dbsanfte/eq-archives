I just started a shaman and was wondering if people could give me a quick sum up of a shaman's responsibilities in groups. Is there more to it than just buffing and slowing? Any info would be appriciated.

Gloriana
Buffs:
Haste - all melee (if enchanter not present)
Hitpoint buff - self, main tank, other melee if mana is sufficient (ask casters/clerics before casting since it interferes with their own buff lines, they might want it)
str/dex - melees, if you have the mana, str for overburdened clerics/casters if applicable
agi/sta - A little more AC/stamina never hurt anyone, give to everyone if mana is sufficient
regen - good for you, the main tank and any necros if applicable
SoW - puller, everyone else if they're scaredy cats
infravision/ultravision - good for night blind races (barbs, humans, erudites) wizards/enchanters get their own line

Debuffs:
Slow - everything
Malo-line - helpful if you have mana (especially with nuking casters present)
Cripple - not hugely useful at the lower levels

Heals:
Your job if you're the only healer
Assist heal if other healers are present
Watch out for enchanters, they drop fast. If a mob is mezzed, make sure no one breaks mez until the mob is taunted, if no taunt, root it. Sit aggro to save an enchanter is sometimes necessary.

Offense:
Root park if necessary, get the mob's attention (slow works nicely) walk it over out of harm's way and park it
Nukes - if mana is sufficient
Dots - if mobs live long enough for it to be mana efficient

Can dance mana back during fight so when fight ends you are full health/full mana ready for next pull. Downtime = the devil.
