Anomalos
My XP PC is P3 850 with 512 MB ram, and a Geforce 32mb. I can run 2 EQW flawlessly, but you kinda have to restrict alot of stuff to make it run. A few changes need to be made to the eqclient.ini and eqw.ini. In eqclient.ini be sure to turn OFF velious armor. In eqw.ini under background set VSYNC=TRUE, and UPDATE=FALSE. Next, before you goto connect on EQ goto options, remove ALL luclin models. On the next screen turn off mip mapping, texture compression, set resolution as low as possible, and texture resolution to low. Then lastly when you get on EQ goto display, make sure you're running lowest possible resolution, particle density off, 3d sky off, and clip plane all the way down. Be sure to not hit F10 and goto full screen as it lags like mad. When i'm running 2 chars this is how I setup my backup (utility) class, and the main class i'm playing (my monk) is identical except I run at 800/600 in full view to see what i'm doing better.
