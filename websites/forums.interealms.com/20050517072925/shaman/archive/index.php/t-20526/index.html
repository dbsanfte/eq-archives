you can find it at EQWindows (http://www.eqwindows.com)

For those that don't know EQW is a program that lets you run EQ in a window, and does it much more effectively than EQ can

New features include;

- added support for DirectX 9
- hotkey support for: Win (windows key)
- custom window title
- GlobalActivate hotkey to activate a specific EQW window from any app (including other EQW sessions), default is Ctrl Alt A
- if you want to use this feature with multiple isntances you'll need a unique EQW directory and GlobalActivate hotkey for each instance
- EQWSwitch hotkey will cycle through all EQW windows instead of just 2

What I had to do was to roll back my video driver. I had updated to the latest Nvidia (56.64) after prompting from EQ yesterday. I rolled back, restarted Windows, Downloaded a new 56.64 from CNet (NOT the Nvidia website) and installed and the rebooted.

Just a note that EQW 2.41 is out and correct the gamma problems. Also a few other tiny fixes:

- some DX9 fixes (framerate and problems after server select)
- gamma fixed
- lag/choppiness starting at EULA screen fixed
- alt-enter no longer switches to full screen or crashes the game but can still be used to clear up "frame lag" as in previous versions
- added hotkeys to launcher
- hotkey support for numpad, use numpad0 numpad1 numpad* numpad/ etc.
- hotkey support for f11 and f12
- custom titles will have to be reentered
