Today Sony Online Entertainment released to the world their upcoming Alternate Ability and Spell List for the soon-to-be-released expansion, Omens of War. Included in this expansion is a heretofore unattainable ability for two classes (Druid and Shaman), the ability to rez a fallen compatriot.

Should a Shaman decide to purchase the ability via AA, the cost is a nominal 70 AA points. ("We like the symmetry between level obtained and AA cost," said one unnamed developer)

I didn't want to see any more of this "giving away unique class abilities", I wanted a return to each class doing one job and one job well.

May seem ****ty to some, but to be perfectly honest, class blurring is one of the best ways to address a lot of the class balance issues currently plaguing the game.

Frankly, we have way too many classes for there to be "unique class abilities" to not be given away. When you have 16 classes, especially considering that the base group is only 6, you simply have to have a degree of class blurring. You can't have "required abilities" that are unique to a class; for true balance, you can't have 16 required classes in a 6 man group, and you can't have some required classes and others only optional.

For true balance, archetypes should be relatively well-defined, and each required. The classes that comprise each archetype, however, should be interchangable, however. They can have different advantages and different means of achieving their role, but one shouldn't be universally superior to the others.

Now, with the way EQ has developed, achieving that in a pure sense is practically impossible, particularly because of warriors and to a lesser extent, clerics, but that doesn't mean it's not a good concept. Any opportunity to reduce the extent to which one class is required over other members of the same archetype should be embraced - this pseudo-rez is one such opportunity.

But I digress... I guess writing 4 paragraphs in response to one sentence might not be the best idea.



This abilitiy, while it may not single-handedly fix Shamans (or Druids), is certainly a boon. Sure, Shamans still need healing improvements to be more in-line with clerics/druids and to help set them apart from being a low-DPS beastlord, but that doesn't make this any less beneficial...
