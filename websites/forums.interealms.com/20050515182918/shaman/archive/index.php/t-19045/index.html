Mez and pacify both use charisma for the secondary check.

On mez, the primary check is "does it land?" which is based on level vs level and MR. The second check is "if it lands, does it wipe aggro?" and that includes a check on the caster's CHA.

For pacify, the primary's the same as mez. The secondary check is "if it does NOT land, does the mob aggro?" and is likewise checked with a formula including CHA.

So for a lot of classes it's not useless, but not doing much for shamans.
