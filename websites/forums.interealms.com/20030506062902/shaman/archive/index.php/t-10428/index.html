Harla Dar (http://eqbeastiary.allakhazam.com/search.shtml?id=7056)

Spawn point is roughly 0, -3500 in Western Wastes. It wanders around, but fight it there because it gates. Claws of Veeshan faction, so wyverns and dragons who wander by will assist. It rampages. It casts enchanter spells (tash, slow, stun, charm, fear, dispell) so I would say buff magic resist. Takes 2-5 groups to kill it.
