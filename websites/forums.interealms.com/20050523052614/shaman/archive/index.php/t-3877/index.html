One of my guildies had the same problem..

This is the reply that one of our techie boys posted for him :)

Symptoms
When you play a file using Windows Media Player or run the DxDiag.exe tool, you may receive the following error message:

Ddhelp.exe caused an invalid page fault in module Mgaxdd32.dll at 015F:BAAL521F
Cause
This issue can occur if your computer includes any of the following items:
Matrox Millennium II video adapter
Microsoft DirectX 6.1
An incompatible video adapter
Resolution
To resolve this issue, contact the manufacturer of the video adapter to obtain an updated video driver

Hope this helps you! Seemed to be a driver problem for the video card. He downloaded updated drivers and it fixed it.
