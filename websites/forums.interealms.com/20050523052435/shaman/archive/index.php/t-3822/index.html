Taladaen
Do you know how to fully remove the drivers? You need to find all the files related to the video card drivers and delete them, not just uninstall the driver. While you're doing this you'll have to be in VGA Mode, 16 bit color, all the really big, blocky graphics that windoze wants to give you. The safest way of doing this is switch to VGA settings & reboot. (Have a boot disk handy in case you really screw up, too) Create a new directory somewhere and move everything that even looks close to a video driver there. Create a text file with where you moved things from so if the system is unstable you can move them back.

Once you've done that & rebooted to make sure you didn't snag a file windoze needed, then go thru the process of installing drivers. You want every system file in there to be the newest version & freshly installed.

Since you said you're upgrading soon, an interm fix and relatively cheap is what I did:

P-III, 500 MHz
512 Meg RAM (bought it before xmas when it was dirt cheap)
Replaced Viper 770 Ultra (32 Meg TNT2 card) with
GeForce 420 (low end GeForce 4 card) for 99 bucks.

Basically, it's a GeForce 2 card, stripped down some, then overclocked up a smidge. I'm able to play in Luclin with the clipping plane back to full again.
