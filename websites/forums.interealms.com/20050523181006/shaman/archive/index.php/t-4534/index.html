As far as I can tell, DoTs DO cause agro every tick. Many many times I have cast the epic DoT on a mob, and when it gets to the last few ticks, the mob switches from the tank to me.

This effect is more noticable if I EBolt the mob as well as epic, which suggests the EBolt is also causing agro every tick.

There is some initial agro from a DoT as well, but unless you cast loads of DoTs all together I shouldn't imagine tanks would have a problem taunting the mob off you again.

(Talking about xp mobs here, not some uber death creature).
