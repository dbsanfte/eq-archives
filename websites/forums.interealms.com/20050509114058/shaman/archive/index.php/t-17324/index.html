This is not only true, shaman in my guild do it regularly.

This is the way we can have a single shaman buff a raid in no time.

You click the button when you first enter the zone - it will deactivate and the timer start over if you zone or camp. Wait however long you need to to begin your buffing. In the background, the timer counts down. If it takes long enough to begin buffing, it is possible to MGB back-to-back.
