Quinner
The recent change introduced on the TEST server seems to have some people in an uproar because it may become no longer more efficient to solo than to group. It and several other things have set my mind to thinking up new and interesting challenges for SOE to add to the game.

1. Solo Challenges - Instance Zones seem to be the new "thing" for MMORPG's so why not expand on this and create dungeons that are meant to be soloed by a specific class. Every class should have their own set of challenges to tackle using skills that only they would have to complete it. (Like a Shaman needing alchemy to complete the challenge.) Rewards could vary or be chosen by the player and should diminish as the overall experience level of the player increases.

2. Group Challenges - Along the same line as the solo challenges, create instance only dungeons that require a specific combination of classes to activate. Having players go LD in the middle might cause some problems, booting the players out of the zone but the point of the challenge would be to have certain classes overcome it together. Some might argue that this would cause too much frustration but then nobody is being forced to experience the challenge.

Each idea would work even better if there were some variables thrown in so challenge would be different if done multiple times by the same player.
