K, don't have all the technical info on this, but I know what is showing up in. The problem we've been seeing is people having troubles opening up MS Word, gives an error message of, "Insufficient System Resources, unable to load this application". This only happens with MS Word, everything else works fine. The new spin off Nimbda and at least 100 other Word targetted viruses infect the normal.dot file ( this is the template file that Word is opening when you see it's title screen ). By infecting the normal.dot file it makes it too big and keeps Word from opening. Delete the normal.dot file and restart Windows and reopen a blank Word doc and it should be fixed, if not uninstall/reinstall Word. When I find out more about this new spin off I'll post it, its spreading fast again just like the original version ( I had 65 calls on it over the past week and a half ). Just FYI, be cautious with e-mail and keep virus scanners up to date.

Just a heads up for a rare case.

Normal.dot can also store information placed there by programs you may need to use(possibly at work)

One example of this is GoldMine and it's document merge function with Word.

When you install this goldmine link it modifies your normal.dot to create the linked fields for your merge forms.

So... if you have any programs that do this sort of thing, make sure to re-install that function from the program after deleting the template.

In Goldmine it is as easy as opening a word document and clicking a button to re-install it.
