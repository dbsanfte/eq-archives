Is it possible to have a different interface for each toon on your account? I ask because frankly the interface I want for my Bard is not the same as the one I would want for my Shaman.

taraddar
You can have different UI's loaded for different chars. You need to make a different directory for each and just copy the files you want in there. You can modify the files yourself or download them from someplace like eqinterface.com. For most that you download you simply just decompress the file into a directory and thats all you have to do.
Jaruden
Set your 2 ui's up in their own folders (c:\...\skins\ui1\ and \ui2\, etc). Log on your shaman, type /loadskin ui1 1 (the 1 at the end makes it remember your settings). Then log on the bard, type /loadskin ui2 1.

Now when you swap around, it should automatically remember which UI you are using on each character. If you want to change it manually, those savings are in the files Henora listed.

Usually installing UI's is easy, but once you patch, look out, who knows what VI has changed. Thank god they quit making the whole game crash if your UI is out of date, I haven't updated mine in 3 months, you just get a message every time you zone about files being incompatible, but it still runs fine.
Schpleef
In your EverQuest directory there is a file called UIErrors.txt. Open it and it will tell you what's wrong with your current UI - i.e. the reason you're getting that error message. If you're a bit technical you can monkey around in the file(s) it lists and fix it. Worth it just to get rid of that annoying 3-line error message you get every time you zone or switch resolutions :)
