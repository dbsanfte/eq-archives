Zyphyr
The web info is mostly correct. The 428 number is what is in the SPDAT. What is not being factored in is the 10% non-cleric healing penalty.

10% of 428 = 43
428-43= 385. <-- what you actually get.

Every heal spell in the game is effected by the cleric/non-cleric situation even if the spell isn't available to clerics.

Something you many want to try : Go look up Superior Heal on the same site. I can pretty much guarantee that it will say 300. When it is cast by a Shaman/Druid/Ranger/Paladin/Beastlord it only does 270.

BTW, Clerics get a 5% bonus so THEY get to do 315 with Superior. If Clerics were to be given Chloroblast it would do 449 or 450 (actual number 449.4 - depends on if they round the number normally or not) for them.
