Grammar:

Yes, with VP3 that situation actually could arise. I am not sure if you would get the agro immediately, or when root broke, or when you next did something harmful to the mob, but there is the definite possability of sending the mob where you don't want it.

That being said, the mob always moves in the exact direction you are facing, so if you want it moved into a corner, face the corner. If you want it moved off the the side, face the side you want it to go. The mob will not go through walls or anything like that, so you've got pretty good control of where you want to put it. "Knockback" is kind of miselading -- it should really be called "knockwhereverimlooking". This is true of all knockback effects, like enchanter stuns, paladin stuns, wizard knockback spells, etc.
