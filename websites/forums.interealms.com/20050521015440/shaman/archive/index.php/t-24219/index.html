Agree Malo is a no brainer, helps on every mob, and required on most mini's. However, my philosophy is that since Malosinia and slow have the same MR adjust, zero, if Malosinia lands, then you could have landed the slow. Sure, landing slow is easier after landing Malosinia, but why take the extra step.

Also, some mini's are only partially slowable, so don't be surprised if you get the partially slowed message. Finally, depending on the wing, (I always forget which is which until I'm there for a couple of pulls, have not been there in a few weeks), mobs may have high MR and if you are pulling singles may be best to Malo each on inc, mana providing of course. With Canni 5 and KEI I typically Malos everything on incoming just to be safe, never know when you'll get an add. For the most part, I would start with malo and follow up with slow. If it is resisted, malosinia. I would alternate malosinia and turgurs until the mob is slowed. This is what I used to do when I was in a tower/mini raid, and tower mobs were resistant if I recall. Been ages since I killed a tower boss or a mini.

Expect to get beat on. I used to die a lot on raids - for the most part, it was do to a mob resisting slow. As you progress your character, you will be able to take more hits and your survivability will greatly increase.

Find a style that works best for you and stick with it. Some people like to wait a few seconds for the tank to build aggro before malo(x) and slow. Some Shaman tend to cast Malo(x) on incoming followed by slow. This could result with the mob making a bee-line to you. This could be detrimental on a raid if the mob was already positioned for placement by the main tank. Food for thought...
Ragack
First if you are asking which to try first, always use Malo/Malos(when you get it) as these are unresistable. Only use malosina if you need extra debuffing.

Some more points to think about, the Malo line has very small amount of aggro, so depending how mobs are pull an arrow or such can normally out aggro the Malo line so always land it asap.

With named, you are going to find slowing them tough, same with there placeholders Krigers (these can be almost as tough as named minus the AoE stuff)

If you have a Mage get them to Malosina if they can, also tash helps but due to it's aggro chanters don't tend to cast it early. It can help alot to be close enough to the mobs and have the emotes like XXX yawns easy to see as you can then tell if a mage/chanter has debuffed.

Personally on named, I malos, try a slow if it fails, I tend to either slow/malosina alternate if the tank is ok with aggro, else will stick with malosina if the the mobs not happy with me.

Learn which spells work on which wing as each has it's own resist set and once you learn which works where you save loads of mana.
