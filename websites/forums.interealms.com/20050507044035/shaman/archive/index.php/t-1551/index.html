Azhdarkhos
There are two issues here:

1) The enchanter will need sufficient Truespirit faction for the turn in - if not enough faction, Rak will eat the Tear without respawning aggro with Scale.

2) The shaman will need to make sure they have sufficient faction to turn in the Scale - if there is a faction hit when turning in the Tear, the enchanter will get the boost instead of the shaman, which might leave the shaman without enough faction.

Both of the above can theoretically be solved by doing prior steps in the quests repeatedly until max Ally faction is acquired.
