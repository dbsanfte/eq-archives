Chutoi
It's not a ring of fire type encounter at all. You get 3 mobs at a time. Unfortunately, he calls them pretty fast, so you keep getting 3 mobs every couple of minutes. Eventually, the caller gets tired, and sits down for a rest.

There are 3 types of mobs he can call. The larger fungus beasts, the smaller ones that hit really fast, and the named. All are random. You can even get 3 named.

The script goes on until you hand him the note to stop it. (He gives you the note when you start the script).

The named are uncommon (possibly rare?), and I think there are 3 named. Takacha's rare drop is the shield. You can run the numbers and see that the shield is NOT a common loot.

(uncommon percentage/3 named) x rare percentage = bloody small number!

This is a fun camp for exp with a group of six high levels, but it can be easy to get behind. 2 group should be able to handle this with ease.
