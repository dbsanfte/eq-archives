Zyphyr
skill based sounds cool

In pen-and-paper games with a good GM, Skill-based is really the way to go. It is very flexible, and lacking in stupid arificial constraints. The GM has all the tools needed to prevent the excesses that tend to occur.

For MMORPGs, it is a disaster waiting to happen. One or Two sets of skills quickly gets identified as being "The Best", and everyone takes them. If you don't, your character is considered to be "gimped". Then the developers notice the problem and nerf that setup in some way. Within days, the new "Best" skill set is identified and every busts ass to get that set instead... repeat ad-infinitum.

Then you run into another problem : in most skill-based games there is some absolute maximum number of skills (or skill points) which one can aquire. If the players aren't permitted to respend the points they had spent on their now-nerfed skills, the whining that ensues makes the stereotypical druid complaints sound like calm and perfectly rational discussion. If you do let them respend the points, all you have done is slow things down for a day or two.


To make it work, the developers have to not only make sure each skill is fair and balanced, but that each combination is as well. To the best of my knowledge, nobody has managed to do that yet.
