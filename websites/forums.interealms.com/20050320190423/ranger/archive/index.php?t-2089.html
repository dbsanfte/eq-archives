Run fdisk. Delete ALL partitions. Create a new, single partition that uses all of the hard drive's space. Reboot.

Run the Windows setup from floppy. Just after initializing the CDRom it will tell you that you have unformatted space, and will ask if you wish to format it. Say yes. Install Windows, and you should be all set.
