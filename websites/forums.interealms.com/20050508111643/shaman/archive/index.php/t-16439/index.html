Picked up the Mirror 1 proc for 30 points in Everfrost last night and slapped it on my sharkspine club. Since I couldn't find any info on it anywhere I figured I'd share my experience.

First of all, Mirror 1-4 is a 5-20% chance to reflect all incoming spells. It is a proc, and it only lasts 3 ticks. I didn't get a screenshot of it, but it also reflected AE slow cast by a mob (of course the mob resisted it :x).

The good news is it procs all the time. Just using my mace between nukes and dots I was able to keep the effect up over 75% of the time, the only time it really dropped was moving from corpse to mob if the run was far.

PVP note: this is going to mess things up. I was manaburned last night with Mirror up and if it had fired I sure as heck wouldn't be posting a cleric nuke screenshot...

Anyways just thought I'd post this info.
