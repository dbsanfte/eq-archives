Patch ui changes automatically update your default ui, not your customised ui, so you can't leave it as is.
I like to leave my default ui untouched, so I delete my customised ui from everquest\uifiles after a patch, copy and rename the default ui, then unzip my playerwindow etc back into the renamed ui file.
