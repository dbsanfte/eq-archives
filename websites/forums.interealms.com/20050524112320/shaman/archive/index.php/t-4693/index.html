Advantages of the horse -

1. No sit aggro, ever. This can be a pretty big deal.

2. If you have all luclin models off (including horse model) then no horsehead and you run at SOW speed

3. Medding all the time unless moving or attacking. I think this still includes JBB, Canniablize, and epic cast time but not 100% sure, I haven't actually tested it.

4. Even if #3 is not true, you are medding all the time while waiting for DOTs to go off, while waiting for snared mobs to catch up, etc. This does add up.

Basically, I do see a significant mana regen difference in outdoor zones. Unfortunately, it costs me Greater Bear Form, so I lose +10 WIS and some regen that I care about.

If I had to decide today, I would NOT buy the horse, since the +2 regen and +10 WIS of greater bearform would help me feel better about my normal mana regen rate. But it's useful enough in some situations that I am choosing to keep it around until I need the plat.
