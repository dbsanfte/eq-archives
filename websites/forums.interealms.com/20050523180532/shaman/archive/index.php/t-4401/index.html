Seredith
To protect your UI when new files are added, only have the UI files you have downloaded/adjusted in your custom folder, leave everything else in the default folder. The sites that tell you how to load a skin all say copy everything from default into your custom folder, and then add the new ones, dont! Just put the new ones in there.

EQ now looks for UI files in default if it can't find them in your custom folder.

If the same thing happens after doing this then one of your custom files needs overwriting too.
