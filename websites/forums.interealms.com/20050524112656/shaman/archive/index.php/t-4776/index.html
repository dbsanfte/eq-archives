there were TWO changes affecting fungi staff.

first one, if i recall correctly, made it so you had to equip it to use (originally could be clicked from inventory just like sow booties). not a big deal for shaman as we CAN equip it but probably some rogues or something were upset. this one only affected new staves, so you may be able to find one of the old variety around.

second one was a fix to the stacking problems so you can no longer trick it into stacking with regular regrowth. this affects the actual effect, not the item, so it can't be evaded by getting an old staff.
Bigguss
Let me clarify this one a bit. There are two nerfs.

1. Originally, the fungi staff used to be usable from inventory. This meant any race/class could use it. Shortly, Verant realized that this wasn't a good idea (necros) and nerfed it to be usable only when equipped. There are very few "pre-nerf" fungi staves out there- and they're worth a TON (hundreds of thousands of pp, I'd think.)

2. The stacking of the effect of the fungi staff was nerfed to no longer be stackable with regen spells. Arguably this was just fixing a bug, as the spell regen was never supposed to stack with the staff effect- but in any case this affected all fungi staves, since it was the effect that changed.

So there are two types of fungi staves out there today- those that work from inventory and those that don't.

Hope this helps.
