Hey Barb, figured I would try to help a bit if I could.

Spells: from GoD we get a new HoT, new disease dot (oh boy!), new situation slow (has a negative magic resist modifier, but crappy spell duration.), and a new canni that has been flagged as ancient (so we are not going to be seeing that one.) If I am not mistaken most of the others drop while doing GoD Trials. Not real sure because I haven't really done much of the new GoD content.

Abilities: Not much really to be honest. Spell Casting Subtlety seems to be praised as well as the Burning Affliction AA's. That's about it really. We get 2 more levels of Spirit Call, but upon achieving the 4th rank our dogs each have roughly 250hps.....

The only good news I can give you is there seems to be a way to max out our alchemy skill with a combine that will only run us about 8pp per attempt. There was a post somewhere about it in the General Section.

So......nothing really outstanding.
