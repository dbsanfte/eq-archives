These changes coincide with smart game design changes that are going on.
They will provide a break in the game's various bottleneck's (CHeal, Crowd Control, etc.), and give each class added utility in a wider variety of areas in the game.

VI is reformatting EQ into a basic game design balance.
With each key game ability being spread over a finite number of classes (2-3 it seems like) with each one tiered to provide a different level of that ability (1 > 2 > 3).
Opens up bottlenecks, and (given a 6 person limit per group)provides flexibility to really have some interesting combos.

Limits like the number of spells/songs one can have memmed at one time, number of people in one group, etc. are key to providing good and varied gameplay.
Scarcity of short term resources combined with abundance of long term resource choices is a great mix.

I would say the biggest factors in this new move by VI are:
1) smart people at the helm
2) the funding (time, people, and $$) to make the changes
3) maturation of the game, large/active smart player base (that intelligently discusses game issues), active smart designers have provided an understanding what elements are the major play strategy variables in the game.
