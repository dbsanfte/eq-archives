There is a difference between "directx 9 compatible" and "directx 9 hardware support".

As long as the drivers support directx 9 then extra features are emulated in software. Cards which are dx9 hardware accelerated have these new features in the gpu so it will get better performance.

If you want a DX9 hardware accelerated card then get any Radeon 9xxx is a dx9 card (except the 9100). All GeforceFX cards are dx9 as well.

A radeon 9000 pro will give you almost the exact same performance (as the g3-ti200) but will have dx9 in hardware. Those are pretty cheap. Anything faster will be a serious upgrade.
