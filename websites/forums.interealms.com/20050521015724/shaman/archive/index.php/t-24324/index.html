Guild tribute is an extension of person tribute. You can use it if you have GoD or Omens or both. It was originally bugged in that officers with Gates couldn't turn it on /off and couldn't use the tribute master to turn in items for the guild.

It still has some bugs in the guild window where donations are showing up wrong until you sort the window once or twice and then its showing up correctly. It also has some bugs sometimes where certain officers can't start it because the button is grayed but as soon as another officer starts it, then the originally bugged officer's button ungrays and then can turn it on and off.

I'm an officer in my guild and I have Gates but not Omens, so some of my problems may be related to that, though the posted info from Sony says I should be able to do all of it without problems :-)
