Barm McLir
Here's something handy I found if you're like me and never load Gate until it's too late...

1) Un-mem all spell gems.
2) Load Gate in the slot you normally swap it into.
3) Save that list as "GATE!" or something.

Now if things go south, I used the GATE! spell set to quick load my escape. It ignores all the gem slots you left empty -- i.e. only unmems the one spell in the slot for gate. And no page flipping.
