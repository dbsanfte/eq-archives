This is all from memory about KT.

First all trash can drop the mask and range items (I have mask, check magelo if interested)

There are 3 major 'camps' in KT. North, West and south. These are on the respective wall where the ZI's to the Ikky RAIDS are. Each camp should be near a temple on their wall, each temple should have an archer in the entrance, this is a ph for a named, these are easy to single pull if you use lev to get on the side of the wall and single the golems as they walk (watch and you will see). There are also a few places around the center temple where named spawn, i can't remember them all i don't pull KT too much...

Also a named hynid/craigbeast can spawn out of any hynid/craigbeast spawn spot, so keep these killed also :)

As far as farming Ikky: Get someone preferably a rog flagged for all 3 (anyone can do it, easier if the person can check the trials also). Get the Ikky 1 group trial. have rog/monk zone in (monks can check T7, anyone can check T8 really). Run in check for named. If it's up group moves in, kills, does loots, goes back to KT and keep checking the other trials for named.

The drops are:
T7 - melee only earring
T8 - Carved vrex totem
T9 - Melee only 46% haste bracer

And of course all of these can drop the glyphed prayer ring.

The other 'named' mobs in Ikky can drop spells but rarely and they also drop 35hp or 35 mana augs (might drop end augs haven't seen it though)

If you are feeling ambitious you can actually beat the trial for even more lewts but that takes quite a bit more effort than farming them, but really they aren't too bad. I've beaten them all w various sub 10K war's. Also i have healed a 9.5K Pal for the names in T7/8 with Xil and a boxed cleric as the only source of healing, though it was preslowed...
