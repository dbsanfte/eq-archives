To answer (or at least waffle about) the other question.

1 AA point costs roughly the same amount of XP as level 51. So if you set AA to 50% while you were in level 51, the 2 bars would move at the same rate.

But if you then levelled to 52, and left AA at 50%, because level 52 is "worth" more xp than 51, the AA bar would be moving faster than the normal xp bar.

Another thing to note is that AA xp is not affected by any racial penalties. So a troll starting out with a full yellow bar to go in 51, and no AA xp, and AA at 50%, would get the AA point before they levelled. Presumably a halfling would level before they got an AA point, but I'm not sure about that.
