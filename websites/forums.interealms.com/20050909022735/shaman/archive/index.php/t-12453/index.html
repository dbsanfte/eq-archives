I was having a similar problem yesterday.

Before this patch, when a new patch came out and I tried to start EQ, my firewall(zone alarm) would pop up a little window saying the program has changed. I just had to say ok to give eqgame.exe access to the internet.

Well, the patch that came out yesterday caused EQ to just lock up, so it seemed. What was really happening is that the little firewall confirmation box was popping up, but not coming into view above EQ like it always had before.

I reset the computer again. As soon as the first splash screen for EQ came up, I hit the alt +enter for window mode. That way when the firewall confirmation box popped up, I would be able to see it.

I hope this helps.
