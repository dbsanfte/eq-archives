In my case, you click my name in red, to see what my HP's and equip, etc are. If you want one of yourself, go to Magelo.com and register, creating a profile of your own. You then enter all your skills, atributes and equip.

BTW, here's the code that allows you to click through my name to show my profile, as always the []'s have been replaced with ?? so you can see what the code is.

#url=http://www.magelo.com/eq_view_profile.html?num=2935##color=red##b##size=3#FinalJustice#/size##/b##/color##/url#

If you are using Internet Explorer try this:

Click the "Tools" menu item, select "Internet Options"; Click on "Advanced" tab; scroll down to "HTTP 1.1 Settings"; make sure both "Use HTTP 1.1" and "Use HTTP 1.1 through proxy connections" are checked.
