No it is not possible. When you arrive to or from FV (or any other server except stormhammer) you lose all of your gear. This is done to prevent exactly what you are thinking of doing and thus preserves server economies.

There have been a few occasions when new servers were opened (like venril server for example) and for $50 you could transfer with your gear, but those were limited-time offers only and only a certain number were allowed to do so.
