To actually explain it........

Left number is the Ping......

Time in milliseconds that it takes a request to get from your system to the game server and back again.

next number is the Percent of loss... Each sent bit of information is called a packet.... this number shows you what percent of the packets you sent have been lost.

Next number is Percent of loss... Same as above but is the percent of inbound packets that were lost.

The last number is the Amount of information actually recieved in that time frame ..... 500 would be 500mbits

Hope that helps...
