Posted by Taradarr, on another thread:

When you do the gage for hps it automatically adds the name as part of the gauge but there is also a Label tag for {EQType}28{/EQType} that gives the target name. That label is what he uses and what gives mob_01.

The trick comes in at getting the name that shows up by default not to show. To do that you have to monkey with the guage properties.

What I did was set {font}0{/font} to set the font size to as small as possible. You then need to set the location to something like {x}3{/x} {y}-20{/y}. This puts the gauge starting to display outside of the bounds of the window. Then use{GaugeOffsetY}36{/GaugeOffsetY} to slide the gauge down back into the same spot. Don't forget to change the {size}{{Size}CX}408{/CX}{CY}44{/CY}{/Size}
y value to account for the fact that the gauge now starts 20 pixels off screen.

In short change the loc line so that it's -20 of it's current value, change the gauge ofset so it pushes it back down 20 and change the size so the guage is 20 bigger. You now have you target gauge with the name of the mob being displayed off window so just use a standard label with the eqtype 28 and boom you've got your numbers.

Obviously all the {} should be replaced with greater than and less thans int he xml file just didn't feel like taking the time to code it so it showed properly in the post =)

Taken from the rangers glade, posted by Braslo:

Basically, every character can have their own settings. And even if characters are using the same skin, there as still a lot of character-specific settings.

In the main EverQuest directory, there is now a UI_{name}_{number}.INI file for each of your characters. This file stores the character specific settings of:
- which skin this char prefers
- the position and size of each window this char prefers
- visible setting for each window this char prefers
- the fade time/percent for each window this char prefers
- (and i think it stores character specific tint colors for the windows, if you have tint on instead of background image)
