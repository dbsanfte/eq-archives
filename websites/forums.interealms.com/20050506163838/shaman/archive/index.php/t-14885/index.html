The only one I know of (there might be more) is using pause, for example when I'm lazy and don't feel like chain canni'ing I have this one set up:

/pause 35, /cast 8
/pause 35, /cast 8
/pause 35, /cast 8
/pause 35, /cast 8
/pause 35, /cast 8

this way, it casts 5 times in a row. Of course, it fizzles once in a while, but for the most part its fine.

for some reason, I believe a pause is the only additional command you can have on the same line (no /cast7, /cast8 for example), and the pause HAS to be first, even though logically, the pause occurs after you cast.

So, now that I think about it, there really isn't any other commands you can put on same line, just a pause command and another regular command.

I'm almost 100% on this one, I don't get too fancy with the macros, so if I'm wrong sorry to have misled you :)
