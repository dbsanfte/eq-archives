Tidykat
I run two accounts on this machine.
I have an AthlonXP 2800+ (Runs at 2.04 ghz), 1 gig of PC-2700 ram, and a GeForce FX5900 128mb vid card.

I run my desktop at 1280x960, and EQw runs 2 instances of EQ at 1152x864(roughly), so that I can see a smidgen of the other screen as well.

Another really helpful thing I found, is create 2 install directories. I have Sony/Everquest1/EQ files and Sony/Everquest2/EQ files. plus another EQw directory for each one.

In each main directory with the INI files, toss in a nosound.txt file to kill your sound on both accounts. As well, my main account (#1) runs with every option turned on, allmodels, mipmapping, etc, but has spell effects off. Account #2 has ALL models off, and most of the lighting/mip mapping stuff off (texture cache for both is on). Account 2 usually has pc names off as well as spell effects off.

While it does take a little bit to load the second account, once things are allloaded up, I can both through small raids and xp groups with no trouble whatsoever. In a full 50+ man raid, or even a 40+ man raid, I notice performance issues and it becomes hard to play. Also, my account 1 clip pane is 100%, account2 is usually around 50% or less.

just my 2cp.

Jaidin
You can run more than one instance of eq using eqw and eqplaynice on machines with pretty low specs. I can do 2 just fine on a 1.13 Thunderbird, GeForce3 and I think 768RAM. I even do it on my laptop, a p4-1.6 with 768ram.

The computer you spec'd should have no problems running 4 or 5 instances, just don't have all the options set to max. Bazaar is the only zone that is painful :( I have to set the clipping plane to minimum.

Visit these sites for more info.

www.eqwindows.com
www.lavishsoft.com

My normal setup is an Athlon 1800, 1gig ram, GeForce4 4800. I only have one install directory (i didn't notice any difference using two other than it being a pain to keep the character files synched for multiple accounts) but you will want to make the nosound changes. I run as many old models as I can (I think I have about 5 new models turned on), low details, 1600x1200 for usually 2 sometimes 3 EQW instances.

Make sure that if you use eqplaynice you set the background mode to framerate limited rather than cpu limited, otherwise autofollow won't work too welln and the background window will fall behind.
