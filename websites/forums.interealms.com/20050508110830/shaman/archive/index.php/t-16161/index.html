howler, there is a "Height" button on the interactive map. Click it so that it is not down. That will get rid of the elevation issue. Also, go to maps.eq-toolbox.com and download a whole set of new maps. They are helpful.

From stonebrunt you need to run south to the waters edge and then turn left and run along the cliff until you see a boat. Run onto the boat and click on the doors. That will zone you into Gunthark. Use levitation before you make the run from the book so you don't need to worry about running off the cliff.

Get a port to Stonebrunt, the run is kinda long from pok. Take the pok book to toxxulia forest, then run to Paineel, then the warrens, then stonebrunt. A druid port is often much quicker.
