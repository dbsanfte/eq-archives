Anywho, I have not yet bought my extra spell gem (Still havent figured out how to fix my UI for that)
Open up your EQUI_CastSpellWnd.xml with any text editor (notepad works fine, as do a ton of programming tools).

Edit: Until I figure out how to tell the board to show html code in the code tags I have to replace all html tags with "{" and "}".

It should contain a few entries looking like this:

{SpellGem item = "CSPW_Spell0"}
{ScreenID}CSPW_Spell0{/ScreenID}
{RelativePosition}true{/RelativePosition}
{Location}
{X}1{/X}
{Y}1{/Y}
{/Location}
{Size}
{CX}40{/CX}
{CY}32{/CY}
{/Size}
{SpellGemDrawTemplate}
{Holder}A_SpellGemHolder{/Holder}
{Background}A_SpellGemBackground{/Background}
{Highlight}A_SpellGemHighlight{/Highlight}
{/SpellGemDrawTemplate}
{SpellIconOffsetX}8{/SpellIconOffsetX}
{SpellIconOffsetY}4{/SpellIconOffsetY}
{/SpellGem}

These are the individual spell gems (numberered 0 through 7). Copy one of yours and paste it after the 8th spell gem (labeled 7). Change the number to 8 and change the "Location"-"Y"-number according to what your other gems look like. It should be easy to figure out if you have a look at how the numbers increase.

If you have a spellbook icon you will also need to look for {Button item = "CSPW_SpellBook"} and increase the "Location"-"Y"-number by the height of one of your spell gems (a little more if you don't want the icons to touch).

Also at the end of the file there should be something similar to this:

{Screen item = "CastSpellWnd"}
{RelativePosition}false{/RelativePosition}
{Location}
{X}0{/X}
{Y}25{/Y}
{/Location}
{Size}
{CX}397{/CX}
{CY}42{/CY}
{/Size}
{Text}Spells{/Text}
{Style_Transparent}false{/Style_Transparent}
{DrawTemplate}WDT_RoundedNoTitle{/DrawTemplate}
{Style_Titlebar}false{/Style_Titlebar}
{Style_Closebox}true{/Style_Closebox}
{Style_Minimizebox}false{/Style_Minimizebox}
{Style_Border}true{/Style_Border}
{Style_Sizable}false{/Style_Sizable}

{Pieces}CSPW_Spell0{/Pieces}
{Pieces}CSPW_Spell1{/Pieces}
{Pieces}CSPW_Spell2{/Pieces}
{Pieces}CSPW_Spell3{/Pieces}
{Pieces}CSPW_Spell4{/Pieces}
{Pieces}CSPW_Spell5{/Pieces}
{Pieces}CSPW_Spell6{/Pieces}
{Pieces}CSPW_Spell7{/Pieces}
{Pieces}CSPW_SpellBook{/Pieces}
{/Screen}

Look for the "Size" tag and then the "CY" tag. This is the height of the window. Increase it by however high your new spell gem is (and probably a couple of pixels more so the gems don't touch.

After {Pieces}CSPW_Spell7{/Pieces} add a new line {Pieces}CSPW_Spell8{/Pieces}

Finished.
