You can fire 2 insta-clickies at once. Just put both clickies in the hotkey bank. Say you put them in 1 & 2. Then go to Option, Keyboard (I think), Macros, and set the hotkeys to use the same button... same way you tie (for example) Sense Heading to a movement arrow key.
