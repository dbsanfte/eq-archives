First try typing /dopropertinting while in EQ. Do all the usual stuff (log out, zone, etc.. - test it out) and see if it works / helps or not.

If not, do you have luclin models on or off? If you have any new models on, or if you switch back and forth, its safest to turn all velious armor textures off - go into your eqclient.ini file and, somewhat near the top, you'll see lines: (this is a direct cut & paste from my file)

LoadArmor17=TRUE
LoadArmor18=TRUE
LoadArmor19=TRUE
LoadArmor20=TRUE
LoadArmor21=TRUE
LoadArmor22=TRUE
LoadArmor23=TRUE

- try setting all of these to FALSE, then restarting EQ

Mine are all set to true because I don't have much ram, and so I never have any of the new models turned on.
