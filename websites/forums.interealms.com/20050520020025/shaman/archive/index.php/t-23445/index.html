Win XP will run multiple instances of svchost.exe. The path displayed is legit, can't remember the name of the virus that runs svchost.exe from the winnt folder /shrug, but that specific bug isn't your issue. Yay, now only 5 billion other things it can be ;).

Basically, it can be a botch, or something malicious.

for malicious software, do what's been advised here already and visit the various web sites that check for viruses online. Norton has a decent one that also checks for security holes. Some malware will try to open your web browser. I personally keep a firewall, pop-up blocker, AV, and active spy ware nuker (I run it after every time I surf), and I keep it all mostly up to date.

For botch ups, it's likely you'll have to bring your machine in for service, or run the repair console (if you have the origional disks), or reinstall all the drivers.

There might be items in your startup, delete everything in the startup menu to the recycle bin, you can restore it later if you're still hosed. Beyond that there are items in your registry that will launch items, but I don't recommend you edit that unless you have some experience editing registries.

Check the task manager to see whats running. Kill processes, if it's important windows won't let you kill it, and see if the problem goes away. If it's svchost.exe, which it likely is, you shouldn't be able to kill it and your pretty much back to square one.
