Luclin's Arriana
That's a bad quote, because it is very very wrong.

AUGMENTS WILL ONLY FIT INTO THE SAME TYPE THEY LIST!!! You cannot fit a type 4 into a type 6. You cannot fit a Type 1 in a Type 3. You cannot fit a Type 3 in a Type 1. You cannot fit a Type 6 into a Type 5.

Type 1: Single stat improvements. HP+12, Wis+5, Dex+6, Resists Magic +6

Type 2: Multiple stat improvements, or large single stat. HP+40, Str/Sta/Wis+5, Resist_All+2

Type 3: Worn effects. Regeneration, Flowing Thought, Vengeance, Enduring Breath, Spell haste IV.

Type 4: Weapon procs...DD, Lifetap, Taunt increase

Type 5: Improve base weapon damage, typically elemental

Type 6: Improve base weapon damage, more. Often not elemental improvements, thus not putting in a resist issue.

Type 7: May not exist. Nothing has been found with this Type, either augment or slot.

Type 8: Exclusive to Time drops. No augments have been found that fits into this slot, but plenty of items have it.

There are augments will specifically list themselves to go in ANY slot. These are generally type 1 in their power level. These are the only augments (so far) which will fit into Type 8 slots on Time gear. I believe this is to make it so that already very strong Time equipment doesn't grow to outright ridiculous proportions. Most any-slot augments are not very good, or the ones that are good are lore and very difficult to get.

There's a lot of bad information going around about augments and about how adventure points are earned/spent. Please don't add to it by quoting grossly erroneous information.
