Hopefully this will answer some basics about signature links and pic links. Make sure that your pics meet the requirements set by the administrators. Also please make sure that the site you use to save your pictures allows for remote linking of pictures, bad example would be Tripod, all you get is their logo when trying to link a picture from there. You should be able to copy/paste the code; you might have to edit a bit of it. An example would be the number for your magelo profile and such. Here are several examples:


This is your basic link:
<a href="http://www.eqshaman.com" target="_blank">EQ Shaman</a>
Notice the --- target="_blank" --- This tells the browser to use a new window for your link. Should look like this: <a href="http://www.eqshaman.com" target="_blank">EQ Shaman</a>


Editing for Magelo would look like this:
<a href="http://www.magelo.com/eq_view_profile.html?num=1" target="_blank">PlayerName1</a>
Change the number to match that of your profile, along with your name. Should look like this: <a href="http://www.magelo.com/eq_view_profile.html?num=1" target="_blank">PlayerName1</a>


Images: Again I suggest you make sure that they meet the criteria posted by the admins

<img border="0" src="http://www.mywebspace.com/myfiles/image.jpg" width="250" height="100">

This is where remote linking come in; this will try to retrieve the file and display it. You can omit the --- border="0" --- statement if you wish. I personally kept the height and width in there to ensure that they meet the size specs. If you wanted people to see your profile when they click on it just replace the earlier "PlayerName1" with the last image code. Here is mine:
<a href="http://www.magelo.com/eq_view_profile.html?num=80612" target="_blank">
<img border="0" src="http://personal.atl.bellsouth.net/mco/b/o/bonetn/CrucibleSig1.JPG" width="250" height="100"></a>
Hope that helps some to those who may know very little about html. Pop-ups and non-resizable windows require javascript at least, that tends to get confusing pretty quick.
