You can single pull him at 51.

- Bind at the zoneline.
- Invis up and run to where the bridge going into Mistmoor is.
- Walk over to the edge where you can see him below you.
- Cast whatever you like and eat the 500-odd HT.
- Gate back.

He'll come striding up all alone. I did this twice in the same evening at level 51. Cake.
