The random number generator isn't always 'random'. For a better random number people will *seed* the roll by rolling for a random number between 0 and 1000. Then that number becomes the random number.

ie:

** a random number between 0 and 1000 MekuDuku rolls a 878.

ok now everyone in the group rolls vs 878.

** a random number between 0 and 878 MekuDuke rolls a 602.
** a random number between 0 and 878 Snapdragn rolls a 788.
