After upgrading my computer, I was crashing to the desktop, or having the video freeze about once an hour. This with a very reputable set of components and a clean install of WinXP Pro, SP1, critical updates, and fresh drivers from the manufacturer.

The problem was using on-board audio. Discussion with tech support at EQ revealed that these on-board audio devices are *not* officially supported. Switching to my old Soundblaster Live 5.1 (about $30 these days) seems to have completely resolved the problem.
