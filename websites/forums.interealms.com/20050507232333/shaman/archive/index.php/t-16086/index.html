>>No one really knows if cha helps

Charisma has been proven help bards in a few ways:

http://pub147.ezboard.com/ftheconcerthallfrm8.showMessage?topicID=77.topic

Notice a few posts down for log data (not zeitosx's first post he used the wrong song, see his 2nd post). Here was the summary:

charisma: 148 missed notes: 7.98%
charisma: 198 missed notes: 7.09%
charisma: 269 missed notes: 5.29%
charisma: 305 missed notes: 4.07%

So a cha buff reduces fizzle by 25.6% on a 198 cha bard!

Now bards cast a song every 3sec so a 2% more fizzles means you fizzle one more time every 2.5 minutes (basically every pull you fizzle an extra time).

Also charisma affects the chance of a mob aggroing a bard after lull. So it affects the number of times a bard would have to use Fading Memories when puller (low charisma could cause bard to spend 900mana).

IMO You should cast on every bard, or at worst ask if he wants it. Of course it doubly helps if the bard is your puller. Note that charisma helps high-end bards (near 200cha) more than it helps lower end bards (near 150cha. So it is alot more worthwhile at 65 on well geared bards.
