A RL friend suggested that I post my current UI settings, so here goes. If you want me to post the XML sheets I used let me know. I did not create any of the XML myself, just gathered them from various web sites.

My screen in set to a resolution of 1024x768 with a /viewport of 800x600. The additional screen space contains all of my main boxes. I get a full 800x600 screen with no clutter.
Entire Screen (http://www.drachan.com/images/ui/m.gif)

On the bottom right, you can see the Text Buff window, and the Main Chat Screen.
Bottom Right (http://www.drachan.com/images/ui/br.gif)

The Text Buffs display Blue for benificial buffs and Red for negative buffs.
Buffs (http://www.drachan.com/images/ui/buff.gif)
