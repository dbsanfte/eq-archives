Chifari
I bought crit aff 3 right off (since I solo decent amount). 9% looks to be about right, based on a tiny parse of 300 ticks or so (got 28 crits)... although, as has been pointed out, it doubles AFTER adding a BA type focus, so this works out to be more than 9% as compared to nukes (which double before applying focus).

If x is your BA% average (i.e. .13 for 25%, .155 for 30%, etc.), then:

Doubling then adding BA is 1.09 + x.

Doing BA first then doubling is 1.09 * (1+x). (25.895% effective damage increase for CritAff3 + 30% dot focus).

If you want to compare to nuke crits then, it isn't really a 9% benefit, but a little higher, more like 10.

(1+y) + x = 1.09 + 1.09x
y = 0.09(1+x)

So for 25% BA, it translates to 10.17% increase.
For 30% BA, it translates to 10.395% increase.
(in Nukeworld terms, if we were forced to double before applying BA)
