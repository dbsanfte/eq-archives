Actually, evertime you put in an autojoin line in EQ, you overwrite the old one. So everytime you write one you have to write in all previous channels you still want to join.

So if you first type /autojoin channel1

And then type /autojoin channel2

You will only autojoin channel2 since channel one is overwritten by the new input. So, to autojoin a bunch of channels you have to set them in the same line like this:

/autojoin channel1:password,channel2:password,channel3...

Or you could just edit your eqclient.ini in your Everquest directory.
