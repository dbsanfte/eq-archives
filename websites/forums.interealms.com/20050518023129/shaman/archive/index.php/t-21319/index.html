Ordee
For my Windows 2000 machine I deleted the Maps folder and let EQ re-install them all and then deleted all of the CJ_charname_Bertoxx.txt{Something similar to that} files for the character that was locking up. That file seems to be the storage spot for all text your character has ever recieved. I saw it was full of mob aggro messages and quest text from ages ago. Delete all the ones with the locked up character name and the TXT at the end and try it.

For Windows XP. Right click on your EQ shortcut and look under properties for a option to run it in Compatibility Mode. Set the Compatibility Mode to Windows 98/ME. That has kept me lockup free on the XP machine.

Hope this helps,
