Kianor
It's fairly straightforward, but needs some planning beforehand.

The event itself is based in Ferubi. Someone needs to have done some farming of nameds there to get the 4 trigger items (needs a strong group), and to have done the first stages of the Breakdown in Communications quest (needs some farming of bits all over GoD).

Quest spoilers: http://www.frmug.org/~archer/Breakdown.html
http://www.allakhazam.com/db/quest.html?quest=2781

Once someone has done the prep work, you can do the event. It will only give out 54 flags, but you can have as many people along as you want. CoH works in the zone, so you can even park alts at zone in and CoH them to get flagged when the flag mob is up. (Note, evac point in Ferubi is the Riwwi zone line, not the Qinimi one which you'll probably zone in through).

Oh, and there seems to be some bug with the event. My guild has done it twice and both times several people (myself included) hailed the flag mob, got the text, but didn't get the flag (there's no way of knowing this until you check by hailing the High Priest/Scribe).

I've heard that it's a bug with the z-axis, and you need to be on the same level as the mob, and not levitated, though I don't remember being levved myself, so not really sure if this is the case or not.

I never got a reply from the GMs on this (Euro timezone sucks for that), so I don't know if they can/will fix it. In the end I simply did the trial I was missing.
