Does prodgiy keep a log of emails on user side?

No. Almost every ISP keeps copies on the server side, but good luck getting them unless you have a court order. If you are using a mail client like Outlook express, then there may be pieces of you mail left about on the client (you) side.

Also, is email, when opened (prodigy or nonprodigy) stored on a computer or just on Prodigy's side?

Depends. Are you using an IMAP (i.e. web-based client), or something else, like Outlook Express? If it is IMAP, it is always stored on their side, if it is a client program like outlook, it is yours unless you've checked the option to leave it on the server.
