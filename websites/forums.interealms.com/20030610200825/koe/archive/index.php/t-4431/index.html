Just a couple heads ups-

I) On the topic of tweaking Luclin:

First thing you need to do is go into your Bios an see what your AGP appeture is set to. The memory value there should be as close to without exceeding half of the physical memory of your system. For my system (512 MEgs Ram) this means 256MEgs, as an example.

Second, you should go into the Windows System.ini file (found by typing SYSEDIT in the run line) and add this line to the section called "[386Enh]" :

ConservativeSwapfileUsage=1

Finally, you should alter your Virtual memory settings (My Computer Properties, farthest Tab then click on the button) so that the minimum and maximum Virtual Memory Size is exactly double the physical memory of your machine (1024MEgs in my example) then reboot.

I dont recomend doing this to lower end systems, but for anything 800Mhz and up with 256 or more memory, this will greatly improve your performance.

As far as system tweaks, one other I have found helpful is to go into the options screen just before login and select Texture compression from the screen just after the one where you select the models you would like to display. Also, I cranked texture quality down to low. This is supposed to degrade the quality of the graphics, but danged if I can tell the difference.


Also, I think everyone here at KoE has been made aware of the benefits of using memturbo. Might want to spread this one to the Corps folks as well. Memturbo (http://www.memturbo.com) scrubs your RAM meory and forces backgroup apps to release all memory not currently being used to the available pool. On my machine, I went from around 240 megs of available RAM at startup to just over 310 megs available. Just download and run the program, scrub your RAM before starting EQ, and set the application to check for available RAM every fifteen minutes.
