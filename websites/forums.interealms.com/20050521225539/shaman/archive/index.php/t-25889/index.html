I honestly think that anything but a tank and a healer will be limiting.
Options here might include paladin as both thank and healer.

The reason I think it might be hard is that so many encounters are multi-mob. I would guess that the pets will only be able to hold aggro on one mob.

There are plenty of duos that work very well indeed. But I think they will run into problems against the multi-mob encounters. Possibly if one of the classes is an illusionist?
