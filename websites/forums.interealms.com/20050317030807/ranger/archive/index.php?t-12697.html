Different abilities have different requirements. If you read the discriptions of each ability it tells you. Most of the new planar abilities require a certain level before you can purchase a point in it.

Several abilities you can buy more than once to improve it and to do so you usually need to be a higher level each time. 61 for the first point, 62 for the second, etc.
