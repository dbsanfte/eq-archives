To upgrade the eqw performance on many systems, create a txt file called "nosound.txt" it disables all sounds in game (obviously). It can cut zone times from crazy stuff like 10min down to around normal zone times. However, if you zone 2 at once it sometimes will kick 1 or take 1 an extreme amount of time to zone.

Bandicoot
I used to 2 box with 2 computers but now use 1 even more effectively.

Suggestions:
1) run EQW for both
2) use Windows XP (runs much better with EQW than 98 does... I upgraded recently)
3) use a completely different installation of EQ on another disc for the second account. You can just copy all the files across to the second disc.
4) put the nosound.txt file in the "secondary" account eq directory.
5) turn off all unessential models on the second account
6) install more RAM. It is cheap!

Do a search on the EQW forums for this stuff. It is well documented.

I just got a brand spankin' new machine and it is the best I could afford so I have been two boxing on one machine for a month or so now and HANDS DOWN the biggest asset to my two boxing is RAM count. I went from 512 MB of 133 MHZ SDRAM to 2 Gigs of pc-3200 400MHZ dual channel DDR. Biggest reason is that EQ was programed to devour system recources to hinder those how wanted to run programs in the back groud in order to gain an unfair advantage. Thus RAW RAM count and processor power make all of the difference (thats the only reason the sub corperate G-Force FX cards can out perform the ATI Raedon 9000 series in EQ ... and only EQ). Thus I would recomend going to a Gig of ram and if you are using SDRAM or any thing below 233 Mhz DDR, chainge the style of ram you are using al together.
Another HUGE asset to one machine 2 boxing is the use of the Logitech MX 300 mouse. I know it sounds crazy but hear me out. MX technology is one of the best laser eyes out there so it is a very accurate mouse (EXCEDINGLY important to me as an FPS fan). At any rate the MX 300 has a tiny little button inbetween the two primary buttons that is a program swaper. Click the button and it will bring up a menu, drag the cursor over the desired program and BAM! there it is. EQ does not really like this feature so it will not be friendly to the swaping menu but if you double click the swap button it will switch you to your other EQ program so be sure the EQ was the last 2 things you computer opened.
