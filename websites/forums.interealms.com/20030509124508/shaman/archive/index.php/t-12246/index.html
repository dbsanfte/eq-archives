Sorry, I haven't had this particular problem before, but some things you can try:
-Make sure you're going to www.magelo.com. Just magelo.com doesn't appear to resolve correctly.
-Make sure your security settings are not High (for IE, not sure what NN uses). Magelo appears to use scripting pretty extensively, and if scripting is blocked, the site does not appear.
