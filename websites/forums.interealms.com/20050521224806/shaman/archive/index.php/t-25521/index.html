Bigsnack For'Mobs
Can't really tell you for sure but can give you a couple things to try.

1. If you are camped in PoK try to make a run to another zone before you go LD. A lot of people have problems with PoK, especially logging on in that zone.

2. Turn your journal off. Use Alt-J to access. Journal records NPC emotes and seems to cause frame freezes a lot or worse.

3. Turn your log off. /log off. Also delete or rename and move your log file if it is very large.

4. Use sky off or set to simple. Turn down your graphics settings in Options.

5. If you are getting booted before you log in from character select screen (quickly booted not like normal LD boot) you will probably need to contact SoE tech. That happened to me last Monday. I visisted the chat room and it looked like it was happening to random people on all servers. It cleared up for me so I haven't worried about it.

Sounds like a eqclient.ini fault ....

Firstly, try going into the 'options' screen .... from that screen, change your screen resolution down one ..... sometimes, if you have played with screen resolutions in game, the eqclient.ini file can get screwed up ....

Secondly, if the first one didn't work, find eqclient.ini file in the everquest directory, and delete it .... then re-try to play EQ ....

Hopefully one of the above will work, good luck )
