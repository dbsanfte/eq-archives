"/loadskins custom 1"

I dont think you can have a space in the foldername.

1. Create a new folder in the uifiles directory and call it bert (or ernie)
2. Download one of the spellbook zip files (Arcane definately works)
3. Download the spell icons zip file
4. Unzip both of these files in your Bert folder
5. Run EQ and type '/loadskin bert'

They should then be there. Apparently some people have had to then reload EQ for it to show the new spell icons and stuff but I didnt.
