I see all these people saying how they got 1 gig RAM, etc, thinking it will make EQ run better.... I am sure it doesn't hurt, but if your whole system is high end, and recent equipment, you don't need that much just for EQ.

When I built my new comp last fall, I got an Athlon xp1600+, 64 meg GEForce2, and 256 megs RAM... planned on upgrading more RAM when SoL came out, but found out it runs flawlessly in raid situations with models on... I run in 1024x768 on a 21" monitor. If I get no video lag with 256 on a raid, why would I need 4 times that amount?

I would suggest getting a single 512 for now, and if for some reason that lags on a raid with the vid card/cpu you listed, then get more... but in no way should it.
