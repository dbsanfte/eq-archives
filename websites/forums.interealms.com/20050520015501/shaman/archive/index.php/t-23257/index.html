2. Radiant cure is different because it also can remove a set number of detrimental effects. The higher level RC, the higher % chance it will remove an effect and also the more effects it can possibly remove. This is why it can remove effects such as Malo which is neither poison, disease, or curse.

3. VP is extremely successful, so much so that if you use it with Malicious Decay it will last it's full duration 99% of the time. VP1 the root lasts about 1.6 minutes, VP2 1.8 minutes, and VP3 is 2.1ish minutes. At VP3, it will actually last longer than the recast time which is 2 minutes so you can keep a mob perma-rooted.
