I use a heavily modded version of Telgar's Icey Blue UI, patched and working with all of the latest updates. I've kept it simple to give the most viewable area but it conatins all the essentials and then some.

The screenshot is in 1280 x 1024 but it works just as well in 1024 x 768. Any smaller and you'd most likely have to delete a chat window or 2 but it should still fit.

http://www.speakeasy.org/~overshoe/Shaman_UI.bmp

You can download the Zip file HERE (http://www.speakeasy.org/~overshoe/Shaman%20UI/)

Installing it is easy. Unzip the files into a folder named ModA in your uifiles folder. Rename and the UI_Character_Number.ini file to your character name and server number and copy it to your main EQ folder. Load EQ and enjoy.
