runs an excelent website detailing how to tweak windows xp and 2K showing how to reconfigure the background services; which ones are necessary and which ones are not. Windows XP itself is a memory hog and the 256 meg of ram you have is what I would consider minimunm for just runing the operating system. EQ has become quite the memory hog also, and will use everything it can grab for itself. You might be able to get that system to run EQ with everything that can be disabled, particle effects turned off, basicly everything in eqclient.ini having to do with advanced modeling or rendering set to false, but you'd probably be stuck with playing in only the old zones from the original EQ. Honestly I believe you'd do better runing Windows 98 second edition on the machine. It would free up a lot of resources for the game to use. DirectX 8.1 still supports Win 98 SE and is sufficient to run EQ (for now). You're on the hairy edge of not being able to run EQ on that machine, all I can tell you is good luck geting it to work and come back and let us know how it went. Best of luck.
