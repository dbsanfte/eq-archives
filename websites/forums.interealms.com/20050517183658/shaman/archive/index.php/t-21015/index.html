Ancagalon
Hybrids in EQ are mixtures of two parent classes. To add in a third parent class, they would have to water down the core abilities of this class to the point of being non-effective.

The one hybrid that really has not been done is a warrior/wizard combo, which should be called a Warlock (if that name were not already used to describe necromancers in EQ). Paladin/SK melee offence/defence capabilities with a line up of single target nukes slightly below that of druids/mages.

The current knight classes are strong defensive, good utility and weak offensive characters. This knight class would be strong defensive, strong offensive with no utility. A crowd pleaser for the min/max players that are tired of playing their warriors and clerics.
