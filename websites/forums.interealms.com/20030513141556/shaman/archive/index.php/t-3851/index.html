Yes, this is also an easy change to make.

I rarely if ever use a second window so I disabled resizing.

Look in EQUI_ChatWindow.xml.

Go all the way to the bottom of the file and you will see a line

{Style_Sizable}false{/Style_Sizable}

Change it to

{Style_Sizable}true{/Style_Sizable}

where the { and } are < and > respectively.

use /loadskin <directoryname> to reload the xml and you will be set.
