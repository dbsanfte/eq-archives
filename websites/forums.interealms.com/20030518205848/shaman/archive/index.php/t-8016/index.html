Torl
~The Epic Begins~
( ) Receive a tiny gem from a lesser spirit
( ) Receive an opaque gem from a greater spirit

~The Test of Patience~
( ) Receive a broken arrow from Abe the Abandoned (no known use)
( ) Receive a small gem from a greater spirit

~The Test of Wisdom~
( ) Kill Tabien the Goodly and loot Marr's Promise
( ) Kill Glaron the Wicked and loot Woe and Envy
( ) Receive Shield of Falsehood from a wandering spirit (mine to keep)
( ) Receive a sparkling gem from a wandering spirit

~Test of Might~
( ) Kill Black Dire and loot his pelt
( ) Receive Black Fur Boots from Spirit Sentinel #1 (mine to keep)
......Spirit Sentinel #1 is in a ruin at +1250, +300 EJ.

~The City of Mist Reports~
( ) Personal Diary Page
( ) Crier's Scroll
( ) Merchant's Letter
( ) Written Annoucement
( ) Priest's Diary Page
( ) Student Log
......Reports are combined in a booklet and given to Spirit Sentinel #1.

~Lord Ghiosk's Books~
( ) Kill Lord Ghiosk and loot three books
......The books are given to Spirit Sentinel #2 underwater at +3685, -640 EJ.

~Battle in The Hole and the Queen's Diary~
( ) High Scale Icon
( ) Kill High Scale Kirn and loot Engraved Ring
( ) Kill Neh'Ashiir and loot her diary
......Neh'Ashiir must be given Engraved Ring to drop diary.

~The Plane of Fear~
( ) Kill Iksar Child and loot Child's Tear
......Diary must be given to Spirit Sentinel #2 before Child's Tear is given to Lord Rak'Ashiir.

~The Final Fight~
( ) Kill Lord Rak'Ashiir and loot Iksar Scale
( ) Give Iksar Scale to Sentinel Spirit #2 and receive Spear of Fate
