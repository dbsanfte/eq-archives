Grippa
And Verant said from the start, DoT stacking only affects DoTs cast by different PCs.

Think of it like this:

When you cast a DoT, the mob gets an effect called "Envenomed Bolt" or whatever. Any subsequent "Envenomed Bolt" will overwrite the one currently on the mob.

In the new scheme, when I cast a DoT the mob gets an effect called "Evenomed Bolt - Grippa". If I cast another DoT, it has the same name, so overwrites the original. But "Evenomed Bolt - Twoflower" is a different spell (for these purposes) and thus can be on the mob at the same time.
