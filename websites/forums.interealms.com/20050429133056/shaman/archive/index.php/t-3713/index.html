MagnoMightyMidget
/viewport lets you resize the display area of the main viewscreen so that you can create an emtpy border around the viewing area. You can then push your chat boxes outside the main viewing area so that they don't block your view of the world.

To use /viewport just type "/viewport (x) (y)" where (x) is the number of horizontal pixels you want to be displayed and (y) is the number of vertical pixels you want to be displayed.

So for instance say your in 1024x768 mode and you want to shrink the main display so that it leaves an empty spot at the bottom of the screen where you can put your chat box. You could type /viewport 1024 500. That would give you 268 vertical pixels of empty space to place your chat box in.

This really isn't the fix that people who are missing /hitsmode would like but it's the only way I've found so far to get around the issue. I would LOVE to hear any other ideas or even better if anyone has found a way to get /hitsmode to work with the new UI.
