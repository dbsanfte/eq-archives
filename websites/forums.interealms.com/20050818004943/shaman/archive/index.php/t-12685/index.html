Magelo appears to love to cache itself (serve you an old version of the page along with any problems you once had), not sure why.

Heres what fixes problems for most people using IE I've found:

Garbled text
Go to Internet Options > Advanced
Under HTTP 1.1 settings ensure 'Use HTTP 1.1' is checked.

Page/profile doesnt update on refresh
Internet Options > General. Temporary Internet Files > Settings. Check 'Every visit to the page. Alternatively click the cursor on the area of the page thats being problematic, right click and choose refresh from the fly out menu, just hitting refresh on the browser toolbar seems to just refresh the frame and not the frame content.

If those don't fly then check you've got exactly the same version browser on both PCs. Help > About Internet Explorer will reveal which version.
