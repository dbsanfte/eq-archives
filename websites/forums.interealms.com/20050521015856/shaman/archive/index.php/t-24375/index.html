Barm McLir
Those enchanter pet weapons are no-rent.

Pets dot not equip no-drop items. You can hand them to a pet, but it poofs.

Also, pets ignore the race and class restrictions of a weapon, but obey the slot restrictions, i.e. handing it a secondary slot weapon will make it dual wield. They also obey the level restrictions on procs.

I believe there is special coding in the game to prevent pets from using a Weighted Axe. Probably a few more items like that.
