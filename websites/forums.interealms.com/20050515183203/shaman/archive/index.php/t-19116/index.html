Grafsha McDaughts
To generalize, every other class has AA's that enhance and improve their core class design. Wizards get faster casting, less mana, improved damage AA's to support nukage. SK's get improved lifetap/ht AA's. Go down the list the and each class gets AA's that improve what they are as a class.

Shaman get borderline improvements to our class specific abilities. More dogs (taken from mages), improved canni (no complaint here) ... the ability to bump our alchemy (/boggle.)

Shaman do not get an enhancement to our core class: slowing. If there is any change of benefit to the shaman class this is the area. AA's that reduce the casting time/refresh rate. AA's that reduce the resist rates of slows. AA's that decrease slow mitigation by 2, 5, 10%. None of these are unbalancing ... no more so than unresistable HT's or Manaburn or the other class AA's that actually focus on the class core ability.

There - thought I would toss that out into the mix of shaman fixes flooding these pages of late.
