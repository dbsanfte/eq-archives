Short answer: yes

Long answer: haste stacks as follows

Best Item Haste + Best Spell Haste + 1 or more stacking Bard songs will stack. Most bard songs don't.

Basically if an item has an effect of "Haste (Worn)", it will stack with spell haste. If it has a buff icon, it is 99% likely to be simple spell haste. There are a few bard songs that stack on top of this can exceed the usual haste cap, if you really want the gory details of bard haste click here (http://pub142.ezboard.com/feqdivabardsofeverquestfrm58.showMessage?topicID=905.topic)
