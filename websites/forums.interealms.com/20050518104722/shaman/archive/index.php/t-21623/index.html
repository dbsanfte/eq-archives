Thanks for the /whoall command!

/em newspaper
Your character opens up a newspaper (though it's the same one every day *grin*). You can also /em any of the commands off that emote button on your chat bar. /em yoga /em cheer etc.

f = follow, or stick, as per DAoC. Indispensible for a scrapper or tanker, as the nukers really really want to you pick the next target quickly. So when the old one dies, tab to the new target, f to stick to him, and start smacking attack power keys.

/bind_load is a command I use every time I make a new character. /bind_save creates a file with your current keymap assignments in the root of your C: drive. (full info here http://coh.warcry.com/scripts/faq/faq.phtml?category=744&site=36&id=153#5562 ) Once the file is created, you can change a lot of key bindings that don't come up in the interface.

I'm not sure how much of the other information in that guide is current, or if it's leftover from beta, but the key binding and saving worked nicely.
