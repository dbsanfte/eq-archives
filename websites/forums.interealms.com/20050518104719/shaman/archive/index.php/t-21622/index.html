Thorzion
Being the only person on the server. Or any server for that matter.

Happened to me this morning. Read the patch message here, and decided that I really didnt want to lose the ability to use EQW (new login screws with it) so, I didnt patch.

But I was still able to log in. And when I logged in, I was the only person on the server. It was just after 9AM, so I figured that I must be one of the lucky few to get in early and beat the log in server jam. So went to the bank and unloaded some crap, and did a few tradeskills while I waited for a few more people to show up in the hopes of getting an adventure.

But after 45 minutes and no one else showing up, I began to get curious. I mean, its all well and good to be able to run around PoK without 200 mindless /ooc's running through my chat window but to be utterly and totally alone on the server gets old quick.

So I logged out and tried another server that I have some characters on. And I was alone there to. All told I tried 21 servers, and every time I was the only person there. Was genuinely an odd experience.

But I decided to make the best of it. I ran over to droga and hit the king (hello CHoS) and a few other named that I could find up (not great drops, but good tribute stuff), but even that got boring. I had now been logged in for about 2 hours, and was really starting to get bored.

Just as I was tossing the stuff in the bank and getting ready to camp, I got a tell from GM Yinpus. He(she?) asked if I could log out, because the server was not availible for play. When I asked how I was able to get in if it wasnt supposed to be availible, GMYinpus said that I just must have been lucky. I told him(her) that I had not patched and (s)he said that could have been it, but I still should not have gotten past the log in screen, as the servers shouldnt have been allowing anyone on yet.

An odd experience, to be sure. I did manage to make some plat from it via drops. Im wondering if they will still be there when I log in next. /shrug With all the rollbacks my server has had in the last week or so (6 that I can remember) Im not holding my breath, but time will tell.

Valsuvious
You can bypass the patch server by clicking on the eqgame.exe file instead of the everquest.exe file. It will send you straight to the login server instead of the patch server. This is how people like me have been still able to use EQW this past week. There is also a way to redo the two files that were changed with the advent of the the new login page but that is explained in another thread here.
Makelike
Change the targeted file on your EQ shortcut from "C:\Blah\BlahEverquest.exe" to "C:\Blah\Blah\eqgame.exe" patchme. You can do this simply by right clicking on the shortcut and choosing properties. Works like a charm and cuts out on the sitting around waiting for the patch server to respond telling your system it doesnt have any new files.
