kapanen
There is a mandatory 5% rate of failure on every item, regardless of your skill. So you could make those easy arrows at skill 250 and still fail about 1 in 20. Every once in awhile you'll get a long batch of successes, but the failures catch up eventually.

The formula I have seen is:

Chance = (YourSkill * 1.15) - (Trivial - 70)

When I want to know what skill I need to have the best chance for a given trivial, I use this:

YourSkill = (95 + (Trivial - 70)) / 1.15

I used the 95 to reflect the 95% chance, but maybe it should be 100. It's Mystery Math, but it gets pretty close. I add a few skill points to make sure anyway.
