I use "Email notifcation by default" to send me email for threads I've posted on here.

I'm not sure this is what you're after, but it does seem that you can subscribe to a thread without it emailing you. Then you could view the list in your User Control Panel to see which ones have been replied to recently. I can't see a way to set this by default though - I'm not sure if simply replying on a thread is enough to make you subscribe if you don't have the "Email notifcation by default" option ticked. Maybe you could experiment with this.

I will say that email notification seems to work pretty well. You only get one email per thread, until you visit the board again, so you don't get like 50 emails if it's a very busy thread and you're away for a couple of days.

I also have hotmail, and sometimes have MSN up, which gives me a notification when I get email.

I hope this helps.
