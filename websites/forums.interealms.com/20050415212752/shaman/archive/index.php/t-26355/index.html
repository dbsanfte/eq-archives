The way merged server character slots worked with the Zek server merge was that you had 8 slots available as with any server - if you had 5 characters on Server A, and 5 characters on Server B, all characters would be combined, but only 8 would be visable - they woulf be listed alphabetically so if say your main character happened to have a name beginning with "Z", and the first 9 characters began with lesser letters, your main would not be visable until you delete at least three of the other characters to get the "Z" character into that 8th slot.

As for the moves after the merge - it is an "all or nothing" move - all your characters from the "new" merged server are allowed to move to any server of your choice (excluding the indentified "special rules" servers), but they all move as one - you cannot specify induvidual moves/stays for your characters. If you move to a server you already have characters on you would have the same 8 slot limitation and have the same options listed above to get down to the 8 visible characters.

Some key things that can't be said enough - make sure you have nothing in any of your shared bank slots on either server being combined, this includes shared cash. If you have items on both servers in shared slots, only one will get moved and the other will be lost so it's just best to have them empty. Make sure you have no corpses left prior to your merge date - if you have a corpse with gear and your server is merged it will be lost. Make sure you login characters 10th level and below soon before the merger or they will get purged with all there gear and any cash they have in their bank.
