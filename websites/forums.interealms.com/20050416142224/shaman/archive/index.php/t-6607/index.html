Some options to help block the pop up messenger messages:

1 - Disable messenger in windows - but this could cause printing problems for you particularly if your using a network based printer

2 - If your using an internet router to allow you to run multiple systems off of a singe cable/dsl connection you can also go into your router setup and block the following ports:

TCP 135 - 139, 5000.
UDP 135 - 139.

Not only will that fix your problem with messenger but it will also close a lot of other potential problems, unless of course you were deliberately sharing your machine's files over the internet.

3 - Try add on packages such as Zone Alarm(pc based firewall) and Ad-Aware(scans your system and removes unwanted add type applications installed with 3rd party software)

If your using Kazaa - download and install Kazaa-lite (www.kazaalite.com) instead...the full version of Kazaa has add based spyware built in to it - but you should also get the Ad-Aware packeage when you do it - just uninstalling Kazaa and installing Kazaa-Lite will NOT get rid of the already installed add spyware.

Good luck!
