Sudedor
Tanking ability of all non-tank classes decreases dramatically once the sub-class level is hit (L20).

Basically, the way the game works is that as each character grows, they become more specialized. Before L10, *anyone* can tank effectively. Between 11 and 20 you start to see the Fighter archtype pull ahead by a little, but some others can still do it in the right circumstances. Post 20, the Fighter tree, and particularly the Warrior and Crusader branches, really start to pull away.
