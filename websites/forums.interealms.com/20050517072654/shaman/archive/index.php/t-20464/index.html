Tigran
That's not accurate though. There's no program that can keep EQ from taking up memory. It will still take up a lot of memory whether you're running EQPlayNice or not. EQPlayNice forces EQ to better share the CPU time, and is most noticable on single-processor systems without hyperthreading.

If you want to reduce the memory EQ uses, turn off Luclin models and enable compression.
