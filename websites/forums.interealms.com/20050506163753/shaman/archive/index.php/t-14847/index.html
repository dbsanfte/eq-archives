Gilador
You might also try altering Windows XP performance options to allow more CPU time to background services:

- Right-click on 'My Computer' and choose 'Properties (or open 'System' control panel if you do not have 'My Computer' on your desktop)
- Go in the 'Advanced' tab
- Press the 'Settings' button in the 'Performance' frame
- Choose the 'Advanced' tab
- Select 'Background services' instead of 'Programs' in the 'Processor scheduling' frame.

This trick has been very helpful for people running EQ Companion along with EQ on the same computer so it might help with 2 EQ's as well.
