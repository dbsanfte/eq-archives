Welcome to the boards. That is an interesting idea. I think it would be too much of a change to mechanics. Still, I hate people running around for buffs before they can hunt.

I like the idea of getting buffed by whoever is actually there. If that were the case, perhaps you could refine the buffs so that all intelligence casters had essentially equal mana regen buffers that overwrote each other, all priest classes had essentially equal hit point buffs that overwrote each others, all melee dps had essentially equal attack and haste buffs which overwrite each others, or could be dualed (since there are 6 melee dps classes), and all plate tanks had a/c and shielding buffs which were essentially the same and could not stack. Thus, each group would want at least one of each type of class and two melee dps. Each "buff" supports that class's essential functional class.

Furthermore you could have raid buffs where each class can supply one of perhaps three buffs for raid, making the attendance of every single in game class be helpful on every single raid. Nothing you could not do without, but something where you would say, cool we have a berserker raid buff, I am glad she came tonight. Thus with three of each class you would have the maximum 39 raid buffs.
