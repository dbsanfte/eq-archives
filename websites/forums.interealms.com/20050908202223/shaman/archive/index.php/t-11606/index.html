First and foremost, let me admit that a vast majority of this UI was not made by me :P I've done a few simple changes myself, but for the most part I've just cut the little bits from different UI's that I liked.

The inventory slots in the hotkeys were specifically adapted for shaman and what I personally had for my shaman, but they can be modified if you chose. In the inventory XML file you can find the appropriate "EQType" number that corresponds with each inventory slot. From there, just open up the hotbuttons file, scroll down to the portion with the inventory slots, and replace the number in <EQtype>#</EQtype> with whatever slot you'd like.
