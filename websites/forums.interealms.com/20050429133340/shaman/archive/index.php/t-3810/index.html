Gloriana
Well, that depends. You up to manipulating the .xml files yourself, or do you just want to download other people's that you like?

If you just want to install what others have created, I can help with how to do that:

Go to your EQ folder and look in the folder called Uifiles. Inside that folder, is a folder called "default." Open it.

On the second column of files, there are a bunch of xml files that start with EQUI. If you look at them, you can see that each one is responsible for part of our user interface.

EQUI_groupwindow - this is our group window
EQUI_inventory - inventory
EQUI_spellbookwndow - spellbook

etc.

In order to make changes, in the Uifiles folder, create a new folder (inside the Uifiles folder, not inside default) and name it whatever you like (I called mine "skins"). Copy everything from the default folder into it. When you are modifying your interface, you can replace each file singly, or download a complete mod that somone has created.

Here is an interesting layout from the Ranger's Glade.

http://forums.interealms.com/ranger/showthread.php?threadid=6582

You can see he has a link to download something called "Sean.zip" Go ahead and save this file to your desktop.

Look at the picture of Sean's UI, is there anything you like about it? I liked his inventory window, so I went to his folder saved to my desktop and copied EQUI_inventory. I pasted that file into my new folder (in the Uifiles folder) that I created called Skins, and it replaced the previous file. Repeat with every file of Sean's that you like.

For my spellbook, I got it from this site:

http://newui.technosapien.net/misc.htm

(this is a great site to browse around and see what items you like).

I went to "Compact Spellbook" and downloaded it. Once I had the item on my desktop, I opened it and did copied EQUI_spellbookwnd and pasted it into my my "skins" folder.

Once you find all the xml files that you like, go ahead and start the game. Once you get your character loaded, type /loadskin (name of your new folder that contains your new UI folders), my command was /loadskin skins. All of the new stuff will pop in, and then you have to take a few minutes to "place" everything where you want it on the screen. If you also want the new UI windows on other characters, you must log them in and type /loadskin (folder name) and also position the new windows.

The xml is all plain text and mostly obvious. Style_VScroll. Well lets break that down. Ok style is how something looks so this is probably gonna control how the window looks.

V? who knows right now.

Ok scroll, hmm only scroll i can think of normally on a window would be the scroll bars.

Hmm that would mean V would probably be vertical.

So this means that you don't show a vertical scroll bar on the window.


<Style_Titlebar>true</Style_Titlebar>
<Style_Closebox>true</Style_Closebox>
<Size>
<CX>104</CX>
<CY>395</CY>
</Size>


Following that same pattern the first line says that the window has a title bar.

The second line says that it has the X to close the window in the title bar.

The rest says that it's 104 pixels wide and 395 pixels tall.

In most cases it's that simple. If you can't figure out what something is, try changing it or looking for other spots it's used. There is also one document in there that kinda defines everything and tells you what attributes things have and what the defaults are.
