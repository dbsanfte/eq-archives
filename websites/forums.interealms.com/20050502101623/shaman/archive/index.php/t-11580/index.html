While I have not parsed it eitehr I take the mitigation msgs to mean the following:

Immune to Changes in attack speed: 0 % effective slow => 0% slow

Slightly: 25 % effective of normal slow => 0.25 x 0.75 = 18.75 % slow

Partially: 50% effictive of normal slow => 0.5 x 0.75 = 37.5 % slow

Mostly: 75% effective of normal slow => 0.75 x 0.75 = 56.25% slow

Full: 100% effective slow => 75% slow

This is using Turgurs at lv 65 (75%) in this example.

While I cannot be sure of this but it seems to follow the logic that was posted by VI about slow mitigation that the best slows would still be the best (Ie Turgurs) since it is a percentage of the normal effect. Id love to see some parses to see if it is in fact what I have described, but Mob dmg varies very greatly in excess of 25% so it isnt really possible to make conclusions this precise with that kinda of variation in the parses.
