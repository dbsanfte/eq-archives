I used to have this problem. I used to have a dial up modem, and have since upgraded to DSL. SOE Tech Support took six tries to fix the problem, after I had updated my drivers and such. If you are running Windows XP (that's what I run, and being a non-software expert I don't know if other Windows versions support this or have this) then you have something called a QOS Packet Scheduler. This program apparently dictates the size and timing between packets of data received and processed by your modem and CPU. Turn this feature off -- I believe you can access it through Network Connections in the Control Panel. Once I did that, my modem's LD problems were solved. You might also take a look at the datarate.txt file in your EQ Directory. Change the number from 5 to 3, and save it.

Hope this helps!
