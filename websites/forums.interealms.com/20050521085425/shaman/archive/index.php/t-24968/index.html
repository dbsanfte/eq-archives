The item links thread that is back on the front page reminded me of this and thought I'd share.

You can get yourself a unique last name with a little ini editing also.

My current name on tunare looks like this in game:
"Monjay Ж"

Using the same method described in the item-links thread, edit one of your hotkeys with the /surname command, then use the Character Map program (comes with windows) to select some untypeable characters to put in your last name.

my hotkey simply looked like this:

/surname Ж

Since I petitioned to have my surname removed a long time ago I had no last name and just pressed my hotkey button to set my new one.

If you do this, be prepared to get bombarded with tells about your new last name. I've had mine for nearly a year now and still get constant tells about it.

have fun.
