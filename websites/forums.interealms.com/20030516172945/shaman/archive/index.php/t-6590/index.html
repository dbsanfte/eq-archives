FROM the other post - this fixed it for me and others:

Was doing the same thing to me - trying to download a file when goingto the main page - or displaying giberish when going to a magelo profile display - this is what fixed it for me:

*** MAKE SURE YOU HAVE THE LATEST PATCH FOR IE6 ***

1 - Cleared Cache and Cookies.

2 - Go into Tools -> Internet Options -> Security -> Custom Level -> and confirm it is set for MEDIUM level of security.

3 - Go into Tools -> Internet Options -> Advanced -> scroll down and look for HTTP 1.1 Settings -> Click on and make sure they are selected for: Use HTTP 1.1 AND use HTTP 1.1 through proxy connections

Good luck!!!
