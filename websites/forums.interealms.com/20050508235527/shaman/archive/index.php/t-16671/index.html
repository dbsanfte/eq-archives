I would go for Run1, Regen3, then spend 2 on STA or something else. Reason being that Run1 is enough to outrun MOST mobs... and how often do you really run from the mobs anyway? Add SOW and Shrew to the mix and the odds that Run2/Run3 will actually be useful to you are low.

Spending any aa's on stats at that point is a mistake in IMO especially now. Stats can all be maxed out with gear especially with augments now. Need more wisdom throw some +6 wis augments in a couple peices and add more wis. Need more sta? throw some sta in other peices. Once you can cap stats from gear those points spent on innate stats becomes worthless.

There are really only 3 things worth spending any aa points on in general initially. Run 3, regen 3 and metabolism 3. I'd probably recommend run 3 and regen 3 especially if the poster doesn't have stoicism yet.
