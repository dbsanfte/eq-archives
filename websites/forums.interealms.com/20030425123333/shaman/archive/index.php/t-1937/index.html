Dandi
According to this thread (http://pub126.ezboard.com/feqtraderscornerfrm13.showMessage?topicID=1242.topic) on the EQ Traders forum, there is a possible formula for predicting the average rate of successes:

Success rate = (1.15*Skill) - (Trivial - 70)

So to use the current example, a gate potion at a skill of 156, you could expect an average success rate of (1.15*156)-(170-70), or 79%.

There is still a lot of speculation regarding this, but so far I have yet to see anyone produce results from a sufficiently large set of data to refute it.
