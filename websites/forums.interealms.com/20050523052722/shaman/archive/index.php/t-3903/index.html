I probably shouldnt even post this ....

But im trusting the Shaman community not to abuse this information somehow.

If you are still using the old UI this type of thing just shows up in HTML code

But with the new UI it shows up as a color message

Do whats typed below and you can change the color in your guild or chat channel.

Expect this to be changed soon if folks keep impersonating GM's and the like with it.


To make your message in chat or whatever channel appear like this...

Type this in your channel:


<html><body><font color="white"> Your message will go here </fontcolor></body></html>

(took me a long time to figure out how to post this so it wouldnt just say Your message here in white)

To change color just substitute another color where it is "white" like "purple" or "blue" or whatever you fancy.


<html><body><font color="green">A generic text message in green or </fontcolor></body></html>

<html><body><font color="white"> A white and very hard to read text message </fontcolor></body></html>

<html><body><font color="red"> Some red text on the screen message</fontcolor></body></html>

<html><body><font color="blue"> Some blue message message</fontcolor></body></html>

<html><body><font color="yellow"> A message in yellow text on the screen, i'm sure you get the idea </fontcolor></body></html>

<html><body><font color="black"> Your message here in black or whatever </fontcolor></body></html>
