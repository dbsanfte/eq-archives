Chazic
If you are having trouble finding that just right color there is one trick to get the color close to what you’re looking for.

And that hint is ...
1. Open up Paint. (Or try MS Paint but my very slightly)
2. Select Edit Colors
3. Select Define Custom Colors (Button Near the bottom)
4. Play with the different Red, Green, Blue (RGB) settings till you get near the color you like.
5. Record the numbers for Each RGB setting.
6. Open /body menu in EQ.
7. Insert the respective color in the corresponding color.
8. Preview your characters new look.
9. When you get the color you like write down the RGB setting for reference.
10. Select Save when all colors are the way you want them.

This enables you to use your mouse to scroll around to find those colors you want. If you select one of the Basic Colors you can copy down the Red/Green/Blue RGB values to duplicate them in EQ. Just remember some colors appear darker/lighter depending on what color is next to it so slight modifications might need to be made. (Jeez too many art classes for me)

**Note: You do not have to have LoY to play with the custom colors. You do NEED to have Loy to SAVE those settings though.
