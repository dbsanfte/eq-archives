Got PoP? If so :

Fine Antique Ring (AC 3, HP 30, Mana 20, Mana Preservation II, No Drop, Ring). Fairly easy quest. (http://everquest.allakhazam.com/db/quest.html?quest=2193)

Fine Cut, Diamond Inlaid Mask (AC 3, Int 3, Wis 3, Sta 3, MR 3, HP 10, Mana 10, Spell Haste II, No Drop, Face) Another easy quest (http://everquest.allakhazam.com/db/quest.html?quest=2203). Need invis to do it, but one or 2 vendor purchased Invis potions should be sufficient if you run the quest below that level. Only component to turn in is Sarnak Blood - dirt cheap to aquire.

Beginner's Magic Manual (AC 7, Int 6, Wis 6, HP 15, Mana 15, No Drop, Secondary/Range, Rec Level 45 but usable well below that)Yet another easy quest (http://everquest.allakhazam.com/db/quest.html?quest=2218). Only component needed is purchasable for 100pp or so in bazaar.
