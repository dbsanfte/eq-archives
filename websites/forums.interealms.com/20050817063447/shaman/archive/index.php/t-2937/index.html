Jaradis
Magelo will soon support item searches with the new version coming out soon. It already works but is currently only accessable by mods/admins. It has the following search options:


Race
Class
Deity
Slot
Category (1H Slash 2H Slash Piercing Armor Shield Brass instrument Percussion instrument Potion Poison Lockpick Jewelry Spell Reagent, etc)
Modifier 1 (Str, Sta, HP, etc)
Modifier 2 (Str, Sta, HP, etc)
Modifier 3 (Str, Sta, HP, etc)
Text (User entered text)

You can easily pull up a list of all items that go in the Arm slot that have Wis, HP, Sta, used by a Barbarian Shaman, and sorted in order of your choosing.


As for having zone/mob info linked to the Item in Magelo, it isn't possible for many reasons. One being there is no way to guarantee the info to be correct, items can be handed to charmed pets and then break charm and kill them. When you loot them they would have the items you handed them. So its very possible someone could intentionally hand a very nice droppable item such as a Mithril Helm to a charmed Kobald in Sol B, then kill the Kobald, loot the Helm back and then Magelo would show it dropping in Sol B off a Kobald. There are also many other situations that would cause data corruption.

As for entering that info in by hand, thats something that isn't going to be done on Magelo. There are far too many items in the database (19282 as of this post) and as you can see on many other sites, having humans entering info just leads to wrong info being put in on many items.
