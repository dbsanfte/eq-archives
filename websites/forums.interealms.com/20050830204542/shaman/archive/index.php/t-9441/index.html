Rathelon
Basically, in your Everquest directory is a folder called UI files. In that folder is a folder called 'Default'. That contains all the files for the new interface.

If you want to make a custom interface, copy all the files in the default folder into a new folder and rename it. Then you can download (or create) custom files and you put them in the new folder you created. Once finished, just type /loadskin 'xxxx' while in the game ('xxxx' being the name you gave the new folder).
Valgoth Ulv'Sagart
You don't need to copy any of the files from the Default directory to your new UI directory. In fact, this will cause more problems than it was intended to fix. The EQ client is now a little "smarter" in that it will now load any files it doesn't find in your custom UI folder from the default folder.

1) Find a UI you like.
2) create a new folder (name it whatever you want) under the UIFILES folder in your EverQuest directory.
3) Unzip all of the files in the new UI into that directory.
4) Start EQ
5) Once in game type "/loadskin <directoryname> true"

Most complete UI's come with a "UI_charactername_##.ini" file that you should rename to your characters name/server. This is the file that contains all of the windows/UI parts x,y coordinates. Otherwise you'll be manually positioning them which can be a real pain in the butt.

http://www.eqinterface.com is the best source for custom EQUI mods I've found.

G'luck!
