Fixed it, files available for download again :)

If you want to change the item slots right of the hotbuttons you have to open EQUI_HotButtonWnd with any text editing programm. Then if you want to change for example the neck slot to legs you search for

<InvSlot item="Newslot5">
<ScreenID>Newslot5</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>90</X>
<Y>156</Y>
</Location>
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
<Background>A_MiniInvNeck</Background>
<EQType>5</EQType>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
</InvSlot>

and change the bold text to

<InvSlot item="Newslot5">
<ScreenID>Newslot5</ScreenID>
<RelativePosition>true</RelativePosition>
<Location>
<X>90</X>
<Y>156</Y>
</Location>
<Size>
<CX>32</CX>
<CY>32</CY>
</Size>
<Background>A_MiniInvLegs</Background>
<EQType>18</EQType>
<Style_VScroll>false</Style_VScroll>
<Style_HScroll>false</Style_HScroll>
<Style_Transparent>false</Style_Transparent>
</InvSlot>

The tags and numbers can be found in the added readme.
