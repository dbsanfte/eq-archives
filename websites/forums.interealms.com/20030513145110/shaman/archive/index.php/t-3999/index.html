For the clickables window:

Open the EQUI_PlayerWindow.xml in your custom directory (where my UI files were placed) with notepad.

In the file EQUI_PlayerWindow.xml you will see each of the inventory spots I put in the Clickables box.

To change which one is which all you have to do is:

1. Find the slot you want to change
2. Look for a reference called EQ Type>#<

Change the number in ## to one of the below... depending on which slot you want it to be:

Inventory Slots
1 = Left Ear
2 = Head
3 = Face
4 = Right Ear
5 = Neck
6 = Shoulders
7 = Arms
8 = Back
9 = Left Wrist
10 = Right Wrist
11 = Range
12 = Hands
13 = Primary
14 = Secondary
15 = Left Fingers
16 = Right Fingers
17 = Chest
18 = Legs
19 = Feet
20 = Waist
21 = Ammo
22 = General Inventory Slot 1
23 = General Inventory Slot 2
24 = General Inventory Slot 3
25 = General Inventory Slot 4
26 = General Inventory Slot 5
27 = General Inventory Slot 6
28 = General Inventory Slot 7
29 = General Inventory Slot 8

The background image will have to be changed also, but I don;t have a list of the references. You can find the one you need for each slot by looking in EQUI_Inventory.xml. Look for a reference to background in that file.

taraddar
OK I'll try to explain this as best I can.

When you do the gage for hps it automatically adds the name as part of the gauge but there is also a Label tag for {EQType}28{/EQType} that gives the target name. That label is what he uses and what gives mob_01.

The trick comes in at getting the name that shows up by default not to show. To do that you have to monkey with the guage properties.

What I did was set {font}0{/font} to set the font size to as small as possible. You then need to set the location to something like {x}3{/x} {y}-20{/y}. This puts the gauge starting to display outside of the bounds of the window. Then use{GaugeOffsetY}36{/GaugeOffsetY} to slide the gauge down back into the same spot. Don't forget to change the {size}{{Size}CX}408{/CX}{CY}44{/CY}{/Size}
y value to account for the fact that the gauge now starts 20 pixels off screen.

In short change the loc line so that it's -20 of it's current value, change the gauge ofset so it pushes it back down 20 and change the size so the guage is 20 bigger. You now have you target gauge with the name of the mob being displayed off window so just use a standard label with the eqtype 28 and boom you've got your numbers.

Obviously all the {} should be replaced with greater than and less thans int he xml file just didn't feel like taking the time to code it so it showed properly in the post =)

Copied and pasted from the last time i posted it. If you want a mod that actually uses it check mine at www.cohort.cc/~taraddar/TaraddarUI.zip

I've found that mod to be amazingly useful when you get 2+ mobs with the same name in camp. It's so easy to know you slowed mob_01 and mob_02 but not mob_03.
