No problem and thanks for writing!

Ok go to your UI directory and you will see a "default" folder. Right click on that and chose copy. The pick a blank spot in that same UI directory and chose paste. that will give you a new folder called "copy of default" right click on that folder and chose rename... change the name to Faren or whatever you would like it called.

Now take the files you downloaded from my guilds server and open that zip file. go to edit and "chose all" then edit again and "copy"... Then navigate back to your ui directory / faren folder and "paste" the stuff from the .zip file into it . It will ask you if you would like to over write the files, just say "yes to all"....

When that is all finished the Faren UI should work just fine now :)

Let me know if you got it to work, or if any of this did not make sense.. And thanks again!
