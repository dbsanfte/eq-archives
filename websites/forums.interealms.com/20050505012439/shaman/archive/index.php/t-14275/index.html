Hail,

First off, this is not a brand new ability for dogdog from the 11June03 patch. It has been in game for a few months now.

In my experience, you don't have to be at <50% health. Generally speaking, dogdog positioning is the key to having this occur. You have to be close alongside dogdog and have solid aggro. The shield seems to take effect after a particularly heavy round of melee taken by dogdog's owner (us). So if a mob CAN double attack/kick/bash, and gets all 4 to hit, that seems to trigger dogdog shielding more than a single hit and a few whiffs by the mob. Please note, this is anecdotal evidence only, and I haven't taken any logs or done any analysis for this.
