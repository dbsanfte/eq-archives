Weyalan
Level 53 is a really good time to start doing KD arena raids (assuming you are on Dragon / Dorf faction) - you will get good exp, faction, loot and some good armor upgrades.

Some of the Runecrafter's gear is not that good (bracers, gloves). Some is ok (helm) and some is damn fine (BP, legs, boots). Anyway, if you have a level 60 tank, a FD puller and a CH cleric, then you can do Arena with 6, otherwise, 2 or 3 groups is good.

Other possible upgrades
Shoulders: You souold be able to solo elder Wolfhide in Hollwshade Moor
Neck: Ivory Imbued Collar
Ear (not Solstice): Earring of cleansing/purity/twisted leaves
Shield: Sarnak Battle Shield
Fingers: Circled toe bones, Shadel bandit ring
