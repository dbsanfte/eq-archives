Dragon Necropolis is good if you have means to get there and back and are not KoS to CoV faction.

Inner Acrylia has some nice drops and all dark blues right through to 65.

Velketors is an old classic, along with the already-mentioned Sebilis. Pickup some velium and some named drops for extra pp.

If you will want to get a VP key, then try some shard-dropping zones to kill 2 birds with one stone. Akheva, ME, Acrylia, The Grey etc are all reasonable xp.
