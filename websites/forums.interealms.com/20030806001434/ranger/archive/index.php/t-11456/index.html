1% chance to kill a mob in one shot means that some unlucky saps will never actually see it go off. Along those lines I would like to see a "Called Shot" ability. Much like this:

Called Shot - Reuse time 1 hour. Activated.

Your next arrow will automatically hit for full critical damage. Consequently this will also render archery unusable for 3 minutes and cannot be used in conjunction with the Trueshot Discipline and cancels Trueshot when used.

The consequences make it so this would be a end fight ability used after Trueshot Disc. The shot would still use damage multipliers ie stationary mob and archery mastery, but would also suffer the same LoS penalties as archery currently does.

I don't think this would be overpowering, but still give us that one great shot when we need/want it.

Just a thought...discuss
