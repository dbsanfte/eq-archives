When you're adding an item, you have to type in part of a name to match against the database. For example, you could type "solstice", and it would return a list of all earrings that have the word "solstice" in them. If you don't type anything, you get a list of everything. But, the list that is filled in for you is limited to a certain size, most likely to improve the speed and the bandwidth it takes up.

You have to know what item you want to put in the slot before searching. If you purchase the premium magelo account, there is a search feature you can use to find upgrades.
