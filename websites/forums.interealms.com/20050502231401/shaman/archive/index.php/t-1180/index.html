Chutoi
Warriors used to get a bonus. They no longer do (and haven't for a number of months--it was abolished when the class penalties were changed).

Based on 100% being the 'base' experience to level, the races need the following to level:

Iksar 120%
Troll 120%
Ogre 115%
Barbarian 105%
Halfling 95%
All other races 100%

Thus, it takes longer for any race over 100% to level, and halflings level faster than all other races. There are no longer any class penalties.

snapdragn
This is from Absor, back when he made the changes (Jan 2001):

Race-based penalties are appropriate. An ogre, for instance, does indeed make a better warrior than a halfling. It is not so little that the faction and size problems make up for it, and not so much that it is really unbalancing at upper levels, but enough that the penalty should apply. Secondly, the penalty is not so severe (compared with class-based penalties) that it would cause groups to break up on the journey from one to sixty due to level differences.

Class-based penalties are not appropriate. Classes are roughly equivalent in power throughout the level ranges, and the versatility does not make up for that penalty. In fact, the majority of changes made to classes in the name of balance in the last year were based on the assumption that, at the high end, each class should still be roughly as needed and balanced as any other.

Class-based experience bonuses (which warriors and rogues get) are also not appropriate, as they cannot be so if penalties are not. However, we've decided to leave this as-is, since the bonus is not so severe as to be unbalancing. Bottom line: we don't feel the bonus is enough to warrant a fix that could be interpreted as a 'nerf'.
