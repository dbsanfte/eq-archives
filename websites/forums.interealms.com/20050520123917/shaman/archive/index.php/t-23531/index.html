taraddar
This is a new look at the same topic I posted previously. Will refer back to http://forums.interealms.com/shaman/showthread.php?threadid=23302 from time to time.

The seem to have taken care of all the Shaman-omen-Ph spells. Some have been convered to spells others have just been removed it seems.

Yoppa's Spear of Venom
Name change since last post. Otherwise same as before. Ends up being less efficient and roughly the same dps as our cold nukes. Increase efficiency and decrease cast time more or make a larger nuke to up the dps some.

Spirit of Sense and Tailsman of Sense
Once again name change only from Ancestral Grace. Check last thread.

Spirit of Perseverance and Tailsman of Perseverance
Once again name change only from Ancestral Regrowth. Check last thread.

Putrid Decay
No significant changes to the spell. Check last thread.

Crippling Spasm
No significant changes to the spell. Check last thread.

Spirit Veil
Looks to be a self only fixed duration invis. Not a bad thing to have and had been requested by some. One of our shaman-omen-ph spells converted. Nice addition if a relatively minor one.

Spirit of Might and Talisman of Might
Spirit of might got lowered in level to match a druids level on their spell. Druids is still slightly better tho with slightly less mana cost, faster cast time and slightly longer duration. Again just don't see why druids should all of a sudden match shaman on thier "strength buff" and even have slight advantages given past advantages shaman had.

The group version of the spell gets kinda funny. The group version that was in last time is gone and I was unable to find what it had been changed to if anything. The shaman-omen-ph that had been the group haste spell got converted to talisman of might. The spell still has some reminants of the being a haste spell, namely the spell category and description but I see no reason to beleive that it's actually still a haste spell in any way.

Farrel's Companion
No real changes to spell. Check previous thread.

Breath of Wunshi
Name change from Breath of Wanvu. Check previous thread.

Ancestral Bulwark
No changes at all. Check previous thread.

Yoppa's Rain of Venom
Name change from Obkin's Rain of Venom. Check previous thread.

Yoppa's Mending
Name changed from Obkin's Mending. Also got amount healed increased and mana cost decreased. Now at the predicted 47% of a clerics heal and slightly more efficient than previous spell. Still beleive shaman and druids need to be much closer in healing to clerics but thats another arguement. Givens sony's current healing plan the spell is now what I'd expect.

Wunshi's Focusing and Talisman of Wunshi
Name change from Wanvu's Focusing. Also knocked 27 hps off the spells which I'm not sure why that was needed. As noted previously would still like to see the dmg mod added to spell instead of having to cast this and other buff.

Spirit of Fortitude and Talisman of Fortitude
Changed name from Ancestral Vigor and bumped both spells up a level. Still only seems to go 40 sta above cap instead of entire amount of spell. See previous thread for more comments.

Pained Memory
No changes from last post. See previous post for notes.

Ice Age
No changes from last post. See previous post for notes.

Curse of Sisslak
Name change from Ancestral Vengeance. See previous post for notes.

Vindictive Spirit
Another one of one of the omen-shaman-ph spells converted to a useful spell. PB ae 50% slow. Neg 100 sv magic and autocasts a a buff on the shaman that will absorb 50% of incoming dmg for the first 3200dmg (1600 dmg absorbed). Not a bad looking spell. Somewhat small ae range and shorter duration than recast time. Still looks like a good way to buy yourself some time to get things under control with out killing yourself, hopefully. Seems to mirror the druids snare spell.

Still some issues that need to be addressed with the spell. Spell type is still benificial, category is still hastes and game description still describes haste too. Might effect the functionality of the spell.

Spiritual Serenity
No changes from last post. Still needs to be upgraded to match upgrades other HoT's got.

Pure Spirit
Another one of the omen-shaman-ph spells. Has the potential to be interesting. Single target "cure" that has one chance to remove a detrimental spell. Judging from my experience with Radiant cure one chance to remove a detrimental spell is not that good. Radiant cure 3 seems to have a good chance of removing a deterimental spell but doesn't always and it has 6 chances to remove detirmental spells. Up pure spirit to this level and I'd say we've got a pretty good spell and a new role on some raids. It's possible Pure spirit has a better chance to remove a spell than radiant cure does but if it's the same only 1 chance isn't gonna cut it.

Champion
Several changes since last time. It got the attack and stats dropped back down to the level of the previous version and converted into a group spell. As it is I'd call it a pretty nice spell but would still ask what is so powerful about this spell that it has to still have such a short duration, long recast and high mana cost. I'd also ask why drop the stats and attack down so that it's just a group version of previous spell with the dmg mod added. Up the stats and attack back up to 160 or better. Also while group spells are nice for when they are needed not having a single target versions means always casting the group version even if only 1 person needs. To sum it up bump the stats back up, make the spell less of a hastle to use still and add a single target version or retoractively change ferine avatar so we have a single target spell too.

Blood of Yoppa
Name change from Blood of Obkin. See previous version.

Ancient Wilslik's Mending
10% more healed and slightly more efficient. Pretty much same thing as clerics ancient heal. As I said with the other heal, i think shaman and druid need to be closer to clerics on heals but this is parr for the plan sony seems to have for now.

Ancient Ancestral Calling
Nothing changed since last post.

All in all some nice improvements, some new spells but there are still a number of things that will hopefully be addressed. If i wasn't so sceptical i might think that sony actually read my last post and addressed some of the issues raised.
