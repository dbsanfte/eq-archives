Dude , over the summer I built a brand spankin new PC it has a P4C 2.6ghz (not overclocked) with hyperthreading FSB@800mhz, MSI NEO2LsR, 1024 mb PC3200 RAM, AIW radeon 9800 pro 128mb, SATA seagate barracuda 120gb hdd, and a soudnblaster audigy 2. I also run it on a 21 inch viewsonic p225fb.

Now I expected running two instances of a 4 year old game on such a machine would be a lag-free experience most of the time. Well, to answer the question you won't like what you experience at the highest graphic setting. At 1280x1024, full models, no FSAA or ANSIO, and clipping plane at 100 on both comps. Things start to get a bit sluggish, usually when turning, though mostly for Luclin,PoP,LoY,LDoN zones.

I used to play before on a AMD athlon t-bird1.3ghz with 384 mb ram, geforce gts 32mb, and it would run as smooth as that (maybe slightly faster) for one instance. What I wanted to say is don't be disappointed if you can't run both at the max settings even with the fastest system out right now.

It takes a bit of getting used to 2boxing... since you have to be able to switch between windows fast, and it becomes a hassle sometimes. right now my problem is with the damn mouse pointer getting stuck while I switch windows. But once I can fix this and get everything to run as intended, then I think it won't be so bad after all.
shamuofxev
Running 2 instances of EQ on one machine is a huge hassle. I'd just turn off all luclin models on the old machine and run the two characters on two machines. Otherwise you will quite often be severly hampered, for example you cannot position your melee character correctly while casting at the same time with your shaman.
Maybe start upgrading the old machine in steps (the 700 MHz processor hurts a lot in a cpu-intensive game like EQ). MB with CPU that takes your memory and video card in the 1.7 GHz range should be able to be had for less then $150.

Extraor Dinary
Ctrl + Alt + S will switch between the two eq windows. This will keep you from having to Ctrl + Alt + Del. Also, Ctrl + Alt + R will release the mouse so you can minimize the screen or click on the other window in the taskbar if you so choose to do that. These shortcuts can be changed to something quicker if you want to mess with the eqwindow ini file.
