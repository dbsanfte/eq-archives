The catch is, you weren't in a pet group. You were in a non-tank group, with what appears to be decent DPS. The problem is, you don't have anything to really shield you in case something does go wrong with aggro.

In a true pet group you should have 4-5 "power" pets (mage, necro, bl) available to pound on the mob and take the aggo quickly. In a true pet group, slow is actually optional because nobody cares if a couple of pets get shredded on a mob.

That's what some folks have a hard time getting the idea of. We're all so used to keeping the tank alive, we need slow, we need CH, we need Mez. None are required (but they are appreciated), because a dead pet doesn't hold a grudge. :)
