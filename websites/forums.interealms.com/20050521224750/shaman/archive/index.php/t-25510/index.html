First off, my appologies to anyone who might have taken me seriously and gone off to find Macroquest and install it. My answer was meant tongue in cheek.

pulled from Magelo's FAQ

<i>Magelo Update doesn't sniff the packet stream, which would violate the EULA (End User License Agreement). What it does is read the relevant memory locations. </I>
Which is exactly what Macroquest does. It doesn't snoop the datastream, it reads (and changes) Everquest memory. It just happens that a lot of the structures are the same in memory as they are on the wire, which is why the ShowEQ and MQ folks worked from the same base in a lot of cases. The difference between Magelo and Macroquest is that Macroquest is designed to change gameplay and violates section 9 of the EULA. ShowEQ, however, doesn't "modify the software to change Game Play" either so why is it any less legal than Magelo?

On a related note, It always amused me how Magelo would so coincidentally start working immediately after the MQ crew figured out the new structures. I'm sure I know where Jelan is getting most of his information.
