Tugela
Since there are no roamers in Time they could simply leave an instance open once it has been activated. It would have to be done a little differently from regular instances of course. I've been giving this thought lately and IMO what might work would be to redo time as follows:

1) have a lockout time after each phase as you do in an IK trial. So, if you complete phase 1, the lockout time becomes 12 hours. If you complete phase 2, it becomes 36 hours. If you complete phase 3, it becomes 3 days. 4 days for phase 4, 5 days for phase 5 and 6 days for phase 6.

2) If you do not complete the phase before the timer is up you are booted out of the instance and get to retry when your lockout timer is up. This would allow guilds working on earlier phases for the first time to retry on a more frequent basis while still maintaining the flow of items into the game at roughly the same rate.

3) If you want to put the timer on hold your raid leader could hail a NPC in timeA, which would suspend the instance. If you did this it would start a 1 hour timer that would prevent re-initialization of the instance.

I think that if they did something like this they could fully instance time without adversely affecting the current flow of items into the game.
