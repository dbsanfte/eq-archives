Schpleef
If you use a custom UI, try typing:
/loadskin (custom UI foldername) 1 (/loadskin default 1 if you use the default UI)
The 1 at the end of it tells it to save window positions

As far as the windows changing positions, they should stay in the same place in windowed mode as in fullscreen, provided you didn't change the resolution you run EQ in. Make sure you're in the resolution you want to be in - this is easiest in game, alt-O to bring up options and click on the display tab. It can also be done in "Options" before you get to the password screen loading EQ, or in the eqclient.ini file, although that last one is a pain.
