Here is the solution: EverQuest: All NPC's in certain zones appear as human males.

1. Ensure that when you start the everquest patcher that you go to the advanced options, and enable the downloads for any expansions you have by checkmark the appropriate expansions.

2. Go to your everquest folder (usually c:\program files\everquest or c:\program files\sony\everquest) and look for files that end in .txc and delete those, then restart the patcher

3. If neither of the above work then you can try reinstalling the game
To safely reinstall the files, please follow these steps, in order:

1. Use the Add/Remove control panel and uninstall ALL EverQuest and EverQuest expansion installations.
2. Open the Find/Search command from your Start menu and select Files or Folders
3. Type "EverQuest" in the Named field; set the Located In field to "All Local Drives", click Find/Search Now, and delete everything that appears.
4. Type "*.s3d" in the named field; set the Located In field to "All Local Drives", click Find/Search Now, and delete everything that appears.
5. Go into your C: drive, double-click the "Program Files" folder, double-click the "InstallShield Installation Information" folder. Note: this is a hidden folder and so you must be showing all files to see it.
6. From the InstallShield folder, delete these folders: {2848FB25-CF81-11D5-B880-00A0CC58DEE4} (Luclin), {B769E280-9708-11D5-B880-00A0CC58DEE4} (Trilogy) and {DD494BAD-BA13-11D4-8737-00A0CC58DEE4} (Velious)
7. Reinstall from the CDs starting with your Original, Kunark Full, or Trilogy discs, then your expansion discs, from oldest to newest.
8. Do not start EverQuest until you have installed from all of your discs.
