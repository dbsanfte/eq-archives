Gloriana
Yep.

Right click the augment, there is a new message in the item description that states the type of augment remover required.

Then, head over to the BB adventure camp and find the little gnome guy that sells solvent. Buy the correct type of solvent.

Move over to the birdbath. Combine the augmented item with the solvent.

Presto.

Yes, you lose neither the item or the augment. Each augmented item, if right clicked, shows the augment removal solvent type you need to buy. I believe you get them from the vendors in the camps (I haven't removed one yet so I don't know), then you use the augmentation table method to remove the augment by putting the solvent in and the item.

Yea, Gnome NPC in BB adventure camp sells augmentation distillers. Look on the augment you're willing to remove to see what class you need.

Ranges from class I to class XVIII, where I is the cheapest and XVIII the most expensive.

Combine the appropriate aug. distiller with the item you're willing to remove an augment from, in any bird pool. Will return both the item and the augment.
