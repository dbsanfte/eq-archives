Download a skin. Unzip and read the readme.txt. If it says to run some .exe, and you trust it, run that to install it. Otherwise, just unzip it to a folder in the "uifiles" folder in your EQ directory. Then in game, type /loadskin.

Just to make it a little easier for you...
Goto your EQUI folder in your everquest directory, and copy all of the files in your default folder... Then paste those files into a new folder named whatever you want it to be called as long as you can remember it... Then download what ever mods you want to add to your UI and when you unzip the files paste them into that new directory... and when you load the ui in game (/loadskin "your new foldername here") you will see the changes!! Hope this helps a little!
