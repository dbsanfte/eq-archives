Caanus Grey
It means when you log in that run mode is active, however if you ever hit CTRL - R then you deactivate runmode, this is VERY easy to do on accident while trying to SHIFT ALT R out of the game in normal window mode, and even easier to do on accident if you use EQW with CTRL ALT R.

YOu can go to options and keyboard, then remove the capability to use CTRL R to toggle run/walk, then you never have to worry about it again.
