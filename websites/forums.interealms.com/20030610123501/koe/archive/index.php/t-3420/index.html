Alan VanCouvering unleashed some important news on the Developer's Corner. Catch the full story here:

In the next month or so, prior to the release of EverQuest: The Shadows of Luclin, we will be moving to DirectX 8. This will be true not just for those purchasing Shadows of Luclin, but for everyone playing EverQuest.

In fact, some time soon, prior to the new interface rollout, we will be giving everyone the DirectX 8 version of the graphics engine dll. This will give players the option of using the new dll to help us find any hardware incompatibilities prior to the time we move to DirectX 8 game-wide. We'll be testing as many configurations as we possibly can in-house, but with the huge number of possible hardware configurations we can't test them all. The plan here is to nail the problems down now and have as many of you updated to DirectX8 as soon as we can before we make the switchover.

Why are we doing this? Well, you'll need it to use the new interface and we need it for all of the good things it lets us do for the game, primarily improving framerates (yes, even in the old world and even if you don't buy Shadows of Luclin). For a good idea of the new features we'll be able to implement with DirectX 8, check out our latest In Development (the EVERQUEST TECH DOCUMENT at the bottom).

Upgrading to DirectX 8 is pretty simple. Here's what you do:

1) Download DirectX 8.0a. Store it in a directory so you can find it when you need it in a few minutes, but don't install it yet. You can find them on Microsoft's site. Make certain to get the driver that is appropriate for your operating system.

2) Locate the most recent drivers for you video card and download them into the same folder (this makes them easier to find). Don't install these yet either. Do this even if you have the latest drivers installed. If they have a text file with installation instructions, you might want to get that too (or print it out).

3) Locate the most recent drivers for you sound card and download them into the same folder. Again, don't install these yet, and go ahead and download them even if you have the latest drivers installed. You might also want to get any installation instructions for the drivers. You're going to need to reinstall them shortly, so it's best to have them on hand.

4) Set your video adapter to the Standard Display Adapter (VGA)
The steps required to do this will probably vary for each of you a little bit. But here are the steps I followed to change this on my computer (keep in mind that I am using Win 98 and the procedure will be different if you're using Win 2k or XP):
a) Click Start, then Settings, then Control Panel.
b) Double Click on Display
c) Click on the Settings Tab. Then click on the Advanced Button.
d) Click on the Adapter Tab, then choose Change.
e) Click Next.
f) Select the radial button (the little circle) next to the option for Display a list of all drivers and click Next.
g) Select the button next to the option for Show all hardware.
h) Scroll to the top of the list and select (standard display types). Then from the window on the right choose standard display adapter (VGA) and click Next.
i) It will ask you if you really, really want to use this adapter. Choose yes.

5) Now install DirectX 8.0a by finding it in your folder and double clicking it. Follow the instructions for installation.

6) NOW you need to reinstall your video drivers. Use the new ones that you saved in that folder. Follow the instructions for installation given by the manufacturer.

7) Install the latest sound drivers for you sound card, also following the instructions given by the manufacturer.

That should be it. You may want to reboot after this if the last driver installation didn't already force you to do it, just to make sure.

NOTE: It is important to reinstall the latest video and sound drivers after you install DX 8.
