Giden Grooverider
The basic difference is duration. The higher level the root, the longer it lasts.

When I am soloing, I typically use the root spell because it has the same duration as the poison spell. I pull with slow, send pet, let pet get a little agro, disease, poison, then root and sit down to med. When poison wears off, u redot and reapply root.

I use the lower level spell because I know better when it's going to wear off, so I know how long till the mob begins to try to run and the casting cost is cheaper. If you are recasting your dot and instill / immobilize wears off, then you have to spend more mana to recast the root again. A cheap root is more practicle IMO.

In a grouping situation, when root is serving as ghetto mez, I use the best root that I have. You want the mob to be parked away from you as long as possible.

I really haven't noticed a big difference in resists, they all seem to be equilly resisted to me.
