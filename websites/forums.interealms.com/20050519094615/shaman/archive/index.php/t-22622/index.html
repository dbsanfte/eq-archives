Hi, I'm the author of EQ Companion. It really sounds like you are using an old version of EQC. When you download the full installer, it installs version 0.6. Version 0.794 is the latest version and it parses all spell damage, DD's and DoT's. You can download that from the downloads page.

The only manual stuff you might have to do is if you want to see which spell did the actual non-melee damage. You accomplish this by creating entries in the 'Weapon Procs' list (I really should rename this back to 'non-melee' list...)

For example creating entry:

Spell name: Spear of Torment
Phrase: screams in torment.
Title: DD: Spear of Torment
Damage: 900
Comment: Spear of Torment DD

will look for lines in the log file that have the string 'screams in torment.' in it. It then knows that the past non-melee message up-to 900 damage was caused by spell 'Spear of Torment' and it will be shown under non-melee damage as 'DD: Spear of Torment'. If you get a resist message like 'Your target resisted Spear of Torment', EQ Companion will increase the 'miss' counter for that line.
