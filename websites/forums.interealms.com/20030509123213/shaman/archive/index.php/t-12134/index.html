You can actually download all the files from the patch server through your browser, it's a bit complitacted, but you need to put the file names of the files you need at the end of this address.

http://64.37.156.21:7000/patch/everquest/everquest-changes.xml

eg:-
http://64.37.156.21:7000/patch/everquest/kunark/citymist.s3d.gz

then uncompress this into the Everquest directory.

Not the easiest way, but it can be done.
