Azzikai
Targar

When you right click on the main window you get the generic windows drop down menu.

On that list is a "filter" option that when highlighted has it's own drop down list of all the different chats and things that normall go into the one main window of the old interface. Highlighting "say" for instance will bring up the message "say will now go into Main chat window" in the main chat window. You do that for all the stuff you want in that window.

If you create a new window (easy to do, right click, new window, and you have it) you can separate stuff. By right clicking on the new window and selecting "spells" from your filter options you'll see the message "spells will now go into chat 1 (or whatever they call that 1st new window). Repeat as necessary.

If you decide you don't want spells in the new window, just do the right click etc. stuff and highlight it again. It will default back to the main window on it's own.

Okay, that makes a bit of sense, easier to understand when you're looking at the interface :)
