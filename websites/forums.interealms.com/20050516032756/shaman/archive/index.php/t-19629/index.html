Not too shaman related, but interesting. The field in item collectors that determines proc rate of a weapon has been found.

The field is called unknown070 by the eqemu database. There are not that many weapons that have a non-0 value for the field.

Can read more about it from the warriors,
http://www.thesteelwarrior.org/forum/showthread.php?s=&threadid=5570

They refer to the SK board, which has this great post:
http://www.shadowknight.org/forums/showthread.php?threadid=19192&postid=247095&perpage=1
