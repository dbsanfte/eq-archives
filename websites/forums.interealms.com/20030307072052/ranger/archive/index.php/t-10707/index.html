Shenlong
Now for the explanation. Most Rangers want a class defining ability, most Rangers want that ability to involve archery (from what I've read). Considering the amount of spells they add with each expansion, this seems to be the easiest thing to do, add spells. This would give Rangers a reason to use archery before 50+. This would make us the hands down best archers in the game, even before lvl 50. The reason I recommend the mana cost and duration of these buffs is because this would be our class defining ability, it shouldn't be a hassle to keep them up. I don't think it would unbalance us, think about it. The Arrow spells end up being only 1 higher than our damage bonus to our main hand at the last spell. At any given level, we would have to choose between a slightly higher archery only haste, or a slightly lower haste from an Enchanter that will affect melee also. I'm only level 40 at the moment, so I haven't included anything above 49. Well, any feedback will be appreciated. I have thought about sending this to Verant, but I don't know how to, and I wanted to get some feedback from other Rangers first anyways. Thanks for reading this :) .
