Is the file size of his GDI32.dll the same as yours? If not it may have been altered by a virus. You can replace the dll by booting to dos mode (use a win98 boot disk is an easy way to do this) then replacing the file. It wont be in use under dos as it is the standard windows drawing interface (nothing to do with DirectX)

If the file size is the same, I would first do a "complete file scan" from EQ in the advanced menu. If that doesnt help i'd try replacing the dll using the above method.
