Jaruden
Halfling faction (I believe) is the only one opposite neriak, and I don't remember if that particular quest messed with it. What I did (as an iksar, kos to both to start):
kill halflings in nektulos. Takes a while, but gets guard faction up. Once nonkos to guards in nek (and some inside), go the library in 3rd gate and handin red wine in mass quantities to a scribe whose name I forget. Eventually that'll get you nonkos to a lot of neriak. It's still a varied enough zone that there are guards/npc's all over on different factions, so I still don't feel safe just strolling down the street non-invis. I think there are a few ogre/trolls on different factions as well that you can't fix.
If you want to fix halflings up after that, there's one near the port in area that wants bandages. Handin in a ton of those to go from max kos to nonkos. I did this one when you could do the handin / invis trick (he's too low level to agro you even if kos), might require some more work now.
