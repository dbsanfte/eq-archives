KumoeSpearbear
As a RL friend of a level 65 chanter, we duo very well.

Snare glands (if not a troll with necklace) are extemely helpful. Other snare abilities work as well.
Method i use is:
1- malos
2- root (short lived root if possible, VP if needed)
3- malosinia and tash.
4- Snare gland
5- Either pull a mob to it, to break root with melee, or wait for root to drop.

Keep the chanter a long long way away. Runes are vital. Haste the charmed pet if you want. When we duo, i mainly debuff the mob the pet is on, and keep Q on the pet all the time.

A neat trick, if you are not too worried about the DPS, is use torpor instead of Q. So when it break, the mob is snared. Not sure if that still works, but it used to.

Hope this helps :)
