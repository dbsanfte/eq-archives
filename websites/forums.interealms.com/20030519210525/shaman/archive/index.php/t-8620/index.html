There is a relatively decent range item that can be quested if you have the PoP expansion. Also, they progress in difficulty/reward similiar to the Thurg shawls.

It starts out with the following:

Willamina's Needles, very easy quest, involves you running around PoK talking to various people and giving them various items. Anyone of any level can do this part.

http://everquest.allakhazam.com/db/quest.html?quest=2183

LORE ITEM MAGIC ITEM NO DROP
AC: +1 Cha: +2
Weight: 1.0
Classes: All Classes
Races: All Races
Inventory Slot: Secondary Melee
Can be equipped in the range slot

This in itself sucks, but don't worry you can upgrade it to this:

http://everquest.allakhazam.com/db/item.html?item=16506

LORE ITEM MAGIC ITEM NO DROP
AC: +7 Wis: +6 Int: +6 HP: +15 Mana: +15
Weight: 1.0
Classes: All Classes
Races: All Races
Inventory Slot: Secondary Melee
Can be equipped in the range slot

This in turn, can be upgraded to this:

http://everquest.allakhazam.com/db/item.html?item=16538

Classes: All Classes
