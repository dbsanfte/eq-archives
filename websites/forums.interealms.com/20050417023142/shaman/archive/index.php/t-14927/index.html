Open the EQUI_HotButtonWnd.xml file and start at this line:

<!--First Slot, Third Row for Neck Inventory 3,1 -->

What you want to change is the number in the EQType as shown here:

<EQType>5</EQType>

To change this from neck to left wrist you would change the 5 to a 9, as shown from the list below:

1 = Left Ear
2 = Head
3 = Face
4 = Right Ear
5 = Neck
6 = Shoulders
7 = Arms
8 = Back
9 = Left Wrist
10 = Right Wrist
11 = Range
12 = Hands
13 = Primary
14 = Secondary
15 = Left Fingers
16 = Right Fingers
17 = Chest
18 = Legs
19 = Feet
20 = Waist
21 = Ammo

The background image for the slot is in the line above the EQType. You can look at the default UI Inventory window if you like to get the default background images for the appropriate slot.
