Just an FYI in case you guys aren't familiar with it (though I'm sure you are!):

If you're debuffing disease on raid mobs, don't overlook Insidious Malady. Insidious decay is like malo and insidious malady is like malosini. ID does -40 dr and is unresistable, but IM does -60. So, ID them IM, then POI. :)
