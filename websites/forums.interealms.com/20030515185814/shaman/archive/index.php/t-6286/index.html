Like what other folks have said ...

1. Cold boot PC ( shut down out of windows to power off and then turn back on again ). Note the Ram Check messages during the Non-Windows bootup period. If you have some Vendor graphic during bootup ... normally pressing the Tab or Esc key will drop the graphic showing the bootup messages. If you still can't see any information on boot up ... you can change the Bios setting to show RAM check BUT itss a bit dangerous for the faint of heart to mess around in a PC's Bios.

2. Confirm that you have not changed the settings in Windows from Automatic to some manual setting. Check by right clicking on "my computer" icon - select properties - performance options - make sure it's set to "Let windows select" with all other options grey'd out.

3. If your PC is a branded model (ie Compaq, Dell, Gateway, etc) then go to their web sites for troubleshooting. Input your systems model number for advanced information on your motherboard, bios, etc. Many vendors offer free utilities for troubleshooting which may check your Ram sticks. Plus these programs will check other things such as drivers, bios, etc for problems. Check the ram configuration diagram off these sites for pattern for installing your Ram sticks. Many motherboards require that you install Ram in even sets ... ie 2x128 sticks instead of (1x128 & 1x256). Also confirm that your sticks are same BUS speed as required for your PC. Example ... you may not be able to put a PC-133 stick in a system that only takes PC-100's ... your vendor should be able to confirm this. Best bet is to match your older Ram Sticks other than improved RAM speed.

4. Rather than list this under 3 ... when checking your spec's on motherboard ... confirm that your system can take 512 Ram. I have a PC that will NOT upgrade pass 386 Ram ... so it could be due to the motherboard rather than Win98. If you can't confirm by PC vendor ... your motherboard shown have a code in one of the corners denoting it's type. You will have to crack (open) the case to find this number before checking it on the web. Do a search via search engines for motherboard makers till you find a site that offers information on either that maker or do a lookup via the code number. Then confirm that the motherboard can take the ram, configuration, and amount being used to install.

5. Finally ... reseat all sticks. Best bet is to do this first before anything else. Just make sure everything is seated properly and in the correct position.

Windows 98 has some ugly memory limitations, which are adjustable in the registry. Your Windows installation sets your memory limit to the memory available when you install, supposedly to save search time for free memory.

I adjusted my registry settings and got a bit more memory out of it.

Then I decided to take the plunge to Win2000 (NOT M.E.), and I've found it has great memory handling, is pretty darn stable, and doesn't have all the stupid extra graphics that ME makes you endure. HOWEVER, if you are using SIMM's in a DIMM system, or some other kind of wrong memory you will find that you will get less memory reporting, or the system simply won't start up.

So now I am sucking it up and buying a whole new motherboard with FOUR slots that takes all the newest memory, and my old eq machine is being sent to the office..I only hope you don't have to do this.
