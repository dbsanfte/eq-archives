stygiana
While playing around with some UI setting I wondered if it would be considered handy to have the option to “block” certain buffs from landing on you.
As I see that in a lot of cases players are far from happy when a MGB is done that overwrites a buff they have I think it would be great to be able to prevent this by just selecting a few (15?) buffs that will never take hold on you (and thus never cancel the buff you would like to keep).
So I would like to hear your suggestions/opinions on this.
finaljustice

Even if you could set up an autodelete on certain spells, wouldn't they already cancel the first spell before the unwanted buff goes poof?
