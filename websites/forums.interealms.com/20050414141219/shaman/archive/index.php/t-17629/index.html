If you're on XP - I went into network properties, then LAN properties (works for modem as well, fixed it for a friend) and uninstalled QoS packet scheduler. From what I've read, it's a pretty much useless component that's installed with XP by default. Something with the new patch apparently gums it up pretty bad. My disconnects (which started with this patch) have pretty much stopped since I did it.

Hope that does it.
