stormrazor
It's not hard...go to Magelo (http://www.magelo.com)
and register. Create an ID and password for yourself and select Create my Account.
Scroll to the bottom-left hand side of the Magelo homepage and click on Characters.
It will then take you to a screen and select Add a Character. This is where you'll manage your toons profile.
There are several links at the bottom of the page
Update this character- manages stats, Surname, Server, Race, Class, Level, Deity, Money.

Manage his profile - This is where you use the Magelo database to put in what gear you're wearing, Magelo will add up everything for you as long as your starting stats/naked stats are correct. Once you have all your gear in name that profile and save it. The bottom of the page will show your stored profiles and below that will show you your default profiles web link. Click on that and it will display your characters stats, gear, etc, etc, etc.
