then note that although both the shaman and necro can stack VoS or E-bolt, you cannot stack VoS AND E-bolt.

This is false. All shaman dots stack. Period. VoS and Ebolt are shared by necros and shamans. Because they are cast by different people, a necro/shaman pair could each have both spells running at the same time.

The all-dots-stack thing is EXCLUSIVE to shamans! Necros can only stack the poison dots which they share with us. Druid insect dots, necro heat blood dots do NOT internally stack with each other.

A shaman could run envenomed breath, venom of the snake, envenomed bolt, bane of nife, ancient: scourge of nife. and blood of saryrn all on the same mob at the same time. A second shaman could run all of these spells on the same mob at the same time.

A necro could run venom of the snake and envenomed bolt on the same mob at the same time, but they could not run heat blood and boil blood at the same time...the higher level one will overwrite the lesser.
