the hole, down deep, might be really good aa experience at 60. i seem to remember it being pretty decent for old world content at lvl 65. ugly cr's though. mage duo is perfect down there. there's a great xp modifier down there, and i know it's great starting even in early 50's.

in dulak, by the forge (which is located between some buildings), if you pull to the forge, mobs don't run away from you when they're low on life. not sure if that helps, but it saved me from having to root every stupid mob i pulled. should help speed up kills if you're not already using it.

speaking of loy, if you're low 50's, lava dudes in nadox (?) might be decent aa. there are a couple really good posts in this area that can you help you out with specifics there.

at 60, i remember fungus grove being pretty decent. might want to go 61 so you get next pet. epic really shines here. pull a bunch of mobs, root all but one, slap epic on them, and then refocus on one.
