Something I should have added to the original post:

After you go back to a production server from Test (and back to the original GUI), you will have to reset all your user-defined settings (like text colors, window positions, AutoRun setting, etc.)

Sorry this is a bit late, but I just found out about it yesterday myself.
So for those that have yet to try this, you might want to save your original eqclient.ini file as something else and then revert back to it after you are done checking out the changes on Test.
