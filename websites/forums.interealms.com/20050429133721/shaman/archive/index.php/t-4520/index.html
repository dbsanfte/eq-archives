Here is what you do:

1. Delete your copy of the default folder
2. Create a new folder but don't copy any files into it
3. Copy all the files from your UI zip to the new folder
4. Start the game and /loadskin it up

VI recently made a change to the way the UI checks for files, basically if it doesnt find the files in your custom folder then it just takes them from default. This should allow you to keep your custom mods even if they make more changes to the UI.
