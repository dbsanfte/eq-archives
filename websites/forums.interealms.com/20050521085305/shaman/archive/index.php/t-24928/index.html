drib
500 is sadly not enough of a sample size to judge anything by. Your best bet would be to leave your computer running overnight, and have a thumper or something set up to chaincast minor healing for a relatively large sample, say 10000 casts.

500 is more than enough data to calculate a decent confidence interval on a Bernoulli random variable. You are estimating the crit rate on heals, which is p.hat here. The maximum likelihood estimator (an unbiased estimator) of a Bernoulli random variable is the sum of the number of times the event occurs divided by the total number of events observed.

n = 500
X-bar = p.hat = 80/500 = 0.16
1 - X-bar = q.hat = 0.84
Standard error = se.hat = sqrt(p.hat * q.hat / n) = sqrt(0.16 * 0.84 / 500) = 0.016

A 95% confidence interval would be p.hat +/- 2*se.hat = 0.16 +/- 0.032 = (0.128, 0.192). 18% is still within that confidence interval.

If you want a confidence interval that differs by less than a percent from the center on each side, then you'll need n = (0.16 * 0.84) / 0.004999999^2 = 5379 samples for a suspected 16% crit rate. n = (0.18 * 0.82) / (0.004999999^2) = 5904 samples for a suspected 18% crit rate. If you want less than 0.1% difference on each side, you'll have to collect around 540000-600000 data points. For each order of magnitude of precision you want, you'll have to collect 100 times more data (inverse square relationship).

Using statistics you'll never get the exact answer, only an estimate of what you're looking for along with a measure of confidence of how good that estimate is. Just looking at the data over the 50-sample sets, I see the same interval closing in around 16-17%.

For real fun, you should have collected crit heal data before you got the OoW AA and then compared it to the data after and done some hypothesis testing.

I think the data is leading more to showing a 1.5% increase per level than 2%. If it were 2% per level and assuming 16% is the crit rate before any OoW instant heal AAs, then I'd expect it to drift closer to 22% than it currently is. 22.7% is the top of the interval, and I don't like that.
