Taking Wolfherz's post, I've boiled down the entire HO tree. The following covers almost everything... the holes in current knowledge are noted. I'll try to keep this post up-to date using Wolfherz's info.

1 Class Chains

Fighter starts -> Fighter (Sword)
1) Sky Cleave single (Horn) = Slashing DD
2) Crushing Anvil single (Horn) = Encounter Crushing AoE DD
3) Hero's Armor single (Horn) = Armor Increase Buff

Priest starts -> Priest (Hammer)
1) Divine Judgement single (Hammer) = Divine DD
2) Inspiring Piety single (Hammer) = Divine DD, +10 Inspiration Buff
3) Blessing of Faith single (Chalice) = Single Power Regen buff

Scout starts -> Scout (Coin)
1) Ringing Blow single (Dagger) = Piercing DD
2) Swindler's Luck single (Dagger) = To Hit Buff
3) Bravo's Dance single (Dagger) = Short Attack Speed Buff

Mage starts -> Mage (Lightning)
1) Arcane Fury single (Lightning) = Magic DD
2) Arcane Storm single (Star) = Encounter Magic AoE DD
3) Arcane Inspiration single (Flame) = Power Regeneration Buff that gives a chance to Regenerate Power when you are hit

2 Class Combo Chains

Fighter starts -> Priest (Chalice)
1) Divine Blade unordered (Hammer) + (Sword) = Divine DD
2) Crippling Shroud unordered (Horn) + (Moon) = sometimes decreases attack rate of any that damage you
3) Chalice of Life single (Chalice) = Insta Heal + Ward + Regen Buff

Fighter starts -> Scout (Coin)
1) Luck's Bite unordered (Cloak) + (Boot) = Piercing DD
2) Swindler's Gift ordered (Cloak) -> (Arm) = Group Attack Buff
3) Raging Sword ordered (Cloak) -> (Horn) = Powerful DD

Priest starts -> Scout (Coin)
1) Piercing Faith ordered (Moon) + (Cloak) = Piercing DD *needs confirmed
2) Divine Trickery ordered (Hammer) -> (Cloak) = Divine DD *needs confirmed
3) Faith's Bulwark ordered (Chalice) -> (Coin) = *unknown / needs confirmed

Mage starts -> Priest (Hammer)
1) Arcane Chalice ordered (Flame) -> (Chalice) = Group Power Regen
2) Suffocating Wrath unordered (Hammer) + (Lightning) = AE Magic DD + Possible Debuff
3) Ancient Crusible single (Moon) = Large Power and Health Instant Regen

Mage starts -> Scout (Coin)
1) Trickster's Grasp ordered (Flame) -> (Mask) = 4 tick Magic DoT
2) Arcane Trickery unordered (Lightning) + (Dagger) = Magic DD
3) Rain of Daggers unordered (Lightning) + (Dagger) = *unknown

4 Class Combo Chain

Scout Inits -> Fighter (Horn) -> Priest (Moon) -> Mage (Fire)
1) Strength in Unity unordered (Arm) + (Chalice) = Str, Wis, Int and Agility Buff
2) Ancient Demise ordered (Boot) -> (Coin) -> (Lightning) + (Hammer) = *unknown
3) Past's Awakening ordered (Fist) -> (Star) -> (Cloak) = *unknown

Note: Scout Mask Combat Art, when the Mask art is the final step used to finish the Heroic Opportunity (whether ordered or unordered), appears to require the scout to target a single mob while using the skill. This is a known issue.

Helpful Hints:

1) A cleric never inits except for cleric-scout duo chain or a solo HO
2) A mage never inits except for a mage-scout duo chain, or mage-cleric duo chain or a solo HO
3) In a duo with a scout, the scout never inits unless it's to do a solo HO.
4) In a 3-class or a 4-class group with a scout, the scout inits to create all 3-class or 4-class chains. If no scout is present in the 3-class group then it's the fighter to inits for the 3-class chain.

They wanted scouts to be the masters of the HO wheel. The last point, combined with the coin to reroll results, certainly makes them control it more than other classes.

Mage starts -> Priest (Hammer)
1) Arcane Chalice ordered (Flame) -> (Chalice) = Group Heal + Weak Power Regeneration
2) Suffocating Wrath unordered (Hammer) + (Lightning) = Magic DD + Possible Debuff
3) Ancient Crusible single (Moon) = Large Power and Health Regen Over Time *needs confirmed

As a 2 Boxer that uses this particular HO Chain almost exclusively I'll add some corrections and notes:

1) Arcane Chalice (Common) -> (Chalice) = Group Power Regen ONLY (Regens around 1/3-1/2 bubble of power on completion, to trigger it the fastest I used Contagion after the Mage triggers, then fire off Static Pulse DOT on the Mage, then hit Totemic Aid on the Shaman)

2) Suffocating Wrath (Uncommon, unless you REALLY wanted that power boost, in which case you'll get this 5x in a row) = AOE Magic DD (Around 35-ish to all encounter mobs) and no noticeable debuff, nor does it have any maintained buff icon.)

3) Ancient Crucible (Note spelling) = INSTANT Power and Health regen (Very large) No regen over time, nor any maintained buff icon etc. Ultra rare.
