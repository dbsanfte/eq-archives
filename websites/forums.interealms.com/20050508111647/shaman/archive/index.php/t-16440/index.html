weissblau
Right-click on MY COMPUTER, right-click on each of your hard disk partitions, select PROPERTIES->TOOLS->Defragment to defrag the drives. Do it to all the drives.

Open your calendar (paper or online) and put an item in to do this again in a month. And a month after that. etc.

Run Windows Update. Take all critical updates and any Recommended Updates that seem to apply (like DirectX and Internet Explorer updates).

Get a virus checker, update it to the latest, and do a thorough scan.

Get rid of ALL extraneous programs that run at startup except a virus checker. AIM/ICQ/MSN, popup stoppers, little search toolbars, archivers/unpackers, etc etc. Boot and notice how fast and smooth everything is. Add the ones you use most back one at a time once you determine you really can't live without them. Consider invoking the programs you use manually when you need them instead if possible.

If those don't work, wipe the machine and reinstall Windows. Seriously. Windows installations rot over time and run slower and slower. Better to shoot them and put them out of your misery, then start fresh.
