Have done this a handful of times. The small is definitely doable by a single group, but I'm thinking you'll want a couple key players (crowd control, for example) to be 60. I think we held the small bottle with shaman, cleric, bard, ranger - all 60 - if that point of reference helps.

I'm wearing the only vest I've seen drop on these trips, I think your chances are a LOT better than getting a regular fungi... and there are nice drops for other folks as well. Not to mention I've never failed to have fun doing this, and the exp seems to fly. But it always seems to when you're having fun. :)

Useful info - nobody starts KoS to the caveman caller folks you need to buy the rotgrub(?)... booze from. You also don't start out KoS to the bandits, which I've avoided killing, since every time I get turned around in that zone, I end up in the middle of a bandit camp. Mmmmm, getting backstabbed by 6 lvl 55+ critters. Sounds fun... Er, anyway, I don't think you need to go thru any bandit camps to get there. If you hug the wall to your right as soon as you get out of the little settlement area, you can run there without invis if you choose.

I've never seen anyone else doing this quest, which surprises me, because it's an absolute blast. Another beautiful point of this spot is you can keep inviting people out. Once you have too many, start another tunnel going. It's the ever-expanding guild fun-nite.
