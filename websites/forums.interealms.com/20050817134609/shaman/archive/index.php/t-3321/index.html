A few weeks ago a lot of the guild members decided to start up a wave of new alts. At the time I only had my main so I soloed for several levels. Now I'm sitting at the edge of reaching lv50. Instead of pressing on, I decided to wait it out and start an alt. As the weeks go by, I noticed that the general guild attendance has been pretty sparse.
