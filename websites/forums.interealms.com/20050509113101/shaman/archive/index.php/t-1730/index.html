Saraquael does requests (but he has a pretty long sig request list at the moment).

What you do is:

1) email him a recent screenshot of your character (as a jpg, not a bmp). (there is a link to his email on his site)

As far as using an image for a signature...

1) You need a place to host it.
2) Once you find a place to host the image, you will use the signature feature of the Crucible (in your profile) to create a default signature that will appear at the end of your posts (once per page).
3) You need a basic (very basic) html skill to create one (mine was made with the help of friends /wave Megumi!). If you ever need help on a signature, I'm sure folks here would help you out with the code.

Gloriana
The picture has to be stored someplace. For example, my sig picture is stored on my friend Jai's website. Everytime someone reads a thread where my signature appears, Jai's website takes the bandwith hit as it is displayed to the viewer (help me here folks!)

That is why a lot of places (like earthlink and angelfire) don't let you store your pictures on their site and then link them to other places, they don't want the increased traffic since increased bandwith use = increased cost.

So, first you need a sig pic, then you need a place to store it, where you will be able to link to it, so others can also see it.
