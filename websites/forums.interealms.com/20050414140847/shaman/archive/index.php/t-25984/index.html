Trezl
as an SK, let me explain the issue with SKs and procs:

SKs get their own table of procs. This started way back when necros had a lifetap proc that scaled with level. An SK with the same level of scaling would have been imbalanced, so they made the SK proc not scale by adding a 2nd proc and putting SKs on a different lookup table for procs. So, every proc buff that an SK can use will have 2 procs in the spell.dat file - one for Sks, one for everyone else :)

The maw buffs didn't have a 2nd proc in the spell.dat. This must be why when I cast leopard on my SK he never actually procs.

I sure hope they don't change the proc rate on these though. Its a nice DPS increase for us that the melee and pet owners love :)
Grizlor
Actually, every proc buff has the actual proc (usually called "[Buffname] Strike". There is a normal one and a SK one. The reason SKs have different procs is because back in the day, necros and SKs got Vampiric embrace. Necros get it at 8, SKs get it at 22. However, Sony wanted it to scale differently between the two classes but not make a different spell or something, so the necro version scales to a 70ish point tap at 50, while with the exact same spell. SKs proccing Vampiric Embrace only tap for 35ish at 50. or something like that.

Since then, Each proc has had a normal version and an SK version.
