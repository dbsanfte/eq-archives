It may well have nothing at all to do with your router. A large number of people on the eqlive tech forum have been complaining about system lockups when zoning and loging on. A related isue seems to be numerous graphics problems like white landscapes and distorted skins. These problems are cleanly across all system types and video cards (ATI and n Vidia) top end and low end alike. As clear as I can tell they messed something up fairly bad in the last set of patches with video memory and are using up the entire amount of available memory to the video card (even the 256 megers) and causing the system to lock. You might try connecting with good old dial-up to see if the problem persists for you before ya go crazy diagnosing the router. If the problem persists on dial-up I'd recomend heading over to the EQ Live Tech Forum and see if there is any recomendations there that may help. Good luck and let us know how it turns out.
