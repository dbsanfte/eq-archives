Search your windows help file for enviroment variables. There is a walk-thru for setting them.

1 Right click My Computer and choose properties to display System Properties.
2 On the Environment tab, in the list you want to add the variable, click any existing variable name.
3 In the Variable box, type the name of the new variable.
4 In the Value box, type the value.
5 Click Set
