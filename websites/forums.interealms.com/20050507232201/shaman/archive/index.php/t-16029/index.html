dugann
Was pretty fun figuring out this event, we usually do with less than 3 groups as well and concentrate on one gravestone only, then split Mujaki from his friends and beatdown.

The three 'areas' you fight in are near enough to each other (bring up map if you have them downloaded), that we just levitate/sow everyone and run up the sides of one of the buildings, then run over to the wall and you can go across to the next 'area', as long as someone goes to the normal corner where you are ported to it will start the next Mujaki (bring it up on map to see how they are close to each other).

If you end up trying to get to the next 'area' and you overshoot the wall, you will go underneath the 'areas' and you can run up through where the gravestone is that mobs spawn to join your raid in the next 'area'. Before we figured this out we just had people agro stuff from above, die and drag their corpse back to the group through the ground.

Not sure if this is considered an exploit, but saves everyone having to leave and come back and the issues of Etumer not sending your groups to the same Mujaki area. He will run the event 3 times and I think Mujaki is a 3 day spawn (for each area).

Not many know about the event or care (was always up unless we took him down), great reward for the risk, he isnt too hard except for the fear part.

edit- he does despawn.
