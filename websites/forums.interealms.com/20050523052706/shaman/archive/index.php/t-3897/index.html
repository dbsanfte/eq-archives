One thing that really annoyed me with the new UI was that you couldnt remove the granite texture from some windows and especially your hotkeys. I have red, green, blue coloured text on my hotkeys, they're almost impossible to read against that granite.

I hadnt noticed many people changing the graphics for their skins, so I made myself some new buttons. Im sure theres nicer out there, or will be, but if like me you really hate that granite and hadnt found a skin that removes it then I thought I'd share these.

Feel free to grab any, preview the colour by viewing the jpgs, they dont include the buttons for the spell book but will replace just about every other button. Extract the file into your skin directory, you can always copy back the original version of the file from the default skin directory to put the granite back.

If you want rid of the granite almost entirely including on windows that you can't remove the texture from via a menu then plain_dark_backgounds.zip contains, well exactly that.
