Installing a UI is simple.

Create a new folder in your Everquest uifiles folder. Make it something simple - Like Shaman or something (No spaces in the name)

Unzip the UI files to that folder

Load EQ and open the Options window (Alt-O) and choose Load UI. A list should pop up of any UI's you have installed. Make sure Use INI is selected and click Load UI

After a few whirs and clicks and what looks like a screen freeze you should see the new UI files in the same arrangement as your old UI. Just move the boxes around to suit the new windows and you're all set.
