If you set the LoadVeliousArmorWithLuclin=TRUE, then characters using new models (including Vah Shir) will sometimes appear to have no armor.

I'm using new models for my main (barbarian), and when I equipped my Rune Crafter's arms I had bare arms except a little piece of gold scale on each shoulder. But my Rune Etched arms appeared just fine. Once I turned that option back to FALSE, then I could see all the armor again.

Typical, they give new models for all the races but apparently didnt code the Velious armor graphics onto the new models. That's why LoadVeliousArmorWithLuclin is set to FALSE by default when you load Luclin.
