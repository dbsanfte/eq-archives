Kriel Darkbane
I've been solo'ing in Sol B lately, doing greater kobolds. I originally went there hoping to find groups, thinking I wouldn't be able to deal with runners without snare. But I've actually done quite well solo. I've had one runner get a significant distance (and train KPC on me :p ) in 3 yellows of solo exp there.

The most important thing is to make sure you have some room for it to run without getting adds. Of course, when you start clearing an area you can't guarantee this, but clear out the running path with your first pulls. After that it's a lot easier.

Still, you want to keep them from getting far. Things that I've found help: Dot'ing the mob certainly helps, as it's dying while running. But I haven't been doing that as I'm in the JBB stage of life and it would waste mana. So I just root when they get to the point that the next JBB blast would send them running. Usually the root lasts until they die. Also, the puppy helps a lot. He often stuns them when they try to run, and he also finishes off runners really well unless they're at max "running" hp. But the biggest thing is to make sure you position yourself so that the mobs have along enough straight-line running path (when they're hurt and slow, at least) that you get a clean shot at rooting them AFTER they start to run. If I root before they start to run, and have enough room to cast another one after they break root and run, I very rarely have a problem. In fact, the only runner I've had so far was when I DIDN'T position myself such that I had a chance to root after they started to run.

You can cast a root pretty fast, and the mob doesn't really get very far in that amount of time, if it's hurt.

None of this guarantees success, but it definitely helps. I've found it works well enough to solo in dungeons with runners, when I expected it to be virtually impossible.

Chutoi
I always try to set up where the mob won't run into anything before he hits the end of his leash. The mobs have a maxium distance that they will run, and it's well inside casting range of even the epic--just don't chase him. In fact, move a little the opposite direction as soon as they start to run so you can control how far off they get. The DoTs or pet will finish eating him.

Alternately, for most mobs, you can just ghetto snare by pulling another one when the first is about to run. Social mobs don't run if there are two in the camp.

Both methods are quite effective. I prefer the first one when available (depending on types of mobs) as the pet can eat them, and they aren't doing any damage while trying to run.

Good luck! :)
