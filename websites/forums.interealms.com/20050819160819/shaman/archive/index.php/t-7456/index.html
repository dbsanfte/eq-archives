Benorf
open your eqclient.ini file and make sure this is at the top (well, make sure it's true wherever it is

newUI=TRUE

Now look for a file named similar to this UI_Benorf_41.ini (will have your char name instead of mine obviously :)). Now, about 8 lines down or so you need to set the UIskin to default. Will read like this

UISkin=default


If you have an old skin that's crashing and/or not working, you can always check the UIErrors.txt file to get an idea of what file is messing you up.
