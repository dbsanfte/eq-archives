Bigguss
Once you put it in the first time, you don't have to put it in again (it remembers.)

There is a separate file: UI_xxxx_yy.ini (where xxxx is your character name and yy is your server number) that contains the UI info.

While the line:

UISkin=xxxxx (where xxxxx = your skin name)

is in there, there is also a ton of other stuff in there. I'd recommend using /load xxxxx from in-game instead.

Here's the longer version- instructions from one of the mods I'm using:

LOADING THE MODIFICATION

To use this UI Mod, you must edit your "eqclient.ini" file
in your base EverQuest directory (defaults on installation
to "C:\Program Files\EverQuest\"). Find the line in "eqclient.ini" that reads "[Defaults]". Under this line, add a new line that reads:

NewUI=TRUE

Now, log into EverQuest. Once you have gotten to the character selection screen you will see that you are now using the new user interface.

Select a character and enter EverQuest. Now, to load a particular
modification you must type "/load @" where "@" is the folder name that you have installed the modification to. To load my modification, if you followed the directions above to install it, type:

/load modname

And my modification will take effect on your user interface.

-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-

PRESERVING YOUR WINDOW POSITIONS AND OPACITY

WARNING: Doing the below may result in problems involving the spacing and positioning of your windows, or crashes. If you experience problems, use the method described below to change to the "default" skin (ie. the line must read "UISkin=default" to change back to the default skin).

When you load a new UI modification or skin, your windows will be reset to certain default positions. Needless to say, these default positions are very, very stupid.

Instead of using the "/load" command in-game to initialize new skins and modifications, I suggest directly editing your character's UI files. A character's UI file will be named something like this:

UI_charname_23.ini

Find the particular character(s) you want to load the modification and open their UI file. Find this line:

[Main]

Under that line will be a line indicating what current UI modification that character is using. Now edit the line below "[Main]" from whatever it currently says to:

UISkin=modname
