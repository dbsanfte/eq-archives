Emmhim
Hello Phfuzz

As far as I know, tribute effects counts as armor effects. Only the best focus you are carrying will be used, so if you are carrying a spell haste item it will not stack with tribute spell haste.
The same apply for haste so tribute haste will not stack with armor haste.


FT and regen are not focus effects, so they stack up to an upper limit (FT15 and regen 35) so tribute regen should stack with your iksar regen chestplate.
