Shandy Orcbane
Horses are warrior mobs.

In short, all mobs default to "warrior" unless clearly given some sort of other class.

As a f'rinstance, sneak/chanter/caller type goblins in Nurga/Droga don't shield. Taskmaster/cave lord types do.

It simply defines a set of abilities which the mob can use, and does not limit weapons. Find a mob you can give a weapon to, and give it a class-only droppable weapon. It'll wield any of them.
