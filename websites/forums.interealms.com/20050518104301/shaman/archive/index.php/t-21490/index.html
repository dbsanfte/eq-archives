61 Shaman here, soloing PoM wall.

My method:

Summon Scooby, buff Scooby.

Pull with Slow. If you have trouble sticking Slow pull with Malo.

Let Scooby get aggro and back out. Lay on first DoT.

Heal Scooby if necessary.

Lay on second DoT (here comes the rascal for sure).

Melee a bit then root and back off.

Keep an eye on Scooby, heal if necessary. Refresh any DoT that runs.

Repeat.

No, it isn't fast. But it works.
