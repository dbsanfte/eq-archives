Valsuvious
When they did the last patch, they fixed leopard so that when it's cast on a SK it reads as "Leopard Maw SK" when procced so that the sks can use it (something to do with their coding). But, when they went to do the same fix to panther, they misscoded it. What this means is that instead of having an ID of 6779 for everyone but sk (Panther Maw (http://lucy.allakhazam.com/spellraw.html?id=6779&source=Live)) they input 6780 instead (Panther Maw SK (http://lucy.allakhazam.com/spellraw.html?id=6780&source=Live)) and for SKs instead of 6780, they imput 6781 (Banish Intruder (http://lucy.allakhazam.com/spellraw.html?id=6781&source=Live)). This is why when everyone that you see proccing this, you see "Panther Maw SK" and when SKs have it on them, they proc "Banish Intruder" instead. All that this really entitles is that to fix it, they just need to change the ID codes to one before. Should be a simple fix I'd think, but I'm not exactly a programmer.
