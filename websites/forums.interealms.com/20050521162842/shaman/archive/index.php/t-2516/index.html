Bigguss
While on a horse, you med while standing if you're not moving and not auto-attacking. You can cast and use right-click items while medding.

If you're outdoors and plan to be root-rotting or JBB'ing, etc. the horse allows you to meditate the whole time without sitting.

In example 2, the one shield had a +med modifier- the point was that if you're medding while standing, and the +med modifier would help you, maybe that shield is more valuable.

Hope this helps.

I'm pretty sure that while you're moving, you're not medding. I've tested it myself, and I believe it's the popular consensus as well.

To be super clear:

If you are on a horse, not moving, and not auto-attacking... you are medding. You can med while casting and you can med while right-clicking... as long as you're not moving.

If you are moving, you're not medding.

If you are auto-attacking, you're not medding.
