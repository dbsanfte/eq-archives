heres what you need to do

1. use a text editor (I personally use http://www.textpad.com)

2. make a NEW COPY of EQ_animation.xml from the default folder

3. compare both versions of EQ_animation.xml
- the new copy from the default folder
- the one in your skin folder

4. Retrofit your skin changes to the NEW COPY of EQ_animation.xml
