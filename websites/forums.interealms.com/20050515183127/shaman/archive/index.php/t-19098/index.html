When a drug is in testing for the FDA's approval as a substitute for another (mostly a generic for a brand-name), they need data to show in vivo (Latin for "living") bioequivalence (sorry, big word). This means the drugs are equivalent and can be substituted for one another at equivalent dosages (with regard to absorption rate). The null hypothesis for these tests is: Drug A and Drug B are bioequivalent. The final outcome is to either REJECT your null hypothesis or NOT REJECT your null hypothesis. You can never prove the null hypothesis is true. They hope their data will result in values that WILL NOT reject the null hypothesis (which is contrary to most hypothesis testing).

Now in these tests they give the original drug that they want the substitute for to the control set and the substitute drug to the test set of subjects. After all these tests are done they simply don't say "Well, only 10% of the test set showed adverse effects, so they must be the same!"
