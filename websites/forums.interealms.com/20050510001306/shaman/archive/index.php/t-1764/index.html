Oishii
the datarate file lets you control how much info comes through.. the higher you set it, the more information you can receive (if you can handle it)
i think 9.0 is the highest you can set it at.. if you are on dial up, then 5.0 is the most you can handle (5 = 56k or so... i share a dial up connection so we both have to set ours at 2.5 each so we don't go linkdead from one person hogging all the packets), setting it higher won't really do anything until you upgrade to a better connection ^_^;;

another thing you can do is use the /shownames command to turn off everyone's names or just show first names. I usually turn off all PC names and just leave NPC names on.
