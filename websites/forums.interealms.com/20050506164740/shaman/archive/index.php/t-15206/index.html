Grizlor
PoJ
Any Trial = PoV/PoS access, Charm Upgrade

PoD
Grummus = CoD access, Charm Upgrade

PoN
Hedge Maze = LoTT access. No charm upgrade

LoTT
Terris Thule = Torment Access. Charm Upgrade.

PoI
Xanamech = Factory Access.
Behemoth = PoTactics Access. Charm Upgrade.

PoV
Aerin'Darr = HoHA Access. Charm Upgrade.

PoS
Askr Quest = BoT Access. Charm Upgrade.

CoD
Carprin Event = Basement Access
Bertoxx = PoTorment Access. Charm upgrade (only if you havent killed terris yet. It's one or the other)

PoTorment
Orbs + Stomach Event = Saryrn's Tower Access.
Keeper of Sorrows (Requires Bertoxx dead) = Saryrn Access.
Saryrn = Part of Soltower Flag.

HoH
All 3 Trials = Temple of Marr Access. No charm upgrade.

BoT
Assemble Tower key = Tower boss access.
Assemble Agnarr Key = Agnarr Access.
Agnarr Dead = Part of Elemental Flag (No charm Upgrade)

Temple of Marr
Mith Marr Dead = Part of Soltower Access. (No Charm Upgrade)

PoTactics
Tallon + Vallon = Part of Solrotower quest. (No charm Upgrade)
Rallos = Part of Elemental Access (No Charm Upgrade) Also part of Sol Ro (The God) Access (No Charm Upgrade)

Solrotower
Access Gained through death of Saryrn, MM, TZ, and VZ. Access to this zone grants a charm upgrade.
5 Minis = Part of Sol Ro (the god himself) Access. (No Charm Upgrade)
Sol Ro Access Requires 5 Minis + Rallos Dead.
Sol Ro Dead = Plane of Fire Access (Charm Upgrade)

Elementals
Agnarr + Rallos = Access to Earth, Water, and Air. (Charm Upgrade)
Mystical Arbiter = PoEB Access (No Charm Upgrade)
3 PoA Avatars = Xegony Access.

Time
Rathe + Corinav + Fennin + Xegony = PoTime access. Charm Upgrade.

There you have it. Took me about 10 min to type out :p
