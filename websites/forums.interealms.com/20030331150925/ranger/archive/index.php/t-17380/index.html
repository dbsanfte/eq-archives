Amise
You can no longer turn in items to NPCs while invisible, so faction raising would be required to do an MQ with this person. The NPC is on True Spirit faction, which there are only a few ways to raise. I found a quest on Allakhazam which may work.

There is an NPC named Solomen in the room with the undead knights that give DF quests. This NPC when hailed spouts some info. If you say "I AGREE" he will give you a note to take to Erudin to a Camin Rah who spawns in the inn. Giving him the note gives Exp and True Spirit Faction gain..after doing this 6 times, went from dubious to amiable.
Gavving
There was something in a patch message a while back about something they put in to help people reset their True Spirit faction. As this faction is used by the Shaman epic, they've had problems where they need to reset it back... Anyway here's the message that was in the patch message:

" There is a fellow named Moltak in Sol Temple. Killing him gives a negative faction adjustment with the True Spirit faction. BUT, if you have a very, very bad faction with True Spirit, killing him will also reset your faction to a more tolerable level. So don't mess with this fellow unless folks on the True Spirit faction are willing to kill you on sight. "

If the Erudite is on Truespirit faction, a quick trip to Sol Temple to kill this guy might solve the problem. /shrug
