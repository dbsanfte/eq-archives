Rellix
I dunno of this will help, but the formula for heals: (base heal + aa modifier)*2 + focus.

so if you were using Tnargs, the base is 1300. Plus let's say Healing Adept 1, which changes it to 1326. 1326*2= 2652 + ornate arms focus modifier of 20% of the 1326. Therefore 2652 + 265 max is 2917.

In your case: Advanced Healing Adept does 1547 for Tnarg's. If you have ele arms, which does 25%, it will be a bonus of 386 max. So: 1547 * 2 = 3094. 3094 + 386 = 3480.

AKA : 3480 is the MAX we can heal with ele modifier and all AA.

Nytmare
Healing focus is applied to the base heal (1300) separately from AA skills.

1300 * 1.19 {AHA3} + 1300 * 0.20 {max IH4} = 1807
1300 * 1.19 {AHA3} + 1300 * 0.25 {max Marr's Gift} = 1872
1300 * 1.19 {AHA3} + 1300 * 0.30 {max Marr's Blessing} = 1937

1300 * 1.19 {AHA3} * 2 {crit} + 1300 * 0.20 {max IH4} = 3354
1300 * 1.19 {AHA3} * 2 {crit} + 1300 * 0.25 {max Marr's Gift} = 3419
1300 * 1.19 {AHA3} * 2 {crit} + 1300 * 0.30 {max Marr's Blessing} = 3484

So to answer the original question, crits aren't capped. It's just that crits don't double focus effects.
