Sounds like you're plugging the ethernet port of one PC directly into the ethernet port of the second. You need a crossover cable to do this (it swaps the transmit and receive pairs).

Chances are, the guy at the store sold you a standard ethernet cable.

The 'technically correct' setup would be to plug both PC's into a hub or a switch using normal cables. Using a crossover between two PC's is kinda a hack; i.e. not guaranteed to work but usually does.

--Edit--

You're also going to need to assign IP addresses to the cards. I *think* when you set the adapter on the main PC to share the internet, Windows gives it the address 192.168.0.1. So set the second PC to 192.168.0.100 with a netmask of 255.255.255.0 and a gateway of 192.168.0.1 and the DNS servers your ISP gave you.
