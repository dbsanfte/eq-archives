You have no pause after you start casting the spell, so you will try to sit down instantly after you begin casting canni, which ofcourse fails.

Try this:

/sit off
/cast 7
/cast 7
/pause 14
/sit on

Not only will you attempt to cast canni twice (in case you fizzle the first), it also got the pause while canni is casting after which you sit down again.

Hope that helps!

chouka
I had the same problem as you in the past. In order to make macros like that just make sure the pause is after the cast and play with the time. I have had macros where the pause is set really high. The problem is that since you can no longer interrupt a cast by sitting, if the pause is not long enough then the sit command gets rejected (you might read real fast the message this command cannot be used while casting). So just play with he pause time and it will eventually work. BTW dont follow the time settings mentioned up above as it will vary wildly from pc to pc based on the system performance and the connection speed. I use both a pc and a laptop to play and the pause times are different on both.

Good luck

This gets me 3 canni IV's for one casting. Saves me mashing that button between pulls and during down time.

Just remember the pause is in tenths of seconds. Three seconds is 30. One point five seconds is 15, but you would probably want 17 due to lag issues in some zones.

Another hot button I use alot is :

/assist (name)
/cast 2 (malo)
/pause 40
/pet attack
/cast 3 (Turgurs)

I use this alot when I am two box'ing with my warrior, as the pull comes in. The pause before the pet attacks gives me long enough to get the mob positioned and a bit of agro built up before hitting it with the slow. It works great for me.
