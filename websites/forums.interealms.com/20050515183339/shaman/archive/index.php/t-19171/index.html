Steelbear
Yes, Taraddar, the percentage will match the hp/mana, but only for a level 65 character. The adventure stone has a recommended level of 65, so those below that level dont see all the bonuses. They need to use 2% per 'block' of 10 wins.

The Magelo page to update it might also be confusing some people. The red text says a 'value between 0 and 200', but the value you enter is actually a percentage between 0 and 100 - as noted by the % sign by the entry field. Ignore the part about 0 and 200 and just enter a percentage. Each adventure stone improvement is 2%, and each flag improvement to the charm is 10%.
