deria
The last patch changed some things about this:

1) on xp, alt+tab is now more effiicient in that other applications will start faster and internet browsing will not be so painful

2) the in-game gamma setting now works even in windowed mode, so you're not tied to your desktops gamma setting anymore

As for the mouse thing, there is no work around that I know of. I think its just an outright bug in EQ. You can minimize it by activating objects using your "U" key, targeting NPCs using your F8 key, and casting spells using alt+#. You can also mem spells (like travel spells) using /memspellset. Not a fantastic work-around, but it gets me by.

LaughingWoman: Just activate your global mouse(with ctrl+alt+r) before you zone, and use the taskbar to switch to the second window. That's how I do it, works without a problem =)
