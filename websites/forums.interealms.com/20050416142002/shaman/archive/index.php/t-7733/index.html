The best way I've found to keep your system virus clean is to format the machine and install Norton Anti-Virus 2002 or newer before doing anything. Then get all updates from MS. After that download/install anything you need from known safe sites only and NEVER run any version of OUTLOOK.

Bah, the #1 rule for virus avoidance is: DO NOT USE A COMPUTER OF ANY KIND. I personally guarantee you will never get a computer worm, virus, or trojan if you religiously stick to this program.

I use Outlook all the time and I have never had a virus problem. I was infected once, it tried to send mail out using my address book, and I stopped it before any damage was done.

If you do use Outlook,

1. Get rid of the "preview pane" and the message preview features. They can prematurely open a message and give a virus a chance to run on your machine.

2. Turn off the option that automatically opens the next message when you delete a message. Same problem as #1.

3. Get a virus scanner that works as an Outlook plug-in.

4. Tell Outlook never to mail anything to the server without you expressly hitting the button. That way you can look at your Outbox, and if there's anything in there that you didn't send, you can delete it.

If you're really paranoid,

5. Delete mail from unknown sources with "suspicious" titles without opening.

Finally, DO run current virus software (I would avoid Symantec, I am very unhappy with their marketing scare tactics for their virus software) and DON'T open cutesy little attachments like Elf Bowling or whatever, just return to sender with a note saying you don't open executable attachments.

Phern
The problem is probably originating from your other networked computer. You Do not want to have an open share on a network this day in age. That includes your home network that "should" be safe. Klez, Nimda and other virus's look for open shares and install themselves on any computer that hasn't been infected and has an open share to the infected computer.

In your case the 2nd computer probably got infected. Everytime you clean the first computer, the second computer checks the open share, sees it is not infected and reinfects it. This is how these types of virus's spread so fast in work place environments. It only takes one person to open a dubious piece of mail and wham, it spreads through the network.

1) Change your shared to drives to password protected shared drives. Only share what you NEED to share. If it is your internal home network you can use super easy to remember passwords. I am unclear whether you can allow the computer to remember the password in the future. Knowing Microsoft, that will allow the virus to still access the share !?? The password protection will stop the virus and stop or hamper anyone who hacks into your system.

2) Once the shares are password protected, reboot each machine, scan each computer and repair anything the scan finds. Reboot and rescan until you are satisfied each machine is clean. Since you have had so much trouble you might want to try scanning with multiple programs to verify the virus is gone.

This worked at work when we got hit with the first virus of this type last year. We have had no problems since.

Well the fact is that klez pretty much disables and/or hides itself from Anti-Virus software already loaded. You should ALWAYS ALWAYS ALWAYS re-install your anti-virus software after a virus has been found and removed.

Like someone said, you should be running the removal tool in safe mode only. I'd say un-install anti-virus. Boot to safe mode. Run the removal tool. Running the removal tool in normal windows is your problem if that's what you did. Klez is a very clever virus, and it works around things like that. If you boot to safe mode, which loads with no drivers or software, the virus won't be running (since no startup operations are done at all in safe mode) and you can run the tool successfully. After all that, it should find and remove what's left of it. If not, then format. That's the ONLY option. If it does find it and remove, re-install norton, update, and off you go.
