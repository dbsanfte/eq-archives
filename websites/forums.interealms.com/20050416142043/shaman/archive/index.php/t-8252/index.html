Kryscha
Hell yeah there's a difference. MX cards are basically GF2's with a few modifications. With such a small bandwidth and slow speed, you're going to find EQ will get graphic lag quite often.

If you're getting a GF of any kind, always go the pure one (ie Ti4200) as opposed to MX cards. (if you can afford it of course... if not, save up.. the performance difference is worth it)

For example, I was running on a P3 733 with 356MB memory a GF2 GTS. The performance difference between it and my friends GF4MX was negligible. I upgraded the GTS to a GF4Ti4200 64MB, and the difference is AMAZING. I am able to run with several Luclin models on, no graphic lag that I've noticed except bazaar, and all this on a P3 733. (now, if I upgrade the CPU and M/B to a P4, I should notice another increase in performance)

So yes, there is a big difference between the two. But go with your budget.
