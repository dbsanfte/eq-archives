The nice thing is that although you must PURCHASE all the expansions, you don't have to actually use the CDs. Just the CD keys. You can use the existing EQ installation.

If you are cloning your EQ installation to a new computer, here's what I do -

Zip the ENTIRE EQ DIRECTORY and all subdirs.

Unzip in the new location.

Delete all the .TXC (texture cache) files.

Voila, done. You won't be able to uninstall EQ from the Add/Remove Programs dialog in the Control Panel, and you'll have to add your own shortcut to the desktop/Start Menu, but other than that everything should work fine.
