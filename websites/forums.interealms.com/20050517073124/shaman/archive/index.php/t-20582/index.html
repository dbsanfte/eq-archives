If are experiencing lagging issues or video related problems since Everquest went to Directx9 then you can disable the calls by adding the following first line to your EQCLIENT.INI file.

HardwareTnL=FALSE

You will loose the nice new stars in the sky as a trade off :-(

Geforce2 and Geforce4MX seem to be the main cards effected.

This won't correct all lag problems but helps with some older video cards.

weissblau
Barbacus, older cards run DX9 through software - there's a Hardware Abstraction Layer (HAL) that runs the routines on the CPU if the card is not capable of doing it in hardware. Thus, older cards are not "really" running DX9 in the sense that newer ones are. The newest cards implement most or all of the DX9 graphics pipeline in hardware, so you get better graphics without increasing the CPU burden.

If you're running EQ well, though, no worries, just do what you've been doing :-)
