Njorls
how does Mana Preservation stack with Spell Casting Mastery?

Copied from a post on the Cleric board here (http://eqcleric.gameglow.com/ubb/Forum1/HTML/008776.html) :
Spec 200 = 11% saving on every single cast.
Spec 50 = 3% saving on every single cast.
SCM3 = 10% saving on every single cast.

(Ladimor - "SCM3 works on alteration if that is your specialization" ... Well, it works on all spells - not just those in your main spec.)

Spec and SCM are additive effects - so:
Spec 200 + SCM3 = 21% saving on every cast
Spec 50 + SCM3 = 13% saving on every cast.

Now - and this is the new info in this thread - MP3. This gives a 1% to 10% saving - on every single cast *except* CH. This effect is also additive with spec and scm - so:

Spec200 + SCM3 + MP3 = 22% to 31% saving on all but CH.
