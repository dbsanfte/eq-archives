I can't view Magelo profiles with either Safari or IE 5.x on my Mac. On Windows, however, Phoenix (now Mozilla Firebird) does work with Magelo. Therefore, I'd suggest trying the following (I haven't tried any of these yet, so no first-hand solutions):

1. Upgrade to the latest edition of IE for OS X.
2. Try Chimera/Camino or another Mozilla based browser for OS X.

* Opera won't read Magelo profiles on Windows, so I wouldn't bother trying the OS X verison.

Technical stuffs:
I've found that Magelo sends profile pages with Content-type: application/gzip. I'm not sure why, but since the pages are html, it should probably be "text/html". Most browsers won't display "application/gzip" type documents and will try to download them or ask what to do with them, since there's usually no default action associated with that content type. IE tends to cover for strange code/website behavior, which might be it works.

Hope that helps.
