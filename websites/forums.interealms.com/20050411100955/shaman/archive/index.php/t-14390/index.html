Jorlaan
things to check...

1. Are you running a custom ui? If so, go back to default ui and see if problem persists. (probably won't help zoning but might help bags/vendors).

2. defrag your harddrive. Reinstall would seem to not have this problem but I've seen reinstalls that leave the harddrive horribly fragmented because of all the temporary files that get created/deleted.

3) what version of directx are you using? What version of video driving are you using? directx9 isn't supported in eq though I haven't heard of any problems with it yet.

4) Have you do a "thorough" file check to make sure all eq files aren't corrupt?

5) Are you sure all 512mb of ram are good? Long zone times can be memory related if network access is okay.

6) I think I recall reading somehwere about lag with container opening.....I think it was some kind of funky sound issue. Try disabling sound inside eq and see if it the problem goes away.

7) What resolution/texture qualities/"new" models do you have set up in options? Try backing them down a level (for example if on high quality, try medium or low) and see if that helps problems. I found that at 512 memory, my system couldn't handle all luclin models without lag, so I had to unclick a few of them until lag got better. With 1.2 gb of memory, I can have all models on, with high texture quality and do pretty good all in most places (still lag in bazaar and occasionally pok).

8) If you are on Bristlebane, it quite probably is something on the server side. Massive reports of 1017 errors during prime time play.

9) PoK and Bazaar are notoriously laggy for me and my friends. try banking/selling somewhere else and see if that improves things.

Tuga
Since it's a fresh install of Windows there's another direction you might want to look, background services. XP esp. is horrible for wanting to run a ton of unnecessary stuff in the background all the time. Do you have all instant messaging set to off (by default it will be on constantly checking for new messages); how about e-mail, is your e-mail program constantly checking for new mail? Are you running a firewall while playing, hardware or software? Do you have the anti-virus disabled while playing, if enabled is it set for heuristic scaning? For the background services the best site I know of for advice on what and how to turn off stuff is:
http://www.blackviper.com/WinXP/servicecfg.htm
Before you start messing with administration services read all the advice he has and read all the descriptions carefully for each service you want to disable. Before playing with the services though I'd try disabling all the IM, e-mail, anti-virus and firewall stuff and see if that helps. Good luck. :)
