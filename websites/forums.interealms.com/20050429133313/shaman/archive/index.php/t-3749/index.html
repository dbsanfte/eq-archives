Jilariz
I took a Player Window Mod I found and furthur modded it myself. Let me know what you think.

http://users.erols.com/daftagious/pwindow.jpg[/img]

The green bar is Pet HP, the Pinkish bar is Casting Time. The orange hp number is your current hp, and the green is your max hp. Everything else should be self-explanitory.

You can download the file, Here (http://users.erols.com/daftagious/EQUI_PlayerWindow.xml). [i]Right-click and Save As.

If you'd like to use this but don't know how:

1) Must have Luclin installed with the new interface stuff downloaded AND enabled.

2) Goto your "EverQuest" folder. Go into the "uifiles" folder. Make a new folder and call it whatever you want (I named mine "mod"). Go into the "default" folder. Copy the contents of that folder into your newly created folder ("mod" for me).

3) Download my "EQUI_PlayerWindow.xml" file and place it into the "mod" folder (Click yes when asked if you want to overwrite the existing file).

4) Go into your "EverQuest" folder again. Find the file named "UI_YourPlayerName_YourServerNumber.ini". Replace "YourPlayerName" with your characters name and "YourServerNumber" with your server number (duh!). For instance, I'd just find the file named "UI_Jilariz_31.ini".

5) Open it up and find the line "UISkin=default" and change the part after the = sign to the name of the folder you created. Mine would look like this "UISkin=mod".

6) That's it, just run EverQuest as you normally would.

Jilariz
.Zip (http://users.erols.com/daftagious/mod2.zip) for the whole shebang.

If you just want the Player Window (that whole thing on the right-hand side is the Player Window, from the top HP bar down to the bottom of the textual buff listing) and the tiny Buff Icon Window, just use these files:

EQUI_PlayerWindow.xml
EQUI_BuffWindow.xml


If you want to use the whole thing and you run a 1280x1024 resolution, I've included my UI_Jilariz_31.ini (change my name to your name and my server# to your server#). If you don't run 1280x1024, you probably want to leave out the EQUI_ChatWindow.xml unless you know how to edit it and turn the borders back on (I have them turned off, but unfortunately to get rid of the queer look also renders the windows unsizeable).

The orange mini bar on top of the Sta bar is the Pet HP bar. The Percentage label next to it is for the Pet HP% (I figured there's not much use in knowing how much Stamina % you've used up).
