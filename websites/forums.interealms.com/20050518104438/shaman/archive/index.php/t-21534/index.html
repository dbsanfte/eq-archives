Angilia
FYI for those having those repeated booting issues in pok and bazaar try the following.

Add HardwareTnL=FALSE to the eqclient Default section.
I used to crash every 5 minutes in pok and bazaar and on occasion zoning. Made this change based on EQwebsite info and no more issues.

My setup is AMD 2600 with a geforce 5200 card and 512 memory. So not a shabby system.

HardwareTnL=TRUE/FALSE - This will force Hardware Texture and Lighting either on or off. This is a line you would manually input into your eqclient.ini, and only for troubleshooting purposes.

They also suggest
VSYNC=1

VSync=0/1 - This either tells EverQuest to wait for the vertical syncing of the monitor before updating the display or no. This can prevent visual tearing (if set to 1). Recommended default is 1. This line is something that would need to be manually inputted into the eqclient.ini, as the game does not provide it by default.
