which one is better? Spirit Caller gloves or coldain skin gloves?
Seredith
Coldain skin have a more rounded set of stats, Spirit Callers slightly more ac, minor stat boosts.

Coldain if you want HP/mana, Spirit Callers if you are concerned with ac dropping below a certain point or youre gunning for a matching set of armour, bear in mind there are higher ac gloves if you need them, Crystal Chitin (20), Barbed Ringmail (19), Gloves of Fungal Flesh (16) for example.
