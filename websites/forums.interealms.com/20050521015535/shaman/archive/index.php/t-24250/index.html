LaughingWoman
Well, I came back, but only because I got too bored at work, and EQ is still better than working, at least. In short, my advice is to clearly define what you want out of EQ before you go back.

Specifically, you KNOW the extents of EQ's entertainment value, you know what you did and didn't like about it. Make sure you set yourself up to do only the things you liked about it. If you loved raiding, focus toward a raiding guild - and pick one within reach so you don't have to level grind to get in (if you hate level grinding).

I have some already that go into a lot of detail about how I set up guidelines for myself when I started again, because I have a tendency to get so caught up in the game that I don't realize I'm making myself hate it all over again.

So far I'm having a lot of fun just questing, soloing or grouping with random people, and doing my own thing. Doing a lot of stuff I didn't make time for before. It's fun, in a slow, EQ kind of way. Expectation and planning are the keys.
