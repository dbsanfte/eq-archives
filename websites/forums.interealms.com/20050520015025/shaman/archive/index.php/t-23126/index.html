Budana
I was wondering if people find it easier to two box on one machine using Eqwindows or using two machines. I have a P4 3.0 @ 3.6 with a Gig of ram and a ATI 9800XT and for a second box use an AMD 2500 @ 2.1 also with a gig of ram and msi 9800pro @ XT speeds. The problem I have is that two keyboards take up alot of room and are just plain hard to watch both characters. I also feel like I lose alot of dps on the character that is playing 2nd box. With Eq windows it sounds like you have both characters on one screen and just use the one mouse and keyboard. This sounds like a much easier way to control both boxes to me. My question I guess would be how are others two boxing and what have you found to be easier one PC or two?

Yes 2 keyboards and mice do take up alot of space but i can reach over hit a button on the other keyboard and move back to main main keyboard alot easier and faster than I can switch between them. Often times I'll actually be doing things at the same time on both keyboards even something that would be impossible to do on 1 computer. It probably depends on the classes. I rarely melee much on my bard on raids so it's essentially like playing 2 casters. I've done both and find that 2-boxing on a single machine with dual monitors is much better. I use WinEQ pro and EQPlayNice pro, (which supports dual monitors). I set my hotkeys to ctrl-z to switch between windows and the \ key to toggle quarter screen / full screen mode. Works like a charm.

Furree
I box 3 and I find it much easier to have 3 keyboards and 3 monitors. Certain things which sound so simple aren't when you can't see what one of your toons is doing. When you're autofollowing across and aggressive zone with invis on , it's a serious pita to not know that invis is dropping on your ranger. There are a number of other things like this but that's the one that comes instantly to me.

Also since the change to the login program I can't run multiple eq instances very well at all. Even my best machine which used to do this without issue becomes really choppy when i do this.

I would also think that if I was alt tabbing around I could easily end up hitting hot keys on the wrong toon. I've become very used to the physical location of each character's pc.

That all said depending on what the "bot" is going to be doing you might be able to get away with the 2 on one machine idea. If you're raising a cleric bot who you intend to use as a one button wonder than I can see it working just fine. If you're going to be doing anything moderately dynamic I think it will be hard without multiple machines.
