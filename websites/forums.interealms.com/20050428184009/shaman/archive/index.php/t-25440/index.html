Valgoth Ulv'Sagart
Did you remove Questminion? or did the new version of CTMod remove it?

I installed the new version last night my desktop (been using on my laptop for a while) and I noticed that QMinion wasn't part of it any more.I removed it from the latest version of my UI remix. The only functionality I was really using it for was the quest level indicator in the quest log. All other features where turned off. Some guildies found it was causing the game to completely lock up for them (they added the stand alone's one by one and noted it started locking up immediately after putting QM back in the AddOns folder), so as a courtesy to them I just removed it. Since CT now includes the quest level indicator, I saw no reason to keep it in the remix.

Besides, there's a better mod out there with similar functionality called "[u]MonkeyQuest[/url]". I haven't used it that much, but it also logs completion date/time, level, reward, exp, etc. You might want to give that a try. You should be able to just copy it into your AddOns folder without any conflicts from any others I've included in the distribution.
