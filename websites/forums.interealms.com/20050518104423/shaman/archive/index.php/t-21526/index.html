veeshanMurtak
There is a general problem with all classes that give up part of their primary ability to gain a secondary ability (paladin healing is a good example). Since they cannot use both abilities at the same time they get to choose which ability they want to be bad at. When you can usually optimize groups to have every needed ability performed by a specialist class being able to do two things worse then these specialist classes is useless.

The solution is obvious (well, to me it is) - instead of being able to do X or Y ability 75% as well as the specialist you should be able to do X and Y 50% as well as that class (or 75/25 or whatever). So far this has been attempted via recast timers on spells. This does not work though as everytime you give that class a minor upgrade to that ability by giving them a new spell you nearly double their ability since they can just mem both spells. Thus you end up going from 75/25 to 75/40.

These abilities should just be AAs with near instant casting times instead (and cheap AAs at that if they are to be used for class balancing). That way a new expansion can just improve the existing AA ability. It would also make class balancing easier overall in that you would not need to worry about casting times of secondary abilities affecting the time available for primary abilities.

Now, not every secondary ability needs this. Buffs are fine as is. Debuffs are probably too much trouble to convert to this system. Rez is fine as is.

But paladin group heals should be a 0.5 second cast time AA. Cannibalize should be an AA. Zefeer's should be an AA. Ranger nukes should be AAs. And so on . .

Of course a lot of the abilities would have to be adjusted in magnitude to make up for nonexistant cast times or mana. And some abilities (like healing) should probably still be available as spells to supplement the AAs.

But overall I think this system would work much better then the current balancing nightmare.

Thoughts? Comments?
