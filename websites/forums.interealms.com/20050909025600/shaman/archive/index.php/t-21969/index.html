Your page file is probably on drive c:. It windows it set up manage it automatically, it will grow or shrink if necessary (I think). With so little space on drive c:, it probably doesn't have room to grow if it needs to.

Definitely move eq over to drive d: if that's not a problem. That should free up quite a bit of space on c: :-)

Also look for other programs that you don't need anymore in your add/remove programs area and get rid of em :-)

After you do that, then defrag and you should see some improvement!

I usually move the swap file onto the d: drive, helps greatly with defragging (since the system doesn't restart the defrag everytime it writes to the sway file). If you don't have a d: drive, have at least 150% of your max swap file size empty on your c: drive (I think standard is 4 gb, so 6 gb would be desired).
