Just "making a copy of the default" folder is the absolute worst advice anyone can give regarding the use of custom UI's. There are files in the default folder that are updated frequently and will cause anyone who just "copied" their default directory endless headaches.

The best thing to do is to create an entirely NEW EMPTY folder in the UIFILES folder and place the new windows/ui files you want to use in there. Then while in game type "/loadskin dirname 1". Any required files it doesn't find in that new custom directory it will load from the original default directory, but it will use the new windows you put in the custom directory as long as they contain all of the required XML data objects.

However, I'd suggest thoroughly perusing eqinterface.com and reading any of the extremely detailed guides, how to's, tips & tricks, and do's and don'ts before taking any advice offered here.

The UI I use is the most unbreakable I've found in a long time (haijacks "Master Control"). Even so, I've had to make a few minor updates to it about every other patch to keep it working. Mainly the EQUI_Animations.xml and occasionally some of the graphics files. I generaly watch the default EQUI_Animiations as well as the EQUI.XML file for changes and updates, then make sure I port those changes over to my files.
