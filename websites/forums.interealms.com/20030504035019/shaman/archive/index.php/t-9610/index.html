I'm guessing the ratio is where they want it balance wise. There is not rule that says the newer spells have to be better mana/hp ratios. Just cause Necros are that way doesn't stand to reason that shaman should be. What you are getting from each higher level canni is faster transfers. We can heal very fast at high levels and very efficiently so you want to be able to chew through lots of hps. Using cann 1 maybe technically more efficient but it would take forever to convert thousands of hps that a 60th shaman (for example) would have, both in base hps and what is healed.
I never wanted to get can 2 since it was so hard to do the quest. By the time I was about 50ish I NEEDED cann 2 because it was taking me so long to get mana with cann 1.
