Start eq up and leave it at the "I accept" screen.

Ctrl + shift + escape to bring up task manager.

Select processes tab.

Scroll down to eqgame.exe.

Right click select "Set Priority-->Below Normal"

Click yes on the whiny warning messagebox that pops up.

Your problem should be solved :)

Been playing EQ lately using the Below Normal Priority Process (thanks for that suggestion, works great) but had grown tired of manually setting it each time. So I figured out how to do it through a batch file.

Just make a .bat file and enter in this info, adjusting path names where necessary:

Start /BELOWNORMAL C:\Progra~1\EQW\eqw.exe

Yeah, I use EQW. If you don't, you'll have to add a bunch more arguements to make it work (if it does..I dunno, tried it once but didn't work and I don't really care :p). If you log on two accounts on the same pc, you'll probably want to change it or leave it at the Normal setting.

Now I just double-click my shortcut to the batch file and off I am, with actual control over my computer again. Outlook opens up mightily fast - the same as it would if I weren't playing EQ. When I need to look up a map or check on my Guild message boards or any other thing, I don't have to cry because my PC doesn't sit there and act like an old sick dog anymore.
