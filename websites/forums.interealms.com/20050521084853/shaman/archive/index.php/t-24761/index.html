Loramin
This is old news. As far back as I can remember (I started right when Velious came out, and a friend of mine went on a language learning crusade not too long after) people have been wondering if there was a secret language they didn't know about. But then it turned out that (surprise surprise) SOE screwed up. In some documents they claim that there is both Dark Evlish and Tier'Dal. But that would be silly, since the two are the same thing, so they only implemented Tier'dal. End result: the game only has 25 languages, and if anyone tells you otherwise ask them to talk to you in Dark Evlish.
