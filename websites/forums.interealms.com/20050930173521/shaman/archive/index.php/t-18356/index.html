When I wanted to solo at 59ish I would head to Dulak and solo on Dandolak's Run or Oceancrasher. Oceancrasher was a bit harder, so that might work well for a 59 duo. The last (stormwave I think) has cleric and caster mobs, so if you get an add with a cleric it can make for a rough time.

Droga is also good at 59. Drops there are good. Veksar can be ok, but mobs there have high resist ates and WAY too many hp. You might be able to do PoI or PoV without too many problems. Adds would make any situation there sketchy at best, but still quite doable.
