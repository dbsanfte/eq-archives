Valgoth Ulv'Sagart
Not difficult at all as long as you have some sort of audio/video production software that performs that type of operation.

I have several myself. Sony Vegas, Sony Sound Forge 7.0, Pinnacle Studio 8.1, Windows Movie Maker 2.0 (which is FREE!). A/V production is yet another hobby of mine. :)

To simply load the MPG and add audio, Windows Movie Maker would be your best bet since it costs absolutely nothing, and it's really, REALLY easy to use. If you're looking for something with a few more features, Sony's new "Screenblast" software is pretty cost effective and has a decent collection of features and functionality.

Enjoy!
