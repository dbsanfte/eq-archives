What I found was that my machine was not CPU limited, it was memory limited. The freezes was when the machine was thrashing using virtual memory. Watch your hard drive light to see if this is the case for you. If this is the case, drive your tuning options for memory rather than CPU.

I also run 2 accounts using EQ windows, (ranger/shaman mostly) and here are a couple of tips I have found..

Use the viewport command. My screen res is 1280 x 1024 but I set the render viewport to 900 700 or somewhere similar. This allows me to push the interface all around the border and eases the strain on the graphics card. Kinda also reminds me of the old original interface. The preferences are remembered per character, so you could set up your bot account to a really tiny size and the main one to a fuller size if you so wish. For those that are wondering, the viewport command is /viewport x y w h where x is the distance from left, y is distance from top, w is width and h is height.

The first thing I do when logging on the bot account is press F9 once for the top down camera. This causes a noticable speedup as the camera has very little detail to sort and zbuffer.

I hope that helps :)
