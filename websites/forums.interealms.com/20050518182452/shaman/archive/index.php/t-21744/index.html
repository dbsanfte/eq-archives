Shabura Naarteban
They are changing the "Category 1" (for shaman, this is Nihil) and "Category 2" (for shaman, this is Daluda) spells to work like the "Category 3" (for shaman, this is Trushar) spells.

Taelosian Geomancy stones, as we knew them before these changes, are becoming Taelosian Geomancy Stone Jelki. The other 2 correspond to the other 2 categories of spells, and work in exactly the same way. Although I'll admit that I don't know which is which at this moment, it should be fairly trivial to find out - whichever Umbiki drops is category 1, and whichever Chavoor drops is category 2.

No more problems with spell drops always being for the wrong classes (*cough* BST *cough*).
