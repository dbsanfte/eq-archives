Not really a bug. There is a server (or set of servers) for LDoN instances. This server(s) may be spread across multiple 'world' servers for all we know. When the instance server has a maximum number of instances running, for whatever reason, it cannot create the instance when you create your mission. You will not be able to enter your dungeon at any point in the 30 minutes because it was never created and is not waiting for you!! Quit the mission (does not count as a loss as you never entered) and get another.
