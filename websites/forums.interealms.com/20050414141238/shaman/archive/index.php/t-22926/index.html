Found the problem(s). Previously I was using 16bit depth on both desktop and in game, but after I stopped playing in March I had set my machine up to do video production work (along with other graphic stuff). I had changed my desktop to 32bit so just naturally set EQ to that as well when I set up my custom UI.

I also disabled HardwareTnL and set VSync=1 in the eqclient.ini. These three things brought my FPS back to normal, though some of my coloring seems to be a bit off. Not bad enough to ruin the game though.
