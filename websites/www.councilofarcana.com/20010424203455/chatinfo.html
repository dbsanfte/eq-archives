Chat Information

The Council of Arcana Chat Room is an Internet Relay Chat-based chat channel. Like all IRC channels, IRC-type commands can be used for functions that you wish to execute. You can access our chat room via the Java Chat page or using an IRC client like mIRC. For client users, the chat channel #CouncilofArcana is on the chat.planetz.net server (port 7000 or 6667 will work). Java Chat users can enjoy the full functionality of chat with great simplicity of interface and use. Higher-end users will want to consider mIRC because of its display functions and control, but it will be confusing to those that don't already have experience with IRC.

Some Commands

/Nick [Newnick] - Changes your nickname to the one you specify in [Newnick]
/quit - Quits you from the Chat Channel.
/me [something] - Emotes your "something".
/float - Floats the Java Chat display window.
/unfloat - Returns the Java Chat display window to its anchored position.
/NickServ REGISTER [password] [e-mail address] - Registers the current nickname you have.
/msg [name] [something] - You send a private message to [name], containing [something].

Other notes

The registration of your nickname will ensure that only you can use a certain name. I highly recommend that if you intend to visit and use our chat room frequently, you consider registering your name. This is done easily by /NickServ REGISTER [password] [e-mail address], where password is a password that you make up, and e-mail address is an accurate e-mail address. It will register the current nickname that you have. You can also /NickServ help for more information.

Users over Javachat will want to double-click on a person's name to open a private dialogue box with each and every individual with whom they want to privately chat, so as to minimize any confusion.
