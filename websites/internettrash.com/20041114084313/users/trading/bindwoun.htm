ind Wounds is a great helpful Skill for non-healing Classes and it is not very hard to increase it. So you don't need to put any Practice Point in this Skill. To this Skill all you need is a lot of Bandages.

This Skill allow you to Bind 1 Hit Point per every 4 Skill Points. So when you will be a Master Binder you will be capable to Bind at least 25 Hit Points. But you can Bind only till the half Hit Points.

Be careful because when you're Binding someone, if the person move you will be Stuck in position, then just Sit down and the problem will resolve, so if someone is Binding you DON'T MOVE or he will be Stuck.

The Begging Skill will raise like this : (Level x 5) + 5; but for certain Classes the cap is 100.

So at level 20 you will become a MASTER BINDER :)

The best Character to become a Master Binder is a Magician because they can Summon Bandages needed and a good Tailor.
