ishing is a very useful Skill for a lot of Characters but it will take you a lot of time. It is not necessary to put any Practice Point in this Skill.

For this Skill you don't need a lot of things, just go buy a Fishing Pole, and a lot of Fishing Baits.

You can do a lot of things when Fishing : it is a good Skill for a Caster because they can Meditate in the same time, you can Beg if you are near a NPC. You can also Fish when you cast a Spell. When you're waiting for a Boat you should Fish to gain an extra few Gold.

Your Fishing Skill increase like the most Skills, with this formulae : (Level x 5) + 5.

So at level 20 you will become a MASTER FISHER :)

The best Character to because a Master Fisher is a Wood Elf, a Druid, a Ranger or a Bard because they can Forage Fishing Grubs needed to Fish.
