Resources: Updated Spell Icon for Brells

This will update the spell icon for the Brell's Mountainous Barrier [aka Brell's Mountain of Poo] to something easily recognizable thus making it difficult for Paladins to stealth Brell you. It should be noticed that this will also change the spell icon for other spells that used the dopey pink heart spell icon. This only works with the new interface.

Brell's Icon Before...

Installation:

  1. Download one of the Brell's updates below.
  2. Decompress this archive to someplace easily accessable on your computer, such as the windows desktop or My Documents folder.
  3. Browse to the EverQuest\uifiles\ on your hard drive.
  4. Copy the new file spells03.tga file to the folder for the skin that you are using.

Note: The file original-spells03.tga is can be found in the EverQuest\uifiles\default folder should you wish to revert back to the original.
