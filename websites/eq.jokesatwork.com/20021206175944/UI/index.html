Kruull's Crappy User Interfaces

Quick HowTo:

In your "Everquest" directory there is a folder called "IUFiles". In that folder there is a folder called "Default". Make a copy of the Default folder. Name this new folder whatever you want your skin to be called, for this example we will us Kruull. Download the new zip and unpack it in the Kruull Directory and click yes to overwrite all files when prompted. Once you load into the game type "/loadskin Kruull 1" (without the "").

Note: this will not work if you do not have the new UI turned on.
