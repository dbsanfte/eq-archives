Shadowbane In-Testing Notes 1.0.2
2003-04-14 10:08:43
Story by - Aidden


Shadowbane In-Testing Notes 1.0.2 was posted over at the Chronicle

This Patch is scheduled to begin internal Quality Assurance testing next week. This testing is expected to take the entire week.  Earliest estimates for a Live server release of this update is the week after next.

FIXES

- Fixed a number of mobs that were dropping extra equipment they weren’t supposed to have
- Fixed a number of NPC’s that would spawn inside graphical assets, trees, counters etc.
- Fixed a number of floating building problems
- Fixed a number of typos and spelling errors for NPC names, titles and dialogs
- Fixed the outfits for a number of NPCs that were missing shoes, sleeves, etc.
 - Fixed a number of NPC spawn radius and placement problems
- Fixed a problem where powers with quick cast times could cause an "infinite" casting animation loop
- Fixed a number of zones where NPC’s were immediately respawning
- Fixed a number of zone entries that would keep some mobs from spawning
- Fixed a few mobs that weren’t assisting when called by friends
 - The Undead Hunter Discipline Rune is now dropping
- Fixed the mouseover descriptions for chants to better indicate that they are chants
- Fixed a couple of zones in the bandit wetlands that was partially in the water
- Fixed the text string for the Wizard’s ’Incantation of Augmentation’
- Fixed the text strings for ’Psychic Healing’ to match the function of this spell
 - Fixed the description for the ’Tough as Nails’ trait to properly indicate the bonuses it confers
