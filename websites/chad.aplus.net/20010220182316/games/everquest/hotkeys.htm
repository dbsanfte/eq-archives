For right now, I am just going to show the hotkeys that I am currently using with my mid-teens level ranger.  I recently changed them around completely, and I am still fine-tuning them, but this might give you some ideas...

To set up your own personalized command, go to the set of Socials that are provided as defaults with the game.  From there, you can right-click on these and change them to whatever you would like, as long is it could normally be entered as a text command in the chat box.

You can now set this command as a hotkey, and place it in one of your banks!

Used strictly for fighting. Includes miscellaneous spells, and a button to quickly enter meditation.  Medding is ended with the Esc key.
Traveling bank. This bank isn't used much right now, but leaves room for growth!
Some miscellaneous skills I hot keyed, so I can maybe practice them while medding, or riding the boat. This bank is occasionally used, but still being fine-tuned.
