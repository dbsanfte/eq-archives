Welcome to TWiki, a flexible, powerful, and simple Web-based collaboration platform. Use TWiki to run a project development space, a document management system, a knowledge base, or any other groupware tool, on an intranet or on the Web.

This is the place to learn about it, download it, and try it out for yourself...

 
The official TWikiTM
site is http://twiki.org/

TWiki Site Tools

Site tools include utilities for navigating and keeping up with site activity on individual TWiki webs. Preferences can also be configured, by web or site-wide. You are currently in the TWiki web.

  • WebSearch: Full text search of the TWiki web
       (extended options in WebSearch)
  • WebChanges: Display recent input on the TWiki web
  • WebIndex: List all TWiki topics in alphabetical order
  • WebPreferences: Preference settings for the TWiki web
  • TWikiPreferences: DLC site-wide preferences
