Syntax Rules

  • %SPACETOPIC% puts spaces into the topic
  • %SPACEWIKI{WikiWord}% spaces out the enclosed WikiWord

Examples

  • %SPACETOPIC% should yield UAFS Plugin
  • %SPACEWIKI{WikiWord}% should yield Wiki Word

Settings

Plugin settings are stored as preferences variables. To reference a plugin setting write %<plugin>_<setting>%, i.e. %EMPTYPLUGIN_SHORTDESCRIPTION%

    • Set SHORTDESCRIPTION = This is a set of special rendering routines for UA Fort Smith.

  • Debug plugin: (See output in data/debug.txt)
    • Set DEBUG = 0

  • Exceptions. If these are found in the SPACETOPIC or SPACEWIKI, then that part of the topic is ignored:
    • Set EXCEPTIONS = TWiki, Mc[A-Z], Mac[A-Z], MySQL?

  • Remove Comments. If this is set to 1, it will remove all HTML Comment tags (<!-- to -->).
    • Set REMCOMMENTS = 0
