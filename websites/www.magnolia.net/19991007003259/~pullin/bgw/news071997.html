This site is devoted to Baldur's Gate, the exciting new game being developed by Interplay and Bioware. Baldur's Gate is a role-playing game set in TSR's Forgotten Realms role-playing world. The current estimate for its release is December.

07/25/97
Some people have e-mailed me asking where they can find this game. This game is still in development and probably will not be released until December at the earliest. Bioware has made statements to the effect that there will be a demo in October. Please remain patient as this game looks well worth the wait.
