Defense

To raise defense quickly as a non-monk, try fighting hand-to-hand for a while. Of course it will only raise defense to the cap which is (5 * level). But still its useful sometimes. - Steve

Swimming

Here is the way to increase swimming. Find a place that is only knee deep, but the game still thinks you are swimming. Then find a wall and start running against it. Once you are swimming forward, into the wall, press "enter" and this will start the auto-run. Then go do something else, while you gain skill. - Phlashbak
