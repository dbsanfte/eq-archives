Factions:

Below is a list of known faction hits for killing certain MOB's. Not all creatures in the game carry a faction hit, some carry a bad one, and some can be used to boost your reputation with races that normally would not like you. The list below was compiled by players, and partly by us here. Feel free to email us with any corrections to this list. Also if there is a certain faction name that you dont recognize, check our Faction Description page for a possible answer.
