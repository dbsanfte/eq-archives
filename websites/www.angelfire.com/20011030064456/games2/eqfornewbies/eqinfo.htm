    		• EQInfo

    EQInfo is a nifty little program designed by Mattias Bergander. Some of the main features:

  • Mana calculator
  • Zone information
  • Zone landmarks
  • Zone mob information
  • Monitor your character using eqlog
  • Server status
  • Links
    The list goes on and on. This program is a wealth of information, all packed in to just 270 KB.

    Download it here for free. The program requires that you have Java Runtime Environment version 1.3 or higher on your computer. This is a little biggish (5MB) but can be downloaded for free here. I highly suggest you take the time to download both of these components, as this program is absolutely phenomenal.
