i like monkeys

what is i like monkeys?

i like monkeys is a semi-anonymous internet story that is endlessly amusing.
i like monkeys is a somewhat strange, vaguely literary e-zine run by padgett arango. the e-zine contains stories that range from the humorous to the annoyingly pretentious.

how can i interact with i like monkeys?

stories can be selected from the menu at the bottom of the screen.

if you would like to contribute to i like monkeys, please send submissions to arango@bigfoot.com
