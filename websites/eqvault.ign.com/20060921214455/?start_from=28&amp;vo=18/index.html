Tuesday, 31 August, 2004


  Exclusive Omens of War Alternate Advancement Abilities  [ Comment ]
Print News | Mail News | Category: Features
Today we have two new OoW Alternate Advancement Abilities from SOE. This round they are for the Paladin and Rogue classes:

Paladin
Steadfast Will - This ability grants a chance to endure what would otherwise be a stunning blow, from any angle, without being stunned.

Rogue
Nerves of Steel - Each rank in this ability gives rogues an increasing chance to remain hidden when a spell hits them. At the highest rank, a resisted spell will never break Hide and there is a good chance that the rogue will remain hidden even when a spell successfully hits them.

Neither are as class-defining as a few of the earlier ones we posted, but both look very useful in some situations.

Aug 24th In-Testing  
Print News | Mail News | Category: EQ Live
The test server has a new update today which includes player-title features.

August, 24th Update 10:00am

Most changes lately have been related to the upcoming expansion and so aren't visible on Test. Here are a few things to look out for though.

- Changed the way no dispell spells work server side. You should not see a change.

- Changed the way MGB spells work server side. You should not see a change.

- Made server side changes to /targetgroupbuff. You should not see a change.

- Items with multiple "All Type" augments in them should not have all the augments pop out if you remove the top most augment.

- Wake the Dead pets should once again cast spells on Test.

- LDoN: Item Collect adventures should once again work on Test.

- Fixed a graphical issue with the guild management window when you resized it.

Custom PC Titles. The details are:

- The window can be brought up using the "+T" key combination, this combo can be remapped via the options menu.

- The window should present a list of all titles available to the player and they can select any title listed.

- Selecting a title in the list should change the title shown above the player's head and appear in yellow.

- Clicking the "Change Title" button will change the player's title, turning it blue and updating all the players in the zone so they see the new title.

- Choosing "Clear Title" should remove any title from the player.

- GM granted titles should appear in the list and are selectable

- All players should retain their current title when they login, so it should not disappear because we have moved to a new system.

If you want to check it out login to the Test Server.

-- @ 2:20PM Tue 24 Aug 2004
