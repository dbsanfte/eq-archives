How Do I install an Interface Mod?

1. Download your chosen mod
2. Find the "UIFILES" folder in your EverQuest directory (c:\everquest\uifiles) inside there will be a folder called Default*
3. Create another folder beside Default, for our example we will call it "CUSTOM"
4. Unzip all of the files from your download into the folder "CUSTOM"
5. Enter EQ
6. In game type "/loadskin CUSTOM true" the screen will look like you're zoning for a moment. The parameter "true" used in the "/loadskin" command will keep the UI from resetting all the boxes back to default posistion. If for any reason you want to go back to the default interface type: "/loadskin default true"

*The default folder CAN NOT be altered in anyway.. any changes made to this folder are over written at Start-Up of the game. THATS IT! You are Done!
