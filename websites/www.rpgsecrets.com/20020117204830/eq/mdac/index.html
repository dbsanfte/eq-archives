Our program requires Microsoft’s Data Access Components (MDAC). We do not make MDAC part of our basic install because it is rather large and comes with many applications (such as Microsoft’s Office 2000) and may already be on your computer. MDAC may not be installed, however. If our program does not run properly and gives you an error about not finding an object, you may need to install MDAC. We use MDAC version 2.5. Search your computer for MDAC_TYP.EXE. If you find MDAC_TYP.EXE, run it to install the MDAC components. If you do not have it on your computer (or if you have an earlier version), MDAC  can be found at:

 www.microsoft.com/downloads.

You must use Version 2.5 in order for this program to run properly. We discovered a bug when using version 2.6 - this will be fixed soon.
