Current Projects as of April 2, 2001

Here is a list of all the current projects being worked on, and soon to be out for the The Unauthorized Secrets of EverQuest! Our new program has been released! You can view some of the functions and screen shots here until we create a page just for the USEQ Items Database Program::

The new item database program will feature all of the items in the game. As you can see from the pictures below, you will be able to simply sort by any column by clicking at the top. So if you were to select Warrior, All Zones and hit enter, it will show you all the weapons a Warrior can wield alphabetically. Now if you want to know which of those weapons is 1 handed slash, simply click on Type and it will break it down by Weapon type. Lets say you want to know which weapon is the best for hitting, just hit Ratio, and it will show you in order from best to worst, which has the best Damage/Delay ratio! This program was released on March 31, 2001!
