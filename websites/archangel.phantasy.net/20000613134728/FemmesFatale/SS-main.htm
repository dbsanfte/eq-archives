Screen Shots will go here....

If you have any screen snaps you would like others to enjoy... send them to Gjiintari.

Information on sending:

  • When you send, give each one a caption and if you want a small narrative describing the snapshot
  • Please use JPEGs (.JPG) only to upload.  Everquest stores the pictures in bitmap (.bmp) format; you must use a graphics package to convert the screenshot first.  You can find a good packages at http://www.tucows.com , http://www.download.com , or http://www.shareware.com.
  • Please use the 640x480 resolution when taking portraits for upload.  This will help reduce the size of the files and also make the picture easier to view in its entirety for those who run a lower resolution.
  • Please try and keep the file size under 100k.  If you followed the recommendations above the picture should be well under that size.
  • Please do not upload pornographic or other questionable materials.  This feature of the site is intended for character portraits.  There are many mailing lists where pornographic submissions would be quite welcome.
  • Enjoy.

-Gjiintari
