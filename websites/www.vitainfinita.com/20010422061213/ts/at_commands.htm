Modem AT Commands

These commands may added to the initialization string (Extra settings) or used in a Hyperterminal window. The "AT" only needs to be entered in an initialization string once - at the beginning.  Example:

{Note: There are no letter O's in modem strings.  It is always a Zero.}

  • AT&F : Resets any Hayes compatible modem to factory defaults
  • AT&F0 : Resets hardware modem to factory defaults
  • AT&F1 : Resets software modem to factory defaults
  • AT~S : Displays modem's serial number
  • ATS7 : Sets the "Wait for carrier" time.  Default is 60 (seconds)
  • ATS11 : Sets the dialer speed (default =70).  Example: ATS11=70
  • ATS6=8 : Disables dial tone recognition
  • ATS32=34 : Disables V.80 protocol (video transfer)
  • ATS32=66 : Disables V.90 protocol
  • ATS15=128 : Disables X2 protocol
  • ATM0 : Disables modem speaker
  • ATM1 : Speaker is enabled until carrier is detected (default)
  • ATL0 to ATL3 : Sets the volume level of the modem speaker
  • A/ : Repeats the last command entered
  • AT&V : Displays the stored and active modem profile settings
  • ATI4 : Displays the stored and active modem profile settings for USR Courier/Sportster
  • AT&W : Write current configuration into stored profile zero
  • ATDT : Dialer.  Example: ATDT { phone number }
  • AT + FCLASS = ? : To determine the type of fax modem. Class1 or Class2 ?
  • , : A comma is used to pause for 2 seconds
  • W : Instructs the modem to wait for a dial tone before dialing
  • 9 : Used to gain an outside line in PBX boxes
  • ATA: Answers an inbound call
  • ATH : Hangs up
  • AT16 : Begins an internal analog loopback
  • AT17 : Displays product information
  • AT19 : Displays PnP identification
  • ATI11 : Displays modulation standard being used
  • +++ : This is the Escape Code Sequence - the modem goes from online to command mode
