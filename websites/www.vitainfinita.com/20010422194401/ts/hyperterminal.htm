Hyperterminal

You can find out if your modem is functioning as it should by running this simple test in a Hyperterminal window.  

Goto Start | Programs | Accessories | Communications | Hyperterminal | Hypertrm.exe.   When the window asking for the Connection Description pops up press the Escape key or click on "Cancel".  This will give you a usable cursor in the Hyperterminal window. If you do not get a blinking cursor then you need to reinstall the Hyperterminal. Do so in Add/Remove Programs | Windows Setup | Communication. Be sure to reboot between deletion and reinstallation, and reboot again after reinstallation.

  1. Turn on the Caps Lock
  2. AT&T1
  3. Enter  (This starts a loopback and will report the modem's info to the terminal screen.)
  4. +++  (Frees the keyboard for additional commands.)
  5. AT16  
  6. Enter (This will echo either "ERROR" to show that no loss of data occurs in the loopback or a string of numeric values denoting the loss statistics.)
  7. ATH  
  8. Enter  (Hangs up the modem.)
