The Scripter



The Scripter Window

Using Scripts: Open up EverHack Scripter in EverHack. Click on open and select the script you want, Set the hot-key to press to activate the scripts (Dont use f1 for some reason EQ wont allow that to be used by any other program)

Writing Scripts:
Skills:
 
 SkillSeek;-Skill-;  tells EH what skill to look for, EverHack needs full name not just part of it.
 SkillEditDelay;0;  This will set the delay for the skill (As in how long before you can use it again).
 SkillImport;Sense Heading.eqs; if you download a *.eqs file and want to use it, put it in here. (remember you still have to do a skillsave; the skill even if you are importing) EverHack knows where to find the skill files so just put in the name.
SkillSave; this tells EH to save the edited skill to memory, (so that EQ can acess the changes, otherwise everytihng done is worthless)
This Needs to be done even if you import a skill.eqs file
