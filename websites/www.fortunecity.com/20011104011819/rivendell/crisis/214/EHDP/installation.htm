Instalation: To install Ever Hack you will need to get the latest setup version of Ever Hack. For a list of locations where you can get the latest version click here. Once that is done extract the .zip file to a folder that you arn't going to install to and runsetup.exe install like normal software. The location of Ever Hack is not important because there is a link to it on the start menu, but if you want to edit anything (like scripts) then the folder must be easily accessable.

After running for the first time EH will ask if you want it to search for EQ. click yes. EH will seach your hard drive and display a list of possible locations of EQ select the EQ directory from the list and click ok.
