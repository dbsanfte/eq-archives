Guides so far:

Willowin's Bard Guide - Contains information on how to use songs, tips on how to be a better bard, and hints to get you started out.
The Wood Elf Bard's Essential Guide to Combat (levels 1-9) - Despite the title, this guide is great for any bards
Written by Nafai
Wileras' Guide to Charming - Great info for people who are new to charming, it covers quite a few aspects of how to use it, as well as the best ways to use it.
Advertise Yourself - An interesting guide written by a Cleric, basically explaining the many powers of the bard.  It explains why bards should be welcome in any group, and how we can advertise ourselves so that we will never have to be alone.
Bards in Combat - This guide will tell you everything you need to know about being in combat as a bard.  Written by Wineyard Wilde
Newbie to Intermediate Bard Guide - This guide tells you a good way to get from level 1 all the way to the intermediate levels.  Written by Minite
