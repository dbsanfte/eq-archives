I spent a couple months fighting with AO. The difficulties that this game had on rollout are now legendary, and it still has many bugs and many problems. They are constantly updating it, and it is gradually improving.

This game has beautiful graphics, the best of any game of this type to date, and the animations are the best too. But the UI still has some serious bugs in it.

AO has a number of really nice features and anyone who is seriously interested in roleplaying games should have a look at this, the price has been reduced from the quite high $50 for the intial release, and so many have left the game in disgust that server lag is no longer much of a problem.

As you can see from what I have said, I have a lot of mixed feelings about this game, it is exciting and very frustrating.

While we may enjoy this game as individuals, the game design itself makes it unsuitable for a NOR duchy, in my opinion. This is because NOR is a neutral guild, and while AO provides for neutral characters, they are disadvantaged in many ways, so that as a player you almost have to choose a side.

They have recently installed a new guild system which provides many options for guild management. I left before this was installed so I have no experience with it. The problem from the point of view of our guild however is that Neutral Characters are not allowed to form guilds, so this system would not be available to us.
