If I were you I would simpley get a Cable/DSL router. Forget about sharing the connection through 1 PC. I am using an SMC Barricade cable/dsl router and it works great. If you are using PPPoE you may need to configure your user ID and password for that in the router but other than that it is pretty much just P&P. Just set the clients up to use DHCP.

This router retails for about $100 and also allows NAT and some firewall features.
