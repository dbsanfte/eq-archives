Hi

There's a lot of suggestions I have for you. Which one to take depends on yer operating system.

Win 95 ye may want to call me ...
Win 98 use SFC (System File Checker) and or scanreg (scan registry). SFC will find corrupted system files. Scanreg will allow you to switch to an earlier registry.
Win ME use System Restore and switch to nearest restore point prior to your problem.
Win NT and Win 2000 I haven't played with much.

As a general rule if it's just your network that's corrupted you can always try uninstalling then reinstalling all of your network adapters.

Just a few things to remember

1. If your registry gets hashed .. rebuild .. or you will eventually regret it. Reloading the OS will not repair a hashed Registry, you gotta blow out. This is why I always multipartition my drives and put nothing but the OS on the C drive .. that way if I need to reload .. no big deal.

2. Unplugging the power strips during the lightning is a bad thing .. because you are also removing your connection to ground .. meaning static can build up and fry stuff.. prob what happened to you if the box was running on shutdown. Just use a good surgeprotector (and UPS if you can afford one).

3. GO Start | RUN type in WINIPCFG in win9x to see what your DHCP is assigning you or to check other boxes for what they are using if you want to manually assign stuff.

4. Internet connection shraring is very picky if any you use that.. especially on ME, and especially if you use a network card it doesnt exactly recognise (IE - IF ME calls an HP card an Intel chipset card you're in trouble.) Sometimes even removing and re-installing doesnt help.

5. If you rebuild the OS and it doesn't help .. you've fried you memory .. or worse. Memory is always the frst to go though.

6. hmm cant think of anything else .. except Network Neighborhood is a GUID in the registry .. try downloading TWEAK UI and installing then use that to remove Netwrok Neighborhood and replace it .. that may work if you really don't want to reinstall the OS. If you're registry is too far gone nothing will fix it.

7. oh yea .. you can try regclean .. if you've got garbage this will clean up registry garbage .. but not corruption. download at
http://support.microsoft.com/support/kb/articles/Q147/7/69.asp
