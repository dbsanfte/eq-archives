  7/24/2002  Today's Patch....

Today's Patch was a huge one.. Lots and lots of changes and lots of stuff that has been awaited has been added.

PATCH MESSAGE

Here's a summary:
  • The new interface is now available to players.
  • The Bazaar is now available to players.
  • A new zone: Jaggedpine Forest is now available to players.
  • Moving NPCs will no longer receive a reduction in DoT damage.
  • Fear & Charm will no longer work against any player on any server when cast by another player.
  • The bug to use charmed pets to have NPC's assist with kills has been fixed.
  • Vah Shir Models for the Old World have been added that are less memory intensive.
  • The Vex Thal key quest has been modified.
  • Pets will now resist spells differently.
A massive patch with lots of new stuff to explore.

  7/21/2002  Scheduled Patch.

SCHEDULED PATCH JULY 24

This from the scheduled downtime page:
    On Wednesday, July 24th, at 3AM PDT, (10AM GMT), all EverQuest servers will be brought down for a scheduled patch. The estimated downtime is 5 hours.

    - The EverQuest Operations Team

We asked Alan VanCouvering if this patch would put the new User Interface and the Bazaar on the Live Servers. Here was the response:
    Yes - As a part of the rollout for the new interface, Beta Two of the new UI and the Bazaar will begin on the Live servers as of the next patch.

    It's had an excellent few weeks overall on the Test Server and we'll be running it on the Live servers in an optional mode until we're satisfied with it. We'll give details about how to turn these options on in the patch message.

    - Alan
