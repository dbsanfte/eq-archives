Now, I know a lot of you are not computer programmers so let me give you a brief summary of what the script does and what you need to change.

The Notes Section does affect the program. It just puts a description of what the script does.

The Macros Section defines what macros you'll be using.  Change the key combination according to what your macros are setup as for hiding and stealth.

Now the Location Section is used as a test point for your script. Load up UO and enter the game using your thief character, and use the hide skill.  Now, from UOAP click SET and click on your characters head.  Go back to UOAP and record the number in the dialog box.

Go into the Variables Section, type in a description (head color) and enter the number you just recorded into the value box.

You should be setup now (if you downloaded the script)  If you didn't enter in the following code into the Script Section:

If Color at Location #1 is NOT Variable #1
Send Macro #1
End If
Send Macro #2

So here's what your computer will be thinking:

If the color (that number you recorded down) is not equal to the same value as the number you entered in the variable section, then you must not be hidden, so use the hide macro again.  Otherwise keep using stealth macro.
