Unconfirmed Bug(s)

  • Mining Macro: This trick goes out to all the people, who want to get a lot of ingots, without the manual labor(or money). To do this you will need to be next to a forge close enough to a mountain(or mine) so that you can mine from right next to it (and several shovels in your backpack). Step 1. First off, set an object for a shovel(make sure it is a general not specific object) Name if Shovel. Step 2. Set an object for the place where you will be mining from(specific)name it mine. Step 3. Set an object for 1 iron ore. name it ore. Step 4. Set an object for a forge(the one you will be smelting on)name it forge. Step 5. Now, Start a new macro like this:
    Use object Shovel
    Delay: 1000
    Target Object Mine
    Delay 2500
    Use Object Ore
    Delay 1000
    Target Object Forge
    Delay 1000

    now just set a loop for that and your all set. (thanks Slaya)
