アップグレードの準備はOK？

・始めよう！
・おめでとうございます！

始めよう！

必須設定:

  ・旧バージョンの "cgi-bin/yabb" に Printpage.pl(.cgi) あるいは Reminder.pl(.cgi) が存在するのであれば、両方とも削除してください。

  ・新バージョンの「Settings.pl ファイル、/Boards フォルダ内、/Messages フォルダ内、/Members フォルダ内、/Variables フォルダ内」以外の "cgi-bin" に入っているすべてのファイルをアップロードしてください。

  ・旧バージョンの YaBBHelp フォルダを空にし、新しい /YaBBHelp をアップロードしてください。

  ・旧バージョンの ubb.js および fader.js を、新しいもので上書きしてください。

  ・旧バージョンの YaBBImages フォルダーを空にし、新しい /avatarsと /YaBBImages をアップロードしてください (現在のアバターを残したい場合、/avatars の内容のアップロードは不要です) 。

  ・新バージョンのテンプレートを参考にし、.hr、 .imgbg、 .imgcatbg、 .imgtitlebg、 .imgwindowbg、.imgmenu クラスの CSS を template.html に追加してください。

  ・以前に "sticky threads" MODを使っていなかった場合のみ、新バージョンの "cgi-bin/yabb/Boards" に入っている "sticky.stk" を /Boards フォルダへと移し、chmod 666 (または 777) と設定してください。

  ・以前に "Ban by membername" MODを使っていなかった場合のみ、新バージョンの "cgi-bin/yabb/Variables" の "ban_memname.txt" をコピーし、/Variables フォルダへと移し、chmod 666 (または 777) と設定してください。


オプション設定:

  ・旧バージョンの template.html にある２つ目の <yabb title> を <yabb position> と書き換えてください。

  ・以下の Java Script を 旧バージョンの template.html の <head> 〜 </head> で囲んであるヘッダ部内に追加してください:
