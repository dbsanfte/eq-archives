How to convert an EverQuest screenshot to a jpg.

Step one-

You must have an image conversion program to convert the large .bmp files to a .jpg format. Here is a link to the free program InfranView that we will be using in this demonstration.

Download IrfanView

You may also use another program such as Adobe PhotoShop, Corel or PaintShop Pro.. 

But I have chosen InfranView since it is FREE and seems to load the fastest when viewing images.

Step two-

Once you have downloaded the .zip file extract the folder to your desktop and run i_view32.

 

Step three-

Open  your EQ screenshot by selecting FILE / OPEN

Step three-

Find the image !!!! The larger the number the newer the screenshot!

Step four-

Time to save it as a different format...

EverQuest saves the screenshot as a .bmp file. BMP files are HUGE in size and not very web or email friendly since they are about 2MB each.

Don't try to send users BMP shots... They will fill up their mailbox or be rejected by Hotmail or another similar ISP.

Step five-

We will be converting the image to a .jpg which will also reduce the size of the file to about 100k or less.

Finished

Once you have reduced the size of the image file it can easily be sent via email or put on to a webpage....

TIPS

Screenshots taken at night will usually be to dark to edit and see. 
When you are taking a shot of your character try to be the only person in the pic.
Check your eqclient file so that you are viewing all the armor types found in the game. 

EXAMPLE:
[Defaults]
Log=FALSE
RunMode=FALSE
ScreenMode=NORMAL
AttackOnAssist=TRUE
ShowInspectMessage=TRUE
LoadArmor17=TRUE
LoadArmor18=TRUE
LoadArmor19=TRUE
LoadArmor20=TRUE
LoadArmor21=TRUE
LoadArmor22=TRUE

If you unsure about editing your client file please consult with The Verant Technical Support...

Good luck!
