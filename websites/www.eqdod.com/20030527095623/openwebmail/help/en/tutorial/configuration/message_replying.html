Message Replying Options set the way that messages are replied to.

Message Replying Options Reference Chart

Term/Setting What it Sets/Changes
Send 'Confirm Reading' receipt Sends you a receipt that the reading of the replied-to message has been confirmed, if set to yes. You can choose "Ask to Confirm", if you want a receipt for only some of the replied messages, and it will ask you before sending a receipt.
Re-paragraph Message Before Reply Re-paragraphs a message before you reply to it
Reply with original message Selects the location, when replying a message, to place the original message
Backup Outgoing message to SENT folder Puts a backup copy of all replied
