Open Webmail Help Tutorials >> Message Filtering Options
The Message Filtering Options determine how and where blocked messages will be filtered to.

Message Filtering Options Reference Chart

Term/Setting What it Sets/Changes
Limit for Repeated Messages Sets the filter limit for redundant messages.
Filter fake SMTP Filters messages with a fake SMTP address.
Filter fake From field Filters messages with a false value in the From field.
Filter fake EXE Filters messages with fake executable applications.
