Want a more appealing, striking look for your company logo, newsletter, catalog or brochure? The Production Doctor (aka Frank Media) will do it.  We  do expert design, image retouching and advertising production using the latest technologies.  Tools of choice include:

Director 6 (indispensable for creating shockwave and of course interactivity for the web or CD-ROM) we like 6 for its ability to cache and stream shockwave movies.

Photoshop 5 (we like working in layers and using filters, effects, and plug-ins).

Illustrator 7 (many of our clients still work with earlier versions; we like this for its ability to rasterize.  This makes illustrator more convenient for creating anti-aliased type and logos for web delivery.

GIF Builder - tool of choice for GIF animation; nice transitions and control over pixel depth absent in other apps.

HomeSite,  BB Edit - tools of choice for website construction.  While there's no single app that does it all, we also find Net Fusion a big time saver for site layout, image management, and link maintenance.  HomeSite and BB Edit are very flexible text editors, with some nice built in features, and we use those to hand code HTML pages.

Shockwave Flash - where plug-in architecture is permissable, we like Flash especially for hot buttons.

Javascript - we'll use javascript for scrolling type and for creating rollovers, for image manipulation and for some animations.  We like javascript for its ability to effect object properties without plug-ins.  Where necessary, we'll use VB as well.

Perl 5 - we'll use Perl to customize CGI applications, making sure your forms interact with your server, process and return data, and do what you want them to.

Adobe Premiere  - tool of choice for video editing, including quick time movie sequencing, transitioning  and  retouching.
