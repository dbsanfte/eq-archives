Any major item can be merited by the guild leader or officers.  We use this system to insure that the proper items go to the proper class and/or people.  Please note that all items will not be decided by merit, but some of the rare items need to be.

Since some may not understand the criteria used in deciding merit, we have the following guidelines.  So everyone as a better idea of how meriting can be done.

 

Guidelines to use for meriting:

1.  Good of the guild.  Merit should go towards advancing the guild.

2.  Common Sense.  A flowing thought IV item is not a warrior item.  High wisdom items are not pure caster items, etc.

3.  Class.  Some classes are more essential to raid success.  While all classes have value, class should still be considered.

4.  Upgrade.  If the item is a major or minor upgrade.  Also when the last major upgrade obtained with the guild.

5.  Attendance.  While no one should be required to attend all raids, raid attendance should be considered.

6.  Morale.  Sometimes a player or the guild needs an item to be merited a particular way for the guild or player morale.

7.  Length of time in guild.  Members that have been in the guild longer than others should be considered.

 

In closing, the above are guidelines to be used when and how an item might be merited.  These are only guidelines and not LAW.  The leadership will make every effort to distribute loot in a fair manner, that’s best for the guild and “Spread the Wealth” when using merit.
