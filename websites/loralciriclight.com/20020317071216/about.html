Tales of Loral Ciriclight

About This Site

Tales of Loral Ciriclight is a collection of stories based around a character in the game Everquest. Loral is hosted on the Quellious server, and was created in May of 2000. He is a member of the prestigious guild, Healers United. This group prides itself on seeking a higher goal than the typical hunting and gathering of powerful beasts and magical weapons. We seek to help all the citizens of Norrath in their time of need. Many of the articles were written for the guild's monthly e-zine, The Healing Zone.

This site was written to work on any device capable of rendering HTML but looks best on a browser that supports Cascading Style Sheets including Netscape 6, Internet Explorer 5 and Opera 5. Downloading to a Palm Pilot

Because this site is built around accessibility you can easily download it to your Palm Pilot using the AvantGo Portable Web Browser. Once you have signed up for the service and installed the software, follow these instructions:

  1. Login to the site.
  2. Click "Create Channel" on the right hand menu.
  3. Enter the following settings:
    • Enter "LoralCiriclight.Com" as the title
    • Enter "http://loralciriclight.com" as the URL
    • Set the max size to 100k
    • Set link depth to "1"
    • Set include images to "no"
    • Set follow off site links "no"
    • Save the Channel.
  4. Resynch your palm pilot.

If you have trouble installing using the above instructions, try this tutorial.
