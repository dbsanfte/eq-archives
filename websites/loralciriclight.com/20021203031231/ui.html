Features

This user interface was designed for simplicity, function, and the excellent fantasy theme of T-King. The following changes were made to the default interface:

  • Keyboard key labels added to group members and hotboxes. Helps identify, remind, and teach hotkeys for both new and expert players
  • Most window titles were removed.
  • Removed many extra button controls. Learn use hotkeys or keyboard mappings. It is far faster.
  • Percentage values for hitpoints, mana, target hps, group hps and AA exp. Thanks Rombus!
  • Chat windows don't use fancy borders.
  • T-King spellbook. Click bottom gem to close book. Click outer page borders to move forward or back. Click center of book to move around. Thanks to T.King and Remelio.
  • Single player / target / group box. Extra player and target boxes removed. Can be toggled on and off with ALT-P. Removed extra player and target window (size 0).
  • Labels for (from left to right) AC, ATT, Fatigue, XP and AAXP added to combined player window.
  • Moved casting bar as a sub-bar of the mana bar on combined player window.
  • Added pet bar under player health.
  • Added T-King Arcane casting globe.
  • Added streamlined bag interface.
  • Removed % signs...in a few days you won't miss them. Saved 3000 pixels of space.
  • New buff window with tiny icons and text descriptions on colored background. Thanks Rombus. Recommend that it is pushed up to the upper right hand corner and set with faded level 0 and normal level 50%. This will remove the rocky background and fade the buffs so they are visible but not too bright.
  • Compass that is the same width as the hotbutton box. Thanks Yaxa!
  • Fix to in-game books (trades, quest, etc) using the same spellbook rules. Thanks Remelio
  • T-King Arcane Inventory window with bubble gauges. It's big but pretty.

How to install this mod:

Here are some better instructions

  1. Click this link, click on "Save" and save it to your everquest/uifiles directory.
  2. Click on "My Computer", "C Drive", "Program Files", "Everquest" (if you have the trilogy, it's in "Sony"), and "uifiles".
  3. Create a new directory inside "uifiles" called "new"
  4. Extract all files from loral_ui.zip to the new directory.
  5. Check that inside the "new" subdirectory are about 18 files, most called EQUI_something.xml. If not repeat step 3.
  6. Start Everquest as you normally would.
  7. When inside the game world, type "/loadskin new 1". This should load up the new UI you created above.
  8. Move windows by clicking in their background or on their edges and place as accordingly. For an example of a good layout, see http://loralciriclight.com/images/loral_screen_layout.jpg

This zip also contains my personal .ini files for screen settings, default keyboard maps and other initialization settings. Feel free to play with them or ignore them.

UI Setup Tips

  • Remove textures from window backgrounds by clicking the window, clicking background and clicking texture.
  • Make the background color black by clicking on the window, clicking background and clicking tint. Than click the black box.
  • Set both normal and fade alpha to 50% for spells, hotboxes and group windows. Set chat to 70%. Set buffs to 0% for faded and 20% for normal. Stick buffs in the upper right corner.
  • Use keyboard shortcuts to activate hotbuttons, spells, targeting group members and assisting.
  • Learn all available "/" commands such as /tar, /assist, /cast 3, /book 2. Use groups of / commands in a social.
  • Remember that the new UI isn't supposed to look pretty, it's supposed to make the game easier to play.
  • On smaller resolution displays, 1024x768, put all your windows on the bottom and use /viewport 0 0 1024 525 to improve your framerate.
