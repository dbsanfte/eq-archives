Report on EQ and DX 7.0
13:28:10 PST
I so far have had no trouble what so ever with EQ and Direct X 7.0, everything works the same as when I was running 6.1. We have mirrored Direct X 7.0 here. This is the email David Schattilly sent in:

DirectX 7 does work I confirmed it with Verant Tech Support line. When I installed it, it cleaned up the issue i had with it crashing out. Well it still does have frame rate problems but that is my system none the less it works great. It looks like they are going to add AGP acceleration DirectX. (Probally with Windows 2000) There is still a issue with it that SBLive cards if set in full acceleration will NOT do the ambient monster sounds. (Bat wings flapping monsters growling wolves howling) you still have to set it down to standard acceleration. The good news is in DXDIAG you can do that and you don't have to set it in multi-media anymore. There is also a few new features this version of DirectX has in DXDIAG. One is the AGP Support but cannot be used yet. Maybe my Voodoo III 2000 AGP card don't have it or it is going to be in place with the next o/s release.

The next feature it has is more troubleshooting tools. Sound has been added. Besides the Msinfo Override and Troubleshoot. The next feature to be added is NETWORK tab. You can test all the registered services, Tcp/IP Direct play, Modem, Ipx and Serial connection. You can test it.

Everquest was much smoother with this version of Direct X however, as with any DirectX there is no way to de-install it. However I had experienced a bug, when your in Faedark and you camp out it was still playing the music along with the charactor selection music. This only happend when I had my Sound Blaster Live on hardware acceleration. (Full acceleration.) This did not happen when i went back to the old setting i have to use to hear the ambient monster sounds.

There is fixes for direct animation, there is also fixes for movie playing that use direct draw. And a fix for the task bar box staying on the task bar. You have to click to make it go away that was fixed. The clarity of the game is improved a little. And a lot of my frame rate issues were eliminated but still happen. Good luck and if you want to mail me feel free to mail me at keiichi@uswest.net.

David Schattilly.
Aka Ataru - Human Druid Xegony
Keiichi Morisato - Barbarian Warrior Xegony

P.S. There is a known fact that the Y1K bug caused the Dark Ages!

Test Server Update (patch info)
17:47:44 PST
The test server was updated with the following:

September 23rd, 4:30pm
----------------------

The following changes have been implemented with the latest patch, most importantly, changes to corpse resurrection. It should be noted that all corpses generated previous to this patch may not be resurrected, however, all corpses generated from this point forward may be resurrected with the changes listed below applying.

  • Corpses that are older than 2 hours cannot be resurrected.
  • Corpses that are produced from a duel cannot be resurrected after 10 minutes.

    General Enhancements

  • A Yes/No request window is now brought up when ever turning in the PK note to the Priest of Discord.
  • It is now possible to manually reorganize icons within the spell book.
  • Pet damage inflicted during PvP combat has been slightly reduced.
  • It is now possible to place items marked as "No Drop" in the bank.

    - The EverQuest Team
