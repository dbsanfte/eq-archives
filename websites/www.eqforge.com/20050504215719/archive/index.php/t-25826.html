Click on it to download it, and choose “save this file to disk”, rather than “open from current location”. Choose where you want to download it. Once this is doen, you should have a file called “Civ3GOTM09.SAV”. You should check that the file is about 78 kb in size. It should be able to be opened from civ III. Note that you have to start civ III, and choose “Opend save game” (or similar) from the menu – you can’t double-click the file to open it. Good luck! :)
