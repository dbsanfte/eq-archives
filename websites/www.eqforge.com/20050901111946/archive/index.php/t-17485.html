Another month... another game. :) This month's game is the first Civ3 GOTM created by Matrix. If this map turns out to be very difficult or ugly, blame matrix! ;)

Game Parameters:

Civilization: Americans
Difficulty Level: Monarch
Map Size: Small
Rivals: 6 Random Rivals
Climate: Arid, Cold
Continents & Water: Random
Earth: 3 billion years
Barbarians: Restless
Victory Conditions: All


Playing the Game:

- You must use the DEFAULT RULES and the v1.17f patch to play the game.

- Use of any cheats/trainers and editing of saved games are not allowed.

Submissions:

- Saved games are due April 2nd, 2002.
- The 0 AD save game is now optional.
- When submitting the game, please include all the info asked in the submission page and zip your saved games. Submissions with incomplete info will not be added.
