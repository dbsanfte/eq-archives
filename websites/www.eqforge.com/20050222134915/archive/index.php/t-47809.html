Ceremonial Burial ~10% if you use Samson’s theory, ~20% if you use Solo’s theory, give or take a few points
Code of Laws ~3%
Currency ~8%
Horseback Riding ~16%
Map Making ~10%
Masonry ~1%
Mysticism ~10%
Pottery ~10%
Warrior Code ~21%
Writing ~1%.
