Broken Images - September 15, 2002 by Hauki
The broken images have been fixed. One of our nightly scripts failed to execute, we will monitor the script and see if we can figure out what happend. This is the 2nd time in the past month that the images have been broken. If any of you come to the site and find that the images are broken, please send an email to info@eqfashion.com.

Server Wide Chat - August 10, 2002 by Hauki
A couple of weeks ago Verant opened up server wide chat and today I had the thought of starting an eqfashion chat channel. I added "eqfashion" to my /autojoin on both of my accounts, so if anyone wishes to contact me in game, please do so by typing the following: /join serverwide.eqfashion If you want this to auto-join when you load up your character use the following: /autojoin serverwide.eqfashion See you in game!

It's been a while... - July 31, 2002 by Hauki
Well, whatta ya know, an update! :) Sorry, things have been pretty busy lately. I was poking around the logs and put together some information that some of you might find interesting. Here goes:
  • The main site has been hit 24,457 times
  • The gallery pages have been accessed 286,483 times (wow!)
  • There are currently 512 images online (Keep 'em coming!)
  • We have transfered just over 1.7 Gigabytes of data since opening
Now with that said, I'd like to thank everyone who uses the site on a regular basis, without you guys (and gals), the site wouldn't be where it is today. There are still a number of things that I would like to do to improve the site, and in time, I'm sure those features will be put in. Anyways, Just thought I would make an update and thank everyone for using the site.
