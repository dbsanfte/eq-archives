Guild News Items paging enabled Thursday, September 23, 2004 - 4:27:07 PM

I enabled paging for the guild news page (default page).  You will see the pager on the top and bottom, if you have more than 5 news items in your guild news page.  Some guilds have a lot of news items, with a lot of images and screenshots from guild raids, resulting in some pretty pathetic page load times.  (Oh, before I forget, COMPRESS YOUR IMAGES!!!). 

Anyway, those guilds with a lot of news items should see a pretty big speed increase when loading the guild news page. 

Next up is the gallery page.  I know this is coming out of nowhere, but I developed a gallery component for another website of mine, and it`s going to fit nicely into guildwizard.  The new gallery will support Albums, which basically allows you to categorize your images.  Also, you will be able to define and customize the gallery quite a bit more, such as being able to specify thumbnail sizes, how many thumbnails per page, number of columns.  Moreover, you will be able to browse images by clicking Previous & Next navigation buttons after you click into an image and are in the "Detail View".

Guild Forums Polls Wednesday, September 22, 2004 - 11:12:11 PM

I deployed the new guild forum poll function.  You will see the button for a New Poll in each forum page that lists the threads.  The button, like all the other forum buttons, are configurable to any image you want in the Forum Editor.  I have it set that only logged in people can Vote on the Polls.  So, if the user has access to the forum that the Poll lives in, they can vote.  Once they vote, it`s recorded, and the results thus far is displayed.  I also added a "Show Vote Results" link, so people don`t have to vote, to see how the poll is going so far.

I will further enhance the polls by allowing color configuration for the result percent bars.  Currently, they are white on green.. I intend on allowing admins to configure the colors.  I will also allow admins to set forums to allow Polls or not. 

Oh, I also fixed a bug that would delete an entire thread if you tried to delete the first reply of a thread on a page other than 1.

Forum Buttons and Icons configurable Tuesday, September 07, 2004 - 4:31:21 PM

Today`s patch enables guilds to change their guild forum icons and buttons.  Previously, guild forums buttons where setup to as text type links where you could change their appearance in the style editors.  I now have them setup as Image Buttons, which allows greater design possibilities for guilds.

If you don`t customize your guild forums buttons/icons, the system will just use some default gray buttons.  To customize your guild forum buttons and icons, login with your admin/officer account and go to the guild Forum Editor.  If you click the Change Forum Icons link, you will be brought to the panel that allows you to change the various icons/buttons.

To change the icons, click the (change) link which brings up the File Manager.  Upload/Select the image you want to use and click the save button.

Image Button Generator Thursday, August 12, 2004 - 11:30:48 PM

I created an Image Button Generator in the Control Editor.  Basically, this allows you to select a blank button and add some text to it.  The when you select Image Button as they Control Type in the Control Editor, you can now generate a button as well as just uploading a premade image button.

Right now, there is about 75 button styles that you can choose from.  When you are in the editor, you can choose the button style by clicking the icon which brings up a new window with a picture list of all the available button types.  Double click the button you want to work with, add the text for the button, and click the Generate button which will generate the button and display it on the screen.  The image is saved, but the Control is NOT saved untill you click the Save button.

You can create some pretty good buttons with this tool.  Keep in mind, it is very basic compared to any half-decent image editor.  If you can use an image maniplution program like PhotoShop, you should create your own buttons.  This tool is for the people who dont know how to use these programs, and want a simple way to create some buttons.

Guild Application Page Tuesday, August 03, 2004 - 10:54:46 PM

I just deployed the new guild application/recruitment request page.  This page allows you to create a form for guild prostects to fill out to join your guild.  The questions are totally configurable by you, as well as the order in which they appear.  You can also mark questions as mandatory/required, which will display an asterix next to the fields that are required if they are not filled in. 

How to setup your guild application page:

  • Login with your admin/officer account
  • Click the admin menu link and click Guild Application Setup
  • The first page sets all the non-question items such as Page Header Title, Description paragraph (this is the paragraph before the questions, maybe to explain your recruitment policy etc), and an email address to send the guild applications to.  Fill in all this information, and click the Submit button to save it
  • Click the Manage Questions selector to change to this panel
  • Add your questions that you want to ask in the New Question section, select if its mandatory, and click the Button to save each question.
  • You can change the order of the questions  in the grid below, as well as delete any questions

I might add some more features to the application depending on the feedback/usage it gets.  I know that most guilds are going to want to have a thread about the applicant, and thats where the voting usually happens.  Will play it by ear to see what people want and make some feature additions from there.
