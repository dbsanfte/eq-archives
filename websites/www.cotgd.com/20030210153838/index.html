Announcement of next changes
Thursday, January 02, 2003 @ 12:38pm - Posted by Aliseisu
We have had a good number of people ask in guild if anyone is a master tailor, smith, brewer, etc. So I am adding checkboxes to the character maintenance page where you can set then to indicate what tradeskills you have mastered. If you check one of these tradeskills, a graphical icon will be shown to reflect that you are a master at this.

Also being added is an epic status dropdown. You will be able to select from 'Not Started',10% thru 90%, and 'Completed'. That way if another member in the guild needs information on a particular part, then they'll know who might have done that step already and can ask them the next time they are on. A graphical icon will also be used for this, and will only show if you select anything other than 'Not Started'

Now, to incorporate these changes, the Hall of Heroes will be changing a bit. I want to keep the information already provided while including the new changes I am implementing. You'll notice the changes as they happen. I will post my progress here as the changes are implemented.
