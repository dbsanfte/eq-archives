Introduction
     
 

For quite some time I've been looking for a way to have my right click items easily accessible. I decided to take things into my own hands and designed a custom target window for myself and my Cleric. I showed it to a few people and they wanted target windows for their characters. Easy enough I thought, so I wrote this script. Using this script you can design a custom target window to your specifications and insert it into ANY interface be it default or custom. You can have up to 6 item or inventory slots ready for use at all times!
