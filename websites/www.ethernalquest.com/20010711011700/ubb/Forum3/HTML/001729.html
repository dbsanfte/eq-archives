ok, I think this is related to the problem the people using the Newbie install guide thread:
http://www.ethernalquest.com/ubb/Forum3/HTML/001411-6.html

what I did to get past the error you are getting was the follow:

vi Makefile.dist

do a search for autoconf until you find a line that reads:

@autoconf -o configure configure.in

Change that line so that it reads:

@autoconf configure.in

that will get you past the error that you are getting but I suspect that you will have the same problem the rest of us are having at that point. Namely not having a configure script to run... Wish I would hope someone is working on because it seems there are a number of us having this problem.
