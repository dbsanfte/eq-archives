Here is what I would try next.

Login
su -

Once you are logged in as root, expand the archive in the /root directory.

Once you have the archive in /root/showeq-2.1.2 then run through the installation process.

Also, run this process from console and not from X. I have not verified that this actually causes a problem, but after installing QT and libpcap on my machines I've always exited X and restarted the machine before running through the installation process. I suspect though that your initial problem is that you are not logged in as root.

Also you need to run ./showeq while logged in as root, or equivalent security. To get it working though I would suggest using the root account, then once it is working you can attempt to scale down an account with which to run SEQ.
