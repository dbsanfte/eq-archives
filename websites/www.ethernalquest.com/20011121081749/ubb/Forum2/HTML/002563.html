Legally I don't think there is much Verant could do other than make threats. Lets look at this legally.

HackersQuest is making a server product from code written from scratch. They did not use the source code of the actual EQ servers. Unless they somehow broke into Verant and stole it. But as they didn't we will say the code was created 100% by the EMU team.

Thus that code is owned by them and cannot be affected anyway by Verant. If Verant/Sony could walk up to you and say..."Hey you are making an mmorpg server stop it!" and actually win that argument in court, well there would be no other mmorpg games on the market. This is not legal.

So the EMU team has simply created an mmorpg server for use by the public. They could in fact sell this product if they so desired. But without a client program to use it who would purchase it?

Could you as the user of this mmorpg server create your own client and graphics, etc to run this mmorpg? Definitely. Will you? Probably not. Instead you will take an existing client program, EQ in this case and use it to attach to HQ's server product. Thus you the end user and the only person that has done anything illegal.

Is Verant going to chase after every person that downloads and runs the emulator with their EQ client? Probably not. What does Verant have to gain by suing the average person? The settlement would not even cover their legal fees.

Fact is HackersQuest owns the code. What they do with their server code is 100% up to them. If you look at the Beosil Emu that everybody says was “Shut Down by Verant”. He states he quit working on it due to a sudden realization of morals and not legal reasons. His url was www.swileys.com/emuq but seems to be down now, tried www.swiley.com as well but wasn’t it. Odd thing was that Beosil also stated that he did not open source the code because of certain networking functions that would be good for use by hackers. Personally the attack of morality and tightness with the source makes me think that me may have gotten a nice check from Verant to cease work and kill the project. But that’s just my opinion and probably is not accurate.

It’s the same story as UOX. They will threaten, they will say you are a bad person, but in the end they will do nothing to stop it. With the possible exception of banning you account if they can prove you used an EMU. That would probably mean scanning your HD, which we know they got into before and it caused them grief.

The only work around they have in the EULA is a statement that reads…
"You may not use any third party software to modify the Software to change Game play. You may not use our intellectual property rights contained in the Game or the Software to create or provide any other means through which the Game may be played by others, as through server emulators. You may not take any action which imposes an unreasonable or disproportionately large load on our infrastructure." (They mean server clones...)

Ok define Intellectual Property Rights contained in EQ. If I can define a set of code in C++ or whatever as Intellectual Property why wouldn’t every software manufacturer be suing each other of code that is similar to theirs? Or websites that are similar, etc.

Second issue on Intellectual Property Rights pertaining to the EMU server. A definition of the portion of the EULA I referenced above. “Creating a server clone would involve using the intellectual property contained in EQ. It's impossible to create a server clone without figuring out what the client and the server say to each other, and it's impossible to figure that out without using the client. It's impossible to use the client without agreeing to the license agreement. Even using the server clone would involve modifying the client.” Basically I interept this as saying the EMU creator used some sort of sniffing device, etc to determine the interaction of the client and server. My response to an allegation such as that….Prove It!
They can’t without hacking your system and reading your files or sniffing you sniffing them. And wouldn’t that be illegal?

Just my 2cents. Flame if you will.
