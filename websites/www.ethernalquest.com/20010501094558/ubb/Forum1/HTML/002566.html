EverTalk is now finished! Anyhow, here's how it works:
download the setup file from:
http://DysassymblerX.tripod.com/
[Edited to reflect correct URL]

The website also contains similar information to this post and the full list of in-game commands.

Run it:
It will instruct you to click yes on the following installers. The install
is actually 6 combined installer programs since it uses MSagent to talk. I
made the install as easy as possible.

Next click yes to Install MicroSoft Agent and TruVoice

The voice character will automatically load.

Install Talking Clipboard anywhere. If this isn't the program that's listed
to install it means you need to clear your
C:\Windows\TEMP folder
However, this error is rare.

Next you'll get a winzip self-extractor. This is EverTalk itself. Type in a
folder to install it to and click unzip. Leave the checkbox checked that
says "run everscut.exe after unzip completes." It will make a shortcut in your
StartMenu\Programs area.

It's almost ready. Next, run EverTalk and select the log file for the
character you are going to play as. Either change eqclient.ini so logging is
always on or turn it on once you enter EQ. Run EQ and listen to people talk!

There are several in game commands you can use to control what is spoken
and what isn't.

They're done in the form of emotes. In EverTalk, the help button displays all
in-game commands.

For example, to completely stop talking:
/em doesn't hear voices.
and to re-enable talking:
/em hears voices.
