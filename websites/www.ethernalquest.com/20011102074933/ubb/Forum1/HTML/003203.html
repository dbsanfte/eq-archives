Xzzy,

Thanks for the script, works great.

A few changes/improvments for you...

unk: $boo[9] is actually itemtype (0=Normal, 1=Bag, 2=Book)

If you're using Ashran's tool to export/import items.db and the Emulator items files, you can use this to convert a current ShowEQ items.db to the format Ashran's program expects or vice versa:

./foo.pl <in.db> <SEQ/ASH> <out.db>

If the input is from ShowEQ, use 'SEQ' otherwise use anything else (must not be null). I didn't want to complicate this with getopt so...

---------
#!/usr/bin/perl

open(ITEMS, $ARGV[0]);


if ($ARGV[1] eq "SEQ") {
$structsize = 244;
$struct = "Z35 Z60 Z6 s Z22 C b b C C S S s L l Z24 c c c c c c c c c c c c c c c Z2 C c c c C C c c C Z2 L Z2 S S Z2 S c c c C c
c S Z26";
$newsize = 240;
$newstruct="Z35 Z60 Z6 s Z22 C b b C C S S s L l Z20 c c c c c c c c c c c c c c c Z2 C c c c C C c c C Z2 L Z2 S S Z2 S c c c C c
c S Z26";
} else {
$structsize = 240;
$struct = "Z35 Z60 Z6 s Z22 C b b C C S S s L l Z20 c c c c c c c c c c c c c c c Z2 C c c c C C c c C Z2 L Z2 S S Z2 S c c c C c
c S Z26";
$newsize = 244;
$newstruct="Z35 Z60 Z6 s Z22 C b b C C S S s L l Z24 c c c c c c c c c c c c c c c Z2 C c c c C C c c C Z2 L Z2 S S Z2 S c c c C c
c S Z26";
}

if ($ARGV[2]) { open(CVT, ">".$ARGV[2]); }

while($foo = read(ITEMS, $data, $structsize)) {

@boo = unpack($struct, $data);
$unboo = pack($newstruct, @boo);
print "Data:
name: $boo[0]
";

if ($ARGV[2]) { print CVT $unboo; }
}

if ($ARGV[2]) { close(CVT); }
---------
