Ok, Got a link to scruffy's new site.
http://www.trifocus.net/~scruffy/
This is where to go to get the cfg files so you can go to Velious zones and such.
He also has some other good info on his page for newbies like me.
For those who are still having connecting issues, make sure you read the README files that are included in the eqfiles.zip and other zips.
They contain valuable information.

Another easy thing to mess up is this:
The first time I was unzipping the EthernalQuest.zip file, I drug and dropped the files from the zip to the directory.
DO NOT DO THIS!!!
Took me a week to get someone to give me an answer about this.
Make sure you let winzip, or whatever unzipping program you have extract the files to your ethernalquest directory.
If done properly, you will have folders inside your ethernalquest folder, one labeled "scripts" and the other labeled "zones".
Then make sure you unzip the emulog.zip that is included into the same directory all the ethernalquest files are in.
Then unzip the eqfiles.zip into that same directory also.
Make sure you let these zip programs overwrite any other files that may be there.
If done properly, you should have no more problems.

Once again:
1. COPY all files from your everquest folder to a new folder named EthernalQuest.
2. Unzip the EthernalQuest.zip into this folder. Double check and make sure it did not create a folder named ethernalquest INSIDE the ethernalquest folder you had created. If so. just drag all those files from the new one to the one you created.
3. Unzip the emulog.zip into the ethernalquest directory.
4. Unzip the EQfiles.zip into the ethernalquest directory.
5. Delete all your old .opt files.
6. Run Zoneserver, Loginserver, Worldserver.exe files. This can be done in any order.
7. Run Emulog.exe
8. Make up anything for station name and password.
9. Log in like normal.
10. Then after you get this to work, then you can use the hexeditor on the zoneserver.exe and make it so the #zone command works and you can change zones.
11. If you do edit the zoneserver.exe correctly, you will be able to issue a command such as #zone tox. This will tell you that you have issued a GM command and so forth.
12. /q, then log back in. You will now be in a new zone.
13. Enjoy.

Good luck,
Z.
