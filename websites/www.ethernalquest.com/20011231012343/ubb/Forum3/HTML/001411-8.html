Zombie 		posted 07-28-2001 08:46 AM            
The only thing you need to do when a new zone comes out is get the map files for it, just login to CVS like normal, do cvs update linux, and either manually copy the new .map files to /usr/local/share/showeq or run make install. If nobody has made a map you can't do a whole lot about it except make your own. There is a site I think run by Mapfiend that usually gets updated maps before they are incorporated into CVS, I forget what the URL is though.
