Date: Monday, December 4th at 7:37 PM EST

Quick Update
Date: Saturday, December 2nd at 3:02 PM EST

     After a bunch of big code updates and a new database backend on the site, things are running a lot faster and the site is more stable. All the newest servers have been added, and a bunch of user submissions were posted.
