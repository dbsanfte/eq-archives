Bug Fixes
Date: Friday, January 19th at 10:49 AM EST

     A whole bunch of bug fixes this morning. The Printable Pricing Guide had a bug where it would only return exact matches if you selected your race/class (i.e. If you chose Iksar Necro it would return IKS-NEC-only items). It now properly returns items that are wearable by ALL, returning far more results.
     The long-standing problem with the No Drop filter has been fixed. It would sometimes exclude items improperly when you specified not to include No Drop items.
     Item effects are now searchable. Just type in the name of the effect as a keyword (for example, type in "Serpent Sight" into the keyword box and perform a search).
     The range of price restrictions have been increased, up to 20,000 pp.
     Most of these were caught by people e-mailing me fixes and suggestions. Keep 'em coming!

Stats Have Returned to the Price Guide
Date: Wednesday, January 17th at 11:28 AM EST

     I originally removed stats from the Printable Pricing Guide because it was taking up so much bandwith. Unfortunatly, it was one of the best features of the site, and so many of you e-mailed me to ask for it back that I decided to compromise. The stats are now included with the guide when you limit it to a Race/Class combination. So go check out the new Printable Pricing Guide, and make your in-game trading easier.
