第32話 みなさんのおかげです 2000/11/10 FRI
　Real Hell に翻弄され、最近のEQのプレイ時間は週に０〜２回、合計10時間も遊べれば多い方という、１年前に比べるとウソのような状況だったりするが、それでもなるべく EQ に Pop しようとしている今日この頃。
　Veeshan ほど日本人率も人口密度も高くない Saryrn では、日本人パーティを現地調達するのは難しく、俺個人の長いブランクは親しい人間とのレベルの差を広げており、パーティに潜り込む手段はギルドに依存している状態だったりする。
　Lv50オーバーが９割を超える我がギルド「Eisen Lowe」では、一緒に組んでも経験値の入りが悪いだろうに……という言い分もあるだろうが、俺的には一切気にしない。
　少しずつでも経験値が入るなら、０より多いってことなんで np なのである。
　言葉の通じる相手とワイワイやるための EQ であって、黙々とソロや無言の外人パーティで、少ないプレイ時間を割くのはイヤだしねー。
　まぁ、Lv39 WIZ の DD など、そんな面子の中では ENC にすら劣るので、戦力不足で悪いかなぁ、とは思うんだけど(苦笑)。

　久々に Pop したら、サクサクと手の空いてるメンバーが集まり、SolB に行くことに。
　Cazic 以降のダンジョン経験のほとんどない俺は、何回目かの SolB にも関わらずオロオロとみんなの後を追って歩き、他のギルドメンバーがキャンプしていた場所に合流し、一緒にクモやらコウモリやらを Pull して倒す。
　やがて人数が減り、パーティを１つにまとめて更に遊んでいたのだが、敵が多すぎて緊急脱出の指示が出る。が、安心しきった俺のスロットには GG が入っていなかった(汗)。
　慌てて魔法の入れ替え。昔ほど長い時間は掛からないが、それでもこんな状況下ではその時間は無限のように感じる。
　そして準備が整い、いざ GG を唱える！ ……が、殴られてまんまと詠唱中断。
　ただでさえ詠唱時間の長い GG を、更にもう一度唱えて、かろうじて死者なしで脱出することが出来たのは、Maxi (PAL54) の LH のおかげ以外の何物でもなかった（汗）。

　再度 SolB への道を走り、中の通路でクモを狩っていると、なにやらいつもと違うクモがやってきた。みんなの話を聞いていると、なにやらいいものを持っているらしい。
　いいものを持っているだけに、俺の魔法が全然入らない程度にそのクモは強かった(苦笑)が多勢に無勢でサクッと倒し、死体を漁ってみると Golden Chitin Bracer を持っていた。
　INT+5 AGI+5 というこのアイテムは全種族／全職業が装備可能だが、INT が WIZ 向きであったこともあり俺がゲット。
　自分から自分のアイテムを集めに行かない Hagane としては、久々の INT 上昇である。
　更にその後、そろそろキャンプ解散しようかという話になり、どうせ最後ならイフリートキャンプを覗いてから帰ろうということに。
　マグマのダンジョンをみんなと逸れないように奥に進むと、イフリートキャンプは空いており、しかも俺達を待っていたかのように、一人たたずんでいるイフリートまでいた(笑)。
　やはり多勢に無勢で、俺がまったく戦闘に貢献しないうちにイフリートは倒され、イフリートの死体を漁ってみると……ありますがな。
　Golden Efreeti Boots 通称成金ブーツとも呼ばれる、INT+9 WIS+9 という、キャスター垂涎の逸品だ。
　もちろんレベルがレベルだけに、他のキャスターはみんなこれを持っていたため、自然とこの金色の靴は俺の足に。
　WIS が９も上昇し、トレードスキルの際のアイテム製作の成功率もアップですか？（笑)
　INT が上がるのは嬉しいが、足元に光る金色が成金チックでダサイので、本気（INT重視)モード用か職人仕事の作業靴に決定。
　往年の Oracle Robe と同じ扱いである……。

　後日、Karnor's Castle にて再びギルドの面々とキャンプ。
　２階のテラス(？)でキャンプをしてたら、Arches (ENC5?) の知人の外人さんが奥からやって来た。
　英会話がさっぱり分からず、ボーッとしてると「Hagane ついてきて」と Arches が先導し始めたので、一緒に奥の部屋へと入っていく。
　なにか意味ありげな立派な部屋に入ると、久々に会う Jyuuza (MNK5?) や Cature (CLR5?) もいて、そこにある死体の品をルートしろとのこと。
　なんじゃらほい？(死語)と死体を漁ってみると Supple Scale Armband とやらが転がっている。
　NoDrop で、使える人はみんな持ってるんで、壊すよりは使える人に取らせてくれるとのこと。「やー、ありがとー」と心で手を合わせつつルートして性能を見てみると WIS+5 INT+5 MANA+25 SVF+10 という鼻血もののアイテム。
　Pop した敵に殴られながらみんなにお礼を言い、その日は更に経験値を稼いで終わったのだった。

　さらに後日、ギルドメンバーの Femto (SHM5?) が JB のクエストアイテムを譲ってくれるとのことなので、みんなと一緒に JB 入手の旅へ。
　一度 W-Com に寄って、もう一つの必要なアイテム Shadowedman のレイピアを取ってから Rathe Mountain に移動。
　JB のクエストキャラが見つからず、Tracker に捜し回ってもらいつつ、HG を狩ってたら大量に襲われて死亡したりもしたが、なんとか JB こと Journeyman's Boots を無事ゲット！
　Trade Skill 用のアイテムを求めて Karana をさまようことも、もうこれで苦ではない！　……いや、やっぱ少しイヤだけど(笑)。

　んでもって、更に後日、FP での団欒モードの時に俺の装備を見た面々に言われた言葉は「あれ？　成金ブーツ取らなかったっけ？」
　持ってるけど普段は履かないの、とオシャレの主張をしてみると、Cabin (WAR5?) から「いいから INT 装備に切り替えんかい」と蛮族感たっぷりの要請を受けたのでいそいそとお着替え。
　予想以上に良い代物を持ってるねと言われるが、いや、ほとんどが人からのもらい物と、ここ最近入手したものばかりなんですが(汗)。
　などと話していると、更に Alexia (WIZ5?) が Sapphire Platinum Necklace (INT+7 STR+7) をくれるとのこと。
　最近持ち物重くて困ってたんで、一石二鳥で大助かり。WIZ のくせに STR 94 になりました(笑)。

　と、以上が１〜２週間の間に起こった事柄。
　従来に比べて一気に INT が 15 も上昇し、INT 装備の際の INT は 173。……普段は 148 なんですが(笑)。
　Human で基本の数値が低いのと、Lv40ってんで信憑性はないが(笑) EQ 発売一ヶ月後という黎明期に EQ を始めたため、Caster にとっての INT の重要性を知らなかったんで、Hagane の基本 INT はなんと 100 。
　Erudite で始めてたら、Caster の INT の目安とされている 200 に達してるのにね(汗)。
