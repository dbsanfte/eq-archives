EQ日記

第23話 世の中アメとムチ 1999/08/20 FRI
　月に一度の締め切りが終わり、さぁ、Hagane を鍛えまくるぜーーー!! と、気合バリバリ(死語)で EQ 開始。
　なにせ、他の面子のレベルの上がりが早く、Hagane は紛れもなく周りから見て最低レベル。
　しかも、最近は死にクセがついており、22Lv の 1bub までは稼いだことがあるものの、決してそれ以降まで稼ぐことができないでいる。

　早い時間からアクセスするも、知り合いは皆無に等しい。って言うか、少なくともパーティを組んだ瞬間、こちらに経験値が入ってこないような人しかいない(笑)。
　仕方がないので禁断のソロを決行！
　……と、その前に、手元にあるトレードスキル用のアイテムを消化するため、少々トレードスキルをば。
　ヤケに簡単なものに失敗が多発したのは、この後の出来事を暗示していたのか。
　初めて Banded の製作に成功するが、首なんて誰も欲しがらねーってば(笑)。

　で、いつものように散財してから、目指すはほとんど足を踏み入れたことのない、E-Karana と Beholder Maze 。
　前に Kukure (SHM 25)と見に来て、それなりに青い敵とかがいて、イイ予感がしたからだ。
　最初は E-Karana に行ったのだが、ダンジョンに潜りっぱなしが多い身なので、広いフィールドはちょっと慣れない上に、Zone が近くにないのも不安なのでさっさと Beholder Maze に足を運ぶ。
　Beholder Maze は ミノタウロスとイビル・アイが徘徊していることでも有名で、前に見た時に、こいつらが青だったのだ。

　さて、いざ青いミノ君に出会うと、やはり怖い。
　Root が効かなきゃ待っているのは死のみということで、及び腰になっているところに緑のミノ君登場。
　喧嘩を売ってみると、Root は効くし、もしも Root が切れても攻撃自体がなかなか当たらないので、改めて Root するのも容易。
　なにより特筆すべきは、EXP まで入るのである!!
　お金も5GP 位持ってる上に、当たりは有名なミノ・アックス付き。
　PC に売る気はないけれど、万年金欠病の身には嬉しい収入だ。

　1匹倒し、2匹目を倒し、完全に慢心した俺は次なる獲物を求めて奥へ進むが、そこに /CON 青の土くれ君が喧嘩を売ってきた。
　全力で行けば負けるはずがない……。そう信じていた俺だったが、敵の攻撃は目くらまし（盲目化）の効果付きだったのだ！
　パニックに陥りながらも、魔法を唱えつづけるが、善戦むなしくギリギリのところで死亡(涙)。
　FP で復活の後、N-Karana まで Gate してから死体回収に走る。
　が、再び喧嘩を売ってくる土くれ君。ここで死んだら、レベルダウンだ！
　殴られながらも、Gate を発動。今度はうまく逃げることができたが、思っていた場所に死体が見当たらなかったことに呆然とする。

　そこで俺と入れ替わりに仕事が忙しくなってきて、EQ できない宣言をしていた Show (MNK 24) に電話して、2nd の Shiki (NEC 21)の出動を懇願することに(涙)。
　GG なんてまだまだ唱えられないので、せめてと 3rd の Gagadan (SHM 09)で入りなおして、SoW をかけて Swanp から一緒に帆走。
　N-Karana で Hagane に切り替えて一緒に走るが、もちろん Hagane には SoW はない(笑)。
　ま、ともかく、Shiki の魔法のおかげで死体回収完了。
　涙目ですわ、ホントに。


　翌日、経験値もお金も稼げず、またお金が5PPくらいしかない状況に嫌気がさし、ふと博打を打つことにする。
　虎の子のHQ熊皮4枚。
　もちろんこれを使って作るのは、トレードスキル・マニア最後の希望、10スロットの手作りバックパックである。
　各種ホームページを調べると、裁縫スキル33でも1度くらいは成功するはず。
　これを売買の BBS で競売にかければ、おそらく100pp以上の収益が望めるはずなのだ！

　ちなみに Hagane 個人は、最高でも 5〜60ppくらいのお金しか持ったことがないので、かなりドキドキもの。
　10スロットのバックパックは、低レベルのプレイヤーには用のない代物であり、そんなプレイヤーからしてみると、なんでたかだかスロットが二つ増えただけのバックパックが 100pp もするのか、不思議がることと思う。
　実際、現時点 (22Lv) の俺から見ても、あまり必要じゃないし、100pp なんて金は見たこともない(笑)。
　しかし、これを必要とするプレイヤーはみな高レベルであり、100pp とでも言わなければ泣けてきてしまうほど、裁縫のスキルは上がり難いのだ。

　案の定、4回挑戦して1回成功！
　2回成功したら、一つは CHR 上昇アイテムとの交換を希望するつもりだったけど、そうは甘くないらしい。
　早速BBSに書きこむが、オークションをかけて無制限に高騰 → 結果としてバックパックや自分自身が悪者になってしまうことを恐れ「100pp からオークション、200ppで即決」という書き込みをしたら、たった2人目で200ppの値がついてしまった。
　上限つけなかったら、いったいいくらで売れたのかな？
　200ppか……、いったいなにが買えるだろう？(笑)

　取り敢えず、このお金を元手に更なるバックパックを作るべく、修行をしまくる所存である。
