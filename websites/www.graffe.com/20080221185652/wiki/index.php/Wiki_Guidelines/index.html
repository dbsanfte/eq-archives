Restricted Content

  1. Posting any type of non-public documentation including, but not limited to, personal contact information such as phone numbers or address or private employee contact information, is forbidden.
  2. No advertisements. This includes but may not be limited to:
    • Outwars / Outwars-clone link spamming / hit farming.
    • Guild recruiting.
    • Link spamming.
    • Gold Farming.
  3. No pornography of any kind. Moderators will make the determination on what gets removed under this policy, which includes but may not be limited to:
    • Pornography
    • Sexually explicit pictures
    • Writing or other material whose purpose is to cause sexual arousal.
    • ASCII porn
    • /cyber logs
    • links to pornography.
  4. No Personal Attacks. This applies to all pages including articles, categories, user pages and talk pages.
  5. No Fanboiism. If your article is about how much your game rocks or how much another game sucks, this applies to you.
  6. No Rants. Use the forums for that.

Users who post restricted content will receive warnings and/or bans against their Graffë's Forum accounts as well as possibly losing Wiki edit access.

Article Etiquette

Game Game Code

EverQuest2 EQII

World of Warcraft WoW
  1. All new articles should be named in the proper manner. Beginning with the code associated with the game, a colon, a space and then the article's name. If the article belongs to a specific sub-category, add that before the article name. Articles that don't follow the proper naming will be renamed so that they do. This is primarily being done to prevent collisions between similar articles in different games. The GameCodes are listed above. An example of a valid name would be
    • EQ: Quests: Epic 2.0
    • WoW: Classes: Mage
  2. All new articles should have at least the appropriate game category attached to it. As well as any other categories you see fit. See Category Guidelines for Category specifications and rules.
  3. Use Templates where applicable. (A template is a chunk of wiki code that can appear on many different articles). To create and use a template:
    1. In the search box, type Template:YourTemplateName and hit go
    2. Apply the necessary text and wikified code to the template, make sure to add:
      <noinclude>[[Category:Templates|TEMPLATENAME]][[Category:GAMECODEHERE:Templates|TEMPLATENAME]]</noinclude>
      to the bottom of the template and save.
    3. Insert it into any articles you feel it should be in by adding {{YourTemplateName}} at the appropriate place in the article.
    4. For more information about using templates, see: http://meta.wikimedia.org/wiki/Help:Template
  4. Use the {{clr}} template to add line breaks when normal wiki coding seems to corrupt normal carriage returns.
  5. If your article can be divided into subsections, use the appropriate formatting to add heading and subheadings so that a Table of Contents can be generated if needed.

Images

Note: Until the GD Library is recompiled, some image features may not work properly.

Image Guidelines

Image uploading has been enable on this wiki. You may upload an image using Special:Upload. Images follow similar rules as the main rules:

  1. No pornographic images.*
  2. No images that create personal attacks against a particular user or group of users.*
  3. Control your image size. Do not upload a huge image that will only be resized in the article.

*Users who post images containing these items will receive warnings and/or bans against their Graffë's Forum accounts as well as possibly losing Wiki edit access.

Inserting Images

  1. To insert an image into follow the instructions for embedding images located here.
  2. Where applicable, please use the caption feature of the images to describe the contents of the image.
  3. If an image is large, but will lose its purpose by being shrunk down, include it as a thumbnail by adding 'thumb' (without the quotes) to the image tag. It should automatically link to the larger image.

Remedies

  1. If someone is not following these guidelines, please bring it to the attention of a Wiki Sysop or relay it to moderators or admins on the Graffë forums.
  2. Not following these guidelines can result in removal of your ability to edit the Wiki as well as warnings or suspensions on your Graffë's forum account.
