If you are viewing this and can't log in, it is probably because your name is in the wrong format. Your login is linked to your Graffë forum user account, so you will use the same login here as you do there, you can't create an account that will only work on the Wiki. If your name contains spaces or punctuation, it will not work on the Wiki and will need to be changed. I have written a script that will change your username automatically if it needs fixing. It is available in the forums under the User Control Panel (UserCP) and should be at the bottom of the menu on the left side. If you have any problems, feel free to post in the Wiki discussion forum on Graffë's about it.
