    Re: Windows 10

    So I click the pop-up and it tells me Windows 10 is ready to install on my machine, and asks me to click "Continue" which will prompt a brief verification after which I can choose to install now or select a time and date to install later.

    I clicked "Continue," there was a progress bar for about ten seconds while it did *something?* after which there was a button to install now and a button to select a later date and time to install.

    Mind you, I did not bother backing up a single thing. None of my favorites, or songs or other media. I assume most of this is automatic at this point.

    I chose to install now. The screen changed to a plain, bright background with a message saying this is going to take awhile (lol). Right after that, my machine rebooted and it came up to what looked like the Windows 8.1 start-up when it's installing updates; the circular progress bar with a message saying Windows Configuration is in progress.

    I should have timed it, but I just returned home from a week on the road so instead caught up on some DVR shows with my wife.

    A few hours later I returned to check on the progress. There's a message saying "Welcome Back." I clicked the continue button and it asked for my Windows login password, which I entered.

    It immediately went to a screen that looked almost identical to the way I had 8.1 configured (no splash screen or apps window, just a stock background with a dozen or so shortcuts). I was impressed that it restored all of the shortcuts I had saved to my desktop: Shutdown, Restart, Explorer, This Computer, Control Panel... Even my Chrome shortcut, heh.

    The only thing I was disappointed with was the screen resolution defaulted to a ridiculously low setting. I was able to increase it to my preference easily enough, but looking at the Device Driver now I see that my NVidia GeForce GTX 750 drivers show in conflict. -Will have to update that I guess, but Windows 10 stock drivers don't seem to be horrible. (Haven't tried any games yet.)

    Cortana is OK I guess. -The OS's helpful avatar to compete with Siri, I guess? It appears she would accept voice recognition commands but I don't have a mic installed. Typing commands just brings up a search result from Bing.

    I'm on this site using Microsoft Edge which seems OK. I did have to manually import my Favorites, though it was a simple one click in Settings.

    The one thing I wish I'd have done before installing is check the free space available on my boot drive. Right now I have no idea how much additional it used, but to be honest it seems like I have more free space now than I did before installing. Explorer shows 21GB currently being used on my 60GB SSD. I thought I had less free space available than that before loading.

    Other than those items, I don't notice much difference yet. Everything is working fine other than those video drivers. I'll look into that and fire up some of my games and see what I notice.

    Re: Windows 10

    On Windows and OS X the version of Chrome I use is: https://iridiumbrowser.de/about

    It focuses on securing the Chromium browser more than the default installation. Also, run your web browser in a virtual machine. Less conveniently but more secure is to boot the virtual machine from a DVD-Rom. SD Cards have a way to turn it to read only but I am not sure if their are attacks to get around that atm. For financial transactions and dealing with the IRS etc. or medical records it is worthwhile.

    Also use umatrix and ublock origin in Chrome.


    https-everywhere: https://www.eff.org/https-everywhere

    ublock: https://github.com/gorhill/uBlock

    umatrix: https://github.com/gorhill/uMatrix

    Use Disconnect search engine: https://disconnect.me/search

    KeePass for password managers: http://keepass.info/

    Chrome doesn't have an extension comparable to Firefox's self-destructing cookies unfortunately. Also, don't fucking install flash, and if you have to have it for some reason keep it on another VM. Turn plugins to click-to-play only aside from flash which you shouldn't be using in 2015 ever.

    And stick with FLOSS copyrighted extensions: https://www.gnu.org/licenses/licenses.html
    I prefer Apache 2, GPLv3, AGPLv3, Mozilla Public License, 3-Clause BSD without dickering that removes its patent compatibilities for legal reasons, and MIT license (same as the X11 license). Any software licensed under other licenses (especially new ones that have not been tested in court) I don't touch.

    Ideally all of our software would have been formall verified but that is probably at least several decades away as likely less than 1% of software developers are competent in it at this point. It takes a lot longer to develop software this way but it is the only way you can have confidence that the algorithms are actually implemented correctly, but at least use probabilistic code testing (used heavily in hardware engineering and why bugs are rare when combined with QC and QA).
