Research Related Downloads

These are files that people have sent me which I think you will find very useful in researching or helping you to further understand what it is your researched spells can do.  If you have a file to submit, please don't hesitate to do so.  As a disclaimer, the Tower does not assume responsibility for any problems these might cause on your system.  Download and use at your own risk (I use them both with no problem).

EqCaster.zip

This file isn't technically research skill related, but it's a great file for all classes in determining what different items do in the game in terms of damage.  It's also a great spell resource.  This comes to you courtesy of Vaslin Slickfinger over at EQCaster.com.  Check his site for updates to this file.

Necro.zip

This file is a must have for Necromancer researchers!  It's in Microsoft Excel format, so if you don't have Excel on your computer, go to this link and download a viewer.  It makes it real easy for necros to know what spell you're ready to research.  This comes to you courtesy of Phisbin Malvagita over on Xegony.

Practice.zip

This file is in excel format.  If you don't have Excel please click on the link under the necro.zip heading above.  This file is a great aid in knowing when you're ready to practice a particular rune.

Wizard.zip

This file is in excel format.  If you don't have Excel please click on the link under the necro.zip heading above.  This file is a must have for all researching wizards.  It'll let you know what spell you're ready to research.
