2   Q: I'd like to edit a category in the Sports section of the ODP. Which ones need help?
A: Have a look at the Sports Help Wanted category at /modules.php?name=Open_directory&file=index&op=modload&?/Test/Help_Wanted/Sports/ and take your pick. If you are already an ODP editor, apply to edit the new category from your Editor Dashboard. If you're not currently an ODP editor, navigate to the desired category and select "become an editor" at the top of the screen. (And if you're not already an editor, make sure you pick a small category to start with!)
