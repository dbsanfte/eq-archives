Finding your way around Norrath's zones can be challenging. What you have to remember to do is to practice Sense Heading as much as you can and know the location of the place in which you are going. Make a Hot Key for Sense Heading and /loc. Find the location of where you are at that point and know the location of your destination. If each loc has an x and y value, the coordinates will look like this x,y.
X Y
N = +
NE= + -
E = -
SE= - -
S = -
SW= - +
W = +
NW= + +
So if you wanted to go from the location 300,-700 from the location 500,-800... you would go SouthWest. Get it? I hope so.

If I have a map and I know which direction I need to go, say I zone and I know the bandit camp is NorthEast of the zone entrance, then I like to think of it like this :

Here's the coordinate pair : x,y
If x = N-S and y = W-E
And bigger number = right movement on number line;
and smaller number = left movement on number line (the number "7" is bigger than the number "-700.")

And if you loc and then move in any direction and loc again :

Then a bigger number for x means you're traveling Northerly. A smaller number for x means you're traveling Southerly.

A bigger number for y means you're traveling Westerly and a smaller number for y means you're traveling Easterly. (I'm catching a menatal block here, sorry, cause it occurs to me that it would make more sense for W-E movement to be reversed...? Help!)

Anyway, bigger numbers for both x and y mean you're traveling which direction? That's right! NorthWesterly! And smaller numbers for both? Right again! SouthEasterly! How about Bigger x but smaller y? (Northerly + Easterly = ) NorthEasterly!
