The "alternate product" message is given in conjunction with the "you lack the skills necessary" message. Both messages appear every time. I suggest you look more closely next time if you do not believe me.

The "alternate product" message is given when you fail at an attempt to create an object of which a certain component is not lost during a failure. Examples of this type of component would include whittling knives and planing tools for fletching, bottles, shot glasses and casks for brewing, and I believe filing tools with smithing, although I haven't gotten my smithing up to that level yet. I'm sure there are others, but that's what's going on.

As an asside, what this really means is that the cost of attempting and failing at a Trueshot is zero given that you already have the planing tool. I count the tool as an expense in a success, not a failure. The cash you get from Maesyn for turning in the parts to get the Treantwood Stave more than makes up for the cost of the components that go into it.
