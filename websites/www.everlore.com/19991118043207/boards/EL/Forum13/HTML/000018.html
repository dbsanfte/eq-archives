Actually I think Verant said something to the effect that any class could be soloed to lvl 50, but at some point it would cease being fun and start to become hard work.
Any class is 'fairly' easy to get to lvl 20, after that things get hard. The classes that are easy to solo after level 20 (IMO) right now are druids (kiting) and necros (nice pets). HOWEVER, on the test server they are implementing changes to 'nerf' both these classes. There is coding (again on the test server) which prohibits DOT spells from taking affect if a MOB is running toward the caster, effectively destroying kiting as a tactic. Also the strength of the necro pets has been severally downgraded.
I support both these changes. Kiting is a cheap tactic that allows druids (to a lesser extent shamans) to solo red cons all day and level pretty darn fast. And the necro pets are REALLY strong right now. I grouped with an equal level necro the other day and his naked unarmed pet out-fought me. Not only that, but the pet heals back to full in about 5 seconds after combat. How fair is that?
It remains to be seen if Verant brings these on to the mainstream servers, but they do seem to be adjusting two of the most powerful (and easily soloable) classes back into the fold so to speak.
