Standard Operating Procedure (SOP) for issues that come up in game.  If you would like to add to this list, please e-mail your suggestions to us:

  1. Turn logging on Use the /log on command.  This will give us hard evidence of any harassment/thievery etc. etc.
  2. Screen Shots Take any screenshots you feel would be vital to help your case.
  3. Get witnesses If you have a Clan mate or friend close by have them come along to see what is going on.
  4. Petition If the situation calls for a petition, by all means petition.  Type /petition <text of your complaint>.
  5. Report For situations that are serious use the report command.  Type /report <toon's name here> 

    Now here is how the report command works;

    In game you are being harassed, you have already turned on text log, taken screen shots, got your witness, but the harassment just wont stop so do this.....

    Get the text of what the person is saying to you in your chat text box. When you use the report command what ever text is visible by yourself in the chat window is exactly what will be sent to the GM's, use the /report <toon's name here>. The text is then sent to the GM's for review. FOLLOW UP the report command with a petition. Both of these things done will get results, albeit usually slow responses :=)

If we can remember to do these things we can put a stop to those who wish to mess with Clan X!!!
