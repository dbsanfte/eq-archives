New Server to be in Amsterdam 3/26/02

The EverQuest team has announced a new server, named Kane Bayle, that will be located in Amsterdam:

We're opening a new server, the second EverQuest server to be located outside of the United States. The new server is called Kane Bayle and it will be located in Amsterdam. We will be opening the server as a test server for a while to help test network and connection issues, so we need to have a lot of people testing it out. You will be able to access the new server by running TestEverQuest.exe. Once you do, Kane Bayle will be one of the servers on the Test Servers selection screen. Remember, though, that this is a Test server and all of the Test server rules apply.

Important note: Because this is a Test server, we will be wiping characters when we take the server Live.

Keeping that in mind, we want to encourage you to log into the server and play for a while. We really want to see how it holds up under a full server load, and for that we need players logged in and playing. Now is also a good time to try it out and see if Kane Bayle will provide you EverQuest with a better Internet connection. So please give it a try.

Thanks for playing EverQuest, and have fun on Kane Bayle!

- The EverQuest Team

Changes on Test Server 3/15/02

These are some of the changes that are being seen on the Test Server.  This doesn't mean that they will go live, but there is hope!

(from EQ Castersrealm)

A new command has been added called /rtarget. This command automatically targets the last person who sent you a tell. This makes it much easier to buff during raids. It also has good value for people needing support heals during fights.

It appears that Gnomes will soon have the ability to play both Shadow Knight and Paladin classes.

Halflings also have the ability to now be Rangers and Paladins

According to players who have been enjoying life as a Gnome Paladin or Halfling Ranger, if you die while level 10 or below you now respawn at your bind point with all your gear.

That's right, no more corpses for players under Level 10! This should significantly help in reducing downtime or lost items due to corpse decay.

Chat channels have now been added to the test server.  This long awaited ability is excellent for improving group communication on raids without spamming zones or guild chat.

These channels can be password protected when creating the channel.  They are used with the following commands (test only currently):

  • /join : (e.g: /join casters:seekrit)
  • /chat (text)
  • /leave

Some fairly exciting possibilities are coming it looks like!

Healing Spells updated 3/12/02

It was brought to my attention that I had an oops on the spells.  I was listing the information straight from the spell database, and in the case of our healing spells, that added a problem. 

For those that don't know, Druids have a 10% penalty on healing spells.  So if the database shows 500 hps healed, we actually only do 450 hps at a cast.

Anyways, I'm going to leave the base number from the spell database in there still, but under the description I've changed the actual healed hitpoints to the more approximate number.

Cazic Thule redone! 3/8/02

For those that hadn't heard Verant has completely redone the mobs in Cazic Thule. 

It is still an outdoors zone, manastones will not work in the zone.  Mobs are VERY tough.  They are all mid to high 50's now, cannot be mezzed, and harmony will not work on them!  They hit in the 125-250 per round and some of the can complete heal.

So if you want to see a stack of corpses by the zoneline, go check it out :)  It is a VERY nasty zone now.

The items that used to drop there, such as ravenscale armor and such have all been moved out into Feerot onto new mobs.
