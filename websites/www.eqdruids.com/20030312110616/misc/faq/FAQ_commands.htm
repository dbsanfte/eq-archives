Keyboard Commands FAQ

 

Below is a list of the current keyboard commands available.  If you are new to the game, we recommend that you take some time and get familiar with these commands.  We also recommend that you re-map any of these commands to whatever keys you feel most comfortable with using.

One key we HIGHLY recommend re-mapping is the Auto-attack key.  Preferably to something not commonly used, such as the "~" key.  This will save several unnecessary deaths.
