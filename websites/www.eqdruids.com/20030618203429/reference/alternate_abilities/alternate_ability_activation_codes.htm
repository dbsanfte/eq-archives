It is possible for you to activate an alternate ability through a hotkey.

 

The command you use is /alt activate #

 

Below you will find a list of the corresponding numbers for different AA's.  I've included all the ones that we've found so far.

 

If you know of one we've missed, please send to me at: spiritual@eqdruids.com so we can add it to the list.
