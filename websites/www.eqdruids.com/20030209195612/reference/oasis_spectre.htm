Geimhreadh Draiocht's Guide to Oasis Spectre Quading

 

V1.2

If you are lower level, orbit the island and harmony specs B/C/D, Proceed to waypoint #1. 

1)      Target E(click on doorway if u cannot see him and you can target him), Harmony E, Clear Target, Face A, Target A, Face B, Snare A, Clear Target, Target B, Run to 2. 

 

2)      Face C, Snare B, Clear Target, Target C, Run to 3.

 

3)      Face D, Snare C, Clear Target, Target D, Run to 4.

 

4)      Snare D, Dive to Water Level, Round Up A-D While Over Water, Proceed to 5.

 

5)      Allow A-D to Get Near 5, Proceed to 6.

 

6)       

 

a)      Allow A-D to make Landfall.

b)      Remain Out of Spell Range (Let them come to you), AOE Nuke At Maximum range, Proceed to 7, Allow AOE Nuke to Refresh.

c)      Stop, Repeat 6-b.

 

7)      When A-D Die, Loot and Sell at Inn, Return to Start Point, Med.

 When Full Mana I would just run up, snare the remaining 4 specs and quad them along the same path towards inn, you can target all 4 specs from the ground, start with spec E.  I also found that as I became higher level, Spec agro radius decreased.  Eventually, harmony became unnecessary as only one spec would agro per snare.

I moved to Oasis to quad around level 43 and I believe I was getting 1 full blue bubble per quad.  I would say I received 25-30pp per quad as a minimum.
