Site Updates: [20-April-2003 @ 20:00 SYD]
Sorry about the loooong delay in getting everything back online. The site really was done within 2 hours of putting up that temporary "maintenance" page. The problem came when my computer died. Power supply, motherboard, even the case! As a result, I lost all data on BOTH my hard drives... I just got the system somewhat up and running today, although it's still randomly crashing and rebooting itself for no aparent reason..../rude Micro$oft.

Anyway, the site is here. I have the Calendar block, as well as the Gallery (for some reason that one did not port over. I am going to try a different calendar - CalZone was nice, but I think the administration and customization of this new one will be nicer.

Finally, the message boards are up and running. The problem with them lies in that version 2 uses a totally different database than the old version 1.4beta that we were using before. Everything works now that did not before, but the old forums and messages are still in the old database, waiting for me to figure out how to port them over. The table structures are completely different, also, so a simple (? over 5,000 messages!) cut & paste is out of the question. There should be a converter/import script out there somewhere - I just haven't found it yet. ;)

Oh, one other thing on the forums... The first time you visit them, you will be "registered" in them. I hacked the core of the user script on the site to do this automatically, so all your information will be the same. The only thing you might have to do is re-type your auto-signature file, which is now done from your Preferences in the forum itself, not the My Account link to the main site.

Once that is done, please send me a private message on the site and let me know which guild you are in and what your rank in that guild is. I hope to have this step automated for new members in the near future, but for now, I'll just do it manually. This is important, because the forums security features are now working, as promised!

That's about it for now. Still lots of work to be done. Since I still can't seem to get into EQ for more than 5 or 10 minutes at a time without crashing, I'll focus on the site for a while. Please shoot me your bug reports, suggestions, etc, also. Thanks!

-X

Site Updates: NEW! Vazaelle.NET version 7.2.3
Beginning today and lasting (hopfully only) over the next 3 days, I will be implimenting the next generation of Vazaelle.NET.

This is a TOTAL OVERHAUL of the site. EVERYTHING will be new.

The software will be the same, but is a total rewrite of the currently installed version. The message boards will finally be updated to a Release version (as opposed to a pre-alpha like the current ones). I will also be upgrading PHP and MySQL on the server, as well as installing some patches to Linux.

Although I am doing this from over 30,000 miles of where the server is physically located, I really do not expect any problems to occur. I either already have, or are in the process of backing up the entire server on my local computer, just in case, and also have a backup job in at the host.

I am taking every measure to ensure that no data, especially user data, is lost. After reviewing the code and database structures of the new version of the site, I do not anticipate any data compatibility issues to arise. Same for the message boards. With luck, the site itself will be back up and running after being offline for about 20 minutes (time to upload and run install scripts). While that is happening, I will have a temporary page up.

Also with this new update, the KGM and NATO sites will no longer be found at this URL. I will post the new URLs when the site is back online, but I think you'll figure it out. ;)
