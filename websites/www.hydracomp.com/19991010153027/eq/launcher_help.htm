GENERAL INFORMATION

What is UniNet ?

UniNet is a new concept in Software Application Deployment and Updating.  We feel that UniNet concept will take computer software to the next level, in creating a new means by which users are able to install, and keep software applications current.   By using the UniNet platform, users can access a single location to see which UniNet compatable Software Applications they have installed on there system, select new software applications to automatically be downloaded and installed, and to ensure that the latest components of that software are always the most currentl that are avalible.
What is the UniNet Launcher ?
The UniNet Launcher is the magic program which helps make all of this happen.  It is responsible for both ensuring that applications are using the most current and up-to-date components each time they are run, as well as providing the means by which users can download and install new UniNet compatable Software Applications, and provide a central location to set global UniNet software settings, such as Proxy Server configurations, etc.
Who develops UniNet Software?
Currently, there are only two software applications being developed to use the UniNet platform, EverMon and EverGuild, both of which are utilities for a Mass-Multiplayer Internet game called EverQuest.
 
However, once the UniNet platform ist stabalized, there will be information provided that will allow other software developers to create UniNet compatable software that users of the UniNet Launcher will be able to be informed about, and automatically download and setup.
 
More information on this topic will be avalible in the future.
Where to get the latest version of the UniNet Launcher ?
Once you have initially downloaded and installed the UniNet Launcher on your system, you will not need to worry about getting new versions of the launcher, as they will automatically be downloaded when they are avalible.

If you have not yet downloaded and installed the UniNet Launcher on your system, you can go to the following website to get the latest version :

http:\\www.hydacomp.com\eq
The latest UniNet news, as well as other UniNet software app news, and other information can also be found at that same site.
 
Where to get the latest version of UniNet Software ?
Once you have initially set up a software application using the Launcher's Configuration mode, then you will always automatically get any future version upgrades for that software application, and no further work on your end will be required to get it.  That's one of the benifits of using the UniNet software platform.
Why are some UniNet Software Applications listed as BETA ?
Several of the software applications which are currently using the UniNet architecture are works in progress themselves.  Because of this, you may see two listings for each software application.  The first listing listing the name of the software, snd the second listing the same software name, with BETA at the end of it.  The BETA Versions of the software are exactly that, they are pre-release versions of the applications which are used for testing.  You should only select these versions if you are an experienced computer user, and are prepared to have to work through bugs, and do light troubleshooting.   If you are only looking for stable apps, then it is suggested that you stay away from the BETA versions of the software, and only use the Non-Beta versions.

INSTALLATION INFORMATION

Installing the UniNet Launcher

1. DOWNLOAD - To Install the UniNet Launcher, you will either need to download uninet-sm.exe (about 350k), or uninet-lg.exe (about 1500k).  If you have previously installed EverMon, or some other application which uses the Visual Basic 6.0 Run Time Libraries (msvbvm60.dll and related files), then you can get the small version.  Otherwise you will need to get the large version.
 
NOTE : It is suggested that you DO NOT save the downloaded file on your desktop.  instead use a directory like C:\SOFTWARE, or D:\DL, etc.
 
2. INSTALL - After you have downloaded the software, you will need to run it.  It will ask you the temp directory to use.  In most cases, you can simply press OK and use the default.  This will launch the VB Installer, and walk you through the setup process.
 
NOTE : If you have problems with the installation process see Section 4a : Trouble Shooting Installation Problems at the bottom of this manual.
Installing UniNet Software Applications
Once you have completed the installation process, you should have a UniNet menu in your Start | Programs menu.  The only icon which should currently be there is one called Launcher Configuration.  By selecting this icon, the UniNet Launcher Application Configuration process will be started.
 
 
Once the Configuration mode starts, you should get a screen such as that shown above.  This will list all the currently avalible UniNet applications.  Additional applications are automatically added to your list as the become avalible.  For each application, the Status will either say INSTALLED or NONE.  Applications that say 'None' are not installed.
 
If this is your first time running the Launcher Configuration, all of the applications will say NONE, even if you ran a previous version of EverMon.  To install an application, double click on it.  The installer will then ask you if you wish to install that app, and will then take you through a few steps needed for the basic setup.
 
After you have finished setting up applications, and Close the configuration program, if you go back to the Start | Programs | UniNet menu, you will notice that new Icons have appeared that are listed as the names of the applications you just selected to install.  Select those new application icons to complete the install, and to run the application.

 
CONFIGURATION INFORMATION
 
Configuring UniNet Proxy Server Settings
All UniNet Applications have built in support for Socks 4, 4a, and 5 Proxy Servers.
 
If you are on a LAN and do not have direct Internet access, or are behind a Firewall, or need to connect to the net via a Proxy Server, then you can configuration UniNet applications including the UniNet Launcher to use your proxy server.

To access the configuration, go to the Start | Programs | UniNet menu, and select the Launcher Configuration icon to get into the UniNet Launcher Configuration screen.  Once you are there, press the Proxy Setup button.  You will see a window that you can use to set the server location of your proxy server.  The port is optional, if you leave it blank, then the standard proxy port 1080 will be assumed.


 
TROUBLE SHOOTING INFORMATION
 
Trouble Shooting Installation Problems
If while attempting to install either the small version (uninet-sm.exe) or the large version (uninet-lg.exe) of UniNet Launcher, you experience one of the following problems :
  1. You get a message saying that setup1.exe can not be started, or the process can not be created
  2. You get a message saying that setup1.exe or setup.exe is currupt, or invalid
  3. You run the uninet-sm.exe or uninet-lg.exe install file, but after it decompresses, instead of getting UniNet 1.0.2 Launcher install screen, you get an install screen for some other application
If you have a problem that matches one of the problems above, or a problem that is very similar to the problems listed above, this may be due to a known problem with the Microsoft Installer, which UniNet uses.  If components of the install files exist in your windows directory, or in your system path, then the installer tends to use those files, instead of using the real files it should be using.
 
In order to fix this problem, you will need to remove the old components from your system.  The componenets should not be there in the furst place, but may end up being left behind when previous installers failed to completely delete all the temperary files they use.

Here is the step of procedures you can take to attempt to fix this problem :

  1. Make sure you our out of the setup program.
  2. Go to Start Menu | Find | Files or Folders to search your hard drive for the file : launcher.cab
  3. If you find any, then hightlight the file, and press DELETE on your keyboard to remove it.
  4. Follow the same set of procedures for Searching and Deleteing any setup.lst or setup1.exe files.
One you have finished cleaning the files, re-run the uninet insall program (Either uninet-sm.exe or uninet-lg.exe, depending on which you need).
 
If after taking these steps, you still can not get UniNet Launcher installed, then please send me an e-mail, describing the exact problem you are having, as well as any error messages you may have been given during the installation attempt.
