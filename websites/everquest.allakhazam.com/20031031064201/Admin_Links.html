Everquest

Items:

  • Fast Item Search -- Please look up every item before you enter it to make sure it isn't in the database.
  • Add new items -- for adding a new item.  Images should be entered as follows:  ---  graphics17/name_of_item.jpg (or gif or whatever) ---  If I change it to 18, I'll let you know.  Forward graphics to me to upload.
  • Creature to Item Table -- Use to link creatures to items they drop.  It will link every creature that comes up in the left column with every item that comes up in the right, so make sure you remove any that don't belong after you do your search.
  • Edit Item Modifier -- If the item has a modifier, use this page to add it.  You will need to replace xxxx with the item number.  (you can also get here from the edit page, but this is faster)
  • Edit Skill -- If you find an item that modifies a skill not on the list, you can add it here.  Make sure forum says "no".
  • New Items -- This page lists all items recently added.
  • Edit Spell
  • Fashion Image Admin
-- Note, to edit an item, click on edit in the description.

Quests:

  • Add new quests -- please check to make sure the quest is not already entered.  It may have a different name.  Copying and pasting a quest from an e-mail usually results in page breaks being artificially added.  The only way to fix it is to go through the whole quest description after you paste it and delete out the breaks.
-- To add a quest item, use the add new item link and check off the quest item box.  You need to add quest items before you add the quest.
-- Note to edit a quest, click on edit in the quest description.

Trade Skills
  •  Basic Recipe Search
  •  Full database search
  •  Tradeskill Recipe Entry Form. -- Item ID is the database ID of an item.  If you enter the info wrong, you can update it by going to that recipe and hitting the update button.  Back arrow brings you back to the previously filled in form, so that in a series of recipes where only the name and the amounts change, you can enter the recipe, back arrow, change the name and ingredients and enter the next one.

 

Dark Age of Camelot

  • Add Item -- You need to add the stats first and then hit enter.  That will bring up the screen to add the modifier.  Condition and Durability default as 100%.  Quality defaults as 80% for storebought items.  Make sure you change the realm slot (I miss it a lot since it is off to the right).  Images should be entered as follows:  ---  images1/name_of_item.jpg ---  If I change it to 2, I'll let you know.  Forward any graphics to me to upload.
  • Add/Edit Quests -- Check carefully to make sure the quest isn't already there.  It may have a different name.
