Patch Server
The patch server has been up for a while, but you may be able to reach the host. Here is a possible reason:

The Patch server is up. Unfortunatly, if you have been trying the patch server over and over again, the DNS info of the old patch server will be cached on your DNS box.

1) Try the patch. If it works, you are home free. If it doesn't, go to step 2.

2) If you are running Windows9x, create a file in your Windows directory called HOSTS (no extension). If you are running Winnt 5.0, create the same file in your WINNT\SYSTEM32\drivers\etc directory. On the first line of this file, put:

209.0.216.89 patch.everquest.com

This will override the DNS lookup for the patch server.

If you can't get to the patch server without doing this, it will be anywhere from 1 to 24 hours before you'll be able to patch.

If you choose to use this bypass, be sure to remove this entry from your HOSTS file later tonight or tomorrow after the local DNS record has refreshed. You can test this by removing the entry and pinging patch.everquest.com. If it returns the IP address above, the record is updated. You want to be sure to remove the entry so future DNS updates will take effect.

-Abashi, Senior Guide
