Oddly enough I was looking for this info today. Here you go:

Text color can be changed ingame with the following command

/usercolor X # # #

X = the corresponding number for a kind of text (there is a table below)
# # # = The RGB combination that corresponds with different colors (there is
another table below)

For Example: to change guild chat to brown.

/usercolor 4 115 80 40
