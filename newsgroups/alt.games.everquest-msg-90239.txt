Ugh what a pain to change!  I wanted to do the same thing, it's a long
command so if you wanna go between different settings often I'd set it on
one of your socials.
Basically the command goes like this:
/wincolor <window name> <redvalue> <greenvalue> <bluevalue> <transparency>
window name can be: Chat, Spell, Player, Party, Target, Buff, Data, Main,
Inv, Track or Hotbox
transparency: 0, 1, 2, 3, or 4  I think 0 is a waste of time so start at 1
To figure out the colors I use this chart
Black 0 0 0
Red 255 0 0
Orange 255 153 0
Yellow 255 255 0
Green 0 255 0
Light blue 152 255 255
blue 0 0 255
dark blue 0 0 63
violet 128 0 128
white 255 255 255

A sample command for green background in chat window  /wincolor Chat 0 255 0
2
