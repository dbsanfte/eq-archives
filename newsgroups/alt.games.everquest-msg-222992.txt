The new 3d targeting indicator has an .ini file associated with
it.  TargetIndicator.ini

In this file there is a section for:
Green ([Trivial])
Cyan ([VeryEasy])
Blue ([Easy])
White ([FairMatch])
Yellow ([Difficult])
Red ([Deadly])

In each section there are Red, Green, Blue and Alpha (0=transparent,
255=opaque)

The default is
Green - R=0, G=255, B=0, A=240
Cyan - R=0, G=255, B=120, A=240
Blue - R=0, G=0, B=255, A=240
White - R=255, G=255, B=255, A=240
Yellow - R=255, G=255, B=0, A=240
Red - R=255, G=0, B=0, A=240

So - instead of bitching about the addition of the target ring, you
should be thanking the devs for finally putting in a way for colorblind
people to see the differences in con color easily.
