    It would look close. The ratio of, say, sale prices at two different
levels of cha/faction would be:

    Px(1 + C x N(cha1,fac1))/Px(1 + C x N(cha2, fac2)) =
[1+CxN(cha1,fac1)]/[1+CxN(cha2,fac2)]. To first approximation, for CxN
larger than one, this ratio will roughly equal CxN(cha1,fac1)/CxN(cha2,fac2)
= N(cha1,fac1)/N(cha2,fac2) - which is what you've observed. I imagine the
only cases where you notice the variance in price is when the item's sale
price changes significantly when you change CHA, which is when CxN >> 1,
which is where the approximation holds.
