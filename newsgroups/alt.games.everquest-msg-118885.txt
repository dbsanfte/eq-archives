Be careful. Many clone builders will use cheap motherboards with
integrated videa and/or sound. The MB is the heart of the system. Make
sure they are using a name brand, and that the board is using one of
the latest chipsets from VIA or Intel.

Avoid RDRAM memory (labelled PC600, PC700, or PC800). You want to be
sure and get DDR (labelled PC2100, PC2600, etc). RDRAM is more
expensive MUCH harder to upgrade later, and a dead end technology.
