There is an XML file in the Everquest directory that describes what is
loaded. Find the reference to old world textures and change True to False.

I found this because I regularly patch one computer and then copy
updated/new files to the others. One of them is lower spec and I didn't
install the old world textures. I accidentally copied this XML file over and
EQ blew up at next launch because it tried to load uninstalled textures!

Change it at your own risk ....
