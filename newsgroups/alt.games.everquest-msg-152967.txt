Some ideas.

1. Use "at" or "cron" from root on a Linux box on your network to do

    ping -f -w 300 -s 1000 ip_address_of_EQ_box

That will cause the Linux box to flood the EQ box with 1000 byte ping
packets for 300 seconds.  Perhaps that will be enough to cause LD?

2. Put the EQ box behind a Linux box, with the Linux box providing IP
masquerading for the EQ box.  Use an "at" or "cron" job to turn off
masquerading, or have the "at" or "cron" job use iptables to change the
firewall rules to block EQ.  This will cause LD.
