> Any way to always allow changed programs without getting promped?

If you start up EQ and see it patching the main program, wait for the patch
to finish then bring up the ZA control panel. Set the program control to Low
(Learning mode) then play EQ. When you are finished playing, bring up the ZA
control panel and set the program control back to what it was. You could
probably even Alt-Return out of EQ once it has started and set ZA back to
its normal setting.
