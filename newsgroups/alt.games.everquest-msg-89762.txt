Not specifically, but here's a repost:
-------------------------------------------

The /usercolor command

syntax:

/usercolor <number> <Red value> <Green value> <Blue value>
with a space between each < >

the numbers for chat lines are...

1  Say
2  Tell
3  Group
4  Guild
5  ooc
6  Auction
7  Shout
8  Emote
9  Spells
10 You hit NPC
11 NPC hits you
12 You miss NPC
13 NPC misses you
14 Broadcasts
15 Skill messages
16 Special Abilities
17 -not used-
18 Default text (what you type)
19 -not used-
20 Merchant's price offer
21 Merchant's Buy/Sell
22 Your death message
23 Others death message
24 Others damaged
25 Others missed
26 Who
27 Yell
28 Non-melee hit (magic)
29 Spell worn off
30 Money spit
31 Loot message
32 Dice roll
33 Others spells
34 Spell failures


Red/Green/Blue values are from 0 to 255

Some common colors:

Black  0 0 0
Red  255 0 0
Orange  255 153 0
Yellow  255 255 0
Green  0 255 0
Lt. Blue  152 255 255
Blue  0 0 255
Dk. Blue  0 0 63
Violet  128 0 128
White  255 255 255

To reset to default colors:  /usercolor reset
To reset a specific chat text color:  /usercolor # reset

Example:  to set "NPC misses you" to orange

/usercolor 13 255 153 0
