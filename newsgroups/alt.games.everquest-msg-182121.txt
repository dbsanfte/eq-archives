ALT-TAB only works if you have another application running. For instance, if
you have EverQuest and Internet Explorer running, you should be able to
ALT-TAB between them, but if you only have EverQuest running, you cannot
ALT-TAB to the desktop.

However, having done the ALT-ENTER, you should be able to either use the
WINDOWS key to bring up the START menu and launch a program, or hit
ALT-SHIFT-R to activate the mouse to the desktop and access the START menu
that way.
