Its been a while... but IIRC this is what I did.

In your Everquest main directory there should be a file for each
player you've ever logged in as with the filename structure:

UI_<playername>_<servername>.txt

Pick the appropriate one and inside of that search on
'ChannelAutoJoin' and remove the channel from the list. Save the file
and rerun the program. You can also search on the channel name and pw
in that file if needed.
