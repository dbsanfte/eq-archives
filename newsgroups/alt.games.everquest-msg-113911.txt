If you are running Windows 98/ME and install more then 512Mb of RAM you will
experience crashing or other problems in EverQuest. You do not need to
remove RAM so that you are less then 512Mb, you will only simply have to
limit the amount of memory you use. Please follow the step by step
directions below to limit the amount of memory Windows 98/ME will use, and
keep EverQuest from crashing.

1) Click on the Start button and select Run
2) Type msconfig in the "open" line and hit enter
3) Click on the Advanced button on the General tab
4) In the advanced troubleshooting settings please check the "Limit memory
to" box and set the value to 512.
