Lance Berg <emporer@dejazzd.com> writes:
> My only observation; the character select screen is ugly.  However, it 
> looks like we can now have 10 characters per server instead of 8.

Not quite.  Apparently you're entitled to an extra two characters
if you have a Station Access account, and due to a bug they're
letting everyone have the extra two characters for now, but they
warn that they're going to take them away again so you shouldn't
use them if you're not entitled to them.

Speaking of the ugly character select screen, apparently in order
to make it that ugly they had to make it incompatible with the
"default_old" UI.  If you get an all-gray screen when you expect
to see character select, you should look in your eqclient.ini file
and see if UISkin=default_old.  If so, change it to UISkin=default.
I don't know if there's a problem with default_old for characters,
but it definitely breaks the char select window.
