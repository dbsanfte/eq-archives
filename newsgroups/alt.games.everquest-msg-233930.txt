On Mon, 02 Feb 2004 16:23:18 GMT, noway@jose.com (Kobe008) wrote:

>Just recently got LDON and have only done 4 dungeons so far with all being 
>successful.  What is the command to autosplit the loot afterwards?  I wouldn't 
>mind being the ML but so far I haven't offered to because I don't know how to 
>split the loot.  Also does the autosplit split all cash I have on me so I 
>would want to start out with empty pockets?  Or does it know to only split 
>what I've just sold from the dungeon?

After you sell everything, type "/split # # # #", each # replaced with
the amounts of pp, gp, sp, and cp in that order that you want to split
The total, not each person's share. For example, if you came out with
300pp 500gp 250cp in the adventure, then it's /split 300 500 0 250.
