It isn't currently built into the GUI, however you can do it this way 
(this is in the release notes if you read them):

Go to C:\WINDOWS\Application Data\Mozilla\Profiles\<user name>\<random 
letters>

Then open up the file in there called prefs.js, in the middle of that 
file somewhere (not then end or begining wont work for some reason), put 
the following line:
user_pref("capability.policy.default.Window.open", "noAccess");

There ya go. Enjoy pop-up free browsing with no plug-ins and it stops 
the pop-up at the source so you wont see it at all.
