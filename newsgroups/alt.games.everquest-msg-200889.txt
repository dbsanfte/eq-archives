Yes, that is saying not to load a velious armor texture on to a luclin
model. You ALWAYS want that to say false.

It has NOTHING to do with loading a velious armor texture on to an OLD
modeld, which velious armor textures work on. Nor does it have
anything to do with using old models with luclin installed, it is only
referring to luclin MODELS.

I have that exact line as false in my ini, and am using velious
textures on old models. That way if I turn on luclin models, I don't
have to turn off velious textures.

here is a run down of the way I have this set up in my ini:

UseLuclinHumanMale=TRUE
UseLuclinHumanFemale=TRUE
UseLuclinBarbarianMale=TRUE
UseLuclinBarbarianFemale=TRUE
UseLuclinEruditeMale=TRUE
UseLuclinEruditeFemale=TRUE
UseLuclinWoodElfMale=TRUE
UseLuclinWoodElfFemale=TRUE
UseLuclinHighElfMale=TRUE
UseLuclinHighElfFemale=TRUE
UseLuclinDarkElfMale=TRUE
UseLuclinDarkElfFemale=TRUE
UseLuclinHalfElfMale=TRUE
UseLuclinHalfElfFemale=TRUE
UseLuclinDwarfMale=TRUE
UseLuclinDwarfFemale=TRUE
UseLuclinTrollMale=TRUE
UseLuclinTrollFemale=TRUE
UseLuclinOgreMale=TRUE
UseLuclinOgreFemale=TRUE
UseLuclinHalflingMale=TRUE
UseLuclinHalflingFemale=TRUE
UseLuclinGnomeMale=TRUE
UseLuclinGnomeFemale=TRUE
UseLuclinIksarMale=TRUE
UseLuclinIksarFemale=TRUE
UseLuclinElementals=TRUE
UseLuclinVahShirMale=FALSE
UseLuclinVahShirFemale=FALSE

(Turns on all luclin models.)


AllLuclinPcModelsOff=TRUE

(Turns OFF all luclin models, 
this does not work on vah shir which
is why I have vah shir set to false above,
to use the old model(there is no froglok old model)
This command is useful so you only have it to change
to jump from using all old models to all luclin)

LoadArmor17=TRUE
LoadArmor18=TRUE
LoadArmor19=TRUE
LoadArmor20=TRUE
LoadArmor21=TRUE
LoadArmor22=TRUE

(Loads velious old armor textures.)

(stops those old armor textures from working on luclin models,
since they are not made for them)

The reason I do it like that is so I only have to change ONE line,
well three if I want new vah shir enabled as well, to enable luclin
models.. I just turn this line to false:

AllLuclinPcModelsOff=FALSE

UseLuclinVahShirMale=TRUE

textures since i have that:

LoadVeliousArmorsWithLuclin=FALSE

in there.
