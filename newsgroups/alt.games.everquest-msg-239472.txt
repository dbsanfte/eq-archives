> Run Regedit ( Start/Run and type in " regedit " )
> Click Edit and Find and enter " gamingcomp " and check Keys, Values, Data.
> Click Find Next.
> When it has found the key " GamingCompatability " under the LogiTech drivers key,
> delete the whole " GamingCompatability " key and it's sub keys and reboot.
> 
> What that key does is disable LogiTech enhancements for your pointer device in the
> listed games or under the listed drivers ( like DirectX ) which would allow games to
> take over certain pointer functions. However EQ doesn't uses any native enhancements
> for pointers ( like wheel scrolling or multi buttons ), it instead uses the pointers
> drivers.... which so happens that key disables.
> You will also most likely notice an increase of mouse sensitivity in game, make
> adjustments via the LogiTech icon in Control Panel then tweak in game.
> 
> If you find another game that would rather use it's own control of the pointer
> device, reinstall the LogiTech drivers, reboot, Regedit and see if that game is
> listed in the GamingCompatability subkeys. If so, delete all the subkeys except the
> one you need.
