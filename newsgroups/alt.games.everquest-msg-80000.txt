What motherboard are you using in that system? If it uses a VIA
chipset, you need to load the latest Vis4in1 driver set and the latest
AGP GART driver set.

www.viahardware.com
4in1 version 4.37 
AGP Gart version 4.10

Next, dowload and install the latest 8500 drivers
www.rage3d.com
Win9X - 4.13.9005 (4.13.9008 is the newest. I am using the 9005s and
they run fine in EQ)
Win2K - 5.13.3281
WinXP - 6.13.3281

If that doesn't solve the problem go into the system Bios and set AGP
to 2X instead of 4X.
