Let me see if I understand this correctly.....

You want to have two EQ installations on the same PC, yes?

Just copy the first EQ installation to a new folder named something
like EQ2. Then create a new shortcut to eqgame.exe (or whichever
startup file you use), that points to the new EQ2 folder. You will now
have two seperate EQ installations, and both will be patched to the
current level of your main EQ installation.

EQ does NOT add any "required" entries into the system registry. You
can copy your entire EQ folder to another PC and run the game without
ever using the installation disks.
