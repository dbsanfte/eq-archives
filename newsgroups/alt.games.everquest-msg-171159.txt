> You create a new folder under the UI folder, and place the player made
> UI files in it. When you load the UI skin in-game, any missing files
> not included with the player UI will be copied automatically from the
> default folder to the player made UI folder.
>
> Directory Structure....
>
> <drive letter>:\<path the EQ main folder>\Everquest\UI Files\<Player
> Made UI folder>
>
> Example...
>
> 1. Download the KhoshUI.zip folder from the Internet.
> 2. Create a folder named Khosh under the EQ\UI files directory.
> (c:\Everquest\UI Files\Khosh)
> 3. Unzip the files (from KhoshUI.zip) in the the Khosh folder you
> created.
> 4. Start EQ
> 5. Login and choose one of your characters.
> 6. Type into the chat buffer "/loadskin Khosh" (the name of the skin
> must match the name of the folder you created.)
