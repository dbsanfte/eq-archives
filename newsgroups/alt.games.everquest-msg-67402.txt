Short answer:

Open your system.ini file.

Locate the [vcache] section.

Add (or modify) the following settings:
MaxFileCache=524288

This tells the disk cache to use a maximum of 512M.  I find the concept of a
half-gig cache a bit ludicrous, so my [vcache] section reads like this:

[vcache]
MinFileCache=16384
MaxFileCache=16384

This reserves 16M for the cache.

There is one other setting that you really want to have in there as well if
you have more than 128M.  Locate the [386enh] section, and add the following
line:
ConservativeSwapFileUsage=1
