The ingame command /usercolor xx xxx xxx xxx, where the first two x's
are the the number/s for the text you want to change. In your case,
the number would be 3, which is the group chat text. The next set of
three digits is red whith a range of 0 to 255, the second group of
three dgits is for the color blue and the third set of digits is for
green. For the command /usercolor 3 255 0 0 woould produce the
brightest color red availble for the text chat.
