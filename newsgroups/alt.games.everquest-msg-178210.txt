On Sat, 01 Feb 2003 04:08:58 GMT, MV <nospam@nospam.com> wrote:

>What rules are there for composing last names?  Once I saw a little 
>gnome whose name was "Amulet the Mystical".  So the last name is "the 
>Mystical"?  Can someone please direct me to a page listing the rules for 
>last names.  I am designing a character whose name I want to be "_____ 
>of Norrath" (I do not want to list the first name otherwise some people 
>might take it).  Thank you kindly!

Last names are a single word, one capital. No space. No apostrophe
unless approved by a guide/GM. If you see an unusual surname like "of
Norrath," it's because the person has won some sort of GM event and
been given a titular surname as a reward.
