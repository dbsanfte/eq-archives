Unfortunately, the server name or number does not appear in the log
anywhere.

There is another workaround for this, though.  The INI file name is derived
from whatever the name of the EXE file is.  So just keep multiple copies of
the EQLog.exe around, and rename them to match the server (eqlog-povar.exe,
eqlog-eci.exe, etc).  Then just run whichever one matches the server the
character lives on.
