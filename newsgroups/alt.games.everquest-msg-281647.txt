I haven't seen this mentioned in the patch notes or a.g.e, so I thought
I'd point out an unheralded addition to the UI.  (I found it when I went
to Options -> Keys -> UI to assign a key for toggling the Audio Triggers
window.)

At the bottom of Options -> Keys -> UI, you can specify a key (possibly
with shift/ctrl/alt, as usual) that will paste the current clipboard into
an EQ window.  I don't see any way to copy text TO the clipboard (or even
to make a selection), but this can still be a big help in some situations.
(E.g., copying quest text triggers from a browser or other window, or
copying part of an earlier log file, etc.)
