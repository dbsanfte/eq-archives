> I want to put my guild chat in a separate window....

If you right-click on the top bar of the main window you get a drop-down
menu.

Select the New Text Window (or similar, don't have an EQ install here :()

If you right-click on the top bar of the new window, you can use FILTER to
select what messages go to this window, and RENAME to make it more obvious.

Repeat for additional windows.

These settings are per-character and are saved in .ini files.
