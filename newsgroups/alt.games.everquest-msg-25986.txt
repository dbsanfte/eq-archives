>Totally reinstall EQ, then copy the entire EQ directory over to the

There's one subtlety in doing that.  If one computer has a Voodoo
graphics card, and the other does not, things may not work right off
the bat.  I copied my EQ directory from my main machine (GeForce 2
GTS) to my secondary machine (Voodoo 3).  I would then get crashes in
eqgfx_dx7.dll, when trying to go to any server from the server select
screen.

I checked the "Options" screen, and it said it was using the Glide
driver.  I changed it to Primary Display Adaptor, and it still failed.
Changing it back to Glide made it work.
