Here's a quick checklist of items that might cause intermittent lag
and slowdowns:

1. Disable virus checking software
2. Disable MS Fastfind
3. Disable ICQ
4. Disable Windows 98 automatic Update
5. Disable AOL messenger
6. Disable PointCast network
7. Close Mail/News readers
8. Disable System board polling (ie temperature, fan, monitors)
9. Defrag drive with EQ
10.verify there is enough room on drive (including root drive for swap).
11. Disable Task Scheduler
12. Disable Web/FTP servers on the machine you are using (you'll be surprised
how many ppl run an ftp server on their eq box and complain about lag =P)
13. Can you see the pattern yet? =)

Basically, if you can time the lag intervals pretty acurately, then it
is possible its comming from your end.

My ritual is:
reboot computer
disable, close all internet/system tasks
run EQ
