There is rumored to be an 1100 "soft cap" on AC . 

" Soft" cap meaning you can have an AC above this point but you will show decreasing marginal returns for the additional AC (like 200+ Int for casters).  There is no Hard Cap (i.e. an upper limit of what the game will dsplay) as far as I know.

However, noone has proved this soft cap really exists via logs (i.e. no one has done a study using a log parser above and below 1100 AC to show the effectiveness of additional AC above & below this point).

If you are interested in more info go to the eq.stratics warrior class board...there is a long thread on it.
