I don't understand how you can be so right (IMHO) in one analysis (your petition regarding warriors shortfalls & EQ dynmaics) & be so missing the point in another (the RL/Fantasy interconnection):

Stephen is pointing out that in order for a fantasy based game to have creedence it must have some modicum of connectiveness to RL life principles. Why? Because it builds an illusion that you, the player, are really in the game (artificial immersion). His examples of RL sword play just illustrate the pragmatic limitations of swinging a sword (and getting pooped as a result).

I think EQ is justified in exaggerating these limitations in order to propagate the illusion that we are like Conan the barbarian. This illusion is what makes ROLE PLAYING work. Ask any old D&D player (ever heard of Gary Gygax? people used to quote him like this "Brad" feller).

The game gets too cartoonish (and mechanical) when these things get too far apart though. I think the jist of this thread was that there needs to be a balance between the 2, strict RL principles & pure fantasy through discussions regarding Stamina etc.

It would be nice to see this conversation continue down this path rather than see discussions regarding one persons physical prowess (or perceived prowess)
