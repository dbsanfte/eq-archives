Speciallization - Verant is already working on this, and the problem is
certainly not class specific! Don't bug them when they are already working
on it!

Haste Spell (for fast travel) - Don't push this one...druids and Shamans
have Spirit of Wolf and I don't really think this is appropriate for
Wizards...maybe the Line of Sight Shadow Step idae would be better...

Max Meditation Skill - Don't bother, we already have a way to decrease our
downtime with meditation...anybody ever played a warrior? lol 20-30 MINUTES
of healing at high levels!

Chat server access while medding - I don't think this will ever

run multiple programs with EQ (which is GREAT for keeping the hack programs
at bay!)...allowing for the code to attach to a different server is probably

Official Feedback on Planned Changes - that's what the test server is all
about...and I, personally would rather if Verant's people were working on
tweaking the code rather than posting messages on the Web about what the
WOULD be working on if they weren't writing HTML updates on things they were
changing...

Keep the letter SHORT and to the POINT.
Don't include complaints about other classes...everybody thins THEIR class
is underpowered (trust me, I've heard from just about
