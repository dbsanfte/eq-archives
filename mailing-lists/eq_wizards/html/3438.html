On Specialization:

Put one point into all 5. There is (or at least used
to be) a bug if you didn't. Then put more points into
the one you want to bump up (for us wizzies, that's
Evoke), and watch them all go up...just make sure the
one you want goes above 50 first. All the others top
out there.

On Research:

There is a "Practice" research, which you can use to
raise your skill. Check www.graffe.com (yet again)
for the details. I was in the middle of things when
it was implemented, and I'm trying to work my way back
in now, but I don't remember any details.

Good luck!
