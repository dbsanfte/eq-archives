Subject: Re: O'Keill's Radiation


I disagree. I think O'Keils radiation is a waste of time when soloing. The reason is because it only lasts about 15 seconds or so. If you are in a battle with more than one creature, you're probably going to die anyway. Not all the time, but a LOT of the time. O'Keils might be ok against a lot of creatures. But it doesn't do enough damage to kill anything by itself.. So, you'll die before they do. (except goblin whelps, etc) ..and if you're fighting those still, then you should be trying for one on one fights only. Wizards are basically looking for one on one fights when solo'ing, so they can time their castings between hits. O'Keils only does 2 dmg per burn, so at higher levels it is almost useless. The only way to rack up dmg on these is to cast them repeatedly. It doesn't stack, so you've got to cast on others. IE. Tanks. Verant made the game this way on purpose.
