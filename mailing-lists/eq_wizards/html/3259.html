The winmodem shouldn't be the cause. Unless you have a P70 or something,
you should have plenty of CPU available for processing. The symptoms
are those of a server-specific route problem: you can patch (different
network), log-in (different network), select world (again a different
network), but cannot enter a specific world.

When this occurs, check other worlds. If you can log on some, you have
a route problem. Now, it might be an ISP-related problem, or a UUNET
problem (if I remember right Cazic is a UUNET hosted farm. Try Quellious.
If you go to the character screen, there you have. UUNET blocked, CERFnet
ok). But in any case, there's nothing you can do on your own. Except
changing ISP, which will only change the frequency and the time where
the next IP problem will occur.
