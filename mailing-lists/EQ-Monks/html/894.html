The reason it didn't work is because you are too high and they run away from
you. To get it to work, when the mob runs away, you need to run away too,
having SOW or jboots works best, and you really need to be in an outside
zone. Run away and turn around so you can see the mob, if you are far enough
away, you'll see it runnin back for you, that's when you FD. Then whoever
kills it will get exp. If it's a mob that doesn't run away, just make sure
when you FD, it stops attacking you. If it doesn't then you didn't clear
your dmg.
