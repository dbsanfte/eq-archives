A side note for the disarm issue. the only time I reccomend using disarm is
when the mob is using a very fast weapon like a dagger or a rapier etc..
what we know is that if the mobs hand damage is the higher damge weapon it
will use the damage from it hands. if it happens to be holding a fs dagger
then it will use the speed of the dagger but not the damage of the dagger.
so it would be doing its hand damage with dagger speed.
