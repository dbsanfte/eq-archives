get ur Window OS disk, got to control panel, then go to Networks. Then
go to ur Window Start button, go to Find Files/Folder, look for the
winsock.dll file under ur C drive. Delete or rename the winsock.dll. then
close off the Find File/Folders window. Then back at the Control Panel
Network, Remove the TCP/IP. Then click on Add -> Protocol -> Manufacturer
(Choose Microsoft) -> Protocol (Choose TCP/IP) . Then click OK all the way.
