For those who have send me mails abt hacking, just wanna let u know tat
hacking ONLY works for open chars. My only objective is to test out all
the skills. This hacking WILL NOT WORK on closed char :)

There are few methods in hacking:

1. FPE2000 (Fixed People Expert 2000) The most popular hacking tool in
taiwan. I guess all those who read chinese game magazines or play chinese
games will find it familiar. Basically this s/w allow u to 'memorise'
memoery and track all the memory when a certain stats changes (eg xp).
After tracking few times it will ultimately find out the memory place to
store this particular stats. So far there is no single solo game I can't
hack with this s/w. Pretty powerful but only work if u can read chinese.
This s/w allow many taiwanese to whack the american upside down during D1,
hehe. Cannot d/l, must get from SLS, CD version.

2. Cheat'O'matic. Work same concept as FPE2000. Allow memory hacking.
But less powerful, can hack one stats at a time (but u can open few
windows at a time, one for each stats like money, hp, xp etc). The only
advanages over FEP2000:
- very small program, can d/l from the net, shareware I think.
- some games will crash when I use FPE2000, but this one less prone to
crash, slower btw.

3. Ice/hexedit
Only for those who know hex editing. Basically only works on save game
files. Since open chars are stored in ya PC, u can find tat file and hack.
Bear in mind tat Diablo dun use straight numbers. Eg in D1, ya hp is a
fornula of x4, ie if ya hp is 100, u have to hack the hp data to 25 (since
25x4=100).
Hededite is a file tat can be d/l from the net.
