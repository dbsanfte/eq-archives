Subject: D2 Patch 1.02 & Realm Status


Patch 1.02 is Live - Gaile [11:36:PST]

Battle.net now automatically updates when you open with Version 1.02 of
Diablo II. So, what have we here, in the patch?

Major Bug repairs include eliminating the "same character name" deletion
error; corrections for a memory usage problem that sometimes made the game
slow down after playing for a while; a fix on CD ejection problems.

Minor Bug repairs will make the "Esc" key work more smoothly in closing
boxes and pop-up screens and will allow faster drawing of the auto-map

Improvements will close some NPC menus when necessary; allow toggling of
the automap when a waypoint is active; reformat game log file names to
follow the D2YearMonthDay.txt format.

Also repaired in the "Game Hint/Spoiler" category is a portal bug in Act
II, details of which you can learn about in the "update" file when you
patch.
The servers remain down, and the scrolling red text on Battle.net mentions
they will probably be up about Noon PDT today.

(Source: http://www.diabloii.net/)
