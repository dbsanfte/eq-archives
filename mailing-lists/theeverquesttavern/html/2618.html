OK, let's see what we can do here. First thing, if you haven't
already, you should visit the official EQ Tech Support page: go to
http://eqlive.station.sony.com, click on Technical Help on the left.

First, your processor. A Pentium II 200 MHz will not cut it, so that
would be the basic problem. Your RAM should come in 64 or 32 MB
increments, so you probably have 128, 192, or 256 MB. You could have
224, but that would be an odd amount; if it's cobbled together using
different kinds of RAM, that could also be a problem (but not one
that would prevent you from running EQ).

At 256 MB of RAM, if you're running the Luclin expansion, you will
only be able to use about half the new models.

Win NT and Linux are different operating systems - you probably have
Windows 95, 98, or XP. I hope you have Windows 98, because Windows
95 does not support DirectX 8.1, which you need to run the new
graphics engine. That could be your basic problem, either that or
the processor.
