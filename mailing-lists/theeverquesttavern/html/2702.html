I had the same issue. And looking at EQ's support page will not tell
you the correct fix for it. They claim that you should download a new
EverQuest.exe file and run it. This will not fix the issue. If you
read further on the page, they briefly talk about the patcher not
working correctly if Internet Explorer is not installed on your pc.
They FAIL to mention that you need to be running IE 5.5 or better for
it to work at all. I was running 5.0 and it would not connect to the
patcher. I downloaded and installed 5.5, rebooted, and WHAM! I'm
online.
