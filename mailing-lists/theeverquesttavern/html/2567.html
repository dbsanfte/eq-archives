As far as lingo, there's not much new I don't think except for two
acronyms. For some reason there's a new love for acronyms.

PST is no longer Pacific Standard Time. It's Please Send Tell.

LFG is looking for group. Which makes sense and is also a tag command
you can use anyway. Course I think LFG may have first occurred on
servers about 8 months ago so you may already know that one.

There's a lot of new Beastlord stuff and content, but that'll be fun
for you to explore.

Only really unobvious one is the command /tgb. This toggles on and
off Targetgroupbuff command which is the command to be able to cast
group buff spells on other groups.
