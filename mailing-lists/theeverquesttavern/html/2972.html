It means just that. data4.cab file is either corrupt,
or the CD is dirty or scratched and it's having
trouble copying the file over to your machine. Does it
give you an option to ignore? Or does it immediately
kick you out of the install? If it allows you to
ignore, then ignore it. Once done installing, open up
the CD and manually copy the data4.cab file form the
CD (by right clicking and copy) and paste it into the
appropriate folder. If not, I suggest cleaning the CD,
and then try re-installing it again. Sometimes it
might give you that error, but doing it over clears it
up.
