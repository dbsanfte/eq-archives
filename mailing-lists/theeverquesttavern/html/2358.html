A new video card may help. Something you might
want to try before spending 100-200 bucks on a new
card is increasing your computers virtual memory. I
bought a new computer and immediately upgraded it to 512
Ram. I still couldn't get on. I checked my virtual
memory and it was set very low. Typically, the virtual
memory's minimum level should be the amount of ram you
have, i.e. 320. The maximum should be set to twice
that. On Windows XP, you can change virtual memory by
selecting the system icon in the control panel...click on
the "advanced" tab. Under "Performance", select
"Settings." Choose "Advanced" Tab and change the vm from
there. Hope this helps!<br><br>Tahksan - 17th Druid
