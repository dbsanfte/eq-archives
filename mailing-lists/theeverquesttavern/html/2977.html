Everquest is compatible with all versions of Windows (with the
exception of 3.11 maybe). It never changes. However, it's
incompatible with certain video cards if you have XP installed.

If it's an old CD, then it's probably scratched or dirty (which can
stop a file from transfering) and you may need to buffer it. Try to
get your hands of a friend's copy of Velious if possible and copy the
file from there. Have you tried canceling the download, and then re-
loading it over? Have you tried cleaning the CD Drive and CD?

You don't need to be some network engineer for that. :)
