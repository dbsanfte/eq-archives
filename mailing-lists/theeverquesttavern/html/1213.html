When I first bought the game I had the same
problem, what I noticed though was that all of the files
downloading were separate and even if it timed out at any
time, if 1 file was downloaded then you would need to
do it again. My advice would be to watch the
download, when one file has finished (you will know because
it will start downloading a file with a different
name) click on cancel and then start again. Takes some
time but it gets there in the end, never had any
troubles since as the size of the files are now so much
smaller
