manually... I go into the EQsupp that patched recently. Then
I look to see what the text color I want to update
is called. (They're all numbered). Then I pull up
Powerpoint or some other RGB enabled program. Then I adjust
the RGB numbers in that program till it's the color I
like. Then I plug those numbers into the RGB 3 code for
that channel.<br><br>It's a lot easier than I'm
explaining it. But again, it's up to your preferences. Other
than a few settings changes the EQ color editor works
