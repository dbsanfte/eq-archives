I buffed my pet a few times this way, and did notice that he hit for more
damage. The problem that I have with the spells are that they take strength
away from you. My outtings last a very long time before I go back to town
and sell, so weight was a huge factor only having a str of 65, so when you
take away 10, I would become encumbered and lose some agi as well, so I quit
using the spell. You are probably meleeing at this level too, so if you did
want to use this spell, use siphon strength with it so that you are back up
to your potential, but that takes 2 spells slots away from a lot of other
more useful spells.
