You need to turn the note back in with the other 3 items. If you look at
the note it says something like reminder: shadowed scythe... Since the
shadowed items disappear when you log out this allows you to get the scythe
at any time instead of it having to be the last thing you get (Like for the
robe, gloves, and boots). So you need all four items. If you have the
other 3 then get the scythe just give him the scythe by itself first to get
the note, then right away give him the note and the other 3 items.
