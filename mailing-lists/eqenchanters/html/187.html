Unfortunately, skill increase is very unpredictable. I've had it go up twice
in a row, I've had it go up on failure, and I've had it take 10 successful
jewelry attempts. I have *no* idea how to know when it will go up. Just keep
using the cheap gems until you get the "too trivial" message and then move
to the next gem up in the list.
