Much about the same thing.

All buff firsthand + pet (no pet in some dungeons). Keeping the pet alive
in the fight is my main startegy solo, so from time to time I enter the
fray to draw the monster to me. It make sure the pet is alive when big
taunt spells (choatic feedback family) are used.

- In group according to case lull/soothe (ex hitting orc camp in EC)
- Tashan is the opener. Doesnt draw a lot od attention nor mana and help
both the group and me.

- Suff. sphere and Enfeeblement
Allow mob to hit so it activates the pet, plus enfeeblement helps a lot in
case of wildlife. Suff. being DOT its better to cast it early.

After that depends on the situation.
- Whack the mob with a pointed stick.
- Chaotic feedback is something else is approaching.
- Root to run away and/or Fear (but you have to wait for the pet then).
Fear is especially useful if another mob start hitting me/grp from behind
(but it is slow).

Note rank 2 pets hit magics wpn only creats (eg wisps).

- In group color flux if against many monsters.

Trouble?. Fear + leave pet to die and zone/run away.
