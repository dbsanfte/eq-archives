Subject: Is Jewelry worthless or what?


I found this web site that displays all the jewelry we can make up to
gold+ruby...Well it really looked worthless to me at first glance. After
thinking a bit, I thought: well, if that's all we can get in the game, then
players will buy it. Then I started a little research around and I found out
that you can find much more powerful magic items that we can ever make on
monsters...so what's the point? Who's gonna pay 150 plats to buy a +3 INT
ring if he can get +6 INT Helmet by camping at a certain place.
