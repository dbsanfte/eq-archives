Looking at the numbers at castersrealm and some of the old
figures I have, the number figure they list matches up with
the attack speed percentages i'm getting.

i.e.
SLtW = 160 = 160% = +60%
Augmentation = 115 = +15%
Quickness = 120-130 = 120%-130% = +20% (L16) - +30% (L18?)
Alacrity = 122-140 = +22% - +40%
Celerity = 128-150 = +28% - +50% (I'm guessing)

The only "sweet" combo I can see when grouped with an
enchanter is Verses (for speedup) + Augmentation (to cut
stamina drain).

Incidently, +60% is not "over double the attack speed." It is
only 1.6x normal attack speed. They are getting the %speedup
confused with the %delay_reduction.
