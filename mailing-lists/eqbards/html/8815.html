True, not only is it impossible to verify this but according to the EQ
Guide document thing that snuck around for a while both apprentice guides
and full guides are not allowed to create items for players, even in the
case of item loss. Senior guides are allowed to but it is usually for a GM
run quest.
So keep in mind that when a guide says they can't help it might not be that
they don't want to or know how to, it could be that their 'rank' doesn't
allow them to. You can always ask for it to be escalated to a higher guide
if you feel that it is important enough (and they are on, of course :)
