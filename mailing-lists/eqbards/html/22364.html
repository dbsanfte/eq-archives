the standard is 3 seconds, because that is the time it takes to start a
song.

the way pauses work is that you add the pause command FIRST on the line,
then the casting command separated by a comma. the pause must be first, but
it will be executed second.

/pause 30, /cast x

hope that helps. the only bad things about macros so far is that they can
not account for missed notes and you can not type while they are running. i
have used them, but not as often as i hoped i would when they came out.
