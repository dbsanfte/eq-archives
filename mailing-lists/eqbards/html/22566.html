I've been partnering up a lot recently with a druid using this tactic and it
is the easiest, fastest, and best EXP I've ever gotten. So easy that I've
gotten 3/5 of the way thru 45th level in less than a week. Druid pulls with
harmony, druid snares the mob he is pulling, druid roots the mob. I start
chanting and he DOTs. Only time we take any damage is if root breaks early.
I imagine getting thru 46 will be a breeze if we continue to use this tactic
since we will kill the mobs even faster when I get my 3d chant.

Best group combo I have been in for chanting was me, a druid, and a wizard
in Skyfire. Wizard would pull, as he got close I would hit it with
curtailing, he would root. Druid stacks dots, I used curtailing + frost +
Clarity. The root would usually break just as the druid dots wore off, wiz
would drop an IC on it and it would either die or turn and run. Was so easy
it almost felt like exploiting :)
