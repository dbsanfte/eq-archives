I've been going there the last 3 or 4 nights now. We typically start up top,

the dungeon is about 15-20 min. So if you have a raider spawn when you get
there, expect to wait 30-40 min for the next one. I'm also guessing there
could be phs so that would mean if you got a ph, it would be 60-80 min in
between.

Anyways, he's there a lot. If you want the coin, I suggest bringing a group
and holding the library for 2 hours, I'd virtually guarantee you leave with
a coin.
