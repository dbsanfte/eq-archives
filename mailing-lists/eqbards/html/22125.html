Galtin,

Open you eqclient.ini file, and locate the following line:

RunMode - FALSE

Replace the FALSE with TRUE, save the document, and restart EQ, it will
default to Run every time.

There are a lot of other interesting defaults in there, too. Take a look at
the following link for more info.

http://eq.castersrealm.com/viewarticle.asp?Article=2076

Hope this helps.
