I use setup Draelon is using, pretty much (including remapping Use CntrScrn
to '+'). I also have Target Nearest PC set to '*' and Target Nearest NPC to
'0', so I can do almost all navigation and manipulation with one hand. The
only times I use the mouse are when traveling through hilly terrain
(MouseLook makes it much easier to see the leader going up and down dunes),
targetting when there are too many mobs (usually I'll just move around and
hit '0', though, if possible) or the mob is so far into the wall that Target
Nearest won't work, or when manipulating inventory.
