I found that the quickest way to /assist someone in your party was to use
the F keys that are already setup for targetting party members. Actually i
modified mine. The default config is F2 for the first party member in your
list all the way down to F6 for the last. I changed this to F1 thru F5 for
party members, just easier for me to remember. I hit the F key for whoever i
want to assist and quickly type "/ASS" which is the short version of
"/ASSIST" =)

No need to type out "/ASSIST JOEBLOW", takes too long, especially if the guy
has a funky long name and you need to get in the fight quickly. /ASS works
as long as you got a player targetted. Well, it usually works...the assist
command can be touchy sometimes.

I hope this helps!
