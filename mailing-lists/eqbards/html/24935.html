Subject: *HOAX* Virus Warning - DO NOT PASS ON!


There is currently a hoax running around that the file named SULFNBK.EXE is
a virus, and should be deleted before it "activates" on June 1st. It is
not a virus. In fact, it is a Microsoft Windows utility that is used to
restore long file names - which means that deleting it could cause you more
problems than it solves.

If you receive this email, the appropriate thing to do is to delete it.

For more information on this hoax, go to
http://www.symantec.com/avcenter/venc/data/sulfnbk.exe.warning.html

And, as always, keep a good, up-to-date virus scanner active on your
computer, and never open any attachments from someone you don't know or
don't trust. Especially don't open anything you're not expecting.
