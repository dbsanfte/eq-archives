Well, the fact that your pages end in .htm implies it's
probably an NT server. But under Unix, assuming the original
page is at "bard/eqbards.htm" all you'd have to do is:

(in the same directory as bard/)
ln -s bard bards

(in the bard/ directory)
ln -s eqbards.htm eqbards.html
ln -s eqbards.htm eqbard.htm
ln -s eqbards.htm eqbard.html
