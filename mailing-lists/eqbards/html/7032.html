As to the philosophical issue, they can scrap it. Taking an hour to bind at
prime time (remember, just because others can bind doesnt mean they will) is
a hassle. Many times I have entered a off the track town (Halas, RV,
Kaladim) and been the only one there. Makes adventuring unfun knowing if you
die its a long walk. I am not saying that any class other than the actual
caster classes should get the bind spell, but being able to be bound by one
of those classes anywhere would not disrupt the balance too much if they
made it equal to all classes by implementing the same thing as after being
resurrected. You respawn at half hps, mana, stat hits, etc. It would skirt
around the issue of exploiting by binding in a lair somewhere, and having
the melee classes constantly charging to die again. (though most forget that
while caster classes dont necessarily need thier equip, melee classes and
hybrids are equip dependant and dont function effectively without it)
On the real subject of the NPCs casting bind, I don't see how it would be
that big of a deal. NPCs are around that will cast healing or cures for
either a donation or an item, the casting of the bind can be set up to do
the same thing. Easiest to do would be set it up on the priest of discord.
Theres one in every city, so it could easily be accomplished (at least in my
mind, if they can set npcs up to cast the heal spell, should be able to
effectively cut and past - crude terminology I know - the code onto another
npc with the bind spell replacing the heal spell.)
As for bards getting the bind spell, it would be nice, but a pipe dream I
think. With my bard I bind at the bank, grab my spare drum (I keep a few
extra) and zip back to corpse in no time. Giving the bard bind opens the
doorway to other classes binding (rangers use druid spells and druids get
binf, pals use cleric spells which can bind, etc) and that is an issue
Verant doesnt want to touch. Gate potions though have been introduced into
the game, so a bind potion as well seems feasible. Make it city only like
the spell. If you try to get the melee class bound outside the city it
fails. I would only assume the potion (procing the same spell) would have
those same guidelines.
