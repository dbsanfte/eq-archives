I found it very easy to kill the fish and not get killed by a guard. Still
took the faction hit though. You need to have the invis song though for
this to work. Also, I recommend against doing this quest now. I don't
think any class but an enchanter can complete it with the recent patch
because the some of the quest items are now NO DROP. If you still want to
know how to do it, see below for spoiler:

Spoiler space




















Stand along the side where the three NPCs are. Look for the fish and
observe its path. It follows the same path every time. About halfway down
the pool, it turns around and starts over. Get into the pool at that
location. Make sure you are behind the column so the guard in the pool
can't see you. Equip your lute and sing underwater breathing song. When
fish nears turnaround point, melee attack. Don't use bellow or chords. The
guard in the pool did not attack me when I did it this way. When you kill
the fish, start singing Invis song. When I got out of the pool, a guard
walked right by me and did not attack.
