Like my friend said, this game is a lot more fun if there's
the threat of death hanging over you. All the overcrowding
and camping IMHO made for really sloppy play. If things had
never gotten that bad, you would only see Ykeshas and such in
the hands of people who knew what they were doing (or knew
someone who knew what they were doing). Mistmoore is a lot
like that - buggy enough to be deadly if you were stupid.

Anyways, here's a tip with Pixie Strike. Once you target a
mob and start a targeted song, you don't have to keep it
targeted. You can switch to a different target, attack,
etc. and the Pixie Strike will still hit the mob you had
targeted when you started the song.

So what I'll usually do if I'm doing both crowd and runner
control is Strike, but retarget the mob everyone else is
fighting and monitor it's hp bar. When it gets near the point
where it'd start to run, I switch to a PS/Chain cycle.

- Pixie Strike
- wait for the nods message (so I still have it targeted if
it's resisted, since it fails way more than Chain)
- target the mob that will run
- start Chain
- immediately switch back to the mob I'm PSing

repeat

Several classes also have quick-casting stun spells which can
be used to stop runners. These can usually give you enough
time to get a Chain or root cast.
