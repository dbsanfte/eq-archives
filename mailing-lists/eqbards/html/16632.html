Subject: Keeping mob in agro range


I use bellow to keep mobs stuck to me that I don't want drilling me in
the back (like stuff with BS).

I use bellow (not sure if there is a better song to pull with at higher
lvls) to pull them, fire up Selo's (it shoudl still be active from when
you were looking for them)

Start the run back, after Selo's pulses, try to bellow on the still
targeted mob. If he is not in range, stop, and keep mashing the bellow
button. As soon as he is in bellow range I take off again, repulse
Selo's and repeat as necesaary. I found this to be the safest way to
pull backstabbers...
