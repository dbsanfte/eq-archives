just a few thoughts:

we've had several machines using the via 133 (not 133a) chipset locking up
with various video cards using the later drivers. unfortunately, it's not
one specific video card, or driver...it seems to be somewhat random.
as for your problem, make SURE that voodoo is not overheating. if it's
being jerky and whatnot sometimes, and locks up with newer drivers, sounds
like you have a clockspeed/heating problem. for gits and shiggles, try
running the machine with the case open and a small house fan blowing through
it. if that seems to cure the problem, then you have airflow blockage, so
you need new fans/more fans, something. also, check to make sure that the
cabling for the hard drive/floppy drive, etc. are not hanging in the way of
the fans/airflow. we use zip-tie strips to get them out of the way of the
airflow when we build machines at work.

good luck
