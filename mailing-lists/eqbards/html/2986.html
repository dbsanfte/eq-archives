Subject: Tick and pulse and Accelerando


Ok, I took my own advice and timed these. From pressing the
song button to when the song effect kicks in is 3.5 seconds
(did this several times and got 3.5 seconds each time). For
pulsed songs (Accelerando, Restoration, etc), the delay
between pulses is 6.0 seconds (timed 5 pulses).

So in theory, you could twist:

<pre>
t t'
0.0 - Push 1st song button
3.5 0.0 1st song effect kicks in, stop song
4.0 0.5 Start 2nd song
7.5 4.0 2nd song effect kicks in, stop song
8.0 4.5 Start 3rd song
9.5 6.0 1st song pulses for second time
11.5 8.0 3rd song effect kicks in, stop song
12.0 8.5 Start 4th song
15.5 12.0 4th song effect kicks in, 1st song pulses 3rd time
16.0 12.5 Start 1st song
19.5 16.0 1st song effect kicks in 2 sec ahead of "4th" pulse
</pre>

So given a really fast computer, connection, and perfect
timing, you should be able to twist 4 songs and still have
2 seconds to spare. :-) Hmm, maybe I should put a metronome
next to my computer...
