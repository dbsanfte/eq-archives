The Psalm series is an "upgrade" to the Rhythm series in that
they are singing based - there is no instrument associated
with them. You can get the full resist bonus while using your
weapons.

As someone else already said, you only get the message when
*you* are hit. If someone else gets hit, you only see the
"burned" message. And the message for Psalm of Cooling is
"chilled to the bone."
