Er, that's the problem with that whole thread. The mode does
not equal the average. So it's really pretty useless for
comparing which weapons you want to use.

If you can prove that the damage distribution for all
weapon/atk combinations is identical, then you can use the
mode in lieu of average for direct comparisons (i.e. given two
weapons, which is better, don't care how much better - this
may actually be the case given some L50 tests I've seen).
But to properly compare two different weapons, you still need
to pull out the average damage by parsing your logfiles.
