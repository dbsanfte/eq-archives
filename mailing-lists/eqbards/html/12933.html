I've had occasions where using the song will not let me start
a quest, but using it with a flute will. Also, the effect
seems to become greater as you level (raises you to a higher
faction). Stuff that I could only take from dubious to
dubious now goes from dubious to apprehensive.

I happened to run into Paglan while helping a friend chase
