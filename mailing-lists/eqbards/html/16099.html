Secret Tip from someone who worked with VI pretty closely for a long time:

Get an email together
get a TON of people to "sign" it
then send a copy of the email to:

<server>@...

Replace <Server> with the name of each server (without spaces) ie:
therathe@... or lanystvil@...

Send a copy of this email to EVERY server

Why?

Sending to that address sends the email to the following people:

All Senior Guides for the server
All GM's assigned to the server
All GM LEAD ADMINS for the server

IE: You can hit every single GM at Verant with the complaint, directly,
rather than posting the complaint to 1 message board where it will rapidly
get burried or ignored.

Just my 2.5 cents ;)
