Absor:

"I have some information about changes we will be making to a few bard
songs with the next patch.

Songs with a damage shield component will have that component removed.
Pixie Strike will have its secondary resist reduced.
Song of Twilight will be singable at any time of the day. Its secondary
resist will be removed completely. It will have a longer duration. And
it will require an instrument.
The single target haste spells will be self-only.

We will also be revising the bardic epic weapon.

This should all go live with the next patch, which for now we are
tentatively scheduling for late next week."



I'm not sure what to think.

The removal of the damage shields is sort of OK, it means that the songs
won't drop now. These songs aren't so great though, and it would have
been nice if something had been added to them. (A damage shield
perhaps).

Pixie Strike is good.

Song of Twilight is good.

Single target haste songs will be self only? Not sure what that means.
As far as I can remember off the top of my head, the only single target
song that isn't self-only is Jonathan's Provocation, which I have rarely
used, but it was useful to use it on other targets sometimes. It's
hardly useful to use it on myself.

Mostly good, I guess, if the secondary resist for Pixie Strike is low
enough. Of course, I'm long past using the song.
