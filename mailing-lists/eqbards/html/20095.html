Its easily possible to retarget the pet using a hotkey.
I have my hotkey set up like this: (Assist/Attack)
/assist <tank>
/pet attack
/pet target

Works great, I've been using it since 39. Occasionally I'll hit the hotkey
just as charm breaks, and end up with the (partially beat) wrong mob targeted.

I have another hotkey set up for when I want the pet to go after a
different mob from the one that everybody else is fighting(or soloing) It is:
/pet attack
/pet target


I've tried putting a /pet backoff in there, but the stupid pet never seems
to listen, so I've got another hotkey for backoff
