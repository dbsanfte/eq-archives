Fear works great in Kunark for the reason you just stated. I've been
fighting a lot recently in the Overthere using the chains/fear trick (or
snare if a ranger or druid in the group). If the group gets more than one
mob, the 2d mob without the chains (or snare) almost always never comes back
when the fear runs off. I'll go so far as to say that the best possible
party in the Overthere (for maximum EXP gain) is a bard, ranger and a rogue.
Ranger snares, bard fears and the rogue backstabs out the ying yang. The
mobs go down quick and hardly anyone takes any damage.

In Kunark if you use AE Fear you can actually fear the mobs so far away that
they don't come back. FUN method of crowd control if a pull gets out of
hand. Snare what you're fighting then fire off Fear. Everything else runs
away at mach speed and usually doesn't come back because it runs outside
it's chase radius.
