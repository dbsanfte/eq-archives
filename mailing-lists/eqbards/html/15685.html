In my experience, the darkness line is inferior to snare.
Stuff with darkness on it will start to run *before* the
darkness will halt their movement. So they'll still manage to
flee a bit depending on how quickly you're damaging it.
Early on Chain acts pretty much like snare. But later (level
thirty-something?) it will stop runners when they're at half
hp. That makes it superior to snare for stopping green
runners, since greenies tend to run when they drop below half.
