It's pretty short range (unless guards or merchants are
involved, in which case the range is as far as you can see
:-). I used this song to practice up my brass skill a bit. I
feared a bunch of snakes, skunks, etc. in Butcherblock and set
myself on a hill near a goblin camp. The critters would run
away, turn around, come back, run away, repeat. The folks
fighting the goblins would use me as a safe zone if they got
in trouble (run to me, let me fear the mob(s) attacking them,
rest and heal up, continue the fight). Anyways, with so many
critters running back and forth, you get a pretty good idea
what the range is as a bunch of them will run away while some
incoming ones just a little further away not be affected. I
would estimate the range as being about the same as melee
range.
