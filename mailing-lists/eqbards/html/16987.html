I'm pretty sure when the server calculates that a debuff
effect has been applied, it just sends the client a message
saying "apply this debuff, duration is 18 seconds (or
whatever)." If there's any server-side lag after this
message, the client's 18 seconds can be over before the
server's 18 seconds are, and the server will apply the effect
after the icon is gone from the client.

Some effects (like Accelerando) are client side, so aren't
affected by this.
