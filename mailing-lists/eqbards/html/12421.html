I read about the MM event on the E'ci board at the Shards website.
http://shards.xrgaming.net/

For the rest of you out there who were not aware of this website, the Shards
website provides message boards for each server in EQ, similar to EQVault.
The good thing about the site is that hardly anyone knows about it so it
allows a core group of players to form a close community. I think the E'ci
board is the only one that actually ever gets used. The other server boards
have little to no traffic. Message traffic on the Shards E'ci board is
actually greater than that on the EQvault E'ci board. Just passing this
link along in case you and your friends want to start using it. For E'ci,
almost all server events like Planes and Dragon raids are coordinated on the
Shards board.

This has been a public service announcement.
