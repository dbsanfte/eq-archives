Hmm...

Well, the general impression from several group members that I have had, is
that Jig dramatically increases the speed of mana regen. An informal timing
(without stopwatch) went like this:
empty mana bar
two bubbles filled, no song
start jig
three bubbles filled, in about the same length of time as the first two

three or four different meditators have reported this effect to me. So, I
have taken to running restoration until heals are nearly to, or all the way
to full, then switching to jig, to get the spellcasters mana up quickly. My
impression is that there is a fair amount of speed increase. (I have also
been using the forage timer for some of this - i.e. click on forage, start
song, wait for forage to stop, compare with not having song running)

I'll try to get some guys in my guild to do some more extensive testing,
with stopwatches, this week, and report the results. Oh, yeah, if this
works the way I think it does, by shorting out the timer on meding, then
level/skill should not matter at all.
