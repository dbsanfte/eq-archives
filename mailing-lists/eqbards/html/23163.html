Here is how it decodes by eqcaster:

Symphonic Harmony
Lowers you on the target's hate list by -100
----------------------------------------------------------------------------
----
Classes: None
----------------------------------------------------------------------------
----
Range to Target: 100 feet
----------------------------------------------------------------------------
----
Skill: Wind Instruments
Allowable Targets: All
----------------------------------------------------------------------------
----
Resistance Check: None
----------------------------------------------------------------------------
----
Spell Duration: Instant
Casting Time: 3.0 seconds
----------------------------------------------------------------------------
----
Spell cast on you: You feel at peace.
Spell cast on someone: Soandso looks peaceful.


Eqcaster is a program that reads your sp_eff.dat file and 'decodes' it.
Yes, this is client side and the server can change stuff but it seems to be
the best spell reference available. http://www.eqcaster.com
