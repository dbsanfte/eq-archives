IMO, to successfully use this technique on a consistent basis you need to be
going against blues. Yellows and reds may resist too much. Could still
work but you make take a lot of damage. Against blues I don't lose much at
all with this technique. Start off with chain, then fear and then start
whacking them from behind. I've been using Furfils as the 3d song to debuff
their resist rate. Bellow might be a good option for you. Biggest thing to
be concerned with is that you don't aggro a second mob while chasing the
first one.
