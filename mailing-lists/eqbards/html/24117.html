Subject: Level 56 songs


I never bothered with these. The descriptions kind of confused me, I
hate the 12-hour timer, and I've been stuck dragon hunting or gathering
exp in Sebilis for a level or so, hardly the places to test things. But
I was looking for the Ancient Jarsath in FV the other night (any hints?
does he have a PH?) and figured I'd try them out.

Song of Highsun - described as "Stun, Cancel Magic, Shadowstep" on
eqsongs. Shadowstep is described on eqcasters as "short distance self
only gate travel (900 feet)". I'm not sure what 900 feet equals. What
this song does (I really should have tested it by targetting myself,
I'll try that next) is Shadowstep the TARGET. So it gates the MOB. The
message is: "<mob> gates." 2 of 7 times I used this, the MOB
returned. The other 5 times it did not. This was in FV, so those MOBs
might have run into some enemies. I need to test it further to see just
HOW far it throws them. It was out of visual range (on a pretty flat
surface) in any case. Has some interesting implications, and sounds
REALLY dangerous. You can probably kill people using this on a blue
server by singing it on the MOB they're fighting and hoping it trains
them. (I'm surprised Verant didn't think of that to begin with.) I did
find it pretty useful soloing; you can use it to summarily dismiss the
add-on junk.

Song of Midnight is described as "target: fear/SoW". In fact, it's AE
fear/SoW, as I discovered when I sang it near some guards. :( This is
in some ways more useful than normal fear, since the SoW lets the MOBs
run a lot further before the song expires, and they have to return at
normal speed. It buys you more time, in other words. And I suppose it
is possible for them to get out of range this way (although they do not
appear to do so if you stand still). The drawback is that, since they
go farther, they can train more stuff back to you. I'd probably use
this instead of the other fear song if it weren't for the 12-hour timer.

Any other observations?
