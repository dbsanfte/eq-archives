Okay, Okay...

I was an admin on an LP MUD for a long time...
I know LP and DIKU are not exactly the same...
As for the XP tables in LP/DIKU...they started out the same way...
These numbers are generalized...if you want the real ones, let me know and
i'll log into my old admin char, and get the actuals from the internals of
the mud.
Level 1 = 100 xp
Level 2 = 250 xp
Level 3 = 600 xp
Level 4 = 1300 or 1400 xp
Level 5 = 3000 xp
Level 6 = 6000 xp
etc.
Level 18 = 225,000 xp
Level 19 = 490,000 xp
Level 20 = 1,000,000 xp
i seem to remember that each level was a bit more than double the one before
EXCEPT the (n*5)+1 levels, which were exactly double the (n*5) levels

This table was manually adjusted for each lp mud, level 20 = 1,000,000 xp in
most
(initally, all lpmuds were based on 20 levels...many lps advanced this to
50, or 100 or even more)
(also, I helped develop the xp tables for 20-50 when our mud decided to let
char's go beyond the 20 limit)
we decided that doubling every time after 20 was too much, i.e. 2 mil, 4
mil, 8 mil, plus there is a limit to the variable in it's value to hold the
requisite number of places (about level 32, you run out of bits:)
we decided on 1.2(prev level xp) for each level over 20, therefore making it
a linear run from 20 to 50.
1 mil, 1.2 mil, 1.44 mil, etc.
the run from 1 to 20 was definately not linear, that i remember, although we
tweaked ours out, due to specifics in our mud...

oh, god, my brain is twitching trying to remember all that lp code.

MacGrough 'Former MUD Admin' Mich'Rathe

p.s. in the old-style text muds, once you hit level 20, you were either
stuck there, or given the opportunity to become a 'wizard', i.e. coder. our
mud changed that to 'anytime from 20 to 50 you desire, and can find an
existing wizard to sponsor you, you can 'wiz' (be promoted to coder).
many others follow(ed) this same suit, from what i understand.
Incidentally, there is a 'mud campus' where all the code is exposed, and you
can learn what code does which, if you so desire.
