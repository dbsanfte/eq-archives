I started playing a bard in beta, and one thing that was added later which
could helped me - If I could have gotten used to it *grin* was to make
macros with something like this:
song1
/stopsong, pause xx? (Its been awhile since I fooled around with it so dont
remember the timing..)
/cast 1

song2
/stopsong, pause xx
cast 2


That way twisting can be done a bit simpler since each hotkey stops the
prior song before starting the next:
hit: song1, song2, song3, song1, song2, song3

If you fizzle you have the small recast delay that is needed after stopsong,
but nothing like if you mapped several songs to one macro.
