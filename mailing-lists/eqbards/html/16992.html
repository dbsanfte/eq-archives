Actually, Accelerando will slow down to non-drum speed whenver
you lose a buff, gain a buff, or zone. I think these events
cause a reappraisal of what buffs you currently have on, and
the spdata.eff file lists song effects without instruments.
So when your buffs are reappraised, the song effects get set
back to non-instrument values (need to test this with a buff
song).

It's usually not a problem because our songs refresh so
quickly. It's annoying enough that I /bugged it a long time
ago though.
