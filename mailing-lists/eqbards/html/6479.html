Here are some additional changes which occured in the patch that were not
announced. I know you all are probably getting
petitions on alot of these so hopefully this information will allow you to
inform players that it was a change and not a bug.

If there are any others we may have missed please report them to your server
GM so that we can verify that they were indeed
intentional changes and not bugs.

Thanks,
Solist

Normal arrows (item 8005)have had their range reduced from 50 to 25. This is
because fletched arrows are range 50 and a
flectcher making his own arrows should have better arrows than generic
storebought ones. No other arrows were affected

-The item Glowing Stein has been changed to Stein of Moggok

-The Lustrous Russet Armor is now warrior only, a previous error had caused
it to be used by several other classes despite the
fact it was always intented to be warrior only

-The Fletching kit now has 8 slots. This is to allow more items added into
the process to make new bows.

-New tailoring items have been added. Tailors can now make the silk series
of armor. The books on how to manufacture these
are also in the game, but are spawn drops and not vendor sales.

-Vendors who previously sold just arrow nocks now sell all of the fletching
materials.
