Subject: RE: Complaints, how do we make them?


> If I have a problem/complaint/suggestion for
> Verant, what is the best way for me to go
> about sending it to them (where are they most
> likely to see it)? Any consensus?

The best way if it's not a serious complaint, is with
the /feedback command. If it's a game bug complaint,
then use the /bug command. If it's about another
player on the server use the /petition command to get
the assistance of a guide. Otherwise, if you want to
speak with someone at Verant by voice, call the
technical support number in the back of your manual.
Page 42 if memory serves correctly.
