It definitely does. I use a minotaur horn all the time. I never really
had to spefically practice my wind skill. When I got invisibility, I ended
up using it so much, that it goes up to max each level on its own.

A tip, if you're in a train prone zone, like Mistmoore for example, and
you're in a group that has mana dumpers. When the party is done being
healed up with Hymn, kick in group invisibility. It's saved my party and
my own life on numerous occasions. Of course, this only applies between
levels 19 and 31. (Not that I would know about the latter level...)
