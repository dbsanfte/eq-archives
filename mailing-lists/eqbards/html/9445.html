Bellow is a DD damage song, I'm not sure, but I think it does 8dmg
now? Basically you can use it like an extra attack (think of it as an extra
sword). Another benefit is that it will sometimes interrupt a caster. If
attacking a mob (with any means) would cause others to come after you, then
so will Bellow. In other words, if there's a single scarab you want to
attack, go ahead, but if there are 3 dervishes, bellow would bring all 3,
just like using ranged weapons would do.
