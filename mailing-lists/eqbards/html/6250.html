I agree with you whole heartedly John. To those who dismiss effects like
the JBoots and SOW, I prize them greatly when I am corpse retrieving for
my party in Mistmoore. Without SOW, I would be left at the standard run
speed which isn't useful when I am try to avoid the nasty trains started
there.

It is unfortunate that at this juncture, no major changes to the
looting system will be made. The best suggestion that I have heard is to
simply flag items with a serial number that incorporates the item number
and user id number. For instance the Shiny Brass Shield is item number
0200 and you are user 00000001 then the serial number for the shield would
be 020000000001. If you ever try to loot another you will not be able to
pick it up and it will be left on the corpse. This is by far the best
solution I have heard, no need for lore items, no need to worry about
farming and the ability to sell such items remains. The only drawback is
what happens when someone tricks you in a trade. BTW if there are
destroyed the item serial number is removed from the database.
