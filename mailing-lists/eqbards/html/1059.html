I got this item while exploring the world. A couple notes:

- It is a quest item. It will not go in the bank, and you
can't give it to anyone. If you're planning on using this
some of the time (to get better prices when buying stuff), and
planning to use real neck armor otherwise, you'll have to
carry it in your inventory all the time. It only weighs 0.1
or 0.2 lbs though.

- It boosted my charisma to 115. I got charged the same
prices as I did when I was at 110 charisma. Maybe it's
broken, maybe there's a cap on the price benefits for charisma
at 110, maybe it only helps with really expensive items.

- It is AC 2, which is not bad so it might be a good item to
have if we can verify any of the charisma effects that we've
speculated about.
