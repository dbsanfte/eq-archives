I managed to put myself into this wedged state without a
magical weapon by hitting my song hotkeys too quickly. So I'm
sure doing so can unstick you (though I don't know how to do
it consistently). Try hitting your song hotkeys really
quickly for a while.

It's a simple programming issue that's causing the problem.
The client and server get out of synch. The client thinks a
song isn't playing, the server says it is. You try to play a
song, the server says you're already playing. In the opposite
case (what happened to me), you try to stop playing a song,
the server says you're already stopped. They just need to add
something to resynch the client and server.
