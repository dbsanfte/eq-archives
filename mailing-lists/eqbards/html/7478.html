When I started playing I had an EQ manual and a cd. There was zero useful
info on how the classes broke down. I picked a class that I thought would
be fun. It is fun. I do enjoy my class. Do I get frustrated by seemingly
arbitrary things my class does not have? Absolutely. Would I play this
class knowing everything I know now if I had not grown attached to the
character that is Kitasi? Probably not, doesn't mean I don't enjoy Kitasi
but if some bug rendered him unrecoverable I am not sure I would start up
another bard without some changes having been made. Not sure what class
would fit my image better, an enchanter maybe, gotta love any spell that
allows me to do more quests which only leaves enchanters and bards. Hard to
say, but just because I am having fun doesn't mean I don't see room for
improvement. After all the reverse argument must hold true if the primary
one does. If binding is not such a big deal, then why don't all the classes
eventually get it at some level. Point is, it is a big deal, just a big
deal that has been arbitrarily assigned to some classes and not others
because of some idea of a world system. Not necessarily bad in most cases,
but I think it is in this case. If I lived in Hawaii like some on this list
I would despair of ever getting bound in my what would be my prime playing
hours. You better know someone real well who can bind and you would
probably be restricted to playing with those people where those people
wanted to play.
