Yes, it is a "wish", but a technically non-challenging as they already have
the code in place for doing this already.

It would just be nice to know how crowded certain zones. I don't think that
this conflicts with any ideals that Verant has as they've already
established precedents. They give you server statistics so you know how
crowded each server is, why not zones?

If Verant really wanted to at least restrict usage of this, they could set
up a "board" in each city that you go to (just like the message board). You
click on it and see a window that lists the number of people in each zone.

Regarding "omnipotence" concerns, if Verant was worried about that, there
wouldn't be a "who" function to begin with. Regarding "anonymity", someone
would have to search each zone in order to find the person. IMHO if Verant
intended "anon" to really hide whether you're in a zone or not, they
wouldn't have anon people show up in a regular /who of the zone you're in.
