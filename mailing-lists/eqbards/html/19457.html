Subject: Chains (was Bard song fixes...)


Can anyone come up with a reason Chains should be in testing? Until the
earlier message today about chains not stopping runners in their tracks at
low health I was always under the impression this song worked fully as
intended. The only reason I can think of that for it to be in testing is
that it wears off prematurely sometimes. Example: I've been using
chains/fear combo a lot the last few times I've played. I weave the two
with Celerity. On numerous occasions the following has happened: 1)
initial chains wears off (yellow text message), 2) restart chains and get
the blue text message it sticks, 3) immediately get yellow text message
saying chains has wore off again. So either one of two things is happening:
1) Mobs are getting a second chance to break the chains earlier than its
normal duration (by design?) or 2) chains doesn't always stick even if you
get a success message (bug).

Something else I've noticed that may be a bug. Sometimes after I chain the
mob they will resist my first Screech attempt. I then restart screech and
if it sticks skip celerity and go straight to another Chains. On a few
occasions I will get the blue message that the mob is chained before I get
the yellow message that the initial chains has wore off. But when I do get
the message that the first chains has wore off, the mob starts to run away
again at full speed. So in effect since the second chains didn't stick even
though I got the message it did. This has only happened on a few occasions
so I didn't think anything of it at the time. I only remembered it when I
saw the message that they were testing Chains for some reason.

Its been a real long time since we've had to discuss Chains on this list so
if these have already been discussed then I've forgotten about it.
