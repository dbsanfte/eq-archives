This has been discussed on the official forum and Abashi commented on it.
If you search the www.eqvault.com archive - you could probably find it.

To paraphrase as best as I can - Verant feels that the "quality" of the game
is in part based on the fact that each zone has a unique look-and-feel based
on the artwork and design. They are afraid if they did this - things would
start appearing "generic", etc. He pointed out the upcoming zones (I
believe there are a couple of lower level ones they are working on) and
"other things" (which would probably be the server split).

On the one hand - perhaps this is a cop-out. On the other - he does have a
point. Perhaps a single duplication wouldn't be bad; but the zone
duplication itself could end up being a cop-out. Instead of really
addressing the issue - just keep dup'ing the zones - and out of necessity
you'd always be in the same place - and it all WOULD like the same.

While their track record in bringing up new servers has been dismal; I
believe the server splits are probably the best solution. Once a server
gets to a certain threshold of average prime-time utilization - split it
into 2. But you damn well better make sure all members of a guild end up on
the same server...

If they could do this in a timely fashion - that would really solve the
problem...
