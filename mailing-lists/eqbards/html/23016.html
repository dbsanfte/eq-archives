The problem with using our Cancel Magic song to remove spell effects is that
it checks against your MR. I think, but am not 100% certain, that Cancel
Magic will always remove the first spell effect. So if you are rooted, in
order to get rid of it the root icon needs to be listed first. Then the
song has to actually succeed a MR check. When I want to remove my DE
illusion, I usually have to try 2-3 times before the song sticks.

Using Disenchanting Melody to remove root would work OK IF you didn't have
any other positive buffs. It could easily remove one of your cleric buffs
instead of the root.
