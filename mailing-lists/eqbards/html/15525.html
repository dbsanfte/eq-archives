This is unrelated to this particular problem, but I'd recommend that you
remove "Fast Find" from your startup folder in Windows.

This little program, installed mostly by default by MS Office, searches your
drives and can cause heavy disk activity at the most inconvenient times. But
like I said, I'm sure this is *not* the cause in this case, but I always
make sure this program never gets loaded on my computers.
