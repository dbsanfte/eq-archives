On all big hunts I have been on there is a
designated looter in each group,
usually the group leader. This person is
responsible for looting the corpse
if the group gets the kill as soon as
feasible. If the group leader dies
before the battle ends then someone in the
group, usually at the direction
of the group leader, will loot and report.

One way to tell if a corpse has been ninja
looted is if it gives no coin.
As, to the best of my knowledge, mobs will
always drop some coin, except of
course newbie mobs.
