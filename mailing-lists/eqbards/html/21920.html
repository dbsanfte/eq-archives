Druids quad kited raptors SOLO? Are you sure about that? Raptors have on
the average about 4k hp. A druid over 49 has one AoE, Lightning Blast,
that does 477 dmg for 225 mana. so assuming no resists, that would be
1800 mana. The other AoE is much less efficient. Lightning Blast is
magic based so it tends to get a lot of partial resists.

I'm not saying it's impossible but it sure looks like you'd be cutting
it real close with mana. It's way more efficient now for a druid to use
root/dotting (rotting). I can solo pull a raptor, stack 3 dots on it,
and med back to about 1 1/2 bubbles of mana by the time it dies if I'm
lucky on resists and root breaks. The L49 (and 56) roots last over 2
minutes with a break check every 30 seconds so unless you run into a
clump of bad rolls, it's pretty likely the first or second root will run
the whole 2 minutes.

I've never seen druids quad kiting when I've been out there. I see a lot
of wizards kiting about 6-8 around though. Now that's impressive. But
I've also seen how hard it is for Wizards to get groups so I guess it
balances out.
