This might be too late to help (I'm having a problem getting any bards email), but I thought I'd throw my 2 cents in.

EQ does not have any (or any important if someone proves me wrong) registry settings. I've reformated my computer and run EQ off the D drive as soon as it was operational again. Knowing this, you can just copy the Directory and files over to the new computer and make a shortcut to Everquest.exe on your desktop. Don't bother wasting your time making a fresh install and taking all the updates. I'm assuming you have a way to copy files from computer to computer.

As a side note, you "could" copy just the Everquest.exe file to a new directory, run it, and it will take ALL the EQ files. I did this in beta when a friend let me check out the game using his account. Shhh, don't tell. : )

2 thoughts...

#1 patches only update the files, so installing fresh, and overwriting the
files should work.

#2 if you are really worried about that, AND other programs you want to
move, get Cleansweep. This program allows you to archive programs, and
includes all files and registry entries, as well as allowing you to clean up
the registry, temp files, and other goodies. use Cleansweep to make
archives of every program you want to move to the other machine, copy the
archives to the new machine, and use Cleansweep to de-archive them.
IMPORTANT NOTE: You are only licensed to use Cleansweep on ONE machine at a
time. So, install cleansweep on the older machine, do your archiving, etc.,
then uninstall. Afterwards, you can install it on the new machine and
dearchive, restoring all files/registry entries. This is probably the least
painful way to move to a new machine, and keep all your settings.
