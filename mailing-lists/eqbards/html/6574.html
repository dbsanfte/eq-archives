To clarify: Target the mob (that should still be Struck from
last round), start up Strike, then immediately move for a
better angle at the next mob. Don't wait for the "glaze" or
resist message before moving. If it's resisted, simply Strike
it again. If it takes, target the next mob (who is now right
in front of you) and repeat the process.
