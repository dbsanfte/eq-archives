I myself run multiple computers through a single DSL connection and single
IP address. I used to run Sygate as my gateway/firewall, but I had that
setup on an extra machine with 2 network cards.

I have since gotten a Linksys DSL/Ethernet router which allows me to have 4
computers all networked together, using the single IP address for internet
access. It also acts as a firewall, so you don't need to install software
on your computer.
