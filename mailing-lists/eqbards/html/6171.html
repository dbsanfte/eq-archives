I was doing the same thing in N.Ro tunnel - off in a cornor giving a concert
for Shady (I want to marry him some day - he must be really rich!)

But instead of just leaving and going to bed - it was middle of the
afternoon - I would go off and do some chores and come back. Well, I
noticed (or so it seems) that if you start and stop the song - you get 2 or
3 fairly quick skill checks then it takes a long time for more.

start song
tick
tick
skill check (you have become better...)
tick
tick
skill check (you have become better...)
tick
tick
and no improves for a long time - some but not really quickly.

So, I'd start song and go off for 20-30 minute. Come back - stop and
re-start and go off. Did this for a couple hours and nearly maxed the skill
- was like 86 I think - in only a couple hours. Without the starts and stops
- it seems like you would need about 8 hours.

Sylly.
