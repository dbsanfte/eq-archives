I really shudder when I hear people recommend that you reformat your drive
to install a new OS. Sure, if the drive's data is totally trashed, that may
be the way to go. But if you would end up losing tons of data by
reformatting (and honestly, how many people actually have a backup device
other than their floppy drives at home???) the easiest way to do a fresh OS
install is just to rename the Windows and Program Files directories.

Rename the windows directory to win.old and the Program Files directory to
prog.old and then install Windows as though it were a newly-formatted drive.
You will obviously need around 300mb free hard disk space to do this.

After everything is done installing, you can go through the win.old and
prog.old directories to restore any settings, data, or whatever files and
delete the rest.
