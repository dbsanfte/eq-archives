It's not an item flag! It's an option in the client to just not allow you
to drop items. It could be a no drop/trade switch for all I care so you
just can't click it in an empty area. It has _nothing_ to do with your
equipment other than not letting you dump it on the ground.

And to those who think it's a difficult code change:

The client already checks to see if you are dropping the item or if you are
giving it to someone. All they'd need to do is add a flag:

If Drop and playerwantsdrops=yes then item drops
Else item stays

That's really rudimentary pseudo code but it would be VERY easy to stick
into the client code. It's not even a server side check.
