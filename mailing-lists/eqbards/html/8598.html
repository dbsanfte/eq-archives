1. Use a brass horn, not the Mino Horn (wind, as previously noted)

2. Sing DDD, not Screech. It is at least remotely possible (they've made this
type of error before) that Brass is NOT the instrument of choice for Screech. It is
not as if there is a valid way to measure the effect reliably, and since it can be
sung, you would not get a message that you were using the wrong instrument. I know
that DDD will work for you. Any remote area is fine.

3. If you enjoy ticking people off, practice this in High Pass. You're perhaps too
high level for this to be fair, but I nearly maxed my skill (and got a lot of exp and
loot) doing nothing but standing at the corner tooting my horn.

and, assuming that you've done all this already (and I'm beating a dead horse):

4. If you play DDD with a brass instrument for oh, say, 30 minutes (5 should do it)
and get no skill increases, you should /petition. You have a problem.
