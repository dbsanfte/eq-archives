yup, make sure you get directx 7.0a...
i'm using about the same setup you are:
piii-600cm, 128 mb (256 when i get the cash), Asus geforce 256, sblive
xgamer, cable modem

couple of things to check:
BIOS SETTINGS! <- make sure those guys are right...and you might see if you
need a bios update, as well (from the motherboard manufacturer, or machine
manufacturer - Gateway used to be great about keeping those things
updated...haven't had one in a couple of years, though)
make sure the bus speeds, etc. match your setup, voltages are right etc.
(don't overclock if you don't have to)

if all of the above doesn't fix it...try loading nVidia's reference drivers
for the geforce and see how it works (you'll prolly lose some creative
functionality, but you can at least check it)
