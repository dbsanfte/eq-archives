At 09:42 AM 11/16/00 -0500, you wrote:
> >Now, number pad 8 still hits hotkey 2, but shift+number pad 8 does nothing.
> >I need to hit shift+top row 2 to change to hotkey bank 2.
>
>Find the section in your eqclient.ini file that has these lines in it:
>KEYMAP_HOTPAGE1=SHIFT+1
>KEYMAP_HOTPAGE2=SHIFT+2
>KEYMAP_HOTPAGE3=SHIFT+3
>KEYMAP_HOTPAGE4=SHIFT+4
>KEYMAP_HOTPAGE5=SHIFT+5
>KEYMAP_HOTPAGE6=SHIFT+6
>KEYMAP_HOTPAGE7=SHIFT+7
>KEYMAP_HOTPAGE8=SHIFT+8
>KEYMAP_HOTPAGE9=SHIFT+9
>KEYMAP_HOTPAGE10=SHIFT+0
>
>
>Either change them to something like:
>KEYMAP_HOTPAGE1=SHIFT+NUMPAD7
>KEYMAP_HOTPAGE2=SHIFT+NUMPAD8
>.
>.
>.
>
>Or change them to something like this:
>KEYMAP_HOTPAGE1=SHIFT+1
>KEYMAP_HOTPAGE1_2=SHIFT+NUMPAD7
>KEYMAP_HOTPAGE2=SHIFT+2
>KEYMAP_HOTPAGE2_2=SHIFT+NUMPAD8
>.
>.
>.
>
>
>The first will mean that you can only change pages using the shift-number
>pad#...the second will mean that you can change pages with the shift-#, or
>with shift-numpad#. I noticed that in-game, you cannot add bindings like
>shift-numpad#, or alt-numpad#, but it supports it, you just have to change
>it in the ini file. Also, in game (at the keyboard mappings page), you
>left click on the first binding spot, and to access the second binding
>spot, right click on it.
>
>An important thing to note....be sure you unbind any keys that are using
>SHIFT-NUMPAD#....else the game will execute both commands at the same
>time. (Oh yeah...I'm using "bind" to refer to keyboard
>assignments...it's an old Quake habit of mine :)
>
>I also noticed some stuff that they didn't put in the patch message.
>1) They moved CTRL-# casting to ALT-# casting (too near the Winblows
>key) (but this can be rebound to ctrl-#)
>2) ALT-casting now works for Bards
>3) A lot of the / commands (like /sit /camp) now work while we're singing
>songs.
>4) You can use one hot key bank for your full inventory and two slots from
>your equipment list (like melee 1 and 2)...since you can open backpacks in
>full screen mode, there's no reason to go to the inventory screen now.
>5) there's a bunch more keyboard shortcuts like CTRL-B for spellbook,
>CTRL-D for disband, CTRL-I for invite/follow, CTRL-S for sitting/standing,
>etc...check your keyboard mappings in game, or read the ini file : )
>
>Couple more things that I like about the patch...the turning off attacking
>on autoassisting (our cleric tends to un-mez every mob sometimes when
>she's trying to heal :) Since I'm so used to using 6 keys for spells,
>etc, I can use the next four keys for my weapons and inventory (I'm
>compulsive about always knowing what I have in my hands...this way I can
>see it on all my hotkey banks : ) 108 new socials means we can do a
