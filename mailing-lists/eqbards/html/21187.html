Subject: Text history loss when switching screen modes


It seems that Verant has inadvertantly screwed bards again...

After the patch last week that added the text wrapping feature (and caused a
slowdown when switching from full screen to windowed; ie. going into
inventory, looting, spell book, etc.) the text in my chat history box would
sometimes disappear. It would only happen once in a while, however, so it
was not that big of a deal.

Last night however, we were doing a practice Kaesora raid with mostly guild
members, and EVERY time I switched from full screen to windowed, my chat
history text would dissapear. Someone in my group mentioned that he only
noticed that problem when grouped with a bard.

I think I'll post this same message on Verant's official message board, not
that they'll notice it, but at least it'll be there.. hehe
