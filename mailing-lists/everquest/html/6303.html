Heres the deal... on a dial up connection aint gonna be a whole heck of a lot you can do about lag.  In the Display Options tab drop your Far clip Plane to nil.  Change Opacity to nil.  those two things will be your biggest help. 
 
Your computer has a built in graphics chip so not a lot of ways to change the video options on it.
 
If virtual memory is low... do CTRL ALT DEL and end any processes in the background that are not necessary... for a new Compaq thats probably quite a bit.
 
Add more memory.  If possible, add a new video card.  If possible, get DSL or cable.
 
If all else fails, use the /viewport command..  The less area yoru graphics processor has to draw, the quicker it can draw it and the less lag you'll see.
