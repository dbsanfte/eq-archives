The best answer to your question is go with whatever your system's motherboard can support.  If you're running a PII, there is a possibility you might not be able to use a 4X AGP card.  The max it may support is 2X.  If that is the case you'll have to go with the card that will fit your form factor regardless of which one it is.   You might even want to open the case to make sure you actually HAVe an AGP slot and that the graphics aren't built into the board.  It would suck to buy a card for $200 only to find out you can't install it.    But all that aside, assuming you can use it and either will fit, then I'd say yes, go with more memory.  I use an ATI Radeon 9000 128MB 4X AGP.  I've had excellent performance with it.  But the choice is yours.  Either way, you'll still get better results.
