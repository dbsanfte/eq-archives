Hardware is fairly inexpensive these days. If your into computers and games
in particular you will have to stay slightly ahead of the curve. Currently
you can pick up memory pretty cheap.. I got mine from googlegear.com.. They
have 512megs of memory for as low as 60$ Keep in mind that only a few years
ago that could have cost you up to 1$ per meg. Besides they released the
requirements for Luclin months ago, there was time to upgrade prior. You
can still certainly play the previous Everquest as well as the prior
expansions without changing much if anything.. If it's the directx 8
problem, you can pick up compatiable cards (geforce2 MXs) for as low as 35$.

Old puters work just fine. The only problem you have is that your video card
*MUST* be dx8 certified. Other than that even going as low as 128 megs will
work. There is not the slightest chance in hell you can install luclin on
that machine tho, cause it will die while trying to render the new graphics.

And you should really get more ram, it isnt that expensive. Ram and a nice
video card and everything works a lot smoother. CPU power isnt that
important.

512mb for $100 isnt steep. And your machine needs to be nicely configured.
For example if you run WinXP old world eq and me would be like 256 megs to
run ok but on 98 128 works ok and 160 is fine.

And dont be so negative, SoL is a whopping expansion worth the new specs.
Graphics are cool (some char models are a bit weird) but mobs/surrondings
are very nice. The mob features are a LOT improved adding more tactics to
fights.
