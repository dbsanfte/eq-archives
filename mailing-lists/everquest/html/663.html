The XP Penalties/bonuses break down as follow:

Human gets a 20% XP bonus over other races - all other races (Warrior class
to avoid class penalty) gain xp at the same rate (unless one of the
following applies)

Hybrids take a 20% xp penalty

XP giving kills in a "dungeon" zone give a 25% xp bonus

So a human pali would level at the exact same rate as a Dwarvish warrior.

There are other class restrictions, but I honestly don't know the
restrictions for them. I do, however, know that all INT based casters level
at the same rate if they're race xp penalty/bonus matches up.
