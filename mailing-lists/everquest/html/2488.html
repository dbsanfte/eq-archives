there are hints about the bandwidth EQ needs on the official EQ site,
but it is somewhat hard to find ... it is sort of information for ISPs
so they know how much traffic within an hour or so can occur playing
this game.

i only tried those "multiple" sessions using ISDN and T-DSL (the "T"
just stands for the german provider "t-online"; they give almost every-
thing a name starting with "t-" ... the only thing you can't get from
them is a t-rex ;-)

what i learned is, that you even can have a good system (fast cpu, good
gfx cars, enough memory), but having a not so good or worse connection
can extremly slow down your system (basically this is a problem of the
protocols used to transfer packets over the internet).

so if you start to think about doing this, check for a very fast and
reliable connection. best thing is to ask high level bards, what ISPs
they use.
