To get to the ftp site:

Enter the ftp address into your Internet Explorer web browser.  It will bring you to the ftp site. The file you need is called EQBetaPatcher.zip.  Just right click on the file and copy it to your desktop.  Unzip the file and copy betaeverquest.exe into your root EverQuest directory (Should be in C:\Program Files\Sony\EverQuest unless you changed your default installation.)

Run this every time prior to logging in to the beta server. You will see the Beta Server on the server list, choose that server and log in.


HOW LONG IS BETA?

The beta test is scheduled to begin on July 20th, 2005 and is scheduled to end on September 13th, 2005, unless we decide to change these dates.


HOW DO I GET TO THE DoD ZONES?

When you get onto the beta server, create your character and head to the Plane of Knowledge. Find Old Man Thorren and he can teleport you to Corathus Creep.


HOW DO I COPY MY CHARACTER ONTO BETA?

For us to copy your favorite EverQuest character to the beta server, please indicate which character you want us to copy by logging into your play server as your favorite character. You indicate that you want us to copy an individual character by typing the command /beta in your chat window. Periodically we will run scripts that will copy characters to the beta server. Keep in mind that we will be running the character copies on an as-needed basis - not necessarily at regular intervals. It may take several days for us to copy your character.


THE /betabuff COMMAND

The level range needed to test all of the variations of dungeons for  DoD is huge.  In order to give you that ability to test as much of the expansion as possible, we have created a /betabuff command.  Here's how you use it.

1. Create a first level character on the beta server and log in with that character (this command will only work on 1st level characters).

2. Type /betabuff CLASS LEVEL - where CLASS is the class of the character you would like to create, and LEVEL is the level that you want that character to be.  LEVEL can only be numbers divisible by 5, starting at 20.  So 20, 25, 30, etc.

That's it.  The character you created will be transformed into a character of the level and class you have chosen.  Your race and even gender may change as well.


BETA FEEDBACK

There are two methods for passing information to us about what you experience during the beta.

/bug should be used to report bugs.  Bugs are gameplay features that do not work, spelling errors, and anything that just does not work as intended.  Please give us as much detail about the bug as you can think of when reporting it.

/feedback should be used to send in comments about parts of the expansion that work properly, but that you would like to see changed.  If you discover an item that you think should have less Wis and more Int, then use /feedback.  If you think a certain encounter is too easy or too hard, use /feedback to report it.

It is important that you distinguish between these two options, as one is read primarily by our testing team and the other is read mostly by our game designers.  Sending a bug through /feedback can delay our ability to fix it, since we'll have to route that to our testers for verification.

Please use this command to explore as many of the adventures as you can and report your findings to us.

If you find yourself in dire need of assistance, you can use /petition to contact us for help.  Please only petition if the situation is one that you can't correct yourself.  For example, if you used /betabuff to create a character and you have lost the corpse, there is no need to /petition.  You can just log out, create a new character and use the /betabuff command again.
