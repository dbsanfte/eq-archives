At 05:36 PM 11/29/00 -0000, you wrote:
>Hello All,
>
> OK, so I'm wondering how I view the details of an Iteam... I see
>iteam details on web pages all the time. I would like to be able to
>get this in the game so the next time I see a Ring Cloak I won't buy
>it thinking I can wear it.

All you have to do is right click on the item, but *hold down* the right
mouse button. That will bring up the stats screen on the item. You have to
do this in a trade window, though, not while someone else is wearing it. If
you right click/hold on an item someone else is wearing it will just
display the name of the item. Right click/hold also works if an item is in
YOUR inventory, or if the item is being offered by a merchant.

Hope this helps,
