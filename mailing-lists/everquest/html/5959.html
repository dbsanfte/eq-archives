Yeah but my motherboard only supports it in 4X mode.  And I did some snooping around ATI's website and found out that the 9600 series cards do not support Fast writing so SMARTGART (a safeguard utility that ATI drivers use for maintaining system stability for unreliable AGP bus communiocation) doesn't even recognize that it runs in 4X compatability mode.  I think it's because Fast writing has to be enabled in the BIOS, and my motherboard doesn't have that option in the BIOS.  so I can either upgrade video cards to a 9800 which does support Fast Writing or upgrade the motherboard to one that supports 8X compatability.  But honestly, I don't think I'm gonna worry about that for awhile.  The system seems to run fine as is and EQ1 and EQ2 are both pretty stable so I'm happy.
