Runs great on my computer, but a little slow on my second computer. But both
computers are easily playable for 2 group situation encounters. You really
need to tune your computer so it will run EQ and SoL the best for your
computer. Verant has some many options now that help cut down on lag.

There is one EQ ini.client program that will help you set all your defaults
to maximize your performance. The web site is
http://www.uncon.com/reverend/eqiniedit/ The owner of this program really
stays on top of patches too. He already has a /hitsmode option to help set
you the way you want it. Although, I still not that happy with the /hitsmode
options, but it is a step in the right direction. What I want is another
window, that I can resize that has ALL combat chat in it. Right now, I still
see my damage shield message, Taunt message and Disarm message. I think
Verant is really working hard to get all these nice features into the game.

But in your ini client file, you can turn off dynamic lighting, on server
side filter, on/off armor texture, Texture Quality, Each character module on
or off and many tracking options. One of the best fixes for SoL on my second
computer was the use of Umbra. This really helped my zone times in the old
world and overall lag in all zones.

Try it out, go to the message board and ask for features. I really like this
program and first used it to get my chat colors just the right color.

Sistily - 60 Warrior
