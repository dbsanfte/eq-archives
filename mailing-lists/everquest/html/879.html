Welcome..

I believe you install and run the program. This includes all the patches.
Next you will come to a screen where you can create an account, Its one of
the buttons on the bottom right. It will as you a few question.. Name..
Location.. etc. There should be a number on your cd's jewel case. It will
ask you for that number and for you to create a login name and password.
Finally after everything is completed you can click on the connect button.
Type in your username and password, this will connect you to the server
select screen. Pick one of the green servers (for they are the least
crowded). That's it... Next screens that come up allow you to create a new
character.


Hopefully this helps.. Imp sure I might have something out of order but the
basic information should be correct. It's been almost two years since I had
to create an account. =)
