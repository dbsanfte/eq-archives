I don't like powerleveling much, either.  As I had stated in my original post, I just want to get the husband's char up a level or two so we can travel together.  He's 10th and I'm 16th.  I've been playing for a while (though I had a long layover) and he's only been playing for a month or so.  If I remember correctly, isn't the formula: level + (level/2) used to determine who can group and still gain exp?
 
I don't think that I would ever want to be powerleveled, myself.  It seems like a big pain in the neck to get all one's stats up to par.  I can also honestly say that I really don't care if I ever get a character up to 60th lvl.  What's the point in reaching the end game as fast as you can?  I say have fun along the way and roleplay like you would with paper and dice.  Okay, maybe not speaking in an mock English accent to the computer screen (at least not when anyone's looking), but still!
 
By the way, has anyone heard anything about the new roleplaying server?  Has there been any word on when/if it will launch?  I'd heard that they were trying to work out the kinks to make sure that no one could communicate if they didn't speak the same language.  I can see that this would be a problem if people used /emote to speak to each other, or even use their character description.
 
How many people would leave their present servers and characters for the RP server, I wonder?  I think I'd give it a try.
