I searched for that dark beginnings game… nothing … that’s odd.  I know there are several private servers trying to build games around the original EQ game engine.  I really like this one.  http://www.shardsofdalaya.com/  It uses old school EQ but some of the improvements are there too.  It isn’t the same as “The Game” as the devs call EQ, it uses pretty much the same zones but many are “re-imagined” with different mobs, drops etc… still, enough is the same that it brings back that OLD school EQ feel.  Here’s the best part… it’s FREE!  I wish SOE would put out a free version of EQ to attract players but I don’t see that happening.  Maybe a server that you can only level up to 25 or something, and then pay to relocate if you want to subscribe… oh well…  This game has it’s own wiki and apparently has found legal sanctuary from SOE.  Some reports suggest that Sony has even helped them out a time or two.  There was a lawsuit early on, but they changed it enough for it to be it’s own game. 

 

Here is the wiki site  http://www.angrygamer.net/sodwiki/index.php?title=Main_Page

 

Hey, I still play live server EQ and although it’s vastly different now than it was in the old days, it’s still my favorite game.  However; to get that old nostalgia going, this Shards  version of EQ is a blast. 

 

Oh, one more thing… check out Youtube for vids of the game in action, there are several.

 

Enjoy
