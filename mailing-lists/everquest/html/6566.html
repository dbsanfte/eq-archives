Yeah, that's what I figured.   This Dark Beginnings is probably another one of those private server deals.  The ULA that Sony puts with the game would tell you that it would be illegal to use Sony software in this particular fashion, but that still doesn't stop people from doing it.  It would have to be free because it's the only way it would be legal.  If the person or persons who own the server profit from it then theoretically they'd ahve to give Sony a share of the profits  for using their software.  However, copyright laws are such that if the game is changed significantly by a certain percentage from the original then it's its own entity and not breaking any laws.  I suspect that's what these people did with the game you mention.  All they'd ahve to do is get permission to use the game engine then they can change whatever they want and run whatever server they want.  If the game develops enough they might be able to go commercial with it, but that's usually after years of development.
