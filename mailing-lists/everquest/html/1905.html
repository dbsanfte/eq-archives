I must say that I too have made the switch, haven't touched EQ since DAoC came out.  Dark Age has had the benefit of hindsight and learning from EQ's mistakes.  They have built upon them.  There is less downtime, more character specialization, less annoying corpse runs, better graphics, easier grouping, and better pvp.  DAoC is better than EQ currently, however, to EQ's credit -- Dark Age wouldn't be what it is if it hadn't been for EQ's experience.
 
If you like EQ, I think you'll like Dark Age if not as much, perhaps better.  The only drawback is that you need a pretty high-end system for smooth play (particularly a good graphics card).
