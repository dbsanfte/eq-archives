I'm running Win2k Pro on:

AthlonXP 3000+
512 mb PC2100 ram
Geforce FX5200 (128 mb)
80 gb ATA133 hd
Custom pagefile - min is 1.5 gb, max is 3 gb

I've got all models except Erudites and Ogres on, music and spell effects
off as a matter of preference...everything else except gamma is set to the
max.

No lag. Runs like a dream.

My connection is 1.5 down/384 up DSL through Alltel, using the provided
modem and shared using another machine as router and firewall.

Someone mentioned defragging....PerfectDisk seems to be quite nice and it
can do it while the machine is idle automatically. I highly recommend it.

I used to run WinME on the machine and it ran like a bat out of hell, even
faster than 2k, but I was eventually forced to drop it during some
troubleshooting. Problem turned out to be bad ram but no one would actually
listen to me when I insisted ME was innocent here, and had ran quite nicely
before the ram started acting up. It took installing another OS before
anyone else thought there was *really* a problem.

Not one word about WinME. I know what's generally thought of it. :-)
