as far as the person with the playing problems. Here
is what I would try:

1. uninstall everything associated with the game
including Direct X., REBOOT,
2. Then download Direct x 8.1 from Microsoft.com.
3. install Direct x, REBOOT
4. Install EQ BUT BUT BUT, when you install it this
time, change the directory to something like c:\EQ and
not the default.
5. after installing eq reboot and try again ti run

NOTES: if it still does not work, make sure your
software firewall is not running, make sure you are
not running TOO high of a resolution for your monitor.
Most important, try shutting down any application
running in the background you do not need, even
disable antivirus software just as a test. I did not
say remove the anti-virus but disable it.

Let us know if any of this helps
