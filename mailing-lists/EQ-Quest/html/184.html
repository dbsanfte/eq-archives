It supposedly heals between 80-100 hps, it's a single charge item, however
the cleric's guild will recharge it for free.

deputy who gave it to you and ask him, again "Am I one with the wall" he
will then return the ring.

So when fighting Nillipuss, if you use the ring, make sure you are prepared
to get to the bank should the ring disappear. I saw it happen, it gets
kinda scary.
