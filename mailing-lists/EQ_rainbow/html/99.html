Looks like you have the job! Now lets make this guild so we can all play
together.

The way guilds are currently made is:
- everyone logs into the chat server
- guildmaster issues a #makeguild command to create a chat room for
the guild. Everyone else already knows a password for the guild
(it's been posted to this list).
- everyone joins the chat room. Everyone in the chat room types
some command to say they approve of the guild.
- once ten people say they approve, the guild master types in a
description of the guild. They we just wait for approval from the
GMs.

As guildmaster, I think it would be fine if you selected the guild name
based on our conversation on this list about what we would like in a guild
name. The next step would be for you to declare a time for us all to log
into the chat server to approve the guild. Any evening should probably be
fine, we only need ten people to get it approved.

I'm looking forward to this guild, it should be a lot of fun.
