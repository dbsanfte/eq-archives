I think it would be great to have more Mystical members on Brell. Rather
than remove all of David's playing time in training everyone individually,
maybe we could have some kind of a newbie event where people who don't
already have characters make new ones and get trained by those of us who
are already on brell. I've done that kind of thing before, and it can be
fun and social.

I don't want to suggest to anyone that they leave their old server and
friends, but it can be fun to be on multiple servers as well. If you're
playing on some other server every night, having a great time, then multi
server stuff may not be for you. If you find you're getting a little
bored though, maybe trying out a fresh character on a fresh server might
be fun.

I also think I mentioned before the idea of having an official "guild
night". The basic idea would be every Tuesday evening (or other randomly
chosen time) would be the official time to play on Brell with your
Mystical Alliance character. This would allow easier grouping, more
chatting fun, etc.
