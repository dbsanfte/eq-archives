I have looked through my EQ directory and have seen a few things.

A.) The graphics/sounds are on your computer. They have names like
qeynoshills.xmi, qeynoshills.s3d. It may be posible to remove
fog/rain/darkness/ocean (the fogyness underwater: you could make water
crysatl clear)
B.) There is a file that contains a list of guildnames and a single player
name next to each of them. I assume this is the guild leader beacuse my name
is next to my guild. This file is full of giberish too which makes it hard
to read.
C.) There is a file that contains the names of every other file followed by
3 small hex values. This may be used to check if files have been modified.
Ex: qeynoshills.xmi d54sd3f46 54df7sd5g 6s8fs69sf

There are no files containing anything that is recognizable (to me) as info
regarding your character. After thinking about it it would not be posible to
store that kind of info on your computer and still have the ability to sell
your acount (on E-bay)
