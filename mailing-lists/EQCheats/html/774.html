A little hint:

I noticed that the *.opt files are kept on my local harddrive. They seem to have the
hotbutton layouts on them. It could save you a little effort in re-mapping the
buttons if you copy them to the laptop. At least the one for the character you are
playing. (eg. if the char you are playing is named Smitty then copy the smitty.opt
file to the laptop's EQ directory. When I found this out, I was tickled. Now I try to
create the same class/name on any server I log into. (E Marr goes down a lot and it's
nice having a backup game on another server.)
