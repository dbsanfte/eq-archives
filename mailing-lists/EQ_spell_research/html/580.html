Subject: ALL spells currently in game....


I have written a program (in QuickBASIC, of all things) that will read a data file in the Everquest directory and give you information about all the spells in the game. Information pulled includes skill used (alteration, abjuration), targeting type (specific target, Area Effect), text shown, etc. This would be considered a SPOILER to many people, so use it if you want. Be advised that many spells that are listed are either not implemented or have been made unavailable, so just because it is LISTED does not mean that you can get it. Examples are Winged Courier, Enchantment of Light, and Enchantment of Brilliance, all of which have been reportedly removed, but which are still listed in the data file.

Anyway, here is the URL. Use it if you like, and give me feedback if you want.
