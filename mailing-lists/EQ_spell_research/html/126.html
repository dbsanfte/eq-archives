Subject: Slightly out-of-context message


Would it be too much to ask that people not post
messages to this list using anything other than
plain text? This is a pretty standard request on
all the mailing lists that I have seen, and it
certainly shouldn't be too difficult for anyone
to do so. Typically, there is an option on the
toolbar with the message, or else you can set
it in the options menu of your email client.

The reason for this is that not everyone uses
a GUI email client, and hence messages that are
sent in via attachments (which some email clients
do when sending html emails) end up being a bit
tedious to navigate through.
